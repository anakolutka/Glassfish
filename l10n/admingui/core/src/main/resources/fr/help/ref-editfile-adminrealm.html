Propriétés spécifiques de la classe FileRealm

Les propriétés suivantes sont requises pour un domaine file.

Contexte JAAS

Contexte JAAS (Java Authentication and Authorization Service), identificateur du module de connexion à utiliser pour ce domaine. La seule valeur valide est fileRealm.

Fichier clé

Chemin d'accès complet et nom du fichier où le serveur va stocker toutes les informations relatives à l'utilisateur, au groupe et au mot de passe du domaine. La valeur par défaut est domain-dir/config/keyfile pour le domaine file et domain-dir/config/admin-keyfile pour le domaine admin-realm .

Au départ, le fichier clé du domaine file est vide, les utilisateurs doivent donc être ajoutés sur la page Utilisateurs de fichiers... avant d'utiliser le domaine file.

Au départ, le fichier clé du domaine admin-realm contient le nom d'utilisateur de l'administrateur, le mot de passe associé dans un format chiffré et le groupe auquel appartient l'utilisateur (asadmin par défaut). Pour plus d'informations sur l'ajout d'utilisateurs au domaine admin-realm, consultez Autoriser l'accès aux outils d'administration.

Remarque - Les utilisateurs du groupe asadmin dans le domaine administrateur sont autorisés à utiliser la Console d?administration et les outils asadmin. N'ajoutez à ce groupe que des utilisateurs ayant des privilèges d'administrateur.

La propriété facultative suivante est disponible pour un domaine file.

Assignation de groupes

Liste de noms de groupes séparés par une virgule. Tous les clients présentant un certificat valide sont affectés à ces groupes, comme par exemple employee, manager qui sont des noms de groupes d'utilisateurs.
