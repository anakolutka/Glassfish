Novo Domínio

Use a página Novo Domínio para criar um domínio.

A página Novo Domínio contém estas opções.

Nome

O nome do novo domínio.

Nome da Classe

O nome da classe do novo domínio.

As opções válidas são mostradas na seguinte tabela.

Nome de Domínio

Nome da Classe

arquivo

com.sun.enterprise.security.auth.realm.file.FileRealm

certificado

com.sun.enterprise.security.auth.realm.certificate.CertificateRealm

ldap

com.sun.enterprise.security.auth.realm.ldap.LDAPRealm

JDBC

com.sun.enterprise.security.auth.realm.jdbc.JDBCRealm

solaris

com.sun.enterprise.security.auth.realm.solaris.SolarisRealm

Para especificar um domínio personalizado, selecione o botão de opção inferior e digite o nome de classe do domínio personalizado que está sendo criado.

Propriedades Adicionais

Propriedades adicionais do domínio. As propriedades válidas dependem do tipo de domínio.

  • Para obter uma descrição das propriedades Domínio de Arquivo, consulte Propriedades específicas da classe FileRealm.

  • Para obter uma descrição das propriedades Domínio de Certificado, consulte Propriedades específicas da classe CertificateRealm .

  • Para obter uma descrição das propriedades Domínio JDBC, consulte Propriedades específicas para a classe JDBCRealm .

  • Para obter uma descrição das propriedades Domínio LDAP, consulte Propriedades específicas da classe LDAPRealm .

  • Para obter uma descrição das propriedades Domínio do Solaris, consulte Propriedades específicas da classe SolarisRealm .
