GlassFish Community - Installation and Configuration Guide

Installing GlassFish is simply unjarring the file.  Upon unjaring both the  the Common Development and Distribution License (CDDL) v1.0 and binary licenses are displayed which you must accept to continue the operation. 

Instructions to unbundle and configure GlassFish

GlasssFish installation requires JDK 5 and Ant 1.62.

  1. Download one of the bundles
  2. Run: 
    java -Xmx256m -jar filename.jar
  3. Run: 
    ant -f setup.xml

Ant is included in the GlassFish bundle that was downloaded. We recommend that you use that version. 

See the GlassFish Quick Start Quide for basic steps to start the server and deploy an application.
