Applications

La page Applications présente la liste des applications déployées sur le Enterprise Server. Vous pouvez afficher et gérer les applications qui sont déjà déployées et vous pouvez déployer d'autres applications.

Pour chaque application, les informations suivantes sont fournies :

Nom

Nom de l'application.

Activée

Coche si l'application est activée ou X si l'application est désactivée.

Composite

True si l'application dispose de plus d'un type de composant ou False si elle n'en dispose que d'un seul.

Moteurs

Types de conteneurs utilisés par l'application. Voici les différents types de conteneurs :

  • web

  • webservices

  • ejb

  • connector

  • appclient

  • jruby

Action

Liens vers les actions que vous pouvez exécuter sur le composant après son déploiement : Redéployer et Redémarrer pour tous les composants et Lancer pour les applications Web.

Déployer

Bouton pour le déploiement d' une application d'entreprise.

Annuler le déploiement

Bouton pour annuler le déploiement d' une application d'entreprise.

Activer

Bouton pour l'activation d'une application d'entreprise sélectionnée.

Désactiver

Bouton pour la désactivation d'une application d'entreprise sélectionnée.

Filtre

Liste déroulante qui filtre les applications par moteur.
