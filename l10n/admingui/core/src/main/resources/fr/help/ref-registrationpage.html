Enregistrement du produit

Pour enregistrer votre installation de Sun GlassFishTMEnterprise Server avec une connexion Sun, vous devez avoir un compte en ligne Sun. Si vous n'avez pas encore de compte en ligne Sun, Enterprise Server vous permet d'en créer un lors de l'enregistrement.

La page Enregistrement du produit contient les options suivantes :

J'ai déjà un compte en ligne Sun ou un compte SDN (Sun Developer Network)

Sélectionnez cette option si vous possédez déjà un compte. Si vous sélectionnez cette option, les champs suivants sont visibles.

Nom de l'utilisateur

Nom d'utilisateur que vous avez spécifié lors de l'enregistrement de votre compte en ligne Sun.

Mot de passe

Mot de passe que vous avez spécifié lors de l'enregistrement de votre compte en ligne Sun.

Hôte proxy

Nom d'hôte du serveur proxy de l'ordinateur sur lequel Enterprise Server s'exécute.

Ce champ est applicable uniquement si l'ordinateur sur lequel Enterprise Server s'exécute se connecte à Internet par l'intermédiaire d'un serveur proxy.

Port proxy

Numéro de port du serveur proxy de l'ordinateur sur lequel Enterprise Server s'exécute.

Ce champ est applicable uniquement si l'ordinateur sur lequel Enterprise Server s'exécute se connecte à Internet par l'intermédiaire d'un serveur proxy.

Je n'ai pas de compte en ligne Sun. M'inscrire.

Sélectionnez cette option si vous n'avez pas de compte et que vous souhaitez enregistrer votre installation de Enterprise Server avec Sun Connection. Si vous sélectionnez cette option, les champs suivants sont visibles.

Adresse e-mail

Cette dernière sera le nom d'utilisateur de votre compte en ligne Sun.

Mot de passe

Mot de passe à utiliser pour votre compte en ligne Sun. Celui-ci est sensible à la casse et doit comporter entre 6 et 10 caractères.

Ressaisir le mot de passe

Vous devez ressaisir le mot de passe spécifié dans le champ Mot de passe.

Prénom

Votre prénom.

Nom

Votre nom.

Société

Société pour laquelle vous travaillez, le cas échéant.

Pays/Territoire

Votre pays ou territoire de résidence.

Hôte proxy

Nom d'hôte du serveur proxy de l'ordinateur sur lequel Enterprise Server s'exécute.

Ce champ est applicable uniquement si l'ordinateur sur lequel Enterprise Server s'exécute se connecte à Internet par l'intermédiaire d'un serveur proxy.

Port proxy

Numéro de port du serveur proxy de l'ordinateur sur lequel Enterprise Server s'exécute.

Ce champ est applicable uniquement si l'ordinateur sur lequel Enterprise Server s'exécute se connecte à Internet par l'intermédiaire d'un serveur proxy.
