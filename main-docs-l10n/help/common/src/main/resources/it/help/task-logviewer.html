Per visualizzare i file di log del server DAS

  1. Selezionare il nodo server (server di amministrazione) nell'albero di navigazione.

    Viene visualizzata la pagina Informazioni generali.

  2. Nella pagina Informazioni generali fare clic sul pulsante Visualizza file di log.

    Viene visualizzata la finestra Visualizzatore log.
