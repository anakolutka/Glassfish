Para eliminar una escucha HTTP

Antes de empezar

Si tiene pensado eliminar las escuchas HTTP predeterminadas http-listener-1, http-listener-2 o admin-listener, primero cree una copia de seguridad de los elementos http-listener en el archivo domain.xml de Servidor empresarial para poder restaurar la configuración en caso necesario.

  1. En el árbol de navegación, expanda el nodo Configuración.
  2. En el nodo Configuración, expanda el nodo Servicio HTTP.
  3. En el nodo Servicio HTTP, seleccione el nodo Escuchas HTTP.
  4. En la página Servicio HTTP, marque la casilla de verificación situada junto al nombre de la escucha HTTP que está eliminando.
  5. Haga clic en Delete.
