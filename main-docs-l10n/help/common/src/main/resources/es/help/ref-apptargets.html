Destinos de aplicaciones

Utilice la página Destinos de Aplicaciones para ver los clusters de destino y las instancias de servidor independientes en los que se puede activar la aplicación.

La página Destinos de Aplicaciones contiene la siguiente información.

Nombre de Destino

Nombre del cluster o instancia de servidor independiente.

Activado

Muestra true si se ha activado la aplicación en el destino, o false si la aplicación está desactivada.

Equilibrador de carga activado

Muestra true si se ha activado el equilibrio de carga para la aplicación en el destino, o false si se ha desactivado el equilibrio de carga para la aplicación.

Servidores Virtuales

Para asociar la aplicación a servidores virtuales específicos en un destino, seleccione Gestionar Servidores Virtuales para ese destino.

La tabla Destinos también contiene las siguientes opciones.

Gestionar Destinos

Botón para gestionar los destinos de aplicaciones.

Más acciones

Lista desplegable con las siguientes acciones.

Activar

Acción para activar la aplicación en el destino seleccionado.

Desactivar

Acción para desactivar la aplicación en el destino seleccionado.

Activación de equilibrador de carga

Acción para activar el equilibrio de carga de la aplicación en el destino seleccionado.

Desactivación de equilibrador de carga

Acción para desactivar el equilibrio de carga de la aplicación en el destino seleccionado.
