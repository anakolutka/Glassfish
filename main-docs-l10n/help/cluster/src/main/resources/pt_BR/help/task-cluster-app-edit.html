Para Editar uma Aplicação que é Implantada em um Cluster

  1. Na árvore de navegação, expanda o nó Clusters.

  2. No nó Clusters expandidos, selecione o nó do cluster.

    A página Informações Gerais do cluster será aberta.

  3. Na página Informações Gerais, clique na guia Aplicações.

    A página Aplicações do cluster se abre.

  4. Na tabela Aplicações Implantadas na página Aplicações, clique no nome da aplicação.

    A página Editar Aplicação será aberta.

  5. Na página Editar Aplicação, forneça as informações necessárias sobre a aplicação e salve as alterações.

    Para obter detalhes, consulte a ajuda on-line de Editar Aplicação.
