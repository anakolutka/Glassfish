So starten Sie eine Webanwendung

Bevor Sie beginnen

Stellen Sie sicher, dass die zu startende Anwendung bereitgestellt ist.

  1. Erweitern Sie in der Navigationsstruktur den Anwendungsknoten.
  2. Wählen Sie den Webanwendungsknoten aus.

    Die Seite "Webanwendungen" wird geöffnet.

  3. Klicken Sie in der Tabelle "Bereitgestellte Webanwendungen" in der Zeile für die zu startende Anwendung auf "Starten".

    Die Anwendung wird in einem neuen Browserfenster geöffnet.
