Editar Protocolo

Utilize a página Editar Protocolo para modificar as definições de um protocolo de rede existente.

A página Editar Protocolo contém as opções a seguir.

Carregar Defaults

Botão para restaurar definições para seus valores default. Definições que não tiveram seus valores default alterados.

Nome da Configuração

O nome da configuração à qual as definições nesta página se aplicam. Este campo é somente para leitura.

Nome

O nome do protocolo. O campo Nome é somente para leitura. Você só pode especificar o nome quando cria um novo protocolo.

Segurança

Se essa opção for selecionada, a segurança será ativada para o protocolo. Se a segurança está ativada, você pode usar a guia SSL para modificar as definições de segurança.

Se você estiver configurando o admin-listener para o servidor de administração de domínios (DAS), que usa a configuração server-config, esta opção será somente para leitura. Para ativar a segurança deste protocolo, é necessário ativar a administração segura. Consulte Administração Segura.
