Service HTTP

La page Service HTTP contient les options suivantes :

Journalisation des accès

Si cette option est activée, elle active la journalisation des accès pour tous les sous-éléments de serveur virtuel qui ne spécifient pas cette propriété. Si cette option n'est pas sélectionnée, ceci désactive la journalisation des accès pour tous les sous-éléments de serveur virtuel qui ne spécifient pas cette propriété.

Taille du tampon du journal d'accès

Taille, en octets, du tampon du journal d'accès, ou valeur inférieure ou égale à zéro pour les journaux d'accès non mis en tampon. La valeur par défaut est 4096.

Intervalle d'écriture du journal d'accès

Nombre de secondes écoulées avant l'écriture du journal sur le disque. Le journal d'accès est écrit lorsque le tampon est saturé ou que l'intervalle expire. Si la valeur est inférieure ou égale à zéro, le tampon est alors toujours écrit même s'il n'est pas plein. Autrement dit, chaque fois que l'utilisateur accède au serveur, le message journal est directement enregistré dans le fichier.

Autres propriétés

Propriétés supplémentaires du service HTTP.

Les propriétés suivantes sont disponibles :

connectionTimeout

Indique le délai (en millisecondes) d'attente pour que les listeners HTTP présentent la ligne d'URI de requête, après avoir accepté une connexion.

La valeur par défaut est 12,000 (12 secondes).

maxKeepAliveRequests

Indique le nombre maximum de requêtes HTTP pouvant être émises en pipeline jusqu'à la fermeture de la connexion par le serveur. Définissez cette propriété sur 1 pour désactiver la connexion Keep-Alive HTTP/1.0, ainsi que la connexion Keep-Alive HTTP/1.1 et le pipelining.

La valeur par défaut est 1000.

traceEnabled

Si elle est définie sur true (vrai), la méthode TRACE est activée. Définissez cette propriété sur false (faux) pour protéger Enterprise Server contre les attaques de type Cross-Site Scripting.

Pour plus d'informations sur la méthode TRACE, reportez-vous au RFC 2616 : Hypertext Transfer Protocol -- HTTP/1.1.

La valeur par défaut est 'true' (vrai).
