Editar Recurso JDBC

Utilice la página Editar recurso JDBC para modificar un recurso JDBC existente.

La página Editar recurso JDBC contiene las siguientes opciones.

Nombre JNDI

Nombre único que identifica el recurso JDBC. El nombre JNDI es un campo de sólo lectura. Sólo puede especificar el nombre JNDI al crear un nuevo recurso JDBC.

Nombre de pool

Pool de conexiones asociado al recurso JDBC.

Descripción

Breve descripción del recurso JDBC.

Estado

Si esta opción está activada, el recurso está disponible en el tiempo de ejecución. Esta opción se encuentra activada por defecto.

Propiedades Adicionales

Propiedades adicionales para el recurso JDBC. GlassFish Server no define ninguna propiedad adicional para recursos JDBC.
