To Create a Load Balancer When Creating a Cluster

As part of the cluster creation process, you can create a load balancer. If you do not want to create a load balancer at this time, Click Next and continue with cluster creation. You can create a load balancer later.

  1. Click the "Use a load balancer for this cluster" checkbox.

  2. To use an existing load balancer:

    1. Click the "Add this cluster to an existing load balancer" checkbox.

    2. From the pull-down menu, select the load balancer.

  3. To create a new load balancer:

    1. Click the "Create a new load balancer for this cluster" checkbox.

    2. Enter a load balancer name. This name must not conflict with any other load balancer groups, agents, configurations, clusters, or sever instances in the domain.

    3. Enter the host where the load balancer will reside, for example the web server host.

    4. Enter the port used to communicate with the load balancer. For example, the web server port. This port must be SSL enabled.

    5. Enter the proxy host used by the load balancer for outbound HTTP.

    6. Enter the proxy post used by the load balancer for outbound HTTP. This port must be SSL enabled.

  4. Click Next.
