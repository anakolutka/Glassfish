Aplicativos

A página Aplicativos exibe uma lista dos aplicativos que são implantados no Enterprise Server. É possível visualizar e gerenciar os aplicativos implantados, e você pode implantar mais aplicativos.

Para cada aplicativo, as seguintes informações são fornecidas.

Nome

O nome do aplicativo.

Ativado

Uma marca de seleção se o aplicativo estiver ativado ou um X se o aplicativo estiver desativado.

Composto

Verdadeiro se o aplicativo tiver mais de um tipo de componente ou Falso, se tiver somente um tipo de componente.

Mecanismos

Os tipos de recipientes usados pelo aplicativo. Os tipos de recipientes podem ser:

  • web

  • webservices

  • ejb

  • connector

  • appclient

  • jruby

Ação

Links para as ações que você pode executar no componentes após a implantação: Reimplantar e Reiniciar para todos os componentes e Iniciar para aplicativos Web.

Implantar

Botão para implantar um aplicativo empresarial.

Cancelar Implantação

Botão para cancelar a implantação de um aplicativo empresarial.

Ativar

Botão para ativar um aplicativo empresarial selecionado.

Desativar

Botão para desativar um aplicativo empresarial selecionado.

Filtro

Lista suspensa que filtra os aplicativos por mecanismo.
