Para crear un recurso JDBC

Un recurso (origen de datos) JNDI (Interfaz de Directorios y Nomenclatura de Java proporciona a las aplicaciones los medios de conexión a una base de datos.

Antes de Empezar

Para crear un recurso JDBC, debe crear primero un pool de conexiones JDBC. Consulte Para crear un pool de conexiones JDBC.

  1. En el árbol de navegación, amplíe el nodo Recursos.

  2. En el nodo Recursos, amplíe el nodo JDBC.

  3. En el nodo JDBC, seleccione el nodo Recursos JDBC.

    Se abrirá la página Recursos de JDBC.

  4. En la página Recursos JDBC, haga clic en Nuevo.

    Se abrirá la página Nuevo recurso JDBC.

  5. En el campo Nombre JNDI, introduzca un nombre de recurso único.

    El nombre JNDI organiza y ubica componentes dentro de un entorno informático distribuido del mismo modo que los ficheros organizan y representan ubicaciones de libros en una biblioteca. Por consiguiente, el nombre JNDI se convierte en un importante método de acceso al recurso JDBC. Por convención, el nombre comienza por la cadena jdbc/, por ejemplo, jdbc/payrolldb. No olvide la barra diagonal.

    El nombre debe contener únicamente caracteres alfanuméricos, de subrayado, guiones y puntos.

  6. En la lista desplegable Nombre de pool, seleccione un pool de conexiones para asociarlo al nuevo recurso JDBC.

  7. En el campo Descripción, introduzca una breve descripción del nuevo recurso JDBC.

  8. Active la casilla de control Estado activado para hacer que el recurso esté disponible en tiempo de ejecución.

    Esta opción se encuentra activada por defecto.

  9. En la sección Propiedades Adicionales, especifique las propiedades adicionales.

    GlassFish Server no define ninguna propiedad adicional para recursos JDBC.

    Para agregar una propiedad, haga clic en el botón Agregar Propiedad. En la fila en blanco que aparece, introduzca el nombre de propiedad en el campo Nombre y escriba el valor de propiedad en el campo Valor.

  10. Seleccione un destino disponible para agregarlo o eliminarlo de la lista destinos seleccionados.

  11. Haga clic en Aceptar.
