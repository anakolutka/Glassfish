檢視 GlassFish Server 的「註冊」頁面

GlassFish Server 的「註冊」頁面可讓您註冊安裝的 GlassFish Server。

註冊成為 GlassFish Server 的使用者可享有某些權益，例如：

  • 新的版本、修正程式及更新的通知

  • 開發者產品、服務及訓練的特別優惠

  • 可存取白皮書、說明文章，並參與 GlassFish 產品小組的線上研討會

  1. 在瀏覽樹狀結構中，選取「常用工作」節點。

    「常用工作」頁面會隨即開啟。

  2. 在「GlassFish 新聞」底下，按一下「註冊」連結。

    GlassFish 的「註冊」網頁會在另外一個瀏覽器標籤或視窗中開啟。
