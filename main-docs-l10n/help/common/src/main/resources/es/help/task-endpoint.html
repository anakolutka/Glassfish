Ver información de punto final para un servicio web

  1. En el árbol de navegación, seleccione el nodo Aplicaciones.

    Se abrirá la página Aplicaciones.

  2. En la página Aplicaciones, haga clic en el nombre del servicio web cuya información de punto final desea ver.

    El nombre puede incluir un identificador de versión opcional, que va a continuación del nombre y está separado de él por dos puntos (:). Para obtener más información sobre las versiones de módulo y de aplicación, consulte el apartado sobre las versiones de módulo y de aplicación en Oracle GlassFish Server Application Deployment Guide.

    Se abrirá la página Editar Aplicación.

  3. En la tabla Módulos y Componentes, haga clic en el enlace Ver Punto Final para el subcomponente.

    Se abrirá la página Información de Punto Final de Servicio Web.

  4. Si está disponible el enlace Probador, haga clic en él para ver una página que permita proporcionar parámetros para probar los métodos del servicio web.

    El enlace Probador sólo está visible si el tipo de despliegue de la aplicación es distinto de RI (por ejemplo, 109). La página Probador se abrirá en una nueva ventana o separador del navegador.

  5. Haga clic en el enlace WSDL para ver el archivo Web Services Definition Language (WSDL) para el punto final.

    El archivo WSDL se abre en una nueva ventana o separador del explorador.

    Tenga en cuenta que algunos exploradores tratan de interpretar el origen de WSDL al hacer clic en el enlace WSDL, de modo que muestran una página en blanco. Para estos navegadores, utilice el comando Ver Origen del explorador en la página en blanco para ver el origen de WSDL.

  6. Haga clic en el enlace Nombre de Aplicación o en el botón Atrás para volver a la página Editar Aplicación.
