업데이트 도구 구성

구성 페이지에서는 업데이트 도구에 대한 구성 정보를 제공하며 이를 사용하여 업데이트 도구의 구성을 변경할 수 있습니다.

업데이트 도구 구성 페이지에는 다음 옵션이 포함되어 있습니다.

설치 경로

설치 이미지의 루트 또는 최상위 수준 디렉토리에 대한 파일 시스템 경로입니다.

기본 저장소

이 설치 이미지와 연결된 기본 네트워크 저장소의 이름이며 예를 들면 glassfish.org입니다.

인터넷 연결

인터넷에 Enterprise Server 연결 방법:

직접 연결

이 옵션을 선택하면 Enterprise Server가 인터넷에 직접 연결됩니다.

프록시를 통해

이 옵션을 선택하면 Enterprise Server가 프록시 서버를 통해 인터넷에 연결됩니다. 이 옵션을 선택하면 프록시 호스트 필드 및 프록시 포트 필드가 이 페이지에 추가됩니다.

프록시 호스트

Enterprise Server가 실행 중인 시스템에 대한 프록시 서버의 호스트 이름입니다.

이 필드는 Enterprise Server가 실행 중인 시스템이 프록시 서버를 통해 인터넷에 연결되는 경우에만 적용됩니다.

프록시 포트

Enterprise Server가 실행 중인 시스템에 대한 프록시 서버의 포트 번호입니다.

이 필드는 Enterprise Server가 실행 중인 시스템이 프록시 서버를 통해 인터넷에 연결되는 경우에만 적용됩니다.

저장소
