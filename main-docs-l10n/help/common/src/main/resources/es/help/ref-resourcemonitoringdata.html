Supervisión de recursos

Utilice la página Supervisión de Recursos para ver los datos de supervisión de un recurso seleccionado.

La página Supervisión de Recursos contiene la información siguiente.

Nombre de instancia

Nombre de la instancia del servidor.

Pool de conexiones

Pool de conexiones para el que se muestran las estadísticas de supervisión.

Para obtener una descripción de las propiedades supervisadas, consulte el capítulo 8, sobre la administración del servicio de supervisión, en Oracle GlassFish Server Administration Guide.
