Configurazioni

Utilizzare la pagina Configurazioni per gestire le configurazioni e visualizzare le istanze server standalone e i cluster che utilizzano le configurazioni.

La configurazione default-config è la configurazione predefinita di GlassFish Server. La configurazione server-config viene creata in modo automatico per il server di amministrazione del dominio (DAS) quando si crea il dominio. Per ulteriori informazioni, vedere About Named Configurations in Oracle GlassFish Server High Availability Administration Guide.

Per ogni configurazione vengono fornite le informazioni riportate di seguito.

Nome

Il nome della configurazione.

Istanza

La lista delle istanze standalone e dei cluster che utilizzano la configurazione.

La tabella Configurazioni contiene inoltre le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare una nuova configurazione.

Elimina

Pulsante che consente di eliminare una o più configurazioni selezionate.
