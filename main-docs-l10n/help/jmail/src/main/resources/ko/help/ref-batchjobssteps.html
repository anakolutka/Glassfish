일괄 처리 작업 실행 단계

[일괄 처리 작업 실행 단계] 페이지에는 특정 일괄 처리 작업 실행의 단계에 대한 세부 정보가 표시됩니다. 다음 정보가 표시됩니다.

인스턴스 이름
클러스터 이름

구성이 적용되는 GlassFish 클러스터 또는 서버 인스턴스 이름입니다.

작업 이름

일괄 처리 작업의 이름입니다.

실행 ID

일괄 처리 작업 실행에 지정된 ID입니다. 새 실행은 작업이 처음 시작될 때, 그리고 기존 실행이 재시작될 때마다 생성됩니다.

작업 단계

특정 일괄 처리 작업 실행의 단계입니다. 각 단계에 대해 다음 정보가 표시됩니다.

단계 이름

단계의 이름입니다.

일괄 처리 상태

일괄 처리 런타임에 의해 설정된 단계 상태입니다.

종료 상태

해당 작업에 대한 작업 XML 또는 일괄 처리 응용 프로그램에 의해 설정된 단계 상태입니다. 종료 상태가 명시적으로 대체되지 않는 한 기본적으로 종료 상태와 일괄 처리 상태는 동일합니다.

시작 시간

단계가 시작된 시간입니다.

종료 시간

단계가 종료된 시간입니다.

단계 측정 단위

읽기, 쓰기, 커밋 횟수 등을 나열하는 단계 측정 단위입니다.
