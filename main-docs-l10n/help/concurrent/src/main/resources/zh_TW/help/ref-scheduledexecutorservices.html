受管理排定的執行程式服務

您可以使用「受管理排定的執行程式服務」頁面來組態排定的受管理執行程式服務資源。

對於每個排定的受管理執行程式服務資源，均提供以下資訊。

JNDI 名稱

可識別資源的唯一名稱。

已啟用

如果資源已啟用，則為 True；如果資源已停用，則為 False。

環境資訊

資源的環境：類別載入器、JNDI、安全性或工作區。

繫線優先順序

要為受管理排定的執行程式服務執行處理建立之繫線的優先順序。預設值為 5。

描述

資源的描述。

「受管理排定的執行程式服務」表格也包含下列選項。

新建

建立新資源的按鈕。

刪除

刪除一或多個所選取資源的按鈕。

啟用

啟用一或多個所選取資源的按鈕。

停用

停用一或多個所選取資源的按鈕。
