컨텍스트 서비스

[컨텍스트 서비스] 페이지에서는 컨텍스트 서비스 리소스를 구성할 수 있습니다.

각 컨텍스트 서비스 리소스에 대해 제공되는 정보는 다음과 같습니다.

JNDI 이름

리소스를 식별하는 고유한 이름입니다.

사용

리소스가 사용으로 설정된 경우 true, 사용 안함으로 설정된 경우 false입니다.

컨텍스트 정보

리소스의 컨텍스트입니다(Classloader, JNDI, Security 또는 WorkArea).

설명

리소스에 대한 설명입니다.

컨텍스트 서비스 테이블에는 다음 옵션도 포함되어 있습니다.

새로 만들기

새 리소스를 생성하는 단추입니다.

삭제

선택한 리소스를 하나 이상 삭제하는 단추입니다.

사용

선택한 하나 이상의 리소스를 사용으로 설정하는 단추입니다.

사용 안함

선택한 하나 이상의 리소스를 사용 안함으로 설정하는 단추입니다.

관련 asadmin 명령

  • create-XXX(1)

  • list-XXX(1)

  • delete-XXX(1)
