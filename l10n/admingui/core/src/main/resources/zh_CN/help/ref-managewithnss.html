用于用户管理的网络安全服务 (Network Security Service, NSS)

可以使用管理控制台（在“文件用户”页面中介绍）或 NSS 工具管理用户。NSS 是一组库，用于支持启用了安全性的客户机和服务器应用程序的跨平台开发。使用 NSS 构建的应用程序可以支持 SSL v2 和 v3、TLS、PKCS #5、PKCS #7、PKCS #11、PKCS #12、S/MIME、X.509 v3 证书，以及其他安全性标准。有关详细信息，请链接到以下 URL：

  • Network Security Service (NSS)，位于 http://www.mozilla.org/projects/security/pki/nss/

  • NSS Security Tools，位于 http://www.mozilla.org/projects/security/pki/nss/tools/

  • Overview of NSS，位于 http://www.mozilla.org/projects/security/pki/nss/overview.html
