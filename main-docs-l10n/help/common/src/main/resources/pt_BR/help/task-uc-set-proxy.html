Para Configurar as Definições de Proxy de Rede para Acessar os Repositórios de Rede

As definições de proxy de rede determinam como o GlassFish Server conecta-se com a Internet. Para ativar o Ferramenta de Atualização a fim de acessar os repositórios da rede por meio da Internet, configure as definições do proxy da sua rede.

  1. Na árvore de navegação, selecione o nó Ferramenta de Atualização.

    É aberta a página Add-ons Disponíveis.

  2. Na página Add-ons Disponíveis, clique na guia Configuração.

    É exibida a página Configuração Ferramenta de Atualização.

  3. Na página Configuração Ferramenta de Atualização, especifique como a máquina na qual GlassFish Server está sendo executado se conecta com a Internet.

    • Se a máquina se conecta com a Internet diretamente, selecione a opção Conexão Direta.

    • Se a máquina se conecta com a Internet por meio de um proxy, selecione a opção Por Proxy.

  4. Se a máquina na qual o GlassFish Server está sendo executado se conectar com a Internet por meio de um proxy, forneça o nome do Host do Proxy e o número da Porta do Proxy.

    Se a máquina na qual o GlassFish Server está sendo executado se conectar com a Internet diretamente, omita esta etapa.

    Um Web browser pode estar sendo executado na máquina na qual está sendo executado o GlassFish Server. Nesta situação, você pode obter o nome do host e o número da porta do proxy do Web browser. Você também pode solicitar essa informação ao administrador do sistema.

  5. Clique em Salvar.
