Livelli di log dei moduli

Utilizzare la pagina Livelli di log dei moduli per configurare i livelli di log per singoli moduli.

Per ogni modulo vengono fornite le informazioni riportate di seguito.

Nome logger

Il nome del logger per il modulo, ad esempio javax.enterprise.system.tools.admin .

Livello di log

Il livello di log corrente per il modulo.

La tabella Impostazioni logger contiene inoltre le opzioni riportate di seguito.

Aggiungi logger

Pulsante che consente di aggiungere un modulo logger. Quando si fa clic sul pulsante, viene aggiunta una riga alla tabella Impostazioni logger. Immettere informazioni sul modulo logger nella nuova riga.

Elimina logger

Pulsante che consente di eliminare uno o più moduli selezionati.

Livello

Elenco a discesa da cui è possibile selezionare un livello di log da applicare a tutti i moduli selezionati.

Cambia livello

Pulsante che consente di modificare il livello di log per uno o più moduli selezionati.

Di seguito sono riportati i livelli di log disponibili. Sono elencati in ordine decrescente dal più alto al più basso.

EMERGENCY

Il server si trova in uno stato inutilizzabile. Si è verificato un errore o un allarme di sistema grave.

ALERT

Un determinato servizio si trova in uno stato inutilizzabile. Il recupero automatico non è possibile.

SEVERE

Eventi che interferiscono con l'esecuzione normale del programma.

WARNING

Avvertenze, incluse le eccezioni.

INFO

Messaggi correlati alla configurazione del server o allo stato del server, esclusi gli errori.

CONFIG

Messaggi correlati alla configurazione del server.

FINE

Livello di dettaglio minimo.

FINER

Livello di dettaglio moderato.

FINEST

Livello di dettaglio massimo.

OFF

Nessun messaggio di log.
