Módulos do Ciclo de Vida

A página Módulos do Ciclo de Vida exibe uma lista de módulos do ciclo de vida disponíveis.

Um módulo do ciclo de vida realiza tarefas quando ele é acionado por um ou mais eventos no ciclo de vida do servidor. Os possíveis eventos do servidor do disparador são: inicialização, pronto para solicitações de serviço e desligamento. Os módulos do ciclo de vida não fazem parte da especificação Java EE, mas são um aprimoramento do Enterprise Server.

Para cada módulo do ciclo de vida, as informações a seguir são fornecidas.

Nome

O nome do módulo do ciclo de vida.

Ativado

Verdadeiro se o módulo ciclo de vida estiver ativado, ou falso se ele não estiver ativado.

Ordem de Carregamento

O valor que especifica a ordem relativa na qual o módulo do ciclo de vida deveser carregado.

Nome da classe

O arquivo de classe do módulo do ciclo de vida.
