編輯 JACC 提供者

使用「編輯 JACC 提供者」頁面可修改所選 GlassFish Server 執行處理或叢集的 JACC 提供者。

「編輯 JACC 提供者」頁面包含下列選項。

組態名稱

套用此頁面上之設定的組態名稱。本欄位是唯讀的。

名稱

JACC 提供者的名稱。此為唯讀欄位。您僅能在建立新的 JACC 提供者時指定名稱。

策略組態

javax.security.jacc.PolicyConfigurationFactory.provider 的完全合格類別名稱，會實行提供者特定的 javax.security.jacc.PolicyConfigurationFactory。

策略提供者

javax.security.jacc.policy.provider 的完全合格類別名稱，會實行 java.security.Policy。

其他特性

JACC 提供者的其他特性。如需適用於 JACC 提供者之其他特性的描述，請參閱JACC 提供者特定的特性。
