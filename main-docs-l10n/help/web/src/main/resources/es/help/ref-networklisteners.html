Listeners de Red

Utilice la página Listeners de Red para configurar los listeners de red.

Al instalar por primera vez GlassFish Server, existen por defecto tres listeners de red denominados admin-listener, http-listener1 y http-listener2. El listener http-listener2, que está configurado con SSL, se encuentra desactivado por defecto.

En cada listener de red se muestra la siguiente información.

Nombre

Nombre del listener de red.

Número de Puerto

Número de puerto en el que recibe el listener de red.

Protocolo

Protocolo que utiliza el listener de red.

Pool de Threads

Pool de threads que utiliza el listener de red.

Activado

El valor true indica que el listener de red se encuentra activado y el valor false indica que no se encuentra activado.

La tabla Listeners de Red contiene además las siguientes opciones.

Nuevo

Botón para crear un nuevo listener de red.

Suprimir

Botón para suprimir uno o varios listeners de red seleccionados.
