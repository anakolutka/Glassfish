To Create an HTTP Load Balancer (Step 1 of 2)

  1. In the tree component, select the HTTP Load Balancers node.

  2. On the HTTP Load Balancers page, click New.

    The New HTTP Load Balancers page appears.

  3. Enter a name for the load balancer.

    This name must not conflict with any other load balancers in the domain.

  4. Select the target or targets for the load balancer.

    The load balancer handles requests for the instances or clusters you select. You can also add targets later.

  5. Click Next to continue with load balancer creation.
