Suppression d'une mappe de sécurité pour un pool de connexions de connecteur

  1. Dans l'arborescence de navigation, développez le noeud Ressources.

  2. Développez le noeud Connecteurs.

  3. Sous le noeud Connecteurs, sélectionnez le noeud Pools de connexions du connecteur.

    La page Pools de connexions du connecteur apparaît.

  4. Sur la page Pools de connexions du connecteur, cliquez sur le nom du pool de connexions contenant la mappe de sécurité à supprimer.

    La page Modifier le pool de connexions du connecteur apparaît.

  5. Cliquez sur l'onglet Mappes de sécurité.

    La page Mappes de sécurité des pools de connexions du connecteur apparaît.

  6. Sur la page Mappes de sécurité des pools de connexions du connecteur, sélectionnez la case en regard du nom de la mappe de sécurité à supprimer.

  7. Cliquez sur Supprimer.
