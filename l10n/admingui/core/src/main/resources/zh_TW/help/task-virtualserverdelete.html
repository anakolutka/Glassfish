刪除虛擬伺服器

  1. 在瀏覽樹狀結構中，展開 [配置] 節點。
  2. 在 [配置] 節點下，展開 [HTTP 服務] 節點。
  3. 在 [HTTP 服務] 節點下，選取 [虛擬伺服器] 節點。
  4. 在 [虛擬伺服器] 頁面上，選取要刪除的虛擬伺服器名稱左側之核取方塊。
  5. 按一下 [刪除]。

    您可以刪除 __asadmin 虛擬伺服器，但是不建議執行此動作。若要執行該動作，請先將 Enterprise Server 的 domain.xml 檔案中的 virtual-server 元素複製到安全的位置，以便可以在需要時復原這些設定。
