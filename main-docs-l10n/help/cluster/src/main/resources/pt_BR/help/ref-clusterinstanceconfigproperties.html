Propriedades de Sistema da Instância

A página Propriedades de Sistema da Instância exibe uma lista das propriedades do sistema Java que estão definidas para a instância atual do GlassFish Server. As propriedades do sistema Java são especificadas para o acionador da aplicação Java por meio da opção -D do acionador da aplicação Java quando GlassFish Server for iniciado.

Essas propriedades substituem as definições de propriedade para definições de porta na configuração da instância. As definições de porta predefinidas deverão ser substituídas se, por exemplo, duas instâncias clusterizadas residirem no mesmo host. Nessa situação, as definições de porta para uma instância devem ser substituídas porque ambas as instâncias compartilham a mesma configuração.

Para obter informações sobre as definições de porta predefinidas, consulte Predefined Configuration System Properties.

O campo Nome da Instância é um campo somente para leitura que exibe o nome da instância atual.

São exibidas as seguintes informações para cada propriedade:

Nome da Variável da Instância

O nome da propriedade do sistema.

Valor Atual

Valor atualmente definido para a propriedade. Este campo é somente para leitura.

Valor Substituto

O valor com o qual a propriedade será definida depois que as alterações forem salvas.

A tabela Propriedades Adicionais também contém as seguintes opções.

Adicionar Propriedade

Botão para adicionar uma propriedade. Clicar neste botão adiciona uma linha à tabela Propriedades Adicionais.

Deletar Propriedades

Botão para deletar uma ou mais propriedades selecionadas. Qualquer propriedade deletada retorna ao seu valor default ou, se nenhum valor tiver sido definido, fica com valor indefinido.
