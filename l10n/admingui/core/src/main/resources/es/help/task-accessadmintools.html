Cómo otorgar acceso a las herramientas de administración.

Sólo los usuarios del grupo asadmin pueden acceder a Consola de administración y a la utilidad de línea de comandos asadmin. Para otorgar acceso de usuario a estas herramientas de administración, agregue los usuarios al grupo asadmin de admin-realm.

  1. En el árbol de navegación, expanda el nodo Configuración.
  2. Si el dominio está configurado para admitir clústeres, seleccione la configuración que esté cambiando.

    Algunos ejemplos son los dominios que se crean con el perfil clúster o con el perfil grandes empresas .

  3. Expanda el nodo Seguridad.
  4. Expanda el nodo Dominios.
  5. Seleccione el nodo admin-realm.
  6. Haga clic en el botón Administrar usuarios de la página Editar dominio.

    Inicialmente tras la instalación el nombre y usuario del administrador introducidos durante la instalación aparecen en un archivo llamado admin-keyfile. Por defecto este usuario pertenece al grupo asadmin, que da derecho a modificar Servidor empresarial. Asigne usuarios a este grupo sólo si les quiere otorgar privilegios de administrador para Servidor empresarial.

    Si agrega usuarios al dominio admin-realm, pero los asigna a un grupo diferente de asadmin, la información de los usuarios permanecerá escrita en el archivo admin-keyfile, pero el usuario no tendrá acceso a las herramientas administrativas ni a las aplicaciones del dominio file.

  7. Haga clic en Nuevo para añadir un nuevo usuario al dominio admin-realm.
  8. Introduzca el ID de usuario, la lista de grupo y la contraseña.

    Para autorizar a un usuario la realización de cambios a Servidor empresarial, incluya el grupo asadmin en la Lista de grupos.

  9. Haga clic en Aceptar para agregar este usuario al dominio admin-realm o haga clic en Cancelar para salir sin guardar.
