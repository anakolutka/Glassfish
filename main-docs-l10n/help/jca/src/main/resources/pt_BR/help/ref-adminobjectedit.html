Editar o Recurso de Objeto Administrado

Use a página Editar o Recurso de Objeto Administrado para modificar as definições de um recurso de objeto administrado.

A página Editar o Recurso de Objeto Administrado contém as seguintes opções.

Carregar Defaults

Botão para restaurar definições para seus valores default. Definições que não tiveram seus valores default alterados.

Nome JNDI

Um nome exclusivo que identifica o recurso do objeto administrado. O nome JNDI é um campo somente para leitura. Você só pode especificar o nome JNDI ao criar um novo recurso de objeto administrado.

Adaptador do Recurso

O adaptador de recurso para o recurso do objeto administrado.

Tipo de Recurso

O tipo Java do recurso de objeto administrado (ou por exemplo,javax.jms.Topic).

Nome da Classe

O nome da classe de implementação associada com o tipo de recurso.

Descrição

A descrição do recurso do objeto administrado.

Status

Se esta opção está selecionada, o recurso do objeto administrado será ativado. Por default, a opção está ativada.

Propriedades Adicionais

Propriedades adicionais para o recurso do objeto administrado. As propriedades disponíveis dependem do adaptador de recursos para o pool de conexões do conector. Os pares de nome e valor especificados nessa tabela podem ser usados para substituir os valores default definidos pelo fornecedor do adaptador de recurso.
