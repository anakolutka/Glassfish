Para exibir os descritores de um aplicativo

  1. Na árvore de navegação, selecione o nó Aplicativos.

    A página Aplicativos é exibida.

  2. Na página Aplicativos, clique no nome do aplicativo cujos descritores devem ser exibidos.

    A página Editar Aplicativo é exibida.

  3. Na página Editar Aplicativo, clique na guia Descritor.

    A página Descritores de Módulos é exibida.

  4. Na tabela Descritores de Implantação, clique no nome do arquivo XML a ser visualizado.

    O arquivo XML é aberto em uma nova janela.
