So wird ein Benutzer der Admin-Realm hinzugefügt

  1. Blenden Sie unter dem Knoten "Konfigurationen" die Serverinstanz oder die Clusterkonfiguration ein, für die Sie Admin-Realm-Benutzer verwalten möchten.

  2. Blenden Sie den Knoten "Sicherheit" ein.

  3. Blenden Sie den Knoten "Realms" ein.

  4. Wählen Sie den Knoten admin-realm.

    Die Seite "Realm bearbeiten" wird geöffnet.

  5. Klicken Sie auf der Seite "Realm bearbeiten" auf die Schaltfläche "Benutzer verwalten".

    Die Seite "Admin-Benutzer" wird geöffnet.

  6. Klicken Sie auf der Seite "Admin-Benutzer" auf "Neu".

    Die Seite "Neuer File Realm-Benutzer" wird geöffnet.

  7. Geben Sie in das Feld "Benutzer-ID" den Namen des Benutzers ein.

  8. Geben Sie in das Feld "Gruppenliste" eine durch Komma getrennte Liste der Gruppen ein, zu denen der Benutzer gehört.

    Verwenden Sie Kommas zum Trennen der Gruppennamen. Diese Gruppen müssen an keiner weiteren Stelle definiert werden.

  9. Geben Sie in das Feld "Kennwort" ein eindeutiges Kennwort ein.

  10. Geben Sie in das Feld "Neues Kennwort bestätigen" das Kennwort des Benutzers erneut ein.

  11. Klicken Sie auf "OK".
