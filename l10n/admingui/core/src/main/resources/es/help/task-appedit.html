Para editar una aplicación implementada

  1. En el árbol de navegación, seleccione el nodo Aplicaciones.

    Se abrirá la página Aplicaciones

  2. En la tabla Aplicaciones implementadas, haga clic en el nombre de la aplicación que se va a editar.

    Se abrirá la página Editar aplicación.

  3. Para una aplicación web, en el campo Raíz de contexto, introduzca la ruta de la aplicación.

    En la URL de la aplicación web, la raíz de contexto aparece justo después del número de puerto (http://host:port /context-root/...). Asegúrese de que la raíz de contexto empieza con una barra, por ejemplo, /hello.

  4. En el campo Descripción, introduzca una descripción de esta aplicación
  5. Anule la selección de la casilla de verificación Estado habilitado para deshabilitar el estado de la aplicación y que no se pueda acceder a ella.

    De manera predeterminada, la aplicación presenta el estado Habilitado una vez implementada.

  6. Seleccione la casilla de verificación Inicio web de Java habilitado para usar el software Inicio web de Java para implementar la aplicación.
  7. Haga clic en Guardar.
