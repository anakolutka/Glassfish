Nuevo pool de conexiones del conector (paso 1 de 2).

Utilice el nuevo pool de conexiones del conector (paso 1 de 2) para crear un nuevo pool de conexiones del conector.

La página Nuevo pool de conexiones del conector (paso 1 de 2) contiene las siguientes opciones.

Nombre

Nombre único que identifica el pool de conexiones de conector.

Adaptador de recursos

Adaptador de recursos asociado al pool de conexiones de conector.

Definición de conexión

Definición de conexión para el pool de conexiones de conector. Las opciones de la lista dependen del adaptador de recursos seleccionado. La definición de conexión identifica un elemento connection-definition del adaptador de recursos en el archivo ra.xml.
