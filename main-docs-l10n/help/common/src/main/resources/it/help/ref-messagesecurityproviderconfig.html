Configurazioni provider

Utilizzare la pagina Configurazioni provider per specificare le configurazioni dei provider di sicurezza dei messaggi per il layer di autenticazione SOAP o HttpServlet.

Per impostazione predefinita vengono configurati quattro provider per il livello di autenticazione SOAP.

Per ogni configurazione di provider vengono fornite le informazioni riportate di seguito.

ID provider

Il nome della configurazione di provider.

Tipo di provider

Il tipo di provider, che può essere uno dei tipi riportati di seguito.

client

Specifica che il provider è il provider di autenticazione client.

server

Specifica che il provider è il provider di autenticazione server.

client-server

Specifica che il provider è un provider di autenticazione sia client che server.

Provider predefinito

Se l'impostazione è true, il provider è il provider predefinito.

Nome classe

Il nome della classe di implementazione Java del provider.

La tabella Configurazioni provider contiene inoltre le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare una nuova configurazione di provider.

Elimina

Pulsante che consente di eliminare una o più configurazioni di provider selezionate.
