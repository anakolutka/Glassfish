So erstellen Sie eine JDBC-Ressource

Eine Java DataBase Connectivity (JDBC)-Ressource (Datenquelle) bietet Anwendungen über die Verbindung zu einer Datenbank.

Bevor Sie beginnen

Um eine JDBC-Ressource zu erstellen, müssen Sie zuerst einen JDBC-Verbindungspool erstellen. Siehe So erstellen Sie einen JDBC-Verbindungspool.

  1. Erweitern Sie in der Navigationsstruktur den Ressourcenknoten.
  2. Wählen Sie den Knoten "JDBC-Ressourcen" aus.
  3. Klicken Sie auf der Seite "JDBC-Ressourcen" auf "Neu".

    Die Seite "Neue JDBC-Ressource" wird geöffnet.

  4. Geben Sie im Feld "J.N.D.I.-Name" einen eindeutigen Ressourcennamen ein.

    Der J.N.D.I.-Name organisiert und ordnet Komponenten innerhalb einer verteilten Computing-Umgebung in ähnlicher Weise an wie ein Karteikartensystem die Bücher in einer Bibliothek katalogisiert und deren Speicherort festhält. Folglich stellt der J.N.D.I.-Name ein wichtiges Mittel für den Zugriff auf eine JDBC-Ressource dar. Gemäß Konvention beginnt der Name mit der Zeichenfolge jdbc/, zum Beispiel jdbc/payrolldb. Vergessen Sie nicht den Schrägstrich (/).

    Der Name darf ausschließlich alphanumerische Zeichen, Unterstriche, Bindestriche oder Punkte enthalten.

  5. Wählen Sie in der Dropdown-Liste "Poolname" einen Verbindungspool aus, welcher der neuen JDBC-Ressource zuzuordnen ist.
  6. Geben Sie im Feld "Beschreibung" eine kurze Beschreibung der neuen JDBC-Ressource ein.
  7. Deaktivieren Sie das Kontrollkästchen "Aktiviert", um die Ressource zu deaktivieren.

    Per Voreinstellung ist die Ressource verfügbar (aktiviert), wenn Sie erstellt wurde.

  8. Klicken Sie auf „OK“.
