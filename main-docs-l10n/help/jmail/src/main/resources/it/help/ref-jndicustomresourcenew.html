Nuova risorsa personalizzata

Utilizzare la pagina Nuova risorsa personalizzata per creare una nuova risorsa JNDI personalizzata.

La pagina Nuova risorsa personalizzata contiene le opzioni riportate di seguito.

Nome JNDI

Il nome univoco che identifica la risorsa.

Tipo di risorsa

La definizione completamente qualificata del tipo di risorsa.

Classe factory

Il nome specificato dall'utente per la classe factory. Questa classe implementa l'interfaccia javax.naming.spi.ObjectFactory.

Descrizione

La descrizione di questa risorsa personalizzata.

Stato

Lo stato della risorsa personalizzata. La risorsa personalizzata può essere abilitata o disabilitata. Lo stato predefinito è Abilitato.

Proprietà aggiuntive

Proprietà aggiuntive per la risorsa personalizzata. GlassFish Server non definisce alcuna proprietà aggiuntiva per le risorse personalizzate.

Destinazioni

I cluster e le istanze standalone per la risorsa personalizzata. Per spostare le destinazioni desiderate nella colonna Destinazioni selezionate, utilizzare i pulsanti Aggiungi e Aggiungi tutto. Per spostare le destinazioni non necessarie nella colonna Destinazioni disponibili, utilizzare i pulsanti Rimuovi e Rimuovi tutto.

Questa opzione viene visualizzata solo se nel dominio sono stati creati cluster o istanze standalone.
