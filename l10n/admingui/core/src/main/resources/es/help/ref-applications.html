Aplicaciones

La página Aplicaciones muestra una lista de aplicaciones que se implementan en Servidor empresarial. Puede ver y administrar las aplicaciones que ya estén implementadas e implementar otras nuevas.

En cada aplicación aparece la siguiente información.

Nombre

El nombre de la aplicación.

Habilitado

Una marca de verificación si la aplicación está habilitada o una X si está deshabilitada.

Compuesto

'True' si la aplicación tiene más de un tipo de componente, o 'False' si existe sólo un tipo de componente.

Motores

Los tipos de contenedores empleados por la aplicación. Entre los tipos de contenedor se incluyen los siguientes:

  • web

  • webservices

  • ejb

  • conector

  • appclient

  • jruby

Action (Acción)

Enlaces a las operaciones que se pueden desarrollar en el componente después de implementarlo: volver a implementar y reiniciar todos los componentes e iniciar aplicaciones web.

Implementar

Botón para implementar una aplicación empresarial.

Anular implementación

Botón para anular la implementación de una aplicación empresarial.

Habilitar

Botón para habilitar una aplicación empresarial seleccionada.

Deshabilitar

Botón para deshabilitar una aplicación empresarial seleccionada.

Filtrar

Lista desplegable que filtra las aplicaciones por motor.
