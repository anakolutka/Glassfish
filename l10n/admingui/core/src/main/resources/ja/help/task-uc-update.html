インストール済みコンポーネントを更新する

インストール済みコンポーネントの更新には次の作業が含まれます。

  • ネットワークリポジトリからシステムに、変更したファイルのみを転送する

  • システム上にある Enterprise Server のインストールイメージ内の変更されたファイルのみを更新する

  • 更新されたコンポーネント間の新しい依存関係 (もしあれば) を適用する

  1. ナビゲーションツリーで、更新ツール ノードを選択します。

    「インストール済みコンポーネント」ページが表示されます。

  2. 「インストール済みコンポーネント」ページで、「利用可能な更新」タブをクリックします。

    「利用可能な更新」ページが表示されます。

  3. 利用可能な更新の表で、更新するコンポーネントを選択します。

    利用可能な更新がない場合、表は空になります。

    • 個々のコンポーネントを選択するには、そのコンポーネントの名前の隣にあるチェックボックスを選択します。
    • すべてのコンポーネントを選択するには、表ヘッダーの「すべてを選択」アイコンをクリックします。
    • すべてのコンポーネントを選択解除するには、表ヘッダーの「すべてを選択解除」アイコンをクリックします。
  4. 「インストール」をクリックします。
  5. ライセンス契約に同意します。

    Enterprise Server はインストールが完了したことを確認します。利用可能な更新の表からコンポーネントが削除されます。

  6. ドメイン管理サーバーを再起動します。

    詳細な手順については、次のマニュアルを参照してください。

    • 『Sun GlassFish Enterprise Server v3 Prelude Administration Guide』の「To Stop a Domain (or Server)」

    • 『Sun GlassFish Enterprise Server v3 Prelude Administration Guide』の「To Start a Domain (or Server)」
