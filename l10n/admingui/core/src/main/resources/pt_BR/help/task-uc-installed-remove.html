Para desinstalar componentes

A desinstalação de um componente envolve a remoção dos seguintes itens:

  • Todos os arquivos e diretórios introduzidos pela instalação original do componente

  • Todos os arquivos e diretórios introduzidos por atualizações do componente, depois da instalação original

  1. Na árvore de navegação, selecione o nó Ferramenta de Atualização.

    É aberta a página Complementos Disponíveis.

  2. Na página Complementos Disponíveis, clique na guia Componentes Instalados.

    A página Componentes Instalado é exibida.

  3. Na tabela de componentes instalados, selecione os componentes que você está desinstalando.
    • Para selecionar um componente individual, marque a caixa de seleção adjacente ao nome do componente.
    • Para selecionar todos os componentes, clique no ícone Selecionar Todos, no cabeçalho da tabela.
    • Para desmarcar todos os componentes, clique no ícone Selecionar Todos, no cabeçalho da tabela Desmarcar Tudo.
  4. Clique em Remover.

    Uma caixa de diálogo pede que você confirme se os componentes selecionados devem ser desinstalados.

  5. Na caixa de diálogo, clique em OK.

    O Enterprise Server confirma que a desinstalação está concluída. Os componentes são removidos da tabela de componentes instalados e adicionados à tabela de complementos disponíveis na página Complementos Disponíveis.
