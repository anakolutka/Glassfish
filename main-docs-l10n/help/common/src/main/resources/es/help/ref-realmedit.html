Editar dominio

Utilice la página Editar Dominio para modificar un dominio.

La página Editar Dominio contiene las siguientes opciones.

Nombre de Configuración

Nombre de la configuración a la que se aplican los valores de esta página. Este campo es de sólo lectura.

Nombre de dominio

Nombre del dominio. Éste es un campo de sólo lectura. Únicamente puede especificar el nombre al crear un nuevo dominio.

Nombre de clase

Nombre de clase del dominio que está editando.

Propiedades Adicionales

Propiedades adicionales del dominio. Las propiedades válidas dependen del tipo de dominio.

  • Propiedades específicas de la clase FileRealm

  • Propiedades específicas de la clase CertificateRealm

  • Propiedades específicas de la clase JDBCRealm

  • Propiedades específicas de la clase LDAPRealm

  • Propiedades específicas de la clase SolarisRealm

  • Propiedades específicas de la clase PamRealm
