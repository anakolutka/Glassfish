Afficher les données de contrôle du conteneur Web

  1. Dans l'arborescence de navigation, sélectionnez le nœud Application Server.
  2. Dans la page Informations générales, cliquez sur l'onglet Contrôle.

    La page Contrôle du conteneur Web s'ouvre.

  3. (Facultatif) Dans la liste déroulante Application, sélectionnez l'application pour laquelle vous souhaitez afficher les données de contrôle.
  4. (Facultatif) Dans la liste déroulante Serveur virtuel, sélectionnez le serveur pour lequel vous souhaitez afficher les données de contrôle.
  5. (Facultatif) Cliquez sur le bouton Actualiser pour mettre à jour les données de contrôle affichées.
