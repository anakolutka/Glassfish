Módulos de Auditoria

A auditoria é como o Enterprise Server grava eventos significativos, como erros ou vulnerabilidades de segurança. O Enterprise Server registra todos os eventos de autenticação em arquivos de log. Um log de acesso completo fornece uma pista sequencial dos eventos de acesso do Enterprise Server.

A página Módulos de Auditoria exibe uma lista dos módulos de auditoria disponíveis. Para cada módulo de auditoria, as seguintes informações são fornecidas

Nome

O nome do módulo de auditoria.

Nome da Classe

O nome totalmente qualificado da classe que implementa o módulo. O nome da classe do módulo de auditoria padrão é com.sun.enterprise.security.Audit .
