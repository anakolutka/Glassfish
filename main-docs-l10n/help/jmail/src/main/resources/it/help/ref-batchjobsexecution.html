Dettagli esecuzione processo batch

Nella pagina Dettagli esecuzione processo batch vengono visualizzati i dettagli relativi a una esecuzione di processo batch specifica. Vengono visualizzate le informazioni riportate di seguito.

Nome istanza
Nome cluster

Il nome del cluster o dell'istanza server di GlassFish cui si applica la configurazione.

Nome processo

Il nome del processo batch.

ID esecuzione

L'ID assegnato all'esecuzione del processo batch. Una nuova esecuzione viene creata al primo avvio di un processo e a ogni riavvio dell'esecuzione esistente.

Conteggio passi

Il numero di passi nell'esecuzione del processo batch.

Stato batch

Lo stato dell'esecuzione del processo batch impostato dal runtime batch.

Stato di uscita

Lo stato dell'esecuzione del processo batch impostato dall'XML di processo per il processo o dall'applicazione batch. Per impostazione predefinita, lo stato di uscita e lo stato del batch sono uguali a meno che lo stato di uscita non venga sostituito in modo esplicito.

Ora di inizio

L'ora in cui è stata avviata l'esecuzione del processo batch.

Ora di fine

L'ora in cui è stata terminata l'esecuzione del processo batch.

Parametri processo

Le proprietà passate al runtime batch per l'esecuzione del processo batch, elencate come coppie nome-valore.
