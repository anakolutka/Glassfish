Para suprimir un servicio de ejecutor programado gestionado

  1. En el árbol de navegación, amplíe el nodo Recursos.

  2. En el nodo Recursos, amplíe el nodo Recursos simultáneos.

  3. En el nodo Recursos simultáneos, seleccione Servicios de ejecutor programados gestionados.

    Se abrirá la página Servicios de ejecutor programados gestionados.

  4. En la página Servicios de ejecutor programados gestionados, active la casilla de control situada junto al nombre del recurso que desee suprimir.

  5. Haga clic en Suprimir.
