응용 프로그램 모니터링 데이터를 보는 방법

시작하기 전에

모니터링 데이터를 보려면 먼저 모니터링을 구성해야 합니다. 자세한 내용은 모니터링 서비스를 구성하는 방법을 참조하십시오.

  1. 탐색 트리에서 Application Server 노드를 선택합니다.

    일반 정보 페이지가 열립니다.

  2. 일반 정보 페이지에서 모니터 탭을 누릅니다.

    응용 프로그램 모니터링 페이지가 열립니다.

  3. 응용 프로그램 드롭다운 목록에서 해당 응용 프로그램에 대한 모니터링 데이터를 볼 응용 프로그램을 선택합니다.
  4. 모니터링 통계를 검색하려면 통계 가져오기 버튼을 누릅니다.

    테이블에 모니터링 통계가 나타납니다.

  5. (옵션) 갱신 버튼을 눌러 표시된 모니터링 데이터를 업데이트합니다.
참조

  • 모니터링 서비스를 구성하는 방법

  • Enterprise Server에 대한 일반 정보를 보는 방법

  • 서버 로그를 보는 방법

  • 고급 응용 프로그램 구성을 설정하는 방법

  • 고급 도메인 속성을 설정하는 방법

  • 관리자 비밀번호를 변경하는 방법

  • 시스템 등록 정보를 관리하는 방법

  • 서버 모니터링 데이터를 보는 방법
