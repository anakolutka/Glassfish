Die GlassFish-Community entwickelt kostenlose Open Source-Unternehmenssoftware in Produktionsqualität. Die Hauptprodukte sind ein Anwendungsserver, die Java EE 5-Referenzimplementierung und die Referenzimplementierung der Java Persistence-API, TopLink Essentials. Darüber hinaus stellt die Community das Maven-Repository, Tools und vieles mehr bereit.

Projekt SailFin, eine Open Source-Version der SIP Servlets-Technologie für GlassFish
SIP (Session Initiation Protocol) ist die Technologie, die hinter vielen populären Diensten steht, die wir täglich nutzen, so z. B. VoIP-Telefondienst (Voice-over-IP), Instant Messaging, Anwesenheits- und Buddy-Listenverwaltung und Webkonferenzen. Das SailFin-Projekt fügt die SIP Servlet-Technologie dem GlassFish-Anwendungsserver hinzu. Weitere Informationen zu diesem Thema finden Sie unter https://sailfin.dev.java.net.

  
  
  
  
Erste Schritte

Nutzen Sie die GlassFish-Beispiele und Blueprint-Beispiele für einen schnellen Einstieg in Java EE 5. Erfahren Sie mehr über die Migration vorhandener J2EE-Anwendungen auf Java EE 5.

Werden Sie aktiv
Tragen Sie zur Verbesserung des GlassFish-Servers bei, oder informieren Sie sich darüber, wie die GlassFish-Community Technolgien und Funktionen wie Grizzly, EJB 3.0, TopLink Essentials, Admin-Tools und mehr implementiert.

Roadmap
Vor kurzem wurde GlassFish V2 veröffentlicht. Sie können diese Version über die Downloadseite, das Java EE 5-SDK-Update 3 oder über den Sun Java System Application Server 9.1-Download abrufen. Die Community arbeitet ab sofort auf GlassFish V3.
