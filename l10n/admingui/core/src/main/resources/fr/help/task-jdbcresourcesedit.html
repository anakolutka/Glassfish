Modifier une ressource JDBC

  1. Dans l'arborescence de navigation, développez le nœud Ressources.
  2. Sélectionnez le nœud Ressources JDBC.
  3. Dans la page Ressources JDBC, cliquez sur le nom de la ressource que vous modifiez.

    La page Modification d'une ressource JDBC s'ouvre.

    Remarque – Le champ Nom J.N.D.I. est en lecture seule. Vous ne pouvez spécifier le nom J.N.D.I. qu'à la création d'une nouvelle ressource JDBC.

  4. Dans la liste déroulante Nom du pool, sélectionnez un pool de connexions à associer à la ressource JDBC.
  5. Dans le champ Description, entrez une brève description de la ressource JDBC.
  6. Cliquez sur Enregistrer.
