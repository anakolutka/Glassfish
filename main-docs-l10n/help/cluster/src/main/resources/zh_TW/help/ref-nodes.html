節點

「節點」頁面會顯示網域中的節點清單。對於每個節點，都會顯示以下資訊：

名稱

建立節點時指定給節點的名稱。按一下該名稱，隨即會開啟節點的「編輯節點」頁面。

節點主機

節點代表的主機名稱。

類型

下列其中一種的節點類型：

組態

節點不支援遠端通訊。

DCOM

此節點支援透過分散式元件物件模型 (DCOM) 遠端協定進行通訊。DCOM 協定僅在 Windows 系統上提供。

SSH

節點支援透過安全 shell (SSH) 進行通訊。

執行處理

位於節點上的執行處理清單，指出各個執行處理是否正在執行中：

  • 如果正在執行執行處理，便會顯示勾選記號 (✓)。

  • 如果執行處理已停止，便會顯示驚嘆號 (!)。

按一下執行處理名稱，隨即會開啟執行處理的「一般資訊」頁面。

動作

僅適用於 DCOM 與 SSH 類型的節點：節點的 Ping 動作的連結。按一下 Ping 連結，會測試節點是否可經由節點支援的協定連線。

針對 CONFIG 節點，沒有可用的動作。

「節點」表也包含下列選項。

新建

建立節點的按鈕。

刪除

刪除一或多個所選取節點的按鈕。

刪除並解除安裝

刪除一個或多個所選節點並解除安裝每個所刪除節點代表之主機中的 GlassFish Server 軟體的按鈕。

備註:

GlassFish Server 軟體不會從 CONFIG 類型之節點所代表的任何主機上解除安裝。對於 CONFIG 類型的節點，此按鈕的動作等同於「刪除」按鈕的動作。
