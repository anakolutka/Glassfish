Propiedades específicas de módulos de auditoría

La siguientes propiedades están disponibles para un módulo de auditoría.

auditOn

Un valor true origina la carga del módulo de auditoría y garantiza que lo llame la biblioteca de auditoría de GlassFish Server en los puntos de auditoría. Un valor false desactiva el registro de auditorías para el módulo.
