동시 리소스

[동시 리소스] 페이지에서는 다음 유형의 동시 리소스를 구성할 수 있습니다.

  • 컨텍스트 서비스

  • 관리 대상 스레드 팩토리

  • 관리 대상 실행기 서비스

  • 관리 대상 예약된 실행기 서비스

동시 리소스는 JSR 236 사양, Concurrency Utilities for Java EE에 정의된 대로 Java EE 응용 프로그램에 동시성 기능을 제공하는 관리되는 객체입니다.

GlassFish Server에서는 동시 리소스를 구성하여 서블릿, EJB 등의 응용 프로그램 구성 요소에서 사용할 수 있도록 제공합니다. 동시 리소스는 JNDI 조회 또는 리소스 삽입을 통해 액세스됩니다.
