Configurar la Disponibilidad del Contenedor EJB

  1. En el árbol de navegación, amplíe el nodo Configuración.

  2. Seleccione la configuración que desea cambiar.

  3. En el nodo de configuración, seleccione el nodo Servicio de Disponibilidad.

    Se abrirá la página Servicio de disponibilidad.

  4. En la página Servicio de Disponibilidad, haga clic en el separador Disponibilidad de Contenedor EJB.

    Se abrirá la página Disponibilidad de contenedor EJB.

  5. Active la casilla Servicio de disponibilidad activado para activar la disponibilidad del contenedor EJB.

    Para desactivar la disponibilidad, desactive la casilla Servicio de disponibilidad.

  6. El tipo de persistencia HA es replicated.

    Éste es el mecanismo de persistencia utilizado cuando Servicio de disponibilidad activado está activada. La persistencia se basa en failover en un cluster. No cambie este valor.

  7. El tipo de persistencia SFSB es file.

    Éste es el mecanismo de persistencia utilizado cuando Servicio de disponibilidad activado no está activada. La persistencia se basa en el sistema de archivos. No cambie este valor.

    El campo Ubicación del almacén de sesión determina dónde continúan activos los estados del punto de control SFSB. Para definir la ubicación del almacén de sesión, consulte Definir la Configuración de EJB.

  8. Para restablecer los valores por defecto, haga clic en el botón Cargar valores por defecto.

  9. En la sección Propiedades Adicionales, especifique las propiedades adicionales.

    GlassFish Server no define ninguna propiedad adicional para la disponibilidad del contenedor EJB.

    • Para agregar una propiedad, haga clic en el botón Agregar Propiedad. En la fila en blanco que aparece, introduzca el nombre de propiedad en el campo Nombre y escriba el valor de propiedad en el campo Valor.

    • Para modificar una propiedad, edite el campo Valor.

    • Para suprimir una propiedad, active la casilla de control a la izquierda del campo Nombre de la propiedad que está suprimiendo y, a continuación, haga clic en el botón Suprimir Propiedades.

  10. Haga clic en Guardar.
