활성 감사 모듈을 설정하는 방법

  1. 탐색 트리에서 [구성] 노드를 확장합니다.

  2. 구성할 인스턴스를 선택합니다.

    • 특정 인스턴스를 구성하려면 해당 인스턴스의 구성 노드를 확장합니다. 예를 들어, 기본 인스턴스 서버에 대해 server-config 노드를 확장합니다.

    • 모든 인스턴스의 기본 설정을 구성하려면 default-config 노드를 확장합니다.

  3. [구성] 노드에서 [보안] 노드를 선택합니다.

    [보안] 페이지가 열립니다.

  4. [감사 모듈] 드롭다운 목록에서 서버가 사용할 감사 모듈 이름을 선택합니다.

    기본적으로 default라는 이름의 감사 모듈 하나만 사용할 수 있습니다.

  5. [저장]을 누릅니다.

관련 항목

  • 감사 모듈을 생성하는 방법

  • 감사 모듈을 편집하는 방법

  • 감사 모듈을 삭제하는 방법

  • 감사 로깅을 사용 또는 사용 안함으로 설정하는 방법

  • 기본 감사 모듈을 사용하는 방법
