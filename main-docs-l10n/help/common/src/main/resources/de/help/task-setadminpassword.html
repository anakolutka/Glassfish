So wird das Administratorkennwort geändert

  1. Wählen Sie in der Navigationsbaumstruktur den Knoten "Domain".

    Die Seite "Domainattribute" wird geöffnet.

  2. Klicken Sie auf die Registerkarte "Administratorkennwort".

    Die Seite "Neues Administratorkennwort" wird geöffnet

    Die Seite zeigt Benutzer-ID und Gruppenname für den aktuellen Administratoraccount an. Bei der Benutzer-ID handelt es sich standardmäßig um admin und beim Gruppennamen um asadmin.

  3. Geben Sie im Feld "Neues Kennwort" das neue Kennwort für den aktuellen Administratoraccount ein.

  4. Geben Sie in das Feld "Neues Kennwort bestätigen" das Kennwort des Benutzers erneut ein.

  5. Klicken Sie auf "Speichern".
