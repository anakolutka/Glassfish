To Enable GMS for a Cluster

You must enable the Group Management Service (GMS) for each cluster individually.

  1. In the tree component, expand the Clusters node.
  2. Select the node for the cluster for which you are enabling GMS.
  3. In the General Information tab, ensure that the Heartbeat Enabled option is checked.
  4. If necessary, change the default port and IP address for GMS.
  5. Click Save.
