Para Interromper um Cluster

Interromper um cluster interrompe todas as instâncias de GlassFish Server no cluster.

  1. Na árvore de navegação, selecione o nó Clusters.

    A página Clusters será aberta.

  2. Na página Clusters, marque a caixa de seleção ao lado do nome do cluster.

  3. Clique em Interromper Cluster.

    Será solicitado que você confirme se deseja interromper o cluster.

  4. Confirme se você deseja interromper o cluster.

    O cluster é interrompido. O status de cada instância do cluster é atualizado na página Clusters quando a instância é interrompida.
