Risorse

Utilizzare la pagina Risorse per eseguire i task seguenti sull'istanza standalone di GlassFish Server selezionata:

  • visualizzazione e gestione delle risorse esistenti dell'istanza;

  • creazione di risorse aggiuntive per l'istanza.

Il campo Nome istanza è un campo di sola lettura in cui viene visualizzato il nome dell'istanza corrente.

Nella pagina Risorse viene visualizzata la lista delle risorse esistenti dell'istanza. Per ogni risorsa vengono fornite le informazioni riportate di seguito.

Nome risorsa

Il nome assegnato alla risorsa al momento della creazione. Se si fa clic sul nome, viene visualizzata una delle pagine di modifica seguenti, in base al tipo di risorsa:

  • Modifica risorsa JDBC

  • Modifica sessione JavaMail

  • Modifica risorsa personalizzata

  • Nuova risorsa personalizzata

  • Modifica risorsa esterna

  • Modifica risorsa connettore

  • Modifica risorsa oggetto amministrato

Abilitato

Indica se la risorsa è abilitata:

  • se la risorsa è abilitata, viene visualizzato un segno di spunta (✓);

  • se la risorsa è disabilitata, viene visualizzata una croce (✗).

Tipo

Il tipo della risorsa.

La tabella Risorse contiene inoltre le opzioni riportate di seguito.

Abilita

Pulsante che consente di abilitare una o più risorse selezionate.

Disabilita

Pulsante che consente di disabilitare una o più risorse selezionate.

Nuovo

Elenco a discesa che consente di creare una risorsa del tipo selezionato. Se si seleziona un tipo di risorsa dalla lista, viene visualizzata una delle pagine seguenti per creare una risorsa del tipo selezionato:

  • Nuova risorsa JDBC

  • Nuovo Connection Factory JMS

  • Nuova risorsa di destinazione JMS

  • Nuova sessione JavaMail

  • Nuova risorsa personalizzata

  • Nuova risorsa esterna

  • Nuova risorsa connettore

  • Nuova risorsa oggetto amministrato

Filtro

Elenco a discesa che consente di filtrare la lista delle risorse visualizzate per tipo.
