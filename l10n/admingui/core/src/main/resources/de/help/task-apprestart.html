Eine bereitgestellte Anwendung erneut starten

Durch erneutes Starten der Anwendung wird die Anwendung, ohne erneute Bereistellung, lediglich neu gestartet, wobei die Einstellungen und anderen Optionen der ursprünglichen Anwendungsbereitstellung beibehalten werden.

  1. Wählen Sie in der Navigationsstruktur den Anwendungsknoten.

    Die Seite "Anwendungen" wird geöffnet.

  2. Klicken Sie in der Tabelle "Bereitgestellte Anwendungen" in der Zeile für die neu bereitzustellende Anwendung auf "Erneut starten".

    Die Seite "Anwendungen" öffnet sich erneut mit einer Meldung, die angibt, ob der erneute Start erfolgreich war.
