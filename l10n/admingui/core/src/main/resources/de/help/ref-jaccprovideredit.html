JACC-Anbieter bearbeiten

Verwenden Sie die Seite JACC-Anbieter bearbeiten, um einen JACC-Anbieter zu ändern.

Die Seite JACC-Anbieter bearbeiten enthält die folgenden Optionen.

Name

Der Name des JACC-Anbieters. Dies ist ein schreibgeschütztes Feld. Sie können den Namen nur angeben, wenn Sie einen neuen JACC-Anbieter erstellen.

Richtlinienkonfiguration

Der Name der Klasse, die die Richtlinienkonfigurations-Factory implementiert.

Richtlinienanbieter

Der Name der Klasse, die die Richtlinien-Factory implementiert.

Additional Properties (Weitere Eigenschaften)

Weitere Eigenschaften für den JACC-Anbieter. Eine Beschreibung der weiteren Eigenschaften, die für die JACC-Anbieter zur Verfügung stehen, finden Sie unter Besondere Eigenschaften von JACC-Anbietern.

Verwandte Aufgaben

  • So erstellen Sie einen JACC-Anbieter

  • So bearbeiten Sie einen JACC-Anbieter

  • So löschen Sie einen JACC-Anbieter

  • So stellen Sie den aktiven JACC-Anbieter ein

Verwandte asadmin-Befehle

Keine.
