Auditoría de eventos de Servidor empresarial

Auditoría de cómo Servidor empresarial registra ciertos sucesos, como errores o violaciones de seguridad. Servidor empresarial registra todos los sucesos de autenticación en archivos de registro. Un registro de acceso completo ofrece un rastro secuencial de los sucesos de acceso de Servidor empresarial.

La auditoría de eventos de Servidor empresarial incluye las tareas siguientes:

  • Cómo crear un módulo de auditoría

  • Cómo editar un módulo de auditoría

  • Cómo eliminar un módulo de auditoría

  • Cómo habilitar o deshabilitar el registro de auditoría

  • Cómo establecer el módulo de auditoría activo

  • Cómo utilizar el módulo de auditoría predeterminado
