Nouveau module d'audit

La page Nouveau module d'audit permet de définir un nouveau module d'audit.

Elle contient les options ci-après.

Nom de configuration

Nom de la configuration à laquelle s'appliquent les paramètres de cette page. Ce champ est en lecture seule.

Nom

Nom du nouveau module d'audit.

Nom de classe

Nom qualifié complet de la classe qui implémente ce module. Le nom de classe du module d'audit par défaut est com.sun.enterprise.security.Audit.

Autres propriétés

Propriétés supplémentaires pour le module d'audit. Pour obtenir une description de la propriété supplémentaire disponible dans les modules d'audit, reportez-vous à Propriétés propres aux modules d'audit.

Tâches connexes

  • Création d'un module d'audit

  • Modification d'un module d'audit

  • Suppression d'un module d'audit

  • Activation ou désactivation de la journalisation d'audit

  • Définition du module d'audit actif

  • Utilisation du module d'audit par défaut

Commandes asadmin connexes

  • list-audit-modules(1)
