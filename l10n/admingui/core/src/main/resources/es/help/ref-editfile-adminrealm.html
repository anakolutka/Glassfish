Propiedades específicas de la clase FileRealm

Un dominio file requiere las siguientes propiedades:

Contexto JAAS

Tipo de módulo de inicio de sesión para usar en este dominio. El único valor válido es fileRealm.

Archivo de claves

Ruta completa y nombre del archivo en el que el servidor almacenará toda la información de este dominio relativa al usuario, el grupo y la contraseña. El valor predeterminado es domain-dir/config/keyfile para el dominio file y domain-dir/config/admin-keyfile para el dominio admin-realm .

El archivo de claves del dominio file está vacío inicialmente, de manera que los usuarios deben de registrarse en la página Archivar usuarios antes de utilizar el dominio file.

El archivo de claves del dominio admin-realm contiene inicialmente el nombre de usuario del administrador, la contraseña del administrador en formato cifrado y el grupo al que el usuario pertenece (de manera predeterminada es asadmin).

Nota – Los usuarios del grupo asadmin del dominio admin-realm tienen autorización para utilizar la Consola de administración y las herramientas asadmin. Agregue a este grupo sólo los usuarios que tienen privilegios de administración de servidores.
