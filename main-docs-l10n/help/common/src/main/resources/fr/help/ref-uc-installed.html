Composants installés

La page Composants installés contient des informations sur les composants de l'image d'installation déjà installés.

Pour désinstaller des composants installés, commencez par arrêter le domaine GlassFish Server. Utilisez ensuite l'une des commandes suivantes :

  • glassfish3/bin/updatetool, qui démarre l'utilitaire graphique de l'outil de mise à jour

  • glassfish3/bin/pkg, version de ligne de commande de l'outil de mise à jour

Pour plus d'informations sur l'utilisation de la commande pkg, reportez-vous au chapitre 11, Extending and Updating GlassFish Server, dans le manuel Oracle GlassFish Server Administration Guide. L'utilitaire de l'outil de mise à jour possède sa propre aide en ligne.

Redémarrez le serveur après avoir terminé la désinstallation des composants.

La page Composants installés contient les options ci-après.

Chemin d'installation

Chemin du système de fichiers du répertoire racine ou du répertoire de niveau supérieur de l'image d'installation.

Composants installés

Pour chacun des composants installés, cette table fournit les informations ci-après.

Composant

Nom du composant.

Catégorie

Catégorie du logiciel auquel le composant est associé (par exemple, serveurs d'applications, outils système et bases de données). La catégorie est affectée au composant par le fabricant au moment de la création du composant.

Version

Numéro de version du package comprenant le composant.

Taille après installation

Espace disque utilisé pour l'installation du composant.

Source

Nom du Référentiel réseau à partir duquel le composant a été obtenu.
