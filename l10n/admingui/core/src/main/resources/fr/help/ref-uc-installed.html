Composants installés

La page Composants installés contient des informations sur les composants de l'image d'installation déjà installés.

La page Composants installés contient les options suivantes :

Chemin d'installation

Chemin du système de fichiers du répertoire root ou du répertoire supérieur de l'image d'installation.

Supprimer

Ce bouton supprime les composants sélectionnés.

Installed Components

Le tableau présente les informations ci-dessous pour chaque composant installé :

Composant

Nom du composant.

Catégorie

Catégorie du logiciel à laquelle le composant est associé, comme par exemple serveurs d'application, outils système et bases de données. La catégorie est affectée au composant par le fabricant du composant au moment de sa création.

Version

Numéro de version du package comprenant le composant.

Taille de l'installation

Espace disque utilisé pour l'installation du composant.

Source

Pour un composant installé, nom du Référentiel réseau à partir duquel le composant a été obtenu.

Pour un composant add-on ou une mise à jour, nom du référentiel réseau à partir duquel le composant a été obtenu.
