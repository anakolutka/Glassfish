Editar de ciclo de vida

Consulte la página Editar Módulo de Ciclo de Vida para modificar la configuración del módulo de ciclo de vida.

La página Editar Módulo de Ciclo de Vida contiene las opciones siguientes.

Nombre

Nombre del módulo de ciclo de vida. Este campo es de sólo lectura. Sólo puede especificar un nombre al crear un nuevo módulo de ciclo de vida.

Nombre de clase

Nombre completo del archivo de clase del módulo de ciclo de vida. La clase debe implantar la interfaz com.sun.appserv.server.LifecycleListener.

Classpath

Classpath para el módulo de ciclo de vida. La classpath especifica la ubicación del módulo de ciclo de vida. El valor por defecto es domain-dir/applications. Si el módulo ya se encuentra en este directorio (es decir, en la classpath del servidor), este campo puede dejarse en blanco.

Orden de Carga

Orden en el que este módulo de ciclo de vida se cargará al inicio. Los módulos con valores de orden de carga de enteros inferiores se cargarán antes. El valor puede estar entre 101 y el MAXINT del sistema operativo. Los valores de 1 a 100 están reservados.

Descripción

Descripción del módulo de ciclo de vida.

Estado

Si se selecciona esta opción, el módulo de ciclo de vida estará activado. Esta opción se encuentra activada por defecto.

Ante Fallo de Carga

Si se selecciona esta opción, el servidor se apagará si se produce un fallo en la carga del módulo de ciclo de vida. Esta opción se activada por defecto.
