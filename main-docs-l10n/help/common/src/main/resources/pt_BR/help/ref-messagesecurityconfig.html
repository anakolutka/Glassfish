Configurações de Segurança de Mensagem

Use a página Configurações de Segurança de Mensagem para exibir e modificar as configurações disponíveis para provedores de segurança de mensagens.

Para configurar a segurança de mensagens, você precisa de uma compreensão básica de conceitos de segurança e Web Services. Para obter mais informações, consulte Capítulo 3, Administering Message Security, em Oracle GlassFish Server Security Guide e Capítulo 6, Administering Web Applications, em Oracle GlassFish Server Administration Guide.

O GlassFish Server implementa a segurança de mensagens usando provedores de segurança de mensagens integrados em sua camada de processamento SOAP. Os provedores de segurança de mensagens dependen de outros recursos de segurança de GlassFish Server.

Para cada configuração, as seguintes informações são fornecidas.

Camada de Autenticação

A camada de mensagens na qual a autenticação é realizada. Por default, uma configuração SOAP é fornecida. Também é possível fornecer uma configuração HttpServlet.

Provedor Default

O provedor do servidor default desta configuração. Por default, nenhum provedor de servidor default está configurado.

Provedor Cliente Default

O provedor de cliente default desta configuração. Por default, nenhum provedor de cliente default está configurado.

A tabela Configuração de Segurança de Mensagem também contém as seguintes opções.

Novo

Botão para criar uma nova configuração.

Deletar

Botão para deletar uma ou mais configurações selecionadas.
