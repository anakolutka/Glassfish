활성 감사 모듈을 설정하는 방법

시작하기 전에

서버에서 사용할 감사 모듈을 지정하려면 먼저 감사 로깅을 활성화하거나 비활성화하는 방법에서 설명한 대로 감사 로깅을 활성화합니다.

  1. 감사 모듈 필드에 서버에서 사용할 감사 모듈 이름을 선택합니다.

    기본값은 default입니다. 이 감사 모듈의 auditOn 등록 정보가 기본 감사 모듈을 사용하는 방법에서 설명한 대로 true로 설정되어 있는지 확인하십시오.

  2. 저장을 누릅니다.
  3. 콘솔에 다시 시작해야 함이 표시되면 Enterprise Server를 다시 시작합니다.
참조

  • 감사 모듈을 만드는 방법

  • 감사 모듈을 편집하는 방법

  • 감사 모듈을 삭제하는 방법

  • 감사 로깅을 활성화하거나 비활성화하는 방법

  • 기본 감사 모듈을 사용하는 방법
