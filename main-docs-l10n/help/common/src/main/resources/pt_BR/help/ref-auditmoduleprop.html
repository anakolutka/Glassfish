Propriedades Específicas para os Módulos de Auditoria

A propriedade a seguir está disponível para um módulo de auditoria.

auditOn

Um valor de true causa o carregamento do módulo de auditoria e garante que ele seja denominado pela biblioteca de auditoria GlassFish Server em pontos de auditoria. Um valor false desativa o log de auditoria do módulo.
