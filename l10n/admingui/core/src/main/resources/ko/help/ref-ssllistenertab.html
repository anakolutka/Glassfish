SSL 수신기

SSL 수신기가 서버 인증서를 HTTP 수신기 포트에 바인딩합니다. SSL 설정은 새 HTTP 수신기를 만들 때 새 HTTP 수신기(단계 2/2) 페이지를 사용하여 정의합니다. 이후에 SSL 수신기 페이지에서 SSL 설정을 추가로 편집할 수 있습니다.

클라이언트 인증

이 수신기를 사용할 때 클라이언트가 자신을 서버에 인증하도록 하려면 활성화합니다.

인증서 별명

기존 서버 키 쌍 및 인증서의 이름입니다.

SSL3

이 옵션을 활성화하면 수신기에서 SSL3 보안 프로토콜이 사용됩니다.

SSL2

이 옵션을 활성화하면 수신기에서 SSL2 보안 프로토콜이 사용됩니다.

TLS

이 옵션을 활성화하면 수신기에서 TLS 보안 프로토콜이 사용됩니다.

공통 암호화 제품군

프로토콜이 사용할 수 있는 암호 제품군입니다.

사용 후 삭제되는 Diffie-Hellman 암호화 제품군

프로토콜이 사용하는 삭제되는 암호 제품군입니다.

관련 작업

  • SSL을 사용하도록 HTTP 수신기를 구성하는 방법

  • HTTP 수신기를 편집하는 방법

  • HTTP 수신기를 삭제하는 방법
