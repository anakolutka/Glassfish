Gestire le destinazioni di una sessione JavaMail

  1. Espandere il nodo Risorse nell'albero di navigazione.

  2. Nel nodo Risorse selezionare Sessioni JavaMail.

    Viene visualizzata la pagina Sessioni JavaMail.

  3. Nella tabella Sessioni fare clic sul nome della sessione di cui si desidera gestire le destinazioni.

    Viene visualizzata la pagina Modifica sessione JavaMail.

  4. Fare clic sulla scheda Destinazione.

    Viene visualizzata la pagina Destinazioni sessioni JavaMail.

  5. Nella tabella Destinazioni, scegliere l'azione da eseguire:

    • per aggiungere o rimuovere destinazioni, fare clic su Gestisci destinazioni;

    • per abilitare una destinazione, fare clic sulla casella di controllo accanto all'istanza server o al cluster desiderato e fare clic su Abilita;

    • per disabilitare una destinazione, fare clic sulla casella di controllo accanto all'istanza server o al cluster desiderato e fare clic su Disabilita.
