Sitzungseigenschaften

Verwenden Sie die Seite "Sitzungseigenschaften", um Eigenschaften für den Webcontainer hinzuzufügen oder zu ändern. Die Seite "Sitzungseigenschaften" umfasst die folgenden Optionen:

Sitzungszeitüberschreitung

Gibt die Anzahl der Sekunden für die Gültigkeit der Sitzung an.

Weitere Eigenschaften

Weitere Sitzungseigenschaften.

  • Zum Hinzufügen einer Eigenschaft klicken Sie auf die Schaltfläche "Eigenschaft hinzufügen". Geben Sie in der daraufhin angezeigten leeren Zeile den Eigenschaftsnamen in das Feld "Name" und den Eigenschaftswert in das Feld "Wert" ein. Klicken Sie auf "Speichern".

  • Um eine Eigenschaft zu ändern, bearbeiten Sie das Feld "Wert" dieser Eigenschaft. Klicken Sie auf "Speichern".

  • Um eine Eigenschaft zu löschen, aktivieren Sie das Kontrollkästchen links neben dem Feld "Name" der zu löschenden Eigenschaft, und klicken Sie anschließend auf die Schaltfläche "Eigenschaften löschen". Klicken Sie auf "Speichern".

Verwandte Aufgaben

  • So konfigurieren Sie Manager-Eigenschaften

  • So konfigurieren Sie die Speichereigenschaften
