Como establecer el dominio predeterminado

El dominio predeterminado es el dominio que Servidor empresarial utiliza para la autenticación y autorización en caso de que el descriptor de implementación de una aplicación no especifique un dominio.

  1. En el árbol de navegación, expanda el nodo Configuración.
  2. Si el dominio está configurado para admitir clústeres, seleccione la configuración que esté cambiando.

    Algunos ejemplos son los dominios que se crean con el perfil clúster o con el perfil grandes empresas .

  3. Seleccione el nodo Seguridad.

    Se abrirá la página Seguridad.

  4. En el campo Dominio predeterminado, escoja el dominio deseado de la lista desplegable.
  5. Haga clic en Guardar para guardar los cambios.
  6. Reinicie el servidor si aparece el mensaje Reinicio necesario en la consola.
