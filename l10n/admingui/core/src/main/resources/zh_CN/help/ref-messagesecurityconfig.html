消息安全性配置

“消息安全性配置”页面列出消息安全性提供者的可用配置。

要配置消息安全性，需要基本了解安全性和 Web 服务概念。有关更多信息，请参见《Sun GlassFish Enterprise Server v3 Administration Guide》中的第 13  章 "Administering Message Security"和《Sun GlassFish Enterprise Server v3 Administration Guide》中的第 6  章 "Administering Web Applications"。

Enterprise Server 使用在其 SOAP 处理层中集成的消息安全性提供者来实现消息安全性。消息安全性提供者取决于 Enterprise Server 的其他安全性工具。

对于每个配置，“消息安全性配置”页面包含以下信息：

验证层

执行验证的消息层。默认情况下，已提供 SOAP 配置。还可以提供 HttpServlet 配置。

默认提供者

此配置的默认服务器提供者。默认情况下，不配置任何默认服务器提供者。

默认客户机提供者

此配置的默认客户机提供者。默认情况下，不配置任何默认客户机提供者。
