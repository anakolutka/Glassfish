Virtuelle Server

Ein virtueller Server, auch als virtueller Host bezeichnet, ist ein Objekt, mit dessen Hilfe derselbe physikalische Server als Host für mehrere Internet-Domänennamen fungieren kann. Alle auf demselben physikalischen Server gehosteten virtuellen Server nutzen dieselbe IP-Adresse dieses physikalischen Servers. Ein virtueller Server verbindet einen Domänennamen für einen Server (wie etwa www.sun.com) mit dem Server, auf dem Enterprise Server ausgeführt wird.
