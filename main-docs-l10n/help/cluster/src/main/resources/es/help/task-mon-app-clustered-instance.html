Ver datos de supervisión de aplicaciones para una instancia de GlassFish Server en cluster

Antes de Empezar

Para poder ver datos de supervisión, debe configurar el servicio de supervisión.

  1. En el árbol de navegación, amplíe el nodo Clusters.

  2. En el nodo Clusters ampliado, seleccione el nodo para el cluster.

    Se abre la página Información General para el cluster.

  3. En la página Información General, haga clic en el separador Instancias.

    Se abrirá la página Instancias del Servidor en Cluster para el cluster.

  4. Seleccione la instancia para la que desea ver los datos de supervisión de aplicaciones.

    Se abrirá la página Información General para la instancia.

  5. En la página Información General, haga clic en el separador Supervisar.

    Se abrirá la página Supervisión de Aplicación.

  6. En la lista desplegable Ver, seleccione la aplicación para la cual desea ver los datos de supervisión.

  7. Para actualizar los datos de supervisión mostrados, haga clic en Refrescar.
