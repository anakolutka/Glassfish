Para excluir um usuário do domínio arquivo

  1. Na árvore de navegação, expanda o nó Configuração.
  2. Expanda o nó Segurança.
  3. Expanda o nó Domínios.
  4. Selecione o nó Arquivo.

    A página Editar Domínio é exibida.

  5. Na página Editar Domínio, clique no botão Gerenciar Usuários.

    A página Usuários do Arquivo é exibida.

  6. Na página Usuários do Arquivo, marque a caixa de seleção à esquerda do nome do usuário a ser excluído.
  7. Clique em Excluir.

    Uma caixa de diálogo pede que você confirme a exclusão do usuário.

  8. Na caixa de diálogo, clique em OK.
