Enterprise Server 이벤트 감사

감사는 Enterprise Server에서 오류나 보안 위반과 같은 중요한 이벤트를 기록하는 방법입니다. Enterprise Server는 모든 인증 이벤트를 로그 파일에 기록합니다. 전체 액세스 로그는 Enterprise Server 액세스 이벤트의 순차적인 추적을 제공합니다.

Enterprise Server 이벤트 감사에는 다음 작업이 포함됩니다.

  • 감사 모듈을 만드는 방법

  • 감사 모듈을 편집하는 방법

  • 감사 모듈을 삭제하는 방법

  • 감사 로깅을 활성화하거나 비활성화하는 방법

  • 활성 감사 모듈을 설정하는 방법

  • 기본 감사 모듈을 사용하는 방법
