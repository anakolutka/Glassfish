Informations générales

Utilisez la page Informations générales pour vérifier les paramètres d'Enterprise Server et afficher les données de la Java Virtual Machine.

La page Informations générales contient les informations suivantes.

Bouton Stop (Arrêter)

Cliquez sur le bouton Stop pour arrêter le Enterprise Server.

Bouton Redémarrer

Cliquez sur le bouton Redémarrer pour redémarrer le Enterprise Server.

Bouton Afficher les fichiers journaux

Cliquez sur le bouton Afficher les fichiers journaux pour afficher le journal du serveur.

Nom

Nom du serveur.

Rapport JVM

Si vous cliquez sur Rapport JVM, une fenêtre distincte s'ouvre et affiche des rapports sur la Java Virtual Machine, notamment un rapport de synthèse, un rapport de gestion de mémoire et de collecte de déchets, un rapport de chargement de classes et le vidage des threads en cours.

Répertoire de configuration

Répertoire de la machine hôte qui contient les fichiers de configuration, tels que domain.xml.

Version installée

Identifiant de la version du produit.

Débogage

Indique si le débogage est activé. Dans l'affirmative, le numéro de port est également affiché.

Port(s) HTTP

Ports actuellement configurés pour les requêtes et les réponses HTTP.

Port(s) IIOP

Ports actuellement configurés pour les requêtes et les réponses IIOP.
