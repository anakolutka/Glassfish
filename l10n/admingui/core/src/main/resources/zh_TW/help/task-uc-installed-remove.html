解除安裝元件

解除安裝元件包括移除下列項目：

  • 最初安裝元件時加入的所有檔案和目錄

  • 最初安裝元件後更新元件時加入的所有檔案和目錄

  1. 在瀏覽樹狀結構中，選取 [更新工具] 節點。

    [可用的附加元件] 頁面會隨即開啟。

  2. 在 [可用的附加元件] 頁面上，按一下 [已安裝的元件] 標籤。

    [已安裝的元件] 頁面會隨即開啟。

  3. 在已安裝的元件表格中，選取您要解除安裝的元件。
    • 若要選取個別元件，請選取元件名稱旁的核取方塊。
    • 若要選取所有元件，請按一下表格標頭中的 [全選] 圖示。
    • 若要取消選取所有元件，請按一下表格標頭中的 [全不選] 圖示。
  4. 按一下 [移除]。

    對話方塊隨即出現，請您確認的確應解除安裝所選的元件。

  5. 在對話方塊中，按一下 [確定]。

    Enterprise Server 會確認解除安裝已完成。元件便會從已安裝的元件表格中移除，並且增加到 [可用的附加元件] 頁面上可用的附加元件表格中。
