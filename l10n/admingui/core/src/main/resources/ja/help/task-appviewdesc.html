アプリケーションの記述子を表示する

  1. ナビゲーションツリーで、「アプリケーション」ノードを選択します。

    「アプリケーション」ページが開きます。

  2. 「アプリケーション」ページで、記述子を表示するアプリケーションの名前をクリックします。

    「アプリケーションを編集」ページが開きます。

  3. 「アプリケーションを編集」ページで、「記述子」タブをクリックします。

    「モジュール記述子」 ページが開きます。

  4. 「配備記述子」表で、表示する XML ファイルの名前をクリックします。

    新規ウィンドウで XML ファイルが開きます。
