관리 대상 스레드 팩토리 편집

[관리 대상 스레드 팩토리 편집] 페이지에서는 관리 대상 스레드 팩토리 리소스를 편집할 수 있습니다.

[관리 대상 스레드 팩토리 편집] 페이지에는 다음 옵션이 포함되어 있습니다.

기본값 로드

기본값을 사용하는 설정을 기본값으로 복원하는 단추입니다. 기본값을 사용하지 않는 설정은 변경되지 않습니다.

JNDI 이름

리소스를 식별하는 고유한 이름입니다. [JNDI 이름]은 읽기 전용 필드입니다. 새 리소스를 생성할 때만 JNDI 이름을 지정할 수 있습니다.

논리적 JNDI 이름

리소스의 논리적 JNDI 이름입니다. 리소스에 액세스할 때 사용하는 Java EE 표준 이름입니다. [논리적 JNDI 이름]은 읽기 전용 필드입니다. 자세한 내용은 논리적 JNDI 이름을 참조하십시오.

컨텍스트 정보

스레드로 전달할 컨텍스트입니다(Classloader, JNDI, Security 또는 WorkArea). 컨텍스트 전달은 사용 또는 사용 안함으로 설정할 수 있습니다. 사용으로 설정되면 선택된 컨텍스트가 전달됩니다. 사용 안함으로 설정되면 선택된 컨텍스트를 포함하여 어떤 컨텍스트도 전달되지 않습니다. 기본값은 [사용]이며 4가지 컨텍스트가 모두 선택되어 있습니다.

상태

런타임에 리소스를 사용할 수 있는지 여부입니다.

기본 서버 인스턴스인 server가 도메인에 존재하는 경우에만 리소스의 런타임 가용성을 사용 또는 사용 안함으로 설정할 수 있습니다. 런타임 가용성은 기본적으로 사용으로 설정됩니다.

도메인에 클러스터 또는 독립형 인스턴스가 생성된 경우 [대상] 탭을 사용하여 해당 대상에서 리소스의 런타임 가용성을 제어할 수 있습니다.

스레드 우선순위

생성된 스레드에 지정되는 우선순위입니다. 기본값은 5입니다.

배치 순서

이 리소스의 배치 순서입니다. 서버 시작 시 낮은 번호의 리소스가 먼저 로드됩니다. 기본값은 100입니다.

설명

해당 리소스에 대한 설명입니다.

추가 속성

리소스에 대한 추가 속성입니다. GlassFish Server에서는 이 리소스 유형에 대한 추가 속성을 정의하지 않습니다. 또한 현재 이 리소스 유형은 추가 속성을 사용하지 않습니다.
