Externe Ressourcen

Eine externe JNDI-Ressource greift auf ein externes JNDI-Repository zu, wie z.B. einen LDAP-Server.

Auf der Seite "Externe Ressourcen" können Sie externe JNDI-Ressourcen konfigurieren.

Für jede externe Ressource werden die folgenden Informationen zur Verfügung gestellt.

JNDI-Name

Ein eindeutiger Name, der die externe Ressource identifiziert.

Aktiviert

"True", wenn die externe Ressource aktiviert ist, bzw. "False", wenn sie deaktiviert ist.

Ressourcentyp

Der Typ der externen Ressource.

Beschreibung

Eine Beschreibung der externen Ressource.

Die Tabelle "Ressourcen" enthält darüber hinaus die folgenden Optionen.

Neu

Schaltfläche zum Erstellen einer neuen externen Ressource.

Löschen

Schaltfläche zum Löschen von mindestens einer ausgewählten externen Ressource.

Aktivieren

Schaltfläche zum Aktivieren von mindestens einer ausgewählten benutzerdefinierten Ressource.

Deaktivieren

Schaltfläche zum Deaktivieren von mindestens einer ausgewählten externen Ressource.

Verwandte Aufgaben

  • So wird eine externe JNDI-Ressource erstellt

  • So wird eine externe JNDI-Ressource bearbeitet

  • So wird eine externe JNDI-Ressource gelöscht

Zugehörige asadmin-Befehle

  • create-jndi-resource(1)

  • list-jndi-resources(1)

  • delete-jndi-resource(1)
