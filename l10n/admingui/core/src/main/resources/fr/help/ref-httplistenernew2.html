Nouveau listener HTTP (étape 2 sur 2)

La page Nouveau listener HTTP (étape 2 sur 2) contient les options suivantes :

Authentification du client

Sélectionnez cette option pour forcer les clients à s'authentifier auprès du serveur lors de l'utilisation de ce module d'écoute.

Surnom du certificat

Nom d'une paire de clés de serveur existant et de certificat.

SSL3

Si cette option est activée, le protocole de sécurité SSL3 est appliqué au listener.

SSL2

Si cette option est activée, le protocole de sécurité SSL2 est appliqué au listener.

TLS

Si cette option est activée, le protocole de sécurité TLS est appliqué au listener.

Suites de chiffrement fréquentes

Suites de chiffrement utilisées par les protocoles.

Suites de chiffrement Diffie-Hellman temporaires

Suites de chiffrement temporaires utilisées par les protocoles.
