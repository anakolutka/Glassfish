Para editar um aplicativo implantado

  1. Na árvore de navegação, selecione o nó Aplicativos.

    A página Aplicativos é exibida.

  2. Na tabela Aplicativos Implantados, clique no nome do aplicativo a ser editado.

    A página Editar Aplicativo é exibida.

  3. Para um aplicativo Web, no campo Raiz de Contexto, digite o caminho do aplicativo.

    Na URL do aplicativo Web, a raiz de contexto aparece imediatamente após o número da porta (http://host:port/ context-root/...). Verifique se a raiz de contexto começa com uma barra; por exemplo, /hello.

  4. No campo Descrição, digite uma descrição para o aplicativo.
  5. Desmarque a caixa de seleção Status Ativado para desativar o status do aplicativo, para que ele fique indisponível.

    Por padrão, o status de um aplicativo é Ativado, assim que é implantado.

  6. Marque a caixa de seleção Java Web Start Ativado para usar o software Java Web Start para implantar o aplicativo.
  7. Clique em Salvar.
