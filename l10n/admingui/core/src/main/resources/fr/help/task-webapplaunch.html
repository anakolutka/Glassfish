Lancer une application Web

Avant de commencer

Assurez-vous que l'application Web que vous voulez utiliser est déployée.

  1. Dans l'arborescence de navigation, développez le nœud Applications.
  2. Sélectionnez le nœud Applications Web.

    La page Applications Web s'ouvre.

  3. Dans le tableau Applications Web déployées, cliquez sur Lancer sur la ligne de l'application que vous voulez lancer.

    L'application s'ouvre dans une nouvelle fenêtre de navigation.
