JDBC 자원

JDBC(Java DataBase Connectivity) 자원(데이터 소스)은 응용 프로그램을 데이터베이스에 연결하는 방법을 제공합니다일반적으로 관리자는 도메인에 배포된 응용 프로그램에서 액세스하는 각 데이터베이스에 대해 JDBC 자원을 만들지만 데이터베이스에 대해 둘 이상의 JDBC 자원을 만들 수 있습니다.

응용 프로그램은 J.N.D.I.(Java Naming and Directory Interface) API 트리에서 데이터 소스를 찾은 다음 연결을 요청하여 연결 풀로부터 데이터베이스 연결을 가져옵니다. 데이터 소스와 연결된 연결 풀은 응용 프로그램에 대한 연결을 제공합니다.

JDBC 자원 페이지에서는 Application Server에서 만든 정보 목록을 표시합니다. 각 자원에 대해 제공되는 정보는 다음과 같습니다.

  • J.N.D.I. 이름

  • 상태

  • 연결 풀

  • 설명
