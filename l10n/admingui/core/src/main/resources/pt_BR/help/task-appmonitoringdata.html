Para exibir dados de monitoramento do aplicativo

Antes de começar

Antes de você poder exibir os dados de monitoramento, é preciso configurar o monitoramento. Consulte Para configurar o serviço de monitoramento para obter os detalhes.

  1. Na árvore de navegação, selecione o nó Servidor de Aplicativos.

    A página Informações Gerais é exibida.

  2. Na página Informações Gerais, clique na guia Monitor.

    A página Monitoramento do Aplicativo é exibida.

  3. Na lista suspensa Aplicativo, selecione o aplicativo para visualizar os dados de monitoramento desse aplicativo.
  4. Clique no botão Obter Estatísticas para recuperar as estatísticas de monitoramento.

    As estatísticas de monitoramento aparecem na tabela.

  5. (Opcional) Clique no botão Atualizar para atualizar os dados de monitoramento exibidos.
