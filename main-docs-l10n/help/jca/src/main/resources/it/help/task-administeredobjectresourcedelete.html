Per eliminare una risorsa oggetto amministrato

  1. Espandere il nodo Risorse nell'albero di navigazione.

  2. Espandere il nodo Connettori.

  3. Nel nodo Connettori selezionare il nodo Risorse oggetto amministrato.

    Viene visualizzata la pagina Risorse oggetto amministrato.

  4. Nella pagina Risorse oggetto amministrato selezionare la casella di controllo in corrispondenza del nome della risorsa oggetto amministrato da eliminare.

  5. Fare clic su Elimina.
