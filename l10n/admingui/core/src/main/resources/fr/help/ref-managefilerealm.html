Utilisateurs de fichiers...

Gérez les utilisateurs du domaine fichier à l'aide de la Console d?administration. Les utilisateurs et les groupes du domaine file sont répertoriés dans le fichier clé, dont l'emplacement est spécifié par la propriété file.

Remarque - Vous pouvez également utiliser ces étapes pour ajouter des utilisateurs à un domaine de fichier, y compris le domaine administrateur. Entrez simplement le nom du domaine cible à la place du domaine fichier référencé dans cette section.

Un utilisateur du domaine fichier peut appartenir à un a groupe Java, c'est-à-dire une catégorie d'utilisateurs classés par caractéristiques communes. Par exemple, les clients d'une application d'e-commerce appartiendront au groupe CUSTOMER, tandis que les plus gros clients appartiendront au groupe PREFERRED. La classification des utilisateurs dans des groupes facilite le contrôle de l'accès d'un grand nombre d'utilisateurs.

Une fois le Enterprise Server installé, le seul utilisateur est l'administrateur configuré lors de l'installation. Par défaut, cet utilisateur appartient au groupe asadmin, du domaine administrateur, qui autorise la modification du Enterprise Server. Les utilisateurs rattachés à ce groupe ont des privilèges d'administrateur, c'est-à-dire qu'ils ont accès à l'outil asadmin et à la Console d?administration.

La page Utilisateurs de fichiers affiche une liste des utilisateurs pour le domaine sélectionné. Pour chaque utilisateur, les informations suivantes sont fournies :

ID utilisateur

Nom d'utilisateur.

Liste des groupes

Groupes auxquels l'utilisateur appartient.
