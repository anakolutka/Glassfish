JDBCRealm クラスに固有のプロパティー

Web ページや Web アプリケーションを保護するために、登録ユーザーのみがそれらにアクセスできるようにセキュリティーを設定できます。これは認証機能として知られています。この種類のレルムでは、データベース内にユーザーの証明書を格納します。Enterprise Server は、データベース情報および設定ファイル内の有効化された JDBC レルムオプションを使用します。JDBC レルムに必要なプロパティーは次のとおりです。

JAAS context

このレルムを使用するログインモジュールの識別子。有効な値は jdbcRealm のみです。

J.N.D.I.

このレルムの J.N.D.I. 名。デフォルト値は jdbc/security です。

User Table

このレルムの承認ユーザーのリストを含むテーブル。デフォルト値は usertable です。

User Name Column

User Table 内の、ユーザーのリストを含む列の名前。デフォルト値は userid です。

Password Column

User Table 内の、各ユーザーのパスワードを含む列の名前。デフォルト値は password です。

Group Table

デフォルト値は grouptable です。

Group Name Column

デフォルト値は groupid です。
