Hosts de JMS

Utilize a página Hosts JMS para configurar hosts do serviço JMS.

Para cada aplicação, as informações a seguir são fornecidas.

Nome

O nome do host JMS.

Host

O nome ou o endereço do Protocolo Internet (IP) do sistema que o host JMS está sendo executado.

Porta

O número da porta do serviço JMS no host.

O quadro Hosts também contém as opções a seguir.

Novo

Botão utilizado para criar um novo host JMS.

Deletar

Botão utilizado para deletar um ou mais hosts JMS selecionados.
