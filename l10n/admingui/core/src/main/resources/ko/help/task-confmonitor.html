모니터링 서비스를 구성하는 방법

  1. 탐색 트리에서 구성 노드를 선택합니다.
  2. 구성 노드에서 모니터링 노드를 선택합니다.

    모니터링 서비스 페이지가 열립니다.

  3. HTTP 서비스 드롭다운 목록에서 낮음 또는 높음을 선택하여 HTTP 서비스 모니터링을 활성화합니다.

    이 릴리스에서는 낮음 및 높음 옵션에서 제공되는 모니터링 수준 간에 차이가 없습니다. HTTP 서비스 모니터링을 비활성화하려면 해제를 선택합니다. 기본값은 해제입니다.

  4. 컨테이너 드롭다운 목록에서 낮음 또는 높음을 선택하여 웹 컨테이너 모니터링을 활성화합니다.

    이 릴리스에서는 낮음 및 높음 옵션에서 제공되는 모니터링 수준 간에 차이가 없습니다. 웹 컨테이너 모니터링을 비활성화하려면 해제를 선택합니다. 기본값은 해제입니다.

  5. JVM 드롭다운 목록에서 낮음 또는 높음을 선택하여 JVM 모니터링을 활성화합니다.

    이 릴리스에서는 낮음 및 높음 옵션에서 제공되는 모니터링 수준 간에 차이가 없습니다. JVM 모니터링을 비활성화하려면 해제를 선택합니다. 기본값은 해제입니다.

  6. 스레드 풀 드롭다운 목록에서 낮음 또는 높음을 선택하여 스레드 풀 모니터링을 활성화합니다.

    이 릴리스에서는 낮음 및 높음 옵션에서 제공되는 모니터링 수준 간에 차이가 없습니다. 스레드 풀 모니터링을 비활성화하려면 해제를 선택합니다. 기본값은 해제입니다.

  7. 저장을 누릅니다.
참조

  • HTTP 서비스 모니터링 데이터를 보는 방법

  • 웹 컨테이너 모니터링 데이터를 보는 방법

  • JVM 모니터링 데이터를 보는 방법

  • 스레드 풀 모니터링 데이터를 보는 방법
