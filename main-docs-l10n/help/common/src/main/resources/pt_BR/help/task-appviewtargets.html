Para Exibir Alvos da Aplicação

  1. Na árvore de navegação, selecione o nó Aplicações.

    A página Aplicações será aberta.

  2. Selecione uma aplicação para a qual você deseja exibir os alvos.

    O nome da aplicação pode incluir um identificador de versão opcional, que segue o nome e é separado do nome por dois-pontos (:). Para obter mais informações sobre versões modulares ou da aplicação, consulte Module and Application Versions em Oracle GlassFish Server Application Deployment Guide.

    A página da aplicação se abre.

  3. Selecione a guia Alvo.

    A página Alvos da Aplicação se abre e os alvos atualmente especificados para a aplicação são exibidos.

  4. Para ativar ou desativar uma aplicação em uma instância ou cluster de alvo específico, clique na caixa de seleção ao lado do alvo e selecione Ativar ou Desativar no menu drop-down Ações.

  5. Para ativar o balanceamento de carga de um alvo em uma instância ou cluster do alvo específico, clique na caixa de seleção ao lado do alvo e selecione Ativar Balanceador de Carga ou Desativar Balanceador de Carga no menu drop-down Mais Ações.

  6. Para associar uma aplicação a servidores virtuais específicos, selecione Gerenciar Servidores Virtuais na instância ou cluster do alvo.

    A página Alvos do Servidor Virtual será aberta.

    Para adicionar servidores virtuais do alvo, selecione-os na lista Alvos Disponíveis e utilize o botão Adicionar para movê-los para a lista Alvos Selecionados.

    Por default, o módulo ou aplicação está associado ao servidor virtual default, com o nome server. Utilize a botão Adicionar Tudo para adicionar todos os alvos. Utilize os botões Remover e Remover Tudo para remover os alvos da lista Alvos Selecionados.

    Selecione Salvar para voltar à página Alvos da Aplicação.

  7. Para adicionar ou deletar instâncias ou clusters do alvo da aplicação, selecione Gerenciar Alvos.

    A página Gerenciar Alvos será aberta.

    Para adicionar alvos, selecione-os na lista Alvos Disponíveis e utilize o botão Adicionar para movê-los para a lista Alvos Selecionados.

    Utilize a botão Adicionar Tudo para adicionar todos os alvos. Utilize os botões Remover e Remover Tudo para remover os alvos da lista Alvos Selecionados.

    Selecione Salvar para voltar à página Alvos da Aplicação.

  8. Clique em Salvar.

    A aplicação está agora disponível na lista de alvos alterada.
