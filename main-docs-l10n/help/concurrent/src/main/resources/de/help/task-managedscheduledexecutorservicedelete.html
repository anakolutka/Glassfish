So wird ein verwalteter geplanter ExecutorService gelöscht

  1. Blenden Sie in der Navigationsbaumstruktur den Knoten "Ressourcen" ein.

  2. Blenden Sie unter dem Ressourcenknoten den Knoten "Nebenläufige Ressourcen" ein.

  3. Wählen Sie unter dem Knoten "Nebenläufige Ressourcen" den Knoten "Verwaltete geplante ExecutorServices".

    Die Seite "Verwaltete geplante ExecutorServices" wird geöffnet.

  4. Aktivieren Sie auf der Seite "Verwaltete geplante ExecutorServices" das Kontrollkästchen neben dem Namen der zu löschenden Ressource.

  5. Klicken Sie auf "Löschen".
