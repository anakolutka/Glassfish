To Disable a JBI Service Assembly

Before You Begin

***update the task id

  1. In the tree view, select the Integrations node and then the JBI Deployments node.

  2. From the list of JBI Service Assemblies, click the service assembly you want to disable.

  3. Click Disable. The Status column will display that the Service Assembly is enabled.
