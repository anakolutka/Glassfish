Configuration de l'Outil de mise à jour

La page Configuration contient les informations de configuration de l'Outil de mise à jour et permet de modifier le mode de connexion à Internet de l'&UpdateToolAdminConsoleNode. Pour modifier d'autres aspects de la configuration Outil de mise à jour, utilisez l'application updatetool autonome située dans as-install-parent/bin.

La page Configuration de l'Outil de mise à jour contient les options suivantes :

Chemin d'installation

Chemin du système de fichiers du répertoire racine ou du répertoire de niveau supérieur de l'image d'installation.

Référentiel préféré

Nom du référentiel réseau préféré qui est associé à cette image d'installation, par exemple glassfish.org.

Connexion Internet

Mode de connexion de GlassFish Server à Internet :

Connexion directe

Cette option signifie que GlassFish Server se connecte directement à Internet.

Via un proxy

Cette option signifie que GlassFish Server se connecte à Internet via un serveur proxy. Lorsque cette option est sélectionnée, les champs Hôte proxy et Port proxy sont ajoutés à la page.

Hôte proxy

Nom d'hôte du serveur proxy de l'ordinateur sur lequel GlassFish Server est exécuté.

Ce champ est applicable uniquement si l'ordinateur sur lequel GlassFish Server est exécuté se connecte à Internet via un serveur proxy.

Port proxy

Numéro de port du serveur proxy de l'ordinateur sur lequel GlassFish Server est exécuté.

Ce champ est applicable uniquement si l'ordinateur sur lequel GlassFish Server est exécuté se connecte à Internet via un serveur proxy.

Référentiels

Cette table présente les informations ci-dessous pour chaque référentiel réseau associé à cette image d'installation :

Référentiel

Nom du référentiel, par exemple, dev.glassfish.org .

Tâches connexes

Configuration des paramètres de proxy réseau permettant d'accéder à des référentiels réseau

Commandes asadmin connexes

Aucune.
