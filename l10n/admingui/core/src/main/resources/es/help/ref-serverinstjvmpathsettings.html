Configuración de ruta de clase JVM

La ruta de clase es la lista de archivos JAR en la que el entorno de ejecución de Java busca clases y otros recursos.

Ruta de clase de entorno

Si se activa la casilla de verificación Ignorar, se omitirá la variable de entorno CLASSPATH. La variable de entorno CLASSPATH es práctica para tutoriales básicos de programación, pero no se recomienda para entornos empresariales.

Ruta de clase de servidor

Campo de sólo lectura para ver la ruta de clase de Application Server.

Ruta de clase del sistema

La ruta a las clases del sistema.

Prefijo de ruta de clase

Rutas completas a los archivos JAR que se van a anteponer al prefijo de ruta de clase del servidor, una por línea.

Sufijo de ruta de clase

Rutas completas a los archivos JAR que se van a anexar al sufijo de ruta de clase del servidor, una por línea.

Prefijo de ruta de biblioteca nativa

Rutas completas a los archivos JAR que se van a anteponer a la ruta de biblioteca nativa, una por línea.

Sufijo de ruta de biblioteca nativa

Rutas completas a los archivos JAR que se van a anexar a la ruta de biblioteca nativa, una por línea.

La ruta de biblioteca nativa es una concatenación de la ruta relativa del servidor de sus bibliotecas compartidas nativas, la ruta de biblioteca nativa del entorno de ejecución estándar, el valor del entorno shell (LD_LIBRARY_PATH en sistemas UNIX® ) y cualquier ruta especificada en la página Configuración de analizador JVM.
