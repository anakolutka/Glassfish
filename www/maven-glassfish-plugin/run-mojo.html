glassfish:run

Executes Glassfish by the current module (and all the other modules needed for a particular distribution of GF.)

Mojo Attributes :

  • Requires a Maven 2.0 project to execute.
  • Executes as an aggregator plugin.
  • Requires dependency resolution of artifacts in scope: runtime
  • Automatically executes within the lifecycle phase: compile

Required Parameters

Name Type Description
distribution ArtifactInfo Distribution of Glassfish to be used as a basis.
localRepository ArtifactRepository No description.

Optional Parameters

Name Type Description
args String[] Command-line options to be passed to {@link StartupContext}.
artifactFactory ArtifactFactory No description.
artifactMetadataSource ArtifactMetadataSource No description.
artifactResolver ArtifactResolver No description.
finalName String The file name of the created distribution image. Default value is ${project.build.finalName}.zip.
outputDirectory File The directory where the final image will be created. Default value is ${project.build.directory}.
projectBuilder MavenProjectBuilder No description.
rootDir File The root directory of the launched module system. If unspecified, the base installation image from {@link #distribution} will be used.
session MavenSession No description.
