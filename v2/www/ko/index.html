GlassFish 커뮤니티는 무료, 오픈 소스, 양산 수준 품질의 기업용 소프트웨어를 빌드합니다. 커뮤니티의 주요 제공물로는 응용 프로그램 서버, Java EE 5 참조 구현 및 Java Persistence API 참조 구현 TopLink Essentials 등이 있습니다. 또한 Maven 저장소, 도구 등을 제공합니다.


시작하기 다운로드 통신 기여


GlassFish용 오픈 소스 SIP 서블릿 기술인 Project SailFin
VoIP(Voice-over-IP) 전화 서비스, 인스턴트 메시징, 상태 확인, 친구 목록 관리 및 웹 회의 등과 같이 오늘날 우리가 향유하고 있는 주요 서비스들을 가능하게 하는 비결은 SIP(Session Initiation Protocol)입니다. SailFin 프로젝트는 GlassFish 응용 프로그램 서버에 SIP 서블릿 기술을 추가합니다. 자세한 내용은 https://sailfin.dev.java.net을 참조하십시오.

  
  
시작하기
GlassFish를 사용하여 최신 Java EE 5 기술을 학습하고 응용 프로그램 서버를 사용하여 응용 프로그램 개발을 시작합니 다. GlassFish 샘플 및 Blueprints 샘플을 사용하여 Java EE 5를 빠르게 실습할 수 있습니다. 기 존 J2EE 응용프로그램을 Java EE 5로 마이그레이션하는 방법을 살펴보십시오.

배워 보기
GlassFish 서버를 향상시키거나 GlassFish 커뮤니티에서 Grizzly, EJB 3.0 및 TopLink Essentials, 관리 도구와 같은 기술 및 기능을 구현하는 방법을 배워 보십시오.
