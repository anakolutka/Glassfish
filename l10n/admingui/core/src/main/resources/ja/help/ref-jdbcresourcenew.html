新しい JDBC リソース

JNDI 名

JNDI 名によって、分散コンピューティング環境に配置されたコンポーネントの編成と場所の管理を行います。この方法は図書館の蔵書の分類と保管場所の管理に用いるカード式の目録と似ています。そのため、JNDI 名は JDBC リソースにアクセスする際の重要な要素になります。規則により、名前は jdbc/ 文字列で始まります。たとえば、jdbc/payrolldb などです。スラッシュを忘れないでください。

プール名

JDBC リソースに関連付けられた接続プール。

説明

JDBC リソースの簡単な説明。

ステータス

JDBC リソースの状態。リソースは、有効または無効の状態になります。
