モジュールログレベル

「モジュールログレベル」ページを使用して、モジュールごとのログレベルを設定します。

「モジュールログレベル」ページには、モジュールごとに次の情報が表示されます。

ロガー名

モジュールのロガーの名前 (例: javax.enterprise.system.tools.admin)。

ロガーレベル

モジュールの現在のログレベル。

次のログレベルを指定できます。ログレベルの高い順に示しています。

SEVERE

通常のプログラム実行に干渉するイベント。

WARNING

警告。例外を含みます。

INFO

サーバー設定またはサーバーの状態に関連するメッセージ。ただし、エラーは除きます。

CONFIG

サーバー設定に関するメッセージ。

FINE

最小の詳細レベル。

FINER

中間の詳細レベル。

FINEST

最大の詳細レベル。

オフ

ログメッセージを出力しません。

関連操作

  • 「ログの一般設定を行う」

  • 「ログレベルを設定する」

  • 「サーバーログを表示する」

関連する asadmin コマンド

  • set-log-level
