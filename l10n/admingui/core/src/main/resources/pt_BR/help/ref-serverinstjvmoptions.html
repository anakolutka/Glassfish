Opções de JVM

Use a página Opções de JVM para especificar as opções do lançador de aplicativos Java (java) que executa o Enterprise Server. As opções -D designam as propriedades específicas do Enterprise Server.

  • Para modificar uma opção, edite o campo Valor dessa opção.

  • Para adicionar uma opção, clique no botão Adicionar Opção de JVM. Na linha em branco que é exibida, digite as informações da opção do campo Valor.

  • Para excluir uma opção, marque a caixa de seleção à esquerda do campo Valor (referente à opção a ser excluída) e clique em Excluir.
