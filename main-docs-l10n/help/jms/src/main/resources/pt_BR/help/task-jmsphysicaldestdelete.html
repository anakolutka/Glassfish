Para Deletar um Destino Físico JMS

Não é recomendável deletar o destino do sistema, mq.sys.dmq.

  1. Na árvore de navegação, expanda o nó Configuração.

  2. No nó Configuração, expanda o nó do Serviço JMS.

  3. No nó Serviço JMS, selecione o nó Destinos Físicos.

    A página Destinos Físicos é aberta.

  4. Na página Destinos Físicos, marque a caixa de seleção ao lado nome do destino que está sendo deletado.

  5. Clique em Deletar.
