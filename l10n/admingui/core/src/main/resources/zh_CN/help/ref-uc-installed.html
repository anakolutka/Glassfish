安装的组件

“安装的组件”页面提供了有关已安装的安装映像组件的信息。

“安装的组件”页面包含以下选项：

安装路径

安装映像的根目录或顶层目录的文件系统路径。

删除

此按钮用于删除所选组件。

Installed Components

此表提供了有关安装的每个组件的以下信息：

组件

组件名称。

类别

与组件相关联的软件类别，例如 "Application Server"、“系统工具”和“数据库”。类别是在创建组件时由组件作者分配到组件中的。

版本

提供组件的软件包的版本号。

安装大小

安装组件所消耗的磁盘空间大小。

源

对于已安装的组件，指从中获取该组件的网络系统信息库的名称。

对于附加组件（或更新），指从中获取该组件的网络系统信息库的名称。

相关任务

卸载组件
