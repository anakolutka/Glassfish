Using the Help System

When you click the help link, a new browser window opens containing help for the Administration Console. The text displayed in the right frame of the help window describes the functionality of the console screen you are using and contains links to related tasks. Use the left frame of this window to navigate to other help topics using the Table of Contents, the alphabetical Index, or the Search function. Use the > (back) button or < (forward) button to step through previously viewed pages. Use the Print button to print the current topic.

Note - Some topics in the Admin Console Online Help pertain to features that are available only in domains that are configured to support clusters. Examples of domains that support clusters are domains that are created with the cluster profile or the enterprise profile. If your domain is not configured to support clustering, and you require to use clustering features of Application Server, add cluster support. For more information, see To Add Cluster Support to a Domain.
