JDBC 자원을 편집하는 방법

  1. 탐색 트리에서 자원 노드를 확장합니다.
  2. JDBC 자원 노드를 선택합니다.
  3. JDBC 자원 페이지에서 편집할 자원의 이름을 누릅니다.

    JDBC 자원 편집 페이지가 열립니다.

    주 – J.N.D.I. 이름은 읽기 전용 필드입니다. 새 JDBC 자원을 만들 때만 J.N.D.I 이름을 지정할 수 있습니다.

  4. 풀 이름 드롭다운 목록에서 이 JDBC 자원과 연결할 연결 풀을 선택합니다.
  5. 설명 필드에 이 JDBC 자원에 대한 간단한 설명을 입력합니다.
  6. 저장을 누릅니다.
