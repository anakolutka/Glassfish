Per raccogliere e scaricare i log del dominio

È possibile raccogliere tutti i file di log disponibili per un'istanza o un cluster di GlassFish Server e compilarli in un unico archivio ZIP.

Nota:

Se si specifica un cluster, l'archivio ZIP includerà anche il file di log per il server di amministrazione del dominio (DAS).

  1. Selezionare il nodo Dominio nell'albero di navigazione.

    Viene visualizzata la pagina Attributi dominio.

  2. Selezionare la scheda Log di dominio.

    Viene visualizzata la pagina Log di dominio.

    Se i cluster esistono, vengono visualizzati gli elenchi a discesa per i cluster e per le istanze. Se i cluster non esistono, viene visualizzato solo l'elenco a discesa per le istanze.

  3. Selezionare Cluster o Istanza.

  4. Selezionare il cluster o l'istanza per cui si stanno raccogliendo e scaricando i file di log.

  5. Fare clic su Raccogli log.

    Il meccanismo di download del browser richiede di scaricare il file log-files-nome.zip, dove nome è il nome del cluster o dell'istanza.
