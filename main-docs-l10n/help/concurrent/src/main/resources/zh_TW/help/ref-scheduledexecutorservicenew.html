新建受管理排定的執行程式服務

您可以使用「新建受管理排定的執行程式服務」頁面來建立排定的受管理執行程式服務資源。

「新建受管理排定的執行程式服務」頁面包含下列選項。

JNDI 名稱

可識別資源的唯一名稱。

環境資訊

資源的環境：類別載入器、JNDI、安全性或工作區。

狀態

資源狀態。您可啟用或停用資源。預設值為「啟用」。

繫線優先順序

要為受管理排定的執行程式服務執行處理建立之繫線的優先順序。預設值為 5。

長時間執行的作業

XXX.

停滯前的時間

作業在被視為停滯之前可執行的時間 (以秒為單位)。

核心大小

要保留在繫線集區中的繫線數目。預設值為 0。

集區的最大大小

集區可以包含的繫線數目上限。預設值為 2147483647。

持續作用時間

當繫線數目大於核心數目時，繫線可維持閒置的時間 (以秒為單位)。預設值為 60。

繫線存留時間

清除繫線之前繫線可保留在繫線集區中的時間 (以秒為單位)，無論繫線數目是否大於核心大小，或繫線是否為閒置狀態。

作業佇列容量

保留等待執行之已提交作業的佇列大小。預設值為 2147483647。

描述

此資源的描述。

其他特性

資源的其他特性。GlassFish Server 未定義排定的受管理執行程式服務資源的任何其他特性。
