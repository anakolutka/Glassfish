Editar Recurso Personalizado

Utilize a página Editar Recurso Personalizado para editar um recurso JNDI personalizado.

A página Editar Recurso Personalizado contém as opções a seguir.

Nome JNDI

Um nome exclusivo que identifica o recurso. O nome JNDI é um campo somente para leitura. Você pode especificar o nome JNDI ao criar um novo recurso personalizado.

Tipo de Recurso

A definição de tipo totalmente qualificado para o recurso.

Classe de Fábrica

O nome especificado pelo usuário para a classe de fábrica. Essa classe implementa a interface javax.naming.spi.ObjectFactory.

Descrição

Uma descrição do recurso personalizado.

Status

O status do recurso personalizado. O recurso personalizado pode ser ativado ou desativado. O valor default é Ativado.

Propriedades Adicionais

Propriedades adicionais para o recurso personalizado. O GlassFish Server não define quaisquer propriedades adicionais para recursos personalizados.
