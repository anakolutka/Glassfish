管理器属性

会话管理器提供了用于配置会话的创建和销毁方式、存储会话状态的位置以及可用会话的最大数量的方法。

使用“管理器属性”页面可以添加或修改会话管理器属性。“管理器属性”页面包含以下选项 ：

清除时间间隔

从库中删除非活动会话数据之前经过的秒数。

最大会话数

允许的最大会话数。

会话文件名

包含会话数据的文件名。

会话 ID 生成器类名

在 Enterprise Server 类路径中指定的 ID 生成器类名。

其他属性

会话管理器的其他属性。

  • 要添加属性，请单击“添加属性”按钮。在显示的空白行中，在“名称”字段中键入属性名称，在“值”字段中键入属性值。单击“保存”。

  • 要修改属性，请编辑该属性的“值”字段。单击“保存”。

  • 要删除属性，请选中位于您要删除属性的“名称”字段左边的复选框，然后单击“删除属性”按钮。单击“保存”。
