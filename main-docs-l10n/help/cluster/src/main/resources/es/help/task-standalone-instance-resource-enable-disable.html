Activar o desactivar un recurso para una instancia de GlassFish Server independiente

  1. En el árbol de navegación, seleccione el nodo Instancias Independientes.

    Se abrirá la página Instancias de Servidor Independientes.

  2. En la página Instancias de Servidor Independientes, seleccione la instancia.

    Se abrirá la página Información General para la instancia.

  3. En la página Información General, haga clic en el separador Recursos.

    Se abrirá la página Recursos para la instancia.

  4. Active la casilla de control que aparece junto al recurso.

  5. Haga clic en el botón correspondiente.

    • Para activar el recurso, seleccione Activar.

    • Para desactivar el recurso, seleccione Desactivar.

      Al desactivar un recurso no se elimina.
