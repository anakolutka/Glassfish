更新工具密钥概念

更新工具管理 Enterprise Server 软件组件（包含在您系统上的单个文件系统目录中）的安装。

在下面的主题中描述了更新工具的密钥概念。

组件

组件是由更新工具管理的可安装单元。组件由安装映像中作为一个单元安装到您系统上的各个文件组成。

安装映像

安装映像是包含软件组件的目录，其中包含目录内容。作为下载并安装 Enterprise Server 软件的结果，安装映像将添加到更新工具。为了启用 Enterprise Server 软件组件的添加和更新，每个安装映像都要包含有关安装组件的信息。

网络系统信息库

网络系统信息库包含以下项：

  • 可以添加到安装映像的组件

  • 对安装映像中现有组件的更新

安装映像与一个或多个网络系统信息库相关联。每个网络系统信息库都由名称以及关联的 URL 表示。每个安装映像的系统信息库名称都是唯一的，并且代表管理系统信息库和已定义完备的内容范围的组织或项目。

附加组件

附加组件 是尚未安装的安装映像的组件。
