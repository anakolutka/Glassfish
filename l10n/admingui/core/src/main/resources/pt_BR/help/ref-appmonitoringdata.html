Monitoramento do Aplicativo

Usar a página Monitoramento do Aplicativo para exibir os dados de monitoramento de um aplicativo selecionado. Para uma descrição das propriedades monitoradas, consulte o Capítulo 8, Administering the Monitoring Service, no Sun GlassFish Enterprise Server v3 Administration Guide.

  • Para exibir dados de monitoramento do aplicativo

  • Para exibir dados de monitoramento do servidor
