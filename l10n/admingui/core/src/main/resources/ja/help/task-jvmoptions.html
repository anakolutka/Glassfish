JVM オプションを設定する

「JVM オプション」ページを使用して、Enterprise Server を実行する Java アプリケーション起動ツール (java) のオプションを指定します。-D オプションは、Enterprise Server に固有のプロパティーを指定します。

  1. ナビゲーションツリーで、「Application Server」ノードを選択します。
  2. 「一般情報」ページで、「JVM 設定」タブをクリックします。
  3. 「JVM 一般設定」ページで、「JVM オプション」タブをクリックします。

    「JVM オプション」ページが開きます。

  4. オプションを変更するには、そのオプションの「値」フィールドを編集します。
  5. オプションを追加するには、次の手順を実行します。
    1. 「JVM オプションを追加」ボタンをクリックします。
    2. 表示される空白行に、「値」フィールドのオプション情報を入力します。
  6. オプションを削除するには、次の手順を実行します。
    1. 削除するオプションの「値」フィールドの左側にあるチェックボックスを選択します。
    2. 「削除」をクリックします。
  7. 「保存」をクリックします。
  8. サーバーを再起動します。
