Activer l'authentification mutuelle pour toutes les applications

Le Enterprise Server utilise le domaine certificat pour l'authentification HTTPS.

  1. Dans l'arborescence de navigation, développez le nœud Configuration.
  2. Sous le nœud Configuration, développez le nœud Sécurité.
  3. Sous le nœud Securité, développez le nœud Domaines.
  4. Sous le nœud Domaines, sélectionnez le domaine certificat.

    La page Modification du domaine s'ouvre.

  5. Dans le tableau Autres propriétés de la page Modification du domaine, cliquez sur le bouton Ajouter propriété.
    1. Dans le champ Nom, saisissez clientAuth.
    2. Dans le champ Valeur, saisissez true.
  6. Cliquez sur Enregistrer.
  7. Redémarrez le Enterprise Server si « Redémarrage requis » s'affiche dans la console.

    Une fois le serveur redémarré, l'authentification client est requise pour toutes les applications utilisant le domaine certificat.
