檔案使用者

透過管理主控台管理 file 範圍使用者。金鑰檔案中會列出 file 範圍中的使用者和群組，金鑰檔案的位置則由 file 特性指定。

備註 – 也可以使用這些步驟將使用者增加到任何 file 範圍，包括 admin-realm。僅需以目標範圍的名稱替代本節提到的 file 範圍。

file 範圍中的使用者可以屬於 Java 群組，該群組是依照一般特性分類的使用者種類。例如，電子商務應用程式的用戶可能屬於 CUSTOMER 群組，但是大用戶可以屬於 PREFERRED 群組。將使用者進行群組分類可以簡化有大量使用者時的存取控制。

剛安裝完 Enterprise Server 之後，唯一的使用者是在安裝期間輸入的管理員。依預設，該使用者屬於 admin-realm 範圍中的 asadmin 群組，該群組有權修改 Enterprise Server。任何指定到該群組的使用者都將具有管理特權，亦即可以存取 asadmin 工具和管理主控台。
