고급 도메인 속성을 설정하는 방법

  1. 탐색 트리에서 Application Server 노드를 선택합니다.

    일반 정보 페이지가 열립니다.

  2. 일반 정보 페이지에서 고급 탭을 누릅니다.

    응용 프로그램 구성 페이지가 열립니다.

  3. 응용 프로그램 구성 페이지에서 도메인 속성 탭을 누릅니다.

    도메인 속성 페이지가 열립니다.

  4. 응용 프로그램 루트 필드에 배포된 응용 프로그램이 있는 디렉토리의 전체 경로를 입력합니다.

    기본값은 domain-dir/applications입니다.

  5. 로그 루트 필드에 서버 로그 파일을 저장할 디렉토리의 전체 경로를 입력합니다.

    기본값은 domain-dir/logs입니다.

  6. 호스트의 기본 로켈을 사용하려면 로켈 필드를 비워 둡니다.

    로켈은 언어 및 지역의 특정 조합을 지정하는 식별자입니다. 예를 들어 미국 영어의 로켈은 en_US이고 일본어의 로켈은 ja_JP입니다. 비영어 로켈을 사용하려면 Enterprise Server가 현지화(영어를 다른 언어로 번역하는 작업이 포함된 프로세스)되어야 합니다.

  7. 저장을 누릅니다.
  8. 서버를 다시 시작합니다.
참조

  • Enterprise Server에 대한 일반 정보를 보는 방법

  • 서버 로그를 보는 방법

  • 고급 응용 프로그램 구성을 설정하는 방법

  • 관리자 비밀번호를 변경하는 방법

  • 시스템 등록 정보를 관리하는 방법

  • 응용 프로그램 모니터링 데이터를 보는 방법

  • 서버 모니터링 데이터를 보는 방법
