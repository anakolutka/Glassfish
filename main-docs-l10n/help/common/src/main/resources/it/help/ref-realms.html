Realm

GlassFish Server è preconfigurato con tre realm: file, certificate e admin-realm. È inoltre possibile creare i realm ldap, JDBC, solaris e realm personalizzati. In genere, in un server è disponibile un realm di ciascun tipo. Tuttavia, in GlassFish Server sono presenti due realm file: file e admin-realm. Si tratta di due realm dello stesso tipo utilizzati per due scopi differenti. È inoltre possibile disporre di un database dei certificati distinto per ciascun server virtuale del sistema.

Utilizzare la pagina Realm per configurare i realm.

Per ogni realm vengono fornite le informazioni riportate di seguito.

Nome

Il nome del realm.

Nome classe

Il nome di classe per il realm.

La tabella Realm contiene inoltre le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare un nuovo realm.

Elimina

Pulsante che consente di eliminare uno o più realm selezionati.
