Pasos de ejecución de trabajo por lotes

En la página Pasos de ejecución de trabajo por lotes se muestran detalles sobre los pasos de la ejecución de un determinado trabajo por lotes. Aparece la siguiente información.

Nombre de la instancia
Nombre del cluster

Nombre de la instancia de servidor o cluster de GlassFish al que se aplica la configuración.

Nombre del trabajo

Nombre del trabajo por lotes.

ID de ejecución

ID asignado para la ejecución del trabajo por lotes. La primera vez que se inicia un trabajo y cada vez que se reinicia la ejecución existente, se crea una nueva ejecución.

Pasos del trabajo

Pasos de la ejecución de un determinado trabajo por lotes. Para cada paso, se muestra la siguiente información:

Nombre del paso

Nombre del paso.

Estado de lote

Estado del paso tal y como lo ha definido el tiempo de ejecución por lotes.

Estado de salida

Estado del paso tal y como lo ha definido el XML de trabajo para el trabajo o la aplicación por lotes. Por defecto, el estado de salida y el estado de lotes son el mismo a menos que el estado de salida se sustituya de forma explícita.

Hora de inicio

Hora a la que se inicia el paso.

Hora de finalización

Hora a la que termina el paso.

Métricas de paso

Métricas del paso, que muestran el número de elementos leídos, escritos, confirmados, etc.
