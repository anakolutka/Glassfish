Next Steps After Creating a Node Agent Placeholder

After creating a node agent placeholder, create and start the node agent. To create and start a node agent, use the Application Server CLI. For more information, see:

  • Creating a Node Agent in Sun Java System Application Server 9.1 High Availability Administration Guide

  • Starting a Node Agent in Sun Java System Application Server 9.1 High Availability Administration Guide

You can start the node agent's server instances only after the node agent is created and started.

To return to the Node Agents page, click OK.
