启用对所有应用程序的双向验证

Enterprise Server 使用 certificate 领域进行 HTTPS 验证。

  1. 在导航树中，展开“配置”节点。
  2. 在“配置”节点下，展开“安全性”节点。
  3. 在“安全性”节点下，展开“领域”节点。
  4. 在“领域”节点下，选择 certificate 领域。

    将打开“编辑领域”页面。

  5. 在“编辑领域”页面的“其他属性”表中，单击“添加属性”按钮。
    1. 在“名称”字段中，键入 clientAuth。
    2. 在“值”字段中，键入 true。
  6. 单击“保存”。
  7. 如果控制台中显示“必须重新启动”，请重新启动 Enterprise Server。

    重新启动服务器之后，需要对使用 certificate 领域的所有应用程序进行客户机验证。
