Installer des composants add-ons

L'installation d'un composant add-on comprend les opérations suivantes :

  • Transférer le contenu du composant et de ses dépendances depuis un référentiel de réseau

  • Installer le composant sous l'Enterprise Server image d'installation dans votre système

Si un composant nécessite un autre composant qui n'est pas déjà installé, ce dernier est également installé.

  1. Dans l'arborescence de navigation, sélectionnez le nœud Outil de mise à jour.

    La page Composants installés s'affiche.

  2. Dans la page Composants installés, cliquez sur l'onglet Add-ons disponibles.

    La page Add-ons disponibles s'affiche.

  3. Dans le tableau des composants add-ons disponibles, sélectionnez les composants à installer.

    Si aucun composant add-on n'est disponible, le tableau est vide.

    • Pour sélectionner un composant, cochez la case située en regard de son nom.
    • Pour sélectionner tous les composants, cliquez sur l'icône Sélectionner tout dans l'en-tête du tableau.
    • Pour désélectionner tous les composants, cliquez sur l'icône Désélectionner tout dans l'en-tête du tableau.
  4. Cliquez sur Installer.

    Une boîte de dialogue vous demande de confirmer que les composants sélectionnés, ainsi que tout autre composant requis, doivent être installés.

  5. Dans la boîte de dialogue, cliquez sur OK.

    On vous demande d'accepter les contrats de licence des composants que vous installez.

  6. Acceptez les contrats de licence des composants que vous installez.

    Enterprise Server confirme que l'installation est terminée. Les composants sont supprimés du tableau des composants add-ons et sont ajoutés au tableau des composants installés dans la page Composants installés.

  7. Redémarrez le serveur d'administration du domaine.

    Pour consulter des instructions détaillées, reportez-vous à la documentation ci-dessous:

    • To Stop a Domain (or Server) du Sun GlassFish Enterprise Server v3 Prelude Administration Guide

    • To Start a Domain (or Server) du Sun GlassFish Enterprise Server v3 Prelude Administration Guide
