Per modificare una risorsa JNDI personalizzata

  1. Espandere il nodo Risorse nell'albero di navigazione.

  2. Nel nodo Risorse espandere il nodo JNDI.

  3. Nel nodo JNDI selezionare Risorse personalizzate.

    Viene visualizzata la pagina Risorse personalizzate.

  4. Nella pagina Risorse personalizzate fare clic sul nome della risorsa da modificare.

    Viene visualizzata la pagina Modifica risorsa personalizzata.

    Nota:

    Il campo Nome JNDI è di sola lettura. È possibile specificare il nome JNDI solo quando si crea una nuova risorsa personalizzata.

  5. Per modificare il tipo di risorsa effettuare una delle operazioni riportate di seguito.

    • Selezionare una definizione di tipo dall'elenco a discesa.

    • Selezionare il secondo pulsante di scelta e digitare una definizione di tipo completamente qualificata nel formato . xxx.

  6. Nel campo Classe factory digitare un nome di classe factory per la risorsa personalizzata.

    Il valore Classe factory è il nome specificato dall'utente per la classe factory. Questa classe implementa l'interfaccia javax.naming.spi.ObjectFactory.

  7. Nel campo Ordine di distribuzione digitare un numero per specificare l'ordine di distribuzione della risorsa personalizzata.

    Le risorse con numero inferiore vengono caricate per prime all'avvio del server. Il valore predefinito è 100.

  8. Nel campo Descrizione digitare una breve descrizione della risorsa.

    La descrizione è un valore stringa con lunghezza limitata a 250 caratteri.

  9. Selezionare la casella di controllo Abilitato dell'opzione Stato per abilitare la risorsa.

    Per impostazione predefinita la risorsa è abilitata, ovvero disponibile, non appena creata.

  10. Nella sezione Proprietà aggiuntive specificare le proprietà aggiuntive per la risorsa personalizzata.

    GlassFish Server non definisce alcuna proprietà aggiuntiva per le risorse personalizzate.

    • Per aggiungere una proprietà, fare clic sul pulsante Aggiungi proprietà. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Nome e il valore della proprietà nel campo Valore.

    • Per modificare una proprietà, è necessario modificare il valore del campo Valore della proprietà.

    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Nome corrispondente alla proprietà da eliminare, quindi fare clic sul pulsante Elimina proprietà.

  11. Fare clic su Salva.
