配置監視服務

  1. 在瀏覽樹狀結構中，展開 [配置] 節點。
  2. 在 [配置] 節點下，選取 [監視] 節點。

    [監視服務] 頁面會隨即開啟。

  3. 從 [HTTP 服務] 下拉式清單中，選取 [LOW] 或 [HIGH] 以啟用 HTTP 服務監視。

    在此發行版本中，[低] 與 [高] 選項所提供的監視層級並無不同。若要停用 HTTP 服務監視，請選取 [OFF]。預設值為 [OFF]。

  4. 從 [Web 容器] 下拉式清單中，選取 [LOW] 或 [HIGH] 以啟用 Web 容器監視。

    在此發行版本中，[低] 與 [高] 選項所提供的監視層級並無不同。若要停用 Web 容器監視，請選取 [OFF]。預設值為 [OFF]。

  5. 從 [JVM] 下拉式清單中，選取 [LOW] 或 [HIGH] 以啟用 JVM 監視。

    在此發行版本中，[低] 與 [高] 選項所提供的監視層級並無不同。若要停用 JVM 監視，請選取 [OFF]。預設值為 [OFF]。

  6. 從 [執行緒池] 下拉式清單中，選取 [LOW] 或 [HIGH] 以啟用執行緒池監視。

    在此發行版本中，[低] 與 [高] 選項所提供的監視層級並無不同。若要停用執行緒池監視，請選取 [OFF]。預設值為 [OFF]。

  7. 按一下 [儲存]。
