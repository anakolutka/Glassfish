Protocole HTTP

La page Protocole HTTP contient les options suivantes :

Version

Version du protocole HTTP utilisé (HTTP/1.0 ou HTTP/1.1). La valeur par défaut est HTTP/1.1.

Recherche DNS activée

Si cette option est sélectionnée, le recherche de l'entrée DNS pour le client est activée. Par défaut, cette option n'est pas sélectionnée.

Protocole SSL activé

Si cette option est sélectionnée, le protocole SSL est utilisé pour tout listener dont la sécurité est activée. Si cette option n'est pas sélectionnée, la sécurité est désactivée sur l'ensemble du serveur. Cette option est sélectionnée par défaut.

Type forcé

Type de réponse utilisé si aucun mappage MIME correspondant à l'extension n'est disponible. La valeur par défaut est text/html; charset=iso-8859-1.

Type par défaut

La valeur par défaut est text/html; charset=iso-8859-1. La valeur est une chaîne de valeurs séparées par des points-virgules et indiquant le type de contenu , le codage, la langue et le jeu de caractères.
