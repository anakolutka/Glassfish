관리 대상 실행기 서비스를 편집하는 방법

  1. 탐색 트리에서 [리소스] 노드를 확장합니다.

  2. [리소스] 노드에서 [동시 리소스] 노드를 확장합니다.

  3. [동시 리소스] 노드에서 [관리 대상 실행기 서비스]를 선택합니다.

    [관리 대상 실행기 서비스] 페이지가 열립니다.

  4. [관리 대상 실행기 서비스] 페이지에서 편집할 리소스의 이름을 누릅니다.

    [관리 대상 실행기 서비스 편집] 페이지가 열립니다.

    주:

    [JNDI 이름]은 읽기 전용 필드입니다. 새 리소스를 생성할 때만 JNDI 이름을 지정할 수 있습니다. [논리적 JNDI 이름]도 읽기 전용 필드입니다.

  5. 컨텍스트를 스레드에 전달하려면 [컨텍스트 정보 사용] 체크 박스를 선택하고 전달할 컨텍스트를 선택합니다.

    기본적으로 이 옵션은 사용으로 설정되고 모든 컨텍스트가 선택되어 있습니다. 컨텍스트를 여러 개 선택하려면 Shift 또는 Ctrl 키를 누른 채로 누르십시오. 이 옵션을 사용 안함으로 설정하면 컨텍스트가 선택된 경우에도 전달되지 않습니다.

  6. 리소스를 사용으로 설정하려면 [상태 사용] 체크 박스를 선택합니다.

    이 옵션은 기본적으로 사용으로 설정되어 있습니다.

  7. [스레드 우선순위] 필드에 생성된 스레드에 지정할 우선순위를 입력합니다.

    기본값은 5입니다.

  8. 리소스를 오랫동안 실행 중인 작업에 사용하려면 [오랫동안 실행 중인 작업 사용] 체크 박스를 선택합니다.

    사용으로 설정되면 오랫동안 실행 중인 작업이 정지된 것으로 보고되지 않습니다. 이 옵션은 기본적으로 사용 안함으로 설정되어 있습니다.

  9. [작업 정지 간주 시간] 필드에 응답이 없는 것으로 간주되기 전에 작업을 실행할 수 있는 시간(초)을 입력합니다.

    기본값은 0으로, 이 경우 작업이 응답이 없는 것으로 간주되지 않습니다.

  10. [배치 순서] 필드에 리소스의 배치 순서를 지정하는 숫자를 입력합니다.

    서버 시작 시 낮은 번호의 리소스가 먼저 로드됩니다. 기본값은 100입니다.

  11. [설명] 필드에 리소스에 대한 간단한 설명을 입력합니다.

    이 설명은 최대 250자가 포함될 수 있는 문자열 값입니다.

  12. [코어 크기] 필드에 스레드의 유휴 상태 여부에 관계없이 스레드 풀에 유지할 스레드 수를 입력합니다.

    새 작업이 제출되고 실행 중인 스레드 수가 코어 크기보다 적으면 새 스레드가 생성되어 요청을 처리합니다. 기본값은 0으로, 이 경우 작업 대기열이 꽉 찼거나 리소스가 직접 대기열 지정을 사용하지 않는 한 새 스레드가 생성되지 않습니다. 직접 대기열 지정은 작업 대기열 용량이 0이거나, 작업 대기열 용량이 2147483647이고 코어 크기가 0일 때 발생합니다.

  13. [최대 풀 크기] 필드에 스레드 풀이 포함할 수 있는 최대 스레드 수를 입력합니다.

    기본값은 2147483647로, 이 경우 스레드 풀에 본질적으로 제한이 없으므로 개수에 관계없이 스레드를 포함할 수 있습니다.

  14. [연결 유지] 필드에 스레드 수가 코어 크기보다 큰 경우 스레드가 유휴 상태로 남아 있을 수 있는 시간(초)을 입력합니다.

    기본값은 60입니다.

  15. [스레드 수명] 필드에 스레드 수가 코어 크기보다 큰지 여부 또는 스레드가 유휴 상태인지 여부에 관계없이 비우기 전에 스레드가 스레드 풀에 남아 있을 수 있는 시간(초)을 입력합니다.

    기본값은 0으로, 이 경우 스레드를 비우지 않습니다.

  16. [작업 대기열 용량] 필드에 실행 대기 상태로 작업 대기열에 저장할 수 있는 제출된 작업 수를 입력합니다.

    기본값은 2147483647로, 이 경우 작업 대기열에 본질적으로 제한이 없으므로 개수에 관계없이 제출된 작업을 저장할 수 있습니다.

  17. [추가 속성] 섹션에서 속성을 추가하지 마십시오.

    GlassFish Server에서는 이 리소스 유형에 대한 추가 속성을 정의하지 않습니다. 또한 현재 이 리소스 유형은 추가 속성을 사용하지 않습니다.

  18. [저장]을 누릅니다.
