Agregar Recursos

Utilice la página Agregar Recursos para crear uno o más recursos desde un archivo glassfish-resources.xml en el destino especificado.

La página Agregar Recursos contiene la información siguiente.

Ubicación

Ubicación del archivo glassfish-resources.xml:

Archivo XML que Se Va a Cargar en el Servidor

Ruta de acceso al archivo en una máquina remota.

Archivo XML local al que Se Puede Acceder desde GlassFish Server

Ruta de acceso al archivo en la máquina del servidor de administración de dominio (DAS).

Destino

Lista desplegable de clusters y de instancias de servidor independientes.

Tareas relacionadas

Agregar recursos configurados en un archivo XML

Comandos asadmin Relacionados
