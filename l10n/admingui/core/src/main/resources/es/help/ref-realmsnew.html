Nuevo dominio

Utilice la página Nuevo directorio para crear un directorio.

La página Nuevo dominio contiene las siguientes opciones.

Nombre

Nombre del dominio nuevo.

Nombre de clase

Nombre de clase del dominio nuevo.

En la tabla siguiente se muestran las opciones válidas.

Nombre de dominio

Nombre de clase

file

com.sun.enterprise.security.auth.realm.file.FileRealm

certificate

com.sun.enterprise.security.auth.realm.certificate.CertificateRealm

ldap

com.sun.enterprise.security.auth.realm.ldap.LDAPRealm

JDBC

com.sun.enterprise.security.auth.realm.jdbc.JDBCRealm

solaris

com.sun.enterprise.security.auth.realm.solaris.SolarisRealm

Para especificar un dominio personalizado, seleccione el botón de opción inferior y el tipo de nombre de clase para crear el dominio personalizado.

Propiedades adicionales

Propiedades adicionales del dominio. Las propiedades válidas dependen del tipo de dominio.

  • Para obtener una descripción de las propiedades del dominio de archivo, consulte Propiedades específicas de la clase FileRealm.

  • Para obtener una descripción de las propiedades del dominio de certificado, consulte Propiedades específicas de la clase CertificateRealm .

  • Para obtener una descripción de las propiedades del dominio JDBC, consulte Propiedades específicas de la clase JDBCRealm .

  • Para obtener una descripción de las propiedades del dominio LDAP, consulte Propiedades específicas de la clase LDAPRealm .

  • Para obtener una descripción de las propiedades del dominio Solaris, consulte Propiedades específicas de la clase SolarisRealm .
