应用程序

“应用程序”页面显示在 Enterprise Server 上部署的应用程序列表。您可以查看和管理已部署的应用程序，并且可以部署更多的应用程序。

对于每个应用程序，均提供以下信息：

名称

应用程序名称。

已启用

如果应用程序已启用，则显示复选标记；如果应用程序已禁用，则显示 X。

复合

如果应用程序有多种组件，则为 True；如果应用程序只有一种组件，则为 False。

引擎

应用程序使用的容器类型。容器类型可为以下任意一种：

  • web

  • webservices

  • ejb

  • connector

  • appclient

  • jruby

操作

部署组件以后，可对组件执行的操作（对于所有组件，为“重新部署”和“重新启动”；对于 Web 应用程序，为“启动”）的链接。

部署

用于部署企业应用程序的按钮。

取消部署

用于取消部署企业应用程序的按钮。

启用

用于启用选定企业应用程序的按钮。

禁用

用于禁用选定企业应用程序的按钮。

过滤

按引擎过滤应用程序的下拉式列表。
