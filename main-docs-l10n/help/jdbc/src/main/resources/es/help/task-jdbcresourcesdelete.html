Para suprimir un recurso JDBC

Los recursos JDBC se suprimen automáticamente si se suprime el pool de conexiones JDBC asociado al recurso.

  1. En el árbol de navegación, amplíe el nodo Recursos.

  2. En el nodo Recursos, amplíe el nodo JDBC.

  3. En el nodo JDBC, seleccione el nodo Recursos JDBC.

    Se abrirá la página Recursos de JDBC.

  4. En la página Recursos JDBC, active la casilla de control situada a la izquierda del recurso que se va a suprimir.

  5. Haga clic en Suprimir.
