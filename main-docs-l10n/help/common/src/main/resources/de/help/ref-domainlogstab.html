Domainlogs

Auf der Seite "Domainlogs" können Sie alle verfügbaren Logdateien für eine angegebene Instanz oder ein angegebenes Cluster von GlassFish Server sammeln und herunterladen und die Logs in einem einzigen ZIP-Archiv kompilieren.

Hinweis:

Wenn Sie ein Cluster angeben, enthält das ZIP-Archiv außerdem die Logdatei für den Domain Administration Server (DAS).

Die Seite "Domainlogs" umfasst die folgenden Optionen.

Cluster

Eine Dropdown-Liste der Cluster, sofern Cluster vorhanden sind.

Instanz

Eine Dropdown-Liste der Serverinstanzen.

Logs sammeln

Schaltfläche, mit der Logs in einer log-files-name.zip-Datei gesammelt werden, wobei name der Name des Clusters oder der Instanz ist.

Die log-files-name.zip-Datei wird mit dem Herunterladverfahren Ihres Browsers heruntergeladen.

Verwandte Aufgaben

  • So werden Domainlogs gesammelt und heruntergeladen

  • So werden Logdateien für DAS angezeigt

Entsprechende asadmin-Unterbefehle

collect-log-files(1)
