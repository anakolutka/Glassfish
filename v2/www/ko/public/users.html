GlassFish는 Java EE 5 플랫폼의 최신 기능을 구현하는 무료 오픈 소스 응용 프로그램 서버입니다. Java EE 5 플랫폼에는 JavaServer Pages(JSP) 2.1, JavaServer Faces(JSF) 1.2, Servlet 2.5, Enterprise JavaBeans 3.0, Java API for Web Services(JAX-WS) 2.0, Java Architecture for XML Binding(JAXB) 2.0, Web Services Metadata for the Java Platform 1.0 등과 같은 다양한 최신 버전의 기술이 포함되어 있습니다.

최신 Java EE 5 기술을 사용하여 응용 프로그램 배우기 및 작성

GlassFish 응용 프로그램 서버는 모든 최신 Java EE 5 기술을 구현합니다. 예를 들어, JSF1.2와 같은 최신 웹 계층 기술을 사용하여 응용 프로그램을 작성합니다. 또한 최신 기술의 사용 외에도 기존 J2EE 1.4 이하 버전의 응용 프로그램을 GlassFish에서 실행하여 보다 유리한 조건에서 마이그레이션을 수행할 수 있습니다.
  • GlassFish에 대한 시작 설명서를 참조하십시오. 이 설명서에는 GlassFish 서버를 설치 및 실행하기 위한 지침 외에 배포를 위한 간단한 샘플 응용 프로그램이 포함되어 있습니다.
  • Java EE 5에 적용되는 작업에 대한 자세한 내용은 Java BluePrints 홈 페이지를 참조하십시오. 이 페이지에는 Java EE 5 플랫폼에서 AJAX 응용 프로그램을 빌드할 수 있는 BluePrints가 포함되어 있습니다.
  • 간단한 hello world 응용 프로그램 및 Java EE 5 지속성을 사용하는 고급 응용 프로그램 중 하나를 사용하여 NetBeans 사용 Java EE 응용 프로그램을 개발합니다.
  • javadocs는 온라인입니다. 이러한 설명서는 매 승격된 빌드 시 업데이트됩니다.
