變更管理密碼

將管理密碼加密可增強應用程式的安全性。如需有關安全性程序的更多詳細資訊，請參閱「Sun GlassFish Enterprise Server v3 Prelude Administration Guide」中的第 6 章「Administering System Security」。

  1. 在瀏覽樹狀結構中，展開 [配置] 節點。
  2. 若將網域配置為支援叢集，請選取您要變更的配置。

    例如，透過 cluster 設定檔或 enterprise 設定檔建立的網域。

  3. 展開 [安全性] 節點。
  4. 展開 [範圍] 節點。
  5. 選取 [admin-realm] 節點。
  6. 在 [編輯範圍] 頁面中按一下 [管理使用者] 按鈕。
  7. 選取名為 admin 的使用者。
  8. 輸入新的密碼並確認密碼。
  9. 按一下 [儲存]。
