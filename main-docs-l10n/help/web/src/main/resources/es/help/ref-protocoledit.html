Editar protocolo

Utilice la página Editar Protocolo para modificar la configuración de un protocolo de red existente.

La página Editar Protocolo contiene las siguientes opciones.

Cargar Valores por Defecto

Botón para restaurar configuración que tiene valores por defecto a sus valores por defecto. Las configuraciones que no tienen valores por defecto no se modifican.

Nombre de Configuración

Nombre de la configuración a la que se aplican los valores de esta página. Este campo es de sólo lectura.

Nombre

Nombre del protocolo. El campo Nombre es un campo de sólo lectura. Únicamente puede especificar el nombre al crear un nuevo protocolo.

Seguridad

Si se selecciona esta opción, se activa la seguridad para el protocolo. Si la seguridad está activada, puede utilizar el separador SSL para modificar la configuración de seguridad.

Si está configurando el admin-listener para el servidor de administración de dominio (DAS), que usa la configuración server-config, esta opción será de sólo lectura. Para activar la seguridad de este protocolo, debe activar primero la administración segura. Consulte la sección Administración Segura.
