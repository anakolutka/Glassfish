模組記錄層級

使用 [模組記錄層級] 頁面可配置個別模組的記錄層級。

[模組記錄層級] 頁面可顯示每則訊息的下列資訊。

記錄程式名稱

模組的記錄程式名稱 (例如 javax.enterprise.system.tools.admin)。

記錄程式層級

模組的目前記錄層級。

可用的記錄層級如下，以由高至低的順序列出。

SEVERE

干擾正常程式執行的事件。

WARNING

包括異常在內的警告。

INFO

與伺服器配置或伺服器狀態相關的訊息，不包括錯誤。

CONFIG

與伺服器配置相關的訊息。

FINE

最低詳細程度。

FINER

中等詳細程度。

FINEST

最高詳細程度。

OFF

關閉記錄訊息功能。

相關作業

  • 配置一般記錄設定

  • 配置記錄層級

  • 檢視伺服器記錄
