데이터베이스 액세스를 설정하는 방법

데이터를 저장, 구성 및 검색하기 위해 대부분의 응용 프로그램에서는 관계형 데이터베이스를 사용합니다.

  1. 지원되는 데이터베이스 제품을 설치합니다.

    지원되는 데이터베이스의 목록에 대해서는 Sun GlassFish Enterprise Server v3 Prelude Release Notes를 참조하십시오.

  2. 데이터베이스에 대한 JDBC API를 지원하는 드라이버(JDBC 드라이버)를 설치합니다.

    데이터베이스 연결을 만드는 데 사용하려는 드라이버가 데이터 소스를 배포하려는 모든 서버에 설치되어 있는지 확인합니다.

  3. 도메인의 서버 인스턴스가 드라이버의 JAR 파일에 액세스할 수 있는지 확인합니다.

    JDBC 드라이버를 통합하는 방법을 참조하십시오.

  4. 데이터베이스를 만듭니다.

    일반적으로 응용 프로그램 공급자가 데이터베이스를 만들고 입력하기 위한 스크립트를 제공합니다.

  5. 데이터베이스에 대한 연결 풀을 만듭니다.

    JDBC 연결 풀을 만드는 방법을 참조하십시오.

  6. 연결 풀을 가리키는 JDBC 자원을 만듭니다.

    JDBC 자원을 만드는 방법을 참조하십시오.
