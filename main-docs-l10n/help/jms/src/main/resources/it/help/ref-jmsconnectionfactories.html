Connection Factory JMS

Utilizzare la pagina Connection Factory JMS per configurare le risorse Connection Factory JMS.

Per ogni risorsa vengono fornite le informazioni riportate di seguito.

Nome pool

Il nome univoco che identifica il Connection Factory.

Nome JNDI

Il nome JNDI che identifica il Connection Factory.

Abilitato

L'impostazione è True se il Connection Factory è abilitato e False se il Connection Factory è disabilitato.

Tipo di risorsa

Il tipo di risorsa del Connection Factory.

Descrizione

La descrizione del Connection Factory.

La tabella Connection Factory contiene inoltre le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare un nuovo Connection Factory.

Elimina

Pulsante che consente di eliminare uno o più Connection Factory selezionati.

Abilita

Pulsante che consente di abilitare uno o più Connection Factory selezionati.

Disabilita

Pulsante che consente di disabilitare uno o più Connection Factory selezionati.
