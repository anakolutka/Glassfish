Para Exibir Dados para Monitoramento do Servidor Virtual da Instância do GlassFish Server Clusterizada

Antes de Começar

Antes de exibir os dados de monitoramento, é preciso configurar o serviço de monitoramento.

  1. Na árvore de navegação, expanda o nó Clusters.

  2. No nó Clusters expandidos, selecione o nó do cluster.

    A página Informações Gerais do cluster será aberta.

  3. Na página Informações Gerais, clique na guia Instâncias.

    A página Instâncias de Servidor Clusterizado será aberta com informações sobre o cluster.

  4. Selecione a instância para a qual você deseja exibir os dados de monitoramento de servidores virtuais.

    A página Informações Gerais da instância se abre.

  5. Na página Informações Gerais, clique na guia Monitor.

    A página Monitoramento da Aplicação será aberta.

  6. Clique na guia Servidor.

    A página Monitoramento do Servidor será aberta.

  7. Na lista drop-down Exibir, selecione o componente cujos dados de monitoramento você deseja exibir.

  8. Para atualizar os dados de monitoramento exibidos, clique em Atualizar.
