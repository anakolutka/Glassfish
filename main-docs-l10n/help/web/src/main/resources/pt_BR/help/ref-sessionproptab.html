Propriedades da Sessão

Use a página Propriedades da Sessão para adicionar ou modificar propriedades da sessão de contêiner Web.

A página Propriedades da Sessão contém as opções a seguir.

Carregar Defaults

Botão para restaurar definições para seus valores default. Definições que não tiveram seus valores default alterados.

Nome da Configuração

O nome da configuração à qual as definições nesta página se aplicam. Este campo é somente para leitura.

Time-out para Sessão

Especifica o número máximo de segundos durante o qual uma sessão inativa é válida. Se o valor é definido como 0, a sessão nunca expirará. O valor default é 1.800.

Propriedades Adicionais

Propriedades adicionais das sessões de contêiner Web. Para obter uma descrição das propriedades disponíveis, consulte Propriedades Específicas para as Sessões de Contêiner Web.
