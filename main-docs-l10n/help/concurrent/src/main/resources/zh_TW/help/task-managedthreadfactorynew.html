建立受管理繫線工廠

受管理繫線工廠指定 XXX。

  1. 在瀏覽樹狀結構中，展開「資源」節點。

  2. 在「資源」節點下，展開「並行資源」節點。

  3. 在「並行資源」節點下，選取「受管理繫線工廠」。

    就會開啟「受管理繫線工廠」頁面。

  4. 在「受管理繫線工廠」頁面上按一下「新建」。

    就會開啟「新建受管理繫線工廠」頁面。

  5. 在「JNDI 名稱」欄位中，鍵入唯一的名稱。

    該名稱將在 JNDI 命名服務中進行註冊。JNDI 名稱可用來為分散式運算環境中的元件加以組織與定位，就如同圖書館裡的卡片目錄，可用來將書籍分門別類並表示擺放位置。因此，JNDI 名稱便成為存取資源的重要方法。

  6. XXX

  7. 在「描述」欄位中，鍵入資源的簡短描述。

    該描述是一個字串值，最多可以包含 250 個字元。

  8. 選取「啟用狀態」核取方塊，以啟用資源。

    依預設會啟用資源，表示資源於建立後就可以使用。

  9. 在「其他特性」區段中，指定資源的其他特性。

    GlassFish Server 未定義受管理繫線工廠資源的任何其他特性。

    若要增加特性，請按一下「增加特性」按鈕。在出現的空白列中，於「名稱」欄位中鍵入特性名稱，然後在「值」欄位中鍵入特性值。

  10. 按一下「確定」。
