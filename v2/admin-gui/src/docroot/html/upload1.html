To Deploy Enterprise Applications/Modules

  1. In the Applications tab, click Deploy.

    The Deploy Enterprise Applications/Modules page appears.

  2. From the Type drop down list, select the type of application you want to deploy.
  3. Use the Browse button to select the archive file you want to deploy.
  4. In the Application Name field, type the name of the application.
  5. Click the Enabled check box next to the Status field to enable the application.
  6. Optionally, click the Enabled check box next to the Run Verifier field if you want to run the Verifier utility to check if the application is Java EE and Application Server compliant.
  7. (Optional) In the Description field, type a brief introduction for the application.
  8. Select a target for the application using the Add button.
  9. Click OK.
