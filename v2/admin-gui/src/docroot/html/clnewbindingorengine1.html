To Install a JBI Component

  1. Click the Install button.
  2. In the Install JBI Binding or Engine page, specify the location of the JBI component you want to install and click Next.

    You can either upload a JBI component from your local file system or copy a JBI component from the Application Server host's filesystem. Use the Browse Files button to navigate to the directory where the archive is located and select the archive.

  3. Click Enabled if you want to start the JBI component immediately after it is installed.
  4. (Optional) In the Add to Available Targets table, by default, a target is specified. You can specify additional targets for installation or specify no targets. If you specify no targets, the component is installed into the JBI repository.

    Note - Make sure that all the Shared Libraries on which a JBI component depends are installed to the same target as the JBI component.

  5. (Optional) In the Configuration Properties table, provide values to override the default values for the configuration properties.
  6. (Optional) In the Additional Properties table, add properties and values by clicking the Add Property button. See the component's documentation for additional properties it supports, if any.
  7. Click Finish.

    The Components page lists the new component.
