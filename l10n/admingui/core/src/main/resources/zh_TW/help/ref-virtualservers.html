虛擬伺服器

虛擬伺服器 (有時又稱為虛擬主機) 是允許相同實體伺服器代管多個網際網路網域名稱的物件。相同實體伺服器上代管的所有虛擬伺服器，會共用該實體伺服器的網際網路協定 (IP) 位址。虛擬伺服器會建立伺服器的網域名稱 (例如 www.sun.com) 與執行 Enterprise Server 之特定伺服器的關聯。
