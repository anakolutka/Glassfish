Nuova risorsa oggetto amministrato

Utilizzare la pagina Nuova risorsa oggetto amministrato per creare una nuova risorsa oggetto amministrato.

La pagina Nuova risorsa oggetto amministrato contiene le opzioni riportate di seguito.

Nome JNDI

Il nome univoco che identifica la risorsa oggetto amministrato.

Resource Adapter

Il Resource Adapter per la risorsa oggetto amministrato.

Tipo di risorsa

Il tipo Java per la risorsa oggetto amministrato, ad esempio javax.jms.Topic.

Nome classe

Il nome della classe di implementazione associato al tipo di risorsa.

Descrizione

La descrizione della risorsa oggetto amministrato.

Stato

Se questa opzione è selezionata, la risorsa oggetto amministrato è abilitata. Questa opzione è abilitata per impostazione predefinita.

Proprietà aggiuntive

Proprietà aggiuntive per la risorsa oggetto amministrato. Le proprietà disponibili dipendono dal Resource Adapter per il connection pool del connettore. Le coppie nome-valore specificate in questa tabella possono essere utilizzate per sostituire i valori predefiniti per le proprietà specificate dal fornitore del Resource Adapter.

Destinazioni

I cluster e le istanze standalone per la risorsa oggetto amministrato. Per spostare le destinazioni desiderate nella colonna Destinazioni selezionate, utilizzare i pulsanti Aggiungi e Aggiungi tutto. Per spostare le destinazioni non necessarie nella colonna Destinazioni disponibili, utilizzare i pulsanti Rimuovi e Rimuovi tutto.

Questa opzione viene visualizzata solo se nel dominio sono stati creati cluster o istanze standalone.
