Para establecer los valores de la ruta de JVM

La ruta de clase es la lista de archivos JAR en la que el entorno de ejecución de Java busca clases y otros recursos.

  1. En el árbol de navegación, seleccione el nodo Configuración.
  2. En el nodo Configuración, seleccione el nodo Configuración de JVM.

    Se abrirá la página Configuración general de JVM.

  3. En la página Configuración general de JVM, haga clic en la ficha Configuración de ruta.

    Se abrirá la página Configuración de ruta de clase de JVM.

  4. Marque la casilla de verificación Ignorar ruta de clase de entorno para omitir la variable de entorno CLASSPATH.

    La variable de entorno CLASSPATH es práctica para tutoriales básicos de programación, pero no se recomienda para entornos empresariales.

  5. En el campo Ruta de clase del sistema, introduzca la ruta a las clases del sistema.
  6. En el campo Prefijo de ruta de clase, introduzca las rutas completas de los archivos JAR que se van a anteponer a la ruta de clase del servidor.

    Para anteponer varios archivos, introduzca una ruta por línea.

  7. En el campo Sufijo de ruta de clase, introduzca las rutas completas de los archivos JAR que se van a anexar a la ruta de clase del servidor.

    Por ejemplo, tendría que especificar el archivo JAR de un controlador de base de datos. Consulte To Integrate a JDBC Driver.

    Para anexar varios archivos, introduzca una ruta por línea.

  8. En el campo Prefijo de ruta de biblioteca nativa, introduzca las rutas completas de los archivos JAR que se van a anteponer a la ruta de biblioteca nativa.

    Para anteponer varios archivos, introduzca una ruta por línea.

  9. En el campo Sufijo de ruta de biblioteca nativa, introduzca las rutas completas de los archivos JAR que se van a anexar a la ruta de biblioteca nativa.

    Para anexar varios archivos, introduzca una ruta por línea.

    La ruta de biblioteca nativa consiste en una concatenación de las rutas siguientes:

    • Ruta relativa del servidor para las bibliotecas compartidas nativas

    • Ruta de biblioteca nativa de entorno de tiempo de ejecución estándar

    • Configuración del entorno del shell (LD_LIBRARY_PATH en los sistemas UNIX®)

    • Todas las rutas especificadas en la página Configuración de analizador JVM

  10. Haga clic en Guardar.
  11. Reinicie el servidor.
