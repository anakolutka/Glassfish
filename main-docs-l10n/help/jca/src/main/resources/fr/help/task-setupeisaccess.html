Configuration de l'accès au système EIS

  1. Déployez (installez) un connecteur. Reportez-vous à Déploiement d'un module connecteur.

  2. Créez un pool de connexions pour le connecteur. Reportez-vous à Création d'un pool de connexions de connecteur.

  3. Créez une ressource de connecteur associée au pool de connexions. Reportez-vous à Création d'une ressource de connecteur.
