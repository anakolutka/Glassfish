Para crear un recurso JDBC

Un recurso de Java DataBase Connectivity (JDBC) (fuente de datos) proporciona a las aplicaciones un método para conectarse a una base de datos.

Antes de empezar

Para crear un recurso JDBC, debe crear primero un conjunto de conexiones JDBC. Consulte Para crear un conjunto de conexiones JDBC.

  1. En el árbol de navegación, expanda el nodo Recursos.
  2. Seleccione el nodo Recursos JDBC.
  3. En la página Recursos JDBC, haga clic en Nuevo.

    Se abrirá la página Nuevo recurso JDBC.

  4. En el campo Nombre J.N.D.I., introduzca un nombre de recurso exclusivo.

    El nombre J.N.D.I. organiza y ubica componentes dentro de un entrono informático distribuido del mismo modo que los ficheros organizan y representan ubicaciones de libros en una biblioteca. Por consiguiente, el nombre J.N.D.I. se convierte en un importante método de acceso al recurso JDBC. Según convención, el nombre empieza por la cadena jdbc/, por ejemplo, jdbc/payrolldb . No olvide la barra oblicua.

    El nombre debe contener únicamente caracteres alfanuméricos, de subrayado, guiones y puntos.

  5. En la lista desplegable Nombre de conjunto, seleccione un conjunto de conexiones para asociarlo al nuevo recurso JDBC.
  6. En el campo Descripción, introduzca una breve descripción del nuevo recurso JDBC.
  7. Para deshabilitar el recurso de manera que no esté disponible, anule la selección de la casilla de verificación Habilitado.

    De forma predeterminada, el recurso está disponible (activado) en cuanto se crea.

  8. Haga clic en Aceptar.
