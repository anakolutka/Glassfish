Configurar valores de proxy de red para acceder a repositorios de red

La configuración de proxy de red determina la manera en que GlassFish Server se conecta a Internet. Para activar Herramienta de Actualización para que acceda a los repositorios de red a través de Internet, configure el proxy de red.

  1. En el árbol de navegación, seleccione el nodo Herramienta de Actualización.

    Se abrirá la página Complementos Disponibles.

  2. En la página Complementos Disponibles, haga clic en el separador Configuración.

    Se muestra la página Configuración de Herramienta de Actualización.

  3. En la página Configuración de Herramienta de Actualización, especifique cómo se conecta a Internet la máquina en la que se está ejecutando GlassFish Server.

    • Si la máquina se conecta a Internet directamente, seleccione la opción Conexión Directa.

    • Si la máquina se conecta a Internet a través de un proxy, seleccione la opción A través de un Proxy.

  4. Si la máquina en la que se está ejecutando GlassFish Server se conecta a Internet a través de un proxy, obtenga el nombre de host y el número de puerto del proxy.

    Si la máquina en la que se está ejecutando GlassFish Server se conecta directamente a Internet, omita este paso.

    Un explorador web debe estar ejecutándose en la máquina en la que se está ejecutando GlassFish Server. En esta situación, puede obtener el nombre de host y el número de puerto del proxy del explorador web. De lo contrario, póngase en contacto con el administrador del sistema para obtener más información.

  5. Haga clic en Guardar.
