Neuer Dateibereichsbenutzer

Über die Seite "Neuer Dateibereichsbenutzer" können Sie einem neuen Benutzer Zugriff auf den aktuellen Bereich gewähren.

Bereichsname

Ein schreibgeschütztes Feld mit dem Namen des aktuell ausgewählten Bereichs.

Benutzer-ID

Die Benutzer-ID für das neue Benutzerkonto.

Gruppenliste

Gruppen, zu denen der neue Benutzer gehören wird. Trennen Sie mehrere Gruppennamen durch Kommata.

Neues Passwort

Passwort für den neuen Benutzer.

Neues Passwort bestätigen

Passwort für den neuen Benutzer.

Verwandte Aufgaben

  • So verwalten Sie Dateibenutzer

  • So fügen Sie einen Benutzer hinzu

  • So bearbeiten Sie Benutzerinformationen

  • So löschen Sie einen Benutzer

Verwandte asadmin-Befehle

  • list-file-users(1)

  • create-file-user(1)

  • update-file-user(1)

  • delete-file-user(1)
