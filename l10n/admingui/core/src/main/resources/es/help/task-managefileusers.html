Cómo administrar usuarios del dominio file

La administración de usuarios de file se lleva a cabo en Servidor empresarial a través del nodo de dominios de seguridad.

  1. En el árbol de navegación, expanda el nodo Configuración.
  2. Si el dominio está configurado para admitir clústeres, seleccione la configuración que desea cambiar.

    Algunos ejemplos son los dominios creados con el perfil clúster o con el perfil grandes empresas .

  3. Expanda el nodo Seguridad.
  4. Expanda el nodo Dominios.
  5. Seleccione el nodo file.
  6. Haga clic en el botón Administrar usuarios de la página Editar dominio.

    Se abrirá la página Usuarios de file. En esta página tiene que realizar las tareas siguientes:

    • Para agregar un usuario

    • Cómo editar la información de usuario

    • Para eliminar un usuario
