新しいレルム

「新しいレルム」ページを使用して、レルムを作成します。

「新しいレルム」ページには、次のオプションが含まれます。

名前

新しいレルムの名前。

クラス名

新しいレルムのクラス名。

選択可能なクラス名を次の表に示します。

レルム名

クラス名

file

com.sun.enterprise.security.auth.realm.file.FileRealm

certificate

com.sun.enterprise.security.auth.realm.certificate.CertificateRealm

ldap

com.sun.enterprise.security.auth.realm.ldap.LDAPRealm

JDBC

com.sun.enterprise.security.auth.realm.jdbc.JDBCRealm

solaris

com.sun.enterprise.security.auth.realm.solaris.SolarisRealm

カスタムレルムを指定するには、下側のラジオボタンをオンにし、作成するカスタムレルムのクラス名を入力します。

追加プロパティー

レルムの追加プロパティー。有効なプロパティーはレルムのタイプによります。

  • ファイルレルムプロパティーの詳細については、「FileRealm クラスに固有のプロパティー」を参照してください。

  • 証明書レルムプロパティーの詳細については、「CertificateRealm クラスに固有のプロパティー」を参照してください。

  • JDBC レルムプロパティーの詳細については、「JDBCRealm クラスに固有のプロパティー」を参照してください。

  • LDAP レルムプロパティーの詳細については、「LDAPRealm クラスに固有のプロパティー」を参照してください。

  • Solaris レルムプロパティーの詳細については、「SolarisRealm クラスに固有のプロパティー」を参照してください。
