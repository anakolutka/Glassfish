ファイルレルムユーザーの編集

「ファイルレルムユーザーの編集」ページを使用して、現在のレルムに対するユーザーのアクセス権限を変更します。

レルム名

現在選択されているレルムの名前を表示する読み取り専用フィールド。

ユーザー ID

編集するアカウントの名前を表示する読み取り専用フィールド。

グループ リスト

このユーザが属するグループ。複数のグループ名をコンマで区切ります。

新しいパスワード

このユーザーの新しいパスワード。

新しいパスワードを確認

このユーザーの新しいパスワード。
