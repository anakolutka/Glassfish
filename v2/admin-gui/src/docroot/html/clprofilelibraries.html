To Manage Shared Libraries in the Cluster Profile

The Shared Libraries page enables you to view information about shared libraries currently deployed and also allows you to install or delete them.

  1. To install a new Shared Library, click Install. The Install JBI Shared Library page appears using which you can install the new Shared Library. For detailed steps, see the help on To Install Shared Libraries.

  2. To delete a Shared Library, click the check box next to the Shared Library that you want to delete and click Delete.
