Para Exibir Informações Gerais do DAS

  1. Na árvore de navegação, selecione o nó do servidor (Servidor Admin).

    A página Informações Gerais será aberta.

  2. Na página Informações Gerais, execute qualquer uma das etapas a seguir.

    • Observe a exibição de Status somente para leitura para ver se o servidor está em execução.

    • Observe a exibição de Período de Disponibilidade somente para leitura para ver o número de horas e minutos durante os quais o servidor esteve em execução contínua.

    • Clique no botão Interromper para interromper o GlassFish Server.

    • Clique no botão Reiniciar para interromper e reiniciar o GlassFish Server.

    • Clique no botão Exibir Arquivos de Log para exibir arquivos de log de instâncias ou clusters do GlassFish Server.

      Consulte Para Exibir Arquivos de Log do DAS para obter detalhes.

    • Clique no Relatório JVM para abrir uma janela separada que lhe permite exibir relatórios no Java Virtual Machine, incluindo um relatório de resumo, um relatório de gerenciamento de memória, um relatório de carregamento de classes e um dump de threads atuais.
