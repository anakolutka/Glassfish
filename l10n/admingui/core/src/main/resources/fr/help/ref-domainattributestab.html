Attributs de domaine

La page Attributs de domaine permet de définir les attributs de domaine avancés de Enterprise Server.

La page Attributs de domaine contient les options suivantes :

Racine application

Chemin complet du répertoire où les applications déployées résident. La valeur par défaut est domain-dir/applications.

Racine journal

Identifie l'emplacement de stockage des fichiers journaux du serveur. La valeur par défaut est domain-dir/logs .

Paramètres locaux

Si ce champ n'est pas renseigné, les paramètres locaux par défaut de l'hôte sont utilisés.

Un environnement linguistique est un identifiant qui spécifie une combinaison particulière de langues et de régions. Par exemple, l'environnement linguistique pour l'anglais des États-Unis est en_US et pour le japonais, ja_JP. Pour utiliser un environnement linguistique non anglophone, Enterprise Server doit être localisé, c'est-à-dire adapté de l'anglais en une autre langue.
