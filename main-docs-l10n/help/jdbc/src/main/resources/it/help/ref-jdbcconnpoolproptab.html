Modifica proprietà connection pool JDBC

Utilizzare la pagina Modifica proprietà connection pool JDBC per modificare le proprietà aggiuntive di un connection pool JDBC esistente.

Proprietà aggiuntive

Proprietà aggiuntive per il connection pool JDBC.

Per la descrizione di alcune delle proprietà disponibili, vedere Proprietà specifiche dei connection pool JDBC.

Se non si specifica un valore per una proprietà, la proprietà interessata verrà rimossa dalla tabella al termine della modifica del connection pool. Se non si desidera impostare il valore di una proprietà in questa fase, conservando tuttavia la proprietà nella tabella in modo da impostarne il valore in un secondo momento, digitare un set di parentesi vuote nel campo Valore:

()
