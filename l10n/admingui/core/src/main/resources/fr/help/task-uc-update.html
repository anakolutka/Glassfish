Pour installer des mises à jour de composants

La mise à jour d'un composant installé comprend les opérations suivantes :

  • Transférer uniquement les fichiers modifiés depuis un référentiel de réseau vers votre système

  • Mettre à jour uniquement les fichiers modifiés dans l'Enterprise Server image d'installation dans votre système

  • Appliquer de nouvelles dépendances (si nécessaire) entre les composants mis à jour

  1. Dans l'arborescence de navigation, sélectionnez le nœud Outil de mise à jour.

    La page Add-ons disponibles s'affiche.

  2. Dans la page Add-ons disponibles, cliquez sur l'onglet Mises à jour disponibles.

    La page Mises à jour disponibles s'affiche.

  3. Dans le tableau des mises à jour disponibles, sélectionnez les composants à mettre à jour.

    Il n'y a aucune différence de version ni de taille entre la Version installée et la Nouvelle version d'un composant, inutile donc de mettre le composant à jour. Si aucune mise à jour n'est disponible, le tableau est vide.

    • Pour sélectionner un composant, cochez la case située en regard de son nom.
    • Pour sélectionner tous les composants, cliquez sur l'icône Sélectionner tout dans l'en-tête du tableau.
    • Pour désélectionner tous les composants, cliquez sur l'icône Désélectionner tout dans l'en-tête du tableau.
  4. Cliquez sur Installer.
  5. Acceptez le contrat de licence.

    Enterprise Server confirme que l'installation est terminée.

  6. Redémarrez le serveur.

    Pour consulter des instructions détaillées, reportez-vous à la documentation ci-dessous:

    • To Stop a Domain du Sun GlassFish Enterprise Server v3 Administration Guide

    • To Start a Domain du Sun GlassFish Enterprise Server v3 Administration Guide
