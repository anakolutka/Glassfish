Servicio de supervisión

Utilice la página Servicio de supervisión para configurar opciones de supervisión de servicios o componentes del servidor individual.

Para cada opción de supervisión están disponibles los niveles BAJO, ALTO Y DESACTIVADO. No existe diferencia alguna entre los niveles de supervisión que ofrecen las opciones BAJO y ALTO de esta versión.
