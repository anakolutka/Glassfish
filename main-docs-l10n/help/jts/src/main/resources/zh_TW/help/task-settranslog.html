設定作業事件日誌位置

為了保持被呼叫資源的資料完整性，同時為了能夠從故障中恢復，作業事件日誌將記錄有關每個作業事件的資訊。作業事件日誌儲存在「作業事件日誌位置」欄位所指定目錄的 tx 子目錄中。這些不是一般人可解讀的日誌。

  1. 在瀏覽樹狀結構中，展開「組態」節點。

  2. 選取要組態的執行處理：

    • 若要組態特定的執行處理，請選取該執行處理的組態節點。例如，對於預設執行處理 server，請選取「server-config」節點。

    • 若要組態所有執行處理的預設設定，請選取「default-config」節點。

  3. 選取「作業事件服務」節點。

    「作業事件服務」頁面會隨即開啟。

  4. 在「作業事件日誌位置」欄位中，鍵入作業事件日誌的位置。

    tx 子目錄便會建立，並且會在該目錄下保存作業事件日誌。

  5. 按一下「儲存」。
