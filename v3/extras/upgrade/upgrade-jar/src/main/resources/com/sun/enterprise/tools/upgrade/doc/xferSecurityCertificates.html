Transfer Security Certificates Screen

Use the Transfer Security Certificates screen to specify domains that contain a set (or multiple sets) of security certificates that need to be migrated to Sun Java Systems Application Server 9:

  1. Press the Add button to display the Add Domain dialog where the name of the domain and the passwords for the certificate files are entered.

    The Domains with Certificates to be Transferred panel displays domains that were added.

  2. Repeat this process until all the domains that contain certificates that need to be transferred have been added.
  3. To delete a domain that was added, select the domain to delete, then press the Delete Domain button.
  4. When done, press the Next button.

    The Upgrade Results screen, showing the status of the upgrade.
