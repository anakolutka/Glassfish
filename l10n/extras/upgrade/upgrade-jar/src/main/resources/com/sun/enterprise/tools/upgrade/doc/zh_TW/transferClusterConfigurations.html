【傳送叢集配置】螢幕

若要將 Sun Java System Application Server 8.x Enterprise Edition (EE) 叢集配置升級到 Sun Java System Application Server 9.1，需要有效的 clinstance.conf 檔案。「升級工具」會認定 clinstance.conf 檔案精準反映 8.x EE 叢集。如果，基於某些原因，此檔案無法精準代表實際的叢集配置時，則可能需要手動編輯叢集定義，之後才能執行升級。

使用【傳送叢集配置】螢幕來指定要遷移到新安裝的叢集配置檔案。

  1. 按下【新增】按鈕，【選取 clinstance.conf 檔案】對話方塊即顯示。
  2. 選取 clinstance.conf 叢集檔案，其包含要遷移的叢集配置資訊。
  3. 按下【開啟】按鈕。
  4. 重複此程序，直到必須遷移的所有叢集配置檔案都已加入。
  5. 完成後，請按【下一步】按鈕。將顯示【升級結果】螢幕，指示升級的狀態。
