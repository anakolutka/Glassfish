Suprimir un módulo de ciclo de vida

  1. En el árbol de navegación, seleccione el nodo Módulos de Ciclo de Vida.

    Se abrirá la página Módulos de Ciclo de Vida.

  2. En la página Módulos de ciclo de vida, active la casilla de control que aparece a la izquierda del módulo de ciclo de vida que desea suprimir.

  3. Haga clic en Suprimir.
