Para crear una asignación de seguridad para un pool de conexiones de conector

Las asignaciones de seguridad para los pools de conexiones del conector permiten crear una asignación entre los usuarios y grupos de usuarios de Java EE y los de EIS.

  1. En el árbol de navegación, amplíe el nodo Recursos.

  2. Amplíe el nodo Conectores.

  3. En el nodo Configuración, seleccione el nodo Pools de conexiones de conector.

    Se abrirá la página Pools de conexiones de conectores.

  4. En la página Pools de conexiones de conectores, haga clic en el nombre del pool de conexiones que desee editar.

    Se abrirá la página Editar pool de conexiones del conector.

  5. En la página Editar pool de conexiones de conector, haga clic en el separador Asignaciones de seguridad.

    Se abrirá la página Asignaciones de seguridad de pool de conexiones de conector.

  6. En la página Asignaciones de seguridad del pool de conexiones del conector, haga clic en Nuevo.

    Se abrirá la página Asignaciones de seguridad del pool de conexiones del conector

  7. En el campo Nombre de asignación de seguridad escriba un nombre para la asignación de seguridad.

  8. Realice una de las siguientes acciones, pero no ambas:

    • Active el botón de radio Grupos de Usuarios y escriba los nombres de los grupos de usuarios, separados por comas.

    • Active el botón de radio Principales y escriba los nombres del principal, separados por comas.

  9. En el campo Nombre de Usuario, especifique el nombre de usuario del principal de backend requerido por el EIS.

  10. En el campo Contraseña, especifique la contraseña del principal de backend requerida por el EIS.

  11. Haga clic en Aceptar.

  12. Reiniciar GlassFish Server.
