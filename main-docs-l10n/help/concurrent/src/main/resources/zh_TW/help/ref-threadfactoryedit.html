編輯受管理繫線工廠

您可以使用「編輯受管理繫線工廠」頁面來編輯受管理繫線工廠資源。

「編輯受管理繫線工廠」頁面包含下列選項。

載入預設值

用於將具有預設值的設定復原為設定值的按鈕。沒有預設值的設定則保持不變。

JNDI 名稱

可識別資源的唯一名稱。「JNDI 名稱」是唯讀欄位。您僅能在建立新資源時指定 JNDI 名稱。

邏輯 JNDI 名稱

資源的邏輯 JNDI 名稱。這是可用來存取資源的 Java EE 標準名稱。邏輯 JNDI 名稱是唯讀欄位。請參閱 Logical JNDI Names 以瞭解詳細資訊。

環境資訊

要傳輸至繫線的環境：Classloader、JNDI、Security 或 WorkArea。您可以啟用或停用環境傳輸。如果啟用，就會傳輸選取的環境。如果停用，就不會傳輸任何環境 (包括選取的環境)。預設值為「啟用」(包括全部的四個環境)。

狀態

是否可在執行階段使用資源。

如果網域中只有預設伺服器執行處理 server 存在，可以選擇啟用或停用資源的執行階段可用性。預設會啟用執行階段可用性。

如果已在網域中建立任何叢集或獨立執行處理，可使用「目標」頁籤控制其目標之資源的執行階段可用性。

繫線優先順序

要指定給已建立繫線的優先順序。預設值為 5。

建置順序

此資源的建置順序。系統會在伺服器啟動時先載入數字較小的資源。預設值為 100。

描述

此資源的描述。

其他特性

資源的其他特性。GlassFish Server 並未定義此資源類型的任何其他特性。此外，此資源類型目前也並未使用任何其他特性。
