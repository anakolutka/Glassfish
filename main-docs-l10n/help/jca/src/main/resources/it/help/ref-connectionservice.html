Servizio connettore

Utilizzare la pagina Servizio connettore per modificare le impostazioni generali per il servizio connettore che gestisce i Resource Adapter.

La pagina Servizio connettore contiene le opzioni riportate di seguito.

Carica impostazioni predefinite

Pulsante per ripristinare i valori predefiniti delle impostazioni che dispongono di valori predefiniti. Le impostazioni che non dispongono di valori predefiniti non vengono modificate.

Nome configurazione

Il nome della configurazione a cui si applicano le impostazioni presenti in questa pagina. Questo campo è di sola lettura.

Timeout chiusura

Il numero massimo di secondi consentito durante la chiusura di GlassFish Server per il completamento del metodo ResourceAdapter.stop di un'istanza di modulo connettore. I Resource Adapter che richiedono tempi di chiusura più lunghi vengono ignorati e la procedura di chiusura di GlassFish Server continua. Il valore predefinito è 30 secondi.

Criteri di caricamento classi del connettore

I criteri da utilizzare per il caricamento delle classi. Di seguito sono riportate le scelte disponibili.

derived

Indica che i Resource Adapter vengono forniti in base ai riferimenti di un'applicazione a qualsiasi risorsa del Resource Adapter. Questo è il criterio predefinito.

global

Indica che tutti i Resource Adapter saranno visibili per tutte le applicazioni.

Questo criterio non si applica per il Resource Adapter di sistema JMS preinstallato, jmsra, che è sempre visibile per tutte le applicazioni.
