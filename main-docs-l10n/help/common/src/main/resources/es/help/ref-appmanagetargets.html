Gestionar Destinos

Utilice la página Gestionar Destinos para cambiar los clusters de destino y las instancias de servidor independientes en los que se puede activar una aplicación. La aplicación sólo se puede activar en los destinos de la columna Destinos seleccionados.

La página Gestionar Destinos contiene la siguiente información.

Destinos Disponibles

Clusters e instancias en los que la aplicación no está desplegada.

Destinos Seleccionados

Clusters e instancias en los que la aplicación está desplegada.

Agregar

Botón para mover el destino seleccionado de Destinos Disponibles a Destinos Seleccionados.

Agregar Todos

Botón para mover todos los destinos disponibles a Destinos Seleccionados.

Eliminar

Botón para mover el destino seleccionado de Destinos Seleccionados a Destinos Disponibles.

Eliminar Todos

Botón para mover todos los destinos seleccionados a Destinos disponibles.
