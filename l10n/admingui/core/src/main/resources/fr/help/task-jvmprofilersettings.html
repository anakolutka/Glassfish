Configurer les paramètres du profileur JVM

Un outil de profilage génère des données servant à analyser les performances et à identifier les éventuels goulots d'étranglement.

  1. Dans l'arborescence de navigation, sélectionnez le nœud Application Server.
  2. Dans la page Informations générales, cliquez sur l'onglet Paramètres JVM.
  3. Dans la page Paramètres généraux JVM, cliquez sur l'onglet Profileur.

    La page Paramètres du profileur JVM s'ouvre.

  4. Définissez les paramètres du profileur.

    Les informations saisies dépendent du profileur utilisé. Pour obtenir des exemples et des instructions, reportez-vous au Chapitre 3, Debugging Applications du Sun GlassFish Enterprise Server v3 Prelude Developer’s Guide.

  5. Cliquez sur Enregistrer.
  6. Redémarrez le serveur.
