新建托管线程工厂

使用“新建托管线程工厂”页可创建托管线程工厂资源。

“新建托管线程工厂”页包含以下选项。

JNDI 名称

标识资源的唯一 JNDI 名称。按照惯例，名称以资源类型指示符和斜杠开头，例如 concurrent/、jdbc/ 或 jms/。

上下文信息

传播到线程的上下文：类加载器、JNDI、安全性或工作区。可以启用或禁用上下文传播。如果启用，则将传播所选上下文。如果禁用，则不传播任何上下文，即使已选中也是如此。默认值为“已启用”并选中所有四个上下文。

状态

资源的状态。可以启用或禁用资源。默认值为“已启用”。

线程优先级

分配给已创建线程的优先级。默认值为 5。

说明

此资源的说明。

其他属性

资源的其他属性。GlassFish Server 不定义此资源类型的任何其他属性。此外，此资源类型当前不使用任何其他属性。

目标

(如果域中已创建任何集群或独立实例，则显示此选项。)

资源的集群和独立实例。使用“添加”和“全部添加”按钮将所需目标移动到“所选目标”列。使用“删除”和“全部删除”按钮将任何不需要的目标移动到“可用目标”列。
