Suplementos Disponíveis

A página Suplementos Disponíveis fornece informações sobre os componentes da imagem de instalação ainda não instalados.

A página Suplementos Disponíveis contém as seguintes opções.

Caminho da Instalação

O caminho do sistema de arquivos do diretório raiz ou de nível superior da imagem de instalação.

Instalar

Este botão instala os componentes de suplemento selecionados.

Suplementos Disponíveis

Esta tabela fornece as seguintes informações sobre cada componente da imagem de instalação, os quais ainda não foram instalados.

Componente

O nome do componente.

Categoria

A categoria de software que é associada ao componente; por exemplo, Servidores de Aplicativos, Ferramentas do Sistema e Bancos de Dados. A categoria é atribuída pelo autor do componente quando o componente é criado.

Versão

O número de versão do pacote em que o componente foi fornecido.

Tamanho Instalado

A quantidade de espaço em disco que a instalação do componente consome.

Origem

Para um componente instalado, o nome do Repositório de rede de onde o componente foi obtido.

Para um componente de suplemento ou para uma atualização, o nome do repositório de rede de onde o componente deve ser obtido.
