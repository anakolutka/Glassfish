Modullogebenen

Verwenden Sie die Seite "Modul-Logebenen", um die Logebenen für einzelne Module zu konfigurieren.

Folgende Informationen werden für jedes Modul zur Verfügung gestellt.

Logger-Name

Der Name des Loggers für das Modul (z.B. javax.enterprise.system.tools.admin).

Logger-Ebene

Die aktuelle Logging-Ebene für das Modul.

Die Tabelle "Logger-Einstellungen" enthält darüber hinaus die folgenden Optionen.

Logger hinzufügen

Schaltfläche zum Hinzufügen eines Logger-Moduls. Durch Klicken auf diese Schaltfläche wird eine Zeile zur Tabelle "Logger-Einstellungen" hinzugefügt. Geben Sie Informationen über das Logger-Modul in die neue Zeile ein.

Logger löschen

Schaltfläche zum Löschen von mindestens einem Modul.

Ebene

Dropdown-Liste, aus der Sie eine Logging-Ebene auswählen können, die auf alle ausgewählten Module angewendet werden soll.

Ebene ändern

Schaltfläche zum Ändern der Logging-Ebene für mindestens ein ausgewähltes Modul.

Die folgenden Logebenen sind verfügbar. Sie sind in der Reihenfolge von der höchsten zur niedrigsten aufgelistet.

Schwerwiegend

Ereignisse, die die normale Programmausführung stören.

Warnung

Warnungen, einschließlich Ausnahmen.

Info

Meldungen, die mit Serverkonfiguration oder Serverstatus in Zusammenhang stehen, ausgenommen Fehler.

CONFIG

Meldungen, die mit der Serverkonfiguration in Zusammenhang stehen.

Genau

Minimaler Ausführlichkeitsgrad.

Genauer

Mittlerer Ausführlichkeitsgrad.

Sehr genau

Maximaler Ausführlichkeitsgrad.

Deaktiviert

Es werden keine Meldungen protokolliert.

Verwandte Aufgaben

  • So werden allgemeine Logging-Einstellungen konfiguriert

  • So werden Logebenen konfiguriert

  • So werden Logdateien für DAS angezeigt
