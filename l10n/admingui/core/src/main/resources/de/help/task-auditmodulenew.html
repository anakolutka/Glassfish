So erstellen Sie ein Überwachungsmodul

Enterprise Server bietet ein einfaches standardmäßiges Überwachungsmodul; weitere Informationen finden Sie unter So verwenden Sie das standardmäßige Überwachungsmodul.

  1. Erweitern Sie den Konfigurationsknoten.
  2. Wählen Sie die zu konfigurierende Instanz:
    • Zum Konfigurieren einer bestimmten Instanz erweitern Sie den config-Knoten der Instanz. Für die Standardinstanz server erweitern Sie z. B. den Knoten server-config.
    • Zum Konfigurieren der Standardeinstellungen für alle Instanzen erweitern Sie den Knoten default-config.
  3. Erweitern Sie den Knoten "Sicherheit".
  4. Wählen Sie den Knoten "Überwachungsmodule".
  5. Klicken Sie auf der Seite "Überwachungsmodule" auf "Neu".
  6. Geben Sie auf der Seite "Überwachungsmodul erstellen" die folgenden Informationen ein:

    • Name – Der Name zur Identifizierung dieses Überwachungsmoduls.

    • Klassenname – Der vollqualifizierte Name der Klasse, welche dieses Modul implementiert. Der Klassenname für das standardmäßige Überwachungsmodul lautet com.sun.enterprise.security.Audit.

  7. Zum Hinzufügen von JVM-Eigenschaften zu diesem Modul klicken Sie auf "Eigenschaft hinzufügen". Geben Sie einen Namen und einen Wert für jede Eigenschaft ein.

    Eine der zulässigen Eigenschaften ist auditOn, welche angibt, ob diese Implementierungsklasse zu aktivieren ist oder nicht. Zulässige Werte sind true und false.

  8. Klicken Sie auf "OK", um Einträge zu speichern, oder klicken Sie auf "Abbrechen", um den Vorgang zu beenden, ohne die Änderungen zu speichern.
