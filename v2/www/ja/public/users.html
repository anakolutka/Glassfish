GlassFish は、Java EE 5 プラットフォームの最新機能を実装した、フリーのオープンソースで開発されたアプリケーションサーバーです。 Java EE 5 プラットフォームでは、JavaServer Pages(JSP) 2.1、JavaServer Faces(JSF) 1.2、 Servlet 2.4、 Enterprise JavaBeans 3.0、 Java API for Web Services(JAX-WS) 2.0、 Java Architecture for XML Binding(JAXB) 2.0、 Web Services Metadata for the Java Platform 1.0 などの最新のテクノロジーを提供しています。

GlassFish についてあまりご存じでない場合は、実際に GlassFish をダウンロードしてインストールし、スタートガイドに沿っていろいろと試したり、スクリーンキャストを見てみたり、チュートリアルにあるいくつかのサンプルを実際に使ったりしながら学んでみてください。 また、 GlassFish の概要ページも参照してください。

ダウンロードして、実行してみる

GlassFish アプリケーションサーバーの安定したバージョン、あるいは最新のアーリーアクセスバージョンをダウンロードできます。 ダウンロードしたファイルには、GlassFish Java EE 5 application server のバイナリが含まれていますので、あとは簡単なインストラクションに従って、それを実行するだけです。ダウンロードサイトには、主なオペレーティングシステム用のダウンロードファイルが置かれています。

最新の Java EE 5 テクノロジーを用いてアプリケーションを書きながら学んでみる

GlassFish アプリケーションサーバーには最新の Java EE 5 テクノロジーが実装されています。  例えば、JSF1.2 などの最新の Web Tier テクノロジーを用いてアプリケーションを書いてみてください。  最新のテクノロジーについて試してみるではなく、 既存の J2EE 1.4 (もしくはそれ以前のバージョン) のアプリケーションを GlassFish 上で動かして、マイグレーションも試してみてください。
  • GlassFish のスタートガイドをチェックしてみてください。このページには、GlassFish サーバーをインストールして実行するための手順に加えて、簡単なサンプルアプリケーションの配備方法も載っています。
  • Java EE 5 に関して、Java BluePrints ホームページもチェックしてみてください。このページには、Java EE 5 プラットフォーム上で AJAX アプリケーションをビルドする際のガイドラインが載っています。
  • NetBeans の簡単な "hello world" アプリケーション、あるいは Java EE 5 Persistence を使ったより高度なアプリケーションを使って、Java EE アプリケーションを開発してみてください。
  • javadocs はオンラインドキュメントです。  ビルドごとに定期的に更新されます。
  • フォーラム上で質問してください。
  • GlassFish で Java EE 5 テクノロジーを用いた例については、growing list of tech tips and blogs をチェックしてみてください。
  • GlassFish 関連のテクノロジーの最新情報については The Aquarium をチェックしてみてください。
