GlassFish est un serveur d'application open source libre, permettant d'implémenter les fonctionnalités les plus récentes dans la plate-forme Java EE 5. La plate-forme Java EE 5 inclut les dernières versions de technologie, telles que JavaServer Pages(JSP) 2.1, JavaServer Faces(JSF) 1.2, Servlet 2.5, Enterprise JavaBeans 3.0, Java API for Web Services(JAX-WS) 2.0, Java Architecture for XML Binding(JAXB) 2.0, Web Services Metadata for the Java Platform 1.0, et bien d'autres.


Vous ne connaissez pas encore GlassFish ? Apprenez comment le télécharger et l'installer, consultez son guide de démarrage rapide, visualisez des screencasts (captures vidéo) ou testez des exemples du didacticiel. Consultez la page de présentation de GlassFish.

Téléchargement et exécution

Vous pouvez télécharger la version stable ou la toute dernière version d'accès du serveur d'application GlassFish. Le téléchargement comprend une version binaire du serveur d'application GlassFish Java EE 5, il vous suffit donc de suivre les instructions données et de l'exécuter. Les versions du téléchargement sont compatibles sur tous les systèmes d'exploitation standard.
  • Téléchargez maintenant

Apprentissage et écriture d'applications avec les dernières technologies Java EE 5

Le serveur d'application GlassFish implémente les technologies Java EE 5 les plus récentes. Par exemple, vous pouvez écrire une application à l'aide des dernières technologies tiers Web, telles que JSF1.2. En outre, vous avez la possibilité d'exploiter vos applications J2EE 1.4 (ou version antérieure) existantes sur GlassFish et d'établir de bonnes bases pour la migration.
  • Découvrez le guide de démarrage rapide de GlassFish. Celui-ci comprend les instructions d'installation et d'exécution du serveur GlassFish, ainsi qu'un exemple simple d'application pour un déploiement.
  • Consultez la page d'accueil de Java BluePrints pour les tâches associées à Java EE 5. Celle-ci contient notamment des informations relatives à la création d'applications AJAX sur la plate-forme Java EE 5.
  • Développez des applications Java EE à l'aide de NetBeans en suivant l'un de ces deux didacticiels, une simple application hello world et une application plus complexe utilisant la persistance Java EE 5.
  • Les documents Java sont en ligne. Ils ont été mis à jour pour chaque version promue.
  • Postez toutes vos questions sur notre forum.
  • Consultez la liste croissante d'astuces et de blogs techniques, comportant des exemples d'utilisation des technologies Java EE 5 dans GlassFish.
  • Consultez The Aquarium pour découvrir toutes les actualités relatives aux technologies associées à GlassFish.
