Para suprimir un listener HTTP

  1. En el árbol de navegación, amplíe el nodo Configuración.

  2. En el nodo Configuración, amplíe el nodo Servicio HTTP.

  3. En el nodo Configuración de Red, seleccione el nodo Listeners HTTP.

    Se abrirá la página Listeners HTTP.

  4. En la página Listeners HTTP, active la casilla de control situada a la izquierda del listener HTTP que desee suprimir.

  5. Haga clic en Suprimir.
