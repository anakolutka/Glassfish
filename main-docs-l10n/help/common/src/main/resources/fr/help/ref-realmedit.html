Modifier le domaine

La page Modifier le domaine permet de modifier un domaine.

Elle contient les options ci-après.

Nom de configuration

Nom de la configuration à laquelle s'appliquent les paramètres de cette page. Ce champ est en lecture seule.

Nom de domaine

Nom du domaine. Ce champ est en lecture seule. Vous ne pouvez spécifier le nom qu'à la création du domaine.

Nom de classe

Nom de classe du domaine que vous modifiez.

Autres propriétés

Autres propriétés du domaine. Les propriétés valides dépendent du type de domaine.

  • Propriétés propres à la classe FileRealm

  • Propriétés propres à la classe CertificateRealm

  • Propriétés propres à la classe JDBCRealm

  • Propriétés propres à la classe LDAPRealm

  • Propriétés propres à la classe SolarisRealm

  • Propriétés propres à la classe PamRealm

Tâches connexes

  • Création d'un domaine

  • Modification d'un domaine

  • Suppression d'un domaine

  • Création d'un domaine personnalisé

Commandes asadmin connexes

  • create-auth-realm(1)

  • delete-auth-realm(1)

  • list-auth-realms(1)
