Impostazioni percorso JVM

Utilizzare la pagina Impostazioni percorso JVM per configurare il percorso della libreria nativa. Il percorso della libreria nativa è una concatenazione dei percorsi seguenti:

  • Prefisso del percorso della libreria nativa

  • Il percorso del server per le proprie librerie native (ovvero as-install/lib)

  • Le directory della libreria nativa a 64 bit, se applicabile

  • Il valore della variabile java.library.path per la JVM utilizzato per avviare GlassFish Server (varia in base al sistema operativo in uso)

  • Qualsiasi percorso specificato nella pagina Impostazioni profiler JVM, se il profiler è abilitato

  • Suffisso del percorso della libreria nativa

Altre impostazioni vengono visualizzate come valori di sola lettura per i domini aggiornati da una release precedente di GlassFish Server. Non sono supportate nella release v3.

La pagina Impostazioni percorso JVM contiene le opzioni riportate di seguito.

Nome configurazione

Il nome della configurazione a cui si applicano le impostazioni presenti in questa pagina. Questo campo è di sola lettura.

Classpath di ambiente

Se è selezionata la casella di controllo Ignora, la variabile di ambiente CLASSPATH viene ignorata. La variabile di ambiente CLASSPATH è utile per le esercitazioni di base di programmazione, ma non è consigliabile per gli ambienti enterprise. Questo campo è di sola lettura.

Classpath di sistema

Il percorso delle classi del sistema. Questo campo è di sola lettura.

Prefisso classpath

I percorsi completi dei file JAR da aggiungere come prefisso al classpath del server, uno per riga. Questo campo è di sola lettura.

Suffisso classpath

I percorsi completi dei file JAR da aggiungere come suffisso al classpath del server, uno per riga. Questo campo è di sola lettura.

Prefisso percorso di libreria nativa

I percorsi completi dei file JAR da aggiungere come prefisso al percorso della libreria nativa, uno per riga.

Suffisso percorso di libreria nativa

I percorsi completi dei file JAR da aggiungere come suffisso al percorso della libreria nativa, uno per riga.
