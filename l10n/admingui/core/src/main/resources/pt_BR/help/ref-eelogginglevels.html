Níveis de Log de Módulo

Use a página Níveis de Log de Módulo para configurar os níveis de log de módulos individuais.

Para cada módulo, a página Níveis de Log de Módulo contém as seguintes informações.

Nome do Agente de Log

O nome do agente de log do módulo (por exemplo, javax.enterprise.system.tools.admin .

Nível do Agente de Log

O nível de log atual do módulo.

Os seguintes níveis de log estão disponíveis. Eles estão listados do mais alto para o mais baixo.

SEVERE

Eventos que interferem com a execução normal do programa.

WARNING

Avisos, inclusive exceções.

INFO

Mensagens relacionadas à configuração do servidor ou status do servidor, excluindo erros.

CONFIG

Mensagens relacionadas à configuração do servidor.

FINE

Verbosidade mínima.

FINER

Verbosidade moderada.

FINEST

Verbosidade máxima.

OFF

Nenhuma mensagem de log.
