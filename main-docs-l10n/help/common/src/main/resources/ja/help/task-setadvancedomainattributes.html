ドメイン属性を設定するには

  1. ナビゲーション・ツリーで、「ドメイン」ノードを選択します。

    「ドメイン属性」ページが開きます。

  2. 「DAS起動後にコンソールをロード」フィールドで、ドロップダウン・リストから次のいずれかのオプションを選択します。

    使用状況ベース

    コンソールが過去24時間以内に使用された場合、またはドメイン管理サーバー(DAS)以外の他のサーバー・インスタンスが存在する場合、DAS起動後に管理コンソールをロードします。これはデフォルトです。

    常時

    DAS起動後に管理コンソールを常にロードします。コンソールは、GlassFish Serverの主要な起動順序の完了後にロードされます。

    なし

    DAS起動後に管理コンソールをロードしません。コンソールは、アクセスがリクエストされたときにロードされます。

  3. 「アプリケーション・ルート」フィールドに、デプロイされたアプリケーションが配置されたディレクトリのフルパスを入力します。

    デフォルト値はdomain-dir/applicationsです。

  4. 「ログ・ルート」フィールドに、サーバーのログ・ファイルを格納するディレクトリのフルパスを入力します。

    デフォルト値はdomain-dir/logsです。

  5. 「ロケール」フィールドを空白のままにして、ホストのデフォルト・ロケールを使用するようにします。

    ロケールは、言語と地域の特定の組合せを指定する識別子です。たとえば、米国英語のロケールはen_USで、日本語のロケールはja_JPです。英語以外のロケールを使用するには、英語を目的の言語に翻訳して、GlassFish Serverをローカライズする必要があります。

  6. 「保存」をクリックします。

  7. サーバーを再起動します。
