응용 프로그램 편집

응용 프로그램 편집 페이지에서는 기존 응용 프로그램을 편집할 수 있습니다.

응용 프로그램 편집 페이지에는 다음 옵션이 포함되어 있습니다.

이름

편집 중인 응용 프로그램의 이름을 표시하는 읽기 전용 필드입니다.

가상 서버

이 응용 프로그램에 연결된 가상 서버입니다.

컨텍스트 루트

웹 응용 프로그램의 경우 응용 프로그램 경로를 지정합니다. 웹 응용 프로그램의 URL에서 컨텍스트 루트 바로 앞에는 포트 번호(http://host:port /context-root/...)가 있습니다. 컨텍스트 루트는 슬래시로 시작해야 합니다(예: /hello).

설명

응용 프로그램에 대한 간단한 설명입니다.

상태

사용 가능 확인란이 선택되어 있으면 응용 프로그램을 사용할 수 있습니다. 이 옵션은 기본적으로 선택됩니다.

Java Web Start

사용 가능 확인란이 선택되어 있으면 응용 프로그램에서 Java Web Start 소프트웨어를 사용합니다. Java Web Start를 사용하면 전용 JVM(Java Virtual Machine)에서 실행되는 Java 응용 프로그램을 브라우저와는 상관 없이 배포할 수 있습니다.

디렉토리 배포

이 확인란이 선택되어 있으면 응용 프로그램이 디렉토리에서 배포됩니다.

위치

배포된 응용 프로그램의 위치를 표시하는 읽기 전용 필드입니다.

라이브러리

응용 프로그램에 필요한 라이브러리 JAR 파일을 표시하는 읽기 전용 필드입니다.

하위 모듈

응용 프로그램의 하위 모듈과 관련 엔진의 이름을 표시하는 테이블입니다.

관련 작업

  • 배포된 응용 프로그램을 편집하는 방법

  • 웹 응용 프로그램을 배포하는 방법

  • 엔터프라이즈 응용 프로그램을 배포하는 방법

  • 응용 프로그램 클라이언트를 배포하는 방법

  • 커넥터 모듈을 배포하는 방법

  • EJB Jar를 배포하는 방법

  • Ruby 응용 프로그램을 배포하는 방법

  • 응용 프로그램의 설명자를 보는 방법

  • 응용 프로그램을 시작하는 방법

  • 배포된 응용 프로그램을 재배포하는 방법

  • 배포된 응용 프로그램을 다시 시작하는 방법
