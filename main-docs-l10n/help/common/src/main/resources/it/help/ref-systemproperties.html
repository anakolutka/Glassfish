Proprietà di sistema

Utilizzare la pagina Proprietà di sistema per visualizzare e gestire le proprietà di GlassFish Server.

Una proprietà di sistema definisce un valore comune per un'impostazione a livello di server. È possibile fare riferimento a una proprietà di sistema in un campo di testo della console di amministrazione racchiudendola tra parentesi graffe dopo il simbolo del dollaro. Ad esempio, per fare riferimento a una proprietà denominata nome-prop, utilizzare la sintassi ${nome-prop}.

La pagina Proprietà di sistema contiene le informazioni riportate di seguito per ogni proprietà.

Nome

Il nome della proprietà.

Valore

Il valore della proprietà.

Descrizione

La descrizione facoltativa della proprietà.

Alcune proprietà di sistema sono predefinite, ma non vengono visualizzate in questa pagina. Per i dettagli, vedere Proprietà di sistema predefinite.
