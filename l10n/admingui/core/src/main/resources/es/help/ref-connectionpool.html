Conjunto de conexiones

La página Conjunto de conexiones contiene las siguientes opciones:

Total máximo pendiente

Especifica el número máximo de conexiones pendientes permitidas para una escucha HTTP. El valor predeterminado es 4096.

Tamaño de cola

Especifica el tamaño máximo, en bytes, de la cola de conexión. Este valor también especifica el número máximo de conexiones pendientes que puede tener el servidor. El valor predeterminado es 4096.

Tamaño del búfer de recepción

Especifica el tamaño, en bytes, del búfer de recepción para una escucha HTTP. El valor predeterminado es 4096.

Tamaño del búfer de envío

Especifica el tamaño, en bytes, del búfer de envío para una escucha HTTP. El valor predeterminado es 8192.
