Web アプリケーションを起動する

始める前に

起動する Web アプリケーションが配備されていることを確認してください。

  1. ナビゲーションツリーで、「アプリケーション」ノードを展開します。
  2. 「Web アプリケーション」ノードを選択します。

    「Web アプリケーション」ページが開きます。

  3. 「配備されている Web アプリケーション」の表で、起動するアプリケーションの行にある「起動」をクリックします。

    新規ブラウザウィンドウが開いて、アプリケーションが表示されます。
