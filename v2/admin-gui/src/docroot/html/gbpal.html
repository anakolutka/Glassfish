To purge messages in a JMS Physical Destination

  1. In the tree component, expand the Configurations node and then expand the Java Message Service node.
  2. Select the Physical Destinations node.
  3. On the Physical Destinations page, select the checkbox next to the name of the destination to be deleted.
  4. Click Flush.

    The messages in the specified physical destination will be purged.

More Information
Equivalent asadmin command

flush-jmsdest
