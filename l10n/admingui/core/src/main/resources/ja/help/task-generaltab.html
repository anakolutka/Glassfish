Enterprise Server の一般情報を表示する

  1. ナビゲーションツリーで、Enterprise Server ノードを選択します。

    「一般情報」ページが開きます。

  2. 「一般情報」ページで、次の手順のいずれかを実行します。
    • Enterprise Server を停止するには、「停止」ボタンをクリックします。
    • Enterprise Server を停止して再起動するには、「再起動」ボタンをクリックします。
    • サーバーログを表示するには、「ログファイルを表示」ボタンをクリックします。

      詳細は、「サーバーログを表示する」を参照してください。

    • 「JVM レポート」をクリックすると、Java 仮想マシンに関するレポート (概要レポート、メモリー管理レポート、クラスローディングレポート、現在のスレッドのダンプなど) を、新しいウィンドウに表示できます。
