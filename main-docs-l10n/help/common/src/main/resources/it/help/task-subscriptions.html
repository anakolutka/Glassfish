Per visualizzare la pagina Sottoscrizioni per GlassFish Server

  1. Selezionare il nodo Task comuni nell'albero di navigazione.

    Viene visualizzata la pagina Task comuni.

  2. In Notizie su GlassFish fare clic sul collegamento Sottoscrizioni.

    La pagina Web Sottoscrizioni di GlassFish viene visualizzata in una scheda o finestra distinta del browser.
