Contrat de licence

La page Contrat de licence affiche les contrats de licence de tous les composants sélectionnés pour être installés ou mis à jour. Le contrat de licence d'un composant est fourni avec ce dernier par le fabricant du composant. Différents composants dans le référentiel réseau peuvent être publiés sous différentes licences, par exemple :

  • Common Development and Distribution License (CDDL)

  • GNU General Public License (GPL)

  • Apache License

  • wxWindows Library Licence

La page Contrat de licence contient les options suivantes :

Accepter

Ce bouton permet d'accepter les conditions des licences de tous les composants sélectionnés, puis de poursuivre l'installation ou la mise à jour.

Refuser

Ce bouton permet de refuser les conditions des licences de tous les composants sélectionnés, puis de retourner sur la page sur laquelle vous avez sélectionné les composants.
