Surveillance d'application

La page Surveillance d'application permet d'afficher les données de surveillance d'une application sélectionnée.

Elle contient les informations ci-après.

Nom d'instance

Nom de l'instance de serveur.

Application

Application pour laquelle les statistiques de surveillance sont affichées.

Composant

Composant d'application pour lequel les statistiques de surveillance sont affichées.

Pour obtenir une description des propriétés surveillées, reportez-vous au chapitre 8, Administering the Monitoring Service, du manuel Oracle GlassFish Server Administration Guide.
