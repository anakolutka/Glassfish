JVM-Profilereinstellungen

Ein Profiler erzeugt Daten, die zur Analyse der Leistung sowie zur Ermittlung von potenziellen Engpässen verwendet werden können.

Geben Sie die Profilereinstellungen an. Die eingegebenen Informationen richten sich danach, welchen Profiler Sie verwenden. Beispiele und Anweisungen finden Sie in Kapitel 3, Debugging Applications in Sun GlassFish Enterprise Server v3 Prelude Developer’s Guide.

Verwandte Aufgaben

  • So konfigurieren Sie die allgemeinen JVM-Einstellungen

  • So konfigurieren Sie die JVM-Pfadeinstellungen

  • So konfigurieren Sie die JVM-Optionen

  • So deaktivieren Sie den Sicherheitsmanager

  • So konfigurieren Sie die JVM-Profilereinstellungen

  • So legen Sie die erweiterte Anwendungskonfiguration fest

  • So legen Sie die erweiterten Domänenattribute fest

Verwandte asadmin-Befehle

  • create-profiler(1)

  • delete-profiler(1)
