GlassFish est un serveur d'application open source libre, permettant d'implémenter les fonctionnalités les plus récentes dans la plate-forme Java EE 5. La plate-forme Java EE 5 inclut les dernières versions de technologie, telles que JavaServer Pages(JSP) 2.1, JavaServer Faces(JSF) 1.2, Servlet 2,4, Enterprise JavaBeans 3.0, Java API for Web Services(JAX-WS) 2.0, Java Architecture for XML Binding(JAXB) 2.0, Web Services Metadata for the Java Platform 1.0, et bien d'autres.

Vous ne connaissez pas encore GlassFish ? Apprenez comment le télécharger et l'installer, consultez son guide de démarrage rapide, visualisez des screencasts (captures vidéo) ou testez des exemples du didacticiel. Consultez la page de mise en route.

Téléchargement et exécution

Vous pouvez télécharger des versions d'accès récentes du serveur d'application GlassFish. Le téléchargement comprend une version binaire du serveur d'application GlassFish Java EE 5, il vous suffit donc de le décompresser et de l'exécuter. Les versions du téléchargement sont compatibles sur tous les systèmes d'exploitation standard.
