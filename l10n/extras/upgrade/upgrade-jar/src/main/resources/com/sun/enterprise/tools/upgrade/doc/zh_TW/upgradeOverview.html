升級工具：概述

使用【升級工具】可將配置資訊從 Application Server 8.x 或 Application Server 9 安裝傳送到 Application Server 9.1 安裝。傳送的資訊包括，有關已部署應用程式、檔案範圍、安全性憑證以及其他資源和伺服器配置設定的資料。

此工具會自動偵測指定來源伺服器安裝的版本。

注意：在開始升級程序之前，請確定來源伺服器 (要升級的伺服器) 和目標伺服器 (要升級至的伺服器) 均已停止。

升級術語

以下是與升級程序相關的重要術語：

  • 來源伺服器：要升級到新版本的安裝。

  • 目標伺服器：要升級至的安裝。

  • 網域根目錄：建立網域的目錄。此目錄預設為 asenv.conf 中 AS_DEF_DOMAINS_PATH 所指定 的位置

  • 網域目錄或 domain-dir：與特定網域對應的目錄 (位於網域根目錄內)。與網域相關的所有配置和其他資料都位於此目錄中。

  • 安裝根目錄：安裝 Application Server 的目錄。

如果在 Application Server 安裝程序中選取了【升級】核取方塊，則在安裝完成之後將自動顯示【升級精靈】螢幕。升級程序即開始執行。
