Novo Alias de Senha

Utilize a página Novo Alias de Senha para criar um novo alias de senha para ser utilizado em um arquivo de senha ou no arquivo de configuração do domínio.

A página Novo Alias de Senha contém as seguintes opções:

Nome do Alias

O nome do alias. Este nome deve ser exclusivo em todos os aliases do domínio.

Senha

A senha que o alias de senha armazena na forma criptografada.

Confirmar Senha

A senha que o alias de senha armazena na forma criptografada. Os valores de Senha e Confirmar senha são comparados para garantir que a senha tenha sido introduzida sem erros.
