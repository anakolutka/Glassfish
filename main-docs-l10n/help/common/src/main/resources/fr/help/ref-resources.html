Ressources

La page Ressources permet de configurer les ressources suivantes pour le cluster ou l'instance GlassFish Server sélectionné :

  • JDBC

  • Connecteurs

  • Configurations d'adaptateurs de ressources

  • Ressources JMS

  • Sessions JavaMail

  • JNDI

  • Ressources simultanées

La page Ressources contient également l'option suivante :

Ajouter des ressources

Bouton permettant de créer des ressources à partir d'un fichier glassfish-resources.xml.
