HTTP-Protokoll

Die Seite "HTTP-Protokoll" umfasst die folgenden Optionen:

Version

Die Version des verwendeten HTTP-Protokolls (HTTP/1.0 oder HTTP/1.1). Der Standardwert ist HTTP/1.1.

DNS-Suche aktiviert

Wenn Sie diese Option auswählen, wird die DNS-Eintrag-Suche für den Client aktiviert. Diese Option ist standardmäßig deaktiviert.

SSL aktiviert

Wenn Sie diese Option aktivieren, wird SSL für alle Zielgeräte verwendet, bei denen die Sicherheitsfunktion aktiviert ist. Wenn Sie diese Option deaktivieren, wird die Sicherheitsfunktion im Server global deaktiviert. Diese Option ist standardmäßig aktiviert.

Erzwungener Typ

Der Antworttyp, der verwendet wird, wenn keine MIME-Zuordnung für die Erweiterung verfügbar ist. Der Standardwert lautet text/html; charset=iso-8859-1.

Standardtyp

Der Standardwert lautet text/html; charset=iso-8859-1. Der Wert ist eine durch Semikolon getrennte Zeichenfolge bestehend aus content-type , encoding, language und charset.
