Para editar um módulo do ciclo de vida

  1. Na árvore de navegação, selecione o nó Módulos do Ciclo de Vida.

    A página Módulos do Ciclo de Vida é exibida.

  2. Na página Módulos do Ciclo de Vida, clique no nome do módulo do ciclo de vida a ser editado.

    A página Editar Módulo do Ciclo de Vida é aberta.

  3. No campo Nome da Classe, digite o nome totalmente qualificado do arquivo de classe do módulo do ciclo de vida.

    A classe deve implementar a interface com.sun.appserv.server.LifecycleListener.

  4. No campo Caminho da classe, digite o caminho da classe do módulo do ciclo de vida.

    O caminho da classe especifica onde o módulo do ciclo de vida está localizado. O valor padrão é domain-dir/applications . Se o módulo já estiver neste diretório (isto é, no caminho da classe do servidor), esse campo poderá ser deixado em branco.

  5. No campo Ordem de Carregamento, digite um valor especificando a ordem na qual esse módulo do ciclo de vida deve ser carregado na inicialização.

    Os módulos com valores inteiros menores de ordem de carregamento são carregados antes. O valor pode variar de 101 até o MAXINT do sistema operacional. Valores de 1 a 100 são reservados.

  6. Marque a caixa de seleção Status Ativado para habilitar o módulo do ciclo de vida.

    Por padrão, a opção está ativada.

  7. Marque a caixa de seleção Inicialização de Instância de Prevenção de Falha de Carregamento para especificar que o servidor seja desligado, em caso de falha do carregamento do módulo do ciclo de vida.

    Por padrão, a opção está desativada.

  8. Na lista suspensa Servidores Virtuais, selecione o servidor ou servidores virtuais a serem associados com o módulo do ciclo de vida.

    Por padrão, nenhum servidor virtual é selecionado.

  9. No campo Descrição, digite uma descrição do módulo do ciclo de vida.
  10. (Opcional) Na seção Propriedades Adicionais, especifique essas propriedades.

    O Enterprise Server não define nenhuma propriedade adicional para os módulos do ciclo de vida.

    • Para adicionar uma propriedade, clique no botão Adicionar Propriedade. Na linha em branco que é exibida, digite o nome da propriedade no campo Nome e digite o valor da propriedade no campo Valor.
    • Para modificar uma propriedade, edite o campo Valor da propriedade.
    • Para excluir uma propriedade, marque a caixa de seleção à esquerda do campo Nome da propriedade a ser excluída. Clique no botão Excluir Propriedades.
  11. Clique em Salvar.

    Dica - Para restaurar as configurações padrão, clique no botão Carregar Padrões e depois em Salvar.
