HTTP サービスの監視データを表示する

  1. ナビゲーションツリーで、「Application Server」ノードを選択します。
  2. 「一般情報」ページで、「監視」タブをクリックします。
  3. 「Web コンテナの監視」ページで、「HTTP サービス」タブをクリックします。

    「HTTP サービスの監視」ページが開きます。

  4. (省略可能) 「仮想サーバー」ドロップダウンリストから、要求の統計情報を表示するサーバーを選択します。
  5. (省略可能) 「HTTP リスナー」ドロップダウンリストから、要求の統計情報を表示するリスナーを選択します。
  6. (省略可能) 表示されている監視データを更新するには、「更新」ボタンをクリックします。
