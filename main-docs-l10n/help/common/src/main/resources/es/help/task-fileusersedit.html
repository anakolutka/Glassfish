Editar información de usuario de dominio de archivo

  1. En el nodo Configuraciones, amplíe la configuración de instancia o cluster del servidor para el que desee gestionar los usuarios del dominio de administración.

  2. Amplíe el nodo Seguridad.

  3. Amplíe el nodo Dominios.

  4. Seleccione el nodo file.

    Se abrirá la página Editar Dominio.

  5. En la página Editar Dominio, haga clic en el botón Gestionar Usuarios.

    Se abrirá la página Usuarios de Archivo.

  6. En la página Usuarios de Archivo, en la columna ID de Usuario de la tabla, haga clic en el nombre del usuario que desee modificar.

    Se abrirá la página Editar Usuario de Dominio de Archivo.

  7. Edite el campo Lista de Grupos para modificar los grupos a los que pertenece el usuario.

    Utilice comas para separar los nombres de grupo. Estos grupos no hace falta definirlos en otro lugar.

  8. Escriba una nueva contraseña en los campos Contraseña Nueva y Confirmar Contraseña Nueva para modificar la contraseña del usuario.

  9. Haga clic en Guardar.
