Per creare un modulo del ciclo di vita

  1. Selezionare il nodo Moduli del ciclo di vita nell'albero di navigazione.

    Viene visualizzata la pagina Moduli del ciclo di vita.

  2. Nella pagina Moduli del ciclo di vita fare clic su Nuovo.

    Viene visualizzata la pagina Nuovo modulo del ciclo di vita.

  3. Nel campo Nome digitare il nome per il modulo del ciclo di vita.

  4. Nel campo Nome classe digitare il nome completamente qualificato del file di classe del modulo del ciclo di vita.

    La classe deve implementare l'interfaccia com.sun.appserv.server.LifecycleListener.

  5. Nel campo Classpath digitare il classpath per il modulo del ciclo di vita.

    Il classpath specifica la posizione del modulo del ciclo di vita. Il valore predefinito è domain-dir/applications. Se il modulo si trova già in questa directory (ovvero nel classpath del server), questo campo può essere lasciato vuoto.

  6. Nel campo Ordine di caricamento digitare un valore per specificare l'ordine di caricamento del modulo del ciclo di vita all'avvio.

    I moduli con valori di ordine di caricamento interi più bassi vengono caricati per primi. Il valore può essere compreso tra 101 e il valore MAXINT del sistema operativo. I valori compresi tra 1 e 100 sono riservati.

  7. Nel campo Descrizione digitare la descrizione del modulo del ciclo di vita.

  8. Selezionare la casella di controllo Abilitato dell'opzione Stato per abilitare il modulo del ciclo di vita.

    Questa opzione è abilitata per impostazione predefinita.

  9. Selezionare la casella di controllo In caso di errore di caricamento: Impedisci avvio istanza per impostare la chiusura del server se il caricamento del modulo del ciclo di vita non riesce.

    Questa opzione è disabilitata per impostazione predefinita.

  10. Per aggiungere istanze server e cluster di destinazione in cui distribuire il modulo del ciclo di vita, selezionarli dalla lista Destinazioni disponibili e utilizzare il pulsante Aggiungi per spostarli nella lista Destinazioni selezionate.

    L'opzione Destinazione viene visualizzata se esistono cluster o istanze server standalone oltre all'istanza server predefinita (server). Per impostazione predefinita, il modulo del ciclo di vita viene abilitato nell'istanza server predefinita, denominata server. Utilizzare il pulsante Aggiungi tutto per eseguire la distribuzione in tutte le destinazioni. Utilizzare i pulsanti Rimuovi e Rimuovi tutto per rimuovere le destinazioni dalla lista Destinazioni selezionate.

  11. Fare clic su OK.
