Valeurs de l'instance

La page Valeurs de l'instance permet de modifier les valeurs d'une propriété système de configuration pour les instances GlassFish Server qui référencent la configuration nommée sélectionnée.

La page Valeurs de l'instance affiche la liste des instances qui référencent la configuration sélectionnée dans laquelle la propriété est définie.

Nom d'instance

Nom affecté à l'instance lors de sa création. Lorsque vous cliquez sur le nom, la page Informations générales de l'instance apparaît.

Nom du cluster

Nom du cluster dont l'instance est membre, le cas échéant. Ce champ est en lecture seule.

Valeur par défaut

Valeur définie pour la propriété dans la configuration nommée que l'instance référence. Ce champ est en lecture seule.

Valeur de remplacement

Valeur de la propriété définie pour l'instance sélectionnée. Cette valeur remplace la valeur par défaut.
