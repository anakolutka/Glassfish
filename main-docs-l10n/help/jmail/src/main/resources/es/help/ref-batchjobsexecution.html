Detalles de ejecución de trabajo por lotes

En la página Detalles de ejecución de trabajo por lotes se muestran detalles sobre la ejecución de un determinado trabajo por lotes. Aparece la siguiente información.

Nombre de la instancia
Nombre del cluster

Nombre de la instancia de servidor o cluster de GlassFish al que se aplica la configuración.

Nombre del trabajo

Nombre del trabajo por lotes.

ID de ejecución

ID asignado para la ejecución del trabajo por lotes. La primera vez que se inicia un trabajo y cada vez que se reinicia la ejecución existente, se crea una nueva ejecución.

Recuento de pasos

Número de pasos de la ejecución del trabajo por lotes.

Estado de lote

Estado de la ejecución del trabajo por lotes tal y como lo ha definido el tiempo de ejecución por lotes.

Estado de salida

Estado de la ejecución del trabajo por lotes tal y como lo ha definido el XML de trabajo o la aplicación por lotes. Por defecto, el estado de salida y el estado de lotes son el mismo a menos que el estado de salida se sustituya de forma explícita.

Hora de inicio

Hora a la que se inicia la ejecución del trabajo por lotes.

Hora de finalización

Hora a la que termina la ejecución del trabajo por lotes.

Parámetros de trabajo

Propiedades transferidas al tiempo de ejecución por lotes para la ejecución del trabajo por lotes, mostradas como pares de nombre/valor.
