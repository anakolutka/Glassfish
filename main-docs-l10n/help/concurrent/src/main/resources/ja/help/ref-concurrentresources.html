並行処理リソース

次のタイプの並行処理リソースを構成するには、「並行処理リソース」ページを使用します。

  • コンテキスト・サービス

  • 管理対象スレッド・ファクトリ

  • 管理対象エグゼキュータ・サービス

  • 管理対象スケジュール済エグゼキュータ・サービス

同時リソースは、JSR 236仕様(Concurrency Utilities for Java EE)に定義されているように、Java EEアプリケーションに同時実行機能を提供する管理対象オブジェクトです。

GlassFish Serverでは、同時リソースを構成し、サーブレットやEJBなどのアプリケーション・コンポーネントが使用できるようにします。同時リソースには、JNDIルックアップやリソース・インジェクションを使用してアクセスします。
