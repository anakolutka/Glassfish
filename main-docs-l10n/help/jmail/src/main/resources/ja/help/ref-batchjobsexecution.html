バッチ・ジョブ実行の詳細

バッチ・ジョブ実行の詳細ページには、特定のバッチ・ジョブの実行に関する詳細が表示されます。次の情報が表示されます。

インスタンス名
クラスタ名

構成が適用されるGlassFishクラスタまたはサーバー・インスタンスの名前。

ジョブ名

バッチ・ジョブの名前。

実行ID

バッチ・ジョブの実行に割り当てられたID。新しい実行は、ジョブが初めて開始されたとき、および既存の実行が再開されたときに作成されます。

ステップ数

バッチ・ジョブ実行のステップ数。

バッチ・ステータス

バッチ・ランタイムにより設定されたとおりのバッチ・ジョブ実行のステータス。

終了ステータス

ジョブのジョブXMLまたはバッチ・アプリケーションにより設定されたとおりのバッチ・ジョブ実行のステータス。終了ステータスが明示的にオーバーライドされないかぎり、デフォルトでは、終了ステータスとバッチ・ステータスは同一です。

開始時間

バッチ・ジョブ実行の開始時間。

終了時間

バッチ・ジョブ実行の終了時間。

ジョブ・パラメータ

バッチ・ジョブ実行のバッチ・ランタイムに渡されるプロパティで、名前と値のペアとしてリストされます。
