HTTP プロトコル

「HTTP プロトコル」ページには、次のオプションが含まれます。

バージョン

使用する HTTP プロトコルのバージョン (HTTP/1.0、HTTP/1.1 のいずれか) 。デフォルト値は HTTP/1.1 です。

DNS 検索

このオプションを選択すると、クライアントの DNS エントリの検索が有効になります。このオプションは、デフォルトでは選択されていません。

SSL を有効

このオプションを選択すると、セキュリティーが有効になっているリスナーに SSL が使用されます。このオプションが選択解除されていると、サーバー全体でセキュリティーが無効になります。これは標準でオンになっています。

強制タイプ

拡張子に一致する利用可能な MIME マッピングが見つからない場合に使用する応答タイプ。デフォルト値は、text/html; charset=iso-8859-1 です。

デフォルトタイプ

デフォルト値は、text/html; charset=iso-8859-1 です。値は、 コンテンツタイプ、 エンコーディング、 言語、および文字セットから構成される、セミコロンで区切られた文字列になります。
