Módulos de Auditoría

Utilice la página Módulos de auditoría para configurar los módulos de auditoría.

La auditoría es el modo en que GlassFish Server registra ciertos eventos, como errores o violaciones de seguridad. GlassFish Server registra todos los eventos de autenticación en archivos log. Un log de acceso completo ofrece una pista secuencial de los eventos de acceso de GlassFish Server.

En cada módulo de auditoría se proporciona la siguiente información.

Nombre

Nombre del módulo de auditoría.

Nombre de clase

Nombre completo de la clase que implanta este módulo. El nombre de clase para el módulo de auditoría por defecto es com.sun.enterprise.security.Audit.

La tabla Módulos también contiene las siguientes opciones.

Nuevo

Botón para crear un nuevo módulo de auditoría.

Suprimir

Botón para eliminar uno o más módulos de auditoría seleccionados.
