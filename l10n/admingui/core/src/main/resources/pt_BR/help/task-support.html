Para exibir a página Notícias do GlassFish do Sun GlassFish Enterprise Server

A página Notícias do GlassFish traz uma coleção de links para informações adicionais sobre o Sun GlassFish Enterprise Server, como newsletters, artigos, dicas técnicas, fóruns de discussão e blogs relevantes.

  1. Na árvore de navegação, selecione o nó Notícias do GlassFish.

    A página Notícias do GlassFish é exibida.

  2. Clique em qualquer link para exibir uma página em uma guia ou janela separada do navegador.
