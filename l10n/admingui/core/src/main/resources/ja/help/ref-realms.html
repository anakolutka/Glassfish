レルム

Enterprise Server には、file、certificate、および admin-realm の 3 つのレルムが事前に設定されています。さらに、ldap、JDBC、solaris、またはカスタムレルムも作成できます。通常、サーバーにはタイプによってレルムが 1 つずつありますが、Enterprise Server の場合は、file および admin-realm の 2 つの file レルムがあります。これらは、異なる 2 つの目的に使用される同じタイプの 2 つのレルムです。また、システムの各仮想サーバーで異なる証明書データベースを持つこともできます。

「レルム」ページでは、使用可能なレルムのリストが表示されます。レルムごとに、次の情報が表示されます。

  • 名前

  • クラス名
