So wird eine Ressourcenadapterkonfiguration gelöscht

  1. Blenden Sie in der Navigationsbaumstruktur den Knoten "Ressourcen" ein.

  2. Wählen Sie unter dem Ressourcenknoten den Knoten "Ressourcenadapterkonfigurationen".

    Die Seite "Ressourcenadapterkonfigurationen" wird geöffnet.

  3. Aktivieren Sie auf der Seite "Ressourcenadapterkonfigurationen" das Kontrollkästchen neben dem Namen der zu löschenden Ressourcenadapterkonfiguration.

  4. Klicken Sie auf "Löschen".
