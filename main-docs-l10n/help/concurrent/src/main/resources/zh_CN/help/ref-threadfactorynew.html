新建托管线程工厂

使用“新建托管线程工厂”页可创建托管线程工厂资源。

“新建托管线程工厂”页包含以下选项。

JNDI 名称

标识资源的唯一名称。

上下文信息

资源的上下文：类加载器、JNDI、安全性或工作区。

状态

资源的状态。可以启用或禁用资源。默认值为“已启用”。

线程优先级

要为托管线程工厂实例创建的线程优先级。默认值为 5。

说明

此资源的说明。

其他属性

资源的其他属性。GlassFish Server 不定义托管线程工厂资源的任何其他属性。

相关的 asadmin 命令
