La comunidad GlassFish está creando software para empresas gratuito, de código abierto, con calidad de producción. Los principales elementos que se distribuyen son: un servidor de aplicaciones, la implementación de referencia de Java EE 5, y la implementación de referencia de Java Persistence API, TopLink Essentials. La comunidad también ofrece Depósito Maven, herramientas y otras muchas cosas.

Tecnología de servlets SIP de código abierto, proyecto SailFin, para GlassFish
SIP (protocolo de iniciación de sesión) está detrás de numerosos servicios muy conocidos que disfrutamos hoy en día, como el servicio telefónico voz sobre IP (VoIP), la mensajería instantánea, la gestión de listas de presencia y de conocidos, o la conferencia en Web. El proyecto SailFin agrega tecnología de servlets SIP al servidor de aplicaciones GlassFish. Si desea más información, visite https://sailfin.dev.java.net.

  
  
  
  
Procedimientos iniciales
Utilice GlassFish para saber más sobre las últimas tecnologías Java EE 5 y comenzar a crear aplicaciones usando el servidor de aplicaciones. Pruebe los ejemplos de GlassFish y de Blueprints para aprender mediante prácticas con Java EE 5. Sepa cómo migrar sus aplicaciones J2EE actuales a Java EE 5.
