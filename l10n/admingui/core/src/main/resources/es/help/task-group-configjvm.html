Configuración de la JVM

La máquina virtual de JavaTM (JVMTM) permite ejecutar código de bytes de Java en un equipo físico. Al ajustar los valores de configuración de JVM, mejora el rendimiento y se evitan errores de asignación de memoria.

La configuración de JVM incluye las siguientes tareas:

  • Cómo establecer la configuración general de JVM

  • Para establecer los valores de la ruta de JVM

  • Para configurar las opciones de JVM

  • Para deshabilitar el administrador de seguridad

  • Cómo configurar los valores del analizador JVM

  • Para establecer la configuración de aplicaciones avanzada

  • Para establecer los atributos de dominio avanzados
