JVM 프로필러 설정

프로필러 도구는 성능을 분석하고 잠재적 병목 현상을 식별하는 데 사용되는 데이터를 생성합니다.

프로필러 설정을 지정합니다. 지정하는 정보는 사용 중인 프로필러 제품에 따라 다릅니다. 예제 및 지침에 대해서는 Sun GlassFish Enterprise Server v3 Preview Application Development Guide의 4 장, Debugging Applications를 참조하십시오.
