Para exibir a página Inscrições do Sun GlassFish Enterprise Server

  1. Na árvore de navegação, selecione o nó Tarefas Comuns.

    A página Tarefas Comuns é exibida.

  2. Em Notícias do GlassFish, clique no link Inscrições.

    A página da Web Inscrições do GlassFish é exibida em uma guia ou janela separada do navegador.
