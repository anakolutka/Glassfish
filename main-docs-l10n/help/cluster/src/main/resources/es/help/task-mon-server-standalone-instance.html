Ver datos de supervisión de servidor virtual para una instancia de GlassFish Server independiente

Antes de Empezar

Para poder ver datos de supervisión, debe configurar el servicio de supervisión.

  1. En el árbol de navegación, seleccione el nodo Instancias Independientes.

    Se abrirá la página Instancias de Servidor Independientes.

  2. Seleccione la instancia para la cual desea ver los datos de supervisión de los servidores virtuales.

    Se abrirá la página Información General para la instancia.

  3. En la página Información General, haga clic en el separador Supervisar.

    Se abrirá la página Supervisión de Aplicación.

  4. Haga clic en el separador Servidor.

    Se abrirá la página Supervisión de Servidor.

  5. En la lista desplegable Ver, seleccione el componente para el cual desea ver los datos de supervisión.

  6. Para actualizar los datos de supervisión mostrados, haga clic en Refrescar.
