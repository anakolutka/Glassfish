Pour définir le fournisseur JACC actif

  1. Dans l'arborescence de navigation, développez le nœud Configuration.
  2. Sous le nœud Configuration, sélectionnez le nœud Sécurité.

    La page Sécurité s'ouvre.

  3. Dans la liste déroulante JACC, sélectionnez le fournisseur JACC à utiliser par le serveur.

    Par défaut, deux fournisseurs JACC sont disponibles, avec les noms par défaut et simple. Le fournisseur par défaut est default.

  4. Cliquez sur Enregistrer.
