So deaktivieren Sie den Sicherheitsmanager

Wenn Sie den Sicherheitsmanager von Enterprise Server deaktivieren, kann dies die Leistung für einige Anwendungstypen verbessern. Die Java-Funktionen für Autorisierung und Authentifizierung arbeiten auch dann noch, wenn der Sicherheitsmanager deaktiviert wurde. Sie können den Sicherheitsmanager in einer Entwicklungsumgebung deaktivieren, in einer Produktionsumgebung wird hiervon jedoch abgeraten.

  1. Wählen Sie in der Navigationsstruktur den Anwendungsserverknoten.
  2. Klicken Sie auf der Seite "Allgemeine Informationen" auf die Registerkarte "JVM-Einstellungen".
  3. Klicken Sie auf der Seite "Allgemeine JVM-Einstellungen" auf die Registerkarte "JVM-Optionen".

    Die Seite "JVM-Optionen" wird geöffnet.

  4. Aktivieren Sie das Kontrollkästchen neben dem Feld "Wert", das die Option -Djava.security.policy enthält.
  5. Klicken Sie auf „Löschen“.
  6. Klicken Sie auf "Speichern".
  7. Starten Sie den Server neu.
