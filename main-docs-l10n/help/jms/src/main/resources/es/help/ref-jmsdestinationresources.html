Recursos de destino JMS

Utilice la página Recursos de Destino JMS para configurar los recursos de destino JMS.

Para cada recurso de destino, aparecerá la siguiente información.

Nombre JNDI

Nombre único que identifica el recurso de destino.

Activado

(Esta columna sólo se muestra si la instancia del servidor por defecto, server, existe en el dominio).

Una marca de control si el recurso está activado, o una X si está desactivado.

Estado

(Esta columna sólo se muestra si en el dominio se ha creado algún cluster o instancia independiente).

Número de clusters e instancias independientes a los que se dirige el recurso y el número de estos destinos en los que el recurso está activado. Por ejemplo, "Activado en 2 de 4 de destinos" significa que el recurso se dirige a cuatro clusters e instancias independientes y que está activado en dos de estos cuatro destinos.

Tipo de recurso

Tipo de recurso del recurso de destino.

Descripción

Descripción del recurso de destino.

La tabla Recursos de Destino también contiene las siguientes opciones.

Nuevo

Botón para crear un nuevo recurso de destino.

Suprimir

Botón para suprimir uno o más recursos de destino seleccionados.

Activar

Botón para activar uno o más recursos de destino seleccionados.

Desactivar

Botón para desactivar uno o más recursos de destino seleccionados.
