Monitoramento da Aplicação

Use a página Monitoramento da Aplicação para exibir os dados de monitoramento de uma aplicação selecionada.

A página Monitoramento da Aplicação contém as seguintes informações.

Nome da Instância

O nome da instância do servidor.

Aplicação

A aplicação para a qual estatísticas de monitoramento são exibidas.

Componente

O componente da aplicação para a qual estatísticas de monitoramento são exibidas.

Para obter uma descrição das propriedades monitoradas, consulte Capítulo 8, Administering the Monitoring Service em Oracle GlassFish Server Administration Guide.
