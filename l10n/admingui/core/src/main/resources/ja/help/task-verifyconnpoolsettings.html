接続プール設定を検証する

  1. ナビゲーションツリーで、「リソース」ノードを展開します。
  2. 「リソース」ノードで、「JDBC」ノードを展開します。
  3. 「JDBC」ノードで、「接続プール」ノードを選択します。

    「接続プール」ページが開きます。

  4. 「接続プール」ページで、検証する接続プールの名前をクリックします。

    「接続プールを編集」ページが開きます。

  5. 「接続プールを編集」ページで、「Ping」ボタンをクリックします。

    管理コンソールがデータベースに接続を試みます。エラーメッセージが表示される場合は、データベースサーバーが再起動しているかどうかを確認します。
