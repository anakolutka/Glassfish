JDBC 연결 풀 등록 정보를 편집하는 방법

  1. 탐색 트리에서 자원 노드를 확장합니다.
  2. 자원 노드에서 JDBC 노드를 확장합니다.
  3. JDBC 노드에서 연결 풀 노드를 선택합니다.

    연결 풀 페이지가 열립니다.

  4. 편집할 연결 풀의 이름을 누릅니다.

    연결 풀 편집 페이지가 열립니다.

  5. 연결 풀 편집 페이지에서 추가 등록 정보 탭을 누릅니다.
  6. 등록 정보를 수정하려면 등록 정보의 값 필드를 편집합니다.
  7. 등록 정보를 추가하려면 다음을 수행합니다.
    1. 등록 정보 추가 버튼을 누릅니다.

      빈 행이 표시됩니다.

    2. 이름 필터에서 등록 정보 이름을 입력합니다.
    3. 값 필드에서 등록 정보 값을 입력합니다.
  8. 등록 정보를 삭제하는 방법:
    1. 삭제할 등록 정보의 이름 필드 왼쪽에 있는 확인란을 선택합니다.
    2. 등록 정보 삭제를 누릅니다.
  9. 저장을 누릅니다.
