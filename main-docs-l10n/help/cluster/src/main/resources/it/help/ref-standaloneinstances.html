Istanze server standalone

Utilizzare la pagina Istanze server standalone per creare e gestire le istanze standalone di GlassFish Server.

Nella pagina Istanze server standalone viene visualizzata la lista di tutte le istanze standalone del dominio. Per ogni istanza vengono fornite le informazioni riportate di seguito.

Nome

Il nome assegnato all'istanza quando è stata creata. Se si fa clic sul nome, viene visualizzata la pagina Informazioni generali per l'istanza.

Peso LB

Il numero intero che rappresenta il peso, o fattore ponderale, per il bilanciamento del carico dell'istanza.

Il fattore ponderale per il bilanciamento del carico determina la proporzione di tutte le richieste inviate al cluster che l'istanza dovrebbe elaborare.

Ad esempio, in un cluster a due istanze può essere necessaria un'istanza per elaborare una di quattro richieste e un'altra istanza per elaborarne tre di quattro. In questa situazione, impostare su 1 il peso dell'istanza che elabora una delle quattro richieste e su 3 il peso dell'altra istanza. Se si preferisce utilizzare valori percentuali, impostare i valori di peso delle istanze rispettivamente su 25 e 75.

Il valore di peso predefinito è 100.

Configurazione

La configurazione selezionata per l'istanza al momento della creazione dell'istanza. Se si fa clic sulla configurazione, viene visualizzata la pagina Configurazioni per l'istanza.

Nodo

Il nome del nodo in cui risiede l'istanza. Se si fa clic sul nome, viene visualizzata la pagina Modifica nodo per il nodo.

Stato

Indica se l'istanza è in esecuzione o non è in esecuzione.

La tabella Istanze server contiene inoltre le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare un'istanza standalone.

Elimina

Pulsante che consente di eliminare una o più istanze standalone selezionate.

Avvia

Pulsante che consente di avviare una o più istanze standalone selezionate.

Arresta

Pulsante che consente di arrestare una o più istanze standalone selezionate.
