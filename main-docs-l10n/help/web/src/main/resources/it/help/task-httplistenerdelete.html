Per eliminare un listener HTTP

  1. Espandere il nodo Configurazione nell'albero di navigazione.

  2. Nel nodo Configurazione espandere il nodo Servizio HTTP.

  3. Nel nodo Configurazione di rete selezionare il nodo Listener HTTP.

    Viene visualizzata la pagina Listener HTTP.

  4. Nella pagina Listener HTTP selezionare la casella di controllo a sinistra del listener HTTP da eliminare.

  5. Fare clic su Elimina.
