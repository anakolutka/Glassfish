將 HTTP 偵聽程式配置為使用 SSL

將 HTTP 偵聽程式配置為使用 SSL，讓 Enterprise Server 將特定連接埠連結至伺服器憑證。

  1. 在瀏覽樹狀結構中，展開 [配置] 節點。
  2. 在 [配置] 節點下，展開 [HTTP 服務] 節點。
  3. 在 [HTTP 服務] 節點下，選取 [HTTP 偵聽程式] 節點。
  4. 在 [HTTP 偵聽程式] 頁面上，按一下您要配置的偵聽程式名稱。
  5. 在 [編輯 HTTP 偵聽程式] 頁面上，按一下 [SSL] 標籤。
  6. 選取 [用戶端認證] 核取方塊，以強制用戶端在使用此偵聽程式時向伺服器進行自我認證。
  7. 在 [憑證暱稱] 欄位中，鍵入現有伺服器金鑰組與憑證的名稱。
  8. 選取 [啟用 SSL3] 核取方塊，可在偵聽程式上啟用 SSL3 安全性協定。
  9. 選取 [啟用 SSL2] 核取方塊，可在偵聽程式上啟用 SSL2 安全性協定。
  10. 選取 [啟用 TLS] 核取方塊，可在偵聽程式上啟用 TLS 安全性協定。
  11. 在 [密碼組] 區段中，選取協定可使用的密碼組，然後按一下 [增加] 按鈕。

    若要啟用所有的密碼組，請按一下 [全部加入] 按鈕。

  12. 按一下 [儲存]。
  13. 重新啟動伺服器。
