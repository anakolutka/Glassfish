更改管理员密码

  1. 在导航树中，选择 "Application Server" 节点。

    将打开“常规信息”页面。

  2. 选择“管理员密码”选项卡。

    此页面显示当前管理员帐户的用户 ID 和组名。默认情况下，用户 ID 为 anonymous，组名为 asadmin。

  3. 在“新密码”字段中，键入当前管理员帐户的新密码。
  4. 在“确认新密码”字段中，再次键入新密码。
  5. 单击“保存”。
