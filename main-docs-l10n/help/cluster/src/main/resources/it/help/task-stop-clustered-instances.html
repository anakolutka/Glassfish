Per arrestare le istanze di GlassFish Server in cluster

  1. Espandere il nodo Cluster nell'albero di navigazione.

  2. Selezionare il nodo per il cluster sotto il nodo Cluster espanso.

    Viene visualizzata la pagina Informazioni generali per il cluster.

  3. Nella pagina Informazioni generali, fare clic sulla scheda Istanze.

    Viene visualizzata la pagina Istanze server in cluster per il cluster.

  4. Per ogni istanza da arrestare, selezionare la casella di controllo in corrispondenza dell'istanza nella tabella Istanze server.

  5. Fare clic su Arresta.

    Viene richiesto di confermare l'arresto delle istanze.

  6. Confermare che si desidera arrestare le istanze.

    Lo stato di ogni istanza viene aggiornato nella tabella Istanze server quando l'istanza viene arrestata.
