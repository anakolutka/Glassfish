JMS 目标资源

使用“JMS 目标资源”页可以配置 JMS 目标资源。

对于每个目标资源，将提供以下信息。

JNDI 名称

标识目标资源的唯一名称。

已启用

(如果域中只有默认服务器实例 server，则显示此列。)

如果资源已启用，则显示复选标记，如果资源已禁用，则显示 X。

状态

(如果域中已创建任何集群或独立实例，则显示此列。)

资源定向到的集群和独立实例数以及在多少个这些目标上启用了资源。例如，“在 2 个 (共 4 个) 目标上启用”表示资源定向到四个集群和独立实例，并且在这四个目标中的两个目标上启用。

资源类型

目标资源的资源类型。

说明

目标资源的说明。

“目标资源”表还包含以下选项。

新建

用于创建新目标资源的按钮。

删除

用于删除一个或多个所选目标资源的按钮。

启用

用于启用一个或多个所选目标资源的按钮。

禁用

用于禁用一个或多个所选目标资源的按钮。
