GlassFish 是一款免費的開放原始碼應用程式伺服器，可實作 Java EE 5 平台中的最新功能。Java EE 5 平台包含最新版本的技術，如 JavaServer Pages (JSP) 2.1、JavaServer Faces (JSF) 1.2、Servlet 2.4、Enterprise JavaBeans 3.0、Java API for Web Services (JAX-WS) 2.0、Java Architecture for XML Binding (JAXB) 2.0、Web Services Metadata for the Java Platform 1.0 以及其他許多新技術。

您是 GlassFish 的新手嗎？瞭解如何下載並安裝 GlassFish、嘗試閱覽「入門指南」、查看某些螢幕擷取影片或嘗試指導中的某些範例。請查看入門頁面。
