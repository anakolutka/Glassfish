Para exibir o log do servidor

  1. Na árvore de navegação, selecione o nó Enterprise Server.

    A página Informações Gerais é exibida.

  2. Na página Informações Gerais, clique no botão Exibir Arquivos de Log.

    A janela Visualizador de Logs é exibida.

  3. Use a área Critérios de Pesquisa para personalizar e filtrar o visualizador de logs.
    • Clique em Pesquisa Avançada na área Critérios de Pesquisa para fazer ajustes adicionais ao visualizador de logs.
    • No campo Carimbo de Hora:
      • Para exibir as mensagens mais recentes, selecione Mais Recente (o padrão).
      • Para exibir as mensagens apenas de um determinado período de tempo, selecione Intervalo Específico e digite os valores de data e hora nos campos De e Para que aparecem.

        Para o valor Hora, a sintaxe deve ter a seguinte forma (SSS significa milissegundos):

        hh:mm:ss.SSS

        Por exemplo:

        17:10:00.000

        se o valor De for posterior ao valor Para, aparecerá uma mensagem de erro.

    • Na lista suspensa Nível de Log, selecione um nível de log para filtrar mensagens por nível de log. Marque a caixa de seleção rotulada “Não incluir mais mensagens graves” para exibir mensagens apenas no nível escolhido

      Por padrão, a exibição inclui todas as mensagens que aparecem no log do servidor, no nível de log escolhido e nos níveis mais graves.

      Para assegurar que as mensagens que você deseja exibir apareçam no log do servidor, primeiro defina os níveis de log apropriados na página Níveis de Log do Módulo. Ver Para configurar níveis de log

      Se você escolher filtrar as mensagens de log com base no nível de log, apenas as mensagens que correspondem aos critérios de filtro específicos serão mostradas. Contudo, essa filtragem não afeta quais mensagens são registradas no log do servidor.

  4. Se você clicou na Pesquisa Avançada na área Critérios de pesquisa, realize estas etapas.
    1. Na lista suspensa Agente de Log, selecione um ou mais espaços para nome para filtrar por módulo.

      Use shift-clique ou control-clique para escolher múltiplos espaços para nomes.

      Selecionar um espaço para nomes em um nível mais elevado seleciona todos os espaços para nomes abaixo dele. Por exemplo, selecionar javax.enterprise.system também seleciona os agentes de log de todos os módulos naquele espaço para nomes: javax.enterprise.system.core , javax.enterprise.system.tools.admin e assim por diante.

    2. No campo Agente de Log Personalizado, digite os nomes dos agentes de log, um por linha, para exibir mensagens de agentes de log específicos em um determinado aplicativo.

      Se o aplicativo tiver diversos módulos, você poderá exibir qualquer um deles. Por exemplo, suponha que o aplicativo tenha agentes de log com os seguintes nomes:

      com.mycompany.myapp.module1
com.mycompany.myapp.module2
com.mycompany.myapp.module3

      Para exibir as mensagens de todos os módulos no aplicativo, digite com.mycompany.myapp . Para exibir apenas mensagens do module2, digite com.mycompany.myapp.module2.

      Quando você especifica um ou mais agentes de logs, as mensagens dos módulos do Enterprise Server aparecem apenas se você as especificar explicitamente na área do Agente de Log.

    3. No campo Pares nome-valor, digite o nome e o valor da chave daquele segmento para exibir a saída de um segmento específico.

      O nome da chave é _ThreadID. Por exemplo:

      _ThreadID=13

      Suponha que com.mycompany.myapp.module2 é executado em diversos segmentos. Para refinar o visualizador de logs para mostrar apenas a saída de um único segmento, especifique agente de log daquele módulo no campo Agente de Log Personalizado e especifique o ID do segmento neste campo.

    4. Na lista suspensa Exibir, selecione um valor para exibir esse número de mensagens por vez.

      O número padrão de mensagens exibidas é 40. Outras opções são 100, 250 e 1.000.

      Para exibir os rastreamentos de pilha, desmarque a caixa de seleção “Limitar mensagens excessivamente longas”. Por padrão, os rastreamentos de pilha não aparecem no visualizador; para exibi-las, clique no link (detalhes) para obter uma mensagem.

    5. Clique em Pesquisa Básica para ocultar a área Opções Avançadas.
  5. Na área Resultados da Pesquisa, personalize a exibição das mensagens de log.

    A área Resultados da Pesquisa exibe as 40 entradas mais recentes no log do servidor, com as configurações especificadas nas páginas Configurações de Log e Níveis de Log.

    • Clique nas setas nas colunas para classificar as mensagens de formas diferentes.
    • Clique no link rotulado (detalhes) para exibir uma versão formatada de qualquer mensagem.

      Uma janela rotulada Detalhes da Entrada de Log é exibida, com uma versão formatada da mensagem.

    • Clique nos botões na parte superior e inferior da tabela para exibir as entradas mais antigas ou mais recentes do arquivo de log.
