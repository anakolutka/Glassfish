批次工作執行步驟

「批次工作執行步驟」頁面顯示與特定批次工作執行中的步驟有關的詳細資訊。會顯示以下資訊。

執行處理名稱
叢集名稱

要套用組態之 GlassFish 叢集或伺服器執行處理的名稱。

工作名稱

批次工作的名稱。

執行 ID

指定給批次工作之執行的 ID。在工作第一次啟動時及每次重新啟動現有的執行時，都會建立新的執行。

工作步驟

特定批次工作執行中的步驟。每個步驟均會顯示以下資訊：

步驟名稱

步驟的名稱。

批次狀態

批次執行階段設定的步驟狀態。

結束狀態

由工作的工作 XML 或批次應用程式設定的步驟狀態。根據預設，除非已明確置換結束狀態，否則結束狀態應與批次狀態相同。

起始時間

步驟的起始時間。

結束時間

步驟的完成時間。

步驟度量

步驟的度量，列出已讀取、已寫入、已確認等等的項目數。
