逻辑 JNDI 名称

创建资源时，为资源指定唯一 JNDI 名称。应用程序使用此名称访问资源。

Java EE 标准指定使特定的默认资源对应用程序可用，并为这些默认资源定义特定 JNDI 名称。GlassFish Server 通过使用逻辑 JNDI 名称使得这些名称可用，逻辑 JNDI 名称将 Java EE 标准 JNDI 名称映射到特定 GlassFish Server 资源，如下所示：

java:comp/DefaultContextService

此 Java EE 标准名称映射到 concurrent/__defaultContextService 资源。

java:comp/DefaultManagedThreadFactory

此 Java EE 标准名称映射到 concurrent/__defaultManagedThreadFactory 资源。

java:comp/DefaultManagedExecutorService

此 Java EE 标准名称映射到 concurrent/__defaultManagedExecutorService 资源。

java:comp/DefaultManagedScheduledExecutorService

此 Java EE 标准名称映射到 concurrent/__defaultManagedScheduledExecutorService 资源。

java:comp/DefaultDataSource

此 Java EE 标准名称映射到 jdbc/__default 资源。

java:comp/DefaultJMSConnectionFactory

此 Java EE 标准名称映射到 jms/__defaultConnectionFactory 资源。
