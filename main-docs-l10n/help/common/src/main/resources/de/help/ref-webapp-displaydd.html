Deployment-Deskriptor

Verwenden Sie die Seite "Deployment-Deskriptor", um den Deployment-Deskriptor für ein Modul anzuzeigen.

Die Seite "Deployment-Deskriptor" enthält die folgenden Informationen.

Anwendungsname

Der Name der Anwendung.

Der Name kann eine optionale Versions-ID enthalten, die dem Namen folgt und von diesem durch einen Doppelpunkt (:) getrennt ist. Weitere Informationen zu Modul- und Anwendungsversionen finden Sie unter Module and Application Versions im Oracle GlassFish Server Application Deployment Guide.

Modulname

Name des Moduls, das den Deployment-Deskriptor enthält.

Deskriptordateiname

Name der Deployment-Deskriptordatei.

Verwandte Aufgaben

  • So werden Deskriptoren einer Anwendung angezeigt

  • So wird eine Liste der bereitgestellten Anwendungen angezeigt

  • So wird eine Webanwendung bereitgestellt

  • So wird eine Unternehmensanwendung bereitgestellt

  • So wird ein Anwendungsclient bereitgestellt

  • So stellen Sie ein Connector-Modul bereit

  • So wird ein EJB-JAR bereitgestellt

  • So wird eine bereitgestellte Anwendung bearbeitet

  • So werden Anwendungsziele angezeigt

  • So werden End Point-Informationen für einen Webservice angezeigt

  • So wird eine Anwendung gestartet

  • So wird ein Anwendungsclient gestartet

  • So werden Anwendungsclient-Stubs heruntergeladen

  • So wird eine bereitgestellte Anwendung erneut bereitgestellt

  • So wird eine bereitgestellte Anwendung erneut geladen

  • So werden anwendungsspezifische Ressourcen angezeigt

  • So werden anwendungsspezifische Ressourcen geändert

Zugehöriger asadmin-Befehl

list-applications(1)
