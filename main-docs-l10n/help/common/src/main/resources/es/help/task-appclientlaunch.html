Iniciar un cliente de aplicaciones

Antes de Empezar

Compruebe que el cliente de aplicaciones que vaya a iniciar esté activado. Debe ser un cliente web que se pueda ejecutar con Java Web Start.

  1. En el árbol de navegación, seleccione el nodo Aplicaciones.

    Se abrirá la página Aplicaciones.

  2. En la tabla Aplicaciones desplegadas, haga clic en el nombre del cliente de aplicaciones que vaya a iniciar.

    El nombre puede incluir un identificador de versión opcional, que va a continuación del nombre y está separado de él por dos puntos (:). Para obtener más información sobre las versiones de módulo y de aplicación, consulte el apartado sobre las versiones de módulo y de aplicación en Oracle GlassFish Server Application Deployment Guide.

    Se abrirá la página Editar Aplicación.

  3. Compruebe que esté activada la casilla de control Java Web Start.

    La opción Java Web Start debe estar activada para poder ejecutar un cliente de aplicaciones. Si la casilla de control Java Web Start no está activada, actívela y haga clic en Guardar.

  4. En la tabla Módulos y Componentes, en la columna Acción, haga clic en Iniciar.

    Se abrirá la página Iniciar Cliente de Aplicaciones.

  5. En la página Iniciar cliente de aplicaciones, escriba los argumentos que se transferirán al cliente de aplicaciones.

  6. Haga clic en Iniciar.
