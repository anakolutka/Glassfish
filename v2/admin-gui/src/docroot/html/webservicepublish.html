To Publish a Web Service to a Registry

  1. In the tree view, expand the Web Services node and select the web service to publish.
  2. Click the Publish tab.
  3. Select the registry from the list of Available Registries list. Click Add.

    To select more than one registry from the list hold the control key while you select other registry names from the list.

    To publish to all the available registries, click the Add All button.

    All the selected registries to which the web service will be published is displayed in the Selected list.

    1. Enter categories under which this web service will show up. Use comma to separate each category. The categories are defined in the registry you are using.
    2. Enter a description for this web service. This description will help the user understand the service provided by this web service.
    3. Enter the name of the organization.
  4. Enter the Load Balancer host name, port number, and the SSL port number. If you are publishing the web service to an external registry, where the WSDL can be found over the internet, these options will replace the hostname and port name specified in the WSDL to the one of the load balancer.
