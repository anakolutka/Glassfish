配置 HTTP 服务连接池

  1. 在导航树中，展开“配置”节点。
  2. 在“配置”节点下，选择“HTTP 服务”节点。
  3. 从“HTTP 服务”页面中，单击“连接池”选项卡。

    此时将打开“连接池”页面。

  4. 在“最大未决计数”字段中，键入针对一个 HTTP 侦听器允许的最大未决连接数。

    默认值为 4096。

  5. 在“队列大小”字段中，键入连接队列的最大大小（以字节为单位）。

    此值还指定服务器可以拥有未解决的连接的最大数目。默认值为 4096。

  6. 在“接收缓冲区大小”字段中，键入 HTTP 侦听器的接收缓冲区的大小（以字节为单位）。

    默认值为 4096。

  7. 在“发送缓冲区大小”字段中，键入 HTTP 侦听器的发送缓冲区的大小（以字节为单位）

    默认值为 8192。

  8. 单击“保存”。

    提示 – 要返回到默认设置，请单击“装入默认值”按钮，然后单击“保存”。
