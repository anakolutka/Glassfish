監視服務頁面

使用 [監視服務] 頁面可配置個別伺服器元件或服務的監視選項。

JVM

JVM 監視層級。可以使用的層級為「低」、「高」和「關」。在此發行版本中 [低] 與 [高] 選項所提供的監視層級並無不同。

HTTP 服務

HTTP 服務監視層級。可以使用的層級為「低」、「高」和「關」。在此發行版本中 [低] 與 [高] 選項所提供的監視層級並無不同。

Web 容器

Web 容器監視層級。可以使用的層級為「低」、「高」和「關」。在此發行版本中 [低] 與 [高] 選項所提供的監視層級並無不同。

執行緒池

執行緒池監視層級。可用的層級為「低」、「高」和「關」。在此發行版本中 [低] 與 [高] 選項所提供的監視層級並無不同。
