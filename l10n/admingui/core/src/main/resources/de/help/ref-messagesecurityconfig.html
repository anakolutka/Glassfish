Nachrichtensicherheitskonfigurationen

Auf der Seite Nachrichtensicherheitskonfigurationen sind die verfügbaren Konfigurationen für Nachrichtensicherheitsanbieter aufgelistet.

Um Nachrichtensicherheit zu konfigurieren, benötigen Sie ein grundlegendes Verständnis von Sicherheits- und Webservice-Konzepten. Weitere Informationen finden Sie in Kapitel 13, Administering Message Security in Sun GlassFish Enterprise Server v3 Administration Guide und Kapitel 6, Administering Web Applications in Sun GlassFish Enterprise Server v3 Administration Guide.

Enterprise Server implementiert die Meldungssicherheit mithilfe von Meldungssicherheitsanbietern, die auf SOAP-Verarbeitungsebene integriert sind. Die Meldungssicherheitsanbieter hängen von den Sicherheitsfunktionen von Enterprise Server ab.

Für jede Konfiguration enthält die Seite Nachrichtensicherheitskonfigurationen die folgenden Informationen.

Authentifizierungs-Layer

Die Nachrichtenlayer, in der Authentifizierung durchgeführt wird. Standardmäßig wird eine SOAP-Konfiguration bereitgestellt. Es ist darüber hinaus möglich, eine HttpServlet-Konfiguration bereitzustellen.

Standardanbieter

Der Standard-Serveranbieter für diese Konfiguration. Standardmäßig ist kein Serveranbieter konfiguriert.

Standard-Clientanbieter

Der Standard-Clientanbieter für diese Konfiguration. Standardmäßig ist kein Standard-Clientanbieter konfiguriert.
