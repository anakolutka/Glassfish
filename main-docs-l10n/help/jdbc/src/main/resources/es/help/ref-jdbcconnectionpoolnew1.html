Nuevo pool de conexiones JDBC (paso 1 de 2)

Utilice la página Nuevo pool de conexiones JDBC (Paso 1 de 2) para crear un nuevo pool de conexiones JDBC.

La página Nuevo pool de conexiones JDBC (paso 1 de 2) contiene las siguientes opciones.

Nombre de pool

Nombre del pool de conexiones.

Tipo de recurso

Los tipos de recursos disponibles son javax.sql.XADataSource (transacciones globales), java.sql.ConnectionPoolDataSource (transacciones locales, posibles mejoras de rendimiento), javax.sql.DataSource (sólo transacciones locales) y java.sql.Driver.

Proveedor de Controladores de la Base de Datos

Nombre del proveedor del producto de la base de datos.

Introspección

Si se activa esta opción, el controlador de la base de datos se introspecciona en función del tipo de recurso y del proveedor de controladores de base de datos, y se muestra cualquier origen de datos o nombre de clase de controlador detectados en la página Nuevo pool de conexiones JDBC (paso 2 de 2) al hacer clic en Siguiente.

Para que la introspección funcione correctamente, el controlador JDBC debe estar disponible para GlassFish Server. Consulte Para integrar una controlador JDBC.
