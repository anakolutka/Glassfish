生命週期模組

[生命週期模組] 頁面會顯示可用生命週期模組的清單。

伺服器生命週期中的一個或多個事件觸發生命週期模組時，該模組便會執行作業。可能觸發該模組的伺服器事件包括：初始化、啟動、準備處理請求以及關機。生命週期模組不屬於 Java EE 規格，而是 Enterprise Server 的增強功能。

對於每個生命週期模組，均提供以下資訊。

名稱

生命週期模組的名稱。

已啟用

若已啟用生命週期模組，則為 True；若未啟用，則為 False。

載入順序

指定載入生命週期模組時，應採用的相對順序值。

類別名稱

生命週期模組的類別檔案。

  • 編輯生命週期模組

  • 刪除生命週期模組
