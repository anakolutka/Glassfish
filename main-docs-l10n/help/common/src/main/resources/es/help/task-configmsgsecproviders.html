Configurar utilidades de GlassFish Server para su uso por parte de proveedores de seguridad de mensajes

  1. Si utiliza un token de nombre de usuario, configure una base de datos de usuarios en caso de que sea necesario.

    Si utiliza un token de nombre de usuario/contraseña, primero tiene que configurar un dominio apropiado y una base de datos de usuarios apropiada para el dominio.

    La configuración de una base de datos de usuarios se aborda en Editar un dominio.

  2. Gestione certificados y claves privadas en caso necesario.

Pasos Siguientes

Una vez ha configurado las utilidades de GlassFish Server para que las utilicen los proveedores de seguridad de mensajes, los proveedores instalados con GlassFish Server pueden activarse tal como se describe en Editar una configuración de seguridad de mensajes.
