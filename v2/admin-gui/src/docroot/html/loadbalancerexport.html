To Export Load Balancer Configuration

  1. In the tree component, select the HTTP Load Balancers node.
  2. In the right pane, click the load balancer to edit.
  3. Click the Export tab.
  4. Click the Apply Changes Now button to apply changes to push any changes to the load balancer configuration to the load balancer. This needs to be done if you have not selected the Automatically Apply Changes check box in the General tab.
  5. Click Export Now button if you want to manually copy the loadbalancer.xml to the load balancer configuration directory.
