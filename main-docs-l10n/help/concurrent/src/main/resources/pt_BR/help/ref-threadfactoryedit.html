Editar Fábrica de Threads Gerenciados

Use a página Editar Fábrica de Threads Gerenciados para criar um recurso de fábrica de threads gerenciados.

A página Editar Fábrica de Threads Gerenciados contém as opções a seguir.

Carregar Defaults

Botão para restaurar definições para seus valores default. Definições que não tiveram seus valores default alterados.

Nome JNDI

Um nome exclusivo que identifica o recurso. O nome JNDI é um campo somente para leitura. Você pode especificar o nome JNDI ao criar um novo recurso.

Informações de Contexto

O contexto do recurso: Classloader, JNDI, Segurança ou Área de Trabalho.

Status

O status do recurso. O recurso pode ser ativado ou desativado. O valor default é Ativado.

Prioridade do Thread

A prioridade de threads a ser criada para a instância da fábrica de threads gerenciados. O default é 5 .

Descrição

Uma descrição deste recurso.

Propriedades Adicionais

Propriedades adicionais para o recurso personalizado. O GlassFish Server não define quaisquer propriedades adicionais para recursos da fábrica de threads gerenciados.
