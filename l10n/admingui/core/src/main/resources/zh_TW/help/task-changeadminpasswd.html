變更管理範圍中使用者的密碼

若要變更目前管理員帳號的密碼，請參閱變更管理員密碼。如果 admin-realm 有多個使用者，您也可以變更其他使用者的密碼。

將管理員密碼加密可增強應用程式的安全性。如需有關安全性程序的更多資訊，請參閱「Sun GlassFish Enterprise Server v3 Administration Guide」中的第 11 章「Administering System Security」。

  1. 在瀏覽樹狀結構中，展開 [配置] 節點。
  2. 展開 [安全性] 節點。
  3. 展開 [範圍] 節點。
  4. 選取 [admin-realm] 節點。

    [編輯範圍] 頁面會隨即開啟。

  5. 在 [編輯範圍] 頁面上，按一下 [管理使用者] 按鈕。

    [檔案使用者] 頁面會隨即開啟。

  6. 在 [檔案使用者] 頁面上，按一下您要為其增加或變更密碼的使用者。

    [編輯 File 範圍使用者] 頁面會隨即開啟。

  7. 在 [新密碼] 和 [確認新密碼] 欄位中，鍵入新密碼。
  8. 按一下 [儲存]。
