Per visualizzare la pagina Registrazione per GlassFish Server

La pagina Registrazione per GlassFish Server consente di registrare l'installazione di GlassFish Server.

Gli utenti registrati di GlassFish Server usufruiscono dei seguenti vantaggi:

  • notifica di nuove versioni, patch e aggiornamenti;

  • offerte speciali su prodotti, servizi e formazione per sviluppatori;

  • accesso a documentazione tecnica, articoli informativi e Webinar dal vivo con il team di GlassFish.

  1. Selezionare il nodo Task comuni nell'albero di navigazione.

    Viene visualizzata la pagina Task comuni.

  2. In Notizie su GlassFish fare clic sul collegamento Registrazione.

    La pagina Web Registrazione di GlassFish viene visualizzata in una scheda o finestra distinta del browser.
