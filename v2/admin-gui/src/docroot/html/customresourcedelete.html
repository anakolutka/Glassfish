To delete a custom resource

  1. Expand the Resources, JNDI, and Custom Resources node.

    If any custom resources have been created already, they are listed in the Resources table.

  2. From the Resources table, click in the box next to the name of the resource to be deleted.
  3. Click Delete.
More Information
Equivalent asadmin command

delete-custom-resource
