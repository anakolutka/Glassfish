To view, deploy, and manage EJB modules

  1. To control the level of detail you want to view for each deployed application, use the View drop-down list.
  2. To deploy an enterprise application, click Deploy. On the Deploy Enterprise Applications/Modules page, provide details of the application you want to deploy. See To deploy an EJB module.
  3. To undeploy an application, click the check box next to the required application and click Undeploy.
  4. To enable or disable an application or module on all targets, click the check box next to the required application and click Enable or Disable, as required.
  5. To make changes to a deployed application, click the required application from the list of deployed applications. For more details, see To edit a deployed EJB module.
  6. To redeploy an application that has been undeployed, click the Redeploy link. For more details, see To redeploy an application.
