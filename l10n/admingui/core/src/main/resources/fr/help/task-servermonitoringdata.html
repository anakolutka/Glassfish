Pour afficher les données de contrôle des serveurs

Avant de commencer

Avant d'afficher les données de contrôle, vous devez configurer le contrôle. Consultez Configurer le contrôle du service pour en savoir plus.

  1. Dans l'arborescence de navigation, sélectionnez le nœud Application Server.

    La page Informations générales s'affiche.

  2. Dans la page Informations générales, cliquez sur l'onglet Contrôle.

    La page Contrôle des applications s'ouvre.

  3. Cliquez sur l'onglet Serveur.

    La page Contrôle des serveurs s'ouvre.

  4. Dans la liste déroulante Serveur virtuel, sélectionnez le serveur pour lequel vous souhaitez afficher les données de contrôle.
  5. Cliquez sur le bouton Obtenir les statistiques pour récupérer les statistiques de contrôle.

    Les statistiques de contrôle s'affichent dans le tableau.

  6. (Facultatif) Cliquez sur le bouton Actualiser pour mettre à jour les données de contrôle affichées.
