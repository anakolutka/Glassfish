Informations générales

La page Informations générales contient les informations suivantes :

Nom

Le nom du serveur .

Statut

Le statut actuel du serveur. Le serveur peut être arrêté, démarré ou en cours d'exécution.

Équilibrage de charge HTTP

Le nom de l'équilibreur de charge, s'il est activé.

Port(s) HTTP

Ports actuellement configurés pour les requêtes et les réponses HTTP.

Port(s) IIOP

Ports actuellement configurés pour les requêtes et les réponses IIOP.

Répertoire de configuration

Répertoire de la machine hôte contenant le fichier de configuration, tel que domain.xml.

Version installée

Identifiant de la version du produit.

Débogage

Indique si le débogage est activé. Dans l'affirmative, le numéro de port est également affiché.
