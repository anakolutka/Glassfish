Modification des informations d'un utilisateur de domaine de fichiers

  1. Dans l'arborescence de navigation, développez le nœud Configuration.
  2. Développez le nœud Sécurité.
  3. Développez le nœud Domaines.
  4. Sélectionnez le nœud Fichier.

    La page Modification du domaine s'ouvre.

  5. Sur la page Modification du domaine, cliquez sur le bouton Gérer les utilisateurs.

    La page Utilisateurs de fichiers s'ouvre.

  6. Dans la colonne ID utilisateur du tableau de la page Utilisateurs de fichiers, cliquez sur le nom de l'utilisateur à modifier.

    La page Modification du domaine fichier s'ouvre.

  7. Modifiez le champ Liste de groupes pour modifier les groupes auxquels l'utilisateur appartient.

    Utilisez des virgules pour séparer les noms des groupes. Ces groupes n'ont pas besoin d'être définis ailleurs.

  8. Saisissez un nouveau mot de passe dans les champs Nouveau mot de passe et Confirmer nouveau mot de passe pour modifier le mot de passe de l'utilisateur.
  9. Cliquez sur Enregistrer.
