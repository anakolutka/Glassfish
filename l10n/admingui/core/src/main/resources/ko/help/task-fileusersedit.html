사용자 정보를 편집하는 방법

  1. 파일 사용자 페이지의 테이블에 있는 사용자 ID 열에서 수정할 사용자 이름을 선택합니다.

    파일 영역 사용자 편집 페이지가 열립니다.

  2. 비밀번호 필드와 비밀번호 확인 필드에 새 비밀번호를 입력하여 사용자 비밀번호를 변경합니다.
  3. 그룹 목록 필드에서 그룹을 추가하거나 삭제하여 사용자가 속한 그룹을 변경합니다.

    그룹 이름은 쉼표로 구분합니다. 그룹을 미리 정의할 필요는 없습니다.

  4. 이 사용자에 적용된 편집 내용을 저장하려면 저장을 선택하거나 저장 없이 종료하려면 뒤로를 선택합니다.
