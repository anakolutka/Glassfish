Alvos do Servidor Virtual

Utilize a página Alvos do Servidor Virtual para alterar os servidores virtuais do alvo aos quais uma aplicação ou módulo está associado.

A página Alvos do Servidor Virtual contém as seguintes informações.

Alvos Disponíveis

Servidores virtuais aos quais não há aplicações ou módulos associados.

Alvos Selecionados

Servidores virtuais aos quais uma aplicação ou módulo está associado.

Adicionar

Botão para mover o alvo selecionado de Alvos Disponíveis para Alvos Selecionados.

Adicionar Tudo

Botão para mover todos os Alvos Disponíveis para os Alvos Selecionados.

Remover

Botão para mover o alvo selecionado de Alvos Selecionados para Alvos Disponíveis.

Remover Tudo

Botão para mover todos os Alvos Selecionados para Alvos Disponíveis.
