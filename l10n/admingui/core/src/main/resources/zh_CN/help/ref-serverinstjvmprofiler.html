JVM 事件探查器设置

事件探查器工具可生成用于分析性能和识别潜在瓶颈的数据。

指定事件探查器设置。指定的信息取决于所使用的事件探查器产品。有关示例和说明，请参见《Sun GlassFish Enterprise Server v3 Preview Application Development Guide》中的第 4  章 "Debugging Applications"。
