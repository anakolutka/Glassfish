Manager-Eigenschaften

Mit dem Sitzungsmanager können Sie konfigurieren, wie Sitzungen erstellt und zerstört werden, wo der Sitzungsstatus gespeichert wird und wie viele Sitzungen maximal zulässig sind.

Auf der Seite "Manager-Eigenschaften" können Sie Sitzungsmanager-Eigenschaften hinzufügen oder ändern. Die Seite "Manager-Egenschaften" umfasst die folgenden Optionen:

Reap-Intervall

Die Anzahl der Sekunden, bis die Daten zur inaktiven Sitzung aus dem Speicher gelöscht werden.

Maximale Sitzungen

Die maximale Anzahl zulässiger Sitzungen.

Sitzungsdateiname

Der Name der Datei mit den Sitzungsdaten.

Klassenname für Sitzungs-ID-Generator

Der Klassenname für den ID-Generator, der im Enterprise Server-Klassenpfad angegeben wird.

Weitere Eigenschaften

Weitere Eigenschaften für den Sitzungsmanager.

  • Zum Hinzufügen einer Eigenschaft klicken Sie auf die Schaltfläche "Eigenschaft hinzufügen". Geben Sie in der daraufhin angezeigten leeren Zeile den Eigenschaftsnamen in das Feld "Name" und den Eigenschaftswert in das Feld "Wert" ein. Klicken Sie auf "Speichern".

  • Um eine Eigenschaft zu ändern, bearbeiten Sie das Feld "Wert" dieser Eigenschaft. Klicken Sie auf "Speichern".

  • Um eine Eigenschaft zu löschen, aktivieren Sie das Kontrollkästchen links neben dem Feld "Name" der zu löschenden Eigenschaft, und klicken Sie anschließend auf die Schaltfläche "Eigenschaften löschen". Klicken Sie auf "Speichern".
