Para Exibir as Atualizações Disponíveis para os Componentes

A página Atualizações Disponíveis fornece informações sobre as versões mais recentes de componentes da imagem da instalação que são instalados.

  1. Na árvore de navegação, selecione o nó Ferramenta de Atualização.

    É aberta a página Add-ons Disponíveis.

  2. Na página Add-ons Disponíveis, clique na guia Atualizações Disponíveis.

    É aberta a página Atualizações Disponíveis. Se não houver atualizações disponíveis, a tabela estará vazia.

Próximas Etapas

Para instalar as atualizações disponíveis, interrompa primeiro o domínio do GlassFish Server. A seguir, use um destes comandos:

  • as-install-parent/bin/updatetool, que inicia o utilitário gráfico da Ferramenta de Atualização

  • as-install-parent/bin/pkg, uma versão da linha de comandos da Ferramenta de Atualização

Para obter mais informações sobre o uso do comando pkg, consulte o Capítulo 11, Extending and Updating GlassFish Server, no Oracle GlassFish Server Administration Guide. O utilitário Ferramenta de Atualização tem sua própria ajuda on-line.

Reinicie o servidor após ter finalizado a instalação das atualizações.
