Pools de connexions du connecteur

La page Pools de connexions du connecteur permet de configurer les pools de connexions de connecteur.

Un pool de connexions de connecteur est un groupe de connexions réutilisables pour un système EIS donné.

Pour chaque ressource, les informations suivantes sont fournies.

Nom du pool

Nom unique identifiant le pool de connexions de connecteur.

Adaptateur de ressources

Adaptateur de ressources associé au pool de connexions de connecteur.

Définition de connexion

Nom de classe de la définition de connexion pour le pool de connexions de connecteur.

Description

Description du pool de connexions de connecteur.

La table Pools contient également les options ci-après.

Nouveau

Bouton permettant de créer un pool de connexions de connecteur.

Supprimer

Bouton permettant de supprimer les pools de connexions de connecteur sélectionnés.
