So wird die Registrierungsseite für GlassFish Server angezeigt

Auf der Registrierungsseite für GlassFish Server können Sie Ihre GlassFish Server-Installation registrieren.

Als registrierter Benutzer von GlassFish Server kommen Sie unter anderem in den Genuss der folgenden Vorteile:

  • Benachrichtigung über neue Versionen, Patches und Updates

  • Spezielle Angebote für Entwicklerprodukte, Services und Schulungen

  • Zugriff auf White Paper, informative Artikel und Live-Webinars mit dem GlassFish-Produktteam

  1. Wählen Sie in der Navigationsbaumstruktur den Knoten "Allgemeine Aufgaben".

    Die Seite "Allgemeine Aufgaben" wird geöffnet.

  2. Klicken Sie unter "Neuigkeiten zu GlassFish" auf den Link "Registrierung".

    Die Webseite für die GlassFish-Registrierung wird in einer separaten Registerkarte oder in einem Fenster Ihres Browsers geöffnet.
