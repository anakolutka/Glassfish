Conectores

Utilice la página Conectores para configurar los recursos de conectores de los siguientes tipos:

  • Recursos de conectores

  • Pools de Conexiones de Conectores

  • Recursos de Objeto Administrado

  • Asignaciones de Seguridad de Trabajo

Un módulo de conector (también llamado un adaptador de recursos) es un componente Java que permite a las aplicaciones interactuar con software de información de empresa (EIS). El software EIS incluye varios tipos de sistemas: planificación de recursos empresariales (ERP), procesamiento de transacciones mainframe, bases de datos no relacionales y sistemas de mensajería como el Java Message Service (JMS), entre otros.

Un pool de conexiones de conector es un grupo de conexiones reutilizables para un determinado EIS.

Un recurso de conector es un objeto de programa que proporciona una conexión a un EIS a una aplicación.

Un recurso de objeto administrado proporciona funcionalidades especializadas para una aplicación.

Una asignación de seguridad de trabajo asigna un elemento principal asociado a una instancia de trabajo entrante a un elemento principal del dominio de seguridad de GlassFish Server.
