Visualización de la Página de suscripciones de GlassFish de Sun Servidor empresarial

  1. En el árbol de navegación, seleccione el nodo Tareas comunes.

    Se abrirá la página Tareas comunes.

  2. En Noticias de GlassFish, haga clic en el enlace Suscripciones.

    Se abrirá la página web Suscripciones de GlassFish en una ficha o ventana del explorador independiente.
