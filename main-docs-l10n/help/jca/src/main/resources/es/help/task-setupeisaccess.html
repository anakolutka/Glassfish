Para configurar acceso de EIS

  1. Despliegue (instale) un conector. Consulte Cómo desplegar un módulo de conector.

  2. Cree un pool de conexiones para el conector. Consulte Para crear un pool de conexiones de conector.

  3. Cree un recurso de conector que esté asociado al pool de conexiones. Consulte Para crear un recurso de conectores.
