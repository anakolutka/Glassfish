가상 서버

가상 서버(가상 호스트라고도 함)는 동일한 물리적 서버에서 여러 개의 인터넷 도메인 이름을 호스트할 때 사용할 수 있는 객체입니다. 동일한 물리적 서버에서 호스트되는 모든 가상 서버는 해당 물리적 서버의 인터넷 프로토콜(IP) 주소를 공유합니다. 가상 서버는 서버의 도메인 이름(예: www.sun.com)을 Enterprise Server 가 실행되는 특정 서버에 연결합니다.

관련 작업

  • 가상 서버를 만드는 방법

  • 가상 서버를 편집하는 방법

  • 가상 서버를 삭제하는 방법

관련 asadmin 명령

  • create-virtual-server(1)

  • list-virtual-servers(1)

  • delete-virtual-server(1)
