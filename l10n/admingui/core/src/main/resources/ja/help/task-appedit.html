配備済みアプリケーションを編集する

  1. ナビゲーションツリーで、「アプリケーション」ノードを選択します。

    「アプリケーション」ページが開きます。

  2. 「配備されているアプリケーション」の表から、編集するアプリケーションの名前をクリックします。

    「アプリケーションを編集」ページが開きます。

  3. Web アプリケーションの場合、「コンテキストルート」フィールドにアプリケーションのパスを入力します。

    Web アプリケーションの URL では、コンテキストルートはポート番号の直後に続きます (http://host:port /context-root/...)。たとえば /hello のように、コンテキストルートがスラッシュで始まるようにしてください。

  4. 「説明」フィールドに、このアプリケーションの説明を入力します。
  5. 「状態を有効」チェックボックスをオフにしてアプリケーションの状態を無効にし、使用できないようにします。

    デフォルトでは、アプリケーションの状態は配備後すぐに「有効」になります。

  6. 「Java Web Start を有効」チェックボックスをオンにし、Java Web Start ソフトウェアを使用してアプリケーションを配備します。
  7. 「保存」をクリックします。
