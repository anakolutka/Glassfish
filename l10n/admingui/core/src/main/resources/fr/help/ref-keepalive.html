Keep-Alive

La page Keep-Alive contient les options suivantes :

Nombre de threads

Indique le nombre de threads Keep-Alive à utiliser. La valeur par défaut est 1.

Nb max. de connexions

Indique le nombre maximum de connexions persistantes à conserver. La valeur par défaut est 256.

Délai

Indique la durée maximale (en secondes) pendant laquelle la connexion Keep-Alive doit être maintenue. La valeur par défaut est 30.
