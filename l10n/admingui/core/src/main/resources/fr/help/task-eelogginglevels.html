Configurer les niveaux de journalisation

  1. Dans l'arborescence de navigation, développez le nœud Configuration.
  2. Sous le nœud Configuration, sélectionnez le nœud Paramètres du .

    La page Paramètres du s'ouvre.

  3. Sur la page Paramètres du , cliquez sur l'onglet Niveaux de journalisation.

    La page Niveaux de journalisation de module s'ouvre.

  4. Sur la page Niveaux de journalisation de module, modifiez les niveaux de journalisation pour un ou plusieurs modules, comme suit.
    • Pour modifier le niveau de journalisation d'un module individuel, sélectionnez une nouvelle valeur dans la liste déroulante à l'opposé du module dont vous voulez modifier le niveau de journalisation.
    • Pour modifier le niveau de journalisation d'un groupe de modules :
      1. Cochez les cases en regard des noms des modules dont vous voulez modifier le niveau de journalisation.
      2. Sélectionnez une valeur dans la liste déroulante en haut du tableau.
      3. Cliquez sur le bouton Modifier le niveau.

    Les messages correspondant au niveau sélectionné ou supérieur apparaissent dans le journal. Choisissez l'une des valeurs suivantes (classées de la plus élevée à la plus basse) :

    • GRAVE

    • WARNING

    • INFO

    • CONFIG

    • FINE

    • FINER

    • FINEST

    • Désactivé

    Pour en savoir plus sur la signification des niveaux de journalisation, consultez la rubrique Niveaux de journalisation des modules. Le niveau par défaut est INFO, donc par défaut, le journal contiendra des messages des niveaux INFO, AVERTISSEMENT ou GRAVE.

  5. Cliquez sur Enregistrer.

    Les modifications apportées aux niveaux de journalisation prennent effet immédiatement. Ils sont également sauvegardés pour être utilisés au redémarrage du serveur.
