Proveedores JACC

La página Proveedores JACC muestra una lista de proveedores de contratos de autorización Java para contenedores (JACC) disponibles.

JACC es la parte de la especificación de Java EE que define una interfaz para proveedores de autorización conectables. Esto permite al administrador instalar módulos de complementos de terceras partes para llevar a cabo la autorización. Por defecto Servidor empresarial ofrece un motor de autorización sencillo basado en archivos que cumple con JACC.

Para obtener información acerca de cómo escribir clases de implementación para un proveedor JACC, consulte la especificación JACC en http://jcp.org/en/jsr/detail?id=115.

Se proporciona la siguiente información para cada proveedor JACC.

Nombre

Nombre del proveedor JACC.

Proveedor de directivas

Nombre de la clase que implementa el generador de directivas.
