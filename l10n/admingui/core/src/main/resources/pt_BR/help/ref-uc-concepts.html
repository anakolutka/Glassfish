Principais conceitos do Ferramenta de Atualização

O Ferramenta de Atualização gerencia as instalações de componentes de software do Enterprise Server que estão contidos em um único diretório do sistema de arquivos do seu sistema.

Os principais conceitos do Ferramenta de Atualização estão descritos nas seções a seguir.

Componente

Um componente é a unidade instalável que o Ferramenta de Atualização gerencia. Um componente consiste em arquivos que são instalados como uma unidade no sistema, em uma imagem de instalação.

Imagem de instalação

Uma imagem de instalação é o diretório que contém os componentes de software, incluindo o conteúdo do diretório. A imagem de instalação é adicionada ao Ferramenta de Atualização como resultado do download e da instalação do software Enterprise Server. Para ativar a adição e a atualização dos componentes de software do Enterprise Server, cada imagem de instalação contém informações sobre a instalação de componentes.

Repositório de rede

Um repositório de rede contém estes itens:

  • Componentes que podem ser adicionados a uma imagem de instalação

  • Atualizações para componentes existentes em uma imagem de instalação

Uma imagem de instalação é associada a um ou mais repositórios de rede. Cada repositório de rede é representado por um nome e uma URL associada. Um nome de repositório é exclusivo para cada imagem de instalação e representa a organização ou o projeto que gerencia o repositório e um escopo bem definido de conteúdo.

Componente de suplemento

Um componente de suplemento é um componente da imagem de instalação que ainda não foi instalado.
