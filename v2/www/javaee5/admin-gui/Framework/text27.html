Building a Sample Application

  • viewDescriptor.xml

  • <!DOCTYPE displayViews SYSTEM "dtd/view.dtd"

  • [

  • <!ENTITY frameworkFunctions SYSTEM "view/HandlerDefinitions.xml">

  • <!ENTITY exampleFunctions SYSTEM "view/exampleFunctions.xml">

  • <!ENTITY exampleDisplayItems SYSTEM "view/exampleDisplayItems.xml">

  • <!ENTITY typeDefs SYSTEM "view/typedefs.xml">

  • ]>

  • <displayViews>

  • &typeDefs;

  • &frameworkFunctions;

  • &exampleFunctions;

  • &exampleDisplayItems;

  • </displayViews>


Notes:

This illustrates how you can use the XML parser and an entity resolver to delegate parts of your XML file to other files.
