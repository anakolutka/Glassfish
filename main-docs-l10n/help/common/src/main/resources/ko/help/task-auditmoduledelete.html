감사 모듈을 삭제하는 방법

  1. 탐색 트리에서 [구성] 노드를 확장합니다.

  2. 구성할 인스턴스를 선택합니다.

    • 특정 인스턴스를 구성하려면 해당 인스턴스의 구성 노드를 확장합니다. 예를 들어, 기본 인스턴스 서버에 대해 server-config 노드를 확장합니다.

    • 모든 인스턴스의 기본 설정을 구성하려면 default-config 노드를 확장합니다.

  3. [구성] 노드에서 [보안] 노드를 확장합니다.

  4. [보안] 노드 아래에서 [감사 모듈] 노드를 선택합니다.

    [감사 모듈] 페이지가 열립니다.

  5. [감사 모듈] 페이지에서 삭제하려는 감사 모듈의 왼쪽에 있는 체크 박스를 선택합니다.

  6. [삭제]를 누릅니다.

관련 항목

  • 감사 모듈을 생성하는 방법

  • 감사 모듈을 편집하는 방법

  • 감사 로깅을 사용 또는 사용 안함으로 설정하는 방법

  • 활성 감사 모듈을 설정하는 방법

  • 기본 감사 모듈을 사용하는 방법
