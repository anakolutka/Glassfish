Atualizando com o Upgrade Wizard

Use a tela Upgrade Wizard para especificar os diretórios de instalação dos servidores de origem e destino, o usuário de administração, a senha de administração e a senha principal do servidor de aplicativos de destino.

  1. No campo Source Installation Directory, insira o local da instalação existente a partir do qual importar a configuração.  É necessário inserir o diretório do domínio. Por exemplo, <install-root>/domains/domain1.

  2. No campo Target Installation Directory, insira o local da instalação do Application Server para o qual transferir a configuração. É necessário fornecer a raiz de domínios como a entrada deste campo.
  3. Forneça o nome de usuário de administração, a senha de administração e a senha principal do servidor de aplicativos de destino.
  4. Clique no botão Next ao terminar.
  5. O painel Upgrade Results é exibido, mostrando o status da operação de atualização.
  6. Clique no botão Finish para fechar a Upgrade Tool quando o processo de atualização tiver terminado.
