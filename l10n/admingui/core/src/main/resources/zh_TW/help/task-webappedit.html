編輯已部署的 Web 應用程式

  1. 在瀏覽樹狀結構中，展開 [應用程式] 節點。
  2. 選取 [Web 應用程式] 節點。
  3. 從 [已部署的 Web 應用程式] 表格中，按一下您要編輯的應用程式名稱。

    [Web 應用程式] 頁面會隨即開啟。

  4. 在 [環境根目錄] 欄位中，鍵入此 Web 應用程式的路徑。

    在 Web 應用程式的 URL 中，環境根目錄緊跟在連接埠號碼之後 (http://host: port/context-root/...)。請確保環境根目錄的開頭是正斜線，例如 /hello。

  5. 在 [說明] 欄位中，鍵入此 Web 應用程式的說明。
  6. 按一下 [儲存]。
