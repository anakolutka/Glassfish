Afficher les descripteurs d'une application Web

  1. Dans l'arborescence de navigation, développez le nœud Applications.
  2. Sous le nœud Applications, sélectionnez le nœud Applications Web.

    La page Applications Web s'ouvre.

  3. Sur la page Applications Web, cliquez sur le nom de l'application Web dont vous affichez les descripteurs.

    La page Application Web s'ouvre.

  4. Sur la page Application Web, cliquez sur l'onglet Descripteur.

    La page Personnalisation du descripteur de déploiement s'ouvre.

  5. Dans le champ Descripteurs de déploiement, cliquez sur le nom du fichier XML affiché.

    Le fichier XML s'ouvre dans une nouvelle page.

  6. Accédez au fichier, puis cliquez sur Fermer pour retourner sur la page Personnalisation du descripteur de déploiement.
