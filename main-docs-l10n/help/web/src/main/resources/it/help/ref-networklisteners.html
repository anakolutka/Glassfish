Listener di rete

Utilizzare la pagina Listener di rete per configurare i listener di rete.

Quando si esegue la prima installazione di GlassFish Server, per impostazione predefinita sono disponibili tre listener di rete denominati admin-listener, http-listener1 e http-listener2. Il listener http-listener2, configurato tramite SSL, è disabilitato per impostazione predefinita.

Per ogni listener di rete vengono fornite le informazioni riportate di seguito.

Nome

Il nome del listener di rete.

Porta n.

Il numero della porta di ascolto del listener di rete.

Protocollo

Il protocollo utilizzato dal listener di rete.

Pool di thread

Il pool di thread utilizzato dal listener di rete.

Abilitato

L'impostazione è True se il listener di rete è abilitato oppure è False se il listener di rete è disabilitato.

La tabella Listener di rete contiene inoltre le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare un nuovo listener di rete.

Elimina

Pulsante che consente di eliminare uno o più listener di rete selezionati.
