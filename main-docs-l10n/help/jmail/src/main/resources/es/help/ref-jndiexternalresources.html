Recursos Externos

Un recurso JNDI externo accede a un repositorio JNDI externo como, por ejemplo, un servidor LDAP.

Utilice la página Recursos externos para configurar recursos JNDI externos.

Para cada recurso externo, se proporciona la siguiente información.

Nombre JNDI

Nombre único que identifica el recurso externo.

Activado

(Esta columna sólo se muestra si la instancia del servidor por defecto, server, existe en el dominio).

Una marca de control si el recurso está activado, o una X si está desactivado.

Estado

(Esta columna sólo se muestra si en el dominio se ha creado algún cluster o instancia independiente).

Número de clusters e instancias independientes a los que se dirige el recurso y el número de estos destinos en los que el recurso está activado. Por ejemplo, "Activado en 2 de 4 de destinos" significa que el recurso se dirige a cuatro clusters e instancias independientes y que está activado en dos de estos cuatro destinos.

Tipo de recurso

Tipo de recurso externo.

Descripción

Descripción del recurso externo.

La tabla Recursos también contiene las siguientes opciones.

Nuevo

Botón para crear un nuevo recurso externo.

Suprimir

Botón para suprimir uno o más recursos externos seleccionados.

Activar

Para activar uno o más recursos externos seleccionados.

Desactivar

Botón para desactivar uno o más recursos externos seleccionados.
