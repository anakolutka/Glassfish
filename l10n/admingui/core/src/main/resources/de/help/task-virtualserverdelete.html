So löschen Sie einen virtuellen Server

  1. Erweitern Sie in der Navigationsstruktur den Konfigurationsknoten.
  2. Erweitern Sie unter dem Konfigurationsknoten den Knoten "HTTP-Dienst".
  3. Wählen Sie unter dem Knoten "HTTP-Dienst" den Knoten "Virtuelle Server" aus.
  4. Aktivieren Sie auf der Seite "Virtuelle Server" das Kontrollkästchen links neben dem Namen des zu löschenden Servers aus.
  5. Klicken Sie auf „Löschen“.

    Es ist möglich, den virtuellen Server __asadmin zu löschen, dies wird jedoch nicht empfohlen. Wenn Sie dies vorhaben, kopieren Sie zunächst die virtual-server-Elemente der domain.xml-Datei von Enterprise Server an einen sicheren Speicherort, damit diese Einstellung ggf. wiederhergestellt werden kann.
