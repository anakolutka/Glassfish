「セキュリティー証明書を転送」画面

「セキュリティー証明書を転送」画面を使用して、Sun Java Systems Application Server 9 への移行が必要なセキュリティー証明書の (1 つ以上の) セットを含むドメインを指定します。

  1. 「追加」ボタンを押して、「ドメインを追加」ダイアログを表示します。このダイアログで、ドメインの名前と証明書ファイルのパスワードを入力します。

    「Domains with Certificates to be Transferred」パネルに、追加したドメインが表示されます。

  2. この手順を繰り返して、転送が必要な証明書を含むすべてのドメインを追加します。
  3. 追加したドメインを削除するには、削除するドメインを選択し、「Delete Domain」ボタンを押します。
  4. 操作が完了したら、「次へ」ボタンを押します。

    「アップグレードの結果」画面に、アップグレードの状態が表示されます。
