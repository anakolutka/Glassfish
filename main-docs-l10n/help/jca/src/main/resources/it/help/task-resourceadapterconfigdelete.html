Per eliminare una configurazione Resource Adapter

  1. Espandere il nodo Risorse nell'albero di navigazione.

  2. Nel nodo Risorse selezionare il nodo Configurazioni Resource Adapter.

    Viene visualizzata la pagina Configurazioni Resource Adapter.

  3. Nella pagina Configurazioni Resource Adapter selezionare la casella di controllo in corrispondenza del nome della configurazione Resource Adapter da eliminare.

  4. Fare clic su Elimina.
