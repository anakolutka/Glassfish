To Configure GMS

Configuring the Group Management Service (GMS) for your environment determines how frequently GMS checks for failures.

  1. In the tree component, expand the Configurations node.
  2. Select the configuration to edit.
  3. Select the Group Management Service node.
  4. Set the required values for the following properties:
    Protocol Maximum Trial

    Maximum number of attempts to try before GMS confirms that a failure is suspected in the group.

    Protocol Timeout

    Period in milliseconds between monitoring attempts to detect failure.

    Maximum Interval

    Maximum amount of time in milliseconds to wait to collect sub-group information before performing a merge .

    Minimum Interval

    Minimum amount of time in milliseconds to wait to collect sub-group information before performing a merge.

    Ping Timeout

    Milliseconds Amount of time in milliseconds that GMS waits for discovery of other members in this group.

    Verified Timeout

    Interval in milliseconds after which a suspected failure is marked as verified.

  5. Set values for any additional properties that you require.

    Set the value of each additional property as follows:

    1. In the Additional Properties table, click Add Property.
    2. In the Name field, type the property name.
    3. In the Value field, type the value.
  6. Click Save.
