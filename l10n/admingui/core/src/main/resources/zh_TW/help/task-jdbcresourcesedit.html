編輯 JDBC 資源

  1. 在瀏覽樹狀結構中，展開 [資源] 節點。
  2. 選取 [JDBC 資源] 節點。
  3. 從 [JDBC 資源] 頁面中，按一下您要編輯的資源名稱。

    [編輯 JDBC 資源] 頁面會隨即開啟。

    備註 – [J.N.D.I. 名稱] 是唯讀欄位。您僅能在建立新的 JDBC 資源時指定 J.N.D.I. 名稱。

  4. 從 [池名稱] 下拉式清單中，選取要與此 JDBC 資源產生關聯的連線池。
  5. 在 [說明] 欄位中，鍵入此 JDBC 資源的簡短說明。
  6. 按一下 [儲存]。
