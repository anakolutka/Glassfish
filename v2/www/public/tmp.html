Instructions to unbundle and configure GlassFish

GlasssFish installation requires JDK 5 and ant 1.6.2 (included).
  1. Download one of the bundles to disk
  2. Run: 
    java -Xmx256m -jar filename.jar
  3. cd glassfish
  4. ant -f setup.xml

We recommend that you use the version of ant included in GlassFish from the \glassfish\lib\ant directory.

See the GlassFish Quick Start Guide for basic steps to start the server and deploy an application.
