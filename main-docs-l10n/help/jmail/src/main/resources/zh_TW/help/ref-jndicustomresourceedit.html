編輯自訂資源

使用「編輯自訂資源」頁面可編輯自訂 JNDI 資源。

「編輯自訂資源」頁面包含下列選項。

JNDI 名稱

可識別資源的唯一名稱。「JNDI 名稱」是唯讀欄位。您僅能在建立新的自訂資源時指定 JNDI 名稱。

資源類型

完全合格的資源類型定義。

工廠類別

使用者指定的工廠類別名稱。該類別實行 javax.naming.spi.ObjectFactory 介面。

建置順序

自訂資源的建置順序。

描述

此自訂資源的描述。

狀態

是否可在執行階段使用資源。

如果網域中只有預設伺服器執行處理 server 存在，可以選擇啟用或停用資源的執行階段可用性。預設會啟用執行階段可用性。

如果已在網域中建立任何叢集或獨立執行處理，可使用「目標」頁籤控制其目標之資源的執行階段可用性。

其他特性

自訂資源的其他特性。GlassFish Server 未定義自訂資源的任何其他特性。
