Para Criar um Serviço de Contexto

Um serviço de contexto é usado para criar objetos proxy dinâmicos que capturam o contexto de um contêiner e permitem que as aplicações sejam executadas dentro desse contexto posteriormente. O contexto do contêiner é propagado para o thread que executa a tarefa.

No GlassFish Server, você configura um recurso de serviço de contexto e o disponibiliza para uso por componentes da aplicação como servlets e EJBs.

  1. Na árvore de navegação, expanda o nó Recursos.

  2. No nó Recursos, expanda o nó Recursos Concorrentes.

  3. No nó Recursos Concorrentes, selecione Serviços de Contexto.

    O contexto Serviços página é aberta.

  4. Na página Serviços de Contexto, clique em Novo.

    A página Novo Serviço de Contexto é aberta.

  5. No campo Nome da JNDI, digite um nome exclusivo que identifique o recurso.

    O nome da JNDI é registrado no serviço de atribuição de nomes JNDI e usado por aplicações para pesquisar o recurso. Use o formato concurrent/name. Por exemplo, concurrent/myContextService. O nome só pode conter caracteres alfanuméricos, sublinhado, traço ou ponto.

  6. Marque a caixa de seleção Informações de Contexto Ativadas para propagar contextos a threads, e selecione os contextos que você deseja propagar.

    Esta opção é ativada e todos os contextos são selecionadas por default. Use Shift-clique ou Control-clique para escolher vários contextos. Se essa opção estiver desativada, os contextos não serão propagados, mesmo que eles estejam selecionados.

  7. No campo Descrição, digite uma descrição curta do recurso.

    Esta descrição é um valor de string e pode incluir no máximo 250 caracteres.

  8. Marque a caixa de seleção Status ativo para ativar o recurso.

    Esta opção é ativada por default, o que significa que o recurso fica disponível assim que é criado.

  9. Na seção Propriedades Adicionais, não adicione propriedades.

    O GlassFish Server não define propriedades adicionais para este tipo de recurso. Além disso, este tipo de recurso não usa atualmente propriedades adicionais.

  10. Clique em OK.
