新規管理対象スケジュール済エグゼキュータ・サービス

管理対象スケジュール済エグゼキュータ・サービス・リソースを作成するには、「新規管理対象スケジュール済エグゼキュータ・サービス」ページを使用します。

「新規管理対象スケジュール済エグゼキュータ・サービス」ページには、次のオプションが含まれます。

JNDI名

リソースを識別する一意の名前。

コンテキスト情報

リソースのコンテキスト: 「クラス・ローダー」、「JNDI」、「セキュリティ」、またはワークエリア

ステータス

リソースのステータス。リソースは、有効または無効の状態になります。デフォルト値は「有効」です。

スレッド優先度

管理対象スケジュール済エグゼキュータ・サービス・インスタンスに作成するスレッドの優先度。デフォルトは5です。

長時間実行中のタスク

XXX。

ハングまでの時間

タスクがハングしたとみなされるまで実行可能である時間(秒単位)。

コア・サイズ

スレッド・プールに保持されるスレッドの数。デフォルトは0です。

最大プール・サイズ

プールに含めることができるスレッドの最大数。デフォルト値は、2147483647です。

キープ・アライブ

スレッド数がコア・サイズよりも大きくなった場合にスレッドがアイドル状態でいられる秒数。デフォルトは60です。

スレッド存続時間

スレッド数がコア・サイズより大きいか否か、またはスレッドがアクティブまたはアイドルであるかにかかわらず、各スレッドがパージされるまでにスレッド・プール内に存続できる秒数。

タスク・キュー・キャパシティ

実行待ちの送信済タスクを保持するキューのサイズ。デフォルトは2147483647です。

説明

リソースの説明。

追加プロパティ

リソースの追加プロパティ。GlassFish Serverは管理対象スケジュール済エグゼキュータ・サービス・リソースの追加プロパティを定義しません。
