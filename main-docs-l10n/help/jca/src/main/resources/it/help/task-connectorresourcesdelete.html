Per eliminare una risorsa connettore

Una risorsa connettore viene eliminata in modo automatico se si elimina il connection pool del connettore ad essa associato.

  1. Espandere il nodo Risorse nell'albero di navigazione.

  2. Espandere il nodo Connettori.

  3. Nel nodo Connettori selezionare il nodo Risorse connettore.

    Viene visualizzata la pagina Risorse connettore.

  4. Nella pagina Risorse connettore selezionare la casella di controllo in corrispondenza del nome della risorsa da eliminare.

  5. Fare clic su Elimina.
