JMS Resources

This section describes how to configure resources for applications that use the Java Message Service (JMS) API.

The Application Server implements the JMS API by integrating the Sun Java System Message Queue (formerly Sun ONE Message Queue) software into the Application Server. For basic JMS API administration tasks, use the Application Server Admin Console. For advanced tasks, including administering a Message Queue cluster, use the tools provided in the MQ-install-dir/imq/bin directory.
