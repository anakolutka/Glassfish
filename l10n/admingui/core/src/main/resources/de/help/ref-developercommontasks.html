Allgemeine Aufgaben

Die Seite "Allgemeine Aufgaben" enthält Tastenkombinationen für allgemeine Administration Console-Aufgaben.

Führen Sie über die Seite "Allgemeine Aufgaben" die folgenden Operationen aus:

  • So registrieren Sie Sun GlassFish Enterprise Server für Sun Connection

  • So zeigen Sie die Supportseite für Sun GlassFish Enterprise Server an

  • So stellen Sie eine Webanwendung bereit

  • So erstellen Sie einen JDBC-Verbindungspool

  • To View Monitoring Data

  • So konfigurieren Sie den Überwachungsdienst

  • To View Installed Components

  • So aktualisieren Sie installierte Komponenten

  • So installieren Sie Add-On-Komponenten

Darüber hinaus finden Sie auf der Seite "Allgemeine Aufgaben" Verknüpfungen zur Enterprise Server-Dokumentation.
