Para editar un servicio de contexto

  1. En el árbol de navegación, amplíe el nodo Recursos.

  2. En el nodo Recursos, amplíe el nodo Recursos simultáneos.

  3. En el nodo Recursos simultáneos, seleccione Servicios de contexto.

    Se abrirá la página Servicios de contexto.

  4. En la página Servicios de contexto, haga clic en el nombre del recurso que desee editar.

    Se abrirá la página Editar servicio de contexto.

    Nota:

    El nombre JNDI es un campo de sólo lectura. Sólo puede especificar el nombre JNDI al crear un nuevo recurso.

  5. XXX

  6. En el campo Descripción, introduzca una breve descripción del recurso.

    Esta descripción es un valor de cadena y puede incluir un máximo de 250 caracteres.

  7. Active la casilla de control Estado activado para activar el recurso.

  8. En la sección Propiedades adicionales, especifique propiedades adicionales del recurso.

    ProductName; no define ninguna propiedad adicional para los recursos del servicio de contexto.

    • Para agregar una propiedad, haga clic en el botón Agregar Propiedad. En la fila en blanco que aparece, introduzca el nombre de propiedad en el campo Nombre y escriba el valor de propiedad en el campo Valor.

    • Para modificar una propiedad, edite el campo Valor.

    • Para suprimir una propiedad, active la casilla de control situada a la izquierda del campo Nombre de la propiedad que está suprimiendo y, a continuación, haga clic en Suprimir propiedades.

  9. Haga clic en Guardar.
