Ressources externes

Une ressource JNDI externe accède à un référentiel JNDI externe, tel qu'un serveur LDAP.

La page Ressources externes permet de configurer les ressources JNDI externes.

Pour chaque ressource externe, les informations suivantes sont fournies :

Nom JNDI

Nom unique identifiant la ressource externe.

Activé

(Cette colonne est affichée si seule l'instance de serveur par défaut server existe dans le domaine.)

Coche si la ressource est activée ou X si la ressource est désactivée.

Statut

(Cette colonne apparaît si des clusters ou des instances autonomes ont été créés dans le domaine.)

Nombre de clusters et d'instances autonomes sur lesquels la ressource est ciblée et nombre de cibles sur lesquelles la ressource est activée. Par exemple, Activé sur 2 cibles sur 4 signifie que la ressource est ciblée sur quatre clusters et instances autonomes et activée sur deux de ces quatre cibles.

Type de ressource

Type de la ressource externe.

Description

Description de la ressource externe.

La table Ressources contient également les options suivantes :

Nouveau

Bouton permettant de créer une ressource externe.

Supprimer

Bouton permettant de supprimer les ressources externes sélectionnées.

Activer

Bouton permettant d'activer les ressources externes sélectionnées.

Désactiver

Bouton permettant de désactiver les ressources externes sélectionnées.
