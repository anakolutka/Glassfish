Cómo configurar los valores del analizador JVM

Una herramienta de tipo analizador genera datos que son utilizados para analizar el rendimiento e identificar atascos potenciales.

  1. En el árbol de navegación, seleccione el nodo Configuración.
  2. En el nodo Configuración, seleccione el nodo Configuración de JVM.

    Se abrirá la página Configuración general de JVM.

  3. En la página Configuración general de JVM, haga clic en la ficha Profiler.

    Se abrirá la página Valores de JVM Profiler.

  4. Especifique la configuración del analizador.

    La información que especifique dependerá del analizador que utilice. Para obtener ejemplos e instrucciones, consulte el Capítulo 4, Debugging Applications de Sun GlassFish Enterprise Server v3 Preview Application Development Guide.

  5. Haga clic en Guardar.
  6. Reinicie el servidor.
