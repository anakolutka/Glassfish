Nouvelle fabrique de threads gérés

Utilisez la page Nouvelle fabrique de threads gérés pour créer une ressource de fabrique de threads gérés.

Elle contient les options ci-après.

Nom JNDI

Nom unique identifiant la ressource.

Informations de contexte

Contexte de la ressource : chargeur de classe; JNDI, sécurité ou espace de travail.

Statut

Statut de la ressource. La ressource peut être activée ou désactivée. La valeur par défaut est Activé.

Priorité des threads

Priorité des threads à créer pour l'instance de fabrique de threads gérés. La valeur par défaut est 5.

Description

Description de la ressource.

Autres propriétés

Propriétés supplémentaires de la ressource. GlassFish Server ne définit aucune autre propriété pour les ressources de fabrique de threads gérés.

Commandes asadmin connexes
