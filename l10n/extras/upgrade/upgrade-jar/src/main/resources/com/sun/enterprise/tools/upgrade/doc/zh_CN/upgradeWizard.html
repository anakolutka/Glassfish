使用“升级向导”进行升级

使用“升级向导”屏幕可以指定源服务器和目标服务器的安装目录，以及源应用服务器的管理员用户、管理员密码和主密码。

  1. 在“源安装目录”字段中，输入要从中导入配置的现有安装的位置。您需要输入域目录。例如，<install-root>/domains/domain1。

  2. 在“目标安装目录”字段中，输入要向其中传输配置的 Application Server 安装的位置。您需要提供域根目录作为此字段的输入。
  3. 提供源应用服务器的管理员用户名、管理员密码和主密码。
  4. 完成后，请单击“下一步”按钮。
  5. 此时将显示“升级结果”面板，表明升级操作的状态。
  6. 当升级过程完成时，单击“完成”按钮关闭升级工具。
