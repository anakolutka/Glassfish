Página Configuración general JVM

La máquina virtual de Java (JVM) viene incluida en la plataforma Java, Standard Edition (plataforma Java SE), que necesita Servidor empresarial. La introducción de valores incorrectos para la configuración de JVM hará que no funcione el servidor; tiene que ir con cuidado a la hora de cambiar estos valores.

Inicio de Java

Nombre del directorio de instalación del software de Java SE, que necesita Servidor empresarial para poder ejecutarse. Para comprobar que la versión del software de Java SE que ha especificado sea compatible con esta versión, consulte Sun GlassFish Enterprise Server v3 Prelude Release Notes. Si introduce un nombre de directorio no existente o un directorio para una versión no compatible del software de Java SE, entonces Servidor empresarial no se podrá iniciar.

Opciones de Javac

Opciones de línea de comandos para el compilador del lenguaje de programación de Java. Servidor empresarial ejecuta el compilador cuando se implementan los componentes de Enterprise JavaBeansTM (EJBTM).

Depuración

Habilita la depuración con Java Platform Debugger Architecture (JPDA).

JPDA lo utilizan los desarrolladores de aplicaciones. Para obtener más información, consulte Sun GlassFish Enterprise Server v3 Prelude Developer’s Guide.

Opciones de depuración

Especifica las opciones de JPDA que enviará a JVM cuando se habilite la depuración.

Opciones de compilación RMI

Opciones de línea de comandos para el compilador rmic. Servidor empresarial ejecuta el compilador rmic cuando se implementan componentes EJB.

Preprocesador de código byte

Nombres de clases para el preprocesamiento de código byte. Utilice comas para separar los nombres de clase. Todas las clases deben implementar com.sun.appserv.Interfaz del preprocesador de código byte. Las clases se llaman en el orden especificado.
