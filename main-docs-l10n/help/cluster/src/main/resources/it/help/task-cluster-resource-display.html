Per elencare le risorse per un cluster

  1. Espandere il nodo Cluster nell'albero di navigazione.

  2. Selezionare il nodo per il cluster sotto il nodo Cluster espanso.

    Viene visualizzata la pagina Informazioni generali per il cluster.

  3. Nella pagina Informazioni generali fare clic sulla scheda Risorse.

    Viene visualizzata la pagina Risorse per il cluster.

  4. Per visualizzare solo le risorse di un tipo specifico, selezionare il tipo di risorsa da visualizzare dall'elenco a discesa Filtro.
