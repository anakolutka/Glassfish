To edit the JMX connector configuration

With the Edit JMX Connector screen, you can edit the configuration of the JSR–160 compliant JMX Connector.

  1. Expand the Configuration node, then the Admin Service node.
  2. Select the System JMX Connector.
  3. Edit the JMX General Settings as follows:

    • JMX Protocol: identifies the .protocol that this JMX connector supports.

    • JMXServiceURL: this read-only URL is a function of the protocol, port, and address, as defined by the JSR–160 1.0 Specification.

    • Security: select the enabled checkbox to indicate that transport layer security should be used in the JMX connector.

      If you enable transport layer security, fill out the SSL section by following the instructions in To configure security for the Admin Service's JMX connector.

    • Address: specify the IP address or host name. The name must contain only alphanumeric, underscore, dash, or dot characters.

    • Port: enter the port of the JMX connector server.

    • Realm Name: enter the name that represents the special administrative realm. All authentication is handled by this realm.

    • Client Hostname: enter the name of the server where the JMX connector is hosted.
