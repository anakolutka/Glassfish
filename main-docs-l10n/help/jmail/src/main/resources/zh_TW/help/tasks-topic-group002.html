檢視批次工作執行

  1. 在瀏覽樹狀結構中，選取您要檢視批次工作執行的 GlassFish Server 叢集或伺服器執行處理。

    此時將開啟「一般資訊」頁面。

  2. 選取「批次」頁籤。

    就會開啟「批次工作執行」頁面。

  3. 在「批次工作執行」頁面中，按一下執行 ID 可檢視與該特定執行有關的詳細資訊。

    就會開啟「批次工作執行詳細資訊」頁面。

  4. 若要檢視與批次工作執行中的步驟有關的詳細資訊，請按一下「執行步驟」頁籤。

    就會開啟「批次工作執行步驟」頁面。

  5. 按一下「返回」按鈕，即可回到「批次工作執行」頁面。
