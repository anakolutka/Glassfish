Nuevo servicio de ejecutor programado gestionado

Utilice la página Nuevo servicio de ejecutor programado gestionado para crear un recurso del servicio de ejecutor gestionado programado.

La página Nuevo servicio de ejecutor programado gestionado contiene las siguientes opciones.

Nombre JNDI

Nombre único que identifica el recurso.

Información de contexto

Contexto del recurso: classloader, JNDI, seguridad o área de trabajo.

Estado

Estado del recurso. El recurso se puede activar o desactivar. El valor por defecto es Activado.

Prioridad de threads

Prioridad de threads que se van a crear para la instancia del servicio de ejecutor programado gestionado. El valor por defecto es 5.

Tareas de larga ejecución

XXX.

Bloquear después

Tiempo en segundos que una tarea se puede ejecutar antes de se bloquee.

Tamaño principal

Número de threads que mantener en un pool de threads. El valor por defecto es 0.

Tamaño de Pool Máximo

Número máximo de threads que el pool puede contener. El valor por defecto es 2147483647.

Mantenimiento de conexiones

Tiempo en segundos que los threads puede permanecer inactivos cuando el número de threads es mayor que el tamaño principal. El valor por defecto es 60.

Duración de threads

Tiempo en segundos que cada thread permanece en el pool de threads antes de que se depure, independientemente de si el número de threads es mayor que el tamaño principal o si el thread está inactivo.

Capacidad de cola de tareas

Tamaño de la cola que contiene las tareas enviadas en espera de ser ejecutadas. El valor por defecto es 2147483647.

Descripción

Descripción de este recurso.

Propiedades Adicionales

Propiedades adicionales del recurso externo. GlassFish Server no define ninguna propiedad adicional para los recursos del servicio de ejecutor gestionado programado.
