Desactivar la Administración Segura

  1. En el árbol de navegación, seleccione el nodo del servidor (servidor de administración).

    Se abrirá la página Información General.

  2. En la página Información General, haga clic en el botón Administración Segura.

    Se abrirá la página Administración segura.

  3. Haga clic en Desactivar Administración Segura.

    Aparece un mensaje que indica que desactivar la administración segura reiniciará DAS de inmediato.

  4. Haga clic en Aceptar.

    Se desactiva la administración segura y se abre la página de reinicio del servidor.
