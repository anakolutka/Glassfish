Suppression d'une fabrique de threads gérés

  1. Dans l'arborescence de navigation, développez le noeud Ressources.

  2. Sous le noeud Ressources, développez le noeud Ressources simultanées.

  3. Sous le noeud Ressources simultanées, sélectionnez Fabriques de threads gérés.

    La page Fabriques de threads gérés est ouverte.

  4. Sur la page Fabriques de threads gérés, cochez la case située en regard du nom de la ressource à supprimer.

  5. Cliquez sur Supprimer.
