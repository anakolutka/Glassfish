Verifier

The Verifier enables you to confirm that your J2EE applications and components conform to the Java(tm) 2 Enterprise Edition Specifications. You may check the following types of items:

  • J2EE Application (EAR file)
  • Enterprise Bean (EJB JAR file)
  • Web Component (WAR file)
  • J2EE Application Client (JAR file)

The following general considerations apply to the verification of all the above items:

  1. If the item you wish to verify is not in the list, click Add and select the item with the file chooser.
  2. If you do not want to verify an item in the list, select the item and click Delete.
  3. Select the radio button that indicates how much verification information will be displayed: All Results, Failures Only, or Failures and Warnings Only.
  4. Click OK to verify the entries selected in the list labelled "Items to be verified."
  5. Select an item in the Results list to display more information in the Details text area.
