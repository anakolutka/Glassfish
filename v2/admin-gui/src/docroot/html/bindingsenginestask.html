To Manage Lifecycle States of JBI Components

  1. Select the JBI components whose lifecycle states you want to manage.
  2. From the Operations drop-down list, select the desired lifecycle state. The lifecycle states of a JBI component are:

    • Start: If a JBI component is shutdown or stopped, you can select Start to start or restart it.

    • Stop: Select Stop only if the JBI component has been started. See the component's documentation for information on when it is appropriate to stop a component.

    • Shutdown: Select Shutdown before you uninstall the JBI component.
