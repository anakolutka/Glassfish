System-Eigenschaften

Über die Seite System-Eigenschaften können Sie die Eigenschaften des Anwendungsservers anzeigen und verwalten.

Eine Systemeigenschaft definiert einen allgemeinen Wert für eine Einstellung auf Serverebene. Sie können eine Systemeigenschaft in einem Textfeld "Admin-Konsole" referenzieren, indem Sie diese mit Dollarzeichen und geschweiften Klammern umschließen. Um beispielsweise eine Eigenschaft namens prop-name zu referenzieren, würden Sie die Syntax ${prop-name} verwenden.

Für jede Eigenschaft enthält die Seite "Systemeigenschaften" die folgenden Informationen.

Name

Der Name der Eigenschaft.

Wert

Der Wert der Eigenschaft.

Description (Beschreibung)

Eine optionale Beschreibung der Eigenschaft.

Einige System-Eigenschaften werden im Voraus festgelegt, jedoch nicht auf dieser Seite angezeigt. Details siehe Vordefinierte Systemeigenschaften.
