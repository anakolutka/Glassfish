Para Configurar Acesso ao Banco de Dados

Para armazenar, organizar e recuperar dados, a maioria das aplicações usa bancos de dados relacionais.

O GlassFish Server inclui uma cópia do banco de dados Java DB e é pré-configurado com a capacidade de acessar esse banco de dados.

  1. Instale um produto de banco de dados compatível.

    Para obter uma lista de bancos de dados suportados, consulte Notas de Release do Oracle GlassFish Server.

  2. Instale um driver que suporte a API da JDBC (driver JDBC) do banco de dados.

    Verifique se o driver a ser utilizado para criar as conexões com bancos de dados está instalado em todos os servidores em que a fonte de dados será implantada.

  3. Certifique-se de o arquivo JAR do driver esteja acessível à instância do servidor do domínio.

    Consulte Para Integrar um Driver JDBC.

  4. Crie o banco de dados.

    Em geral, o provedor da aplicação disponibiliza scripts para criação e preenchimento do banco de dados.

  5. Crie um pool de conexões para o banco de dados.

    Consulte Para Criar um Pool de Conexões do JDBC.

  6. Crie um recurso JDBC que aponte para o pool de conexões.

    Consulte Para Criar um Recurso do JDBC.
