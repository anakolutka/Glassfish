删除托管调度执行程序服务

  1. 在导航树中，展开“资源”节点。

  2. 在“资源”节点下，展开“并发资源”节点。

  3. 在“并发资源”节点下，选择“托管调度执行程序服务”。

    此时将打开“托管调度执行程序服务”页。

  4. 在“托管调度执行程序服务”页上，选中要删除的资源名称旁边的复选框。

  5. 单击“删除”。
