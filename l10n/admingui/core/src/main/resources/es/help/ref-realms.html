Dominios

Servidor empresarial viene preconfigurado con tres dominios: file, certificate y admin-realm. También es posible crear dominios ldap, JDBC, solaris y personalizados. Normalmente se tiene un dominio de cada tipo en un servidor, pero en Servidor empresarial existen dos dominios de archivo: file y admin-realm. Se trata de dos dominios del mismo tipo utilizados para dos fines diferentes. También es posible tener una base de datos de certificados diferente para cada servidor virtual de su sistema.

La página Dominios muestra una lista de los dominios disponibles. De cada dominio se proprociona la información siguiente:

  • Nombre

  • Nombre de clase
