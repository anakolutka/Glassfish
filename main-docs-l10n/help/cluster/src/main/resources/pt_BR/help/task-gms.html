Para Configurar o Serviço de Gerenciamento de Grupos

O Serviço de Gerenciamento de Grupos (GMS) é um serviço em andamento que oferece monitoramento de cluster, associação de cluster e serviços de comunicação de grupos. O GMS realiza as seguintes funções:

  • Notifica os módulos registrados em uma instância do GlassFish Server quando uma ou mais instâncias do membro são adicionadas ou removidas de um cluster, ou quando tais instâncias são suspeitas de ter falhas ou realmente tenham falhas.

  • Fornece a capacidade de enviar e receber mensagens entre membros do cluster.

  1. Na árvore de navegação, expanda o nó Configuração.

  2. Selecione a configuração do cluster que está sendo alterado.

  3. No nó da configuração, selecione o nó Serviço de Gerenciamento de Grupos.

    A página Serviço de Gerenciamento de Grupos será aberta.

  4. No campo Máximo de Pulsações Perdidas, digite o número máximo de ativações perdidas antes que o GMS confirme se há uma suspeita de falha.

    O default é 3.

  5. No campo Frequência das Pulsações, digite o período entre sucessivas transmissões da pulsação de um membro do grupo.

    O default é 2000 milissegundos.

  6. No campo Time-out para Descoberta de Grupo, digite a quantidade de tempo que o GMS espera a descoberta de outros membros do grupo.

    O default é 5000 milissegundos.

  7. No campo Tempo de Espera para Confirmação de Falha, digite a quantidade de tempo a ser aguardado antes de confirmar se um membro com suspeita de falha deve ser relatado como apresentado falha.

    O default é 1500 milissegundos.

  8. Para restaurar os valores default, clique no botão Carregar Defaults.

  9. Na seção Propriedades Adicionais, especifique as propriedades adicionais.

    • Para adicionar uma propriedade, clique no botão Adicionar Propriedade. Na linha em branco que for exibida, digite o nome da propriedade no campo Nome e digite o valor da propriedade no campo Valor.

    • Para modificar uma propriedade, edite o campo Valor da propriedade.

    • Para deletar uma propriedade, marque a caixa de seleção à esquerda do campo Nome da Propriedade a ser deletada. Clique no botão Deletar Propriedades.

  10. Clique em Salvar.
