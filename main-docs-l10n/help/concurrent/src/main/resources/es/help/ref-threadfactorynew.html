Nueva fábrica de threads gestionada

Utilice la página Nueva fábrica de threads gestionada para crear un recurso de la fábrica de threads gestionada.

La página Nueva fábrica de threads gestionada contiene las siguientes opciones.

Nombre JNDI

Nombre único que identifica el recurso.

Información de contexto

Contexto del recurso: classloader, JNDI, seguridad o área de trabajo.

Estado

Estado del recurso. El recurso se puede activar o desactivar. El valor por defecto es Activado.

Prioridad de threads

Prioridad de threads que se van a crear para la instancia de la fábrica de threads gestionada. El valor por defecto es 5.

Descripción

Descripción de este recurso.

Propiedades Adicionales

Propiedades adicionales del recurso. GlassFish Server no define ninguna propiedad adicional para los recursos de la fábrica de threads gestionada.
