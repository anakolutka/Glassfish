Redémarrage d'une application déployée

Redémarrer une application redémarre simplement l'application sans la redéployer tout en conservant les paramètres et les options avec lesquels l'application a été déployée.

  1. Dans l'arborescence de navigation, sélectionnez le nœud Applications.

    La page Applications s'affiche.

  2. Dans le tableau Applications déployées, cliquez sur Redémarrer sur la ligne de l'application que vous redéployez.

    La page Applications s'affiche à nouveau, avec un message indiquant si le redémarrage a réussi.
