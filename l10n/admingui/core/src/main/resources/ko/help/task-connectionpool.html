HTTP 서비스 연결 풀을 구성하는 방법

  1. 탐색 트리에서 구성 노드를 선택합니다.
  2. 구성 노드에서 HTTP 서비스 노드를 선택합니다.
  3. HTTP 서비스 페이지에서 연결 풀 탭을 누릅니다.

    연결 풀 페이지가 열립니다.

  4. 보류 중인 최대 연결 수 필드에 HTTP 수신기에 허용할 보류 중인 최대 연결 수를 입력합니다.

    기본값은 4096입니다.

  5. 대기열 크기 필드에 연결 대기열의 최대 크기(바이트)를 입력합니다.

    이 값은 서버가 보유할 수 있는 해결되지 않은 최대 연결 수도 지정합니다. 기본값은 4096입니다.

  6. 수신 버퍼 크기 필드에 HTTP 수신기의 수신 버퍼 크기(바이트)를 입력합니다.

    기본값은 4096입니다.

  7. 전송 버퍼 크기 필드에 HTTP 수신기의 전송 버퍼 크기(바이트)를 입력합니다.

    기본값은 8192입니다.

  8. 저장을 누릅니다.

    참고 – 기본 설정으로 돌아가려면 기본값 로드 버튼을 누른 다음 저장을 누릅니다.
