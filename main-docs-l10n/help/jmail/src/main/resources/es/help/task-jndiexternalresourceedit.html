Para editar un recurso JNDI externo

  1. En el árbol de navegación, amplíe el nodo Recursos.

  2. En el nodo Recursos, amplíe el nodo JNDI.

  3. En el nodo JNDI, seleccione Recursos Externos.

    Se abrirá la página Recursos Externos.

  4. En la página Recursos Externos, haga clic en el nombre del recurso que va a editar.

    Se abrirá la página Editar Recurso Externo.

  5. Para modificar el tipo de recurso, realice una de las acciones siguientes.

    • En la lista desplegable, seleccione una definición de tipo.

    • Seleccione el segundo botón de radio e introduzca una definición de tipo completamente cualificada que siga el formato xxx.xxx.

  6. En el campo Clase de Fábrica, escriba un repositorio externo de clase de fábrica JNDI para el recurso externo.

    La Clase de Fábrica es el nombre especificado por el usuario para la clase de fábrica, como com.sun.jndi.ldap. Esta clase implanta la interfaz javax.naming.spi.InitialContextFactory.

  7. En el campo Búsqueda JNDI, introduzca el valor JNDI que buscar en el repositorio externo.

    Por ejemplo, cuando se crea un recurso externo para conectar a un repositorio externo, para probar una clase de bean, la búsqueda JNDI puede seguir el formato: cn=testmybean.

  8. En el campo Orden de despliegue, escriba un número para especificar el orden de despliegue de los recursos externos.

    Los recursos con un número inferior se cargan en primer lugar al iniciar el servidor. El valor por defecto es 100.

  9. En el campo Descripción, introduzca una breve descripción del recurso.

    Esta descripción es un valor de cadena y puede incluir un máximo de 250 caracteres.

  10. Active la casilla de control Estado activado para activar el recurso.

    Por defecto, el recurso está activado, lo que significa que está disponible en cuanto se crea.

  11. En la sección Propiedades Adicionales, especifique propiedades adicionales del recurso externo.

    GlassFish Server no define ninguna propiedad adicional para los recursos externos.

    • Para agregar una propiedad, haga clic en el botón Agregar Propiedad. En la fila en blanco que aparece, introduzca el nombre de propiedad en el campo Nombre y escriba el valor de propiedad en el campo Valor.

    • Para modificar una propiedad, edite el campo Valor.

    • Para suprimir una propiedad, active la casilla de control a la izquierda del campo Nombre de la propiedad que está suprimiendo y, a continuación, haga clic en el botón Suprimir Propiedades.

  12. Haga clic en Guardar.
