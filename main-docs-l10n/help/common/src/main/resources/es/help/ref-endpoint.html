Información de punto final de servicio web

Utilice la página Información de Punto Final de Servicio Web para ver información sobre el punto final de un servicio web.

La página Información de Punto Final de Servicio Web contiene la siguiente información.

Nombre de Aplicación

Nombre de la aplicación desplegada. Haga clic en este enlace o en el botón Atrás para volver a la página Editar Aplicación.

El nombre puede incluir un identificador de versión opcional, que va a continuación del nombre y está separado de él por dos puntos (:). Para obtener más información sobre las versiones de módulo y de aplicación, consulte el apartado sobre las versiones de módulo y de aplicación en Oracle GlassFish Server Application Deployment Guide.

Probador

El enlace Probador sólo está visible si el tipo de despliegue de la aplicación es distinto de RI (por ejemplo, 109). Haga clic en este enlace para ver una página que permita proporcionar parámetros para probar los métodos del servicio web.

WSDL

Web Services Definition Language (WSDL) para el punto final. Haga clic en este enlace para ver el archivo WSDL en un nuevo separador o ventana del explorador.

Tenga en cuenta que algunos exploradores tratan de interpretar el origen de WSDL al hacer clic en el enlace WSDL, de modo que muestran una página en blanco. Para estos navegadores, utilice el comando Ver Origen del explorador en la página en blanco para ver el origen de WSDL.

Nombre de Punto Final

Nombre del punto final.

Nombre de Servicio

Nombre del servicio web.

Nombre de Puerto

Nombre del puerto del servicio web.

Tipo de Despliegue

Tipo de despliegue del servicio web (por ejemplo, RI o 109).

Tipo de Implantación

Tipo de implantación del servicio web (por ejemplo, SERVLET o EJB).

Nombre de Clase de Implantación

Nombre de la clase de implantación del servicio web.

URI de Dirección de Punto Final

URI de la dirección del punto final.

Espacio de Nombres

Espacio de nombres de destino asociado al servicio web.

Descripción

Descripción del punto final, si existe.
