Para suprimir un pool de conexiones JDBC

Cuando se suprime un pool de conexiones JDBC, los recursos JDBC que dependen de ese pool de conexiones también se suprimen.

  1. En el árbol de navegación, amplíe el nodo Recursos.

  2. En el nodo Recursos, amplíe el nodo JDBC.

  3. En el nodo JDBC, seleccione el nodo Pools de conexiones.

    Se abrirá la página Pools de conexiones.

  4. En la página Pool de conexiones, active la casilla de control que aparece a la izquierda del pool de conexiones que desee suprimir.

  5. Haga clic en Suprimir.
