Contrôle des performances de Enterprise Server

Le contrôle des performances de Enterprise Server comprend les tâches suivantes :

  • Configurer le contrôle du service

  • Afficher les données de contrôle du service HTTP

  • Afficher les données de contrôle du conteneur Web

  • Afficher les données de contrôle JVM

  • Afficher les données de contrôle du pool de threads
