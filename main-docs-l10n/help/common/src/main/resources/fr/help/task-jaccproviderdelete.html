Suppression d'un fournisseur JACC

  1. Sous le noeud Configurations dans l'arborescence de navigation, développez le cluster ou l'instance GlassFish Server pour lequel vous souhaitez créer un fournisseur JACC.

    Pour créer un fournisseur qui sera disponible pour l'ensemble des instances et clusters, choisissez le noeud default-config.

  2. Sous le noeud développé de configuration du cluster ou de l'instance, développez le noeud Sécurité.

  3. Sous le noeud Sécurité, sélectionnez le noeud Fournisseurs JACC.

    La page Fournisseurs JACC apparaît.

  4. Cochez la case située à gauche du fournisseur JACC à supprimer.

  5. Cliquez sur Supprimer.
