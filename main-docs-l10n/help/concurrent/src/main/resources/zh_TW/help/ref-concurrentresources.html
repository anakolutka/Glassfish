並行資源

使用「並行資源」頁面可組態下列類型的並行資源：

  • 環境服務

  • 受管理繫線工廠

  • 受管理執行程式服務

  • 受管理排定的執行程式服務

並行資源為受管理物件，主要用來提供 Java EE 應用程式並行功能 (如 JSR 236 規格 Concurrency Utilities for Java EE 中所定義)。

在 GlassFish Server 中，您可以組態並行資源，然後將它們提供給像是 Servlet 和 EJB 等的應用程式元件使用。並行資源可透過 JNDI 查詢或資源引入存取。
