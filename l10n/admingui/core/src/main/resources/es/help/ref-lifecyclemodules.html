Módulos de ciclo de vida

La página Módulos de ciclo de vida muestra una lista de módulos de ciclo de vida disponibles.

Un módulo de ciclo de vida realiza tareas cuando es activado por uno o más eventos del ciclo de vida del servidor. Los posibles eventos del servidor que pueden activarlo son: inicialización, arranque, disponible para atender peticiones y apagado. Los módulos de ciclo de vida no forman parte de la especificación de Java EE, sino que son una mejora de Servidor empresarial.

Se proporciona la siguiente información para cada módulo de ciclo de vida.

Nombre

Nombre del módulo de ciclo de vida.

Habilitado

True si el módulo de ciclo de vida está habilitado, o false si no lo está.

Orden de carga

Valor que especifica el orden relativo en el que el módulo de ciclo de vida debería cargarse.

Nombre de clase

Archivo de clase del módulo de ciclo de vida.
