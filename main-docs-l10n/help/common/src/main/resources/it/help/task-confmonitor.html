Per configurare il servizio di monitoraggio

  1. Per aprire la pagina Servizio di monitoraggio, effettuare una delle operazioni riportate di seguito.

    • Selezionare il nodo Dati di monitoraggio e fare clic sul collegamento Configura monitoraggio per l'istanza del server per il quale si desidera configurare il monitoraggio. Per configurare il monitoraggio per un cluster, scegliere un'istanza qualsiasi nel cluster.

    • Nel nodo Configurazioni espandere la configurazione dell'istanza server o del cluster per cui si desidera configurare il monitoraggio, quindi espandere il nodo Monitoraggio.

    Viene visualizzata la pagina Servizio di monitoraggio.

  2. Selezionare la casella di controllo Abilitato dell'opzione Servizio di monitoraggio per abilitare il monitoraggio per GlassFish Server o deselezionarla per disabilitare il monitoraggio.

    Questa opzione è abilitata per impostazione predefinita.

  3. Selezionare la casella di controllo Abilitato dell'opzione MBean di monitoraggio per distribuire tutti gli MBean necessari per il monitoraggio o deselezionare la casella di controllo per annullare la distribuzione degli MBean.

    Questa opzione è abilitata per impostazione predefinita.

    Se la casella di controllo non è selezionata, non è possibile visualizzare i dati di monitoraggio nella Console di amministrazione, anche se è selezionata la casella di controllo Servizio di monitoraggio. In tal caso, per visualizzare i dati di monitoraggio è necessario utilizzare la riga di comando. Se si deseleziona la casella di controllo, l'utilizzo di memoria di GlassFish Server risulterà ridotto.

  4. Selezionare la casella di controllo Abilitato dell'opzione DTrace per abilitare l'utilizzo della funzione DTrace Solaris.

    Per utilizzare la funzione DTrace Solaris è necessario disporre anche di Oracle Enterprise Manager DTrace Monitoring. Questa opzione è disabilitata per impostazione predefinita.

  5. Eseguire una delle azioni riportate di seguito nella tabella.

    • Nella colonna Livello di monitoraggio selezionare BASSO o ALTO dall'elenco a discesa per uno o più componenti della colonna Componente, in modo da abilitare il monitoraggio per i componenti corrispondenti.

    • Selezionare le caselle di controllo in corrispondenza di uno o più componenti, quindi selezionare BASSO o ALTO nell'elenco a discesa Livello nella parte superiore della tabella e fare clic sul pulsante Cambia livello per impostare il livello di monitoraggio sul valore desiderato per tutti i componenti.

    Nella release corrente i livelli di monitoraggio garantiti con le opzioni BASO e ALTO non presentano differenze. Per disabilitare il monitoraggio per uno o più componenti, selezionare DISATTIVATO. Il valore predefinito è DISATTIVATO.

    Prima di configurare il monitoraggio di un'applicazione, un componente o un servizio, è necessario eseguirne la distribuzione.

  6. Fare clic su Salva.
