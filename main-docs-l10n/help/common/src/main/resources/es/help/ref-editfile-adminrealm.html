Propiedades específicas de la clase FileRealm

Las siguientes propiedades son necesarias para un dominiofile.

Contexto JAAS

Contexto JAAS (Java Authentication and Authorization Service) (identificador para el módulo de conexión que se utiliza para este dominio). El único valor válido es fileRealm.

Archivo de claves

Ruta de acceso completa y nombre del archivo donde el servidor almacenará toda la información sobre usuarios, grupos y contraseñas para este dominio. El valor por defecto es domain-dir/config/keyfile para el dominio file y domain-dir/config/admin-keyfile para el dominio admin-realm.

El archivo de claves para el dominio file está inicialmente vacío, así que los usuarios deben agregarse en la página Usuarios de archivos antes de que se utilice el dominio file.

El archivo de claves para el dominio admin-realm inicialmente contiene el nombre de usuario del administrador, la contraseña del administrador en formato cifrado y el grupo al que pertenece este usuario (asadmin por defecto). Para obtener más información sobre cómo agregar usuarios a admin-realm, consulte Otorgar acceso a herramientas de administración.

Nota:

Los usuarios del grupo asadmin en admin-realm están autorizados a utilizar las herramientas Consola de administración y asadmin. Agregue a este grupo sólo usuarios que tengan privilegios administrativos del servidor.

La siguiente propiedad opcional está disponible para un dominio file.

Asignar grupos

Lista separada por comas de nombres de grupos. Todos los clientes que presenten certificados válidos se asignan a estos grupos, por ejemplo, employee,manager , donde éstos son los nombres de los grupos de usuarios.
