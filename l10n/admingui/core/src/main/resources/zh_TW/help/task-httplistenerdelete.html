刪除 HTTP 偵聽程式

開始之前

若計劃刪除預設 HTTP 偵聽程式 http-listener-1、http-listener-2 或 admin-listener，請先為 Enterprise Server 的 domain.xml 檔案建立 http-listener 元素的備份副本，才可在需要時復原這些設定。

  1. 在瀏覽樹狀結構中，展開 [配置] 節點。
  2. 在 [配置] 節點下，展開 [HTTP 服務] 節點。
  3. 在 [HTTP 服務] 節點下，選取 [HTTP 偵聽程式] 節點。
  4. 在 [HTTP 服務] 頁面上，選取您要刪除之 HTTP 偵聽程式名稱旁的核取方塊。
  5. 按一下 [刪除]。
