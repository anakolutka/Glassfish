Pour modifier le mot de passe d'un utilisateur du domaine d'administration

Pour modifier le mot de passe d'un compte administrateur actuel, consultez la rubrique Pour changer le mot de passe administrateur. Si le domaine admin-realm comporte plusieurs utilisateurs, vous pouvez également modifier le mot de passe pour d'autres utilisateurs.

Le chiffrement du mot de passe de l'administrateur renforce la sécurité de votre application. Pour plus d'informations sur les procédures de sécurité, consultez le Chapitre 11, Administering System Security du Sun GlassFish Enterprise Server v3 Administration Guide.

  1. Dans l'arborescence de navigation, développez le nœud Configuration.
  2. Développez le nœud Sécurité.
  3. Développez le nœud Domaines.
  4. Sélectionnez le nœud admin-realm.

    La page Modification du domaine s'ouvre.

  5. Sur la page Modification du domaine, cliquez sur le bouton Gérer les utilisateurs.

    La page Utilisateurs de fichiers s'ouvre.

  6. Dans la page Utilisateurs de fichiers, cliquez sur l'utilisateur pour lequel vous souhaitez ajouter ou modifier le mot de passe.

    La page Modification du domaine fichier s'ouvre.

  7. Dans les champs Nouveau mot de passe et Confirmer le mot de passe, saisissez le nouveau mot de passe.
  8. Cliquez sur Enregistrer.
