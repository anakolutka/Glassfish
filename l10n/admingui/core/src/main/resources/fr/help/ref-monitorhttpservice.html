Contrôle du service HTTP

La page Contrôle du service HTTP affiche les statistiques complètes du pool de connexions et du pool de threads de Enterprise Server, ainsi que les statistiques des requêtes du serveur virtuel et du listener HTTP sélectionnés. Pour une description des propriétés non contrôlées, reportez-vous au Chapitre 10, Monitoring the Enterprise Server du Sun GlassFish Enterprise Server v3 Prelude Administration Guide.
