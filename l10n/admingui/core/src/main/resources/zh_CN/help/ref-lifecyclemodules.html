生命周期模块

“生命周期模块”页面显示可用的生命周期模块列表。

在服务器的生命周期中，如果一个或多个事件触发了生命周期模块，此生命周期模块将执行一些任务。可能的触发服务器事件为：初始化、启动、准备为请求提供服务和关机。生命周期模块并不在 Java EE 规范的定义范围内，而是 Enterprise Server 的增强功能。

对于每个生命周期模块，将提供以下信息：

名称

生命周期模块名称。

已启用

如果启用了生命周期模块，则为已启用；如果未启用生命周期模块，则为未启用。

加载顺序

指定应装入生命周期模块相对顺序的值。

类名

生命周期模块的类文件。

相关任务

  • 创建生命周期模块

  • 编辑生命周期模块

  • 删除生命周期模块

相关的 asadmin 命令

  • create-lifecycle-module(1)

  • list-lifecycle-modules(1)

  • delete-lifecycle-module(1)
