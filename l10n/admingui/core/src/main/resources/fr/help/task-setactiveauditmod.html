Définir le module d'audit actif

  1. Dans l'arborescence de navigation, développez le nœud Configuration.
  2. Sous le nœud Configuration, sélectionnez le nœud Sécurité.

    La page Sécurité s'ouvre.

  3. Dans la liste déroulante Modules d'audit, sélectionnez le nom du module d'audit à utiliser par le serveur.

    Par défaut, il n'y a qu'un module d'audit disponible, nommé valeur par défaut.

  4. Cliquez sur Enregistrer.
