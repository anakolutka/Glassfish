GlassFish コミュニティではフリー、 オープンソース、かつ 実運用品質の エンタープライズ向けソフトウェアを作っています。 主な成果物は Application Server、 Java EE 5 リファレンス実装、 そして Java Persistence API リファレンス実装 TopLink Essentials です。 また、コミュニティは Maven リポジトリ、 ツール、その他いろいろお届けします。

Project SailFin, Open Source SIP Servlets Technology for GlassFish
SIP (Session Initiation Protocol) は私たちが今日体験する多くのサービスを支えて>います。例えば Voice-over-IP (VoIP) 電話サービス、インスタントメッセージング、友達リスト、web 会議などです。SailFin プロジェクトは SIP Servlet テクノロジを GlassFish Application Server に追加します。詳細は https://sailfin.dev.java.net をご覧下さい。

  
  
  
  
始めるには
GlassFish を使って最新の Java EE 5 テクノロジについて 学んだり、 アプリケーション開発を始めることができます。 GlassFish サンプルや Blueprints サンプルを試して実際に Java EE 5 に触れながら学んでみて下さい。また、 既存の J2EE アプリケーションを Java EE 5 に移行するやり方についてもご覧下さい。

参加するには
GlassFish サーバの改善を手伝ったり、GlassFish コミュニティがどのように Grizzly、 EJB 3.0、TopLink Essentials、 管理 ツール、その他たくさんのテクノロジや機能を実装しているのか学びましょう。
