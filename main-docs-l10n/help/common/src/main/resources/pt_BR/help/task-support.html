Para Exibir o Blog do GlassFish Server

O blog do GlassFish Server oferece notícias da comunidade do GlassFish Server e links para informações adicionais sobre o GlassFish Server.

  1. Na árvore de navegação, selecione o nó Tarefas Comuns.

    A página Tarefas Comuns será aberta.

  2. Em Notícias do GlassFish, clique no link Blog.

    O blog do GlassFish Server é aberto em uma guia ou janela separada do browser.
