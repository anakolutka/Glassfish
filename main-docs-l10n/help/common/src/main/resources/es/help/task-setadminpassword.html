Cambiar la contraseña de administrador

  1. En el árbol de navegación, seleccione el nodo Dominio.

    Se abrirá la página Atributos de Dominio.

  2. Seleccione el separador Contraseña de Administrador.

    Se abrirá la página Nueva Contraseña de Administrador.

    La página mostrará el ID de usuario y el nombre de grupo de la cuenta actual de administrador. Por defecto, el ID de usuario es admin, y el nombre del grupo es asadmin.

  3. En el campo Contraseña Nueva, introduzca la nueva contraseña para la cuenta actual de administrador.

  4. En el campo Confirmar Contraseña Nueva, vuelva a escribir la nueva contraseña.

  5. Haga clic en Guardar.
