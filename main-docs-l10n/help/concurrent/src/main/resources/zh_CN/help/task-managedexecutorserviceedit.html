编辑托管执行程序服务

  1. 在导航树中，展开“资源”节点。

  2. 在“资源”节点下，展开“并发资源”节点。

  3. 在“并发资源”节点下，选择“托管执行程序服务”。

    此时将打开“托管执行程序服务”页。

  4. 在“托管执行程序服务”页上，单击要编辑的资源的名称。

    此时将打开“编辑托管执行程序服务”页。

    注:

    JNDI 名称是只读字段。只能在创建新资源时指定 JNDI 名称。

  5. XXX

  6. 在“说明”字段中，键入资源的简短说明。

    此说明是字符串值，最多可以包含 250 个字符。

  7. 选择“启用状态”复选框以启用资源。

  8. 在“其他属性”部分中，指定资源的其他属性。

    GlassFish Server 不定义托管执行程序服务资源的任何其他属性。

    • 要添加属性，请单击“添加属性”按钮。在显示的空白行中，在“名称”字段中键入属性名称，在“值”字段中键入属性值。

    • 要修改属性，请编辑该属性的“值”字段。

    • 要删除属性，请选中要删除属性的“名称”字段左侧的复选框，然后单击“删除属性”。

  9. 单击“保存”。
