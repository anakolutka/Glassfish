管理程式特性

階段作業管理員提供各式方法，以配置建立與銷毀階段作業的方式、階段作業狀態的儲存位置，以及可用的最大階段作業數。

使用 [管理程式特性] 頁面可增加或修改階段作業管理程式特性。[管理程式特性] 頁面包含下列選項：

清除間隔

從存放區刪除非使用中的階段作業資料之前，等待的秒數。

最大階段作業數

所允許的最大階段作業數。

階段作業檔案名稱

包含階段作業資料的檔案名稱。

階段作業 ID 產生器類別名稱

在 Enterprise Server 類別路徑中指定的 ID 產生器類別名稱。

其他特性

階段作業管理員的其他特性。

  • 若要增加特性，請按一下 [增加特性] 按鈕。在出現的空白列中，於 [名稱] 欄位中鍵入特性名稱，然後在 [值] 欄位中鍵入特性值。按一下 [儲存]。

  • 若要修改特性，請編輯特性的 [值] 欄位。按一下 [儲存]。

  • 若要刪除特性，請選取您要刪除之特性 [名稱] 欄位左側的核取方塊，然後按一下 [刪除特性] 按鈕。按一下 [儲存]。
