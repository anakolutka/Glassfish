Para definir os atributos de domínio avançados

  1. Na árvore de navegação, selecione o nó Servidor de Aplicativos.

    A página Informações Gerais é exibida.

  2. Na página Informações Gerais, clique na guia Avançado.

    A página Configuração de Aplicativos é exibida.

  3. Na página Configuração de Aplicativos, clique na guia Atributos de Domínio.

    A página Atributos de Domínio é exibida.

  4. No campo Raiz de Aplicativo, digite o caminho completo do diretório onde os aplicativos implantados residem.

    O valor padrão é domain-dir/applications .

  5. No campo Raiz de Log, digite o caminho completo do diretório onde os arquivos de log de servidor serão mantidos.

    O valor padrão é domain-dir/logs.

  6. Deixe o campo Localidade em branco para usar a localidade padrão do host.

    Uma localidade é um identificador que especifica uma determinada combinação de idioma e região. Por exemplo, a localidade para inglês americano é en_US e para japonês é ja_JP. Para usar uma localidade que não seja o inglês, o Enterprise Server deve ser localizado, um processo que inclui a tradução de inglês para outro idioma.

  7. Clique em Salvar.
  8. Reinicie o servidor.
