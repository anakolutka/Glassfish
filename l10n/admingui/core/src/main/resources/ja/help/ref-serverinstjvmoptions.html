JVM オプション

「JVM オプション」ページを使用して、Enterprise Server を実行する Java アプリケーション起動ツール (java) のオプションを指定します。-D オプションは、Enterprise Server に固有のプロパティーを指定します。

  • オプションを変更するには、そのオプションの「値」フィールドを編集します。

  • オプションを追加するには、「JVM オプションを追加」ボタンをクリックします。表示される空白行に、「値」フィールドのオプション情報を入力します。

  • オプションを削除するには、削除するオプションの「値」フィールドの左側にあるチェックボックスを選択し、「削除」をクリックします。
