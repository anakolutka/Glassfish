Cibles de serveur virtuel

La page Cibles de serveur virtuel permet de modifier les serveurs virtuels cible auxquels une application ou un module est associé.

Elle contient les informations ci-après.

Cibles disponibles

Serveurs virtuels auxquels aucune application ou aucun module n'est associé.

Cibles sélectionnées

Serveurs virtuels auxquels une application ou un module est associé.

Ajouter

Bouton permettant de déplacer la cible sélectionnée de Cibles disponibles vers Cibles sélectionnées.

Tout ajouter

Bouton permettant de déplacer toutes les cibles disponibles vers Cibles sélectionnées.

Enlever

Bouton permettant de déplacer la cible sélectionnée de Cibles sélectionnées vers Cibles disponibles.

Tout enlever

Bouton permettant de déplacer toutes les cibles sélectionnées vers Cibles disponibles.
