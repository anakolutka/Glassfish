Ver Archivos Log para una Instancia de GlassFish Server Independiente

Antes de Empezar

La instancia cuyos archivos log desea ver debe estar en ejecución.

  1. En el árbol de navegación, seleccione el nodo Instancias Independientes.

    Se abrirá la página Instancias de Servidor Independientes.

  2. Seleccione la instancia para la que desea ver los archivos log.

    Se abrirá la página Información General para la instancia.

  3. En la página Información General, haga clic en Ver Archivos Log.

    Se abre una ventana con el visor de logs.

    Nota:

    La instancia ya debe estar en ejecución. Si la instancia no está ejecutándose, el botón Ver Archivos Log estará desactivado.
