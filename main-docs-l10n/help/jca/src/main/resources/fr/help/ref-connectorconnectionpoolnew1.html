Nouveau pool de connexions de connecteur (étape 1 sur 2)

La page Nouveau pool de connexions de connecteur (étape 1 sur 2) permet de créer un pool de connexions de connecteur.

Elle contient les options ci-après.

Nom

Nom unique identifiant le pool de connexions de connecteur.

Adaptateur de ressources

Adaptateur de ressources associé au pool de connexions de connecteur.

Définition de connexion

Définition de connexion pour le pool de connexions de connecteur. Les options disponibles dans la liste dépendent de l'adaptateur de ressources sélectionné. La définition de connexion identifie l'élément connection-definition d'un adaptateur de ressources dans le fichier ra.xml.
