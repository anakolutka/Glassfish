Nuova risorsa esterna

Utilizzare la pagina Nuova risorsa esterna per creare una risorsa JNDI esterna.

La pagina Nuova risorsa esterna contiene le opzioni riportate di seguito.

Nome JNDI

Il nome univoco che identifica la risorsa.

Tipo di risorsa

La definizione completamente qualificata del tipo di risorsa.

Classe factory

Il nome specificato dall'utente per la classe factory. Questa classe implementa l'interfaccia javax.naming.spi.InitialContextFactory.

Ricerca JNDI

Il valore JNDI per eseguire ricerche nel repository esterno.

Ad esempio, quando si crea una risorsa esterna da collegare a un repository esterno, per eseguire i test su una classe di bean, la funzione Ricerca JNDI può avere l'aspetto seguente: cn=testmybean.

Descrizione

La descrizione di questa risorsa esterna.

Stato

Lo stato della risorsa esterna. La risorsa esterna può essere abilitata o disabilitata. Lo stato predefinito è Abilitato.

Proprietà aggiuntive

Proprietà aggiuntive per la risorsa esterna. GlassFish Server non definisce alcuna proprietà aggiuntiva per le risorse esterne.

Destinazioni

I cluster e le istanze standalone per la risorsa esterna. Per spostare le destinazioni desiderate nella colonna Destinazioni selezionate, utilizzare i pulsanti Aggiungi e Aggiungi tutto. Per spostare le destinazioni non necessarie nella colonna Destinazioni disponibili, utilizzare i pulsanti Rimuovi e Rimuovi tutto.

Questa opzione viene visualizzata solo se nel dominio sono stati creati cluster o istanze standalone.
