Editar Provedor JACC

Use a página Editar Provedor JACC para modificar um provedor JACC da instância ou cluster de GlassFish Server selecionado.

A página Editar Provedor JACC contém as seguintes opções.

Nome da Configuração

O nome da configuração à qual as definições nesta página se aplicam. Este campo é somente para leitura.

Nome

O nome do provedor JACC. Este é um campo somente para leitura. Você só pode especificar o nome quando cria um novo provedor JACC.

Configuração da Política

Nome de classe totalmente qualificado do javax.security.jacc.PolicyConfigurationFactory.provider que implementa o javax.security.jacc.PolicyConfigurationFactory específico do provedor.

Provedor de Política

Nome de classe totalmente qualificado do javax.security.jacc.policy.provider que implementa o java.security.Policy.

Propriedades Adicionais

Propriedades adicionais do provedor JACC. Para obter uma descrição das propriedades adicionais que estão disponíveis para provedores JACC, consulte Propriedades Específicas para Provedores JACC.
