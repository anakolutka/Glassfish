Nuovo utente realm

Utilizzare la pagina Nuovo utente realm per concedere l'accesso al realm corrente a un nuovo utente.

Nota:

Se l'amministrazione sicura è abilitata, non è possibile creare un utente di admin-realm con una password vuota.

La pagina Nuovo utente realm contiene le opzioni riportate di seguito.

Nome configurazione

Il nome della configurazione a cui si applicano le impostazioni presenti in questa pagina. Questo campo è di sola lettura.

Nome realm

Il nome del realm attualmente selezionato. Questo campo è di sola lettura.

ID utente

L'ID utente per il nuovo account utente.

Lista gruppi

I gruppi ai quali apparterrà il nuovo utente. Separare i nomi dei gruppi con una virgola.

Per gli utenti admin-realm, questo campo è di sola lettura e il gruppo è asadmin.

Nuova password

La password del nuovo utente.

Conferma nuova password

La password del nuovo utente.
