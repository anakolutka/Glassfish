To Monitor Applications

  1. In the tree component, select the Application Server node.

  2. Click the Monitor tab.

  3. Click the Applications tab.

  4. Select the application to monitor from the Application list.

    The Application list shows all applications deployed to the Application Server.

  5. Select the specific component from the Component list.

    The page displays monitoring data for the selected application and component. For a description of properties that can be monitored, see Chapter 17, "Monitoring Components and Services," in Sun Java System Application Server Enterprise Edition 9.1 Administration Guide.

    If the page does not display monitoring data for the application or component, click Configure Monitoring. To monitor applications, enable monitoring for the container in which they execute. For example, for web applications enable monitoring for the Web Container; for EJBs enable monitoring for the EJB Container.

    If no monitoring data is displayed for applications, it is most likely that the application does not exist or is not exercised. Applications monitoring data is available only when the application exists, and the application is being exercised. Once the application is executed, it is registered in the monitoring registry and the monitoring data displays.
