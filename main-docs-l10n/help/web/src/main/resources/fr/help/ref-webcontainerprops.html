Propriétés propres au conteneur Web

La propriété générale supplémentaire suivante est disponible pour le conteneur Web.

dispatcher-max-depth

Empêche les instructions d'inclusion ou de retransmission récursives de créer une boucle infinie en définissant un niveau de répartition imbriquée maximal. Si le niveau est dépassé, le message suivant est écrit dans le journal du serveur :

Exceeded maximum depth for nested request dispatches

La valeur par défaut est 20.
