Novo Recurso Externo

Use a página Novo Recurso Externo para criar um recurso JNDI externo.

A página Novo Recurso Externo contém as opções a seguir.

Nome JNDI

Um nome exclusivo que identifica o recurso.

Tipo de Recurso

A definição de tipo totalmente qualificado para o recurso.

Classe de Fábrica

O nome especificado pelo usuário para a classe de fábrica. Essa classe implementa a interface javax.naming.spi.InitialContextFactory.

Pesquisa JNDI

O valor de JNDI para pesquisar no repositório externo.

Por exemplo, ao criar um recurso externo para conectá-lo a um repositório externo, para testar uma classe de bean, a Pesquisa de JNDI terá a seguinte aparênciacn=testmybean.

Descrição

Uma descrição do recurso externo.

Status

O status do recurso externo. O recurso externo pode ser ativado ou desativado. O valor default é Ativado.

Propriedades Adicionais

Propriedades adicionais para o recurso externo. O GlassFish Server não define quaisquer propriedades adicionais para recursos externos.

Alvos

(Esta opção só será exibida se os clusters ou instâncias standalone tiverem sido criados no domínio.)

Clusters e instâncias standalone do recurso. Mova os alvos desejados para a coluna Alvos Selecionados utilizando os botões Adicionar e Adicionar Tudo. Mova os alvos desnecessários para a coluna Alvos Disponíveis utilizando os botões Remover e Remover Tudo.
