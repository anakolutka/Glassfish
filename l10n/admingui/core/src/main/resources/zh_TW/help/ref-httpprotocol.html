HTTP 通訊協定

[HTTP 通訊協定] 頁面包含下列選項：

版本

要使用的 HTTP 通訊協定版本 (HTTP/1.0 或 HTTP/1.1)。預設值為 HTTP/1.1。

啟用 DNS 查詢

若選取此選項，則會啟用用戶端的 DNS 項目查詢。此選項預設為取消選取。

啟用 SSL

若選取此選項，則會將 SSL 用於所有啟用了安全性的偵聽程式。若取消選取此選項，則會以全域方式停用伺服器安全性。此選項預設為選取。

強制類型

在沒有與副檔名相符的可用 MIME 對映時，要使用的回應類型。預設值為 text/html; charset=iso-8859-1。

預設類型

預設值為 text/html; charset=iso-8859-1。該值是以分號分隔的字串，由內容類型、編碼、語言和字元集組成。
