檢視 GlassFish Server 部落格

GlassFish Server 部落格會提供來自 GlassFish Server 社群的新聞，以及其他 GlassFish Server 相關資訊的連結。

  1. 在瀏覽樹狀結構中，選取「常用工作」節點。

    「常用工作」頁面會隨即開啟。

  2. 在「GlassFish 新聞」底下，按一下「部落格」連結。

    GlassFish Server 部落格會在另外一個瀏覽器標籤或視窗中開啟。
