So aktivieren Sie die gegenseitige Authentifizierung für alle Anwendungen

Enterprise Server verwendet den certificate-Bereich für die HTTPS-Authentifizierung.

  1. Erweitern Sie in der Navigationsstruktur den Konfigurationsknoten.
  2. Erweitern Sie unter dem Konfigurationsknoten den Sicherheitsknoten.
  3. Erweitern Sie unter dem Sicherheitsknoten den Knoten "Bereiche".
  4. Wählen Sie unter dem Knoten "Bereiche" den Bereich certificate aus.

    Die Seite "Bereich bearbeiten" wird angezeigt.

  5. Klicken Sie auf der Seite "Bereich bearbeiten" in der Tabelle "Weitere Eigenschaften" auf die Schaltfläche "Eigenschaft hinzufügen".
    1. Geben Sie im Feld "Name" den Wert clientAuth ein.
    2. Geben Sie im Feld "Wert" true ein.
  6. Klicken Sie auf "Speichern".
  7. Starten Sie Enterprise Server neu, wenn in der Konsole "Neustart erforderlich" angezeigt wird.

    Nach dem Neustart des Servers ist die Clientauthentifizierung für alle Anwendungen erforderlich, die den Bereich certificate verwenden.
