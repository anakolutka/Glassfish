新しい HTTP リスナー (ステップ 2/2)

「新しい HTTP リスナー (ステップ 2/2)」ページには、次のオプションが含まれます。

クライアント認証

このリスナーを使っているクライアントにサーバーへの認証を強制的に行わせることができます。

証明書のニックネーム

既存サーバーの鍵ペアと証明書の名前。

SSL3

このオプションを有効にすると、リスナーで SSL3 セキュリティープロトコルが使用されます。

SSL2

このオプションを有効にすると、リスナーで SSL2 セキュリティープロトコルが使用されます。

TLS

このオプションを有効にすると、リスナーで TLS セキュリティープロトコルが使用されます。

共通暗号化方式群

プロトコルで使用できる暗号化方式。

Ephemeral Diffie-Hellman 暗号化方式群

プロトコルが使用する一時的な暗号化方式。
