データベースへの JDBC テクノロジを使用したアプリケーションの接続

JDBCTM (JavaTM Database Connectivity) テクノロジは、リレーショナルデータベースへの接続を提供します。Enterprise Server では、JDBC リソース (データソース) をドメインに追加することで、データベースの接続を設定します。データソースは、データベースへの接続を設定する Java Platform, Enterprise Edition (Java EE プラットフォーム) の標準メソッドです。各データソースには、データベース接続のプールが格納されます。アプリケーションは、Java Naming and Directory InterfaceTM (J.N.D.I.) API ツリー上またはローカルのアプリケーションコンテキスト内でデータソースを検索し、getConnection メソッドを使用してデータベース接続を取得します。データソースと接続プールによって提供される接続管理プロセスのおかげで、システムの稼働は維持されます。

データベースへの JDBC テクノロジを使用したアプリケーションの接続には、次の操作が含まれます。

  • 「JDBC ドライバを統合する」

  • 「JDBC リソースを編集する」

  • 「JDBC リソースを削除する」

  • 「JDBC 接続プールを作成する」

  • 「JDBC 接続プールの一般設定を編集する」

  • 「JDBC 接続プールの詳細属性を編集する」

  • 「接続プール設定を検証する」

  • 「JDBC 接続プールのプロパティーを編集する」

  • 「JDBC 接続プールを削除する」
