Connection Factory JMS

Utilizzare la pagina Connection Factory JMS per configurare le risorse Connection Factory JMS.

Per ogni risorsa vengono fornite le informazioni riportate di seguito.

Nome JNDI

Il nome JNDI che identifica il Connection Factory.

Nome JNDI logico

Il nome JNDI logico per la Connection Factory. Per ulteriori informazioni, vedere Nomi JNDI logici.

Abilitato

Questa colonna viene visualizzata se nel dominio esiste solo l'istanza server predefinita, server.

Un segno di spunta indica che la risorsa è abilitata, una X indica che la risorsa è disabilitata.

Stato

Questa colonna viene visualizzata se nel dominio sono stati creati cluster o istanze standalone.

Il numero di cluster e istanze standalone a cui la risorsa è destinata e in quante di queste destinazioni la risorsa è abilitata. Ad esempio, "Abilitato in 2 di 4 destinazione/i" significa che la risorsa è destinata a quattro cluster e istanze standalone e che è abilitata in due di queste quattro destinazioni.

Tipo di risorsa

Il tipo di risorsa del Connection Factory.

Descrizione

La descrizione del Connection Factory.

La tabella Connection Factory contiene inoltre le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare un nuovo Connection Factory.

Elimina

Pulsante che consente di eliminare uno o più Connection Factory selezionati.

Abilita

Pulsante che consente di abilitare uno o più Connection Factory selezionati.

Disabilita

Pulsante che consente di disabilitare uno o più Connection Factory selezionati.
