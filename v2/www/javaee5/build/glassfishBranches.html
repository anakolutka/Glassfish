2.1 The Trunk

The Trunk is reserved for future work ike SJSAS 9.1 and GlassFish.next and hence it is extremely important that we keep the Trunk "buildable" all the time. The nightly build is set up on the trunk for Darwin, Linux, Solaris (Sparc and X86) and Windows. All the changes that are committed as part of SJSAS90_FCS_BRANCH work will be merged back to Trunk so that Trunk has the latest and greatest.

2.2 Building GlassFish on SJSAS90_FCS_BRANCH

The immediate work will be done on SJSAS90_ FCS_BRANCH. Developers need to follow commit procedures when updating the workspace

  • Get the Approval from GlassFish project lead about the bugs to be fixed in SJSAS90_FCS_BRANCH .
  • After the approval,  below are the steps to build from GlassFish FCS Branch:
% mkdir workspace; cd workspace
% cvs -d :pserver:<userid>@cvs.dev.java.net:/cvs checkout -r SJSAS90_FCS_BRANCH glassfish/bootstrap

As glassfish/bootstrap/project.properties from SJSAS90_FCS_BRANCH branch has variable "glassfish.cvstag" set as "glassfish.cvstag=SJSAS90_FCS_BRANCH" so branch sources will be checked out and built accordingly.

  • The commit procedures need to be followed and all the relevant tests should be run prior to checkin.
