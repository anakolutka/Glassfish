連線器資源

使用「連線器資源」頁面可組態連線器資源。

連線器資源是為應用程式提供 EIS 連線的程式物件。

對於每個資源，都將提供以下資訊。

JNDI 名稱

可識別連線器資源的唯一名稱。

已啟用

(網域中有預設伺服器執行處理 server 存在時才會顯示此欄。)

如果已啟用資源則為勾選記號，如果停用資源則為打叉記號 (X)。

狀態

(已在網域中建立叢集或獨立執行處理時才會顯示此欄。)

設定為資源目標的叢集和獨立執行處理數目，以及資源中啟用的目標數。例如，若為「已在 2 個 (共 4 個) 目標上啟用」，表示資源的目標有四個叢集和獨立執行處理，而且已在這四個目標的其中兩個啟用資源。

連線集區

與連線器資源相關聯的連線器連線集區。

描述

連線器資源的描述。

資源表也包含下列選項。

新建

建立新連線器資源的按鈕。

刪除

刪除一或多個所選取連線器資源的按鈕。

啟用

啟用一或多個所選取連線器資源的按鈕。

停用

停用一或多個所選取連線器資源的按鈕。
