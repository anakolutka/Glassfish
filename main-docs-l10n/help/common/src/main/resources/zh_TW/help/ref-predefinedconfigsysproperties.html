預先定義的系統特性

GlassFish Server 會定義下列組態系統特性：

ASADMIN_LISTENER_PORT

此特性指定 HTTP 連接埠或 HTTPS 連接埠的連接埠號碼，DAS 透過該連接埠連線至執行處理以管理執行處理。有效值為 1-65535。在 UNIX 上，建立監聽連接埠 1-1024 的通訊端需要超級使用者權限。

HTTP_LISTENER_PORT

此特性指定要用來監聽 HTTP 請求之連接埠的連接埠號碼。有效值為 1-65535。在 UNIX 上，建立監聽連接埠 1-1024 的通訊端需要超級使用者權限。

HTTP_SSL_LISTENER_PORT

此特性指定要用來監聽 HTTPS 請求之連接埠的連接埠號碼。有效值為 1-65535。在 UNIX 上，建立監聽連接埠 1-1024 的通訊端需要超級使用者權限。

IIOP_LISTENER_PORT

此特性指定用於 IIOP 連線之連接埠的連接埠號碼。此特性指定用於 IIOP 連線之連接埠的連接埠號碼。有效值為 1-65535。在 UNIX 上，建立監聽連接埠 1-1024 的通訊端需要超級使用者權限。

IIOP_SSL_LISTENER_PORT

此特性指定用於安全 IIOP 連線之連接埠的連接埠號碼。有效值為 1-65535。在 UNIX 上，建立監聽連接埠 1-1024 的通訊端需要超級使用者權限。

IIOP_SSL_MUTUALAUTH_PORT

此特性指定用於與從屬端認證之安全 IIOP 連線之連接埠的連接埠號碼。有效值為 1-65535。在 UNIX 上，建立監聽連接埠 1-1024 的通訊端需要超級使用者權限。

JAVA_DEBUGGER_PORT

此特性指定用於 Java 平台除錯程式架構 (JPDA) (http://java.sun.com/javase/technologies/core/toolsapis/jpda/) 除錯程式連線之連接埠的連接埠號碼。有效值為 1-65535。在 UNIX 上，建立監聽連接埠 1-1024 的通訊端需要超級使用者權限。

JMS_PROVIDER_PORT

此特性會指定 Java Message Service 提供者的連接埠號碼。有效值為 1-65535。在 UNIX 上，建立監聽連接埠 1-1024 的通訊端需要超級使用者權限。

JMX_SYSTEM_CONNECTOR_PORT

此特性會指定 JMX 連線器進行監聽的連接埠號碼。有效值為 1-65535。在 UNIX 上，建立監聽連接埠 1-1024 的通訊端需要超級使用者權限。

OSGI_SHELL_TELNET_PORT

此特性指定用於 Apache Felix Remote Shell (http://felix.apache.org/site/apache-felix-remote-shell.html) 連線之連接埠的連接埠號碼。此 Shell 會使用 Felix Shell 服務來與 OSGi 模組管理子系統互動。有效值為 1-65535。在 UNIX 上，建立監聽連接埠 1-1024 的通訊端需要超級使用者權限。
