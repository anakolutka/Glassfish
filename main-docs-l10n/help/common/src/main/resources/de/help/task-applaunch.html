So wird eine Anwendung gestartet

Bevor Sie beginnen

Stellen Sie sicher, dass die zu startende Webanwendung aktiviert ist.

  1. Wählen Sie in der Navigationsbaumstruktur den Anwendungsknoten.

    Die Seite "Anwendungen" wird geöffnet.

  2. Klicken Sie in der Tabelle "Bereitgestellte Anwendungen" in der Zeile für die zu startende Webanwendung auf "Starten".

    Der Anwendungsname kann eine optionale Versions-ID enthalten, die dem Namen folgt und von diesem durch einen Doppelpunkt (:) getrennt ist. Weitere Informationen zu Modul- und Anwendungsversionen finden Sie unter Module and Application Versions im Oracle GlassFish Server Application Deployment Guide.

    Die Anwendung wird in einem neuen Browserfenster geöffnet.
