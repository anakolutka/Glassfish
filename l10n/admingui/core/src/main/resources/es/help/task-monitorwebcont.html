Para visualizar los datos de supervisión del contenedor web

  1. En el árbol de navegación, seleccione el nodo Application Server.
  2. En la página Información general, haga clic en la ficha Supervisar.

    Se abrirá la página Supervisión del contenedor web.

  3. (Opcional) En la lista desplegable Aplicación, seleccione una aplicación para visualizar los datos de supervisión de dicha aplicación.
  4. (Opcional) En la lista desplegable Servidor virtual, seleccione un servidor virtual para visualizar los datos de supervisión de dicho servidor.
  5. (Opcional) Haga clic en el botón Actualizar para actualizar los datos de supervisión mostrados.
