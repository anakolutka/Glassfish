Editar asignación de seguridad del pool de conexiones de conector

Utilice la página Editar asignación de seguridad del pool de conexiones del conector para editar una asignación de seguridad para un pool de conexiones de conector.

La página Editar asignación de seguridad del pool de conexiones del conector presenta las opciones siguientes:

Nombre del pool de conexiones

Nombre del pool de conexiones de conector al que pertenece esta asignación de seguridad. El nombre del pool es un campo sólo de lectura.

Nombre de asignación de seguridad

Nombre de la asignación de seguridad. Este campo es de sólo lectura. Sólo se puede especificar un nombre de asignación de seguridad cuando cree una nueva asignación de seguridad.

Nombre de asignación de seguridad

Nombre de la asignación de seguridad.

Grupos de usuarios

Si este botón de radio está seleccionado, el campo de texto debe contener los nombres de los grupos de usuarios, separados por comas.

Principales

Si este botón de radio está seleccionado, el campo de texto debe contener los nombres de los principales, separados por comas.

Nombre de usuario

Nombre de usuario del principal de backend requerido por el EIS.

Contraseña

Contraseña del principal de backend requerida por el EIS.
