Pools de Conexiones de Conectores

Utilice la página Pools de conexiones de conectores para configurar pools de conexiones de conectores.

Un pool de conexiones de conector es un grupo de conexiones reutilizables para un determinado EIS.

Para cada recurso, aparece la siguiente información.

Nombre de pool

Nombre único que identifica el pool de conexiones de conector.

Adaptador de recursos

Adaptador de recursos asociado al pool de conexiones de conector.

Definición de conexión

Nombre de clase de la definición de conexión para el pool de conexiones de conector.

Descripción

Descripción del pool de conexiones de conector.

La tabla Pools también contiene las siguientes opciones.

Nuevo

Botón para crear un nuevo pool de conexiones de conector.

Suprimir

Botón para suprimir uno o más pools de conexiones de conectores seleccionados.
