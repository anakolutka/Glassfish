사전 정의된 시스템 등록 정보

Enterprise Server에 대해 다음 시스템 등록 정보가 사전 정의되어 있습니다.

com.sun.aas.installRoot

Enterprise Server가 설치되는 디렉토리입니다.

com.sun.aas.instanceRoot

Enterprise Server 도메인의 최상위 디렉토리입니다.

com.sun.aas.hostName

호스트(시스템)의 이름입니다.

com.sun.aas.javaRoot

Java 런타임의 설치 디렉토리입니다.

com.sun.aas.imqLib

Sun GlassFish Message Queue 소프트웨어의 라이브러리 디렉토리입니다.

com.sun.aas.domainName

도메인의 이름입니다. 이 등록 정보는 기본 구성에 사용되지는 않지만 구성을 사용자 정의하는 데 사용될 수 있습니다. 기본값은 domain1입니다.
