Para Exibir a Página de Registro de GlassFish Server

A página de Registro do GlassFish Server permite registrar sua instalação do GlassFish Server.

Como usuário registrado do GlassFish Server, você recebe benefícios como:

  • Notificação sobre novas versões, patches e atualizações

  • Ofertas especiais sobre produtos, serviços e treinamento do desenvolvedor.

  • Acesso a white papers, artigos instrucionais e webinars ao vivo com a equipe do produto GlassFish

  1. Na árvore de navegação, selecione o nó Tarefas Comuns.

    A página Tarefas Comuns será aberta.

  2. Em Notícias do GlassFish, clique no link Registro.

    A página Web Registro do GlassFish é exibida em uma guia ou janela separada do browser.
