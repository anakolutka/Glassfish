Intégrer un pilote JDBC

Un pilote prenant en charge l'API JDBC (pilote JDBC) convertit les appels JDBC d'une application en protocole du serveur de base de données.

  • Pour intégrer le pilote JDBC à un domaine administratif, effectuez l'une des opérations suivantes :
    • Configurez le pilote afin qu'il soit accessible au chargeur de classe commun.
      1. Copiez les fichiers JAR et ZIP du pilote dans le répertoire rép-domaine /lib ou copiez les fichiers de classe dans le répertoire rép-domaine/lib/ext .
      2. Redémarrez le domaine.
    • Configurez le pilote afin qu'il soit accessible au chargeur de classe du système.
      1. Dans l'arborescence de navigation, sélectionnez le nœud Application Server.
      2. Dans la page Informations générales, sélectionnez l'onglet Paramètres JVM.
      3. Dans la page Paramètres généraux JVM, sélectionnez l'onglet Paramètres du chemin.
      4. Dans le champ Suffixe du chemin de classe, saisissez le nom complet du chemin d'accès au fichier JAR du pilote.
      5. Cliquez sur Enregistrer.
      6. Redémarrez le serveur.
