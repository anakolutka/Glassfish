建立稽核模組

Enterprise Server 提供一個簡單的預設稽核模組；如需更多資訊，請參閱使用預設稽核模組。

  1. 展開 [配置] 節點。
  2. 選取要配置的實例：
    • 若要配置特定的實例，請展開該實例的配置節點。例如，對於預設實例 server，請展開 [server-config] 節點。
    • 若要配置所有實例的預設設定，請展開 default-config 節點。
  3. 展開 [安全性] 節點。
  4. 選取 [稽核模組] 節點。
  5. 在 [稽核模組] 頁面上，按一下 [新建]。
  6. 在 [建立稽核模組] 頁面上，輸入以下資訊：

    • 名稱 – 用於識別該稽核模組的名稱。

    • 類別名稱 – 實作該模組的類別之完全合格的名稱。預設稽核模組的類別名稱為 com.sun.enterprise.security.Audit。

  7. 若要將 JVM 特性增加到該模組，請按一下 [增加特性]。指定每個特性的名稱和值。

    有效特性包括 auditOn，該特性指定是否啟用該實作類別。有效值包括 true 和 false。

  8. 按一下 [確定] 以儲存項目，或者按一下 [取消] 以直接退出而不進行儲存。
