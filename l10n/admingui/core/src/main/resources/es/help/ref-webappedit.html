Aplicación de Web

Utilice la página Aplicación web para modificar una aplicación web existente. La página contiene las opciones siguientes:

Nombre

Campo de sólo lectura que muestra el nombre de la aplicación web que está editando.

Raíz de contexto

Especifica la ruta de la aplicación web. En la URL de la aplicación web, la raíz de contexto aparece justo después del número de puerto ( http://host:port/context-root/...). La raíz de contexto debe comenzar por una barra diagonal, por ejemplo, /hello.

Servidores virtuales

Servidores virtuales asociados con esta aplicación web.

Descripción

Descripción de la aplicación web.

Estado activado

Si está opción está seleccionada, entonces la aplicación web está activada. Esta opción está seleccionada de forma predeterminada.

Ubicación

Campo de sólo lectura que muestra la ubicación del archivo de almacenamiento desde donde se implementó la aplicación.
