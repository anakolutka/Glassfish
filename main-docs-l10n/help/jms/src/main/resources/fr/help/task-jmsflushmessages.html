Suppression des messages d'une destination physique JMS

  1. Dans l'arborescence de navigation, développez le noeud Configuration.

  2. Sous le noeud Configuration, développez le noeud Service JMS.

  3. Sous le noeud Service JMS, sélectionnez le noeud Destinations physiques.

    La page Destinations physiques apparaît.

  4. Sur la page Destinations physiques, sélectionnez la case en regard du nom de la destination à partir de laquelle enlever des messages.

  5. Cliquez sur le bouton Supprimer tous les messages.
