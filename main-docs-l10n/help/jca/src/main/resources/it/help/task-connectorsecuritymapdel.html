Per eliminare una mappa di sicurezza per un connection pool connettore

  1. Espandere il nodo Risorse nell'albero di navigazione.

  2. Espandere il nodo Connettori.

  3. Nel nodo Connettori selezionare il nodo Connection pool connettore.

    Viene visualizzata la pagina Connection pool connettore.

  4. Nella pagina Connection pool connettore fare clic sul nome del connection pool che include la mappa di sicurezza da eliminare.

    Viene visualizzata la pagina Modifica connection pool connettore.

  5. Fare clic sulla scheda Mappe di sicurezza.

    Viene visualizzata la pagina Mappe di sicurezza connection pool connettore.

  6. Nella pagina Mappe di sicurezza connection pool connettore selezionare la casella di controllo in corrispondenza del nome della mappa di sicurezza da eliminare.

  7. Fare clic su Elimina.
