Novo Provedor JACC

Use a página de Novo Provedor de JACC para criar um novo provedor de JSR-115/196 compatível que possa ser usado por módulos de autorização de terceiros para aplicações que estejam sendo executadas no GlassFish Server. O provedor de JACC é criado como um elemento jacc-provider dentro do elemento security-service no arquivo domain.xml do domínio.

A instalação default do &ProductName inclui dois provedores de JACC, com o nome default e simple. Todos os novos provedores de JACC criados são somados a estes dois provedores default. Os provedores de JACC do GlassFish Server default implementam um mecanismo simples de autorização baseado em arquivo que está em conformidade com a especificação JACC.

Todos os provedores de JACC do GlassFish Server devem usar a Autenticação Java e APIs de JAAS. O JAAS ativa os serviços para autenticar e fazer cumprir os controles de acesso dos usuários. O JASS implementa a versão de tecnologia Java do framework Pluggable Authentication Module (PAM) padrão.

O JSR 196 permite desenvolver plug-ins em camadas diferentes. É possível definir plug-ins que alterem a forma como o novo mecanismo de autenticação é configurado, como AuthConfigProvider e AuthConfigFactory. Também é possível definir os novos mecanismos de autenticação, como ServerAuthModule e ClientAuthModule.

A página Novo Provedor JACC contém as seguintes opções.

Nome da Configuração

O nome da configuração à qual as definições nesta página se aplicam. Este campo é somente para leitura.

Nome

O nome do provedor JACC. Este nome é utilizado para fazer referência ao elemento jacc-provider no domain.xml.

Configuração da Política

Nome de classe totalmente qualificado do javax.security.jacc.PolicyConfigurationFactory.provider que implementa o javax.security.jacc.PolicyConfigurationFactory específico do provedor.

Provedor de Política

Nome de classe totalmente qualificado do javax.security.jacc.policy.provider que implementa o java.security.Policy.

Propriedades Adicionais

Propriedades adicionais do provedor JACC. Para obter uma descrição das propriedades adicionais que estão disponíveis para provedores JACC, consulte Propriedades Específicas para Provedores JACC.
