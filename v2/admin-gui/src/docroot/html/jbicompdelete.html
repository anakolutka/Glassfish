To Delete Components

In the Developer profile, components are installed or uninstalled for the one domain. In a Clusters profile, components are enabled or disabled for that named cluster target. Consequently, starting, stopping, and shutting down components is available for the Developers profile, and enabling, disabling, or deleting components is only available for the Clusters profile.

  1. In the tree view, select the JBI node and then the Components node.

    A list of Binding Components and Service Engines is displayed. Only disabled components can be deleted.

  2. Click the checkbox beside the component you want to delete.

  3. Click Delete.
