GlassFish es un servidor de aplicaciones de código abierto que implementa las últimas funciones de la plataforma Java EE 5. La plataforma Java EE 5 incluye las últimas versiones de tecnologías como JavaServer Pages (JSP) 2.1, JavaServer Faces (JSF) 1.2, Servlet 2.5, Enterprise JavaBeans 3.0, Java API for Web Services (JAX-WS) 2.0, Java Architecture for XML Binding (JAXB) 2.0, Web Services Metadata para Java Platform 1.0, entre otras muchas.


¿Es nuevo en GlassFish? Aprenda a descargar e instalar GlassFish, pruebe la guía de procedimientos iniciales, vea algunas películas o ensaye nuestros ejemplos en el tutorial. Visite la página de presentación de GlassFish.

Descargar y ejecutar

Puede descargar la versión estable de las últimas versiones de acceso rápido del servidor de aplicaciones GlassFish. La descarga incluye una versión binaria del servidor de aplicaciones GlassFish Java EE 5, de manera que sólo tiene que seguir las sencillas instrucciones y ejecutarla. Hay descargas para ejecutar en los sistemas operativos más conocidos.
  • Consiga ya las descargas.

Aprender y escribir aplicaciones con las últimas tecnologías Java EE 5

El servidor de aplicaciones GlassFish implementa las últimas tecnologías Java EE 5. Por ejemplo, escriba una aplicación usando las últimas tecnologías de nivel de web, como JSF1.2. Además de probar las últimas tecnologías, puede ensayar sus aplicaciones J2EE 1.4 (o de versiones anteriores) en GlassFish y obtener un adelanto sobre la migración.
  • Consulte la Guía de procedimientos iniciales para GlassFish. Incluye instrucciones para instalar y ejecutar el servidor GlassFish, así como una pequeña aplicación de ejemplo para implementar.
  • Vaya a la página inicial de Java BluePrints, donde encontrará trabajos sobre Java EE 5. Incluye BluePrints para crear aplicaciones AJAX en la plataforma Java EE 5.
  • Desarrolle aplicaciones Java EE con NetBeans, usando uno de estos dos tutoriales, una aplicación sencilla del tipo "hola a todos" y otras más avanzadas, mediante Java EE 5 persistence.
  • Los javadocs están en línea. Se actualizan con cada compilación aprobada.
  • Plantee dudas en el foro
  • Compruebe la lista creciente de sugerencias y blogs técnicos con ejemplos de uso de las tecnologías Java EE 5 en GlassFish.
  • Compruebe The Aquarium para enterarse de más novedaes sobre las tecnologías relacionadas con GlassFish.
