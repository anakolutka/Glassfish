Affichage des descripteurs d'une application

  1. Dans l'arborescence de navigation, sélectionnez le nœud Applications.

    La page Applications s'affiche.

  2. Sur la page Applications , cliquez sur le nom de l'application dont vous affichez les descripteurs.

    La page Modification de l'application s'affiche.

  3. Sur la page Modification de l'application, cliquez sur l'onglet Descripteur.

    La page Descripteurs de modules s'affiche.

  4. Dans le tableau Descripteurs de déploiement, cliquez sur le nom du fichier XML à afficher.

    Le fichier XML s'affiche dans une nouvelle fenêtre.
