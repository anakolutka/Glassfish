To configure the JVM Profiler settings

A profiler tool generates data that is used to analyze performance and identify potential bottlenecks.

  1. In the Developer profile, select the Application Server node.
  2. In the Cluster profile, select the Configurations node Cluster.
  3. Click the JVM Setting node or tab, then the Profiler tab.
  4. Define the profiler settings as desired. The information you specify depends on which profiler product you are using.

    For examples and instructions, see Chapter 4, Debugging Applications, in Sun Java System Application Server 9.1 Developer’s Guide.

  5. Click Save.
  6. Restart the server.
