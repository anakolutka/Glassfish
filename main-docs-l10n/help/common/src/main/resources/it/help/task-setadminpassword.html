Per modificare la password dell'amministratore

  1. Selezionare il nodo Dominio nell'albero di navigazione.

    Viene visualizzata la pagina Attributi dominio.

  2. Selezionare la scheda Password amministratore.

    Viene visualizzata la pagina Nuova password amministratore.

    La pagina visualizza l'ID utente e il nome gruppo dell'account dell'amministratore corrente. Per impostazione predefinita l'ID utente è admin e il nome del gruppo è asadmin.

  3. Nel campo Nuova password digitare la nuova password per l'account dell'amministratore corrente.

  4. Nel campo Conferma nuova password digitare una seconda volta la nuova password.

  5. Fare clic su Salva.
