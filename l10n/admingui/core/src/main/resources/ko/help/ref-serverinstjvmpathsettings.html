JVM 클래스 경로 설정

클래스 경로는 Java 런타임 환경이 클래스 및 기타 자원을 찾는 JAR 파일의 목록입니다.

JVM 클래스 경로 설정 페이지에는 다음 옵션이 포함되어 있습니다.

환경 클래스 경로

무시 확인란을 선택하면 CLASSPATH 환경 변수가 무시됩니다. CLASSPATH 환경 변수는 프로그래밍의 기본 자습서에는 편리하지만 엔터프라이즈 환경에서는 사용하지 않는 것이 좋습니다.

서버 클래스 경로

Enterprise Server 클래스 경로입니다. 읽기 전용 필드입니다.

시스템 클래스 경로

시스템에 있는 클래스의 경로입니다.

클래스 경로 접두어

서버의 클래스 경로 앞에 한 줄당 하나씩 추가할 JAR 파일의 전체 경로입니다.

클래스 경로 접미어

서버의 클래스 경로 뒤에 한 줄당 하나씩 추가할 JAR 파일의 전체 경로입니다.

원시 라이브러리 경로 접두어

원시 라이브러리 경로에 한 줄당 하나씩 추가할 JAR 파일의 전체 경로입니다.

원시 라이브러리 경로 접미어

원시 라이브러리 경로에 한 줄당 하나씩 추가할 JAR 파일의 전체 경로입니다.

원시 라이브러리 경로는 서버의 원시 공유 라이브러리에 대한 상대적 경로, 표준 런타임 환경 원시 라이브러리 경로, 쉘 환경 설정(UNIX® 시스템에서는 LD_LIBRARY_PATH) 및 JVM 프로필러 설정 페이지에 지정된 모든 경로가 결합된 것입니다.
