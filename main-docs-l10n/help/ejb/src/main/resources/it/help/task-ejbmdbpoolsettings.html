Per configurare le impostazioni del pool MDB

Il pool per i bean basati su messaggi è simile al pool di bean di sessione. Per impostazione predefinita, il contenitore include un pool di bean basati su messaggi.

Per regolare la configurazione del pool, attenersi alla procedura riportata di seguito.

  1. Espandere il nodo Configurazione nell'albero di navigazione.

  2. Selezionare il nodo Contenitore EJB.

    Viene visualizzata la pagina Enterprise Java Bean (EJB).

  3. Fare clic sulla scheda Impostazioni MDB.

    Viene visualizzata la pagina Impostazioni del pool predefinito MDB.

  4. Nel campo Dimensione pool iniziale e minima digitare il numero minimo di bean di messaggio da gestire nel pool.

    Il valore predefinito è 0.

  5. Nel campo Dimensione massima pool digitare il numero massimo di bean che il contenitore può gestire contemporaneamente nel pool.

    Il valore predefinito è 32.

  6. Nel campo Quantità di ridimensionamento pool digitare il numero di bean rimossi dal pool se inattivi per un periodo di tempo superiore a quello specificato nel campo Timeout inattività pool.

    Il valore deve essere almeno 0 ed essere minore del valore Dimensione massima pool. Il valore predefinito è 8.

  7. Nel campo Timeout inattività pool digitare il periodo di tempo, in secondi, durante il quale un bean nel pool può rimanere inattivo prima di essere rimosso dal pool.

    Allo scadere di questo periodo, il bean viene eliminato. Il valore 0 indica che un bean può rimanere inattivo per un periodo di tempo indeterminato. Il valore predefinito è 600.

  8. Nella sezione Proprietà aggiuntive, specificare le proprietà aggiuntive.

    GlassFish Server non definisce alcuna proprietà aggiuntiva per il pool MDB.

    • Per aggiungere una proprietà, fare clic sul pulsante Aggiungi proprietà. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Nome e il valore della proprietà nel campo Valore.

    • Per modificare una proprietà, è necessario modificare il valore del campo Valore della proprietà.

    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Nome corrispondente alla proprietà da eliminare, quindi fare clic sul pulsante Elimina proprietà.

  9. Fare clic su Salva.

    Suggerimento:

    Per ripristinare le impostazioni predefinite, fare clic sul pulsante Carica impostazioni predefinite, quindi su Salva.

  10. Riavviare GlassFish Server.
