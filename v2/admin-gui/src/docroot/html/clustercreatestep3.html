To Create a High-Availability Database while Creating the Cluster

As part of the cluster creation process, you can create a high-availability database (HADB). If you do not want to create an HADB at this time, click Finish to complete the cluster creation. . You can create an HADB later.

  1. Click the "Use a high-availability database for this cluster" checkbox.

  2. Click Add Nodes to add paired database nodes.

  3. In the First Mirrored Node and the Paired Mirrored Node fields, enter the host names of the machines on which to create the nodes.

    The HADB management agent for these hosts must be running in order for creation to be successful.

  4. Add a port base and device size.

  5. Click Finish to create the cluster and the optional load balancer and high-availability database.
