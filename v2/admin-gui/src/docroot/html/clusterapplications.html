To Configure Applications for a Cluster

  1. In the tree component, expand the Clusters node.
  2. Select the node for the cluster.
  3. Click the Applications tab to bring up the Applications page.

    On this page, you can:

    • Select the check box next to the application and click Deploy to deploy that application. . On the Deployment page that appears, enter the location of the application, name for the application, and the target.

    • From the Filter pull-down list, select a type of application to display in the list.

    • To edit an application, click the application name.

    • Select the checkbox next to an application and from the More Actions pull-down menu, choose Enable or Disable to enable or disable the application for the cluster.

      If the cluster has a load balancer, select the checkbox next to an application and from the More Actions pull-down menu, choose Load Balancer Enable or Load Balancer Disable to enable or disable load balancing for the application.
