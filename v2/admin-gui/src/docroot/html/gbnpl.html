To Configure Call Flow Monitoring

Call Flow Monitoring provides an overview of the success and response time details for each server instance.

  1. In the tree component, select the Application Server node.

  2. Click the Monitor tab.

  3. Click the Call Flow tab.

    The Call Flow Configuration page displays.

  4. Select Enabled to initiate call flow monitoring for the server.

  5. Specify the client host IP address and user ID for which you want to enable monitoring.

    You can specify either the client host IP address or the user's ID. Specify both IP address and user ID to filter for an exact match of user ID and IP address.

  6. Click Save

More Information
To Monitor Call Flow Data

The call flow data will be displayed in the table on this page.

  • Select the type of data you want to view from the Filter drop-down list.

  • Select Refresh to get the most recent data.

  • Use the Delete button to clear individual requests.

  • To remove all data, use the Clear Data button. Data can be cleared only when Call Flow is disabled.
