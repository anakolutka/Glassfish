추가 등록 정보

추가 영역 등록 정보입니다.

  • 등록 정보를 추가하려면 등록 정보 추가 버튼을 누릅니다. 빈 행이 표시되면 이름 필드에 등록 정보 이름을 입력하고 값 필드에 등록 정보 값을 입력합니다. 저장을 누릅니다.

  • 등록 정보를 수정하려면 등록 정보의 값 필드를 편집합니다. 저장을 누릅니다.

  • 등록 정보를 삭제하려면 삭제할 등록 정보의 이름 필드 왼쪽에 있는 확인란을 선택한 다음 등록 정보 삭제 버튼을 누릅니다. 저장을 누릅니다.

  • 파일 영역 등록 정보에 대한 설명은 FileRealm 클래스와 관련된 등록 정보를 참조하십시오.

  • 인증서 영역 등록 정보에 대한 설명은 CertificateRealm 클래스와 관련된 등록 정보를 참조하십시오.
