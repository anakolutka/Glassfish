ドメイン属性

このページを使用して、Enterprise Server に関するドメインの詳細属性を設定します。

アプリケーションルート

アプリケーションが配備されるディレクトリの完全パス。

ログルート

サーバーのログファイルを保持する場所を指定します。

ロケール

このフィールドを空白のままにすると、ホストのデフォルトのロケールが使用されます。

ロケールは、言語と地域の特定の組み合わせを指定する識別子です。たとえば、米国英語のロケールは en_US で、日本語のロケールは ja_JP です。英語以外のロケールを使用するには、英語を目的の言語に翻訳して、アプリケーションサーバー をローカライズする必要があります。
