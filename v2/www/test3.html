GlassFish Distributions

Best When: You Need Commercial Support or Other Vendor Added Value

These are the known distributions which use GlassFish and have gone through additional testing and may have additional content.

Sun's Distributions

Sun Microsystems donated the initial code for the GlassFish Application Server, and continues as one of the project's largest and most active sponsors. It distributes GlassFish in many offerings. These include the Sun Java System Application Server, the Java EE 5 SDK, and the Java Application Platform SDK.

NetBeans Java EE SDK co-bundle

The NetBeans Project provides a bundle containing the award-winning NetBeans 5.5 IDE and the Sun Java System Application Server 9.0 U1 distribution of GlassFish.
