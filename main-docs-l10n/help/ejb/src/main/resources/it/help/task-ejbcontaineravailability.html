Per configurare le impostazioni di disponibilità per il contenitore EJB

  1. Espandere il nodo Configurazione nell'albero di navigazione.

  2. Selezionare la configurazione da modificare.

  3. Nel nodo della configurazione, selezionare il nodo Servizio disponibilità.

    Viene visualizzata la pagina Servizio disponibilità.

  4. Nella pagina Servizio disponibilità fare clic sulla scheda Disponibilità contenitore EJB.

    Viene visualizzata la pagina Disponibilità contenitore EJB.

  5. Selezionare la casella Abilitato dell'opzione Servizio disponibilità per abilitare la disponibilità per il contenitore EJB.

    Per disabilitare la disponibilità, rimuovere il segno di spunta dalla casella Servizio disponibilità abilitato.

  6. Il tipo di persistenza HA è replicated.

    Si tratta del meccanismo di persistenza utilizzato quando la casella di controllo Abilitato dell'opzione Servizio disponibilità è selezionata. La persistenza si basa sul failover in un cluster. Non modificare questo valore.

  7. Il tipo di persistenza SFSB è file.

    Si tratta del meccanismo di persistenza utilizzato quando la casella di controllo Abilitato dell'opzione Servizio disponibilità non è selezionata. La persistenza si basa sul file system. Non modificare questo valore.

    Il campo Posizione di memorizzazione sessione determina la posizione in cui gli stati dei checkpoint SFSB vengono resi persistenti. Per impostare il campo Posizione di memorizzazione sessione, vedere Per configurare le impostazioni EJB.

  8. Per ripristinare i valori predefiniti, fare clic sul pulsante Carica impostazioni predefinite.

  9. Nella sezione Proprietà aggiuntive, specificare le proprietà aggiuntive.

    GlassFish Server non definisce alcuna proprietà aggiuntiva per la disponibilità del contenitore EJB.

    • Per aggiungere una proprietà, fare clic sul pulsante Aggiungi proprietà. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Nome e il valore della proprietà nel campo Valore.

    • Per modificare una proprietà, è necessario modificare il valore del campo Valore della proprietà.

    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Nome corrispondente alla proprietà da eliminare, quindi fare clic sul pulsante Elimina proprietà.

  10. Fare clic su Salva.
