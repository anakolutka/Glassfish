批处理作业执行详细信息

“批处理作业执行详细信息”页显示有关特定批处理作业执行的详细信息。将显示以下信息。

实例名称
集群名称

该配置应用到的 GlassFish 集群或服务器实例的名称。

作业名称

批处理作业的名称。

执行 ID

分配给批处理作业执行的 ID。新执行在作业首次启动以及现有执行每次重新启动时创建。

步骤计数

批处理作业执行中的步骤数。

批处理状态

批处理运行时所设置的批处理作业执行的状态。

退出状态

由作业 XML 为作业设置的批处理作业执行状态或者由批处理应用程序设置的批处理作业执行状态。默认情况下，退出状态和批处理状态相同，除非显式覆盖了退出状态。

开始时间

批处理作业执行的开始时间。

结束时间

批处理作业执行的结束时间。

作业参数

传递到批处理运行时以便用于批处理作业执行的属性，按名称/值对的形式列出。
