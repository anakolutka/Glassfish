Configuration de la machine virtuelle Java

La machine virtuelle JavaTM (JVMTM) permet d'exécuter du code binaire Java sur une machine physique. Vous pouvez régler les paramètres de JVM pour améliorer les performances et empêcher des erreurs d'allocation de mémoire de se produire.

La configuration de JVM comprend les tâches suivantes :

  • Configurer les paramètres généraux du profileur JVM

  • Configurer les paramètres du chemin d'accès à la JVM

  • Configurer les options JVM

  • Désactiver le gestionnaire de sécurité

  • Configurer les paramètres du profileur JVM

  • Définir la configuration avancée des applications

  • Définir les attributs avancés du domaine

Remarque – Les termes « Machine Java Virtual » et « JVM » désignent une machine virtuelle pour plate-forme Java.
