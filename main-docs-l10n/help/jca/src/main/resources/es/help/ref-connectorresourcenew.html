Nuevo recurso de conector

Utilice la página Nuevo recurso de conector para crear un nuevo recurso de conector.

La página Nuevo recurso de conector contiene las siguientes opciones.

Nombre JNDI

Un nombre único que identifica el recurso de conector.

Nombre de pool

El pool de conexiones al que pertenece el nuevo recurso de conector.

Descripción

Descripción del recurso de conector.

Estado

Si está opción está seleccionada, entonces el recurso de conector está activado. Esta opción se encuentra activada por defecto.

Propiedades Adicionales

Propiedades adicionales para el recurso de conector. Las propiedades disponibles dependen del adaptador de recursos para el pool de conexiones de conector. Los pares nombre-valor especificados en esta tabla se pueden usar para sustituir los valores por defecto de las propiedades definidas por el proveedor del adaptador de recursos.

Destinos

Clusters e instancias independientes para el recurso de conector. Mueva los destinos seleccionados a la columna Destinos Seleccionados mediante los botones Agregar y Agregar Todos. Mueva los destinos que no necesite a la columna Destinos Disponibles mediante los botones Eliminar y Eliminar Todos.

Esta opción sólo se muestra si en el dominio se han creado clusters o instancias independientes.
