Création d'une ressource JDBC

Une ressource JDBC (Java DataBase Connectivity) (source de données) fournit des applications dans le but d'établir une connexion à une base de données.

Avant de commencer

Pour créer une ressource JDBC, vous devez commencer par créer un pool de connexions JDBC. Reportez-vous à Création d'un pool de connexions JDBC.

  1. Dans l'arborescence de navigation, développez le noeud Ressources.

  2. Sous le noeud Ressources, développez le noeud JDBC.

  3. Sous le noeud JDBC, sélectionnez le noeud Ressources JDBC.

    La page Ressources JDBC apparaît.

  4. Dans la page Ressources JDBC, cliquez sur Nouveau.

    La page Nouvelle ressource JDBC apparaît.

  5. Dans le champ Nom JNDI, attribuez un nom unique à la ressource.

    Le nom JNDI organise et localise les composants dans un environnement informatique distribué de la même manière que des catalogues sur fiches organisent et représentent des emplacements de livres dans une bibliothèque. Par conséquent, le nom JNDI est une méthode importante d'accès à la ressource JDBC. Par convention, le nom commence par la chaîne jdbc/, par exemple jdbc/payrolldb. N'oubliez pas la barre oblique.

    Le nom ne peut contenir que des caractères alphanumériques, des traits de soulignement, des tirets ou des points.

  6. Dans la liste déroulante Nom du pool, sélectionnez un pool de connexions à associer à la nouvelle ressource JDBC.

  7. Dans le champ Description, entrez une brève description de la nouvelle ressource JDBC.

  8. Cochez la case Activé du champ Statut pour rendre la ressource disponible lors de l'exécution.

    Cette option est activée par défaut.

  9. Dans la section Autres propriétés, spécifiez les propriétés supplémentaires.

    GlassFish Server ne définit aucune autre propriété pour les ressources JDBC.

    Pour ajouter une propriété, cliquez sur le bouton Ajouter une propriété. Sur la ligne vide qui apparaît, saisissez le nom de la propriété dans le champ Nom et la valeur de la propriété dans le champ Valeur.

  10. Sélectionnez une cible disponible pour l'ajouter à la liste Cibles sélectionnées ou l'en enlever.

  11. Cliquez sur OK.
