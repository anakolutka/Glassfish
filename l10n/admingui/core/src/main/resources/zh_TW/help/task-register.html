利用 Sun Connection 註冊 Sun GlassFish Enterprise Server

成為 Sun GlassFishTM Enterprise Server 的註冊使用者後，即可獲得以下好處：

  • 可得知修補程式資訊和錯誤更新

  • 可獲得螢幕擷取教學和指導

  • 可得知新聞和活動

  • 可獲得支援和訓練課程

若要於 Sun Connection 中註冊您的 Sun GlassFish Enterprise Server 安裝，您必須有 Sun Online 帳號。如果您沒有 Sun Online 帳號，Enterprise Server 可讓您在註冊期間建立一個帳號。

備註 - 如果您在註冊 Enterprise Server 時建立了 Sun Online 帳號，則系統會提示您僅提供所需的資訊。註冊完 Enterprise Server 之後，您可以更新您的 Sun Online 帳號來提供其他選擇性資訊。

開始之前

如果執行 Enterprise Server 的機器透過代理伺服器連線至網際網路，請取得代理伺服器的主機名稱和連接埠號碼。如果 Web 瀏覽器正在執行 Enterprise Server 的機器上執行，您可以從 Web 瀏覽器取得此資訊。否則，請詢問您的系統管理員，以取得此資訊。

  1. 在瀏覽樹狀結構中，選取 [註冊] 節點。

    [產品註冊] 頁面會隨即開啟。

  2. 在 [產品註冊] 頁面中，提供所需的資訊。
    • 如果您已具有 Sun Online 帳號，請依照如下方法提供所需的資訊：
      1. 選取 [我已經擁有 Sun Online 帳號或 Sun 開發社群 (SDN) 帳號] 選項。
      2. 在 [使用者名稱] 欄位中，鍵入您在註冊 Sun Online 帳號時指定的使用者名稱。
      3. 在 [密碼] 欄位中，鍵入您在註冊 Sun Online 帳號時指定的密碼。
    • 如果您沒有 Sun Online 帳號，請依照如下方法提供所需的資訊：
      1. 選取 [我沒有 Sun Online 帳號] 選項。
      2. 在 [電子郵件地址] 欄位中，鍵入您的電子郵件地址。

        您的電子郵件地址即為您的 Sun Online 帳號使用者名稱。

      3. 在 [密碼] 欄位中，鍵入要用於 Sun Online 上帳號的密碼。

        密碼要區分大小寫，並且長度必須為 6 至 10 個字元。

      4. 在 [再次鍵入密碼] 欄位中，再次鍵入您在上一步驟中所鍵入的密碼。
      5. 在 [名字] 欄位中，鍵入您的名字。
      6. 在 [姓氏] 欄位中，鍵入您的姓氏。
      7. 在 [公司] 欄位中，鍵入貴公司的名稱 (如果適用)。
      8. 從下拉式清單中，選取您所居住的國家或地區。
  3. 如果執行 Enterprise Server 的機器透過代理伺服器連線至網際網路，請提供代理伺服器的主機名稱和連接埠號碼。

    如果執行 Enterprise Server 的機器直接連線至網際網路，則請略過此步驟。

    Web 瀏覽器可能正在執行 Enterprise Server 的機器上執行。在此情況下，您可以從 Web 瀏覽器取得代理伺服器的主機名稱和連接埠號碼。否則，請詢問您的系統管理員，以取得此資訊。

  4. 按一下 [註冊]。

    如果 [支援] 頁面開啟，則可以確認註冊已完成。
