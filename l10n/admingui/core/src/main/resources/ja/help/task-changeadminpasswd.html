管理レルムのユーザーのパスワードを変更する

現在の管理者アカウントのパスワードを変更するには、「管理者パスワードを変更する」を参照してください。admin-realm に複数のユーザーがある場合は、ほかのユーザーのパスワードも変更できます。

管理者パスワードを暗号化すると、アプリケーションのセキュリティーが向上します。セキュリティー設定手順の詳細については、『Sun GlassFish Enterprise Server v3 Administration Guide』の第 11 章「Administering System Security」を参照してください。

  1. ナビゲーションツリーで、「設定」ノードを展開します。
  2. 「セキュリティー」ノードを開きます。
  3. 「レルム」ノードを開きます。
  4. 「admin-realm」ノードを選択します。

    「レルムを編集」ページが開きます。

  5. 「レルムを編集」ページで、「ユーザーを管理」ボタンをクリックします。

    「ファイルユーザー」ページが開きます。

  6. 「ファイルユーザー」ページで、パスワードを追加する、または変更するユーザーをクリックします。

    「file レルムユーザーを編集」ページが開きます。

  7. 「新規パスワード」および「新しいパスワードを確認」フィールドに、新しいパスワードを入力します。
  8. 「保存」をクリックします。
