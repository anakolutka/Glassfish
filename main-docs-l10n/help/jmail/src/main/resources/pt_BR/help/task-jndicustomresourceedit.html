Para Editar um Recurso JNDI Personalizado

  1. Na árvore de navegação, expanda o nó Recursos.

  2. No nó Recursos, expanda o nó JNDI.

  3. No nó JNDI, selecione Recursos Personalizados.

    A página Recursos Personalizados é aberta.

  4. Na página Recursos Personalizados, clique no nome do recurso a ser editado.

    A página Editar Recurso Personalizado será aberta.

    Observação:

    O nome JNDI é um campo somente para leitura. Você pode especificar o nome JNDI ao criar um novo recurso personalizado.

  5. Para modificar o Tipo de Recurso, efetue um dos procedimentos a seguir.

    • Na lista drop-down, selecione uma definição de tipo.

    • Selecione o segundo botão de opção e digite uma definição de tipo totalmente qualificada que siga o formato xxx.xxx.

  6. No campo Classe de Fábrica, digite um nome de classe de fábrica para o recurso personalizado.

    A Classe de Fábrica é o nome especificado de usuário para a classe de fábrica. Essa classe implementa a interface javax.naming.spi.ObjectFactory.

  7. No campo Descrição, digite uma descrição curta do recurso.

    Esta descrição é um valor de string e pode incluir no máximo 250 caracteres.

  8. Marque a caixa de seleção Status ativo para ativar o recurso.

    Por default, o recurso está ativado, significando que está disponível assim que é criado.

  9. Na seção Propriedades Adicionais, especifique propriedades adicionais do recurso personalizado.

    O GlassFish Server não define propriedades adicionais para recursos personalizados.

    • Para adicionar uma propriedade, clique no botão Adicionar Propriedade. Na linha em branco que for exibida, digite o nome da propriedade no campo Nome e digite o valor da propriedade no campo Valor.

    • Para modificar uma propriedade, edite o campo Valor da propriedade.

    • Para deletar uma propriedade, marque a caixa de seleção à esquerda do campo Nome da Propriedade a ser deletada. Clique no botão Deletar Propriedades.

  10. Clique em Salvar.
