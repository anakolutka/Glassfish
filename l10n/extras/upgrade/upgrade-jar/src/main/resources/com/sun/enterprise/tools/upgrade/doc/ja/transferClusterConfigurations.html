「クラスタ設定の転送」画面

Sun Java System Application Server 8.x Enterprise Edition (EE) のクラスタ設定を Sun Java System Application Server 9.1 にアップグレードするには、有効な clinstance.conf ファイルが必要です。アップグレードツールは、 clinstance.conf ファイルが 8.x EE クラスタを正確に反映していることを想定しています。何らかの理由で、このファイルが実際のクラスタ設定を正確に表していない場合は、アップグレードを実行する前にクラスタ定義を手動で編集する必要があります。

「クラスタ設定の転送」画面を使用して、新しいインストール環境への移行が必要なクラスタ設定ファイルを指定します。

  1. 「追加」ボタンを押すと、「clinstance.conf ファイルを選択します」ダイアログが表示されます。
  2. 移行するクラスタ設定情報を含む clinstance.conf ファイルを選択します。
  3. 「開く」ボタンを押します。
  4. この処理を繰り返して、移行が必要なクラスタ設定ファイルをすべて追加します。
  5. すべてのファイルを追加したら、「次へ」ボタンを押します。「アップグレードの結果」画面が表示され、アップグレードの状態が示されます。
