Modules de cycle de vie

La page Modules de cycle de vie affiche la liste des modules de cycle de vie disponibles.

Un module de cycle de vie accomplit certaines tâches lorsqu'il est déclenché par un ou plusieurs événements dans le cycle de vie du serveur. Les événements de serveur qui peuvent être déclencheurs sont : l'initialisation, le démarrage, les requêtes prêt pour la maintenance et l'arrêt. Les modules de cycle de vie ne font pas partie de la spécification Java EE, mais optimisent le fonctionnement du Enterprise Server.

Pour chaque module de cycle de vie, les informations suivantes sont fournies.

Nom

Nom du module de cycle de vie.

Activée

True si le module de cycle de vie est activé, False s'il ne l'est pas.

Ordre de chargement

Valeur indiquant l'ordre relatif dans lequel le module de cycle de vie doit être chargé.

Nom de classe

Fichier de classe du module de cycle de vie.
