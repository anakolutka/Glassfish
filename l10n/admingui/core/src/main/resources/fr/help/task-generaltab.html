Pour afficher des Informations générales sur le Enterprise Server

  1. Dans l'arborescence de navigation, sélectionnez le nœud Enterprise Server.

    La page Informations générales s'affiche.

  2. Sur la page Informations générales, procédez à l'une des étapes suivantes.
    • Cliquez sur le bouton Stop pour arrêter le Enterprise Server.
    • Cliquez sur le bouton Redémarrer pour redémarrer le Enterprise Server.
    • Cliquez sur le bouton Afficher les fichiers journaux pour afficher le journal du serveur.

      Consultez Afficher le journal du serveur pour en savoir plus.

    • Cliquez sur Rapport JVM pour ouvrir une fenêtre distincte qui vous permet d'afficher des rapports sur la Java Virtual Machine, notamment un rapport de synthèse, un rapport de gestion de la mémoire, un rapport de chargement de classe et un vidage des threads en cours.
