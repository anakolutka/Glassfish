Modifica risorsa di destinazione JMS

Utilizzare la pagina Modifica risorsa di destinazione JMS per modificare le impostazioni per una risorsa di destinazione JMS.

La pagina Modifica risorsa di destinazione JMS contiene le opzioni riportate di seguito.

Carica impostazioni predefinite

Pulsante per ripristinare i valori predefiniti delle impostazioni che dispongono di valori predefiniti. Le impostazioni che non dispongono di valori predefiniti non vengono modificate.

Nome JNDI

Il nome univoco che identifica la risorsa di destinazione. Il campo Nome JNDI è di sola lettura. È possibile specificare il nome JNDI solo quando si crea una nuova risorsa di destinazione JMS.

Nome destinazione fisica

Il nome della destinazione fisica associata a questa risorsa.

GlassFish Server crea in modo automatico la destinazione fisica quando è necessaria e la elimina quando si elimina la risorsa di destinazione.

Tipo di risorsa

Il tipo di risorsa di destinazione. Le scelte disponibili sono javax.jms.Topic e javax.jms.Queue.

Ordine di distribuzione

L'ordine di distribuzione della risorsa di destinazione.

Descrizione

La descrizione della risorsa di destinazione.

Stato

Indica se la risorsa è disponibile in fase di esecuzione.

Se nel dominio esiste solo l'istanza server predefinita server, è possibile abilitare o disabilitare la disponibilità in fase di esecuzione della risorsa. La disponibilità in fase di esecuzione è abilitata per impostazione predefinita.

Se nel dominio sono stati creati cluster o istanze standalone, utilizzare la scheda Destinazione per controllare la disponibilità in fase di esecuzione della risorsa nelle relative destinazioni.

Proprietà aggiuntive

Proprietà aggiuntive per la risorsa di destinazione. GlassFish Server non definisce alcuna proprietà aggiuntiva per le risorse di destinazione JMS.
