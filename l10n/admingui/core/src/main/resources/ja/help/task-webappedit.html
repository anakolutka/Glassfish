配備されている Web アプリケーションを編集する

  1. ナビゲーションツリーで、「アプリケーション」ノードを展開します。
  2. 「Web アプリケーション」ノードを選択します。
  3. 「配備されている Web アプリケーション」の表から、編集するアプリケーションの名前をクリックします。

    「Web アプリケーション」ページが開きます。

  4. 「コンテキストルート」フィールドに、この Web アプリケーションのパスを入力します。

    Web アプリケーションの URL では、コンテキストルートはポート番号の直後に続きます (http://host:port/context-root/...)。たとえば /hello のように、コンテキストルートがスラッシュで始まるようにしてください。

  5. 「説明」フィールドに、この Web アプリケーションの説明を入力します。
  6. 「保存」をクリックします。
