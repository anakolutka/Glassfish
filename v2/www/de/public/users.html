GlassFish ist ein kostenloser Open Source-Anwendungsserver, der die neuesten Features der Java EE 5-Plattform implementiert. Die Java EE 5-Plattform umfasst neueste Versionen von Technologien wie JavaServer Pages (JSP) 2.1, JavaServer Faces (JSF) 1.2, Servlet 2.4, Enterprise JavaBeans 3.0, Java API for Web Services (JAX-WS) 2.0, Java Architecture for XML Binding (JAXB) 2.0, Web Services Metadata für Java 1.0 sowie zahlreiche weitere Technologien.


Neu bei GlassFish? Erfahren Sie, wie Sie GlassFish herunterladen und installieren, machen Sie erste Schritte mit GlassFish, zeigen Sie einige Screencasts an, und testen Sie die Beispiele im Lernprogramm. Hier finden Sie einen Überblick über die GlassFish-Seite.

Download und Ausführung

Sie können die stabile oder neuesten Early-Access-Versionen des GlassFish-Anwendungsservers herunterladen. Der Download umfasst eine Binärversion des GlassFish Java EE 5-Anwendungsservers, Sie müssen lediglich den Anweisungen folgen und den Server ausführen. Es stehen Downloads für die Ausführung auf allen gängigen Betriebssystemen bereit.
  • Hier gelangen Sie zum Download.

Kennenlernen und Schreiben von Anwendungen mit den neuesten Java EE 5-Technologien

Der GlassFish-Anwendungsserver implementiert sämtliche neueste Java EE 5-Technologien. Schreiben Sie beispielsweise eine Anwendung unter Verwendung aktuellster Web-Tier-Technologien wie z. B. JSF 1.2. Zusätzlich zu den aktuellsten Technologien können Sie vorhandene J2EE 1.4-Anwendungen (oder frühere Versionen) auf GlassFish testen und eine Migration durchführen.
  • Lesen Sie die Informationen im Dokument Erste Schritte mit GlassFish. Hier finden Sie Anweisungen für die Installation und Ausführung des GlassFish-Servers sowie eine kleine Beispielanwendung für die Bereitstellung.
  • Auf der Java BluePrints-Seite finden Sie Beispiele zur Verwendung von Java EE 5, etwa BluePrints für die Erstellung von AJAX-Anwendungen auf der Java EE 5-Plattform.
  • Entwickeln Sie Java EE-Anwendungen mit NetBeans unter Verwendung zweier Lernprogramme, einer einfachen Hello World-Anwendung und einer erweiterten Anwendung mit Nutzung der Java EE 5 Persistence-API.
  • In den javadocs können Sie sich online informieren, diese werden mit jedem Promoted-Build aktualisiert.
  • Im Forum haben Sie die Möglichkeit, Fragen zu stellen
  • Die Liste der technischen Tipps und Blogs stellt Beispiele zur Verwendung der Java EE 5-Technologien in GlassFish bereit.
  • In The Aquarium finden Sie Nachrichten zu anderen Technologien mit Bezug zu GlassFish.

GlassFish-Dokumentationsentwürfe

Das GlassFish-Team umfasst auch Dokumentationsautoren, die ihre Entwürfe für die Community bereitstellen. Die Dokumente werden während der Entwicklung von Release V2 aktualisiert, um zusätzliche Features aufzunehmen.
  • Auf der Dokumentationsseite finden Sie Verweise auf die verfügbare Dokumentation. Wir freuen uns über Ihr Feedback!

Tragen Sie zur GlassFish-Community bei

Wir würden uns freuen, wenn Sie sich am Projekt GlassFish beteiligen. Sie haben dazu verschiedene Möglichkeiten:
  • Lassen Sie es uns wissen, wenn Sie Ihre Java EE-Anwendung auf GlassFish ausführen
  • Nehmen Sie an einer Diskussion im Forum teil, und tauschen Sie sich mit anderen GlassFish-Benutzern aus
  • Melden Sie einen Bug
  • Fordern Sie ein Feature an
  • Entwickeln Sie Tests -- weitere Informationen finden Sie im GlassFish-Qualitätsportal
  • Einchecken von Code für den GlassFish-Anwendungsserver
  • Zu GlassFish beitragen/Entwicklen von Java EE-Beispielen
  • Treten Sie dem CATfish-Programm bei

Ausführen gängiger Open Source-Frameworks oder -Anwendungen

Auf GlassFish können viele gängige Open Source-Frameworks und -Anwendungen ausgeführt werden. Prüfen Sie, ob Ihre bevorzugte Anwendung getestet wurde. Sie können Ihre Anwendung nicht finden? Versuchen Sie es, und lassen Sie uns das Ergebnis wissen.
