To view, deploy, and manage deployed web applications

  1. To control the level of detail you want to view for each deployed application, from the View drop-down list, select Summary or Detailed.
  2. To deploy a new web application, click Deploy. On the Deploy Enterprise Applications/Modules page, provide details of the application you want to deploy. For more details, see To deploy a web application.
  3. To edit a web application, click the name of the application. The General tab of the Web Application page appears. For more details, see To edit a deployed web application.
  4. To undeploy an application, click the check box next to the required application and click Undeploy.
  5. To enable or disable an application or module on all targets, click the check box next to the required application and click Enable or Disable, as required.
  6. To launch an application after it is deployed, click Launch in the row corresponding to the required application. The application opens in a new browser window.
