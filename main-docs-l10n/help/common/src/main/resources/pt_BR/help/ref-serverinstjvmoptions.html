Opções JVM

Use a página Opções de JVM para especificar as opções do acionador de aplicações Java (java) que executa o GlassFish Server. As opções -D designam propriedades que são específicas ao GlassFish Server.

  • Para modificar uma opção, edite o campo Valor dessa opção.

  • Para adicionar uma opção, clique no botão Adicionar Opção JVM. Na linha em branco que for exibida, digite as informações da opção do campo Valor.

    Se as informações da opção contiverem um ou mais espaços, coloque toda a string entre aspas.

  • Para deletar uma opção, marque a caixa de seleção à esquerda do campo Valor (referente à opção a ser deletada) e clique em Deletar.
