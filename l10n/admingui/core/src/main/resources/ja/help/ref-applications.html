アプリケーション

「アプリケーション」ページには、Enterprise Server に配備されているアプリケーションの一覧が表示されます。すでに配備されているアプリケーションを表示および管理し、その他のアプリケーションを配備できます。

アプリケーションごとに、次の情報が表示されます。

名前

アプリケーションの名前。

使用可能

アプリケーションが使用可能な場合はチェックマーク、使用不可の場合は X を表示します。

複合

アプリケーションが複数のコンポーネントで構成されている場合は True、1 つのコンポーネントのみで構成されている場合は False です。

エンジン

アプリケーションで使用されるコンテナのタイプ。コンテナのタイプは以下のいずれかになります。

  • Web

  • webservices

  • ejb

  • connector

  • appclient

  • jruby

アクション

配備したコンポーネントに実行できるアクションへのリンク。再配備と再起動はすべてコンポーネントに対して実行できます。Web アプリケーションの場合はさらに起動を実行できます。

配備する

エンタープライズアプリケーションを配備するボタン。

配備取消し

エンタープライズアプリケーションの配備を取り消すボタン。

有効

選択したエンタープライズアプリケーションを有効にするボタン。

無効

選択したエンタープライズアプリケーションを無効にするボタン。

フィルタ

エンジンでアプリケーションをフィルタする際に使用するドロップダウンリスト。
