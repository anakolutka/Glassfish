Per eliminare un trasporto

  1. Espandere il nodo Configurazione nell'albero di navigazione.

  2. Nel nodo Configurazione espandere il nodo Configurazione di rete.

  3. Nel nodo Configurazione di rete selezionare il nodo Trasporti.

    Viene visualizzata la pagina Trasporti.

  4. Nella pagina Trasporti selezionare la casella di controllo a sinistra del protocollo da eliminare.

  5. Fare clic su Elimina.
