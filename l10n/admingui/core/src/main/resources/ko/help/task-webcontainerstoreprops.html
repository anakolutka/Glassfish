저장소 등록 정보를 구성하는 방법

  1. 탐색 트리에서 구성 노드를 선택합니다.
  2. 구성 노드 아래에서 웹 컨테이너 노드를 선택합니다.

    웹 컨테이너 페이지가 열립니다.

  3. 웹 컨테이너 페이지에서 저장소 등록 정보 탭을 누릅니다.
  4. 리프 간격 필드에 비활성 세션 데이터가 저장소에서 삭제되기 전까지의 시간(초)을 입력합니다.
  5. 디렉토리 필드에 세션 데이터가 저장될 파일의 경로를 입력합니다.

    지속성 유형으로 파일을 선택한 경우에만 필요합니다.

  6. (옵션) 추가 등록 정보 섹션에서 추가 등록 정보를 지정합니다.
  7. 저장을 누릅니다.
참조

  • 세션 시간 초과 값을 설정하는 방법

  • 관리자 등록 정보를 구성하는 방법
