Suppression d'une session JavaMail

  1. Dans l'arborescence de navigation, développez le noeud Ressources.

  2. Sous le noeud Ressources, sélectionnez Sessions JavaMail.

    La page Sessions JavaMail apparaît.

  3. Sur la page Sessions JavaMail, cochez la case située en regard du nom de la session à supprimer.

  4. Cliquez sur Supprimer.
