授予對管理工具的存取權限

只有 asadmin 群組中的使用者才能存取管理主控台和 asadmin 指令行公用程式。若要授予使用者對這些管理工具的存取權限，請將他們增加到 admin-realm 中的 asadmin 群組內。

  1. 在瀏覽樹狀結構中，展開 [配置] 節點。
  2. 若將網域配置為支援叢集，請選取您要變更的配置。

    例如，透過 cluster 設定檔或 enterprise 設定檔建立的網域。

  3. 展開 [安全性] 節點。
  4. 展開 [範圍] 節點。
  5. 選取 [admin-realm] 節點。
  6. 在 [編輯範圍] 頁面中按一下 [管理使用者] 按鈕。

    剛安裝完之後，admin-keyfile 檔案中會列出安裝期間所輸入的管理員使用者名稱和密碼。依預設，該使用者屬於 asadmin 群組，有權修改 Enterprise Server。僅當需要授予使用者 Enterprise Server 的管理員特權時，才將他們指定到該群組。

    如果將使用者增加到 admin-realm 範圍，但是將某個使用者指定到 asadmin 以外的群組，則該使用者的資訊仍會寫入名為 admin-keyfile 的檔案，但是該使用者無權存取管理工具或 file 範圍中的應用程式。

  7. 按一下 [新建] 以將新的使用者增加到 admin-realm 範圍。
  8. 輸入使用者 ID、群組清單和密碼。

    若要授予使用者對 Enterprise Server 的修改權限，請將 asadmin 群組納入 [群組清單] 中。

  9. 按一下 [確定] 以將該使用者增加到 admin-realm 範圍，或者按一下 [取消] 以直接退出而不進行儲存。
