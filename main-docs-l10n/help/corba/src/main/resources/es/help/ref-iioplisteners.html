Listeners IIOP

Utilice la página Listeners IIOP para configurar los listeners IIOP.

Se pueden configurar varios listeners IIOP para ORB, cada una de los cuales puede aceptar conexiones en diferentes direcciones de red y combinaciones de puertos. Los listeners se pueden configurar de forma diferente para las comunicaciones segura y la autenticación de clientes.

Por defecto, hay tres listeners IIOP configurados: orb-listener-1, SSL, y SSL_MUTUALAUTH.

Para cada listener IIOP, la página Listeners IIOP contiene la siguiente información.

Nombre

Nombre del listener IIOP.

Activado

"True" si el listener IIOP está activado o "false" si no lo está.

Dirección de red

Dirección IP (Protocolo de Internet) o nombre de host que se pueda resolver con DNS para el listener IIOP.

Puerto del Listener

Número de puerto del listener IIOP.
