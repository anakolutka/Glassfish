SSL

Utilice la página SSL para modificar la configuración de SSL para un protocolo de red.

La configuración de esta página únicamente tiene sentido si se encuentra activada la seguridad en la página Editar Protocolo.

La página SSL contiene las siguientes opciones.

Nombre de Configuración

Nombre de la configuración a la que se aplican los valores de esta página. Este campo es de sólo lectura.

SSL3

Si se activa esta casilla de control, se activará el protocolo SSL3 para el protocolo. Esta opción se encuentra activada por defecto.

TLS

Si se activa esta casilla de control, se activará el protocolo TLS para el protocolo. Esta opción se encuentra activada por defecto.

Autenticación de Cliente

Si esta casilla de control está activada, los clientes se deben identificar en el servidor con cada solicitud. Esta opción se activada por defecto.

Apodo de Certificado

Apodo del certificado del servidor en la base de datos de certificados o token PKCS#11. En el certificado, el formato del nombre es tokenname:nickname. Incluir tokenname: parte del nombre de este atributo es opcional.

Almacén de Claves

Nombre de archivo del almacén de claves (por ejemplo, keystore.jks).

Algoritmo de Confianza

Nombre del algoritmo de gestión de confianza (por ejemplo, PKIX) que se utilizará para la validación de rutas de acceso de certificación.

Longitud de Certificado Máxima

Número máximo de certificados intermedios no emitidos automáticamente que puede haber en una ruta de acceso de certificación. Este campo sólo se utiliza en caso de que el campo Algoritmo de Confianza esté definido en PKIX. Un valor de 0 implica que la ruta de acceso sólo puede contener un certificado. Un valor de -1 implica que la longitud de la ruta de acceso no está restringida (no hay un máximo). La configuración en un valor inferior a -1 inicia una excepción.

Almacén de Confianza

Nombre de un archivo del almacén de confianza (por ejemplo, cacerts.jks).

Conjuntos de Cifrado

Área en la que puede agregar o suprimir conjuntos de cifrado. Si no agrega ningún conjunto de cifrado, se utilizarán todos los conjuntos de cifrado.
