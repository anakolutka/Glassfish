Modification des numéros de port d'une instance GlassFish Server incluse dans un cluster

Les numéros de port d'une instance GlassFish Server incluse dans un cluster sont initialement définis dans la configuration que l'instance hérite de son cluster parent. Vous devez modifier ces numéros de port si, par exemple, deux instances incluses dans un cluster résident sur le même hôte. Dans ce cas, les numéros de port d'une instance doivent être modifiés car les deux instances partagent la même configuration.

Les numéros de port d'une instance sont stockés en tant que propriétés système Java. Ces propriétés sont transmises au lanceur d'application Java via son option -D au démarrage de GlassFish Server.

  1. Dans l'arborescence de navigation, développez le noeud Clusters.

  2. Sous le noeud Clusters développé, sélectionnez le noeud correspondant au cluster.

    La page Informations générales du cluster apparaît.

  3. Sur la page Informations générales, cliquez sur l'onglet Instances.

    La page Instances de serveur clusterisées du cluster apparaît.

  4. Sélectionnez l'instance dont vous souhaitez modifier les numéros de port.

    La page Informations générales de l'instance apparaît.

  5. Sur la page Informations générales, cliquez sur l'onglet Propriétés.

    La page Propriétés système d'instance apparaît.

    Cette page affiche les valeurs en cours des propriétés système suivantes pour l'instance :

    • Numéros de port prédéfinis dans la configuration de l'instance

    • Propriétés personnalisées définies pour l'instance, le cas échéant

  6. Indiquez la nouvelle valeur de chaque propriété existante à modifier dans le champ Valeur de remplacement.

  7. Définissez les valeurs des propriétés supplémentaires requises.

    1. Dans la table Autres propriétés, cliquez sur Ajouter une propriété.

      Une ligne est ajoutée à la table Autres propriétés.

    2. Dans le champ Nom de la variable d'instance, entrez le nom de la propriété.

    3. Dans le champ Valeur de remplacement, indiquez la valeur.

  8. Cliquez sur Enregistrer.
