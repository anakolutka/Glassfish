JDBC リソースを作成する

Java DataBase Connectivity (JDBC) リソース (データソース) は、データベースへの接続手段をアプリケーションに提供します。

始める前に

JDBC リソースを作成するには、まず JDBC 接続プールを作成する必要があります。「JDBC 接続プールを作成する」を参照してください。

  1. ナビゲーションツリーで、「リソース」ノードを展開します。
  2. 「JDBC リソース」ノードを選択します。
  3. 「JDBC リソース」ページで、「新規」をクリックします。

    「新しい JDBC リソース」ページが開きます。

  4. 「JNDI 名」フィールドに、一意のリソース名を入力します。

    JNDI 名によって、分散コンピューティング環境に配置されたコンポーネントの編成と場所の管理を行います。この方法は図書館の蔵書の分類と保管場所の管理に用いるカード式の目録と似ています。そのため、JNDI 名は JDBC リソースにアクセスする際の重要な要素になります。規則により、名前は jdbc/ 文字列で始まります。たとえば、jdbc/payrolldb などです。スラッシュを忘れないでください。

    名前に含めることができるのは、英数字、下線、ダッシュ、またはドットの各文字だけです。

  5. 「プール名」ドロップダウンリストから、新しい JDBC リソースに関連付ける接続プールを選択します。
  6. 「説明」フィールドで、新しい JDBC リソースの簡単な説明を入力します。
  7. リソースを無効にして使用できないようにするには、「有効」チェックボックスの選択を解除します。

    デフォルトでは、リソースは作成時に利用可能 (有効) です。

  8. 「了解 (OK)」をクリックします。
