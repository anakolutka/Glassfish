AMX: Frequently Asked Questions

by Lloyd L Chambers
Last updated: 01 August 2006

Why do some of the AMX interfaces use overly-complicated generic types ?

Generic types were inserted fairly late in the game. Understanding the best way to use Java generic types takes some time and experience—and in a few cases there are superior alternatives. If you think there is a better way to declare a type, please file an RFE and/or submit a code change for review.

The AMX DeploymentMgr makes deployment too hard to use. Is there an easier way?

The original design vision called for layering additional helper classes over this base-level API. That decision has been rethought. The com.sun.appserv.managementy.DeploymentMgr interface is deprecated in version 9.1. Use the JSR 88 DeploymentManager class instead.
