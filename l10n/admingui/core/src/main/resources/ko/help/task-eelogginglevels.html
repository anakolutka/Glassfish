로그 수준을 구성하는 방법

  1. 탐색 트리에서 구성 노드를 선택합니다.
  2. 구성 노드에서 로깅 설정 노드를 선택합니다.

    로깅 설정 페이지가 열립니다.

  3. 로깅 설정 페이지에서 로그 수준 탭을 누릅니다.

    모듈 로그 수준 페이지가 열립니다.

  4. 모듈 로그 수준 페이지에서 다음과 같이 하나 이상의 모듈에 대한 로그 수준을 변경합니다.
    • 개별 모듈에 대한 로그 수준을 변경하려면 로그 수준을 변경할 모듈 반대쪽의 드롭다운 목록에서 새 값을 선택합니다.
    • 모듈 그룹의 로그 수준을 변경하는 방법
      1. 로그 수준을 변경할 모듈의 이름 옆에 있는 확인란을 선택합니다.
      2. 테이블 맨 위의 드롭다운 목록에서 값을 선택합니다.
      3. 수준 변경 버튼을 누릅니다.

    선택한 수준 이상의 메시지가 로그에 기록됩니다. 다음 값 중 하나를 선택합니다(높은 수준에서 낮은 수준으로 나열).

    • SEVERE

    • WARNING

    • INFO

    • CONFIG

    • FINE

    • FINER

    • FINEST

    • 해제

    로그 수준의 의미에 대한 자세한 내용은 모듈 로그 수준을 참조하십시오. 기본 수준은 INFO이므로 기본적으로 로그에는 INFO, WARNING 또는 SEVERE 수준의 메시지가 포함됩니다.

  5. 저장을 누릅니다.

    로그 수준 설정에 대한 변경 사항은 바로 적용됩니다. 또한 서버를 다시 시작할 때 사용할 수 있도록 저장됩니다.
