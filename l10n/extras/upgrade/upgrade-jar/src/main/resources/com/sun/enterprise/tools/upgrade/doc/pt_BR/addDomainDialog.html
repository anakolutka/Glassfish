Caixa de diálogo Add Domain

Use a caixa de diálogo Add Domain para selecionar o nome de domínio e as senhas de certificado de origem e destino.

  1. Clique no campo de opções Source Domain e selecione o domínio que contém os certificados de segurança para a migração.
  2. No painel Passwords, insira as senhas solicitadas. Dependendo do tipo da instalação de origem e de destino, pode ser necessário a introdução de duas ou três senhas:
    • Insira as senhas dos arquivos de certificado NSS, usadas para banco de dados NSS seguro, nos campos Source NSS e/ou Target NSS.
    • Insira a senha do banco de dados de chaves de origem do usuário, usada para o arquivo keystore.jks do JSSE que contém os certificados do par de chaves do usuário, no campo User Key Database.
    • Insira a senha do banco de dados de autoridades de certificado, que é usada para o arquivo cacerts.jks do JSSE que contém os certificados da autoridade de certificado, no campo Certificate Authority Database.
  3. Pressione o botão OK ao terminar. A tela Transfer Security Certificates será exibida, mostrando o domínio adicionado no painel Domains with Certificates to be Transferred.
