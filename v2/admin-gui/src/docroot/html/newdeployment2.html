To Deploy Service Assemblies

  1. In the tree view, select the JBI node and then the Service Assemblies node. If you are using the Cluster profile, you should click the Clusters node, the required cluster and then the JBI tab.

    A list of Service Assembly instances is displayed.

  2. Click Deploy.

    The Deploy JBI Services Assembly page is displayed.

  3. Select the location of the JBI Service Assembly archive to be deployed.

    This archive will be a .jar or a .zip file.

  4. Click Next.

  5. If you are using the Clusters Profile, select the required target for the service assembly from the Available list.

  6. Click Add to move the target to the Selected list.

  7. Click the Enabled checkbox if you want to enable the Service Assembly.

  8. Click Finish.

    The JBI Service Assembly page lists the deployed service assembly.
