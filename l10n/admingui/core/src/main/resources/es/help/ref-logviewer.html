Visualizador de registros

Consulte la página Visualizador de registros para ver los registros del servidor y personalizar la pantalla de registros.

La página Visualizador de registros contiene tres áreas:

  • Criterios de búsqueda

  • Búsqueda avanzada

  • Resultados de búsqueda

El área Criterios de búsqueda contiene las siguientes opciones.

Búsqueda avanzada

Al presionar en este enlace aparecerá una zona que le permitirá añadir elementos al visualizador de registros.

Marca de tiempo

El botón de opción Más reciente muestra los mensajes más recientes. El botón de opción Intervalo específico habilita los campos Desde y Hasta que le permiten ver mensajes de un período concreto.

Para el valor Tiempo, la sintaxis debe tener la forma siguiente (SSS significa milisegundos):

hh:mm:ss.SSS

Por ejemplo:

17:10:00.000

Si el valor introducido en el campo De es posterior al valor introducido en el campo A, aparece un mensaje de error.

Nivel de registro

Nivel de registro seleccionado para el filtrado. De forma predeterminada, la pantalla incluye todos los mensajes que aparecen en el registro del servidor en el nivel de registro seleccionado y niveles más graves. Seleccione la casilla de verificación No incluir más mensajes graves para mostrar mensajes sólo en el nivel seleccionado.

Para asegurarse de que los mensaje que quiere ver aparecen en el registro del servidor, configure previamente los niveles de registro en la página Niveles de registro del módulo. Vea Para configurar niveles de registro.

Si decide filtrar mensajes de registro según nivel de registro, sólo se muestran los mensajes que se ajustan a los criterios de filtro especificados. Sin embargo, este filtro no afecta a qué mensajes están registrados en el registro del servidor.

El área Búsqueda avanzada contiene las siguientes opciones.

Registro

Espacios de nombre del registro para filtrar por módulos. Utilice la tecla Mayús o la tecla Ctrl para seleccionar varios espacios de nombre.

La selección de un espacio de nombre en un nivel superior selecciona todos los espacios de nombre existentes por debajo. Por ejemplo, al seleccionar javax.enterprise.system también se seleccionan los registradores de todos los módulos existentes por debajo de ese espacio de nombre: javax.enterprise.system.core, javax.enterprise.system.tools.admin, etc.

Registro personalizado

Registros específicos de una aplicación determinada, uno por línea.

Si la aplicación tiene varios módulos, puede visualizar uno o todos ellos. Por ejemplo, suponga que la aplicación tiene registros con los nombres siguientes:

com.mycompany.myapp.module1
com.mycompany.myapp.module2
com.mycompany.myapp.module3

Para ver los mensajes de todos los módulos de la aplicación, introduzca com.mycompany.myapp. Para ver los mensajes sólo de module2, introduzca com.mycompany.myapp.module2.

Cuando especifica uno o varios registros personalizados, los mensajes de los módulos de Servidor empresarial aparecen sólo si los indica explícitamente en el área Registro.

Pareja Nombre-Valor

Nombre clave y valor para un subproceso, para ver la salida de un subproceso específico. El nombre de clave es _ThreadID. Por ejemplo:

_ThreadID=13

Suponga que com.mycompany.myapp.module2 se ejecuta en varios subprocesos. Para refinar la búsqueda del visualizador de registros y mostrar sólo el resultado de un único subproceso, especifique este registro del módulo en el campo Registro personalizado y, a continuación, indique el ID del subproceso en este campo.

Visualización

Número de mensajes que se ven a la vez. El número predeterminado de mensajes mostrados es 40. También puede elegir 100, 250 y 1000.

Para ver seguimientos de pilas, anule la selección la casilla de verificación Limitar mensajes demasiado largos. De forma predeterminada, los seguimientos de pilas no aparecen en el visualizador; para visualizarlos, haga clic en el vínculo (details) para un mensaje.

La tabla Resultados de búsqueda muestra los mensajes de registro del servidor.

La tabla Resultados de búsqueda muestra la siguiente información para cada mensaje.

Número de registro

Número del mensaje en el registro del servidor.

Nivel de registro

Nivel de registro del mensaje.

Mensaje

Breve texto del mensaje de registro. Presione el enlace (details) para ver una versión con formato del mensaje.

Registro

Módulo de registro que generó el mensaje de registro.

Marca de tiempo

Hora y fecha en que se emitió el mensaje de registro.

Pareja Nombre/Valor

Nombre de subproceso e Id. del mensaje de registro.
