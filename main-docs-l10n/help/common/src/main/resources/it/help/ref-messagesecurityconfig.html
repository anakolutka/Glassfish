Configurazioni di sicurezza dei messaggi

Utilizzare la pagina Configurazioni di sicurezza dei messaggi per visualizzare e modificare le configurazioni disponibili per i provider di sicurezza dei messaggi.

Per configurare la sicurezza dei messaggi è necessario avere una conoscenza di base delle nozioni fondamentali relative alla sicurezza e ai servizi Web. Per ulteriori informazioni, vedere il Capitolo 3, Administering Message Security, in Oracle GlassFish Server Security Guide e il Capitolo 6, Administering Web Applications, in Oracle GlassFish Server Administration Guide.

GlassFish Server implementa la sicurezza dei messaggi mediante provider di sicurezza dei messaggi integrati nel proprio layer di elaborazione SOAP. I provider di sicurezza dei messaggi dipendono da altre risorse di protezione di GlassFish Server.

Per ogni configurazione vengono fornite le informazioni riportate di seguito.

Layer di autenticazione

Il layer di messaggi in cui viene eseguita l'autenticazione. Per impostazione predefinita, viene fornita una configurazione SOAP. È inoltre possibile fornire una configurazione HttpServlet.

Provider predefinito

Il provider server predefinito per questa configurazione. Per impostazione predefinita, non viene configurato alcun provider server predefinito.

Provider client predefinito

Il provider client predefinito per questa configurazione. Per impostazione predefinita, non viene configurato alcun provider client predefinito.

La tabella Configurazione di sicurezza dei messaggi contiene inoltre le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare una nuova configurazione.

Elimina

Pulsante che consente di eliminare una o più configurazioni selezionate.
