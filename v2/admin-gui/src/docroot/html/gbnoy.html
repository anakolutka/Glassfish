To Monitor and Rollback Transactions

  1. In the tree component, select the Application Server node.

  2. Click the Monitor tab.

  3. Click the Transactions tab.

    The page displays the transactions that are in process.

  4. To rollback a transaction, first click the Freeze button.

    This freezes the transaction subsystem and determines transactions that are in process at the time of the freeze.

  5. Select the checkbox beside the transaction ID and click Rollback.

  6. Click Unfreeze to continue viewing in-process transactions.
