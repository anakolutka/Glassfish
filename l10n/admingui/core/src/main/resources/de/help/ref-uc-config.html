Update-Tool-Konfiguration

Die Seite "Konfiguration" liefert Konfigurationsinformationen zu Update-Tool und ermöglicht Ihnen, die Konfiguration von Update-Tool zu ändern.

Die Update-Tool-Konfigurationsseite enthält folgende Optionen:

Installationspfad

Der Dateisystempfad des Root-Verzeichnisses oder des Verzeichnisses auf der obersten Ebene des Installations-Image.

Bevorzugtes Repository

Der Name des bevorzugten Netzwerk-Repositorys, das mit diesem Installations-Image verbunden ist, zum Beispiel glassfish.org.

Internetverbindung

Wie Enterprise Server eine Verbindung zum Internet herstellt:

Direktverbindung

Diese Option gibt an, dass Enterprise Server direkt mit dem Internet verbunden ist.

Über einen Proxy

Diese Option gibt an, dass Enterprise Server eine Verbindung zum Internet über einen Proxyserver herstellt. Wenn Sie diese Option auswählen, werden dieser Seite die Felder "Proxyhost" und Proxyport" hinzugefügt.

Proxy-Host

Der Hostname des Proxy-Servers für den Computer, auf dem Enterprise Server ausgeführt wird.

Dieses Feld gilt nur, wenn der Computer, auf dem Enterprise Server ausgeführt wird, über einen Proxy-Server mit dem Internet verbunden wird.

Proxy-Port

Die Port-Nummer des Proxy-Servers für den Computer, auf dem Enterprise Server ausgeführt wird.

Dieses Feld gilt nur, wenn der Computer, auf dem Enterprise Server ausgeführt wird, über einen Proxy-Server mit dem Internet verbunden wird.

Datensammlungen

Diese Tabelle liefert die folgenden Informationen zu allen Netzwerk-Repositorys, die mit diesem Installations-Image verbunden sind:

Repository

Der Name des Repositorys, zum Beispiel glassfish.org.
