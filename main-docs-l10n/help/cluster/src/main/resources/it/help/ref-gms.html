Modifica Group Management Service

Utilizzare la pagina Group Management Service per modificare le impostazioni del servizio GMS.

GMS (Group Management Service) è un servizio in-process che offre servizi di monitoraggio del cluster, appartenenza del cluster e comunicazione di gruppo. Il servizio GMS è stato creato nella parte superiore di uno stack configurabile di protocolli di individuazione di appartenenza ai gruppi e di monitoraggio dell'integrità. Il servizio GMS esegue le funzioni seguenti:

  • notifica ai moduli registrati in un'istanza di GlassFish Server l'aggiunta o la rimozione di una o più istanze membro da un cluster oppure un errore sospettato o confermato di tali istanze;

  • consente di inviare e ricevere messaggi tra i membri del cluster.

La pagina Group Management Service contiene le opzioni riportate di seguito.

Carica impostazioni predefinite

Pulsante per ripristinare i valori predefiniti delle impostazioni che dispongono di valori predefiniti. Le impostazioni che non dispongono di valori predefiniti non vengono modificate.

Nome configurazione

Il nome della configurazione a cui si applicano le impostazioni presenti in questa pagina. Questo campo è di sola lettura.

Numero massimo heartbeat mancati

Il numero massimo di heartbeat mancati da un membro oltre il quale si sospetta un errore dell'istanza del server. L'impostazione predefinita è 3.

Frequenza heartbeat

Il periodo di tempo che intercorre tra gli heartbeat mancati da un membro. L'impostazione predefinita è 2000 millisecondi.

Timeout individuazione gruppo

Il periodo di tempo durante il quale il servizio GMS rimane in attesa dell'individuazione di altri membri del cluster. L'impostazione predefinita è 5000 millisecondi.

Tempo di attesa verifica errore

Il periodo di tempo di attesa prima di verificare se segnalare come in stato di errore un membro nel quale si sospetta si siano verificati errori. L'impostazione predefinita è 1500 millisecondi.

Proprietà aggiuntive

Tabella di proprietà aggiuntive per il servizio GMS.

GlassFish Server non definisce alcuna proprietà GMS aggiuntiva.

Per ogni proprietà vengono visualizzate le informazioni riportate di seguito.

Nome

Il nome della proprietà.

Valore

Il valore della proprietà impostato per l'istanza selezionata.

Descrizione

Descrizione di testo che fornisce ulteriori informazioni sulla proprietà.

La tabella Proprietà aggiuntive contiene inoltre le opzioni riportate di seguito.

Aggiungi proprietà

Pulsante che consente di aggiungere una proprietà. Se si fa clic su questo pulsante, viene aggiunta una riga alla tabella Proprietà aggiuntive.

Elimina proprietà

Pulsante che consente di eliminare una o più proprietà selezionate. L'eliminazione comporta il ripristino del valore predefinito della proprietà. Se il valore predefinito non è stato impostato, la proprietà rimane non definita.
