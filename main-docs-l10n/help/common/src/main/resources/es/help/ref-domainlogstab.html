Logs de dominio

Utilice la página Logs de Dominio para recopilar y descargar archivos log para una instancia o cluster de GlassFish Server específicos y para compilar los logs en un solo archivo ZIP.

Nota:

Si especifica un cluster, el archivo ZIP también incluirá el archivo log del servidor de administración del dominio (DAS).

La página Logs de Dominio contiene las siguientes opciones.

Cluster

Lista desplegable de clusters si hay algún cluster.

Instancia

Lista desplegable de instancias de servidor.

Recopilar Logs

Botón para recopilar los logs en un archivo log-files-name.zip, donde name es el nombre del cluster o de la instancia.

El archivo log-files-name.zip se descarga mediante el mecanismo de descarga de su explorador.

Tareas relacionadas

  • Ver archivos log de DAS
