Modificar los números de puerto de una instancia de GlassFish Server independiente

Los números de puerto de una instancia de GlassFish Server independiente se definen inicialmente en la configuración a la que hace referencia la instancia. Debe modificar estos números de puerto si, por ejemplo, dos instancias que hacen referencia a la misma configuración residen en el mismo host. En este caso, los números de puerto de una instancia deben modificarse porque ambas instancias hacen referencia a la misma configuración.

Los números de puerto de una instancia se guardan como propiedades del sistema Java. Estas propiedades se transfieren al iniciador de aplicaciones Java mediante la opción -D del iniciador de aplicaciones Java cuando se inicia GlassFish Server.

  1. En el árbol de navegación, seleccione el nodo Instancias Independientes.

    Se abrirá la página Instancias de Servidor Independientes.

  2. Seleccione la instancia para la cual se modificarán los números de puerto.

    Se abrirá la página Información General para la instancia.

  3. En la página Información General, haga clic en el separador Propiedades.

    Se abrirá la página Propiedades del Sistema de Instancias.

    Esta página muestra los valores actuales de las siguientes propiedades del sistema para la instancia:

    • Números de puerto predefinidos en la configuración de la instancia

    • Propiedades personalizadas que se definen para la instancia, si las hay

  4. Para cada propiedad que vaya a modificar, escriba el nuevo valor en el campo Valor de Sustitución para la propiedad.

  5. Defina los valores que necesite en las demás propiedades.

    1. En la tabla Propiedades Adicionales, haga clic en Agregar Propiedad.

      Se agrega una fila a la tabla Propiedades Adicionales.

    2. En el campo Nombre de Variable de Instancia, escriba el nombre de la propiedad.

    3. En el campo Valor de Sustitución, escriba el valor.

  6. Haga clic en Guardar.
