Per modificare le impostazione della cache di file per un protocollo

  1. Espandere il nodo Configurazione nell'albero di navigazione.

  2. Nel nodo Configurazione espandere il nodo Configurazione di rete.

  3. Nel nodo Configurazione di rete selezionare il nodo Protocolli.

    Viene visualizzata la pagina Protocolli.

  4. Nella pagina Protocolli fare clic sul nome del protocollo da modificare.

    Viene visualizzata la pagina Modifica protocollo.

  5. Nella pagina Modifica protocollo fare clic sulla scheda Cache di file.

    Viene visualizzata la pagina Cache di file.

  6. Selezionare la casella di controllo Abilitato dell'opzione Stato per abilitare la cache di file.

    Questa opzione è selezionata per impostazione predefinita.

  7. Nel campo Durata massima digitare la durata massima, in secondi, per una voce valida della cache.

    Questo parametro definisce il tempo di utilizzo delle informazioni inserite nella cache dopo l'inserimento di un file nella cache. Una voce con durata maggiore della durata massima impostata viene sostituita da una nuova voce per lo stesso file. Se si modifica il contenuto in modo sporadico, impostare un valore più elevato per ottimizzare le prestazioni.

    La durata massima ottimale dipende dalla frequenza con cui si modificano i file esistenti. Ad esempio, se i file vengono modificati quattro volte al giorno a intervalli regolari, si consiglia di impostare una durata massima su 21600 secondi (6 ore). È altrimenti possibile impostare una durata massima corrispondente al periodo più lungo durante il quale si desidera utilizzare la versione precedente di un file dopo che il file è stato modificato.

    Il valore predefinito è 30.

  8. Nel campo Dimensione massima cache digitare la dimensione totale massima (in byte) della cache di file su disco.

    Il valore predefinito è 10485760.

  9. Nel campo Numero massimo di file digitare il numero massimo di file che possono essere memorizzati nella cache di file.

    Se il valore è eccessivamente alto, il server inserisce nella cache i file meno richiesti al fine di evitare sprechi di memoria. Se il valore è eccessivamente basso, l'inserimento nella cache non presenta alcun vantaggio. Provare a specificare valori differenti per questo attributo fino a ottenere la soluzione ottimale per applicazioni specifiche.

    Il valore predefinito è 1024.

  10. Fare clic su Salva.

    Suggerimento:

    Per ripristinare le impostazioni predefinite, fare clic sul pulsante Carica impostazioni predefinite, quindi su Salva.
