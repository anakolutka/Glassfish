Service de contrôle

La page Service de contrôle permet de configurer les options de contrôle pour les composants ou services des serveurs individuels.

Pour chaque option de contrôle, les niveaux disponibles sont BAS, HAUT et ARRETE. Dans cette version, il n'existe aucune différence entre les options BAS et HAUT.
