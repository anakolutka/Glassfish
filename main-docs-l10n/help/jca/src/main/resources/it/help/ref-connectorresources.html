Risorse connettore

Utilizzare la pagina Risorse connettore per configurare le risorse connettore.

Una risorsa connettore è un oggetto di programma che fornisce una connessione a un sistema EIS a un'applicazione.

Per ogni risorsa vengono fornite le informazioni riportate di seguito.

Nome JNDI

Il nome univoco che identifica la risorsa connettore.

Abilitato

L'impostazione è True se la risorsa connettore è abilitata oppure è False se la risorsa connettore è disabilitata.

Connection pool

Il connection pool del connettore associato alla risorsa connettore.

Descrizione

La descrizione della risorsa connettore.

La tabella Risorse contiene inoltre le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare una nuova risorsa connettore.

Elimina

Pulsante che consente di eliminare una o più risorse connettore selezionate.

Abilita

Pulsante che consente di abilitare una o più risorse connettore selezionate.

Disabilita

Pulsante che consente di disabilitare una o più risorse connettore selezionate.
