Monitoraggio risorse

Utilizzare la pagina Monitoraggio risorse per visualizzare i dati di monitoraggio per la risorsa selezionata.

La pagina Monitoraggio risorse contiene le informazioni riportate di seguito.

Nome istanza

Il nome dell'istanza server.

Connection pool

Il connection pool per il quale vengono visualizzate le statistiche di monitoraggio.

Per una descrizione delle proprietà monitorate, vedere il Capitolo 8, Amministrazione del servizio di monitoraggio, in Oracle GlassFish Server Administration Guide.
