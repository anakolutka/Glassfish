Per visualizzare la lista delle applicazioni distribuite

  1. Selezionare il nodo Applicazioni nell'albero di navigazione.

    Viene visualizzata la pagina Applicazioni. Nella tabella sono elencate le applicazioni distribuite.

  2. Selezionare un tipo di modulo dall'elenco a discesa Filtro per filtrare la lista delle applicazioni in base al tipo di modulo.
