So fügen Sie einen Benutzer hinzu

  1. Klicken Sie auf "Neu", um einen neuen Benutzer zum Dateibereich hinzuzufügen.

    Die Seite "Neuer Dateibereichsbenutzer" wird geöffnet.

  2. Geben Sie in das Feld "Benutzer-ID" den Namen des Benutzers ein. Dies ist ein erforderliches Feld.
  3. Geben Sie in das Feld "Passwort" ein eindeutiges Passwort ein. Dies ist ein erforderliches Feld.
  4. Bestätigen Sie das Benutzerpasswort zur Überprüfung. Dies ist ein erforderliches Feld.
  5. Geben Sie in das Feld "Gruppenliste" eine durch Kommas getrennte Liste der Gruppen ein, zu denen der Benutzer gehört. Diese Gruppe muss an keiner weiteren Stelle definiert werden.
  6. Wählen Sie "OK", um diesen Benutzer der Benutzerliste im Dateibereich hinzuzufügen, oder "Abbrechen", um ohne Speichern zu beenden.
