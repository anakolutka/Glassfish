To edit a deployed web application

  1. On the General tab of the Web Application page, change the Context Root or the Description, if desired.
  2. In Cluster Profile, enable or disable high-availability.

    If availability is enabled for an application, it must also be enabled at all higher levels (named configuration and web container or EJB container) as well.

  3. To view the list of module descriptorsfor the application, click the Descriptors tab. For more details, see To view module descriptors of deployed applications
  4. To view the targets on which the application has been enabled, click the Targets tab. You can enable or disable applications on one or multiple targets. See Managing Targets.
