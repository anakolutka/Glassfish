新建 JDBC 資源

使用「新增 JDBC 資源」頁面可建立新的 JDBC 資源。

「新增 JDBC 資源」頁面包含下列選項。

JNDI 名稱

可識別資源的唯一 JNDI 名稱。依慣例會是以資源類型指示符與斜線開頭的名稱，例如 concurrent/、jdbc/ 或 jms/。

集區名稱

與 JDBC 資源相關聯的連線集區。

描述

JDBC 資源的簡短描述。

狀態

若啟用此選項，則資源可在執行階段使用。此選項預設為啟用。

其他特性

JDBC 資源的其他特性。GlassFish Server 未定義 JDBC 資源的任何其他特性。

目標

(已在網域中建立叢集或獨立執行處理時才會顯示此選項。)

資源的叢集和獨立執行處理。可使用「加入」與「全部加入」按鈕，將所需的目標移至「選取的目標」欄。可使用「移除」與「全部移除」按鈕，將任何不需要的目標移至「可用的目標」欄。
