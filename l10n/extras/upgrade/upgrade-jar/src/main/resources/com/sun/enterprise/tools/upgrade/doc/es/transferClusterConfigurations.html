Pantalla Transferir configuraciones de clúster

Para actualizar un clúster de Sun Java System Application Server 8.x Enterprise Edition (EE) a Sun Java System Application Server 9.1 se necesita un archivo clinstance.conf válido. La herramienta de actualización asume que el archivo clinstance.conf refleja de manera correcta el clúster 8.x EE. Si, por algún motivo, este archivo no representa la configuración de clúster actual correctamente, deberá editar manualmente la definición de clúster antes de poder realizar la actualización.

Utilice la pantalla Transferir configuraciones de clúster para especificar el archivo o archivos de configuración de clúster que deben migrarse a la nueva instalación.

  1. Haga clic en Agregar para abrir el cuadro de diálogo Seleccionar archivo clinstance.conf.
  2. Seleccione el archivo de clúster clinstance.conf, que contiene la información de configuración del clúster que se migrará.
  3. Haga clic en el botón Abrir.
  4. Repita este proceso hasta haber añadido todos los archivos de configuración de clúster que se deben migrar.
  5. Haga clic en Siguiente cuando haya terminado. Se abrirá la pantalla Resultados de la actualización, en la que se muestra el estado de la actualización.
