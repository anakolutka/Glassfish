Para configurar ORB

  1. En el componente de árbol, amplíe el nodo Configuración.

  2. Seleccione la instancia que va a configurar:

    • Para configurar una instancia determinada, seleccione el nodo de configuración de la instancia. Por ejemplo, para la instancia por defecto, server, seleccione el nodo server-config.

    • Para definir la configuración por defecto para instancias futuras que utilicen una copia de default-config, seleccione el nodo default-config.

  3. Seleccione el nodo ORB.

    Se abrirá la página ORB.

  4. Seleccione el pool de threads que utilizará ORB de la lista desplegable ID de pool de threads.

    ORB utiliza pools de threads para responder a solicitudes de clientes remotos de beans de empresa y otros clientes que se comunican mediante RMI-IIOP. El pool de threads por defecto para ORB es thread-pool-1, que utiliza valores por defecto apropiados para las solicitudes de RMI/IIOP.

    Consulte Pools de threads para obtener más información.

  5. En la lista desplegable Tamaño máximo de fragmentos de mensaje, seleccione el tamaño máximo para los fragmentos de mensajes IIOP.

    Los mensajes más grandes que este tamaño se fragmentarán. El valor por defecto es de 1024 bytes.

  6. En el campo Nº total de conexiones, defina el número máximo de conexiones entrantes para los listeners IIOP.

    El valor por defecto es 1024.

  7. Active la casilla de control de autenticación necesaria de cliente IIOP para especificar que la autenticación del cliente IIOP es necesaria.

    Si la autenticación de cliente IIOP está activada, el servidor rechazará las solicitudes no autenticadas e insertará un bit de autenticación necesaria en las referencias de objetos remotos interoperables (IOR) enviadas a los clientes. Esta opción se activada por defecto.

  8. En la sección Propiedades adicionales, especifique las propiedades adicionales.

    GlassFish Server no define ninguna propiedad adicional para ORB.

    • Para agregar una propiedad, haga clic en el botón Agregar Propiedad. En la fila en blanco que aparece, introduzca el nombre de propiedad en el campo Nombre y escriba el valor de propiedad en el campo Valor.

    • Para modificar una propiedad, edite el campo Valor.

    • Para suprimir una propiedad, active la casilla de control a la izquierda del campo Nombre de la propiedad que está suprimiendo y, a continuación, haga clic en el botón Suprimir Propiedades.

  9. Haga clic en Guardar.

    Sugerencia:

    Para volver a la configuración por defecto, haga clic en el botón Cargar Valores por Defecto y, a continuación, en Guardar.

  10. Reinicie GlassFish Server.
