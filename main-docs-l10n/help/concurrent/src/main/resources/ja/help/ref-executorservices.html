管理対象エグゼキュータ・サービス

管理対象エグゼキュータ・サービス・リソースを構成するには、「管理対象エグゼキュータ・サービス」ページを使用します。

コンテキスト・サービス・リソースごとに、次の情報が表示されます。

JNDI名

リソースを識別する一意の名前。

有効

リソースが有効な場合はtrue、無効な場合はfalseです。

コンテキスト情報

リソースのコンテキスト: 「クラス・ローダー」、「JNDI」、「セキュリティ」、またはワークエリア

スレッド優先度

管理対象エグゼキュータ・サービス・インスタンスに作成するスレッドの優先度。デフォルトは5です。

説明

リソースの説明。

「管理対象エグゼキュータ・サービス」の表には、次のオプションも表示されます。

新規

新規リソースを作成するボタン。

削除

選択した1つまたは複数のリソースを削除するボタン。

有効

選択した1つまたは複数のリソースを有効にするボタン。

無効

選択した1つまたは複数のリソースを無効にするボタン。

関連するasadminコマンド

  • create-XXX(1)

  • list-XXX(1)

  • delete-XXX(1)
