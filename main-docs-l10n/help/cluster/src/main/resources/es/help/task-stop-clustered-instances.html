Para parar instancias de GlassFish Server en cluster

  1. En el árbol de navegación, amplíe el nodo Clusters.

  2. En el nodo Clusters ampliado, seleccione el nodo para el cluster.

    Se abre la página Información General para el cluster.

  3. En la página Información General, haga clic en el separador Instancias.

    Se abrirá la página Instancias del Servidor en Cluster para el cluster.

  4. Para cada instancia que vaya a parar, active la casilla de control que aparece junto a la instancia en la tabla Instancias del Servidor.

  5. Haga clic en Parar.

    Se le pedirá que confirme que desea parar las instancias.

  6. Confirme que desea parar las instancias.

    El estado de cada instancia se actualiza en la tabla Instancias del Servidor cuando se para la instancia.
