JVM のプロファイラ設定を行う

プロファイラツールは、パフォーマンスを分析し、潜在的な障害を特定するための情報を生成します。

  1. ナビゲーションツリーで、「Application Server」ノードを選択します。
  2. 「一般情報」ページで、「JVM 設定」タブをクリックします。
  3. 「JVM 一般設定」ページで、「プロファイラ」タブをクリックします。

    「JVM プロファイラ設定」ページが開きます。

  4. プロファイラ設定を指定します。

    指定する情報は、使用するプロファイラ製品によって異なります。具体例と手順については、『Sun GlassFish Enterprise Server v3 Prelude Developer’s Guide』の第 3 章「Debugging Applications」を参照してください。

  5. 「保存」をクリックします。
  6. サーバーを再起動します。
