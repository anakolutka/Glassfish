Para usar o módulo de auditoria padrão

O módulo de auditoria padrão registra as solicitações de autenticação e de autorização no arquivo de log do servidor. Para obter informações sobre a alteração do local do arquivo de log, consulte Para Definir as configurações gerais de log.

As entradas de log de autenticação incluem estas informações:

  • Nomes do usuários que tentaram se autenticar

  • O domínio que processou a solicitação de acesso

  • Componente URI ou EJB do módulo Web solicitado

  • Êxito ou falha da solicitação

Independentemente de o modo de auditoria estar ou não ativado, o Enterprise Server registra em log todos os eventos de autenticação recusados.

As entradas de log de autorização incluem estas informações:

  • Nomes de usuários autenticados, se houver

  • Componente URI ou EJB da Web solicitada

  • Êxito ou falha das solicitações

  1. Na árvore de navegação, expanda o nó Configuração.
  2. Sob o nó Configuração, selecione o nó Segurança.

    A página Segurança é exibida.

  3. Na lista suspensa Módulos de Auditoria, selecione o módulo de auditoria denominado padrão.
  4. Clique em Salvar.
