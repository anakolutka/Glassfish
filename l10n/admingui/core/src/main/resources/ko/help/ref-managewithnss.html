사용자 관리를 위한 NSS(Network Security Services)

파일 사용자 페이지에서 설명한 대로 관리 콘솔을 사용하거나 NSS 도구를 사용하여 사용자를 관리할 수 있습니다. NSS는 보안이 활성화된 클라이언트 및 서버 응용 프로그램의 교차 플랫폼 개발을 지원할 수 있도록 설계된 라이브러리 집합입니다. NSS로 구축된 응용 프로그램은 SSL v2 및 v3, TLS, PKCS #5, PKCS #7, PKCS #11, PKCS #12, S/MIME, X.509 v3 인증서 및 다른 보안 표준을 지원할 수 있습니다. 자세한 내용을 보려면 다음 URL로 이동하십시오.

  • NSS(Network Security Services) - http://www.mozilla.org/projects/security/pki/nss/

  • NSS 보안 도구 - http://www.mozilla.org/projects/security/pki/nss/tools/

  • NSS 개요 - http://www.mozilla.org/projects/security/pki/nss/overview.html
