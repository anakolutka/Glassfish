Paramètres du journal

Utilisez la page Paramètres du journal pour configurer la journalisation pour le Enterprise Server.

La page Paramètres du journal contient les options suivantes.

Ecrire dans le journal système

Si cette option est sélectionnée, le résultat de journalisation est envoyé à l'utilitaire syslog en plus du journal du serveur. Cette fonction est disponible sur les systèmes Solaris et Linux uniquement. Cette option est désactivée par défaut.

Consigner sur la console

Si cette option est sélectionnée, NE SAIS PAS ce qui se passe. Cette option est désactivée par défaut.

Fichier journal

Un autre nom ou emplacement pour le fichier du journal du serveur. L'emplacement par défaut est glassfish/domains/domain1/logs/server.log.

Gestionnaire du journal

Nom de classe absolu d'un gestionnaire de journal personnalisé. Un gestionnaire de journal personnalisé vous permet d'envoyer des journaux vers une destination autre que server.log ou syslog. Celui-ci doit venir en extension de la classe java.util.logging.Handler (API compatible JSR 047). Ajoutez la classe du gestionnaire au classpath du Enterprise Server de façon à installer le gestionnaire lors du démarrage du serveur.

Limite de rotation du fichier

La taille en octets que le fichier du journal du serveur doit atteindre avant la rotation. Lorsque le journal du serveur atteint la taille indiquée, le Enterprise Server crée un nouveau fichier vide nommé server.log et renomme l'ancien fichier server.log_date, où date indique la date et l'heure de rotation du fichier. La valeur par défaut est 2 mégaoctets. La valeur minimale est 500 ko. Si vous spécifiez une valeur inférieure, le fichier effectuera la rotation à 500 ko. Pour désactiver la rotation du fichier journal, spécifiez la valeur 0.

Temps limite de rotation du fichier

Nombre de minutes au bout duquel la rotation du journal du serveur doit s'effectuer. La valeur par défaut est 0, ce qui signifie que le fichier effectuera la rotation lorsqu'il aura atteint la taille spécifiée dans le champ Limite de rotation du fichier. Si vous spécifiez une ou plusieurs minutes, le temps limite remplace la taille limite indiquée par le champ Limite de rotation du fichier.
