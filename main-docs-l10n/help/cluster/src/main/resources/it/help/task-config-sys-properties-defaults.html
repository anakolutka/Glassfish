Per impostare i valori predefiniti per le proprietà di sistema della configurazione

I valori predefiniti delle proprietà di configurazione sono definiti nella configurazione alla quale fa riferimento il cluster o l'istanza.

  1. Espandere il nodo per la configurazione nell'albero di navigazione.

  2. Nella lista dei tipi di proprietà del nodo espanso, selezionare Proprietà di sistema.

    Viene visualizzata la pagina Proprietà di sistema per la configurazione selezionata.

    Nella pagina vengono visualizzati i valori correnti delle proprietà di sistema per la configurazione riportate di seguito.

    • I numeri di porta predefiniti al momento della creazione della configurazione

    • Le eventuali proprietà personalizzate aggiunte alla configurazione

  3. Per garantire l'applicazione delle modifiche senza dover riavviare il cluster o l'istanza, selezionare l'opzione Riconfigurazione dinamica.

  4. Per ogni proprietà per la quale si imposta un valore predefinito, digitare il nuovo valore nel campo Valore predefinito per la proprietà.

  5. Impostare i valori predefiniti per eventuali proprietà aggiuntive necessarie.

    1. Nella tabella Proprietà aggiuntive fare clic su Aggiungi proprietà.

      Viene aggiunta una riga alla tabella Proprietà aggiuntive.

    2. Nel campo Nome variabile istanza digitare il nome della proprietà.

    3. Nel campo Valore predefinito digitare il valore.

  6. Fare clic su Salva.
