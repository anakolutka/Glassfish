To Monitor and Roll Back Transactions

  1. Do one of the following:
    • For the developer profile, in the tree component, select the Application Server node. Click the Monitor tab.
    • For the cluster or enterprise profile In the tree component, select Common Tasks, then click View Monitoring Data.
  2. Click the Transactions tab.

    The page displays the transactions that are in process.

  3. To rollback a transaction, first click the Freeze button.

    This freezes the transaction subsystem and determines transactions that are in process at the time of the freeze.

  4. Select the checkbox beside the transaction ID and click Rollback.
  5. Click Unfreeze to continue viewing in-process transactions.
