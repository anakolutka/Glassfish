So löschen Sie einen HTTP-Listener

Bevor Sie beginnen

Wenn Sie vorhaben, die Standard-HTTP-Zielgeräte http-listener-1, http-listener-2 oder admin-listener zu löschen, erstellen Sie zuerst eine Sicherungskopie der http-listener-Elemente der domain.xml-Datei von Enterprise Server, sodass die Einstellungen bei Bedarf wiederhergestellt werden können.

  1. Erweitern Sie in der Navigationsstruktur den Konfigurationsknoten.
  2. Erweitern Sie unter dem Konfigurationsknoten den Knoten "HTTP-Dienst".
  3. Wählen Sie unter dem Knoten "HTTP-Dienst" den Knoten "HTTP-Zielgeräte" aus.
  4. Aktivieren Sie auf der Seite "HTTP-Dienst" das Kontrollkästchen neben dem Namen des zu löschenden HTTP-Zielgeräts.
  5. Klicken Sie auf „Löschen“.
