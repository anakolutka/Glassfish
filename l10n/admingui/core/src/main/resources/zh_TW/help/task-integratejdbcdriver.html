整合 JDBC 驅動程式

支援 JDBC API 的驅動程式 (JDBC 驅動程式) 會將應用程式的 JDBC 呼叫轉譯為資料庫伺服器的協定。

  • 若要將 JDBC 驅動程式整合到管理網域中，請執行以下操作之一：
    • 使通用類別載入器可以存取該驅動程式。
      1. 將驅動程式的 JAR 和 ZIP 檔案複製到 domain-dir/lib 目錄，或將類別檔案複製到 domain-dir/lib/ext 目錄中。
      2. 重新啟動該網域。
    • 使系統類別載入器可以存取該驅動程式。
      1. 在瀏覽樹狀結構中，選取 [Application Server] 節點。
      2. 從 [一般資訊] 頁面中，選取 [JVM 設定] 標籤。
      3. 從 [JVM 一般設定] 頁面中，選取 [路徑設定] 標籤。
      4. 在 [類別路徑後綴] 欄位中，鍵入驅動程式的 JAR 檔案之完全合格的路徑名稱。
      5. 按一下 [儲存]。
      6. 重新啟動伺服器。
