Per rimuovere le applicazioni da un cluster

Con la rimozione l'applicazione viene rimossa solo dalla destinazione corrente. Quando viene rimossa da una destinazione, l'applicazione continua a essere elencata nella pagina Applicazioni, in cui sono visualizzate tutte le applicazioni distribuite in GlassFish Server. L'applicazione rimane inoltre distribuita su eventuali altre destinazioni sulle quali è stata effettuata la distribuzione.

  1. Espandere il nodo Cluster nell'albero di navigazione.

  2. Selezionare il nodo per il cluster sotto il nodo Cluster espanso.

    Viene visualizzata la pagina Informazioni generali per il cluster.

  3. Nella pagina Informazioni generali fare clic sulla scheda Applicazioni.

    Viene visualizzata la pagina Applicazioni per il cluster.

  4. Per ogni applicazione da rimuovere, selezionare la casella di controllo in corrispondenza dell'applicazione nella tabella Applicazioni distribuite.

  5. Fare clic su Rimuovi.

    Viene richiesto di confermare la rimozione delle applicazioni.

  6. Confermare che si desidera rimuovere le applicazioni.

    Le applicazioni vengono rimosse dalla tabella delle applicazioni distribuite visualizzata nella pagina.
