Applicazioni

Utilizzare la pagina Applicazioni per eseguire i task seguenti sull'istanza standalone di GlassFish Server selezionata:

  • visualizzazione e gestione delle applicazioni già distribuite nell'istanza;

  • distribuzione di altre applicazioni nell'istanza.

Il campo Nome istanza è un campo di sola lettura in cui viene visualizzato il nome dell'istanza corrente.

Nella pagina Applicazioni viene visualizzata la lista delle applicazioni distribuite all'istanza selezionata. Per ogni applicazione vengono fornite le informazioni riportate di seguito.

Nome

Il nome dell'applicazione.

Il nome può includere un identificativo di versione opzionale, che segue il nome ed è separato da questo mediante due punti (:). Per ulteriori informazioni sulle versioni di moduli e applicazioni, vedere Module and Application Versions in Oracle GlassFish Server Application Deployment Guide.

Abilitato

Indica se l'applicazione è abilitata:

  • se l'applicazione è abilitata, viene visualizzato un segno di spunta (✓);

  • se l'applicazione è disabilitata, viene visualizzata una croce (✗).

LB abilitato

Indica se l'applicazione è abilitata per il bilanciamento del carico:

  • se l'applicazione è abilitata per il bilanciamento del carico, viene visualizzato un segno di spunta (✓);

  • se l'applicazione è disabilitata per il bilanciamento del carico, viene visualizzata una croce (✗).

Motori

I tipi di contenitori utilizzati dall'applicazione. I contenitori possono essere di uno qualsiasi dei tipi seguenti:

  • web

  • webservices

  • ejb

  • connector

  • appclient

  • weld — contenitore per Contexts and Dependency Injection per le applicazioni della piattaforma Java EE

La tabella Applicazioni distribuite contiene inoltre le opzioni riportate di seguito.

Distribuisci

Pulsante che consente di distribuire un'applicazione.

Annulla distribuzione

Pulsante che consente di annullare la distribuzione di una o più applicazioni selezionate.

L'annullamento della distribuzione comporta la rimozione dell'applicazione da tutte le destinazioni e dal dominio. Quando la distribuzione di un'applicazione viene annullata, l'applicazione non viene più elencata nella pagina Applicazioni, in cui sono visualizzate tutte le applicazioni distribuite in GlassFish Server.

Per rimuovere le applicazioni solo dalla destinazione corrente, utilizzare il pulsante Rimuovi di questa pagina.

Rimuovi

Pulsante che consente di rimuovere una o più applicazioni selezionate.

Con la rimozione l'applicazione viene rimossa solo dalla destinazione corrente. Quando viene rimossa da una destinazione, l'applicazione continua a essere elencata nella pagina Applicazioni, in cui sono visualizzate tutte le applicazioni distribuite in GlassFish Server. L'applicazione rimane inoltre distribuita su eventuali altre destinazioni sulle quali è stata effettuata la distribuzione.

Per rimuovere le applicazioni da tutte le destinazioni e dal dominio, utilizzare il pulsante Annulla distribuzione di questa pagina.

Altre azioni

Elenco a discesa che consente di eseguire azioni aggiuntive su una o più applicazioni selezionate.

Abilita

Abilita una o più applicazioni selezionate.

Disabilita

Disabilita una o più applicazioni selezionate.

Abilitazione load balancer

Abilita una o più applicazioni selezionate per il bilanciamento del carico.

Disabilitazione load balancer

Disabilita una o più applicazioni selezionate per il bilanciamento del carico.
