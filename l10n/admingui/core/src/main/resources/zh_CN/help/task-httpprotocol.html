配置 HTTP 协议

  1. 在导航树中，展开“配置”节点。
  2. 在“配置”节点下，选择“HTTP 服务”节点。
  3. 从“HTTP 服务”页面中，单击“HTTP 协议”选项卡。

    将打开“HTTP 协议”页面。

  4. 在“版本”字段中，键入要使用的 HTTP 协议的版本。

    接受的值有 HTTP/1.0 和 HTTP/1.1。默认值为 HTTP/1.1。

  5. 选中“启用 DNS 查找”复选框，以对客户机启用 DNS 条目查找。

    默认情况下禁用此选项。

  6. 取消选中“启用 SSL”复选框，以全局禁用服务器的安全性。

    如果将此复选框保持为选中状态，则可以将 SSL 用于任何启用了安全性的侦听器。默认情况下启用此选项。

  7. 在“强制类型”字段中，键入没有与扩展名匹配的可用 MIME 映射时使用的响应类型。

    默认值为 text/html; charset=iso-8859-1。

  8. 在“默认类型”字段中，输入默认响应类型。

    默认值为 text/html; charset=iso-8859-1。此值是以分号分隔的字符串，由内容类型、编码、语言和字符集组成。

  9. 单击“保存”。

    提示 – 要返回到默认设置，请单击“装入默认值”按钮，然后单击“保存”。
