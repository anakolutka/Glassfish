To Manage Load Balancer Targets

  1. In the tree component, select the HTTP Load Balancers node.
  2. In the right pane, click the load balancer to edit.
  3. Click the Targets tab.
  4. To add or remove targets, click Manage Targets.

    The Manage Targets page appears.

  5. Add or remove targets by highlighting the cluster or stand-alone instance name and clicking Add or Remove.

    To add or remove all targets click Add All or Remove All.

  6. Click OK.
