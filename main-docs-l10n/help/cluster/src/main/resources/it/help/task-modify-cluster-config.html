Per modificare la configurazione di un cluster

  1. Espandere il nodo Cluster nell'albero di navigazione.

  2. Selezionare il nodo per il cluster.

  3. Fare clic sul nome della configurazione.

    Viene visualizzata la pagina Configurazione per il cluster.

  4. Nella pagina Configurazione selezionare il componente della configurazione da modificare.

    Viene visualizzata la pagina per la modifica del componente della configurazione selezionato.

  5. Nella pagina per la modifica del componente selezionato, eseguire e salvare le modifiche necessarie.

    Per i dettagli, consultare la Guida in linea relativa alla pagina per la modifica del componente selezionato.
