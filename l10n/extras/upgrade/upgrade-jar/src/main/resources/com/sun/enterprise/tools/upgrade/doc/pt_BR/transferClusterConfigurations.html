Tela Transfer Cluster Configurations

A atualização da configuração de um cluster Sun Java System Application Server 8.x Enterprise Edition (EE) para Sun Java System Application Server 9.1 requer um arquivo clinstance.conf válido. A Upgrade Tool pressupõe que o arquivo clinstance.conf é uma cópia exata do cluster 8.x EE. Se, por alguma razão, este arquivo não corresponder exatamente à configuração atual do cluster, a definição do cluster deverá ser editada manualmente antes que a atualização seja efetuada.

Use a tela Transfer Cluster Configurations para especificar o(s) arquivo(s) de configuração que serão migrados para a nova instalação.

  1. Pressione o botão Add e a caixa de diálogo Select clinstance.conf file será exibida.
  2. Selecione o arquivo de cluster clinstance.conf , que contém as informações de configuração do cluster que serão migradas.
  3. Pressione o botão Open.
  4. Repita esse processo até que todos os arquivos de configuração do cluster que deve ser migrado tenham sido adicionados.
  5. Pressione o botão Next ao terminar. A tela Upgrade Results será exibida, mostrando o status da atualização.
