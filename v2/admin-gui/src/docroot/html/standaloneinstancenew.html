To Create a New Standalone Server Instance

  1. In the tree, select the Stand-Alone Instances node.

    The Standalone Server Instances page appears.

  2. To create a new stand-alone instance, click New. The New Stand-Alone Server Instance page appears where you can create the new instance.
  3. In the name field, type a unique name for the new instance.
  4. From the Node Agent drop-down list, select a node agent for the new instance.
  5. From the Configuration drop-down list, select a configuration for the instance

    • To create a stand-alone instance, choose default-config, and leave the radio button labeled Make a copy of the selected Configuration selected. The copy of the default configuration will have the name instance_name-config.

    • To reference another configuration, choose the configuration from the drop-down list and select the radio button labeled Reference the selected Configuration. This action creates a shared instance if the configuration is used by another instance.

  6. Click OK to save your changes.
