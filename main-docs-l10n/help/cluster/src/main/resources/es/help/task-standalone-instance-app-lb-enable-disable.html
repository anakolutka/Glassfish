Activar o desactivar el equilibrio de carga para una aplicación desplegada en una instancia de GlassFish Server independiente

  1. En el árbol de navegación, seleccione el nodo Instancias Independientes.

    Se abrirá la página Instancias de Servidor Independientes.

  2. En la página Instancias de Servidor Independientes, seleccione la instancia.

    Se abrirá la página Información General para la instancia.

  3. En la página Información General para la instancia, haga clic en el separador Aplicaciones.

    Se abrirá la página Aplicaciones para la instancia.

  4. Active la casilla de control que aparece junto a la aplicación.

  5. En el menú desplegable Más Acciones, seleccione el comando pertinente.

    • Para activar el equilibrio de carga para la aplicación, seleccione Activación de Equilibrador de Carga.

    • Para desactivar el equilibrio de carga para la aplicación, seleccione Desactivación de Equilibrador de Carga.
