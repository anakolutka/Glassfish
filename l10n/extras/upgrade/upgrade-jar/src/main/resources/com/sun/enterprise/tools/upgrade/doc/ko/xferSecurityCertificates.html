보안 인증서 전송 화면

보안 인증서 전송 화면을 사용하여 Sun Java Systems Application Server 9로 마이그레이션해야 하는 보안 인증서 집합(한 개 또는 여러 개)이 있는 도메인을 지정합니다.

  1. 추가 버튼을 눌러 도메인 이름 및 인증서 파일 비밀번호를 입력하는 도메인 추가 대화 상자를 표시합니다.

    전송할 인증서가 있는 도메인 패널에 추가된 도메인이 표시됩니다.

  2. 전송해야 하는 인증서가 있는 모든 도메인이 추가될 때까지 이 과정을 반복합니다.
  3. 추가된 도메인을 삭제하려면 삭제할 도메인을 선택한 다음 도메인 삭제 버튼을 누릅니다.
  4. 완료되면 다음 버튼을 누릅니다.

    업그레이드 결과 화면에 업그레이드 상태가 표시됩니다.
