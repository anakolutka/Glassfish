Modifier une application Web déployée

  1. Dans l'arborescence de navigation, développez le nœud Applications.
  2. Sélectionnez le nœud Applications Web.
  3. Dans le tableau Applications Web déployées, cliquez sur le nom de l'application à modifier.

    La page Application Web s'ouvre.

  4. Dans le champ Racine contexte, saisissez le chemin d'accès à cette application Web.

    Dans l'URL de l'application Web, la racine contexte suit immédiatement le numéro de port (http://host: port/context-root/...). Assurez-vous que la racine contexte commence par une barre oblique, comme par exemple /hello.

  5. Dans le champ Description, entrez une brève description de cette application Web.
  6. Cliquez sur Enregistrer.
