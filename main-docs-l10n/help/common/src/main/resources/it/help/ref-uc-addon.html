Componenti aggiuntivi disponibili

La pagina Componenti aggiuntivi disponibili fornisce informazioni sui componenti dell'immagine di installazione non ancora installati.

La pagina Componenti aggiuntivi disponibili contiene le opzioni riportate di seguito.

Percorso di installazione

Il percorso del file system in cui si trova la directory radice o di livello superiore dell'immagine di installazione.

Installa

Pulsante che consente di installare o più componenti aggiuntivi selezionati.

Componenti aggiuntivi disponibili

Per ogni componente dell'immagine di installazione non ancora installato, questa tabella fornisce le informazioni riportate di seguito.

Componente

Il nome del componente.

Categoria

La categoria del software a cui è associato il componente, ad esempio Application Server, strumenti di sistema e database. La categoria viene assegnata al componente dall'autore al momento della creazione del componente.

Versione

Il numero di versione del package in cui viene fornito il componente.

Dimensioni installazione

La quantità di spazio su disco usata per l'installazione del componente.

Origine

Il nome del Repository di rete da cui si ottiene il componente.

Task correlati

Per installare i componenti aggiuntivi

Comandi asadmin correlati
