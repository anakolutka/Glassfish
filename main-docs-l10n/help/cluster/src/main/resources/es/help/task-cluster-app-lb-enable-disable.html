Activar o desactivar el equilibrio de carga para una aplicación desplegada en un cluster

  1. En el árbol de navegación, amplíe el nodo Clusters.

  2. En el nodo Clusters ampliado, seleccione el nodo para el cluster.

    Se abre la página Información General para el cluster.

  3. En la página Información General, haga clic en el separador Aplicaciones.

    Se abre la página Aplicaciones para el cluster.

  4. Active la casilla de control que aparece junto a la aplicación.

  5. En el menú desplegable Más Acciones, seleccione el comando pertinente.

    • Para activar el equilibrio de carga para la aplicación, seleccione Activación de Equilibrador de Carga.

    • Para desactivar el equilibrio de carga para la aplicación, seleccione Desactivación de Equilibrador de Carga.
