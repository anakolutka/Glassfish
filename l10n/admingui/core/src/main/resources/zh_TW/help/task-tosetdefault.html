設定預設範圍

應用程式的部署描述元未指定範圍時，Enterprise Server 可使用 預設範圍 進行認證和授權。

  1. 在瀏覽樹狀結構中，展開 [配置] 節點。
  2. 若將網域配置為支援叢集，請選取您要變更的配置。

    例如，透過 cluster 設定檔或 enterprise 設定檔建立的網域。

  3. 選取 [安全性] 節點。

    [安全性] 頁面會隨即開啟。

  4. 在 [預設範圍] 欄位中，從下拉式清單挑選所需的範圍。
  5. 按一下 [儲存] 以儲存變更。
  6. 如果主控台中顯示 [需要重新啟動]，請重新啟動伺服器。
