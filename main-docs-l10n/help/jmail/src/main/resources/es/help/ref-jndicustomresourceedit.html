Editar Recurso Personalizado

Utilice la página Editar recurso personalizado para editar un recurso JNDI personalizado.

La página Editar recurso personalizado contiene las siguientes opciones.

Nombre JNDI

Nombre único que identifica el recurso. El nombre JNDI es un campo de sólo lectura. Únicamente puede especificar el nombre JNDI al crear un nuevo recurso personalizado.

Tipo de recurso

Definición de tipo totalmente cualificada para el recurso.

Clase de fábrica

Nombre especificado por el usuario para la clase de fábrica. Esta clase implanta la interfaz javax.naming.spi.ObjectFactory.

Orden de despliegue

Orden de despliegue del recurso personalizado.

Descripción

Descripción de este recurso personalizado.

Estado

Indica si el recurso está disponible en el tiempo de ejecución.

Si sólo existe la instancia de servidor por defecto, server en el dominio, la disponibilidad en tiempo de ejecución del recurso puede estar activada o desactivada. La disponibilidad en tiempo de ejecución está activa por defecto.

Si se ha creado algún cluster o alguna instancia independiente en el dominio, utilice el separador Destino para controlar la disponibilidad del recurso en tiempo de ejecución en sus destinos.

Propiedades Adicionales

Propiedades adicionales para el recurso personalizado. GlassFish Server no define ninguna propiedad adicional de los recursos personalizados.
