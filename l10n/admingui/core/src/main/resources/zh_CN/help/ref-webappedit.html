Web 应用程序

使用 Web 应用程序页面可以修改现有 Web 应用程序。此页面包含以下选项：

名称

显示要编辑的 Web 应用程序名称的只读字段。

上下文根目录

指定 Web 应用程序的路径。在 Web 应用程序的 URL 中，上下文根目录要紧跟着端口号 (http://host:port/context-root/...)。上下文根目录以正斜杠开头，例如：/hello。

虚拟服务器

与此 Web 应用程序相关联的虚拟服务器。

说明

Web 应用程序的说明。

启用状态

如果选中此选项，则将启用 Web 应用程序。默认情况下，此选项是选中的。

位置

显示归档文件（应用程序通过此文件进行部署）位置的只读字段。
