JMS-Zielressource bearbeiten

Auf der Seite "JMS-Zielressource bearbeiten" können Sie die Einstellungen für eine JMS-Zielressource ändern.

Die Seite "JMS-Zielressource bearbeiten" enthält die folgenden Optionen.

Standardeinstellungen laden

Schaltfläche zur Wiederherstellung der Standardwerte von Einstellungen, die Standardwerte haben. Einstellungen, die keine Standardwerte haben, werden nicht geändert.

JNDI-Name

Ein eindeutiger Name, der die Zielressource identifiziert. Der JNDI-Name ist ein schreibgeschütztes Feld. Sie können den JNDI-Namen nur festlegen, wenn Sie eine neue JMS-Zielressource erstellen.

Name des physikalischen Ziels

Der Name des mit dieser Ressource zu verknüpfenden physischen Ziels.

Das physische Ziel wird von GlassFish Server bei Bedarf automatisch erstellt und wieder gelöscht, wenn Sie die Zielressource löschen.

Ressourcentyp

Der Typ der Zielressource. Zur Auswahl stehen javax.jms.Topic und javax.jms.Queue.

Beschreibung

Eine Beschreibung der Zielressource.

Status

Der Status der Zielressource. Die Zielressource kann aktiviert oder deaktiviert sein. Diese Option ist standardmäßig aktiviert.

Weitere Eigenschaften

Weitere Eigenschaften für die Zielressource. In GlassFish Server werden keine weiteren Eigenschaften für JMS-Zielressourcen definiert.

Verwandte Aufgaben

  • So werden JMS-Zielressourcen erstellt

  • So werden JMS-Zielressourcen bearbeitet

  • So werden JMS-Zielressourcen gelöscht
