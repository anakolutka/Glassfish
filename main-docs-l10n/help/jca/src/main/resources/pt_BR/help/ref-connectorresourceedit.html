Editar Recurso Conector

Use a página Editar Recurso Conector para modificar as definições de recurso do conector.

A página Editar Recurso Conector contém as seguintes opções.

Carregar Defaults

Botão para restaurar definições para seus valores default. Definições que não tiveram seus valores default alterados.

Nome JNDI

Um nome exclusivo que identifica o recurso do conector. O nome JNDI é um campo somente para leitura. Você somente pode especificar o nome JNDI ao criar um novo recurso conector.

Nome do Pool

O pool de conexões do conector associado com este recurso.

Ordem de Implantação

A ordem de implantação do recurso conector.

Descrição

Uma breve descrição do recurso do conector.

Status

Se o recurso está disponível no runtime.

Se apenas a instância do servidor default, server, existir no domínio, a disponibilidade de runtime do recurso poderá ser ativada ou desativada. A disponibilidade de runtime está ativada por default.

Se clusters ou instâncias standalone tiverem sido criados no domínio, utilize a guia Alvo para controlar a disponibilidade de runtime do recurso em seus alvos.

Propriedades Adicionais

Propriedades adicionais para o recurso conector. As propriedades disponíveis dependem do adaptador de recursos para o pool de conexões do conector. Os pares de nome e valor especificados nessa tabela podem ser usados para substituir os valores default definidos pelo fornecedor do adaptador de recurso.
