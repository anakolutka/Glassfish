日常任务

“日常任务”页面提供日常管理控制台任务的快捷方式。

可使用“日常任务”页面进行以下操作：

  • 在 Sun Connection 中注册 Sun GlassFish Enterprise Server

  • 查看 Sun GlassFish Enterprise Server 的"GlassFish 新闻"页面

  • 查看 Sun GlassFish Enterprise Server 的"订阅"页面

  • 查看已部署应用程序列表

  • 部署 Web 应用程序

  • 部署企业应用程序

  • 部署应用程序客户机

  • 部署连接器模块

  • 部署 EJB Jar

  • 卸载组件

  • 安装附加组件

  • 安装组件更新

  • To Create a JDBC Connection Pool

“日常任务”页面还提供了某些 Enterprise Server 文档的链接。
