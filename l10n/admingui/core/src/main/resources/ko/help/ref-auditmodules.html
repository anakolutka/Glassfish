감사 모듈

감사는 Enterprise Server에서 오류나 보안 위반과 같은 중요한 이벤트를 기록하는 방법입니다. Enterprise Server는 모든 인증 이벤트를 로그 파일에 기록합니다. 전체 액세스 로그는 Enterprise Server 액세스 이벤트의 순차적인 추적을 제공합니다.

감사 모듈 페이지에서는 사용 가능한 감사 모듈 목록을 표시합니다. 각 감사 모듈에 대해 제공되는 정보는 다음과 같습니다.

이름

감사 모듈의 이름입니다.

클래스 이름

이 모듈을 구현하는 클래스의 정규화된 이름입니다. 기본 감사 모듈의 클래스 이름은 com.sun.enterprise.security.Audit입니다.

관련 asadmin 명령

  • create-audit-module(1)

  • list-audit-modules(1)

  • delete-audit-module(1)
