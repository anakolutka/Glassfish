Editar Alias de Senha

Utilize a página Editar Alias de Senha para alterar a senha associada a um alias de senha existente.

A página Editar Alias de Senha contém as seguintes opções:

Nome do Alias

O nome do alias. Esta opção não pode ser editada.

Senha

A senha que o alias de senha armazena na forma criptografada.

Confirmar Senha

A senha que o alias de senha armazena na forma criptografada. Os valores de Senha e Confirmar senha são comparados para garantir que a senha tenha sido introduzida sem erros.
