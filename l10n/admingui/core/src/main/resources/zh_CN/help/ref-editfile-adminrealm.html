特定于 FileRealm 类的属性

对于 file 领域，需要以下属性：

JAAS 上下文

要用于此领域的登录模块类型。唯一的有效值是 fileRealm。

密钥文件

文件（服务器将在其中存储此领域的所有用户、组和密码信息）的完整路径和名称。对于 file 领域，默认值是 domain-dir/config/keyfile；对于 admin-realm 领域，默认值是 domain-dir/config/admin-keyfile。

file 领域的密钥文件最初为空，因此在使用 file 领域之前，必须在文件用户页面上添加用户。

admin-realm 领域的密钥文件最初包含管理员用户名、加密格式的管理员密码，以及此用户所属的组（默认情况下为 asadmin）。

注 – admin-realm 的 asadmin 组中的用户已被授权，可以使用管理控制台和 asadmin 工具。只能将具有服务器管理权限的用户添加到该组中。
