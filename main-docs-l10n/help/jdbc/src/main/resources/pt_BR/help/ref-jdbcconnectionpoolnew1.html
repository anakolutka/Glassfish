Novo Pool de Conexões do JDBC (etapa 1 de 2)

Use a página Novo Pool de Conexões do JDBC (Etapa 1 de 2) para criar um novo pool de conexões.

A página Novo Pool de Conexões do JDBC (Etapa 1 de 2) contém as opções a seguir.

Nome do Pool

O nome do pool de conexões.

Tipo de Recurso

Os tipos de recursos disponíveis são javax.sql.XADataSource (transações globais), java.sql.ConnectionPoolDataSource (transações locais, possíveis melhorias de desempenho) e javax.sql.DataSource (somente transações locais) e java.sql.Driver.

Fornecedor do Driver do Banco de Dados

O nome do fornecedor do produto de banco de dados.

Omitir

Se esta opção for ativada, o driver do banco de dados será omitido com base no Tipo de Recurso e no Fornecedor do Driver do Banco de Dados, e qualquer fonte de dados ou nomes de classe do driver descobertos serão exibidos na página Novo Pool de Conexões do JDBC (etapa 2 de 2) quando você clicar no botão Próximo.

Para que a omissão funcione, o driver JDBC deve estar disponível para GlassFish Server. Consulte Para Integrar um Driver JDBC.
