Para Interromper Instâncias do GlassFish Server Clusterizadas

  1. Na árvore de navegação, expanda o nó Clusters.

  2. No nó Clusters expandidos, selecione o nó do cluster.

    A página Informações Gerais do cluster será aberta.

  3. Na página Informações Gerais, clique na guia Instâncias.

    A página Instâncias de Servidor Clusterizado será aberta com informações sobre o cluster.

  4. Para cada instância que estiver sendo interrompida, marque a caixa de seleção ao lado da instância na tabela Instâncias do Servidor.

  5. Clique em Interromper.

    Será solicitado que você confirme se deseja interromper as instâncias.

  6. Confirme se deseja interromper as instâncias.

    O status de cada instância é atualizado na tabela Instâncias do Servidor quando a instância for interrompida.
