Complementos disponibles

La página Complementos disponibles ofrece información sobre los componentes de la imagen de instalación que todavía no están instalados.

La página Complementos disponibles contiene las siguientes opciones.

Ruta de instalación

Ruta de acceso del sistema de archivos del directorio raíz o nivel superior de la imagen de instalación.

Instalar

Botón para instalar uno o más complementos seleccionados.

Complementos disponibles

Para cada componente de la imagen de instalación que aún no está instalado, esta tabla proporciona la siguiente información.

Componente

Nombre del componente.

Categoría

Categoría del software al que se asocia el componente, por ejemplo, servidores de aplicaciones, herramientas del sistema y bases de datos. El autor del componente le asigna la categoría al mismo cuando se crea el componente.

Versión

Número de versión del paquete en el que se proporciona el componente.

Tamaño de instalación

Cantidad de espacio en disco que consume la instalación del componente.

Origen

Nombre del Repositorio de red del que se obtendrá el componente.

Tareas relacionadas
