웹 컨테이너 모니터링 데이터를 보는 방법

  1. 탐색 트리에서 Application Server 노드를 선택합니다.
  2. 일반 정보 페이지에서 모니터 탭을 누릅니다.

    웹 컨테이너 모니터링 페이지가 열립니다.

  3. (옵션) 응용 프로그램 드롭다운 목록에서 해당 응용 프로그램에 대한 모니터링 데이터를 볼 응용 프로그램을 선택합니다.
  4. (옵션) 가상 서버 드롭다운 목록에서 해당 서버에 대한 모니터링 데이터를 볼 가상 서버를 선택합니다.
  5. (옵션) 갱신 버튼을 눌러 표시된 모니터링 데이터를 업데이트합니다.
