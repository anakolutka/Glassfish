Noeuds

La page Noeuds présente la liste des noeuds du domaine. Pour chaque noeud, les informations suivantes sont affichées :

Nom

Nom affecté au noeud lors de sa création. Lorsque vous cliquez sur le nom, la page Modifier le noeud du noeud s'ouvre.

Hôte du noeud

Nom de l'hôte que le noeud représente.

Type

Type de noeud parmi les suivants :

CONFIG

Le noeud ne prend pas en charge la communication à distance.

DCOM

Le noeud prend en charge la communication via le protocole distant DCOM (Distributed Component Object Model). Le protocole DCOM est disponible uniquement pour les systèmes Windows.

SSH

Le noeud prend en charge la communication via le protocole SSH (shell sécurisé).

Instances

Liste des instances résidant sur le noeud, avec une indication relative au statut d'exécution de chaque instance :

  • Si l'instance est en cours d'exécution, une coche (✓) s'affiche.

  • Si l'instance est arrêtée, un point d'exclamation (!) apparaît.

Lorsque vous cliquez sur le nom d'une instance, la page Informations générales de l'instance s'ouvre.

Action

Noeuds de type DCOM et SSH uniquement : lien vers l'action Ping pour le noeud. Le fait de cliquer sur le lien Ping permet de vérifier si le noeud est accessible via le protocole de communication qu'il prend en charge.

Aucune action n'est possible pour les noeuds CONFIG.

La table Noeuds contient également les options ci-après.

Nouveau

Bouton permettant de créer un noeud.

Supprimer

Bouton permettant de supprimer les noeuds sélectionnés.

Supprimer et désinstaller

Bouton permettant de supprimer les noeuds sélectionnés et de désinstaller le logiciel GlassFish Server de l'hôte représenté par chaque noeud supprimé.

Remarque:

Le logiciel GlassFish Server n'est pas désinstallé des hôtes représentés par un noeud de type CONFIG. Pour les noeuds de type CONFIG, l'action de ce bouton est la même que celle du bouton Supprimer.
