Recursos

Utilize a página Recursos para realizar as seguintes tarefas na instância standalone selecionada do GlassFish Server:

  • Exibindo e gerenciando os recursos existentes da instância

  • Criando recursos adicionais para a instância

O campo Nome da Instância é um campo somente para leitura que exibe o nome da instância atual.

A página Recursos exibe uma lista de recursos existentes da instância. Para cada recurso, as informações a seguir são fornecidas.

Nome do Recurso

Nome designado ao recurso quando o recurso é criado. Ao clicar no nome, abre-se uma das seguintes páginas para editar o recurso, dependendo do tipo do recurso:

  • Editar Recurso JDBC

  • Editar Sessão JavaMail

  • Editar Recurso Personalizado

  • Novo Recurso Personalizado

  • Editar Recurso Externo

  • Editar Recurso Conector

  • Editar Recurso do Objeto de Admin

Ativado

Indicação de que o recurso está ativado:

  • Se o recurso estiver ativado, uma marca de verificação (✓) será exibida.

  • Se o recurso for desativado, um ✗ () é exibido.

Tipo

O tipo de recurso.

A tabela Recursos também contém as opções a seguir.

Ativar

Botão utilizado para ativar um ou mais recursos selecionados.

Desativar

Botão utilizado para desativar um ou mais recursos selecionados.

Novo

Lista drop-down que cria um recurso do tipo selecionado. Selecionar um tipo de recurso na lista abre uma das páginas a seguir para criar um recurso do tipo selecionado:

  • Novo Recurso do JDBC

  • Nova Fábrica de Conexões JMS

  • Novo Recurso de Destino JMS

  • Nova Sessão JavaMail

  • Novo Recurso Personalizado

  • Novo Recurso Externo

  • Novo Recurso do Conector

  • Novo Recurso de Objeto Admin

Filtro

Lista drop-down que filtra por tipo a lista de recursos exibidos.
