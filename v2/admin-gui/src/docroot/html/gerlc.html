To create a Profile

  1. Launch the asadmin utility.

    For instructions see Starting the asadmin Utility.

  2. Use the create-domain command to create the configuration of a domain by typing:

    $ asadmin create-domain --user username --passwordfile passwordfile --profile profile_name mynewdomain

    If a Developer profile is created, clustering and node agent features are not available for the newly created domain. To use clusters and server instances, identify the Cluster or Enterprise profile for the domain.

  3. When prompted identify the admin password and master password.

    Since the port numbers were not identified, the default port numbers for the properties of the domain being created are displayed.
