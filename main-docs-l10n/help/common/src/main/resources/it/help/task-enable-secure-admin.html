Per abilitare l'amministrazione sicura

  1. Selezionare il nodo server (server di amministrazione) nell'albero di navigazione.

    Viene visualizzata la pagina Informazioni generali.

  2. Nella pagina Informazioni generali fare clic sul pulsante Amministrazione sicura.

    Viene visualizzata la pagina Amministrazione sicura.

  3. Se l'installazione di GlassFish Server utilizza valori personalizzati per gli alias di amministrazione o delle istanze, aggiornare i valori predefiniti nei campi Alias di amministrazione e Alias istanze a seconda delle esigenze.

  4. Fare clic su Abilita amministrazione sicura.

    Viene visualizzato un messaggio che indica che l'abilitazione della amministrazione sicura comporta il riavvio immediato del server DAS.

  5. Fare clic su OK.

    L'amministrazione sicura viene abilitata e viene visualizzata la pagina di riavvio del server.
