So zeigen Sie die Beschreibung einer Anwendung an

  1. Wählen Sie in der Navigationsstruktur den Anwendungsknoten.

    Die Seite "Anwendungen" wird geöffnet.

  2. Klicken Sie auf der Seite "Anwendungen" auf den Namen der Anwendung, deren Beschreibung Sie anzeigen möchten.

    Die Seite "Anwendung bearbeiten" wird geöffnet.

  3. Klicken Sie auf der Seite "Anwendung bearbeiten" auf die Registerkarte "Beschreibung".

    Die Seite "Modulbeschreibungen" wird geöffnet.

  4. Klicken Sie auf der Registerkarte "Bereitstellungsbeschreibungen" auf den Namen der XML-Datei, die Sie anzeigen möchten.

    Die XML-Datei wird in einem neuen Fenster geöffnet.
