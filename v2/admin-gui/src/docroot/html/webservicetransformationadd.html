To Add a Transformation Rule

  1. In the tree view, expand the Web Services node.
  2. Select the web service for which you want to add a transformation rule.
  3. Click the Transformation tab.

    A list of transformation rules available for this web service end point is displayed.

  4. Click the Add button to add a transformation rule.
  5. Define the Transformation Rule settings as follows:

    • Name: Identifies the transformation rule to apply for a web service operation.

    • Filepath: Enter the file path location of the XSLT file that contains the transformation rule. All the generated XSLT files are stored in the generated/xml/application or module name/ directory.

  6. You can apply transformation rule to only a SOAP request, SOAP response or both.

    Note - If you add multiple transformation rules for a web service endpoint, the transformation rules are applied in the order in which they are added.
