A página Editar Realm será aberta.

Na página Editar Realm, clique no botão Gerenciar Usuários.

A página Usuários Admin será aberta.

Na página Usuários Administrativos, clique em Novo.

A página Novo Usuário de Realm de Arquivos será aberta.

No campo ID de Usuário, digite o nome do usuário.

No campo Lista de Grupos, digite uma lista dos grupos aos quais o usuário pertence.

Use vírgulas para separar os nomes de grupos. Esses grupos não precisam estar definidos em outro lugar.

No campo Senha. digite uma senha exclusiva.

No campo Confirmar Nova Senha, digite novamente a senha de usuário.

Clique em OK.
