Notes:

function elements define “handlers” that can be used during events. A function element associates a name with a Java method. It also defines the expected inputs and/or outputs that method (handler) expects. This is covered in more detail later in these slides, including features such as automatic type conversion and validating the correct inputs are supplied.

It is worth noting here that on the previous slide, a “call” referenced this “function”.
