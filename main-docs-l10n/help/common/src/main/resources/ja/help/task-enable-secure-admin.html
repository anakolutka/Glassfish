セキュリティ保護された管理を有効にするには

  1. ナビゲーション・ツリーで、サーバー(管理サーバー)ノードを選択します。

    「一般情報」ページが開きます。

  2. 「一般情報」ページで、「セキュリティ保護された管理」ボタンをクリックします。

    「セキュリティ保護された管理」ページが開きます。

  3. GlassFish Serverのインストールでカスタマイズされた管理エイリアスまたはインスタンス・エイリアスの値を使用する場合は、「管理エイリアス」および「インスタンス・エイリアス」フィールドに表示されるデフォルト値を必要に応じて更新します。

  4. 「セキュリティ保護された管理の有効化」をクリックします。

    セキュリティ保護された管理を有効にすると、DASが即時に再起動されることを示すメッセージが表示されます。

  5. 「OK」をクリックします。

    セキュリティ保護された管理が有効になり、サーバーを再起動するページが開きます。
