Propiedades específicas de la clase PamRealm

Un dominio PAM requiere la siguiente propiedad.

Contexto JAAS

Contexto JAAS (Java Authentication and Authorization Service) (identificador para el módulo de conexión que se utiliza para este dominio). El único valor válido es pamRealm.

La siguiente propiedad adicional está disponible para un dominio PAM.

Asignar grupos

Lista separada por comas de nombres de grupos. Todos los clientes que presenten certificados válidos se asignan a estos grupos, por ejemplo, employee,manager , donde éstos son los nombres de los grupos de usuarios.
