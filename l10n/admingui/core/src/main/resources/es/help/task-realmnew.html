Cómo crear un dominio

Servidor empresarial viene preconfigurado con tres dominios: file, certificate y admin-realm. También es posible crear dominios ldap, JDBC, solaris y personalizados. Normalmente se tiene un dominio de cada tipo en un servidor, pero en Servidor empresarial existen dos dominios de archivo: file y admin-realm. Se trata de dos dominios del mismo tipo utilizados para dos fines diferentes. También es posible tener una base de datos de certificados diferente para cada servidor virtual de su sistema.

  1. En el árbol de navegación, expanda el nodo Configuración.
  2. Si el dominio está configurado para admitir clústeres, seleccione la configuración que esté cambiando.

    Algunos ejemplos son los dominios que se crean con el perfil clúster o con el perfil grandes empresas .

  3. Expanda el nodo Seguridad.
  4. Seleccione el nodo Dominios.
  5. En la página Dominios haga clic en Nuevo.

    Aparecerá la página Dominio nuevo.

  6. Escriba un nombre para el dominio en el campo "Nombre".
  7. Especifique el nombre de clase del dominio que está creando.

    En la tabla siguiente se muestran las opciones válidas.

    Nombre de dominio

    Nombre de clase

    file

    com.sun.enterprise.security.auth.realm.file.FileRealm

    certificate

    com.sun.enterprise.security.auth.realm.certificate.CertificateRealm

    ldap

    com.sun.enterprise.security.auth.realm.ldap.LDAPRealm

    JDBC

    com.sun.enterprise.security.auth.realm.jdbc.JDBCRealm

    solaris

    com.sun.enterprise.security.auth.realm.solaris.SolarisRealm

    custom

    Nombre de clase de dominio de inicio de sesión
  8. Agregue las propiedades adicionales que quiera para el dominio.
    1. Haga clic en Agregar propiedad.
    2. En el campo Nombre introduzca el nombre de la propiedad.

      • Para obtener una descripción de las propiedades del dominio de archivo, consulte Propiedades específicas de la clase FileRealm.

      • Para obtener una descripción de las propiedades del dominio de certificado, consulte Propiedades específicas de la clase CertificateRealm .

    3. En el campo Valor introduzca el valor de la propiedad.
  9. Haga clic en Aceptar.
