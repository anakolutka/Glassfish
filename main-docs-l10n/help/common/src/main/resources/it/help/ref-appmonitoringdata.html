Monitoraggio applicazioni

Utilizzare la pagina Monitoraggio applicazioni per visualizzare i dati di monitoraggio per l'applicazione selezionata.

La pagina Monitoraggio applicazioni contiene le informazioni riportate di seguito.

Nome istanza

Il nome dell'istanza server.

Applicazione

L'applicazione per la quale vengono visualizzate le statistiche di monitoraggio.

Componente

Il componente dell'applicazione per il quale vengono visualizzate le statistiche di monitoraggio.

Per una descrizione delle proprietà monitorate, vedere il Capitolo 8, Amministrazione del servizio di monitoraggio, in Oracle GlassFish Server Administration Guide.
