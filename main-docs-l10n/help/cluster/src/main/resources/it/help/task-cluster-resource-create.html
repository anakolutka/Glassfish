Per creare una risorsa per un cluster

  1. Espandere il nodo Cluster nell'albero di navigazione.

  2. Selezionare il nodo per il cluster sotto il nodo Cluster espanso.

    Viene visualizzata la pagina Informazioni generali per il cluster.

  3. Nella pagina Informazioni generali fare clic sulla scheda Risorse.

    Viene visualizzata la pagina Risorse per il cluster.

  4. Nel menu a discesa Nuovo selezionare il tipo di risorsa da creare.

    Viene visualizzata la pagina per la creazione di una risorsa del tipo selezionato.

  5. Nella pagina per la creazione di una risorsa del tipo selezionato, apportare e salvare le modifiche necessarie.

    Per i dettagli, consultare la Guida in linea relativa alla pagina per la creazione di una risorsa del tipo selezionato.

    Nota:

    Quando si crea la risorsa, assicurarsi di specificare il cluster come destinazione.
