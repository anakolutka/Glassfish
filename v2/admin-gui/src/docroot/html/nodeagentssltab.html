Configuring the Node Agent for SSL

  1. Set client authentication.

    To require clients to authenticate themselves to the server when using this listener, check the Enabled box in the Client Authentication field.

  2. Enter a certificate nickname.

    Enter the name of an existing server keypair and certificate in the Certificate NickName field.

  3. Check the security protocol(s) to enable on the listener.

    You must check either SSL3, SSL2, or TLS, or all three protocols.

  4. In the Ciphersuites section, select the cipher suite used by the protocol(s).

    To enable all cipher suites, move all supported cipher suites to the respective Selected lists.

  5. Click Save.
