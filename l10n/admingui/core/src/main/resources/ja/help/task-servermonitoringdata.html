サーバーの監視データを表示する

始める前に

監視データを表示する前に、監視を設定する必要があります。詳細は、「監視サービスを設定する」を参照してください。

  1. ナビゲーションツリーで、Enterprise Server ノードを選択します。

    「一般情報」ページが開きます。

  2. 「一般情報」ページで、「監視」タブをクリックします。

    「アプリケーションの監視」ページが開きます。

  3. 「サーバー」タブをクリックします。

    「サーバーの監視」ページが開きます。

  4. 「仮想サーバー」ドロップダウンリストから、監視データを表示する仮想サーバーを選択します。
  5. 「統計情報の取得」ボタンをクリックして、監視統計情報を取得します。

    表に監視統計情報が表示されます。

  6. (省略可能) 表示されている監視データを更新するには、「更新」ボタンをクリックします。
