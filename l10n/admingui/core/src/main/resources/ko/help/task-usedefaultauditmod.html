기본 감사 모듈을 사용하는 방법

default 감사 모듈은 인증 및 권한 부여 요청을 서버 로그 파일에 기록합니다. 로그 파일의 위치를 변경하는 방법에 대한 자세한 내용은 일반 로깅 설정을 구성하는 방법을 참조하십시오.

인증 로그 항목에는 다음 정보가 포함됩니다.

  • 인증을 시도한 사용자의 이름

  • 액세스 요청을 처리한 영역

  • 요청된 웹 모듈 URI 또는 EJB 구성 요소

  • 요청의 성공 또는 실패 여부

감사 로깅의 활성화 여부에 관계없이 Enterprise Server는 거부된 모든 인증 이벤트를 기록합니다.

권한 부여 로그 항목에는 다음 정보가 포함됩니다.

  • 인증된 사용자의 이름(있는 경우)

  • 요청된 웹 URI 또는 EJB 구성 요소

  • 요청의 성공 또는 실패 여부

  1. 탐색 트리에서 구성 노드를 선택합니다.
  2. 구성 노드에서 보안 노드를 선택합니다.

    보안 페이지가 열립니다.

  3. 감사 모듈 드롭다운 목록에서 default라는 감사 모듈을 선택합니다.
  4. 저장을 누릅니다.
