Para Exibir Dados de Monitoramento do Recurso

Antes de Começar

Antes de você poder exibir os dados de monitoramento, é preciso configurar o monitoramento. Consulte Para Configurar o Serviço de Monitoramento para obter detalhes.

  1. Na árvore de navegação, selecione o nó Dados de Monitoramento.

    A página Monitoramento será aberta.

  2. Na coluna Exibir Dados de Monitoramento, clique no link Recursos que corresponde à instância do servidor que você está monitorando.

    A página Monitoramento de Recursos será aberta.

  3. Na lista drop-down Pool de Conexões, selecione o pool de conexões cujos dados de monitoramento você deseja ver.

  4. Clique no botão Atualizar para atualizar os dados de monitoramento exibidos.

Etapas alternativas:

  1. Na árvore de navegação, selecione o nó do servidor (Servidor Admin).

    A página Informações Gerais será aberta.

  2. Na página Informações Gerais, clique na guia Monitor.

    A página Monitoramento da Aplicação será aberta.

  3. Clique na guia Recursos.

    A página Monitoramento de Recursos será aberta.

  4. Na lista drop-down Pool de Conexões, selecione o pool de conexões cujos dados de monitoramento você deseja ver.

  5. Clique no botão Atualizar para atualizar os dados de monitoramento exibidos.
