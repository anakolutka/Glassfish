Gerenciar Alvos

Utilize a página Gerenciar Alvos para exibir os clusters do alvo o e as instâncias de servidores standalone nos quais uma aplicação pode estar ativada. A aplicação só pode ser ativada nos alvos da coluna Alvos Selecionados.

A página Gerenciar Alvos contém as seguintes informações.

Alvos Disponíveis

Os clusters e instâncias nos quais a aplicação não está implantada.

Alvos Selecionados

Os clusters e instâncias nos quais a aplicação está implantada.

Adicionar

Botão para mover o alvo selecionado de Alvos Disponíveis para Alvos Selecionados.

Adicionar Tudo

Botão para mover todos os Alvos Disponíveis para os Alvos Selecionados.

Remover

Botão para mover o alvo selecionado de Alvos Selecionados para Alvos Disponíveis.

Remover Tudo

Botão para mover todos os Alvos Selecionados para Alvos Disponíveis.
