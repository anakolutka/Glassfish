管理対象スレッド・ファクトリの編集

管理対象スレッド・ファクトリ・リソースを編集するには、「管理対象スレッド・ファクトリの編集」ページを使用します。

「管理対象スレッド・ファクトリの編集」ページには、次のオプションが含まれます。

デフォルトのロード

デフォルト値のある設定をデフォルト値に復元するボタン。デフォルト値のない設定は変更されません。

JNDI名

リソースを識別する一意の名前。JNDI名は読取り専用フィールドです。JNDI名を指定できるのは、新規リソースの作成時のみです。

コンテキスト情報

リソースのコンテキスト: 「クラス・ローダー」、「JNDI」、「セキュリティ」、またはワークエリア

ステータス

リソースのステータス。リソースは、有効または無効の状態になります。デフォルト値は「有効」です。

スレッド優先度

管理対象スレッド・ファクトリ・インスタンスに作成するスレッドの優先度。デフォルトは5です。

説明

リソースの説明。

追加プロパティ

リソースの追加プロパティ。GlassFish Serverは管理対象スレッド・ファクトリ・リソースの追加プロパティを定義しません。
