配置 JVM 效能評測器設定

效能評測器工具產生用於分析效能和識別潛在瓶頸的資料。

  1. 在瀏覽樹狀結構中，選取 [Application Server] 節點。
  2. 在 [一般資訊] 頁面上，按一下 [JVM 設定] 標籤。
  3. 在 [JVM 一般設定] 頁面上，按一下 [效能評測器] 標籤。

    [JVM 效能評測器設定] 頁面會隨即開啟。

  4. 指定效能評測器設定。

    您指定的資訊取決於所用之效能評測器產品。如需範例與說明，請參閱「Sun GlassFish Enterprise Server v3 Prelude Developer’s Guide」中的第 3 章「Debugging Applications」。

  5. 按一下 [儲存]。
  6. 重新啟動伺服器。
