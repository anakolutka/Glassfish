Configurer le protocole HTTP

  1. Dans l'arborescence de navigation, développez le nœud Configuration.
  2. Sous le nœud Configuration, sélectionnez le nœud Service HTTP.
  3. Dans la page Service HTTP, cliquez sur l'onglet Protocole HTTP.

    La page Protocole HTTP s'ouvre.

  4. Dans le champ Version, entrez la version du protocole HTTP à utiliser.

    Les valeurs acceptées sont HTTP/1.0 et HTTP/1.1. La valeur par défaut est HTTP/1.1.

  5. Cochez la case Recherche DNS activée pour activer la recherche de l'entrée DNS pour le client.

    Cette option est désactivée par défaut.

  6. Désélectionnez la case à cocher SSL activé pour désactiver la sécurité pour le serveur au niveau global.

    Laissez cette case à cocher activée pour utiliser SSL pour tout module d'écoute dont la sécurité est activée. Cette option est activée par défaut.

  7. Dans le champ Type forcé, entrez le type de réponse à utiliser si aucun mappage MIME correspondant à l'extension n'est disponible.

    La valeur par défaut est text/html; charset=iso-8859-1.

  8. Dans le champ Type par défaut, entrez le type de réponse par défaut.

    La valeur par défaut est text/html; charset=iso-8859-1. La valeur est une chaîne de valeurs séparées par des points-virgules et indiquant le type de contenu, le codage, la langue et le jeu de caractères.

  9. Cliquez sur Enregistrer.

    Astuce – Pour revenir aux paramètres par défaut, cliquez sur le bouton Charger valeurs par défaut, puis sur Enregistrer.
