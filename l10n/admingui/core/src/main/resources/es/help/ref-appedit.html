Editar aplicación

Utilice la página Editar aplicación para modificar una aplicación existente.

La página Editar aplicación contiene las siguientes opciones.

Nombre

Campo de solo lectura que muestra el nombre de la aplicación que está editando.

Servidores virtuales

Servidores virtuales asociados a esta aplicación.

Raíz de contexto

Para una aplicación web, se especifica la ruta de la aplicación. En la URL de la aplicación web, la raíz de contexto aparece justo después del número de puerto (http://host:port /context-root/...). La raíz de contexto debe comenzar por una barra diagonal, por ejemplo, /hello.

Descripción

Breve descripción de la aplicación.

Estado

Si se selecciona la casilla de verificación Habilitado, se habilita la aplicación. Esta opción está seleccionada de forma predeterminada.

Java Web Start

Si se selecciona la casilla de verificación Habilitado, la aplicación empleará el software Java Web Start. Java Web Start proporciona una forma independiente de los navegadores de implementar aplicaciones Java para que se ejecuten en una máquina virtual de Java específica.

Directorio implementado

Si se selecciona esta casilla de verificación, la aplicación se implementa desde un directorio.

Ubicación

Campo de solo lectura que muestra la ubicación de la aplicación implementada.

Bibliotecas

Campo de solo lectura que muestra los archivos JAR de biblioteca que necesita la aplicación.

Submódulos

Tabla que muestra los nombres de los submódulos de la aplicación y sus motores.
