Konfigurieren der JVM

Die JavaTM Virtual Machine (JVMTM) ermöglicht die Ausführung von Java-Byte-Code auf einem physischen Computer. Wenn Sie die Feinabstimmung der JVM-Einstellungen vornehmen, können Sie die Leistung optimieren und Fehler bei der Speicherzuweisung vermeiden.

Das Konfigurieren der JVM umfasst die folgenden Schritte:

  • So konfigurieren Sie die allgemeinen JVM-Einstellungen

  • So konfigurieren Sie die JVM-Pfadeinstellungen

  • So konfigurieren Sie die JVM-Optionen

  • So deaktivieren Sie den Sicherheitsmanager

  • So konfigurieren Sie die JVM-Profilereinstellungen

  • So legen Sie die erweiterte Anwendungskonfiguration fest

  • So legen Sie die erweiterten Domänenattribute fest

Hinweis – Die Begriffe "Java Virtual Machine" und "JVM" stehen für einen virtuellen Computer der Java-Plattform.
