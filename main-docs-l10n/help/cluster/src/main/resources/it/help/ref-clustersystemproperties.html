Proprietà di sistema del cluster

Nella pagina Proprietà di sistema del cluster viene visualizzata la lista delle proprietà di sistema Java impostate per il cluster selezionato. Le proprietà di sistema Java vengono passate al launcher dell'applicazione Java tramite la relativa opzione -D all'avvio di GlassFish Server.

Il campo Nome cluster è un campo di sola lettura in cui viene visualizzato il nome del cluster selezionato.

Per ogni proprietà vengono visualizzate le informazioni riportate di seguito.

Nome variabile istanza

Il nome della proprietà di sistema.

Valore corrente

Il valore impostato al momento per la proprietà. Questo campo è di sola lettura.

Valore di sostituzione

Il valore su cui verrà impostata la proprietà dopo il salvataggio delle modifiche.

La tabella Proprietà aggiuntive contiene inoltre le opzioni riportate di seguito.

Aggiungi proprietà

Pulsante che consente di aggiungere una proprietà. Se si fa clic su questo pulsante, viene aggiunta una riga alla tabella Proprietà aggiuntive.

Elimina proprietà

Pulsante che consente di eliminare una o più proprietà selezionate. L'eliminazione comporta il ripristino del valore predefinito della proprietà. Se il valore predefinito non è stato impostato, la proprietà rimane non definita.
