檢視 HTTP 服務監視資料

  1. 在瀏覽樹狀結構中，選取 [Application Server] 節點。
  2. 在 [一般資訊] 頁面上，按一下 [監視] 標籤。
  3. 在 [Web 容器監視] 頁面上，按一下 [HTTP 服務] 標籤。

    [HTTP 服務監視] 頁面會隨即開啟。

  4. (可選擇) 從 [虛擬伺服器] 下拉式清單中，選取一個伺服器以檢視該伺服器的請求統計。
  5. (可選擇) 從 [HTTP 偵聽程式] 下拉式清單中，選取一個偵聽程式以檢視該偵聽程式的請求統計。
  6. (可選擇) 按一下 [重新整理] 按鈕可更新顯示的監視資料。
