Para Configurar as Opções JVM

Use a página Opções de JVM para especificar as opções do acionador de aplicações Java (java) que executa o GlassFish Server. As opções -D designam propriedades que são específicas ao GlassFish Server.

  1. Na árvore de navegação, selecione o nó Configuração.

  2. Selecione a configuração que está sendo alterada.

  3. No nó Configuração, selecione o nó Definições de JVM.

    A página Definições Gerais de JVM será aberta.

  4. Na página Definições Gerais de JVM, clique na guia Opções de JVM.

    A página Opções de JVM será aberta.

  5. Para modificar uma opção, edite o campo Valor dessa opção.

  6. Para adicionar uma opção:

    1. Clique no botão Adicionar Opção de JVM.

    2. Na linha em branco que for exibida, digite as informações da opção do campo Valor.

      Se as informações da opção contiverem um ou mais espaços, coloque toda a string entre aspas.

  7. Para deletar uma opção:

    1. Marque a caixa de seleção à esquerda do campo Valor da opção a ser deletada.

    2. Clique em Deletar.

  8. Clique em Salvar.

  9. Reinicie o servidor.
