Configuration des utilitaires GlassFish Server que les fournisseurs de sécurité des messages doivent utiliser

  1. Si vous utilisez un jeton de nom utilisateur, configurez une base de données utilisateur, si nécessaire.

    Si vous utilisez un jeton de nom utilisateur/mot de passe, configurez d'abord un domaine et une base de données utilisateur appropriés pour le domaine.

    La configuration d'une base de données utilisateur est abordée dans Modification d'un domaine.

  2. Gérez les certificats et les clés privées, si nécessaire.

Etapes suivantes

Une fois les utilitaires de GlassFish Server configurés en vue d'une utilisation par les fournisseurs de sécurité des messages, les fournisseurs installés avec GlassFish Server peuvent être activés comme décrit dans Modification d'une configuration de sécurité des messages.
