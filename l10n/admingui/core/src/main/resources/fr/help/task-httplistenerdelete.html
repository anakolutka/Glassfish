Supprimer un module d'écoute HTTP

Avant de commencer

Si vous voulez supprimer les listeners HTTP par défaut http-listener-1, http-listener-2 ou admin-listener, créez d'abord une copie de sauvegarde des éléments http-listener du fichier domain.xml de Enterprise Server de façon à pouvoir restaurer les paramètres si nécessaire.

  1. Dans l'arborescence de navigation, développez le nœud Configuration.
  2. Sous le nœud Configuration, développez le nœud Service HTTP.
  3. Sous le nœud Service HTTP, sélectionnez le nœud Modules d'écoute HTTP.
  4. Dans la page Service HTTP, cochez la case située en regard du nom du listener HTTP à supprimer.
  5. Cliquez sur Delete.
