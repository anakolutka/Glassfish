Risorse concorrenti

Utilizzare la pagina Risorse concorrenti per le risorse concorrenti dei tipi riportati di seguito.

  • Servizi contesto

  • Thread factory gestiti

  • Servizi esecutore gestiti

  • Servizi esecutore pianificati gestiti

Le risorse concorrenti sono oggetti gestiti che forniscono funzioni contemporanee alle applicazioni Java EE secondo quanto definito nella specifica JSR 236, Utility contemporanee per Java EE.

In GlassFish Server, configurare le risorse concorrenti e renderle disponibili per l'utilizzo da parte dei componenti delle applicazioni, come servlet ed EJB. Alle risorse concorrenti si accede mediante ricerca JNDI o inserimento risorse.
