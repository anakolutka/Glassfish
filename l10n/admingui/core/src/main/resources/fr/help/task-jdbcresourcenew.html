Créer une ressource JDBC

Une ressource JDBC (Java DataBase Connectivity) (source de données) fournit des applications via une connexion à une base de données.

Avant de commencer

Pour créer une ressource JDBC, vous devez commencer par créer un pool de connexions JDBC. Voir Créer un pool de connexions JDBC.

  1. Dans l'arborescence de navigation, développez le nœud Ressources.
  2. Sélectionnez le nœud Ressources JDBC.
  3. Dans la page Ressources JDBC, cliquez sur Nouveau.

    La page Nouvelle ressource JDBC s'ouvre.

  4. Dans le champ Nom J.N.D.I., saisissez un nom de ressource unique.

    Le nom J.N.D.I. organise et recherche des composants dans un environnement informatique distribué de la même manière que des catalogues sur fiches organisent et représentent des emplacements de livres dans une bibliothèque. Par conséquent, le nom J.N.D.I. devient une méthode importante d'accès à la ressource JDBC. Par convention, le nom commence par la chaîne jdbc/, comme par exemple dans jdbc/payrolldb. N'oubliez pas la barre oblique.

    Le nom ne peut contenir que des caractères alphanumériques, des caractères de soulignement, des tirets ou des points.

  5. Dans la liste déroulante Nom du pool, sélectionnez un pool de connexions à associer à la nouvelle ressource JDBC.
  6. Dans le champ Description, entrez une brève description de la nouvelle ressource JDBC.
  7. Pour désactiver la ressource et la rendre indisponible, désactivez la case à cocher Activé.

    Par défaut, la ressource est disponible (activée) au moment de sa création.

  8. Cliquez sur OK.
