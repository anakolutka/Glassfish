Cómo iniciar una aplicación

Antes de empezar

Asegúrese de que se ha implementado la aplicación que está iniciando.

  1. En el árbol de navegación, seleccione el nodo Aplicaciones.

    Se abrirá la página Aplicaciones

  2. En la tabla Aplicaciones implementadas, haga clic en Iniciar en la fila de la aplicación que esté iniciando.

    La aplicación se abrirá en una nueva ventana del navegador.
