Affichage des composants installés

La page Composants installés contient des informations sur les composants de l'image d'installation déjà installés.

La désinstallation d'un composant implique la suppression des éléments suivants :

  • Tous les fichiers et répertoires qui figuraient dans l'installation initiale du composant

  • Tous les fichiers et répertoires créés par les mises à jour du composant après l'installation initiale

  1. Dans l'arborescence de navigation, sélectionnez le noeud Outil de mise à jour.

    La page Extensions disponibles apparaît.

  2. Sur la page Extensions disponibles, cliquez sur l'onglet Composants installés.

    La page Composants installés apparaît.

Etapes suivantes

Pour désinstaller des composants installés, commencez par arrêter le domaine GlassFish Server. Utilisez ensuite l'une des commandes suivantes :

  • as-install-parent/bin/updatetool, qui démarre l'utilitaire graphique de l'outil de mise à jour

  • as-install-parent/bin/pkg, version de ligne de commande de l'outil de mise à jour

Pour plus d'informations sur l'utilisation de la commande pkg, reportez-vous au chapitre 11, Extending and Updating GlassFish Server, dans le manuel Oracle GlassFish Server Administration Guide. L'utilitaire de l'outil de mise à jour possède sa propre aide en ligne.

Redémarrez le serveur après avoir terminé la désinstallation des composants.
