编辑已部署的 Web 应用程序

  1. 在导航树中，展开“应用程序”节点。
  2. 选择“Web 应用程序”节点。
  3. 从“已部署的 Web 应用程序”表中，单击要编辑的应用程序的名称。

    此时将打开“Web 应用程序”页面。

  4. “在上下文根目录”字段中，键入此 Web 应用程序的路径。

    在 Web 应用程序的 URL 中，上下文根目录要紧跟着端口号 (http://host:port /context-root/...)。确保上下文根路径以正斜杠开头，例如 /hello。

  5. 在“说明”字段中，键入此 Web 应用程序的说明。
  6. 单击“保存”。
