Para eliminar un servidor virtual

  1. En el árbol de navegación, expanda el nodo Configuración.
  2. En el nodo Configuración, expanda el nodo Servicio HTTP.
  3. En el nodo Servicio HTTP, seleccione el nodo Servidores virtuales.
  4. En la página Servidores virtuales, marque la casilla de verificación situada a la izquierda del nombre del servidor virtual que desee eliminar.
  5. Haga clic en Delete.

    Se puede eliminar el servidor virtual __asadmin , pero no se recomienda. Si tiene pensado eliminar este servidor virtual, primero copie los elementos virtual-server del archivo domain.xml de Servidor empresarial en un lugar seguro para poder restaurar la configuración en caso de ser necesario.
