編輯 File 範圍使用者

使用 [編輯 File 範圍使用者] 頁面可修改使用者對目前範圍的存取權限。

範圍名稱

顯示目前選取範圍之名稱的唯讀欄位。

使用者 ID

顯示正在編輯之帳號名稱的唯讀欄位。

群組清單

此使用者所屬的群組。以逗號分隔多個群組名稱。

新密碼

此使用者的新密碼。

確認新密碼

此使用者的新密碼。
