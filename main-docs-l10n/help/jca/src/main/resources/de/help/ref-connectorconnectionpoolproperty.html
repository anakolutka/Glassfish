Connector Connection-Pool bearbeiten - Eigenschaften

Auf der Seite "Connector Connection-Pool bearbeiten - Eigenschaften" können Sie die Eigenschaften für einen Connector Connection-Pool ändern.

Die Seite "Connector Connection-Pool bearbeiten - Eigenschaften" umfasst die folgenden Optionen.

Weitere Eigenschaften

Weitere Eigenschaften für den Connector Connection-Pool. Die verfügbaren Eigenschaften richten sich nach dem Ressourcenadapter, der von dem Connector Connection-Pool verwendet wird. Mit den in dieser Tabelle angegebenen Name/Werte-Paaren können die vom Ressourcenadapterprovider definierten Standardwerte für die Eigenschaften überschrieben werden.

Wenn Sie für eine Eigenschaft keinen Wert angeben, wird die Eigenschaft nach Abschluss der Bearbeitung des Connection-Pools aus der Tabelle entfernt. Falls Sie zum jetzigen Zeitpunkt keinen Wert für eine Eigenschaft festlegen möchten, die Eigenschaft jedoch in der Tabelle verbleiben soll, damit Sie den Wert später angeben können, geben Sie ein leeres Klammernpaar in das Wertefeld ein:

()
