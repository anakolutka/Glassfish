Lancement de client d'application

Utilisez la page Lancement du client d'application pour lancer un client d'application déployé. Il doit s'agir d'un client Web qui peut être exécuté avec Java Web Start.

La page Lancement du client d'application contient les options ci-après.

Application

Nom de l'application contenant le client d'application.

Le nom peut être suivi d'un identificateur de version facultatif dont il est séparé par un signe deux-points (:). Pour plus d'informations sur les versions des modules et des applications, reportez-vous à Module and Application Versions dans le manuel Oracle GlassFish Server Application Deployment Guide.

Module

Nom du module contenant le client d'application.

Liens

URL à laquelle le client est déployé.

Arguments

Arguments à transmettre à l'application.
