일괄 처리 작업 실행을 보려면

  1. 탐색 트리에서 일괄 처리 작업 실행을 보려는 GlassFish Server 클러스터 또는 서버 인스턴스를 선택합니다.

    [일반 정보] 페이지가 열립니다.

  2. [일괄 처리] 탭을 선택합니다.

    [일괄 처리 작업 실행] 페이지가 열립니다.

  3. [일괄 처리 작업 실행] 페이지에서 특정 실행에 대한 세부 정보를 보려면 실행 ID를 누릅니다.

    [일괄 처리 작업 실행 세부 정보] 페이지가 열립니다.

  4. 일괄 처리 작업 실행의 단계에 대한 세부 정보를 보려면 [실행 단계] 탭을 누릅니다.

    [일괄 처리 작업 실행 단계] 페이지가 열립니다.

  5. [일괄 처리 작업 실행] 페이지로 돌아가려면 [뒤로] 단추를 누릅니다.
