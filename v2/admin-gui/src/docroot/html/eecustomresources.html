Using Custom Resources

A custom resource accesses a local JNDI repository and an external resource accesses an external JNDI repository. Both types of resources need user-specified factory class elements, JNDI name attributes, etc.

Within the Application Server, you can create, delete, and list resources, as well as list-jndi-entities.
