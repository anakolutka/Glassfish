Anwendungsüberwachung

Verwenden Sie die Seite Anwendungsüberwachung, um Überwachungsdaten für eine ausgewählte Anwendung anzuzeigen. Eine Beschreibung der überwachten Eigenschaften finden Sie in Kapitel 8, Administering the Monitoring Service in Sun GlassFish Enterprise Server v3 Administration Guide.

Verwandte Aufgaben

  • So konfigurieren Sie den Überwachungsdienst

  • So zeigen Sie Überwachungsdaten der Anwendung an

  • So zeigen Sie die Server-Überwachungsdaten an

Verwandte asadmin-Befehle

  • monitor(1)
