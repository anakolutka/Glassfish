Ver Archivos Log para una Instancia de GlassFish Server en Cluster

Antes de Empezar

La instancia cuyos archivos log desea ver debe estar en ejecución.

  1. En el árbol de navegación, amplíe el nodo Clusters.

  2. En el nodo Clusters ampliado, seleccione el nodo para el cluster.

    Se abre la página Información General para el cluster.

  3. En la página Información General, haga clic en el separador Instancias.

    Se abrirá la página Instancias del Servidor en Cluster para el cluster.

  4. Seleccione la instancia para la que desea ver los archivos log.

    Se abrirá la página Información General para la instancia.

  5. En la página Información General, haga clic en Ver Archivos Log.

    Se abre una ventana con el visor de logs.

    Nota:

    La instancia ya debe estar en ejecución. Si la instancia no está ejecutándose, el botón Ver Archivos Log estará desactivado.
