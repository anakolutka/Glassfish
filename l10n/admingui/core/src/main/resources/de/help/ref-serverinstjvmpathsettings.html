JVM-Klassenpfadeinstellungen

Der Klassenpfad bezeichnet die Liste der JAR-Dateien, die von der Java-Laufzeitumgebung nach Klassen und anderen Ressourcen durchsucht wird.

Die Seite JVM Classpath Settings (JVM-Klassenpfad-Einstellungen) umfasst die folgenden Optionen.

Environment Classpath (Umgebungsklassenpfad)

Wenn Sie das Kontrollkästchen Ignore (Ignorieren) aktivieren, wird die Umgebungsvariable CLASSPATH ignoriert. Die Umgebungsvariable CLASSPATH ist für grundlegende Lernprogramme in der Programmierung geeignet, wird für Unternehmensumgebungen jedoch nicht empfohlen.

Server Classpath (Serverklassenpfad)

Der Enterprise Server Klassenpfad. Dies ist ein schreibgeschütztes Feld.

System Classpath (Systemklassenpfad)

Der Pfad zu den Klassen auf Ihrem Computer.

Classpath Prefix (Klassenpfadpräfix)

Vollständige Pfade zu den JAR-Dateien, die dem Klassenpfad des Servers - jeweils einer pro Zeile - vorangestellt werden müssen.

Classpath Suffix (Klassenpfadsuffix)

Vollständige Pfade zu den JAR-Dateien, die an den Klassenpfad - jeweils einer pro Zeile - angehängt werden müssen.

Native Library Path Prefix (Präfix für Pfad zur nativen Bibliothek)

Vollständige Pfade zu den JAR-Dateien, die dem Pfad zur nativen Bibliothek - jeweils einer pro Zeile - vorangestellt werden müssen.

Native Library Path Suffix (Suffix für Pfad zur nativen Bibliothek)

Vollständige Pfade zu den JAR-Dateien, die an den Pfad zur nativen Bibliothek - jeweils einer pro Zeile - angehängt werden müssen.

Der Pfad zur nativen Bibliothek ist eine Aneinanderreihung der relativen Serverpfade zu den nativen gemeinsam genutzten Bibliotheken, der Standard-Laufzeitumgebung, der Shell-Umgebungseinstellung (LD_LIBRARY_PATH auf UNIX®-Systemen) und allen Pfaden, die auf der Seite JVM-Profiler-Einstellungen angegeben sind.
