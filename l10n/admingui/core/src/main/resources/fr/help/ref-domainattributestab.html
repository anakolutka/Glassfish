Attributs de domaine

Cette page permet de définir les attributs de domaine avancés d'Enterprise Server.

Racine application

Chemin complet du répertoire dans lequel les applications sont déployées.

Racine journal

Identifie l'emplacement de stockage des fichiers journaux du serveur .

Paramètres locaux

Si ce champ n'est pas renseigné, les paramètres locaux par défaut de l'hôte sont utilisés.

Un environnement linguistique est un identifiant qui spécifie une combinaison particulière de langues et de régions. Par exemple, l'environnement linguistique pour l'anglais américain est en_US , pour le japonais ja_JP. Pour utiliser un environnement linguistique non anglophone, serveur d'application doit être localisé, c'est-à-dire adapté de l'anglais en une autre langue.
