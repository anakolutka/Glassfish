To edit a deployed EJB module

To modify details for a deployed application, click the application name from the list of deployed applications. The page with the details of the enterprise application appears. You can modify the following:

  1. On the EJB Module page, change the description.
  2. If you are running a domain with cluster profile, you can enable or disable high-availability.

    If availability is enabled for a module, it must also be enabled at all higher levels (named configuration and web container or EJB container) as well.

  3. To view the list of module descriptors for the application, click the Descriptors tab. For more details, see To view module descriptors of deployed applications
  4. To view the targets on which the application has been enabled, click the Targets tab. You can enable or disable applications on one or multiple targets. See Managing Targets.
