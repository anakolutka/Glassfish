JVM プロファイラ設定

プロファイラツールは、パフォーマンスを分析し、潜在的な障害を特定するための情報を生成します。

プロファイラ設定を指定します。指定する情報は、使用するプロファイラ製品によって異なります。具体例と手順については、『Sun GlassFish Enterprise Server v3 Preview Application Development Guide』の第 4 章「Debugging Applications」を参照してください。

  • 「JVM パスの設定を行う」

  • 「JVM オプションを設定する」

  • 「セキュリティーマネージャーを無効にする」

  • 「JVM のプロファイラ設定を行う」
