并发资源

使用“并发资源”页可配置以下类型的并发资源：

  • 上下文服务

  • 托管线程工厂

  • 托管执行程序服务

  • 托管调度执行程序服务

并行资源是向 Java EE 应用程序提供并行功能的受管对象，如 JSR 236 规范“Java EE 并行实用程序”中所定义。

在 GlassFish Server 中，可以配置并行资源并使它们可供应用程序组件使用，例如 Servlet 和 EJB。并行资源通过 JNDI 查找或资源注入访问。
