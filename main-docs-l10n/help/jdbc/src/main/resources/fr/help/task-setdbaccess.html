Configuration de l'accès aux bases de données

Pour stocker, organiser et extraire les données, la plupart des applications utilisent des bases de données relationnelles.

GlassFish Server inclut une copie de la base de données Java DB et est préconfiguré pour accéder à cette base de données.

  1. Installez un produit de base de données pris en charge.

    Pour obtenir la liste des bases de données prises en charge, reportez-vous à Oracle GlassFish Server Release Notes.

  2. Installez un pilote prenant en charge l'API JDBC (pilote JDBC) pour la base de données.

    Assurez-vous que le pilote que vous souhaitez utiliser pour créer des connexions à la base de données est installé sur tous les serveurs sur lesquels vous souhaitez déployer la source de données.

  3. Assurez-vous que le fichier JAR du pilote est accessible à l'instance de serveur du domaine.

    Reportez-vous à Intégration d'un pilote JDBC.

  4. Créez la base de données.

    Généralement, le fournisseur d'application offre des scripts pour créer et remplir la base de données.

  5. Créez un pool de connexions pour la base de données.

    Reportez-vous à Création d'un pool de connexions JDBC.

  6. Créez une ressource JDBC pointant vers le pool de connexions.

    Reportez-vous à Création d'une ressource JDBC.
