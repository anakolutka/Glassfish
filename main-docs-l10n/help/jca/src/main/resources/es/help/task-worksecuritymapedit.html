Para editar una asignación de seguridad de trabajo

  1. En el árbol de navegación, amplíe el nodo Recursos.

  2. Amplíe el nodo Conectores.

  3. En el nodo Conectores, seleccione el nodo Asignaciones de seguridad de trabajo.

    Se abrirá la página Asignaciones de seguridad de trabajo.

  4. En la página Asignaciones de seguridad de trabajo, haga clic en el nombre de la asignación de seguridad de trabajo que esté editando.

    Se abrirá la página Editar asignación de seguridad de trabajo.

  5. Utilice el campo Grupo o el campo Principal como se indica a continuación.

    • Si activa el botón de radio Grupo, asigne cada grupo de usuarios EIS de backend al grupo de usuarios correspondiente definido en el dominio de seguridad de GlassFish Server.

      Utilice una lista separada por comas para especificar más de una asignación. Por ejemplo:

      eis-group-1=server-group-1,eis-group-2=server-group-2

    • Si activa el botón de radio Principal, asigne cada principal EIS de backend al principal correspondiente definido en el dominio de seguridad de GlassFish Server.

      Utilice una lista separada por comas para especificar más de una asignación. Por ejemplo:

      eis-principal-1=server-principal-1,eis-principal-2=server-principal-2

  6. En la lista desplegable Adaptador de recursos, seleccione el adaptador de recursos para EIS.

  7. En el campo Descripción, introduzca una descripción de la asignación de seguridad de trabajo.

  8. Active la casilla de control Estado activado para activar la asignación de seguridad de trabajo.

    Esta opción se encuentra activada por defecto.

  9. Haga clic en Aceptar.
