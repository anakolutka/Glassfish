デフォルトレルムを設定する

デフォルトレルムとは、アプリケーションの配備記述子がレルムを指定しない場合に、Enterprise Server が認証と承認に使用するレルムです。

  1. ナビゲーションツリーで、「設定」ノードを展開します。
  2. ドメインがクラスタをサポートするように設定されている場合は、変更する設定を選択します。

    たとえば、クラスタ プロファイルや エンタープライズ プロファイルで作成されるドメインなどです。

  3. 「セキュリティー」ノードを選択します。

    「セキュリティ」ページが開きます。

  4. 「デフォルトレルム」フィールドで、ドロップダウンリストから必要なレルムを選択します。
  5. 「保存」をクリックして変更を保存します。
  6. コンソールに「再起動が必要です」と表示される場合は、サーバーを再起動します。
