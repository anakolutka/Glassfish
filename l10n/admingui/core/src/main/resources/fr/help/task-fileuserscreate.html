Ajout d'un utilisateur au domaine de fichiers

  1. Dans l'arborescence de navigation, développez le nœud Configuration.
  2. Développez le nœud Sécurité.
  3. Développez le nœud Domaines.
  4. Sélectionnez le nœud Fichier.

    La page Modification du domaine s'ouvre.

  5. Sur la page Modification du domaine, cliquez sur le bouton Gérer les utilisateurs.

    La page Utilisateurs de fichiers s'ouvre.

  6. Sur la page Utilisateurs de fichiers, cliquez sur Nouveau.

    La page Nouvel utilisateur du domaine fichier s'ouvre.

  7. Dans le champ ID utilisateur, saisissez le nom de l'utilisateur.
  8. Dans le champ Liste des groupes, saisissez une liste des groupes auxquels l'utilisateur appartient.

    Utilisez des virgules pour séparer les noms des groupes. Ces groupes n'ont pas besoin d'être définis ailleurs.

  9. Dans le champ Mot de passe, saisissez un mot de passe unique.
  10. Dans le champ Confirmer le nouveau mot de passe, saisissez à nouveau le mot de passe utilisateur.
  11. Cliquez sur OK.
