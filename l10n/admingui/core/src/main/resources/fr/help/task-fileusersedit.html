Modifier les informations utilisateur

  1. Dans la colonne ID utilisateur du tableau de la page Utilisateurs de fichiers,sélectionnez le nom de l'utilisateur à modifier.

    La page Modification du domaine fichier s'ouvre.

  2. Changez le mot de passe utilisateur en entrant un nouveau mot de passe dans les champs Mot de passe et Confirmation du mot de passe.
  3. Modifiez les groupes auxquels l'utilisateur appartient en ajoutant ou en supprimant des groupes dans le champ Liste des groupes.

    Séparez les noms de groupe par une virgule. Les groupes n'ont pas besoin d'être définis préalablement.

  4. Cliquez sur Enregistrer pour enregistrer les modifications que vous avez apportées à cet utilisateur ou sur Précédent pour quitter la page sans enregistrer.
