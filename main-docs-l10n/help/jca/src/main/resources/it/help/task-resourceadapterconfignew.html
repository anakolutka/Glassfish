Per creare una configurazione Resource Adapter

  1. Espandere il nodo Risorse nell'albero di navigazione.

  2. Nel nodo Risorse selezionare il nodo Configurazioni Resource Adapter.

    Viene visualizzata la pagina Configurazioni Resource Adapter.

  3. Nella pagina Configurazioni Resource Adapter fare clic su Nuovo.

    Viene visualizzata la pagina Nuova configurazione Resource Adapter.

  4. Nell'elenco a discesa Resource Adapter selezionare il Resource Adapter.

  5. Nell'elenco a discesa ID pool di thread selezionare il pool di thread da utilizzare per la configurazione Resource Adapter.

  6. Nella sezione Proprietà aggiuntive specificare le proprietà aggiuntive per il Resource Adapter.

    Le proprietà disponibili sono i nomi dei metodi setter della classe a cui fa riferimento l'elemento resourceadapter-class nel file ra.xml, che definisce il nome della classe del componente JavaBeans del Resource Adapter. Qualsiasi modifica apportata alle proprietà qui definite sostituisce i valori predefiniti presenti nel file ra.xml.

    • Per aggiungere una proprietà, fare clic sul pulsante Aggiungi proprietà. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Nome e il valore della proprietà nel campo Valore.

    • Per modificare una proprietà, è necessario modificare il valore del campo Valore della proprietà.

    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Nome corrispondente alla proprietà da eliminare, quindi fare clic sul pulsante Elimina proprietà.

  7. Fare clic su OK.
