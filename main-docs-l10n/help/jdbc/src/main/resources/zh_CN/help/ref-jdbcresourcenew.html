新建 JDBC 资源

使用“新建 JDBC 资源”页可以创建新的 JDBC 资源。

“新建 JDBC 资源”页包含以下选项。

JNDI 名称

标识资源的唯一 JNDI 名称。按照惯例，名称以资源类型指示符和斜杠开头，例如 concurrent/、jdbc/ 或 jms/。

池名称

与 JDBC 资源关联的连接池。

说明

JDBC 资源的简短说明。

状态

如果启用此选项，则资源将在运行时可用。默认情况下启用此选项。

其他属性

JDBC 资源的其他属性。GlassFish Server 不定义 JDBC 资源的任何其他属性。

目标

(如果域中已创建任何集群或独立实例，则显示此选项。)

资源的集群和独立实例。使用“添加”和“全部添加”按钮将所需目标移动到“所选目标”列。使用“删除”和“全部删除”按钮将任何不需要的目标移动到“可用目标”列。
