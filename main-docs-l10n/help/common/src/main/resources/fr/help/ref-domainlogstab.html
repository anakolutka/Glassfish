Journaux de domaine

La page Journaux de domaine permet de collecter et de télécharger les fichiers journaux pour une instance GlassFish Server ou un cluster donné, et de les compiler dans une seule archive ZIP.

Remarque:

Si vous spécifiez un cluster, l'archive ZIP inclut également le fichier journal pour le serveur d'administration de domaine (DAS).

La page Journaux de domaine contient les options ci-après.

Cluster

Liste déroulante des clusters, s'il en existe.

Instance

Liste déroulante des instances de serveur.

Collecter les journaux

Bouton permettant de collecter les journaux dans un fichier log-files-name.zip, où name est le nom du cluster ou de l'instance.

Le fichier log-files-name.zip est téléchargé à l'aide du mécanisme de téléchargement de votre navigateur.

Tâches connexes

  • Collecte et téléchargement de journaux de domaine

  • Affichage de fichiers journaux pour le serveur DAS

Sous-commandes asadmin équivalentes

collect-log-files(1)
