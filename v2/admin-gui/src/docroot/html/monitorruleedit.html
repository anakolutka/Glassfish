To Edit a Monitor Rule

  1. Do one of the following:
    • For the developer profile, in the tree component, select the Configuration node.
    • For the cluster or enterprise profile, in the tree component select the Configurations node, then select the configuration to edit.
  2. Select the Management Rules node.
  3. Select the management rule to edit.

    The Edit Management Rules page appears.

  4. Check Enabled to activate this rule.
  5. Change the rule description, if needed.
  6. Select the Record Event checkbox to log the occurrence of the specified event.
  7. Change the log level for this management rule, if needed.
  8. Specify the granularity, in milliseconds, at which data is collected for the observed MBean attribute.
  9. Specify the properties for the monitor type.
    • If the type is Count, specify the following:

      • Difference Mode — Select the checkbox to enable the difference mode flag for all observed MBeans.

      • Number Type — Specify the type of the numeric value being monitored. Valid values are byte, double, float, int, long, and short.

      • Init Threshold — Specifies a value above which notification occurs.

      • Offset — Specifies that the event should be re-triggered when the Init Threshold value plus this offset value is reached.

      • Modulus — Specifies the modulus value common to all observed MBeans.

    • If the type is Gague, specify the following:

      • Difference Mode — Select the checkbox to enable the difference mode flag for all observed MBeans.

      • Number Type — Specify the type of the numeric value being monitored. Valid values are byte, double, float, int, long, and short.

      • Threshold — Specify either a lower limit, upper limit, or both, for the range within which notification occurs.

    • If the type is String, specify the following:

    • Trigger — Specify whether the action is triggered when the string matches or differs from the value specified in the next field.

    • Value — Specify the value to which to match the string.

  10. Specify the event description.
  11. Specify the action you want performed when the monitored data meets the condition specified in the event type.
  12. Click Save.
