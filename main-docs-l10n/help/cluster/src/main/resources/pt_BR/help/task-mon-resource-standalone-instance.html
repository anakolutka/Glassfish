Para Exibir Dados para Monitoramento do Recurso da Instância do GlassFish Server Standalone

Antes de Começar

Antes de exibir os dados de monitoramento, é preciso configurar o serviço de monitoramento.

  1. Na árvore de navegação, selecione o nó Instâncias Standalone.

    A página Instância de Servidor Standalone será aberta.

  2. Selecione a instância para a qual você deseja exibir os dados de monitoramento de recursos.

    A página Informações Gerais da instância se abre.

  3. Na página Informações Gerais, clique na guia Monitor.

    A página Monitoramento da Aplicação será aberta.

  4. Clique na guia Recursos.

    A página Monitoramento de Recursos será aberta.

  5. Na lista drop-down Exibir, selecione o recurso cujos dados de monitoramento você deseja exibir.

  6. Para atualizar os dados de monitoramento exibidos, clique em Atualizar.
