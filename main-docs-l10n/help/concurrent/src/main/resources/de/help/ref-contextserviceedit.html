Kontextservice bearbeiten

Auf der Seite "Kontextservice bearbeiten" können Sie eine Kontextserviceressource bearbeiten.

Die Seite "Kontextservice bearbeiten" enthält die folgenden Optionen.

Standardeinstellungen laden

Schaltfläche zur Wiederherstellung der Standardwerte von Einstellungen, die Standardwerte haben. Einstellungen, die keine Standardwerte haben, werden nicht geändert.

JNDI-Name

Ein eindeutiger Name, der die Ressource identifiziert. Der JNDI-Name ist ein schreibgeschütztes Feld. Sie können den JNDI-Namen nur festlegen, wenn Sie eine neue Ressource erstellen.

Kontextinformation

Der Kontext der Ressource: Classloader, JNDI, Security oder WorkArea.

Beschreibung

Eine Beschreibung dieser Ressource.

Status

Der Status der Ressource. Die Ressource kann aktiviert oder deaktiviert sein. Der Standardwert ist "Aktiviert".

Weitere Eigenschaften

Weitere Eigenschaften für die externe Ressource. In GlassFish Server werden keine weiteren Eigenschaften für Kontextserviceressourcen definiert.
