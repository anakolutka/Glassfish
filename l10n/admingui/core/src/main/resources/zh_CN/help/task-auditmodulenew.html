创建审计模块

审计指的是 Enterprise Server 对错误或安全性破坏等重要事件进行记录。Enterprise Server 在日志文件中记录所有验证事件。完整的访问日志提供了 Enterprise Server 访问事件的顺序线索。

Enterprise Server 提供了一个简单的默认审计模块；有关更多信息，请参见使用默认审计模块。

  1. 在导航树中，展开“配置”节点。
  2. 在“配置”节点下，展开“安全性”节点。
  3. 在“安全性”节点下，选择“审计模块”节点。

    此时将打开“审计模块”页面。

  4. 在“审计模块”页面中，单击“新建”。

    此时将打开“新建审计模块”页面。

  5. 在“新建审计模块”页面的“名称”字段中，键入审计模块的名称。
  6. 在“类名”字段中，键入实现此模块的类的全限定名称。

    默认审计模块的类名为 com.sun.enterprise.security.Audit。

  7. （可选的）在“其他属性”部分中，指定其他属性。

    要添加属性，请单击“添加属性”按钮。在显示的空白行中，在“名称”字段中键入属性名称，在“值”字段中键入属性值。

    有关可用于审计模块的其他属性的说明，请参见特定于审计模块的属性。

  8. 单击“确定”。
