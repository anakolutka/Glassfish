Das Fenster "Sicherheitszertifikate übertragen"

Im Fenster "Sicherheitszertifikate übertragen" können Sie Domänen angeben, die einen Satz (oder mehrere Sätze) von Sicherheitszertifikaten enthalten, die zu Sun Java Systems Application Server 9 migriert werden sollen:

  1. Klicken Sie auf die Schaltfläche "Hinzufügen", um das Dialogfenster "Domäne hinzufügen" zu öffnen, in dem der Name der Domäne und die Passwörter der Zertifikatdateien eingegeben werden.

    Im Fenster "Domains with Certificates to be Transferred" ("Domänen mit zu übertragenden Zertifikaten") werden die hinzugefügten Domänen angezeigt.

  2. Wiederholen Sie diesen Vorgang so lange, bis die Domänen, die zu übertragende Zertifikate enthalten, hinzugefügt wurden.
  3. Um eine hinzugefügte Domäne zu löschen, wählen Sie diese aus und klicken Sie auf die Schaltfläche "Domäne löschen".
  4. Klicken Sie anschließend auf "Weiter".

    Das Fenster "Upgrade-Ergebnisse" wird geöffnet und zeigt den Status des Upgrades an.
