Cómo habilitar o deshabilitar el registro de auditoría

  1. En el árbol de navegación, expanda el nodo Configuracións.
  2. Seleccione la instancia que va a configurar:
    • Para configurar una instancia específica, expanda el nodo de configuración de dicha instancia. Por ejemplo, en la instancia predeterminada server expanda el nodo server-config.
    • Para configurar los valores de configuración predeterminados de todas las instancias, expanda el nodo default-config.
  3. En el nodo Configuración, seleccione el nodo Seguridad.

    Se abrirá la página Seguridad.

  4. Marque la casilla de verificación Registro de auditoría activado para habilitar el registro de auditoría.

    Si quiere deshabilitar esta opción, desmarque la casilla de verificación Registro de auditoría acttivado.

    Si selecciona esta opción, se cargarán los módulos de auditoría y se garantiza que serán llamados por la biblioteca de auditoría de Servidor empresarial en los puntos de auditoría.

  5. Si activa el registro de auditoría, especifique un módulo de auditoría predeterminado tal y como se describe en Cómo establecer el módulo de auditoría activo.
  6. Haga clic en "Guardar".
  7. Reinicie Servidor empresarial si aparece el mensaje Reinicio necesario en la consola.
