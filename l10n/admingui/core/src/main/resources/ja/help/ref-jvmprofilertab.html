「JVM プロファイラ設定」ページ

プロファイラツールは、パフォーマンスを分析し、潜在的な障害を特定するための情報を生成します。

プロファイラ設定を指定します。指定する情報は、使用するプロファイラ製品によって異なります。具体例と手順については、Chapter 3, 「Debugging Applications,」 in 『GlassFish v3 Application Server Developer’s Guide 』を参照してください。
