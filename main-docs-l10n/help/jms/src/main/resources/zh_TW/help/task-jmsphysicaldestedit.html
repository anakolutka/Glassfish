編輯 JMS 實體目的地

  1. 在瀏覽樹狀結構中，展開「組態」節點。

  2. 選取要組態的執行處理：

    • 若要組態特定的執行處理，請選取該執行處理的組態節點。例如，對於預設執行處理 server，請選取 server-config 節點。

    • 若要組態使用 default-config 副本之未來執行處理的預設設定，請選取 default-config 節點。

  3. 在「組態」節點下，展開「Java 訊息服務」節點。

  4. 在「Java 訊息服務」節點下，選取「實體目的地」節點。

    將開啟「實體目的地」頁面。

  5. 在「實體目的地」頁面上，按一下您要編輯的實體目的地名稱。

    將開啟「編輯實體目的地」頁面。
