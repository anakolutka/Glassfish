Para configurar as definições de caminho de JVM

O caminho de classe é a lista de arquivos JAR que o ambiente de tempo de execução JAVA pesquisa para detectar classes e outros recursos.

  1. Na árvore de navegação, selecione o nó Configuração.
  2. Sob o nó Configuração, selecione o nó Configurações de JVM.

    A página Configurações Gerais de JVM é exibida.

  3. Na página Configurações Gerais de JVM, clique na guia Configurações de Caminho.

    A página Configurações de Caminho de Classe de JVM é exibida.

  4. Marque a caixa de seleção Ignorar Caminho de Classe de Ambiente para ignorar a variável de ambiente CAMINHO DE CLASSE.

    A variável de ambiente CAMINHO DE CLASSE é conveniente para tutoriais básicos de programação, mas não é recomendada para ambientes empresariais.

  5. No campo Caminho de Classe do Sistema, digite o caminho para as classes no seu sistema.
  6. No campo Prefixo de Caminho de Classe, digite os caminhos completos dos arquivos JAR a serem colocados no início do caminho de classe do servidor.

    Para colocar vários arquivos no início, digite um caminho por linha.

  7. No campo Sufixo de Caminho de Classe, digite os caminhos completos dos arquivos JAR a serem anexados ao caminho de classe do servidor.

    Por exemplo, especifique o arquivo JAR de um driver da banco de dados. Consulte To Integrate a JDBC Driver.

    Para anexar vários arquivos, digite um caminho por linha.

  8. No campo Prefixo de Caminho de Classe da Biblioteca Nativa, digite os caminhos completos dos arquivos JAR a serem colocados no início do caminho da biblioteca nativa.

    Para colocar vários arquivos no início, digite um caminho por linha.

  9. No campo Sufixo de Caminho de Classe da Biblioteca Nativa, digite os caminhos completos dos arquivos JAR a serem anexados ao caminho da biblioteca nativa.

    Para anexar vários arquivos, digite um caminho por linha.

    O caminho da biblioteca nativa é uma concatenação dos seguintes caminhos:

    • O caminho relativo do servidor para as respectivos bibliotecas nativas compartilhadas

    • O caminho da biblioteca nativa do ambiente de tempo de execução padrão

    • A configuração de ambiente do shell (LD_LIBRARY_PATH nos sistemas UNIX®)

    • Todos os caminhos especificados na página Configurações do Gerador de Perfil de JVM

  10. Clique em Salvar.
  11. Reinicie o servidor.
