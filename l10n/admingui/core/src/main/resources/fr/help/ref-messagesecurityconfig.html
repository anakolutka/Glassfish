Configuration de la sécurité des messages

La page Configuration de la sécurité des messages répertorie les configurations disponibles pour les fournisseurs de sécurité des messages.

Pour configurer la sécurité des messages, vous devez comprendre les bases des concepts de sécurité et des services Web. Pour en savoir plus, consultez le Chapitre 13, Administering Message Security du Sun GlassFish Enterprise Server v3 Administration Guide et le Chapitre 6, Administering Web Applications du Sun GlassFish Enterprise Server v3 Administration Guide.

Le Enterprise Server implémente la sécurité des messages via les fournisseurs de sécurité des messages intégrés à sa couche de traitement SOAP. Les fournisseurs de sécurité des messages dépendent des autres utilitaires de sécurité du Enterprise Server.

Pour chaque configuration, la page Configuration de la sécurité des messages contient les informations suivantes.

Couche d'authentification

Couche message à laquelle l'authentification est effectuée. Par défaut, une configuration SOAP est fournie. Il est également possible de fournir une configuration HttpServlet.

Fournisseur par défaut

Fournisseur de serveur par défaut pour cette configuration. Par défaut, aucun fournisseur de serveur par défaut n'est configuré.

Fournisseur client par défaut

Fournisseur client par défaut pour cette configuration. Par défaut, aucun fournisseur client par défaut n'est configuré.
