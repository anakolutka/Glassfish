删除 JMS 主机

建议不要删除默认 JMS 主机。

  1. 在导航树中，展开“配置”节点。

  2. 选择要配置的实例：

    • 要配置特定的实例，请选择该实例的配置节点。例如，对于默认实例 server，请选择 server-config 节点。

    • 要为以后使用 default-config 副本的实例配置默认设置，请选择 default-config 节点。

  3. 在“配置”节点下，展开“Java 消息服务”节点。

  4. 在“Java 消息服务”节点下，选择“JMS 主机”节点。

    此时将打开“JMS 主机”页。

  5. 在“JMS 主机”页上，选中您要删除的主机名称旁边的复选框。

    注意:

    请勿删除默认的 JMS 主机。这样做可能会导致意外错误。

  6. 单击“删除”。
