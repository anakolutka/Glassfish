So erstellen Sie einen Bereich

Enterprise Server verfügt über drei vorkonfigurierte Bereiche: file, certificate und admin-realm. Darüber hinaus können die Bereiche ldap, JDBC, solaris sowie benutzerdefinierte Bereiche erstellt werden. Im Allgemeinen ist auf jedem Server ein Bereichstyp vorhanden, Enterprise Server verfügt jedoch über zwei Dateibereiche: file and admin-realm. Dies sind zwei Bereiche desselben Typs, die für unterschiedliche Zwecke verwendet werden. Darüber hinaus kann für jeden virtuellen Server auf dem System eine separate Zertifikatdatenbank bereitgestellt werden.

  1. Erweitern Sie in der Navigationsstruktur den Konfigurationsknoten.
  2. Wenn die Domäne für die Unterstützung von Clustern konfiguriert ist, wählen Sie die zu ändernde Konfiguration aus.

    Beispiele sind Domänen, die mit dem Profil Cluster oder 'Unternehmen' erstellt werden.

  3. Erweitern Sie den Knoten "Sicherheit".
  4. Wählen Sie den Knoten "Bereiche".
  5. Klicken Sie auf der Seite "Bereiche" auf "Neu".

    Die Seite zur Erstellung neuer Bereiche wird angezeigt.

  6. Geben Sie im Feld "Name" einen Namen für den Bereich ein.
  7. Geben Sie den Klassennamen für den zu erstellenden Bereich an.

    Zulässige Optionen sind in der folgenden Tabelle aufgeführt.

    Bereichsname

    Klassenname

    file

    com.sun.enterprise.security.auth.realm.file.FileRealm

    certificate

    com.sun.enterprise.security.auth.realm.certificate.CertificateRealm

    ldap

    com.sun.enterprise.security.auth.realm.ldap.LDAPRealm

    JDBC

    com.sun.enterprise.security.auth.realm.jdbc.JDBCRealm

    solaris

    com.sun.enterprise.security.auth.realm.solaris.SolarisRealm

    custom

    Name der Anmeldebereichsklasse
  8. Fügen Sie gegebenenfalls weitere Eigenschaften für den Bereich hinzu.
    1. Klicken Sie auf "Eigenschaft hinzufügen".
    2. Geben Sie im Feld "Name" den Namen der Eigenschaft ein.

      • Eine Beschreibung der Eigenschaften "Dateibereich" finden Sie unter Die für die FileRealm-Klasse spezifischen Eigenschaften.

      • Eine Beschreibung der Eigenschaften "Zertifikatsbereich" finden Sie unter Für die CertificateRealm-Klasse spezifische Eigenschaften.

    3. Geben Sie im Feld "Wert" den Wert der Eigenschaft ein.
  9. Klicken Sie auf „OK“.
