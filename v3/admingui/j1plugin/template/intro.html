[Write your abstract here. An example follows.] JavaServer Faces (JSF) is a standard-based server side user interface component framework for Java technology based Web applications. In this hands-on lab, your will learn basic concept, architecture and features of JavaServer Faces by building, deploying, and running a set of sample Web applications. You will also learn the usage of basic UI components through JavaServer Faces custom tags, specify page selection through JavaServer Faces navigation model, perform input validation, handle event handling, and so on.

Notations used in this documentation

  • <J2EE14_HOME> or <INSTALL_DIR>
    • It is the installation directory of J2EE 1.4 SDK
      • For Windows, the default is C:\Sun\AppServer
      • For Solaris/Linux, the default is /opt/SUNWappserver
  • <lab_root> - directory into which lab zip file is unzipped
    • This document uses <lab_root> to denote the directory under which you have unzipped the lab zip file of this handson lab. The name of the lab zip file of this handson lab is 7622_netbeans40.zip.
    • Once you unzipped the lab zip file under <lab_root>, it will create a subdirectory called netbeans40. For example, under Windows, if you have unzipped the lab zip file in the root of drive E:\, it will create E:\netbeans40. Under Linux/Solaris, if you have unzipped the lab zip file in the /home/sang directory, it will create $HOME/netbeans40 directory.
