Verwaltete geplante ExecutorServices

Auf der Seite "Verwaltete geplante ExecutorServices" können Sie verwaltete geplante ExecutorService-Ressourcen konfigurieren.

Für jede geplante verwaltete ExecutorService-Ressource werden die folgenden Informationen zur Verfügung gestellt.

JNDI-Name

Ein eindeutiger Name, der die Ressource identifiziert.

Aktiviert

"True", wenn die Ressource aktiviert ist. "False", wenn sie deaktiviert ist.

Kontextinformation

Der Kontext der Ressource: Classloader, JNDI, Security oder WorkArea.

Threadpriorität

Die Priorität der Threads, die für die verwaltete geplante ExecutorService-Instanz erstellt werden müssen. Der Standardwert ist 5.

Beschreibung

Eine Beschreibung der Ressource.

Die Tabelle "Verwaltete geplante ExecutorServices" enthält darüber hinaus die folgenden Optionen.

Neu

Schaltfläche zum Erstellen einer neuen Ressource.

Löschen

Schaltfläche zum Löschen von mindestens einer ausgewählten Ressource.

Aktivieren

Schaltfläche zum Aktivieren von mindestens einer ausgewählten Ressource.

Deaktivieren

Schaltfläche zum Deaktivieren von mindestens einer ausgewählten Ressource.
