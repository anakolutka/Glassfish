管理目標

使用「管理目標」頁面可變更能在其上啟用應用程式的目標叢集與獨立伺服器執行處理。您只能在「選取的目標」欄中，啟用目標上的應用程式。

「管理目標」頁面包含以下資訊。

可用的目標

未建置應用程式的叢集與執行處理。

選取的目標

已建置應用程式的叢集與執行處理。

增加

可將選取的目標從「可用的目標」移至「選取的目標」的按鈕。

Add All

可將所有「可用的目標」移至「選取的目標」的按鈕。

移除

可將選取的目標從「選取的目標」移至「可用的目標」的按鈕。

全部移除

可將所有「選取的目標」移至「可用的目標」的按鈕。
