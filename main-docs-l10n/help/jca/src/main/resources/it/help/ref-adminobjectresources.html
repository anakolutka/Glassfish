Risorse oggetto amministrato

Utilizzare la pagina Risorse oggetto amministrato per configurare le risorse oggetto amministrato.

Una risorsa oggetto amministrato fornisce una funzionalità specializzata definita dal Resource Adapter per il modulo connettore distribuito.

Per ogni risorsa oggetto amministrato vengono fornite le informazioni riportate di seguito.

Nome JNDI

Il nome univoco che identifica la risorsa oggetto amministrato.

Abilitato

L'impostazione è True se la risorsa oggetto amministrato è abilitata oppure è False se la risorsa oggetto amministrato è disabilitata.

Tipo di risorsa

Il tipo Java della risorsa oggetto amministrato.

Descrizione

La descrizione della risorsa oggetto amministrato.

La tabella Risorse contiene inoltre le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare una nuova risorsa oggetto amministrato.

Elimina

Pulsante che consente di eliminare una o più risorse oggetto amministrato selezionate.

Abilita

Pulsante che consente di abilitare una o più risorse oggetto amministrato selezionate.

Disabilita

Pulsante che consente di disabilitare una o più risorse oggetto amministrato selezionate.
