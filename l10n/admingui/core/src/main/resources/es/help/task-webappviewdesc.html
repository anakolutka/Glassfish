Cómo ver los descriptores de una aplicación web

  1. En el árbol de navegación, expanda el nodo Aplicaciones.
  2. En el nodo Aplicaciones, seleccione el nodo Aplicaciones web.

    Se abrirá la página Aplicaciones web.

  3. En la página Aplicaciones web, haga clic en el nombre de la aplicación web cuyos descriptores desea ver.

    Se abrirá la página Aplicación web.

  4. En la página Aplicación web, haga clic en la ficha Descriptor.

    Aparecerá la página Descriptor de implementación personalizado.

  5. En el campo Descriptores de implementación, haga clic en el nombre del archivo XML que desea ver.

    El archivo XML se abrirá en una página nueva.

  6. Examine el archivo y haga clic en Cerrar para volver a la página Descriptor de implementación personalizado.
