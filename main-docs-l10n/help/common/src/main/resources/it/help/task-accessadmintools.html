Per concedere l'accesso agli strumenti di amministrazione

Solo gli utenti del gruppo asadmin possono accedere a Console di amministrazione e alla utility della riga di comando asadmin. Per garantire agli utenti l'accesso a questi strumenti di amministrazione, aggiungerli al gruppo asadmin nel realm admin-realm.

  1. Nel nodo Configurazioni espandere la configurazione dell'istanza server o del cluster per cui si desidera configurare i privilegi di accesso di amministrazione.

  2. Espandere il nodo Sicurezza.

  3. Espandere il nodo Realm.

  4. Selezionare il nodo admin-realm.

    Viene visualizzata la pagina Modifica realm.

  5. Nella pagina Modifica realm fare clic sul pulsante Gestisci utenti.

    Viene visualizzata la pagina Utenti amministratori.

    Subito dopo l'installazione, il nome utente e la password di amministratore immessi durante l'installazione vengono elencati in un file denominato admin-keyfile. Per impostazione predefinita questo utente fa parte del gruppo asadmin, che conferisce i diritti per la modifica di GlassFish Server. Assegnare gli utenti a questo gruppo solo se si desidera concedere loro privilegi di amministratore per l'istanza o il cluster di GlassFish Server selezionato.

    Se si aggiungono utenti al realm admin-realm assegnandoli a un gruppo diverso da asadmin, le informazioni sugli utenti verranno comunque salvate nel file admin-keyfile, ma gli utenti non avranno accesso agli strumenti o alle applicazioni di amministrazione nel realm file.

  6. Nella pagina Utenti amministratori fare clic su Nuovo per aggiungere un nuovo utente al realm admin-realm.

    Viene visualizzata la pagina Nuovo utente realm.

  7. Nel campo ID utente digitare il nome del nuovo utente.

  8. Nel campo Lista gruppi digitare i nomi dei gruppi di appartenenza dell'utente.

    Per autorizzare un utente ad apportare modifiche a GlassFish Server, includere asadmin in Lista gruppi.

  9. Nei campi Nuova password e Conferma nuova password digitare la password per il nuovo utente.

  10. Fare clic su OK per aggiungere questo utente al realm admin-realm oppure su Annulla per uscire senza salvare le modifiche.
