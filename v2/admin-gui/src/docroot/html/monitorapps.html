To Monitor Applications

  1. Do one of the following:
    • For the developer profile, in the tree component, select the Application Server node. Click the Monitor tab.
    • For the cluster or enterprise profile In the tree component, select Common Tasks, then click View Monitoring Data.
  2. Click the Applications tab.
  3. Select the application to monitor from the Application list.

    The Application list shows all applications deployed to the Application Server.

  4. Select the specific component from the Component list.

    The page displays monitoring data for the selected application and component. For a description of properties that can be monitored, see Chapter 18, Monitoring Components and Services, in Sun Java System Application Server 9.1 Administration Guide.

    If the page does not display monitoring data for the application or component, click Configure Monitoring. To monitor applications, enable monitoring for the container in which they execute. For example, for web applications enable monitoring for the Web Container; for EJBs enable monitoring for the EJB Container.

    If no monitoring data is displayed for applications, it is most likely that the application does not exist or is not being used. Applications monitoring data is available only when the application exists, and the application is being used. Once the application is executed, it is registered in the monitoring registry and the monitoring data displays.
