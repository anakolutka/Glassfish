새 JDBC 자원

J.N.D.I. 이름

J.N.D.I 이름은 카드 카탈로그가 도서관 내 책의 위치를 조직화하여 나타내는 것과 유사한 방식으로 분산 컴퓨팅 환경 내에서 구성 요소를 조직화하고 구성 요소의 위치를 찾아냅니다. 이에 따라 J.N.D.I 이름은 JDBC 자원에 액세스하는 중요한 방법이 됩니다. 일반적으로 이름은 jdbc/ 문자열로 시작합니다(예: jdbc/payrolldb). 슬래시를 반드시 입력합니다.

풀 이름

JDBC 자원과 연결된 연결 풀입니다.

설명

JDBC 자원에 대한 간단한 설명입니다.

상태

JDBC 자원의 상태입니다. 자원은 사용 가능하거나 사용 가능하지 않을 수 있습니다.
