Serveurs virtuels

Un serveur virtuel, parfois appelé hôte virtuel, est un objet qui permet au même serveur physique d'héberger plusieurs noms de domaines Internet. Tous les serveurs virtuels hébergés sur le même serveur physique partagent la même adresse IP (Internet Protocol) de ce serveur. Un serveur virtuel associe le nom de domaine d'un serveur (tel que www.sun.com) au serveur sur lequel Enterprise Server est exécuté.
