Annulation du déploiement d'applications d'une instance GlassFish Server autonome

L'annulation du déploiement d'une application enlève l'application de toutes les cibles et du domaine. Lorsque le déploiement d'une application est annulé, l'application n'est plus répertoriée sur la page Applications qui affiche toutes les applications déployées dans GlassFish Server.

  1. Dans l'arborescence de navigation, sélectionnez le noeud Instances autonomes.

    La page Instances de serveur autonomes apparaît.

  2. Sur la page Instances de serveur autonomes, sélectionnez l'instance.

    La page Informations générales de l'instance apparaît.

  3. Sur la page Informations générales de l'instance, cliquez sur l'onglet Applications.

    La page Applications de l'instance apparaît.

  4. Pour chaque application dont vous souhaitez annuler le déploiement, cochez la case en regard de l'application dans la table Applications déployées.

  5. Cliquez sur Annuler le déploiement.

    Vous êtes invité à confirmer que vous souhaitez annuler le déploiement des applications.

  6. Confirmez l'annulation du déploiement des applications.

    Le déploiement des applications est annulé et les applications sont enlevées de la table des applications déployées sur la page.
