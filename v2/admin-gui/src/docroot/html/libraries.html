Shared Libraries

A Shared Library provides Java classes that are not private to a single component and is typically shared by more than one JBI Component. For example, the Java EE Service Engine requires the WSDL Shared Library.

From the Admin Console, you can:

  • View all Shared Libraries. For steps to view all Shared Libraries, see To View Shared Libraries.

  • Install Shared Libraries. For steps to install Shared Libraries, see To Install a JBI Shared Library.

  • View General Properties of a Shared Library. For steps to view General Properties of a Shared Library, see Viewing General Properties of a Shared Library.

  • View the descriptor for a Shared Library. For steps to view the descriptor for a Shared Library, see Viewing JBI Descriptors.

  • Uninstall a Shared Library. For steps to uninstall a Shared Library, see To Uninstall a JBI Shared Library.
