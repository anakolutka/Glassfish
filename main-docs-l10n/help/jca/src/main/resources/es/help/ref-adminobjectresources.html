Recursos de Objeto Administrado

Utilice la página Recursos de objeto administrado para configurar los recursos de objeto administrado.

Un recurso de objeto administrado proporciona una funcionalidad especializadas que se define mediante el adaptador de recursos del módulo de conector desplegado.

Para cada recurso de objeto administrado, se proporciona la siguiente información.

Nombre JNDI

Nombre único que identifica el recurso de objeto administrado.

Activado

Verdadero si el recurso de objeto administrado está activado, o falso si el recurso está desactivado.

Tipo de recurso

Tipo Java del recurso de objeto administrado.

Descripción

Descripción del recurso de objeto administrado.

La tabla Recursos también contiene las siguientes opciones.

Nuevo

Botón para crear un nuevo recurso de objeto administrado.

Suprimir

Botón para suprimir uno o más recursos de objeto administrado seleccionados.

Activar

Botón para activar uno o más recursos de objeto administrado seleccionados.

Desactivar

Botón para desactivar uno o más recursos de objeto administrado seleccionados.
