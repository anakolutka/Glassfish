So zeigen Sie Beschreibungen für eine Webanwendung an

  1. Erweitern Sie in der Navigationsstruktur den Anwendungsknoten.
  2. Wählen Sie unter dem Anwendungsknoten den Webanwendungsknoten aus.

    Die Seite "Webanwendungen" wird geöffnet.

  3. Klicken Sie auf der Seite "Webanwendungen" auf den Namen der Webanwendung, deren Beschreibung Sie anzeigen.

    Die Webanwendung wird geöffnet.

  4. Klicken Sie auf der Seite "Webanwendung" auf die Registerkarte "Beschreibung".

    Die Seite "Bereitstellungsbeschreibung anpassen" wird geöffnet.

  5. Klicken Sie im Feld "Bereitstellungsbeschreibungen" auf den Namen der XML-Datei, die Sie anzeigen.

    Die XML-Datei wird in einer neuen Seite geöffnet.

  6. Blättern Sie durch die Datei, und klicken Sie auf "Schließen", um zur Seite "Bereitstellungsbeschreibung anpassen" zurückzukehren.
