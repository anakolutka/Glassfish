Services exécuteurs planifiés gérés

Utilisez la page Services exécuteurs planifiés gérés pour configurer les ressources de service exécuteur planifié géré.

Pour chaque ressource de service exécuteur planifié géré, les informations suivantes sont fournies.

Nom JNDI

Nom unique identifiant la ressource.

Activé

True si la ressource est activée ou False si la ressource est désactivée.

Informations de contexte

Contexte de la ressource : chargeur de classe; JNDI, sécurité ou espace de travail.

Priorité des threads

Priorité des threads à créer pour l'instance de service exécuteur planifié géré. La valeur par défaut est 5.

Description

Description de la ressource.

La table Services exécuteurs planifiés gérés contient également les options suivantes :

Nouveau

Bouton permettant de créer une ressource.

Supprimer

Bouton permettant de supprimer les ressources sélectionnées.

Activer

Bouton permettant d'activer les ressources sélectionnées.

Désactiver

Bouton permettant de désactiver les ressources sélectionnées.

Commandes asadmin connexes

  • create-XXX(1)

  • list-XXX(1)

  • delete-XXX(1)
