Para suprimir una asignación de seguridad para un pool de conexiones de conector

  1. En el árbol de navegación, amplíe el nodo Recursos.

  2. Amplíe el nodo Conectores.

  3. En el nodo Configuración, seleccione el nodo Pools de conexiones de conector.

    Se abrirá la página Pools de conexiones de conectores.

  4. En la página Pools de conexiones de conectores, haga clic en el nombre del pool de conexiones que tiene la asignación de seguridad que está suprimiendo.

    Se abrirá la página Editar pool de conexiones del conector.

  5. Haga clic en el separador Asignaciones de seguridad.

    Se abrirá la página Asignaciones de seguridad de pool de conexiones de conector.

  6. En la página Asignaciones de seguridad de pools de conexiones del conector, active la casilla de control junto al nombre de la asignación de seguridad que va a suprimir.

  7. Haga clic en Suprimir.
