Para Remover Mensagens de um Destino Físico JMS

  1. Na árvore de navegação, expanda o nó Configuração.

  2. No nó Configuração, expanda o nó do Serviço JMS.

  3. No nó Serviço JMS, selecione o nó Destinos Físicos.

    A página Destinos Físicos é aberta.

  4. Na página Destinos Físicos, marque a caixa de seleção ao lado do nome do destino do qual as mensagens devem ser removidas.

  5. Clique no botão Deletar Todas as Mensagens.
