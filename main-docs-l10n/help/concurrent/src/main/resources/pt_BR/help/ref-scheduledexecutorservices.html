Serviços do Executor Programado Gerenciado

Use a página Serviços do Executor Programado Gerenciado para configurar os recursos de serviço do executor gerenciado programado.

Para cada recurso de serviço do executor gerenciado, as informações a seguir são fornecidas.

Nome JNDI

Um nome exclusivo que identifica o recurso.

Ativado

Verdadeiro se o recurso estiver ativado, ou falso se o recurso estiver desativado.

Informações de Contexto

O contexto do recurso: Classloader, JNDI, Segurança ou Área de Trabalho.

Prioridade do Thread

A prioridade de threads a ser criada para a instância de serviço do executor programado gerenciado. O default é 5 .

Descrição

Uma descrição do recurso.

A tabela Serviços do Executor Programado Gerenciado também contém as seguintes opções.

Novo

Botão utilizado para criar um novo recurso.

Deletar

Botão utilizado para deletar um ou mais recursos selecionados.

Ativar

Botão utilizado para ativar um ou mais recursos selecionados.

Desativar

Botão utilizado para desativar um ou mais recursos selecionados.
