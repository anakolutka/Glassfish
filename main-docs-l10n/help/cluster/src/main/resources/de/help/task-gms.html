So wird der Group Management Service konfiguriert

Der Group Management Service (GMS) ist ein prozessinterner Service, der Services für Clusterüberwachung, Clustermitgliedschaft und Gruppenkommunikation bereitstellt. Der GMS führt folgende Funktionen aus:

  • Benachrichtigt registrierte Module in einer GlassFish Server-Instanz, wenn mindestens eine Mitgliedsinstanz zum Cluster hinzugefügt bzw. aus dem Cluster entfernt wurde oder bei einer Mitgliedsinstanz Fehler vermutet bzw. bestätigt wurden.

  • Bietet die Möglichkeit, Nachrichten zwischen Cluster-Membern zu senden und zu empfangen.

  1. Blenden Sie in der Navigationsbaumstruktur den Knoten "Konfiguration" ein.

  2. Wählen Sie die Konfiguration für das zu ändernde Cluster.

  3. Wählen Sie unter dem Konfigurationsknoten den Group Management Service-Knoten.

    Die Seite "Group Management Service" wird geöffnet.

  4. Geben Sie im Feld "Maximale Anzahl nicht ausgelöster Heartbeats" die maximale Anzahl nicht ausgelöster Heartbeats an, bevor der GMS bestätigt, dass ein Fehler vermutet wird.

    Der Standardwert ist 3.

  5. Geben Sie im Feld "Heartbeat-Häufigkeit" den Zeitraum an, der zwischen aufeinanderfolgenden Übertragungen von Heartbeats eines Gruppenmitglieds liegen soll.

    Der Standardwert ist 2000 Millisekunden.

  6. Geben Sie im Feld "Timeout für Gruppenerkennung" den Zeitraum an, den der GMS wartet, bis die Erkennung anderer Cluster-Member fortgesetzt wird.

    Der Standardwert ist 5000 Millisekunden.

  7. Geben Sie im Feld "Wartezeit für Fehlerüberprüfung" die Wartezeit an, bevor geprüft wird, ob ein Mitglied, bei dem ein Fehler vermutet wird, als Fehler gemeldet werden soll.

    Der Standardwert ist 1500 Millisekunden.

  8. Klicken Sie zum Wiederherstellen der Standardwerte auf die Schaltfläche "Standardeinstellungen laden".

  9. Geben Sie im Abschnitt "Weitere Eigenschaften" weitere Eigenschaften an.

    • Um eine Eigenschaft hinzuzufügen, klicken Sie auf die Schaltfläche "Eigenschaft hinzufügen". Geben Sie in die daraufhin angezeigte leere Zeile den Eigenschaftsnamen in das Feld "Name" und den Eigenschaftswert in das Feld "Wert" ein.

    • Um eine Eigenschaft zu ändern, bearbeiten Sie das Feld "Wert" dieser Eigenschaft.

    • Um eine Eigenschaft zu löschen, aktivieren Sie das Kontrollkästchen links neben dem Feld "Name" der zu löschenden Eigenschaft und klicken anschließend auf die Schaltfläche "Eigenschaften löschen".

  10. Klicken Sie auf "Speichern".
