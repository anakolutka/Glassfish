Network Security Services (NSS) für Benutzerverwaltung

Sie können Benutzer entweder über die Administration Console (wie unter Seite "Dateibenutzer" beschrieben) oder mithilfe der NSS-Tools verwalten. NSS ist ein Bibliothekensatz, der die plattformübergreifende Entwicklung von Client- und Serveranwendungen mit Sicherheitsfunktionen unterstützt. Mit NSS erstellt Anwendungen können SSL v2- und v3-, TLS-, PKCS #5-, PKCS #7-, PKCS #11-, PKCS #12-, S/MIME-, X.509 v3-Zertifikate sowie andere Sicherheitsstandards unterstützen. Detaillierte Informationen erhalten Sie über die folgenden Links:

  • Network Security Services (NSS) unter http://www.mozilla.org/projects/security/pki/nss/

  • NSS-Sicherheitstools unter http://www.mozilla.org/projects/security/pki/nss/tools/

  • Überblick über NSS unter http://www.mozilla.org/projects/security/pki/nss/overview.html
