Nuovo connection pool JDBC (passo 1 di 2)

Utilizzare la pagina Nuovo connection pool JDBC (passo 1 di 2) per creare un nuovo connection pool JDBC.

La pagina Nuovo connection pool JDBC (passo 1 di 2) contiene le opzioni riportate di seguito.

Nome pool

Il nome del connection pool.

Tipo di risorsa

I tipi di risorse disponibili sono javax.sql.XADataSource (transazioni globali), java.sql.ConnectionPoolDataSource (transazioni locali, possibili miglioramenti delle prestazioni), javax.sql.DataSource (solo transazioni locali) e java.sql.Driver.

Fornitore driver database

Il nome del fornitore del prodotto database in uso.

Analisi

Se questa opzione è abilitata, viene eseguita l'introspezione del driver del database in base al tipo di risorsa e al fornitore del driver del database. I nomi rilevati per origini dati o classi di driver vengono visualizzati nella pagina Nuovo connection pool JDBC (passo 2 di 2) quando si fa clic sul pulsante Successivo.

Affinché l'introspezione riesca, il driver JDBC deve essere disponibile per GlassFish Server. Vedere Per integrare un driver JDBC.
