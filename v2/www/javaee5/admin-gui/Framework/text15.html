  • HandlerContext:

  • public View getView();

  • public EventObject getEvent();

  • public UseHandlerDescriptor getUseHandlerDescriptor();

  • public HandlerDescriptor getHandlerDescriptor();

  • public Object getInputValue(String name);

  • public void setOutputValue(String name, Object value);

  • public Object getOutputValue(String name);


Notes:

The HandlerContext is the interface to all the framework related information that may be needed while processing the handler.

Most importantly, it contains the getInputValue / setOutputValue methods. Both of these methods perform type conversion based on the declared type (in XML).

UseHandlerDescriptor = Call descriptor

HandlerDescriptor = Function descriptor
