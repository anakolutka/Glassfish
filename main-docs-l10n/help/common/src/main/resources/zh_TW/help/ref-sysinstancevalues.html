執行處理值

使用「執行處理值」頁面可為參照所選取已命名組態之 GlassFish Server 執行處理編輯組態系統特性值。

「執行處理值」頁面會顯示參照所選取組態之執行處理的清單，其中已設定特性。

執行處理名稱

建立執行處理時指定給執行處理的名稱。按一下此名稱，隨即會開啟執行處理的「一般資訊」頁面。

叢集名稱

執行處理所屬的叢集 (如果存在) 名稱。本欄位是唯讀的。

預設值

為執行處理所參照之已命名組態中的特性所設定的值。本欄位是唯讀的。

置換值

為所選取執行處理所設定的特性值。此值會置換預設值。
