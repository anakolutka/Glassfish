Procesamiento de solicitud

La página Procesamiento de solicitud contiene las opciones siguientes:

Total de subprocesos

Número máximo de subprocesos de procesamiento de solicitud. El valor predeterminado es 20.

Total inicial de subprocesos

Número de subprocesos de procesamiento de solicitudes que estarán disponibles cuando se inicie el servidor. El valor predeterminado es 2.

Incremento de subprocesos

Número de subprocesos de procesamiento de peticiones que se agregarán cuando el número de solicitudes exceda el total inicial de subprocesos. El valor predeterminado es 1.

Tiempo de espera de solicitud

Número de segundos antes de que se agote el tiempo de espera de una solicitud. El valor predeterminado es 30.

Longitud de memoria intermedia

Tamaño, en bytes, del búfer utilizado por los subprocesos de procesamiento de peticiones para leer los datos de las solicitudes. El valor predeterminado es 8192.
