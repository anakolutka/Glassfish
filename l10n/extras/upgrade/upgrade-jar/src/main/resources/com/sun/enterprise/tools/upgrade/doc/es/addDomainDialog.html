Cuadro de diálogo Agregar dominio

Utilice el cuadro de diálogo Agregar dominio para seleccionar el nombre de dominio y las contraseñas de certificado de origen y destino.

  1. Haga clic en el menú desplegable Dominio de origen y seleccione el dominio que contenga los certificados de seguridad que migrar.
  2. En el panel Contraseñas, introduzca las contraseñas se soliciten. Según el tipo de instalación de origen y destino, es posible que necesite introducir dos o tres contraseñas:
    • Introduzca las contraseñas de los archivos de certificado NSS, que se utilizan para bases de datos NSS de confianza, en los campos NSS de origen y NSS de destino.
    • Introduzca la contraseña para la base de datos de claves de usuario de origen, utilizada para el archivo JSSE keystore.jks, que contiene certificados de pares de claves de usuario, en el campo Base de datos de claves de usuario.
    • Introduzca la contraseña para la base de datos de autoridad de certificado, utilizada para el archivo JSSE cacerts.jks, que contiene certificados de autoridad de certificado, en el campo Base de datos de la autoridad emisora de certificados.
  3. Haga clic en Aceptar cuando haya terminado. Se abrirá la pantalla Transferir certificados de seguridad, en la que se muestra el dominio añadido en el panel Dominios con certificados que transferir.
