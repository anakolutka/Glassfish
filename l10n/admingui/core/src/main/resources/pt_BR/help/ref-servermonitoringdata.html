Monitoramento do Servidor

Use a página Monitoramento do Servidor para exibir os dados de monitoramento de um servidor virtual selecionado. Para uma descrição das propriedades monitoradas, consulte Capítulo 8, Administering the Monitoring Service, no Sun GlassFish Enterprise Server v3 Administration Guide.
