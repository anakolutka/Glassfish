查看 Web 应用程序的描述符

  1. 在导航树中，展开“应用程序”节点。
  2. 在“应用程序”节点下，选择“Web 应用程序”节点。

    此时将打开“Web 应用程序”页面。

  3. 在“Web 应用程序”页面上，单击您要查看其描述符的 Web 应用程序的名称。

    此时将打开“Web 应用程序”页面。

  4. 在“Web 应用程序”页面上，单击“描述符”选项卡。

    将打开“自定义部署描述符”页面。

  5. 在“部署描述符”字段中，单击您要查看的 XML 文件的名称。

    此时将在新的页面中打开 XML 文件。

  6. 浏览此文件，然后单击“关闭”返回到“自定义部署描述符”页面。
