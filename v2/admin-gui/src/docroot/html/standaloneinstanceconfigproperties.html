Setting Instance Specific Configuration Properties

In the Instance Specific Configuration Properties page, you can modify the instance values of various configuration properties specific to an instance. The Configuration Properties table displays the properties, their default values and the instance values. You can modify the instance values in the Override Value column and click Save.

The default instance values are defined in the configuration bound to this instance. To set default values for configuration properties, click the Stand-alone Instances node and in the Configuration column on Stand-Alone Server Instances page, click the configuration for the required instance. In the Configuration page, click System Properties. In the Configuration System Properties page, click Instance Values for the property you want to edit.
