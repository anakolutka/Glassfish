Per arrestare le istanze standalone di GlassFish Server

  1. Selezionare il nodo Istanze standalone nell'albero di navigazione.

    Viene visualizzata la pagina Istanze server standalone.

  2. Per ogni istanza da arrestare, selezionare la casella di controllo in corrispondenza dell'istanza nella tabella Istanze server.

  3. Fare clic su Arresta.

    Viene richiesto di confermare l'arresto delle istanze.

  4. Confermare che si desidera arrestare le istanze.

    Lo stato di ogni istanza viene aggiornato nella tabella Istanze server quando l'istanza viene arrestata.
