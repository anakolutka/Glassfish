HTTP 协议

“HTTP 协议”页面包含以下选项：

版本

使用的 HTTP 协议的版本（HTTP/1.0 或 HTTP/1.1）。默认值是 HTTP/1.1。

启用 DNS 查找

如果选中此选项，则将启用对客户端 DNS 条目的查找。默认情况下，此选项未选中。

启用 SSL

如果选中此选项，则 SSL 将用于任何启用了安全性的侦听器。如果未选中此选项，则将在服务器中全局禁用安全性。默认情况下，此选项已选中。

强制类型

没有与扩展名匹配的可用 MIME 映射时使用的响应类型。默认值为 text/html; charset=iso-8859-1。

默认类型

默认值为 text/html; charset=iso-8859-1。此值是以分号分隔的字符串，由内容类型、编码、语言和字符集组成。
