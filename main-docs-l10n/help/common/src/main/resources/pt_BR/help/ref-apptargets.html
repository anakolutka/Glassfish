Alvos da Aplicação

Utilize a página Alvos da Aplicação para exibir os clusters e as instâncias de servidores standalone do alvo nos quais a aplicação pode ser ativada.

A página Alvos da Aplicação contém as informações a seguir.

Nome do Alvo

Nome do cluster ou da instância de servidor standalone.

Ativado

Exibirá true se a aplicação estiver ativada no alvo ou false se a aplicação estiver desativada.

Balanceador de Carga Ativado

Exibirá true se a aplicação estiver ativada para balanceamento de carga no alvo ou false se a aplicação estiver desativada para balanceamento de carga.

Servidores Virtuais

Para associar a aplicação a servidores virtuais específicos em um alvo, selecione Gerenciar Servidores Virtuais no alvo em questão.

A tabela Alvos também contém as seguintes opções.

Gerenciar Alvos

Botão para gerenciar os alvos da aplicação.

Mais Ações

Lista drop-down das ações seguintes.

Ativar

Ação para ativar a aplicação no alvo selecionado.

Desativar

Ação para desativar a aplicação no alvo selecionado.

Ativar Balanceador de Carga

Ação para ativar o balanceamento de carga da aplicação no alvo selecionado.

Desativar Balanceador de Carga

Ação para desativar o balanceamento de carga da aplicação no alvo selecionado.
