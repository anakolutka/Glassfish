Pools de Conexões do Conector

Use a página Pools de Conexões do Conector para configurar pools de conexões do conector.

Um pool de conexões do conector é um grupo de conexões reutilizáveis para um determinado EIS.

Para cada recurso, as informações a seguir são fornecidas.

Nome do Pool

Um nome exclusivo que identifica o pool de conexões do conector.

Adaptador do Recurso

O adaptador do recurso associado ao pool de conexões do conector.

Definição da Conexão

O nome da classe da definição da conexão para o pool de conexões do conector.

Descrição

Uma descrição do pool de conexões do conector.

A tabela Pools também contém as seguintes opções.

Novo

Botão para criar um novo pool de conexões do conector.

Deletar

Botão para deletar um ou mais pools de conexões do conector selecionados.
