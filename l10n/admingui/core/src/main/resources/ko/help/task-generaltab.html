Enterprise Server에 대한 일반 정보를 보는 방법

  1. 탐색 트리에서 Enterprise Server 노드를 선택합니다.

    일반 정보 페이지가 열립니다.

  2. 일반 정보 페이지에서 다음 단계 중 하나를 수행합니다.
    • Enterprise Server를 중지하려면 중지 버튼을 누릅니다.
    • Enterprise Server를 중지했다 다시 시작하려면 다시 시작 버튼을 누릅니다.
    • 서버 로그를 보려면 로그 파일 보기 버튼을 누릅니다.

      자세한 내용은 서버 로그를 보는 방법을 참조하십시오.

    • JVM 보고서를 누르면 요약 보고서, 메모리 관리 보고서, 클래스 로딩 보고서 및 현재 스레드 덤프를 포함하여 Java Virtual Machine에 대한 보고서를 볼 수 있는 별도의 창이 열립니다.
참조

  • 서버 로그를 보는 방법

  • 고급 응용 프로그램 구성을 설정하는 방법

  • 고급 도메인 속성을 설정하는 방법

  • 관리자 비밀번호를 변경하는 방법

  • 시스템 등록 정보를 관리하는 방법

  • 응용 프로그램 모니터링 데이터를 보는 방법

  • 서버 모니터링 데이터를 보는 방법
