Per configurare il broker ORB

  1. Espandere il nodo Configurazione nel componente della struttura.

  2. Selezionare l'istanza da configurare:

    • Per configurare un'istanza particolare, selezionare il nodo di configurazione dell'istanza. Ad esempio, per l'istanza predefinita server, selezionare il nodo server-config.

    • Per configurare le impostazioni predefinite per le istanze future che utilizzano una copia di default-config, selezionare il nodo default-config.

  3. Selezionare il nodo ORB.

    Viene visualizzata la pagina ORB.

  4. Scegliere il pool del thread utilizzato dal broker ORB nell'elenco a discesa ID pool di thread.

    Il broker ORB usa i pool di thread per rispondere alle richieste dei client remoti degli Enterprise Bean e di altri client che comunicano mediante il protocollo RMI-IIOP. Il pool di thread predefinito per il broker ORB è thread-pool-1, che utilizza le impostazioni predefinite appropriate per le richieste RMI/IIOP.

    Per ulteriori informazioni, vedere Pool di thread.

  5. Nell'elenco a discesa Dimensione massima frammento messaggio selezionare la dimensione di frammento massima per i messaggi IIOP.

    I messaggi con dimensione superiore alla dimensione specificata vengono frammentati. Il valore predefinito è 1024 byte.

  6. Nel campo Connessioni totali impostare il numero massimo di connessioni in entrata per tutti i listener IIOP.

    Il valore predefinito è 1024.

  7. Selezionare la casella di controllo Autenticazione client IIOP obbligatoria per specificare che è necessaria l'autorizzazione client IIOP.

    Se l'autenticazione client IIOP è abilitata, il server rifiuta le richieste non autenticate e inserisce un bit autenticazione richiesta nei riferimenti IOR (Interoperable Object References) inviati ai client. Questa opzione è disabilitata per impostazione predefinita.

  8. Nell'area Proprietà aggiuntive specificare le proprietà aggiuntive.

    GlassFish Server non definisce alcuna proprietà aggiuntiva per il broker ORB.

    • Per aggiungere una proprietà, fare clic sul pulsante Aggiungi proprietà. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Nome e il valore della proprietà nel campo Valore.

    • Per modificare una proprietà, è necessario modificare il valore del campo Valore della proprietà.

    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Nome corrispondente alla proprietà da eliminare, quindi fare clic sul pulsante Elimina proprietà.

  9. Fare clic su Salva.

    Suggerimento:

    Per ripristinare le impostazioni predefinite, fare clic sul pulsante Carica impostazioni predefinite, quindi su Salva.

  10. Riavviare GlassFish Server.
