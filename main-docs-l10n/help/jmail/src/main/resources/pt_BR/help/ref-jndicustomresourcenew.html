Novo Recurso Personalizado

Utilize a página Novo Recurso Personalizado para criar um recurso JNDI personalizado.

A página Novo Recurso Personalizado contém as opções a seguir.

Nome JNDI

Um nome exclusivo que identifica o recurso.

Tipo de Recurso

A definição de tipo totalmente qualificado para o recurso.

Classe de Fábrica

O nome especificado pelo usuário para a classe de fábrica. Essa classe implementa a interface javax.naming.spi.ObjectFactory.

Descrição

Uma descrição do recurso personalizado.

Status

O status do recurso personalizado. O recurso personalizado pode ser ativado ou desativado. O valor default é Ativado.

Propriedades Adicionais

Propriedades adicionais para o recurso personalizado. O GlassFish Server não define quaisquer propriedades adicionais para recursos personalizados.

Alvos

(Esta opção só será exibida se os clusters ou instâncias standalone tiverem sido criados no domínio.)

Clusters e instâncias standalone do recurso. Mova os alvos desejados para a coluna Alvos Selecionados utilizando os botões Adicionar e Adicionar Tudo. Mova os alvos desnecessários para a coluna Alvos Disponíveis utilizando os botões Remover e Remover Tudo.
