Alias de mot de passe

La page Alias de mot de passe permet de créer et de gérer des alias de mot de passe.

Elle présente la liste des alias de mot de passe du domaine. Pour chaque alias, les informations suivantes sont affichées :

Nom

Nom affecté à l'alias de mot de passe lors de sa création. Lorsque vous cliquez sur le nom, la page Modifier l'alias du mot de passe de l'alias s'ouvre.

La table Alias de mot de passe contient également les options ci-après.

Nouveau

Bouton permettant de créer un alias de mot de passe.

Supprimer

Bouton permettant de supprimer les alias de mot de passe sélectionnés.
