“监视服务”页面

使用“监视服务”页面可为各个服务器组件或服务配置监视选项。

JVM

JVM 监视级别。可用级别有“低”、“高”和“关闭”。在此发行版中，“低”和“高”选项所提供的监视级别之间没有差别。

HTTP 服务

HTTP 服务监视级别。可用的级别有“低”、“高”和“关闭”。在此发行版中，“低”和“高”选项所提供的监视级别之间没有差别。

Web 容器

Web 容器监视级别。可用级别分别为“低”、“高”和“关闭”。在此发行版中，“低”和“高”选项所提供的监视级别之间没有差别。

线程池

线程池监视级别。可用级别有“低”、“高”和“关闭”。在此发行版中，“低”和“高”选项所提供的监视级别之间没有差别。
