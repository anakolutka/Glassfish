Propriedades Específicas das Configurações do Provedor de Segurança de Mensagens

As propriedades a seguir estão disponíveis para uma configuração do provedor de segurança de mensagens

server.config

O diretório e o nome de um arquivo XML que contém as informações de configuração do servidor. O valor padrão é domain-dir/config/wss-server-config-1.0.xml .

debug

Se true, ativa o despejo das mensagens de depuração do provedor do servidor para o log do servidor. O valor padrão é falso.

dynamic.username.password

Se true, assinala o tempo de execução do provedor para coletar o nome de usuário e a senha doCallbackHandler para cada solicitação. Se falso, o nome de usuário e a senha de wsse:UsernameToken(s) são coletados uma vez, durante a inicialização do módulo. Essa propriedade só é aplicável para um ClientAuthModule. O valor padrão é false.

encryption.key.alias

Especifica a chave de criptografia usada pelo provedor. A chave é identificada por seu alias keystore. O valor padrão é s1as.

signature.key.alias

Especifica a chave de assinatura usada pelo provedor. A chave é identificada por seu alias keystore. O valor padrão é s1as.
