Group Management Service bearbeiten

Auf der Seite "Group Management Service" können Sie die GMS-Einstellungen ändern.

Der Group Management Service (GMS) ist ein prozessinterner Service, der Services für Clusterüberwachung, Clustermitgliedschaft und Gruppenkommunikation bereitstellt. Der GMS baut auf einem konfigurierbaren Stapel von Protokollen zu Gruppenmitgliedschaftserkennung und Zustandsüberwachung auf. Der GMS führt folgende Funktionen aus:

  • Benachrichtigt registrierte Module in einer GlassFish Server-Instanz, wenn mindestens eine Mitgliedsinstanz zum Cluster hinzugefügt bzw. aus dem Cluster entfernt wurde oder bei einer Mitgliedsinstanz Fehler vermutet bzw. bestätigt wurden.

  • Bietet die Möglichkeit, Nachrichten zwischen Cluster-Membern zu senden und zu empfangen.

Die Seite "Group Management Service" enthält die folgenden Optionen.

Standardeinstellungen laden

Schaltfläche zur Wiederherstellung der Standardwerte von Einstellungen, die Standardwerte haben. Einstellungen, die keine Standardwerte haben, werden nicht geändert.

Name der Konfiguration

Der Name der Konfiguration, für die die Einstellungen auf dieser Seite angewendet werden. Dieses Feld ist schreibgeschützt.

Maximale Anzahl nicht ausgelöster Heartbeats

Die Anzahl nicht ausgelöster Heartbeats eines Mitglieds, nach der ein Serverinstanzfehler vermutet wird. Der Standardwert ist 3.

Heartbeat-Häufigkeit

Der Zeitraum zwischen nicht ausgelösten Heartbeats eines Mitglieds. Der Standardwert ist 2000 Millisekunden.

Timeout für Gruppenerkennung

Der Zeitraum, den der GMS wartet, bis die Erkennung anderer Cluster-Member fortgesetzt wird. Der Standardwert ist 5000 Millisekunden.

Wartezeit für Fehlerüberprüfung

Die Wartezeit, bevor geprüft wird, ob ein Mitglied, bei dem ein Fehler vermutet wird, als Fehler gemeldet werden soll. Der Standardwert ist 1500 Millisekunden.

Eine Tabelle mit weiteren Eigenschaften für den Group Management Service.

Bei GlassFish Server sind keine weiteren Eigenschaften für den GMS definiert.

Für jede Instanz werden folgende Informationen angezeigt:

Name

Der Name der Eigenschaft.

Wert

Der Wert der Eigenschaft, der für die ausgewählte Instanz festgelegt ist.

Beschreibung

Eine Textbeschreibung mit weiteren Informationen zur Eigenschaft.

Die Tabelle "Weitere Eigenschaften" enthält darüber hinaus die folgenden Optionen.

Eigenschaft hinzufügen

Schaltfläche zum Hinzufügen einer Eigenschaft. Durch Klicken auf diese Schaltfläche wird eine Zeile zur Tabelle "Weitere Eigenschaften" hinzugefügt.

Eigenschaften löschen

Schaltfläche zum Löschen von mindestens einer ausgewählten Eigenschaft. Eine gelöschte Eigenschaft wird auf den Standardwert zurückgesetzt oder, falls kein Standardwert vorhanden ist, nicht definiert.
