共通タスク

「共通操作」ページには、管理コンソールの一般的な操作のためのショートカットが用意されています。

「共通操作」ページを使用して次の操作を実行できます。

  • 「Sun Connection に Sun GlassFish Enterprise Server を登録する」

  • 「Sun GlassFish Enterprise Server の「Support」ページを表示する」

  • 「Web アプリケーションを配備する」

  • 「JDBC 接続プールを作成する」

  • 監視データを表示する

  • 「監視サービスを設定する」

  • インストールされたコンポーネントを表示する

  • 「インストール済みコンポーネントを更新する」

  • 「アドオンコンポーネントをインストールする」

「共通操作」ページには、Enterprise Server マニュアルへのリンクもあります。
