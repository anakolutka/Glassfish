虚拟服务器

虚拟服务器有时称为虚拟主机，是允许同一物理服务器托管多个 Internet 域名的对象。同一物理服务器上托管的所有虚拟服务器都共享该物理服务器的 Internet 协议 (Internet Protocol, IP) 地址。虚拟服务器将服务器的域名（如 www.sun.com）与运行 Enterprise Server 的特定服务器相关联。
