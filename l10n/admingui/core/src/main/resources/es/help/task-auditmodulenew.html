Cómo crear un módulo de auditoría

Servidor empresarial ofrece un sencillo módulo de auditoría predeterminado; para obtener más información, consulte Cómo utilizar el módulo de auditoría predeterminado .

  1. Expanda el nodo de configuraciones.
  2. Seleccione la instancia que va a configurar:
    • Para configurar una instancia específica, expanda el nodo de configuración de dicha instancia. Por ejemplo, en la instancia predeterminada server expanda el nodo server-config.
    • Para configurar los valores de configuración predeterminados de todas las instancias, expanda el nodo default-config.
  3. Expanda el nodo Seguridad.
  4. Seleccione el nodo Módulos de auditoría.
  5. En la página Módulos de auditoría haga clic en Nuevo.
  6. En la página Crear módulo de auditoría introduzca la información siguiente:

    • Nombre – El nombre utilizado para identificar este módulo de auditoría.

    • Nombre de clase – El nombre completo de la clase que implementa este módulo. El nombre de clase para el módulo de auditoría predeterminado es com.sun.enterprise.security.Audit.

  7. Para agregar propiedades de JVM a este módulo haga clic en Agregar propiedad. Especifique un nombre y un valor para cada propiedad.

    Entre las propiedades válidas está auditOn, que especifica si se activa o no esta clase de implementación. Los valores permitidos son true y false.

  8. Haga clic en Aceptar para guardar las entradas o haga clic en Cancelar para salir sin guardar.
