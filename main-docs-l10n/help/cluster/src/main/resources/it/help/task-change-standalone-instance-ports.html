Per modificare i numeri di porta di un'istanza standalone di GlassFish Server

I numeri di porta di un'istanza standalone di GlassFish Server sono impostati inizialmente nella configurazione alla quale fa riferimento l'istanza. È necessario modificare questi numeri di porta se, ad esempio, due istanze che fanno riferimento alla stessa configurazione risiedono nello stesso host. In questa situazione è necessario modificare i numeri di porta per una delle istanze, in quanto le due istanze fanno riferimento alla stessa configurazione.

I numeri di porta di un'istanza vengono memorizzati come proprietà di sistema Java. Queste proprietà vengono passate al launcher dell'applicazione Java tramite la relativa opzione -D all'avvio di GlassFish Server.

  1. Selezionare il nodo Istanze standalone nell'albero di navigazione.

    Viene visualizzata la pagina Istanze server standalone.

  2. Selezionare l'istanza per la quale modificare i numeri di porta.

    Viene visualizzata la pagina Informazioni generali per l'istanza.

  3. Nella pagina Informazioni generali, fare clic sulla scheda Proprietà.

    Viene visualizzata la pagina Proprietà di sistema dell'istanza.

    Nella pagina vengono visualizzati i valori correnti delle proprietà di sistema per l'istanza riportate di seguito.

    • I numeri di porta predefiniti nella configurazione dell'istanza

    • Le eventuali proprietà personalizzate definite per l'istanza

  4. Per ogni proprietà esistente da modificare, digitare il nuovo valore nel campo Valore di sostituzione per la proprietà.

  5. Impostare i valori per eventuali proprietà aggiuntive necessarie.

    1. Nella tabella Proprietà aggiuntive fare clic su Aggiungi proprietà.

      Viene aggiunta una riga alla tabella Proprietà aggiuntive.

    2. Nel campo Nome variabile istanza digitare il nome della proprietà.

    3. Nel campo Valore di sostituzione digitare il valore.

  6. Fare clic su Salva.
