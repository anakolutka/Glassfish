GlassFish Serverブログを表示するには

GlassFish Serverのブログには、GlassFish Serverコミュニティからのニュース、およびGlassFish Serverに関する追加情報へのリンクが用意されています。

  1. ナビゲーション・ツリーで、「共通タスク」ノードを選択します。

    「共通タスク」ページが開きます。

  2. 「GlassFishニュース」の下で、ブログのリンクをクリックします。

    GlassFish Serverのブログが、新しいブラウザ・タブまたはウィンドウで開きます。
