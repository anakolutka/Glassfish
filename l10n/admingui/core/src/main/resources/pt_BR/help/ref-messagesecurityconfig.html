Segurança de Mensagens

A página Configurações de Segurança de Mensagens lista as configurações disponíveis para provedores de segurança de mensagens.

Para configurar a segurança de mensagens, você precisa de uma compreensão básica de conceitos de segurança e de serviços da Web. Para obter mais informações, consulte o Capítulo 13, Administering Message Security, no Sun GlassFish Enterprise Server v3 Administration Guide e o Capítulo 6, Administering Web Applications, no Sun GlassFish Enterprise Server v3 Administration Guide.

O Enterprise Server implementa a segurança de mensagens usando provedores de segurança de mensagens integrados em sua camada de processamento SOAP. Os provedores de segurança de mensagens dependen de outros recursos de segurança de Enterprise Server.

Para cada configuração, a página Configurações de Segurança de Mensagens contém as seguintes informações.

Camada de Autenticação

A camada de mensagens na qual a autenticação é realizada. Por padrão, uma configuração SOAP é fornecida. Também é possível fornecer uma configuração HttpServlet.

Provedor Padrão

O provedor do servidor padrão desta configuração. Por padrão, nenhum provedor de servidor padrão está configurado.

Provedor de Cliente Padrão

O provedor de cliente padrão desta configuração. Por padrão, nenhum provedor de cliente padrão está configurado.
