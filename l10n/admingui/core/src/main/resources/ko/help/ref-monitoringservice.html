모니터링 서비스

모니터링 서비스 페이지에서는 개별 서버 구성 요소 또는 서비스의 모니터링 옵션을 구성할 수 있습니다.

사용 가능한 각 구성 요소에 대해 모니터링 서비스 페이지에는 다음 정보가 나열됩니다.

구성 요소

Enterprise Server 구성 요소의 이름입니다.

모니터링 수준

구성 요소에 대해 설정된 모니터링의 수준입니다. 각 구성 요소에 대해 사용 가능한 수준은 LOW, HIGH 및 OFF입니다. 기본 수준은 끄기입니다. 이 릴리스에서는 낮음 및 높음 옵션에서 제공되는 모니터링 수준 간에 차이가 없습니다.

관련 작업

  • 모니터링 서비스를 구성하는 방법

  • 응용 프로그램 모니터링 데이터를 보는 방법

  • 서버 모니터링 데이터를 보는 방법

관련 asadmin 명령

  • monitor(1)
