配置 HTTP 通訊協定

  1. 在瀏覽樹狀結構中，展開 [配置] 節點。
  2. 在 [配置] 節點下，選取 [HTTP 服務] 節點。
  3. 從 [HTTP 服務] 頁面中，按一下 [HTTP 通訊協定] 標籤。

    [HTTP 通訊協定] 頁面會隨即開啟。

  4. 在 [版本] 欄位中，鍵入要使用的 HTTP 通訊協定版本。

    可接受的值包括 HTTP/1.0 和 HTTP/1.1。預設值為 HTTP/1.1。

  5. 選取 [啟用 DNS 查詢] 核取方塊，可啟用用戶端的 DNS 項目查詢。

    此選項預設為停用。

  6. 取消選取 [啟用 SSL] 核取方塊，可以全域方式停用伺服器安全性。

    將此核取方塊保留為已選取，可將 SSL 用於任何啟用了安全性的偵聽程式。此選項預設為啟用。

  7. 在 [強制類型] 欄位中，鍵入在沒有與副檔名相符的可用 MIME 對映時要使用的回應類型。

    預設值為 text/html; charset=iso-8859-1。

  8. 在 [預設類型] 欄位中，輸入預設的回應類型。

    預設值為 text/html; charset=iso-8859-1。該值是以分號分隔的字串，由內容類型、編碼、語言和字元集組成。

  9. 按一下 [儲存]。

    提示 – 若要回到預設設定，請按一下 [載入預設值] 按鈕，然後按一下 [儲存]。
