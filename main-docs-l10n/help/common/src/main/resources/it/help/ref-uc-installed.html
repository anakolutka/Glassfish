Componenti installati

La pagina Componenti installati fornisce informazioni sui componenti dell'immagine di installazione installati.

Per disinstallare i componenti installati, è necessario in primo luogo arrestare il dominio di GlassFish Server. Utilizzare quindi uno dei comandi riportati di seguito.

  • glassfish3/bin/updatetool, che avvia la utility grafica Strumento di aggiornamento.

  • glassfish3/bin/pkg, la versione da riga di comando dello strumento di aggiornamento.

Per informazioni sull'uso del comando pkg, vedere il Capitolo 11, Extending and Updating GlassFish Server, in Oracle GlassFish Server Administration Guide. La utility Strumento di aggiornamento dispone di una Guida in linea specifica.

Una volta completata la disinstallazione dei componenti, riavviare il server.

La pagina Componenti installati contiene le opzioni riportate di seguito.

Percorso di installazione

Il percorso del file system in cui si trova la directory radice o di livello superiore dell'immagine di installazione.

Componenti installati

Per ogni componente installato, questa tabella fornisce le informazioni riportate di seguito.

Componente

Il nome del componente.

Categoria

La categoria del software a cui è associato il componente, ad esempio Application Server, strumenti di sistema e database. La categoria viene assegnata al componente dall'autore al momento della creazione del componente.

Versione

Il numero di versione del package in cui viene fornito il componente.

Dimensione installata

La quantità di spazio su disco usata per l'installazione del componente.

Origine

Il nome del Repository di rete da cui è stato ottenuto il componente.

Task correlati

Per visualizzare i componenti installati
