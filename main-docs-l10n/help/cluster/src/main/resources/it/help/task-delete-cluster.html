Per eliminare un cluster

L'eliminazione comporta l'eliminazione del cluster e di tutte le istanze di GlassFish Server presenti nel cluster. Non è necessario eliminare le istanze presenti nel cluster prima di eliminare il cluster.

Se la configurazione utilizzata dal cluster non è condivisa, verrà eliminata anche la configurazione.

Prima di iniziare

Assicurarsi che il cluster da eliminare sia arrestato. Per le necessarie istruzioni, vedere Per arrestare un cluster.

  1. Selezionare il nodo Cluster nell'albero di navigazione.

    Viene visualizzata la pagina Cluster.

  2. Nella pagina Cluster selezionare la casella di controllo accanto al nome del cluster.

  3. Fare clic su Elimina.

    Viene richiesto di confermare l'eliminazione del cluster e di tutte le istanze presenti nel cluster.

  4. Confermare l'eliminazione del cluster e di tutte le istanze presenti nel cluster.

    Il cluster viene eliminato e rimosso dalla tabella dei cluster della pagina Cluster.
