Para editar un recurso JDBC

  1. En el árbol de navegación, expanda el nodo Recursos.
  2. Seleccione el nodo Recursos JDBC.
  3. En la página Recursos JDBC, haga clic en el nombre del recurso que desee editar.

    Se abrirá la página Editar recurso JDBC.

    Nota – El campo Nombre J.D.N.I. es un campo de sólo lectura. Sólo puede especificar el nombre J.N.D.I. al crear un nuevo recurso JDBC.

  4. En la lista desplegable Nombre de conjunto, seleccione un conjunto de conexiones para asociarlo al recurso JDBC.
  5. En el campo Descripción, introduzca una breve descripción del recurso JDBC.
  6. Haga clic en "Guardar".
