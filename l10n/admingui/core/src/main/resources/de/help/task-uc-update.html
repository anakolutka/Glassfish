So aktualisieren Sie installierte Komponenten

Das Aktualisieren einer installierten Komponente umfasst folgende Schritte:

  • Nur die geänderten Dateien von einem Netzwerk-Repository auf Ihr System übertragen

  • Nur die geänderten Dateien im Enterprise Server Installations-Image auf Ihrem System aktualisieren

  • Neue Abhängigkeiten (sofern vorhanden ) zwischen den aktualisierten Komponenten anwenden

  1. Wählen Sie in der Navigationsstruktur den Knoten Update-Tool aus.

    Daraufhin wird die Seite Installierte Komponenten angezeigt.

  2. Klicken Sie auf der Seite "Installierte Komponenten" auf die Registerkarte "Verfügbare Updates".

    Die Seite Verfügbare Updates wird angezeigt.

  3. Wählen Sie in der Tabelle der verfügbaren Updates die Komponenten aus, die Sie aktualisieren.

    Wenn keine Updates verfügbar sind, ist die Tabelle leer.

    • Um eine einzelne Komponente auszuwählen, aktivieren Sie das Kontrollkästchen neben dem Namen der Komponente.
    • Um alle Komponenten auszuwählen, klicken Sie in der Kopfzeile der Tabelle auf das Symbol "Alle auswählen".
    • Um die Auswahl aller Komponenten aufzuheben, klicken Sie in der Kopfzeile der Tabelle auf das Symbol "Auswahl für alle aufheben".
  4. Klicken Sie auf „Installieren“.
  5. Akzeptieren Sie die Lizenzvereinbarung.

    Enterprise Server bestätigt den Abschluss der Installation. Die Komponenten werden aus der Tabelle der verfügbaren Updates gelöscht.

  6. Starten Sie den Domänenverwaltungsserver neu.

    Weitere Informationen finden Sie in der folgenden Dokumentation:

    • To Stop a Domain (or Server) in Sun GlassFish Enterprise Server v3 Prelude Administration Guide

    • To Start a Domain (or Server) in Sun GlassFish Enterprise Server v3 Prelude Administration Guide
