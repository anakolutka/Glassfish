领域

Enterprise Server 预先配置了三个领域：file、certificate 和 admin-realm。还可以创建 ldap、JDBC、solaris 和自定义领域。通常，一台服务器上每种类型的领域都会有一个，但在 Enterprise Server 上有两个 file 领域：file 和 admin-realm。这两个领域的类型相同但用于两种不同的目的。还可以在系统中为每台虚拟服务器创建不同的证书数据库。

“领域”页面显示可用领域的列表。对于每个领域，将提供以下信息：

  • 名称

  • 类名
