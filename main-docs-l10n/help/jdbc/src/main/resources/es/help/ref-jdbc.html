  • Pools de Conexiones

La tecnología JDBC ( Java DataBase Connectivity) proporciona conectividad a bases de datos relacionales. En GlassFish Server, configure la conectividad a una base de datos agregando recursos JDBC (orígenes de datos) al dominio. Un origen de datos es un método estándar de plataforma Java, Enterprise Edition (plataforma Java EE) de configuración de la conectividad a una base de datos. Cada origen de datos contiene un pool de conexiones de bases de datos. Las aplicaciones buscan el origen de datos en el árbol de API de JNDI (Interfaz de Directorios y Nomenclatura de Java) o en el contexto de la aplicación local y, a continuación, recuperan la conexión con la base de datos con el método getConnection. Los orígenes de datos y sus pools de conexiones proporcionan procesos de gestión de conexión que ayudan a mantener el sistema en ejecución.

GlassFish Server incluye una copia de la base de datos Java DB y se ha configurado previamente con la capacidad para acceder a esta base de datos.
