網域日誌

使用「網域日誌」頁面可以收集與下載指定 GlassFish Server 執行處理或叢集的日誌檔，然後將其編譯成單一 ZIP 存檔。

備註:

如果指定叢集，則 ZIP 存檔也會包括網域管理伺服器 (DAS) 的日誌檔。

「網域日誌」頁面包含下列選項。

叢集

叢集的下拉式清單 (若叢集存在的話)。

執行處理

伺服器執行處理的下拉式清單。

收集日誌

將日誌收集到 log-files-name.zip 檔案的按鈕，其中 name 是叢集或執行處理的名稱。

log-files-name.zip 檔案可透過瀏覽器的下載機制下載。
