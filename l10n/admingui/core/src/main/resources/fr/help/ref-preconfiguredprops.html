Propriétés système prédéfinies

Les propriétés système suivantes sont prédéfinies pour le Enterprise Server.

com.sun.aas.installRoot

Répertoire sur lequel le Enterprise Server est installé.

com.sun.aas.instanceRoot

Répertoire de niveau supérieur pour un domaine Enterprise Server.

com.sun.aas.hostName

Nom de la machine hôte.

com.sun.aas.javaRoot

Répertoire d'installation de l'exécution Java.

com.sun.aas.imqLib

Le répertoire de bibliothèque pour le logiciel Sun GlassFish Message Queue.

com.sun.aas.domainName

Nom du domaine. Cette propriété n'est pas utilisée dans la configuration par défaut, mais peut être utilisée pour personnaliser la configuration. La valeur par défaut est domain1.
