Propiedades específicas de la clase FileRealm

Un dominio file requiere las siguientes propiedades.

Contexto JAAS

El contexto JAAS (Servicio de autenticación y autorización de Java) (mecanismo identificador para que el módulo de entrada utilice este dominio). El único valor válido es fileRealm.

Archivo de claves

Ruta completa y nombre del archivo en el que el servidor almacenará toda la información de este dominio relativa al usuario, el grupo y la contraseña. El valor predeterminado es domain-dir/config/keyfile para el dominio file y domain-dir/config/admin-keyfile para el dominio admin-realm .

<Check Alignment of PHs>El archivo de claves del dominio file está vacío inicialmente, de manera que los usuarios deben de registrarse en la página Archivar usuarios antes de utilizar el dominio file.

El archivo de claves del dominio admin-realm contiene inicialmente el nombre de usuario del administrador, la contraseña del administrador en formato cifrado y el grupo al que el usuario pertenece (de manera predeterminada es asadmin). <Check Alignment of PHs>Para obtener más información sobre cómo registrar usuarios en el dominio admin-realm, consulte Cómo otorgar acceso a las herramientas de administración..

Nota - Los usuarios del grupo asadmin en admin-realm tienen autorización para utilizar la Consola de administración y las herramientas asadmin. Agregue a este grupo sólo los usuarios que tienen privilegios de administración de servidores.

La siguiente propiedad adicional está disponible para un dominio file.

Asignar grupos

Una lista de nombres de grupo separada por comas. A todos los clientes que presentan certificados válidos se les asigna estos grupos, por ejemplo, employee,manager , donde estos son nombres de los grupos de usuarios.
