Para definir o Provedor de JACC ativo

  1. Na árvore de navegação, expanda o nó Configuração.
  2. Sob o nó Configuração, selecione o nó Segurança.

    A página Segurança é exibida.

  3. Na lista suspensa JACC, selecione o provedor de JACC para ser usado pelo servidor.

    Por padrão, dois provedores de JACC estão disponíveis, com os nomes padrão e simples. O provedor padrão é padrão.

  4. Clique em Salvar.
