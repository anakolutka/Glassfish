To View Load Balancer Targets

  1. In the tree component, select the HTTP Load Balancers node.
  2. In the right pane, click the load balancer to edit.
  3. Click the Targets tab. The Targets table in this page displays the following information.

    • Target Name — Displays the name of the target.

    • LB Enabled — Displays true if the target server is enabled for load balancing and false if the target is not enabled for load balancing. This column is available only if the target is server.

    • Disable Timeout — The amount of time, in minutes, this server takes to reach a quiescent state after having been disabled. This column is available only if the target is server.

    • Policy — Indicates the policy algorithm used for this target. This column is available only if the target is a cluster.

    • Health Checker — Click Edit Health Checker to specify the health checker properties.

  4. To add or remove targets, click Manage Targets.

    The Manage Targets page appears.

  5. Add or remove targets by highlighting the cluster or stand-alone instance name and clicking Add or Remove.

    To add or remove all targets click Add All or Remove All.

  6. Click OK.
