Editar Aplicativo

Use a página Editar Aplicativo para modificar um aplicativo existente.

A página Editar Aplicativo contém as seguintes opções.

Nome

Campo somente leitura que exibe o nome do aplicativo em edição.

Servidores Virtuais

Os servidores virtuais associados ao aplicativo.

Raiz de Contexto

Para aplicativos Web, especifica o caminho do aplicativo. Na URL do aplicativo Web, a raiz de contexto aparece imediatamente após o número da porta (http://host:port/ context-root/...). A raiz de contexto deve iniciar com uma barra; por exemplo, /hello.

Descrição

Uma breve descrição do aplicativo.

Status

Se a caixa de seleção Ativado estiver marcada, o aplicativo está ativado. Por padrão, essa opção está selecionada.

Java Web Start

Se a caixa de seleção Ativado estiver marcada, isso indica que o aplicativo usa o software Java Web Start. O Java Web Start fornece uma maneira não dependente do navegador de implantar aplicativos Java a serem executados em uma Máquina Virtual Java dedicada.

Diretório Implantado

Se a caixa de seleção estiver marcada, o aplicativo será implantado em um diretório.

Local

Arquivo somente leitura que exibe o local do aplicativo implantado.

Bibliotecas

Arquivo somente leitura que exibe os arquivos JAR de biblioteca, os quais são exigidos pelo aplicativo.

Submódulos

Tabela que exibe os nomes dos submódulos do aplicativo e seus respectivos mecanismos.
