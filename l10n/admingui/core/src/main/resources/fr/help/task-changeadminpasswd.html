Changer le mot de passe administrateur

Le chiffrement du mot de passe de l'administrateur renforce la sécurité de l'application. Pour plus d'informations sur les procédures de sécurité, consultez le Chapitre 6, Administering System Security du Sun GlassFish Enterprise Server v3 Prelude Administration Guide.

  1. Dans l'arborescence de navigation, développez le nœud Configuration.
  2. Si la configuration du domaine prend en charge des clusters, sélectionnez la configuration à modifier.

    Exemples : domaines créés avec le profil Cluster ou entreprise .

  3. Développez le nœud Sécurité.
  4. Développez le nœud Domaines.
  5. Sélectionnez le nœud admin-realm.
  6. Cliquez sur le bouton Gestion utilisateurs sur la page Modification du domaine.
  7. Sélectionnez l'utilisateur nommé admin.
  8. Saisissez le nouveau mot de passe puis confirmez-le.
  9. Cliquez sur Enregistrer.
