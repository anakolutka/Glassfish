Activer l'authentification mutuelle

Avec l'authentification mutuelle, l'authentification est activée à la fois du côté du serveur et du client. Pour pouvoir tester l'authentification mutuelle, il doit obligatoirement exister un client avec un certificat valide. Pour plus d'informations sur l'authentification mutuelle, consultez la section Sécurité du Java EE 5 Tutorial.

L'activation de l'authentification mutuelle comprend les tâches suivantes :

  • Activer l'authentification mutuelle pour toutes les applications

  • Activer l'authentification mutuelle d'une application spécifique
