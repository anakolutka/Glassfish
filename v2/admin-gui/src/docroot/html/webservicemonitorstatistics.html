To View and Set Monitoring Statistics for a Web Service

  1. In the tree view, expand the Web Services node.
  2. Select the web service for which you want to view the monitoring statistics.
  3. Click the Monitor tab. Then click the Statistics tab.
  4. In the Show data for server instance drop-down list, select the server for which you want to display statistics.
  5. The Admin Console displays the following monitoring data for the selected server instance:

    • Response Time (last request) in milliseconds.

    • Average Response time in milliseconds for any successful or unsuccessful operation

    • Minimum Response Time in milliseconds

    • Maximum Response Time in milliseconds

    • Throughput: the number of requests per second

    • Total Fault Request Count: number of faults that occurred on this web service endpoint

    • Total Success Request Count: number of successful requests.

    Note - If more than one instance is running on the server, select the instance for which you want to view the data. To view the monitoring data for the server, choose server from the list.

  6. Choose any of the data by selecting the radio button and click the Graph button to get the graphical display of the monitoring statistics.
  7. Click the Clear Data button to reset all statistics for the selected server.
