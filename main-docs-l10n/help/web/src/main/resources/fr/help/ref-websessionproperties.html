Propriétés propres aux sessions de conteneur Web

Les propriétés suivantes sont disponibles pour une session de conteneur Web.

enableCookies

Si la valeur est true, utilise des cookies pour le traçage de session. La valeur par défaut est true.

enableURLRewriting

Permet la réécriture d'URL. Cette propriété permet le traçage de session via la réécriture d'URL lorsque le navigateur Web n'accepte pas les cookies. Vous devez également utiliser un appel encodeURL ou encodeRedirectURL dans le servlet ou sur la page JavaServer Pages (JSP). La valeur par défaut est true.

idLengthBytes

Spécifie le nombre d'octets dans l'ID de session de ce module Web. La valeur par défaut est 128.
