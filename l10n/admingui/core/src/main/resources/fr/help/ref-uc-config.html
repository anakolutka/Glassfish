Configuration de Outil de mise à jour

La page Configuration contient les informations de configuration de Outil de mise à jour et vous permet de modifier la configuration de Outil de mise à jour.

La page Configuration de Outil de mise à jour contient les options suivantes :

Chemin d'installation

Chemin du système de fichiers du répertoire root ou du répertoire supérieur de l'image d'installation.

Référentiel préféré

Nom du référentiel réseau préféré qui est associé à cette image d'installation, comme par exemple glassfish.org.

Connexion Internet

Mode de connexion de Enterprise Server à Internet :

Connexion directe

Cette option signifie que Enterprise Server se connecte directement à Internet.

Via un proxy

Cette option signifie que Enterprise Server se connecte à Internet via un serveur proxy. Lorsque cette option est sélectionnée, les champs Hôte proxy et Port proxy sont ajoutés à la page.

Hôte proxy

Nom d'hôte du serveur proxy de l'ordinateur sur lequel Enterprise Server s'exécute.

Ce champ est applicable uniquement si l'ordinateur sur lequel Enterprise Server s'exécute se connecte à Internet par l'intermédiaire d'un serveur proxy.

Port proxy

Numéro de port du serveur proxy de l'ordinateur sur lequel Enterprise Server s'exécute.

Ce champ est applicable uniquement si l'ordinateur sur lequel Enterprise Server s'exécute se connecte à Internet par l'intermédiaire d'un serveur proxy.
