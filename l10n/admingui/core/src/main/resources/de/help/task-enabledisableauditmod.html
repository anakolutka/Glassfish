So aktivieren oder deaktivieren Sie die Überwachungsprotokollierung

  1. Erweitern Sie in der Navigationsstruktur den Knoten der Konfigurationen.
  2. Wählen Sie die zu konfigurierende Instanz:
    • Zum Konfigurieren einer bestimmten Instanz erweitern Sie den config-Knoten der Instanz. Für die Standardinstanz server erweitern Sie z. B. den Knoten server-config.
    • Zum Konfigurieren der Standardeinstellungen für alle Instanzen erweitern Sie den Knoten default-config.
  3. Wählen Sie unter dem Konfigurationsknoten den Sicherheitsknoten aus.

    Die Seite "Sicherheit" wird geöffnet.

  4. Aktivieren Sie das Kontrollkästchen "Überwachungsprotokollierung aktiviert", um die Überwachungsprotokollierung zu aktivieren.

    Um die Überwachungsprotokollierung zu deaktivieren, deaktivieren Sie das Kontrollkästchen "Überwachungsprotokollierung aktiviert".

    Bei Aktivierung dieser Option werden die Überwachungsmodule geladen, und es wird sichergestellt, dass die Module an Überwachungspunkten durch die Enterprise Server-Überwachungsbibliothek aufgerufen werden.

  5. Bei Aktivierung der Überwachungsprotokollierung geben Sie ein standardmäßiges Überwachungsmodul an, wie unter So legen Sie das aktive Überwachungsmodul fest beschrieben.
  6. Klicken Sie auf "Speichern".
  7. Starten Sie Enterprise Server neu, wenn in der Konsole "Neustart erforderlich" angezeigt wird.
