So wird eine benutzerdefinierte JNDI-Ressource gelöscht

  1. Blenden Sie in der Navigationsbaumstruktur den Knoten "Ressourcen" ein.

  2. Blenden Sie unter dem Ressourcenknoten den JNDI-Knoten ein.

  3. Wählen Sie unter dem JNDI-Knoten den Eintrag "Benutzerdefinierte Ressourcen".

    Die Seite "Benutzerdefinierte Ressourcen" wird geöffnet.

  4. Aktivieren Sie auf der Seite "Benutzerdefinierte Ressourcen" das Kontrollkästchen neben dem Namen der zu löschenden Ressource.

  5. Klicken Sie auf "Löschen".
