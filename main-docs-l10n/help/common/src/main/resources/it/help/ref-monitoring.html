Monitoraggio

Utilizzare la pagina Monitoraggio per configurare il monitoraggio e visualizzare i dati di monitoraggio per le istanze server in cluster e non in cluster.

Per ogni istanza server vengono fornite le informazioni riportate di seguito.

Nome istanza

Il nome dell'istanza server. Fare clic sul nome istanza per andare alla pagina Informazioni generali per l'istanza.

Nome cluster

Il nome del cluster al quale appartiene l'istanza server, se è un'istanza in cluster. Fare clic sul nome cluster per andare alla pagina Informazioni generali per il cluster.

Azione

Collegamenti che consentono di effettuare le operazioni riportate di seguito sulle istanze server.

Configura monitoraggio

Visualizza la pagina Servizio di monitoraggio per la configurazione del cluster o di un'istanza standalone.

Visualizza dati di monitoraggio

Collegamenti che consentono di visualizzare i tipi di dati di monitoraggio riportati di seguito per le istanze server.

Applicazione

Visualizza la pagina Monitoraggio applicazione per l'istanza server.

Server

Visualizza la pagina Monitoraggio server per l'istanza server.

Risorse

Visualizza la pagina Monitoraggio risorse per l'istanza server.

Task correlati

  • Per configurare il servizio di monitoraggio

  • Per visualizzare i dati di monitoraggio delle applicazioni

  • Per visualizzare i dati di monitoraggio dei server

  • Per visualizzare i dati di monitoraggio delle risorse
