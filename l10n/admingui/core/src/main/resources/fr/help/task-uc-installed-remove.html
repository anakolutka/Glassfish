Désinstaller des composants

La désinstallation d'un composant implique la suppression des éléments suivants :

  • Tous les fichiers et répertoires qui figuraient dans l'installation initiale du composant

  • Tous les fichiers et répertoires chargés par les mises à jour du composant après l'installation initiale

  1. Dans l'arborescence de navigation, sélectionnez le nœud Outil de mise à jour.

    La page Add-ons disponibles s'affiche.

  2. Sur la page Add-ons disponibles, cliquez sur l'onglet Composants installés.

    La page Composants installés s'ouvre.

  3. Dans le tableau des composants installés, sélectionnez les composants à désinstaller.
    • Pour sélectionner un composant, cochez la case située en regard de son nom.
    • Pour sélectionner tous les composants, cliquez sur l'icône Sélectionner tout dans l'en-tête du tableau.
    • Pour désélectionner tous les composants, cliquez sur l'icône Désélectionner tout dans l'en-tête du tableau.
  4. Cliquez sur Supprimer.

    Une boîte de dialogue vous demande de confirmer que les composants sélectionnés doivent être désinstallés.

  5. Dans la boîte de dialogue, cliquez sur OK.

    Enterprise Server confirme que la désinstallation est terminée. Les composants sont supprimés du tableau des composants installés et ajoutés au tableau des add-ons disponibles dans la page Available Add-Ons.
