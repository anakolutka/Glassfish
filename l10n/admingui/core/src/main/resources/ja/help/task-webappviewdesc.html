Web アプリケーションの記述子を表示する

  1. ナビゲーションツリーで、「アプリケーション」ノードを展開します。
  2. 「アプリケーション」ノードで、「Web アプリケーション」ノードを選択します。

    「Web アプリケーション」ページが開きます。

  3. 「Web アプリケーション」ページで、記述子を表示する Web アプリケーションの名前をクリックします。

    「Web アプリケーション」ページが開きます。

  4. 「Web アプリケーション」ページで、「記述子」タブをクリックします。

    「配備記述子をカスタマイズ」ページが開きます。

  5. 「配備記述子」フィールドで、表示する XML ファイルの名前をクリックします。

    新規ページで XML ファイルが開きます。

  6. そのファイルを参照し、「閉じる」をクリックして「配備記述子をカスタマイズ」ページに戻ります。
