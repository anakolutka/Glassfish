Para Exibir Arquivos de Log de uma Instância do GlassFish Server Standalone

Antes de Começar

A instância para a qual você está exibindo os arquivos de log deve estar em execução.

  1. Na árvore de navegação, selecione o nó Instâncias Standalone.

    A página Instância de Servidor Standalone será aberta.

  2. Selecione a instância cujos arquivos de log você deseja exibir.

    A página Informações Gerais da instância se abre.

  3. Na página Informações Gerais, clique em Exibir Arquivos de Log.

    A janela Visualizador de Log será aberta.

    Observação:

    A instância já deve estar em execução. Se a instância não estiver em execução, o botão Exibir Arquivos de Log está desativado.
