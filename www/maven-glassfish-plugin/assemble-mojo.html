glassfish:assemble

Creates a glassfish distribution image.

Mojo Attributes :

  • Requires a Maven 2.0 project to execute.
  • Requires dependency resolution of artifacts in scope: runtime
  • Automatically executes within the lifecycle phase: package

Optional Parameters

Name Type Description
finalName String The file name of the created distribution image. Default value is ${project.build.finalName}.zip.
outputDirectory File The directory where the final image will be created. Default value is ${project.build.directory}.
projectBuilder MavenProjectBuilder No description.

Parameter Details

finalName

The file name of the created distribution image.

  • Type : java.lang.String
  • Required : No
  • Expression : ${finalName}
  • Default : ${project.build.finalName}.zip

outputDirectory

The directory where the final image will be created.

  • Type : java.io.File
  • Required : No
  • Expression : ${outputDirectory}
  • Default : ${project.build.directory}

projectBuilder

No Description.

  • Type : org.apache.maven.project.MavenProjectBuilder
  • Required : No
