新建 HTTP 侦听器（步骤 2，共 2 步）

“新建 HTTP 侦听器（步骤 2，共 2 步）”页面包含以下选项：

客户机验证

可以强制客户机在使用此侦听器时向服务器验证自身。

证书昵称

现有服务器密钥对和证书的名称。

SSL3

如果启用此选项，则将在此侦听器上使用 SSL3 安全协议。

SSL2

如果启用此选项，则将在此侦听器上使用 SSL2 安全协议。

TLS

如果启用此选项，则将在此侦听器上使用 TLS 安全协议。

通用加密算法套件

可供协议使用的加密算法套件。

临时 Dittie-Hellman 加密算法套件

协议所使用的临时加密算法套件。
