048        /**
049            Create a new &lt;server&gt; given an existing config and node-agent.
050            These are required parameters for the server instance to be created.
051    
052            @param name the name of the server to create
053            @param nodeAgentName the node agent that the server will reference
054            @param configName the config that the server will reference
055            @param optional properties for this new server
056              This is a Map object consisting of key/value for a given property, that can be applied to
057              this server instance. The Map may also contain additional properties that can be applied to this server instance.
058              <p>Note that Properties that relate to ports of listeners are stored as system-properties and have
059              specific key names and must be specified to override values defined in the config to any of the relevant
060              ports - this is particularly required when the instance being created is on the same machine as other
061              instances in the domain.
062              <p>Legal property keys are those found in {@link ServerConfigKeys}.
063    
064            @return A proxy to the StandaloneServerConfig MBean that manages the newly created server
065         */
066        public StandaloneServerConfig createStandaloneServerConfig(String name, String nodeAgentName,
067                String configName, Map<String,String> optional);
