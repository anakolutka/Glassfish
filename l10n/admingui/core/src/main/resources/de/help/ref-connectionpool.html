Verbindungspool

Die Seite "Verbindungspool" umfasst die folgenden Optionen:

Maximal ausstehend

Gibt die maximale Anzahl an ausstehenden Verbindungen an, die für ein HTTP-Zielgerät zulässig sind. Der Standardwert lautet 4096.

Größe der Warteschlange

Gibt die maximale Größe der Verbindungswarteschlange in Byte an. Dieser Wert gibt auch die maximale Anzahl an ausstehenden Verbindungen an, die für den Server zulässig sind. Der Standardwert lautet 4096.

Größe des Empfangspuffers

Gibt die Größe des Empfangspuffers für ein HTTP-Zielgerät in Byte an. Der Standardwert lautet 4096.

Größe des Sendepuffers

Gibt die Größe des Sendepuffers für ein HTTP-Zielgerät in Byte an. Der Standardwert ist 8192.
