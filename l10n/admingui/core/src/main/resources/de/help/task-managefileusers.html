So verwalten Sie Dateibenutzer

Das Verwalten von Dateibenutzern wird in Enterprise Server über den Sicherheitsbereichsknoten durchgeführt.

  1. Erweitern Sie in der Navigationsstruktur den Konfigurationsknoten.
  2. Wenn die Domäne für die Unterstützung von Clustern konfiguriert ist, wählen Sie die zu ändernde Konfiguration aus.

    Beispiele sind Domänen, die mit dem Profil Cluster oder 'Unternehmen' erstellt werden.

  3. Erweitern Sie den Knoten "Sicherheit".
  4. Erweitern Sie den Knoten "Bereiche".
  5. Wählen den Knoten "Datei" aus.
  6. Klicken Sie auf der Seite "Bereich bearbeiten" auf die Schaltfläche "Benutzer verwalten".

    Die Seite "Dateibenutzer" wird geöffnet. Führen Sie auf dieser Seite die folgenden Aufgaben aus:

    • So fügen Sie einen Benutzer hinzu

    • So bearbeiten Sie Benutzerinformationen

    • So löschen Sie einen Benutzer
