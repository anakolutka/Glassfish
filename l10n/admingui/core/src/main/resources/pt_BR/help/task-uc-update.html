Para instalar atualizações de componentes

A atualização de um componente instalado envolve as seguintes operações:

  • Transferência somente dos arquivos modificados de um repositório da rede para o sistema

  • Atualização somente dos arquivos modificados dentro da imagem de instalação do Enterprise Server no sistema

  • Aplicação de novas dependências (se houver) entre os componentes atualizados

  1. Na árvore de navegação, selecione o nó Ferramenta de Atualização.

    É aberta a página Complementos Disponíveis.

  2. Na página Complementos Disponíveis, clique na guia Atualizações Disponíveis.

    É aberta a página Atualizações Disponíveis.

  3. Na tabela de atualizações disponíveis, selecione os componentes que está atualizando.

    Se não houver diferença na versão ou no tamanho entre a versão instalada e a nova versão do componente, não é necessário atualizar o componente. Se não houver atualizações disponíveis, a tabela estará vazia.

    • Para selecionar um componente individual, marque a caixa de seleção adjacente ao nome do componente.
    • Para selecionar todos os componentes, clique no ícone Selecionar Todos, no cabeçalho da tabela.
    • Para desmarcar todos os componentes, clique no ícone Selecionar Todos, no cabeçalho da tabela Desmarcar Tudo.
  4. Clique em Instalar.
  5. Aceite o contrato de licença.

    O Enterprise Server confirma que a instalação está concluída.

  6. Reinicie o servidor.

    Para obter instruções detalhadas, consulte a seguinte documentação:

    • To Stop a Domain no Sun GlassFish Enterprise Server v3 Administration Guide

    • To Start a Domain no Sun GlassFish Enterprise Server v3 Administration Guide
