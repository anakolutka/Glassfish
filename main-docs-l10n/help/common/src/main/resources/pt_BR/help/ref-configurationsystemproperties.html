Propriedades do Sistema

A página Propriedades do Sistema exibe uma lista das propriedades do sistema Java que estão definidas na configuração nomeada selecionada. Propriedades do sistema Java são especificadas para o acionador da aplicação Java por meio da opção -D do acionador da aplicação Java quando GlassFish Server for iniciado

Estas propriedades definem os valores default das definições da porta de todas as instâncias do GlassFish Server que fazem referência à configuração.

Para obter informações sobre definições de porta predefinida em uma configuração, consulte Propriedades Predefinidas do Sistema.

São exibidas as seguintes informações para cada propriedade:

Nome da Variável da Instância

O nome da propriedade do sistema.

Valor Default

Valor definido para a propriedade na configuração nomeada que a instância faz referência. Este campo é somente para leitura.

Valores da Instância

Um link para a página Valores da Instância para a propriedade.

A tabela Propriedades Adicionais também contém as seguintes opções.

Adicionar Propriedade

Botão para adicionar uma propriedade. Clicar neste botão adiciona uma linha à tabela Propriedades Adicionais.

Deletar Propriedades

Botão para deletar uma ou mais propriedades selecionadas. Qualquer propriedade deletada retorna ao seu valor default ou, se nenhum valor tiver sido definido, fica com valor indefinido.

A página Propriedades do Sistema também contém as seguintes opções:

Nome da Configuração

O nome da configuração à qual as definições nesta página se aplicam. Este campo é somente para leitura.

Reconfiguração Dinâmica

Se esta opção for ativada, são aplicadas alterações na configuração das instâncias que fazem referência à configuração sem a necessidade de reiniciar as instâncias.

Clusters e/ou Instâncias Usando esta Configuração

Lista dos clusters e das instâncias que fazem referência à configuração com uma indicação se cada cluster ou instância está em execução:

  • Se o cluster ou a instância estiver em execução, uma marca de verificação (✓) será exibida.

  • Se a instância ou o cluster for interrompido, um ponto de exclamação (!) será exibido.
