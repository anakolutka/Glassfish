Per installare i componenti aggiuntivi

L'installazione di un componente aggiuntivo richiede le operazioni riportate di seguito.

  • Trasferimento del contenuto e delle dipendenze del componente da un repository di rete

  • Installazione del componente nell'immagine di installazione di GlassFish Server disponibile nel sistema

Se un componente richiede altri componenti che non sono installati, verranno installati anche tali componenti.

  1. Selezionare il nodo Strumento di aggiornamento nell'albero di navigazione.

    Viene visualizzata la pagina Componenti aggiuntivi disponibili.

  2. Nella pagina Componenti aggiuntivi disponibili selezionare i componenti da installare nella tabella dei componenti aggiuntivi disponibili.

    Se non è disponibile alcun componente aggiuntivo, la tabella è vuota.

    • Per selezionare un singolo componente, selezionare la casella di controllo in corrispondenza del nome del componente.

    • Per selezionare tutti i componenti, fare clic sull'icona Seleziona tutto nell'intestazione della tabella.

    • Per deselezionare tutti i componenti, fare clic sull'icona Deseleziona tutto nell'intestazione della tabella.

  3. Fare clic su Installa.

    Viene visualizzata una finestra di dialogo con la richiesta di conferma dell'installazione dei componenti selezionati e di eventuali altri componenti necessari.

  4. Fare clic su OK nella finestra di dialogo.

    Viene richiesto di accettare i contratti di licenza dei componenti da installare.

  5. Accettare i contratti di licenza dei componenti da installare.

    GlassFish Server conferma il completamento dell'installazione. I componenti vengono rimossi dalla tabella dei componenti aggiuntivi e inseriti nella tabella dei componenti installati nella pagina Componenti installati.

  6. Riavviare il server.

    Per istruzioni dettagliate, vedere:

    • To Stop a Domain in Oracle GlassFish Server Administration Guide

    • To Start a Domain in Oracle GlassFish Server Administration Guide
