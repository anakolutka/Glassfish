Per eliminare un servizio contesto

  1. Espandere il nodo Risorse nell'albero di navigazione.

  2. Nel nodo Risorse espandere il nodo Risorse concorrenti.

  3. Nel nodo Risorse concorrenti selezionare Servizi contesto.

    Viene visualizzata la pagina Servizi contesto.

  4. Nella pagina Servizi contesto selezionare la casella di controllo in corrispondenza del nome della risorsa da eliminare.

  5. Fare clic su Elimina.
