To Create an HTTP Load Balancer (Step 2 of 2)

On this page, set the web server communication details that allow automatic updates of the load balancer for the web server. If you do not specify these fields, changes to the load balancer are not automatically copied to the load balancer plug-in. You must copy the load balancer configuration file manually.

  1. Enter the host name of the web server or other load balancer device.

  2. Enter the port used to communicate with the load balancing device. It must be SSL enabled.

  3. Enter the proxy host used for outbound HTTP.

  4. Enter the proxy port used for outbound HTTP.

  5. Click Finish.
