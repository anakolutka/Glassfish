So werden JMS-Zielressourcen bearbeitet

  1. Blenden Sie in der Navigationsbaumstruktur den Knoten "Ressourcen" ein.

  2. Blenden Sie unter dem Ressourcenknoten den Knoten "JMS-Ressourcen" ein.

  3. Wählen Sie unter dem Knoten "JMS-Ressourcen" den Knoten "JMS-Zielressourcen".

    Die Seite "JMS-Zielressourcen" wird geöffnet.

  4. Klicken Sie auf der Seite "JMS-Zielressourcen" auf den Namen der zu bearbeitenden Zielressource.

    Die Seite "JMS-Zielressource bearbeiten" wird geöffnet.

  5. Geben Sie im Feld "Name des physischen Ziels" den Namen des mit dieser Ressource zu verknüpfenden physischen Ziels ein.

    Das physische Ziel wird von GlassFish Server bei Bedarf automatisch erstellt und wieder gelöscht, wenn Sie die Zielressource löschen.

  6. Wählen Sie in der Dropdown-Liste "Ressourcentyp" entweder javax.jms.Topic oder javax.jms.Queue.

  7. Geben Sie im Feld "Beschreibung" eine Beschreibung der Zielressource ein.

  8. Aktivieren Sie das Kontrollkästchen "Status", um die Ressource zur Laufzeit zu aktivieren.

    Diese Option ist standardmäßig aktiviert.

  9. Geben Sie im Abschnitt "Weitere Eigenschaften" zusätzliche Eigenschaften für die Verbindungsressource an.

    In GlassFish Server werden keine weiteren Eigenschaften für JMS-Zielressourcen definiert.

    • Um eine Eigenschaft hinzuzufügen, klicken Sie auf die Schaltfläche "Eigenschaft hinzufügen". Geben Sie in die daraufhin angezeigte leere Zeile den Eigenschaftsnamen in das Feld "Name" und den Eigenschaftswert in das Feld "Wert" ein.

    • Um eine Eigenschaft zu ändern, bearbeiten Sie das Feld "Wert" dieser Eigenschaft.

    • Um eine Eigenschaft zu löschen, aktivieren Sie das Kontrollkästchen links neben dem Feld "Name" der zu löschenden Eigenschaft und klicken anschließend auf die Schaltfläche "Eigenschaften löschen".

  10. Klicken Sie auf "Speichern".

    Tipp:

    Um zu den Standardeinstellungen zurückzukehren, klicken Sie auf die Schaltfläche "Standardwerte laden" und anschließend auf "Speichern".

  11. Klicken Sie alternativ auf die Registerkarte "Ziele", um die Seite "JMS-Zielressourcenziele" aufzurufen. Führen Sie auf dieser Seite die folgenden Aktionen aus:

    1. Klicken Sie auf "Ziele verwalten", um die Seite "Ressourcenziele verwalten" zu öffnen.

      Verschieben Sie auf dieser Seite die Ziele zwischen der Spalte der verfügbaren Ziele und der Spalte der gewählten Ziele. Stellen Sie sicher, dass Sie die Ziele, auf denen Anwendungen, die die Ressource verwenden, bereitgestellt werden sollen, in der Spalte der gewählten Ziele ablegen. Die verfügbaren Ziele schließen die verfügbaren Cluster und Serverinstanzen sowie die Standardserverinstanz server ein. Klicken Sie auf "OK", um die Änderungen zu speichern.

    2. Wählen Sie das Kontrollkästchen für ein Ziel, und klicken Sie anschließend auf "Aktivieren" oder "Deaktivieren", um die Ressource für dieses Ziel zu aktivieren bzw. zu deaktivieren.
