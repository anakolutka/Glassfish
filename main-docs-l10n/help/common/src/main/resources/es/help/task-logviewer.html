Ver archivos log de DAS

  1. En el árbol de navegación, seleccione el nodo del servidor (servidor de administración).

    Se abrirá la página Información General.

  2. En la página Información General, haga clic en el botón Ver Archivos Log.

    Se abre una ventana con el visor de logs.
