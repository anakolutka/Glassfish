JACC 공급자

JACC 공급자 페이지에는 사용 가능한 JACC(Java Authorization Contract for Containers) 공급자의 목록이 표시됩니다.

JACC는 플러그 가능한 권한 부여 공급자의 인터페이스를 정의하는 Java EE 사양의 일부입니다. JACC를 사용하면 관리자는 타사 플러그인 모듈에서 권한 부여를 수행할 수 있도록 설정할 수 있습니다. 기본적으로 Enterprise Server는 간단한 형식으로 된 JACC 호환 파일 기반의 권한 부여 엔진을 제공합니다.

JACC 공급자에 대한 구현 클래스를 작성하는 방법에 대한 자세한 내용은 http://jcp.org/en/jsr/detail?id=115
