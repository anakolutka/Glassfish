Suppression d'instances GlassFish Server autonomes

  1. Dans l'arborescence de navigation, sélectionnez le noeud Instances autonomes.

    La page Instances de serveur autonomes apparaît.

  2. Pour chaque instance à supprimer, cochez la case en regard de l'instance dans la table Instances de serveur.

  3. Cliquez sur Supprimer.

    Vous êtes invité à confirmer que vous souhaitez supprimer les instances.

  4. Confirmez la suppression des instances.

    Les instances sont supprimées et enlevées de la table des instances sur la page.
