Per modificare un modulo del ciclo di vita

  1. Selezionare il nodo Moduli del ciclo di vita nell'albero di navigazione.

    Viene visualizzata la pagina Moduli del ciclo di vita.

  2. Nella pagina Moduli del ciclo di vita fare clic sul nome del modulo del ciclo di vita modificare.

    Viene visualizzata la pagina Modifica modulo ciclo di vita.

  3. Nel campo Nome classe digitare il nome completamente qualificato del file di classe del modulo del ciclo di vita.

    La classe deve implementare l'interfaccia com.sun.appserv.server.LifecycleListener.

  4. Nel campo Classpath digitare il classpath per il modulo del ciclo di vita.

    Il classpath specifica la posizione del modulo del ciclo di vita. Il valore predefinito è domain-dir/applications. Se il modulo si trova già in questa directory (ovvero nel classpath del server), questo campo può essere lasciato vuoto.

  5. Nel campo Ordine di caricamento digitare un valore per specificare l'ordine di caricamento del modulo del ciclo di vita all'avvio.

    I moduli con valori di ordine di caricamento interi più bassi vengono caricati per primi. Il valore può essere compreso tra 101 e il valore MAXINT del sistema operativo. I valori compresi tra 1 e 100 sono riservati.

  6. Nel campo Descrizione digitare la descrizione del modulo del ciclo di vita.

  7. Selezionare la casella di controllo Abilitato dell'opzione Stato per abilitare il modulo del ciclo di vita.

    Questa opzione è abilitata per impostazione predefinita.

  8. Selezionare la casella di controllo In caso di errore di caricamento: Impedisci avvio istanza per impostare la chiusura del server se il caricamento del modulo del ciclo di vita non riesce.

    Questa opzione è disabilitata per impostazione predefinita.

  9. Viene visualizzata la scheda Destinazioni se esistono cluster o istanze server standalone oltre all'istanza server predefinita (server). Selezionare la scheda Destinazioni per visualizzare le destinazioni in cui è stato abilitato il modulo del ciclo di vita. Per istruzioni sulle modalità di abilitazione o disabilitazione di un modulo del ciclo di vita in una o più destinazioni, vedere Per visualizzare le destinazioni di un'applicazione.

  10. Fare clic su Salva.
