最新の安定リリース: GlassFish V2

無料の実運用品質の Java EE 5 Application Server

GlassFish は本番で稼動する、サポートされた無料のオープンソースアプリケーションサーバーで Java EE 5 に準拠しています。サーバーは、Windows、Linux、Solaris、Mac OS などを含む多数のプラットフォーム上で実行されます。これには、Java Persistence API の TopLink Essentials 実装も含まれます。GlassFish V2 リリースはすべて初期の V1 リリースと同じ安定性と機能に加え、多くの新しい機能を提供します。これらには、クラスタリング、インメモリーレプリケーション、Grizzly ベースの HTTP エンジンへのさらなる改良、拡張された管理機能、改良された自己管理機能、高度なパフォーマンスなどが含まれます。

GlassFish V2 を、エンタープライズアプリケーションの開発と配備のために使用して下さい。
