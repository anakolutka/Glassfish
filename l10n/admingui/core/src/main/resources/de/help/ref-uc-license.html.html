Lizenzvereinbarung

Die Lizenzvereinbarungsseite zeigt die Lizenzvereinbarungen aller Komponenten an, die zur Installation oder Aktualisierung ausgewählt sind. Die Lizenzvereinbarung für eine Komponente wird vom Autor der Komponente mit der Komponente geliefert. Verschiedene Komponenten im Netzwerk-Repository könnten unter verschiedenen Lizenzen veröffentlicht werden, wie zum Beispiel:

  • Common Development and Distribution License (CDDL)

  • GNU General Public License (GPL)

  • Apache-Lizenz

  • wxWindows Library Licence

Die Seite "Lizenzvereinbarung" umfasst die folgenden Optionen:

Akzeptieren

Diese Schaltfläche akzeptiert die Bestimmungen der Lizenzen aller ausgewählten Komponenten und setzt die Installation oder Aktualisierung fort.

Ablehnen

Diese Schaltfläche lehnt die Bestimmungen der Lizenzen aller ausgewählten Komponenten ab und wechselt zurück zu der Seite, auf der Sie die Komponenten ausgewählt haben.

Verwandte Aufgaben

  • So aktualisieren Sie installierte Komponenten

  • So installieren Sie Add-On-Komponenten
