Configurer le contrôle du service

  1. Dans l'arborescence de navigation, développez le nœud Configuration.
  2. Sous le nœud Configuration, sélectionnez le nœud Contrôle.

    La page Service de contrôle s'affiche.

  3. Dans la liste déroulante Service HTTP, sélectionnez BAS ou HAUT pour activer le contrôle du service HTTP.

    Dans cette version, il n'existe aucune différence entre les options BAS et HAUT. Pour désactiver le contrôle du service HTTP, sélectionnez OFF. La valeur par défaut est OFF.

  4. Dans la liste déroulante Conteneur Web, sélectionnez BAS ou HAUT pour activer le contrôle du conteneur Web.

    Dans cette version, il n'existe aucune différence entre les options BAS et HAUT. Pour désactiver le contrôle du conteneur Web, sélectionnez OFF. La valeur par défaut est OFF.

  5. Dans la liste déroulante JVM, sélectionnez BAS ou HAUT pour activer le contrôle JVM.

    Dans cette version, il n'existe aucune différence entre les options BAS et HAUT. Pour désactiver le contrôle JVM, sélectionnez OFF. La valeur par défaut est OFF.

  6. Dans la liste déroulante Pool de threads, sélectionnez BAS ou HAUT pour activer le contrôle du pool de threads.

    Dans cette version, il n'existe aucune différence entre les options BAS et HAUT. Pour désactiver le contrôle du pool de threads, sélectionnez OFF. La valeur par défaut est OFF.

  7. Cliquez sur Enregistrer.
