To Enable a Transformation Rule

  1. In the tree view, expand the Web Services node.
  2. Select the web service for which you want to enable a transformation rule.
  3. Click the Transformation tab.

    A list of transformation rules available for this web service end point is displayed.

  4. Select the transformation rule by selecting the check box.
  5. Click the Enable button.

    This enables the transformation rule to be applied to SOAP requests for the web service endpoint.
