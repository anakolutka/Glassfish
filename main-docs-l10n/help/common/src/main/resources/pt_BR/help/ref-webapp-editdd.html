Descritores do Módulo

Use a página Descritores de Módulos para exibir os descritores de módulos de uma aplicação.

A página Descritores de Módulos contém as seguintes opções.

Descritores de Implantação

Uma tabela que exibe o nome da aplicação, o nome do subcomponente (se aplicável) e o nome do arquivo do descritor.
