Supprimer un serveur virtuel

  1. Dans l'arborescence de navigation, développez le nœud Configuration.
  2. Sous le nœud Configuration, développez le nœud Service HTTP.
  3. Sous le nœud Service HTTP, sélectionnez le nœud Serveurs virtuels.
  4. Dans la page Serveurs virtuels, cochez la case située à gauche du nom du serveur virtuel à supprimer.
  5. Cliquez sur Delete.

    Il est possible de supprimer le serveur virtuel __asadmin. Toutefois, cette opération est vivement déconseillée. Si vous envisagez de le faire, copiez d'abord les éléments virtual-server du fichier domain.xml de Enterprise Server à un emplacement sûr afin de pouvoir rétablir les paramètres, si nécessaire.
