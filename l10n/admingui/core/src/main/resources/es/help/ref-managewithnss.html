Network Security Services (NSS) de administración de usuarios

Puede administrar usuarios con la Consola de administración, como se menciona en Página Usuarios de archivo o mediante herramientas NSS. NSS es un conjunto de bibliotecas diseñadas para permitir el desarrollo en distintas plataformas de aplicaciones de cliente y servidor con la seguridad activada. Las aplicaciones generadas con NSS son compatibles con los certificados SSL v2 y v3, TLS, PKCS #5, PKCS #7, PKCS #11, PKCS #12, S/MIME, X.509 v3 y otros estándares de seguridad. Para obtener información detallada, consulte las direcciones siguientes:

  • Network Security Services (NSS) en http://www.mozilla.org/projects/security/pki/nss/

  • Herramientas de seguridad NSS en http://www.mozilla.org/projects/security/pki/nss/tools/

  • Introducción a NSS en http://www.mozilla.org/projects/security/pki/nss/overview.html
