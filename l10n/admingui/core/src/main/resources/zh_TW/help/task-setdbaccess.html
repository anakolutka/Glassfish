設定資料庫存取

大部分應用程式使用關聯式資料庫進行資料的儲存、管理及擷取。

  1. 安裝支援的資料庫產品。

    如需支援之資料庫的清單，請參閱「Sun GlassFish Enterprise Server v3 Prelude Release Notes」。

  2. 為資料庫安裝支援 JDBC API 的驅動程式 (JDBC 驅動程式)。

    請確保已在所有要部署資料來源的伺服器上，安裝了要用於建立資料庫連線的驅動程式。

  3. 請確保網域的伺服器實例可以存取該驅動程式的 JAR 檔案。

    請參閱整合 JDBC 驅動程式。

  4. 建立資料庫。

    通常，應用程式提供者提供了用於建立和寫入資料庫的程序檔。

  5. 為資料庫建立連線池。

    請參閱建立 JDBC 連線池。

  6. 建立指向連線池的 JDBC 資源。

    請參閱建立 JDBC 資源。
