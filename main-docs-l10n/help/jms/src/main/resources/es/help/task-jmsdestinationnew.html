Para crear un recurso de destino JMS

Cuando se crea un recurso de destino JMS, GlassFish Server crea un recurso de objeto de administración para el recurso de destino. Puede visualizar este recurso en el nodo Conectores.

  1. En el árbol de navegación, amplíe el nodo Recursos.

  2. En el nodo Recursos, amplíe el nodo Recursos de JMS.

  3. En el nodo Recursos de JMS, seleccione el nodo Recursos de Destino JMS.

    Se abrirá la página Recursos de Destino JMS.

  4. En la página Recursos de Destino JMS, haga clic en Nuevo.

    Se abrirá la página Nuevo recurso de destino JMS.

  5. En el campo Nombre JNDI, introduzca el nombre JNDI del recurso.

    Es una práctica recomendada utilizar el prefijo de subcontexto de nomenclatura jms/ para recursos de JMS. Por ejemplo: jms/Queue.

  6. En el campo Nombre de Destino Físico, introduzca el nombre del destino físico que se asociará a este recurso.

    GlassFish Server creará el destino físico automáticamente cuando sea necesario y lo suprimirá cuando se suprima el recurso de destino.

  7. En la lista desplegable Tipo de Recurso, seleccione javax.jms.Topic o javax.jms.Queue.

  8. En el campo Descripción, introduzca una descripción del recurso de destino.

  9. Active la casilla de control Estado activado para activar el recurso en tiempo de ejecución.

    Esta opción se encuentra activada por defecto.

  10. En el área de Propiedades Adicionales, indique las propiedades adicionales para el recurso de destino.

    ProductName; no define ninguna propiedad adicional para los recursos de destino JMS.

    Para agregar una propiedad, haga clic en el botón Agregar Propiedad. En la fila en blanco que aparece, introduzca el nombre de propiedad en el campo Nombre y escriba el valor de propiedad en el campo Valor.

  11. En el área Destinos, haga lo siguiente:

    1. En la columna Disponibles, seleccione los destinos en los que se desplegarán las aplicaciones que usen el recurso.

      Los destinos disponibles incluyen los clusters y las instancias de servidor disponibles, así como la instancia de servidor por defecto, llamada server.

    2. Haga clic en Agregar para mover el destino a la columna Seleccionados.

  12. Haga clic en Aceptar.
