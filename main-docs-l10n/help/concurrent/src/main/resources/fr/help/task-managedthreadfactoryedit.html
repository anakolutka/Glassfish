Modification d'une fabrique de threads gérés

  1. Dans l'arborescence de navigation, développez le noeud Ressources.

  2. Sous le noeud Ressources, développez le noeud Ressources simultanées.

  3. Sous le noeud Ressources simultanées, sélectionnez Fabriques de threads gérés.

    La page Fabriques de threads gérés est ouverte.

  4. Sur la page Fabriques de threads gérés, cliquez sur le nom de la ressource à modifier.

    La page Modifier la fabrique de threads gérés est ouverte.

    Remarque:

    Le champ Nom JNDI est en lecture seule. Vous ne pouvez spécifier le nom JNDI qu'à la création d'une ressource. Le champ Nom JNDI logique est également en lecture seule.

  5. Cochez la case Informations de contexte activées pour propager les contextes vers des threads, puis sélectionner les contextes que vous voulez propager.

    Cette option est activée et tous les contextes sont sélectionnés par défaut. Appuyez sur les touches Maj ou Ctrl pendant que vous cliquez pour sélectionner plusieurs contextes. Si cette option est désactivée, les contextes ne sont pas propagés, même s'ils sont sélectionnés.

  6. Cochez la case Activé du champ Statut pour activer la ressource.

    Cette option est activée par défaut.

  7. Dans le champ Priorité des threads, saisissez la priorité à affecter aux threads créés.

    La valeur par défaut est 5.

  8. Dans le champ Ordre de déploiement, saisissez un nombre pour indiquer l'ordre de déploiement de la ressource.

    Les ressources avec les plus petits numéros sont chargés en premier au démarrage du serveur. La valeur par défaut est 100.

  9. Dans le champ Description, entrez une brève description de la ressource.

    Cette description doit se présenter sous la forme d'une chaîne pouvant inclure un maximum de 250 caractères.

  10. Dans la section Autres propriétés, n'ajoutez aucune propriété supplémentaire.

    GlassFish Server ne définit aucune autre propriété pour ce type de ressource. De plus, ce type de ressource n'utilise actuellement aucune propriété supplémentaire.

  11. Cliquez sur Enregistrer.
