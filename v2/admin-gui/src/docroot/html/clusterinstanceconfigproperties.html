To Set Instance-Specific Configuration Properties

Default values of instance-specific configuration properties are defined in the configuration that is bound to this instance. If no override value is set, the default value is used.

  1. In the tree component, expand the Clusters node.
  2. Expand the node for the cluster that contains the server instance.
  3. Select the server instance node.
  4. In the General Information page, click the Properties tab.
  5. If necessary, click the Configuration Properties tab.
  6. In the Configuration Properties table, set the configuration properties to the values that you require.
    • To override the default value of a property, type the override value in the Override Value field.
    • To revert a property to the default value, remove the override value.
  7. Click Save.
