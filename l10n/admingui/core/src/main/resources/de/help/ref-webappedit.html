Webanwendung

Auf der Seite "Webanwendungen" können Sie eine vorhandene Webanwendung ändern. Die Seite enthält die folgenden Optionen:

Name

Schreibgeschütztes Feld mit dem Namen der Webanwendung, die Sie bearbeiten.

Kontext-Root

Gibt den Pfad zu der Webanwendung an. Im URL der Webanwendung folgt der Kontext-Root-Wert gleich hinter der Portnummer (http://Host:Port/Kontext-Root/...). Die Kontext-Root sollte mit einem Schrägstrich beginnen, wie z. B. /hallo.

Virtuelle Server

Die mit dieser Webanwendung verbundenen virtuellen Server.

Beschreibung

Beschreibung für die Webanwendung.

Status Aktiviert

Wenn diese Option ausgewählt wird, ist die Anwendung aktiviert. Diese Option ist standardmäßig aktiviert.

Ort

Schreibgeschütztes Feld mit dem Speicherort der Archivdatei, aus der die Anwendung bereitgestellt wurde.
