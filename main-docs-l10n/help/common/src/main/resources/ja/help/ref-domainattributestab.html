ドメイン属性

GlassFish Serverに関するドメインの詳細属性を設定するには、「ドメイン属性」ページを使用します。

「ドメイン属性」ページには、次のオプションが含まれます。

DAS起動後にコンソールをロード

管理コンソールをロードするタイミングを決定します。

使用状況ベース

コンソールが過去24時間以内に使用された場合、またはドメイン管理サーバー(DAS)以外の他のサーバー・インスタンスが存在する場合、DAS起動後に管理コンソールをロードします。これはデフォルトです。

常時

DAS起動後に管理コンソールを常にロードします。コンソールは、GlassFish Serverの主要な起動順序の完了後にロードされます。

なし

DAS起動後に管理コンソールをロードしません。コンソールは、アクセスがリクエストされたときにロードされます。

アプリケーション・ルート

デプロイされたアプリケーションが存在するディレクトリのフルパス。デフォルト値はdomain-dir/applicationsです。

ログ・ルート

サーバーのログ・ファイルを保持する場所を識別します。デフォルト値はdomain-dir/logsです。

ロケール

このフィールドを空白のままにすると、ホストのデフォルトのロケールが使用されます。

ロケールは、言語と地域の特定の組合せを指定する識別子です。たとえば、米国英語のロケールはen_USで、日本語のロケールはja_JPです。英語以外のロケールを使用するには、英語を目的の言語に翻訳して、GlassFish Serverをローカライズする必要があります。
