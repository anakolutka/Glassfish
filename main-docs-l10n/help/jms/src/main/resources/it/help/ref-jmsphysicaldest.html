Destinazioni fisiche JMS

Gli oggetti di destinazione fisica del servizio JMS (Java Message Service) vengono gestiti dai broker Message Queue. La coda denominata mq.sys.dmq è la destinazione di sistema alla quale vengono reindirizzati i messaggi scaduti e non recapitabili.

Utilizzare la pagina Destinazioni fisiche JMS per configurare le destinazioni fisiche di Message Queue.

Per ogni destinazione fisica vengono fornite le informazioni riportate di seguito.

Nome

Il nome della destinazione fisica.

Tipo

Il tipo della destinazione fisica.

Statistiche

Le statistiche relative alla destinazione fisica.

La tabella Destinazioni contiene inoltre le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare una nuova destinazione fisica.

Elimina

Pulsante che consente di eliminare una o più destinazioni fisiche selezionate.

Elimina tutti i messaggi

Pulsante che consente di rimuovere i messaggi da una o più destinazioni fisiche selezionate.
