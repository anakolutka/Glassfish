Servicio de Conector

Utilice la página Servicio de conector para modificar la configuración general del servicio de conector, que controla los adaptadores de recursos.

La página Servicio de conector contiene las siguientes opciones.

Cargar Valores por Defecto

Botón para restaurar configuración que tiene valores por defecto a sus valores por defecto. Las configuraciones que no tienen valores por defecto no se modifican.

Nombre de Configuración

Nombre de la configuración a la que se aplican los valores de esta página. Este campo es de sólo lectura.

Timeout de cierre

Número máximo de segundos permitidos durante el cierre de GlassFish Server para que se complete el método ResourceAdapter.stop de una instancia del módulo del conector. Los adaptadores de recursos que tarden más en cerrarse se ignoran y continúa el cierre de GlassFish Server. El valor por defecto es 30 segundos.

Política de carga de clases del conector

Política que se utiliza para cargar las clases. Las opciones disponibles son:

derivada

Indica que los adaptadores de recursos se proporcionan en función de las referencias de una aplicación a cualquiera de los recursos del adaptador de recursos. Se trata de la política por defecto.

global

Indica que todos los adaptadores de recursos serán visibles en todas las aplicaciones.

Esta política no se aplica al adaptador de recursos del sistema JMS preinstalado, jmsra, que es siempre visible en todas las aplicaciones.
