GlassFish Server 블로그를 보는 방법

GlassFish Server 블로그는 GlassFish Server 커뮤니티에서 제공하는 뉴스 및 GlassFish Server 추가 정보에 대한 링크를 제공합니다.

  1. 탐색 트리에서 [일반 작업] 노드를 선택합니다.

    [일반 작업] 페이지가 열립니다.

  2. GlassFish 뉴스 아래에서 [블로그] 링크를 누릅니다.

    GlassFish Server 블로그가 별도의 브라우저 탭이나 창에서 열립니다.
