배포된 웹 응용 프로그램을 재배포하는 방법

웹 응용 프로그램을 재배포할 경우 응용 프로그램을 원래 배포했던 설정 및 기타 옵션을 유지한 상태에서 새 버전의 응용 프로그램을 배포합니다.

시작하기 전에

재배포할 웹 응용 프로그램이 이미 배포되었는지 확인합니다.

  1. 탐색 트리에서 응용 프로그램 노드를 선택합니다.
  2. 응용 프로그램 노드에서 웹 응용 프로그램 노드를 선택합니다.

    웹 응용 프로그램 페이지가 열립니다.

  3. 배포된 웹 응용 프로그램 테이블에서 재배포할 응용 프로그램의 행에 있는 재배포를 누릅니다.

    엔터프라이즈 응용 프로그램/모듈 재배포 페이지가 열립니다.

  4. 응용 프로그램의 활성 세션을 유지한 다음 재배포가 완료될 때 복구해야 하는 경우 세션 유지 확인란을 선택합니다.

    응용 프로그램의 활성 세션을 보존 또는 복구하지 못할 경우 재배포가 완료될 때 모든 세션을 사용할 수 없습니다. 그러나 재배포가 계속되고 경고가 기록됩니다.

  5. 재배포할 응용 프로그램이 포함된 아카이브의 위치를 지정합니다.
    • 아카이브가 클라이언트 시스템에 있거나 액세스 가능한 파일인 경우 서버에 업로드할 패키지화된 파일 옵션을 선택합니다.

      클라이언트 시스템은 브라우저를 통해 관리 콘솔을 보는 호스트입니다.

      찾아보기를 눌러 파일을 찾거나, 파일의 전체 경로를 입력합니다.

    • 아카이브가 서버 시스템에 있는 파일이거나 확장된 디렉토리에서 압축 해제된 응용 프로그램인 경우 Application Server에서 액세스할 수 있는 패키지화된 로컬 파일 또는 디렉토리 옵션을 선택합니다.

      서버 시스템은 도메인 관리 서버(Domain Administration, DAS)를 실행하는 호스트입니다.

      파일 또는 디렉토리의 전체 경로 이름을 입력합니다. 확장된 디렉토리에서 배포하는 것은 고급 개발자 작업이며 프로덕션 환경에는 권장하지 않습니다.

  6. 확인을 누릅니다.
참조

  • 웹 응용 프로그램을 배포하는 방법

  • 배포된 웹 응용 프로그램을 편집하는 방법

  • 웹 응용 프로그램의 설명자를 보는 방법

  • 웹 응용 프로그램을 시작하는 방법
