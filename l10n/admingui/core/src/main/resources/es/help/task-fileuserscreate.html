Para agregar un usuario

  1. Haga clic en Nuevo para agregar un usuario nuevo al dominio file.

    Se abrirá la página Nuevo usuario de dominio file.

  2. En el campo Id. de usuario, introduzca el nombre del usuario. Se trata de un campo obligatorio.
  3. En el campo Contraseña, introduzca una contraseña exclusiva. Se trata de un campo obligatorio.
  4. Vuelva a confirmar la contraseña del usuario para su verificación. Se trata de un campo obligatorio.
  5. En el campo Lista de grupos, introduzca la lista de grupos separada con comas a la que el usuario pertenece. Estos grupos no hace falta definirlos en otro lugar.
  6. Seleccione Aceptar para agregar a este usuario a la lista de usuarios en el dominio file, o en Cancelar para salir sin guardar.
