HTTP リスナーを削除する

始める前に

デフォルトの HTTP リスナーである http-listener-1、http-listener-2、または admin-listener を削除する場合は、必要に応じて設定を復元できるように、最初に Enterprise Server の domain.xml ファイルの http-listener 要素のバックアップコピーを作成します。

  1. ナビゲーションツリーで、「設定」ノードを展開します。
  2. 「設定」ノードの下で、「HTTP サービス」ノードを展開します。
  3. 「HTTP サービス」ノードで、「HTTP リスナー」ノードを選択します。
  4. 「HTTP サービス」ページで、削除する HTTP リスナーの名前の隣にあるチェックボックスを選択します。
  5. 「削除」をクリックします。
