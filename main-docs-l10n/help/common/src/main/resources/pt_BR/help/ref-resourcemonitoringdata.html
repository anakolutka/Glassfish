Monitoramento de Recursos

Usar a página Monitoramento de Recursos para exibir os dados de monitoramento de um recurso selecionado.

A página Monitoramento de Recursos contém as seguintes informações.

Nome da Instância

O nome da instância do servidor.

Pool de Conexões

O pool de conexões cujas estatísticas de monitoramento são exibidas.

Para obter uma descrição das propriedades monitoradas, consulte Capítulo 8, Administering the Monitoring Service em Oracle GlassFish Server Administration Guide.
