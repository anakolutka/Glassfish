Per impostare l'accesso a un sistema EIS

  1. Distribuire (installare) un connettore. Vedere Per distribuire un modulo connettore.

  2. Creare un connection pool per il connettore. Vedere Per creare un connection pool del connettore.

  3. Creare una risorsa connettore associata al connection pool. Vedere Per creare una risorsa connettore.
