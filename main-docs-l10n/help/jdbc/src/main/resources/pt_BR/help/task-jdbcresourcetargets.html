Para Editar Alvos do Recurso do JDBC

  1. Na árvore de navegação, expanda o nó Recursos.

  2. Expanda o nó JDBC.

  3. Expanda o nó Recursos JDBC.

  4. Selecione o recurso do JDBC que deseja editar.

  5. Na página Editar Recurso do JDBC, selecione a guia Alvos

  6. Para designar um novo alvo ao recurso, selecione o botão Gerenciar Alvos.

  7. Na página Gerenciar Alvos, selecione um alvo disponível para adicioná-lo ou removê-lo da lista Alvos selecionados.

  8. Clique em OK para aplicar as edições.
