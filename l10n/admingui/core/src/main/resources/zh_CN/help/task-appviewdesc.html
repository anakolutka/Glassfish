查看应用程序描述符

  1. 在导航树中，选择“应用程序”节点。

    此时将打开“应用程序”页面。

  2. 在“应用程序”页面上，单击要查看其描述符的应用程序的名称。

    此时将打开“编辑应用程序”页面。

  3. 在“编辑应用程序”页面上，单击“描述符”选项卡。

    此时将打开“模块描述符”页面。

  4. 在“部署描述符”表中，单击要查看的 XML 文件的名称。

    此时将在新窗口中打开 XML 文件。
