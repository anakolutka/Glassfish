編輯已部署的應用程式

  1. 在瀏覽樹狀結構中，選取 [應用程式] 節點。

    [應用程式] 頁面會隨即開啟。

  2. 從 [已部署的應用程式] 表格中，按一下您要編輯的應用程式名稱。

    [編輯應用程式] 頁面會隨即開啟。

  3. 若為 Web 應用程式，請在 [環境根目錄] 欄位中，鍵入該應用程式的路徑。

    在 Web 應用程式的 URL 中，環境根目錄緊跟在連接埠號碼之後 (http://host: port/context-root/...)。請確保環境根目錄的開頭是正斜線，例如 /hello。

  4. 在 [說明] 欄位中，鍵入此應用程式的說明。
  5. 取消選取 [啟用狀態] 核取方塊，以停用應用程式狀態，使其無法使用。

    應用程式在部署後的狀態預設為 [已啟用]。

  6. 選取 [啟用 Java Web Start] 核取方塊，以使用 Java Web Start 軟體來部署應用程式。
  7. 按一下 [儲存]。
