Per creare una risorsa JNDI esterna

Una risorsa JNDI esterna accede a un repository JNDI esterno, ad esempio un server LDAP.

  1. Espandere il nodo Risorse nell'albero di navigazione.

  2. Nel nodo Risorse espandere il nodo JNDI.

  3. Nel nodo JNDI selezionare Risorse esterne.

    Viene visualizzata la pagina Risorse esterne.

  4. Nella pagina Risorse esterne fare clic su Nuovo.

    Viene visualizzata la pagina Nuova risorsa esterna.

  5. Nel campo Nome JNDI, digitare un nome univoco.

    Il nome digitato verrà registrato nel servizio di denominazione JNDI. Il nome JNDI consente di organizzare e individuare i componenti all'interno di un ambiente di elaborazione distribuito, in modo analogo agli schedari in una biblioteca, con cui viene organizzata e rappresentata la posizione dei libri. Il nome JNDI diventa quindi un metodo importante per accedere alla risorsa.

  6. Nel campo Tipo di risorsa effettuare una delle operazioni riportate di seguito.

    • Selezionare una definizione di tipo dall'elenco a discesa.

    • Selezionare il secondo pulsante di scelta e digitare una definizione di tipo completamente qualificata nel formato . xxx.

  7. Nel campo Classe factory digitare un repository esterno di classe factory JNDI per la risorsa esterna da creare.

    Classe factory è il nome specificato dall'utente per la classe di factory, ad esempio com.sun.jndi.ldap. Questa classe implementa l'interfaccia javax.naming.spi.InitialContextFactory.

  8. Nel campo Ricerca JNDI digitare il valore JNDI per eseguire ricerche nel repository esterno.

    Ad esempio, quando si crea una risorsa esterna da collegare a un repository esterno, per eseguire i test su una classe di bean, la funzione Ricerca JNDI può avere l'aspetto seguente: cn=testmybean.

  9. Nel campo Descrizione digitare una breve descrizione della risorsa.

    La descrizione è un valore stringa con lunghezza limitata a 250 caratteri.

  10. Selezionare la casella di controllo Abilitato dell'opzione Stato per abilitare la risorsa.

    Per impostazione predefinita la risorsa è abilitata, ovvero disponibile, non appena creata.

  11. Nella sezione Proprietà aggiuntive specificare le proprietà aggiuntive per la risorsa esterna.

    GlassFish Server non definisce alcuna proprietà aggiuntiva per le risorse esterne.

    Per aggiungere una proprietà, fare clic sul pulsante Aggiungi proprietà. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Nome e il valore della proprietà nel campo Valore.

  12. Fare clic su OK.
