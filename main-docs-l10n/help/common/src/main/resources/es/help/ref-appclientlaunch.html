Inicio de cliente de aplicaciones

Utilice la página Inicio del Cliente de Aplicaciones para iniciar un cliente de aplicaciones desplegado. Debe ser un cliente web que se pueda ejecutar con Java Web Start.

La página Iniciar Cliente de Aplicaciones contiene las opciones siguientes.

Aplicación

Nombre de la aplicación que contiene el cliente de aplicaciones.

El nombre puede incluir un identificador de versión opcional, que va a continuación del nombre y está separado de él por dos puntos (:). Para obtener más información sobre las versiones de módulo y de aplicación, consulte el apartado sobre las versiones de módulo y de aplicación en Oracle GlassFish Server Application Deployment Guide.

Módulo

Nombre del módulo que contiene el cliente de aplicaciones.

Enlaces

URL en que se despliega el cliente.

Argumentos

Argumentos que se deben pasar a la aplicación.
