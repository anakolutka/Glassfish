Webanwendungen

Auf der Seite "Webanwendungen" wird eine Liste von Webanwendungen angezeigt, die auf dem Anwendungsserver bereitgestellt werden. Für jede Anwendung werden die folgenden Informationen zur Verfügung gestellt:

  • Name

  • Aktiviert

  • Kontext-Root

  • Aktions-Links

Verwandte Aufgaben

  • So stellen Sie eine Webanwendung bereit

  • So bearbeiten Sie eine bereitgestellte Webanwendung

  • So starten Sie eine Webanwendung

  • So stellen Sie eine bereitgestellte Webanwendung erneut bereit
