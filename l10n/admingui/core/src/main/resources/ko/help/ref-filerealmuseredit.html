파일 영역 사용자 편집

파일 영역 사용자 편집 페이지를 사용하여 현재 영역의 사용자 액세스 권한을 수정합니다.

영역 이름

현재 선택한 영역의 이름을 표시하는 읽기 전용 필드입니다.

사용자 ID

편집 중인 계정의 이름을 표시하는 읽기 전용 필드입니다.

그룹 목록

이 사용자가 속할 그룹입니다. 여러 그룹 이름은 쉼표로 구분합니다.

새 비밀번호

이 사용자의 새 비밀번호입니다.

새 비밀번호 확인

이 사용자의 새 비밀번호입니다.

관련 작업

  • 파일 사용자를 관리하는 방법

  • 사용자를 추가하는 방법

  • 사용자 정보를 편집하는 방법

  • 사용자를 삭제하는 방법
