ファイルユーザーを管理する

ファイルユーザーの管理は、security レルムノードを介して Enterprise Server で行われます。

  1. ナビゲーションツリーで、「設定」ノードを展開します。
  2. ドメインがクラスタをサポートするように設定されている場合は、変更する設定を選択します。

    たとえば、クラスタ プロファイルや エンタープライズ プロファイルで作成されるドメインなどです。

  3. 「セキュリティー」ノードを開きます。
  4. 「レルム」ノードを開きます。
  5. 「file」ノードを選択します。
  6. 「レルムを編集」ページで、「ユーザーを管理」ボタンをクリックします。

    「ファイルユーザー」ページが開きます。このページで、次のタスクを実行します。

    • 「ユーザーを追加するには」

    • 「ユーザー情報を編集する」

    • 「ユーザーを削除するには」
