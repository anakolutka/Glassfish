Crear un recurso para una instancia de GlassFish Server independiente

  1. En el árbol de navegación, seleccione el nodo Instancias Independientes.

    Se abrirá la página Instancias de Servidor Independientes.

  2. En la página Instancias de Servidor Independientes, seleccione la instancia.

    Se abrirá la página Información General para la instancia.

  3. En la página Información General, haga clic en el separador Recursos.

    Se abrirá la página Recursos para la instancia.

  4. En el menú desplegable Nuevo, seleccione el tipo de recurso que vaya a crear.

    Se abrirá la página para crear un recurso del tipo seleccionado.

  5. En la página para crear un recurso del tipo seleccionado, realice los cambios oportunos y guárdelos.

    Para obtener más información, consulte la ayuda en pantalla de la página para crear un recurso del tipo seleccionado.

    Nota:

    Cuando cree el recurso, asegúrese de especificar la instancia como destino.
