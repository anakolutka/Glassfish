接続プール

「接続プール」ページには、次のオプションが含まれます。

最大保留カウント

HTTP リスナーに対して許可する保留接続の最大数を指定します。デフォルト値は 4096 です。

キューサイズ

接続キューの最大サイズをバイト単位で指定します。この値は、サーバーが保持できる未処理接続の最大数も指定します。デフォルト値は 4096 です。

受信バッファーサイズ

HTTP リスナーの受信バッファーのサイズをバイト単位で指定します。デフォルト値は 4096 です。

送信バッファーサイズ

HTTP リスナーの送信バッファーのサイズをバイト単位で指定します。デフォルト値は 8192 です。
