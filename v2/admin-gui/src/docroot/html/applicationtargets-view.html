To view application targets

  1. In the tree component, expand the Applications node.

  2. Select the node for the type of application or module for which to view descriptors.

  3. Select the node for the deployed application or module.

  4. Open the Targets tab. The targets currently specified for the application appear in this page.

  5. To enable a target, click the check box next to the application and from the More Actions drop-down list, select Enable.

  6. To enable a target to be load balanced, click the check box next to the application and from the More Actions drop-down list, select Load Balancer Enable.
