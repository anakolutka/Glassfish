Protocolos

Utilice la página Protocolos para configurar los protocolos de red.

Para cada protocolo, se muestra la siguiente información.

Nombre

Nombre del protocolo.

Seguridad Activada

True si la seguridad está activada para el protocolo, o false si no está activada.

La tabla Protocolos contiene además las siguientes opciones.

Nuevo

Botón para crear un nuevo protocolo.

Suprimir

Botón para suprimir uno o varios protocolos seleccionados.
