Web 應用程式

使用 [Web 應用程式] 頁面可修改現有的 Web 應用程式。此頁面包含下列選項：

名稱

顯示正在編輯之 Web 應用程式名稱的唯讀欄位。

環境根目錄

指定 Web 應用程式的路徑。在 Web 應用程式的 URL 中，環境根目錄緊跟在連接埠號碼之後 (http://host:port/context-root/...)。環境根目錄的開頭必須是正斜線，例如 /hello。

虛擬伺服器

與此 Web 應用程式相關的虛擬伺服器。

說明

Web 應用程式的說明。

啟用狀態

若選取此選項，則會啟用 Web 應用程式。此選項預設為選取。

位置

顯示部署應用程式之來源歸檔檔案位置的唯讀欄位。
