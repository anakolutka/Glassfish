Add Domain Dialog

Use the Add Domain dialog to select the domain name and the source and target certificate passwords.

  1. Click on the Source Domain pulldown and select the domain that contains the security certificates to migrate.
  2. In the Passwords panel, enter the passwords you are prompted to enter. Depending on the type of source and target installation, either two or three passwords may need to be entered:
    • Enter the passwords for the NSS certificate files, used for NSS trusted database, in the Source NSS and/or Target NSS fields.
    • Enter the password for the source user key database, used for the JSSE keystore.jks file that contains user keypair certificates, in the User Key Database field.
    • Enter the password for the certificate authority database, which is used for the JSSE cacerts.jks file that contains certificate authority certificates, in the Certificate Authority Database field.
  3. Press the OK button when done. The Transfer Security Certificates screen will display, showing the domain that was added in the Domains with Certificates to be Transferred panel.
