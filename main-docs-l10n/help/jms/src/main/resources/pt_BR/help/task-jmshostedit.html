Para Editar um Host JMS

  1. Na árvore de navegação, expanda o nó Configuração.

  2. Selecione a instância que será configurada:

    • Para configurar uma instância específica, selecione o nó de configuração da instância. Por exemplo, para a instância default, server, selecione o nó server-config.

    • Para configura as definições default das futuras instâncias que utilizam uma cópia de default-config, selecione o nó default-config.

  3. No nó Configuração, expanda o nó do Serviço JMS.

  4. No nó do Serviço JMS, selecione o nó Hosts JMS.

    A página Hosts JMS é aberta.

  5. Na página Hosts JMS, selecione o host que está sendo editado.

    A página Editar Host JMS é aberta.

  6. No campo Host, digite o nome ou o endereço do Protocolo Internet (IP) do sistema em que o host JMS será executado (localhost ou o nome do sistema local ou remoto).

  7. No campo Porta, digite o número da porta do serviço JMS.

    Altere esse campo somente se o serviço JMS a ser utilizado estiver sendo executado em uma porta não default. A porta default é 7676.

  8. Nos campos Nome de Usuário e Senha Admin, digite o nome de usuário e a senha do broker do Message Queue.

    São diferentes do nome de usuário e da senha do GlassFish Server. Edite esses campos somente se os valores do broker do Message Queue tiverem sido alterados com o comando as-install-parent/mq/bin/imqusermgr. Os valores default são admin e admin.

  9. Clique em Salvar.

    Dica:

    Para voltar às definições default, clique no botão Carregar Defaults e depois em Salvar.
