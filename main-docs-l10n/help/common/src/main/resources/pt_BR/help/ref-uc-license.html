Contrato de Licença

A página Contrato de Licença exibe os contratos de licença de todos os componentes selecionados para instalação ou atualização. O contrato de licença é fornecido com o componente, pelo autor do componente. Componentes diferentes em Repositório de Rede podem ser publicados nas diferentes licenças, por exemplo:

  • Common Development and Distribution License (CDDL) (http://www.sun.com/cddl/)

  • Licença Pública Geral GNU (GPL) (http://www.gnu.org/copyleft/gpl.html)

  • Licença do Apache (http://www.apache.org/licenses/)

  • Licença da Biblioteca wxWindows (http://www.opensource.org/licenses/wxwindows.php)

A página Contrato de Licença contém as seguintes opções.

Aceitar

Este botão aceita os termos da licença de todos os componentes selecionados e dá continuidade à instalação ou upgrade.

Recusar

Este botão recusa os termos da licença de todos os componentes selecionados e volta para a página em que os componentes foram selecionados.
