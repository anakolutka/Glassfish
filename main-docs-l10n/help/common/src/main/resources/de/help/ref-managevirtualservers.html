Virtuelle Serverziele

Verwenden Sie die Seite "Virtuelle Serverziele", um virtuelle Serverziele zu ändern, mit denen eine Anwendung oder ein Modul verknüpft ist.

Die Seite "Virtuelle Serverziele" enthält die folgenden Informationen.

Verfügbare Ziele

Die virtuellen Server, mit denen keine Anwendungen oder Module verknüpft sind.

Ausgewählte Ziele

Die virtuellen Server, mit denen Anwendungen oder Module verknüpft sind.

Hinzufügen

Schaltfläche zum Verschieben des ausgewählten Ziels von "Verfügbare Ziele" nach "Ausgewählte Ziele".

Alle hinzufügen

Schaltfläche zum Verschieben aller Ziele von "Verfügbare Ziele" nach "Ausgewählte Ziele".

Entfernen

Schaltfläche zum Verschieben des ausgewählten Ziels von "Ausgewählte Ziele" nach "Verfügbare Ziele".

Alle entfernen

Schaltfläche zum Verschieben aller Ziele von "Ausgewählte Ziele" nach "Verfügbare Ziele".
