Tarefas Comuns

A página Tarefas Comuns fornece atalhos para as tarefas comuns do Console de Administração.

Use a página Tarefas Comuns para as seguintes operações:

  • Para registrar o Sun GlassFish Enterprise Server com conexão Sun

  • Para exibir a página Notícias do GlassFish do Sun GlassFish Enterprise Server

  • Para exibir a página Inscrições do Sun GlassFish Enterprise Server

  • Para exibir uma lista de aplicativos implantados

  • Para implantar um aplicativo da Web

  • Para implantar um Aplicativo Empresarial

  • Para implantar um cliente aplicativo

  • Para implantar um módulo conector

  • Para implantar um EJB Jar

  • Para desinstalar componentes

  • Para instalar componentes complementares

  • Para instalar atualizações de componentes

  • To Create a JDBC Connection Pool

A página Tarefas Comuns também fornece links para alguns dos documentos do Enterprise Server
