Nuevo Recurso de Objeto Administrado

Utilice la página Nuevo recurso de objeto administrado para crear un nuevo recurso de objeto administrado.

La página Nuevo recurso de objeto administrado contiene las siguientes opciones.

Nombre JNDI

Nombre único que identifica el recurso de objeto administrado.

Adaptador de recursos

Adaptador de recursos para el recurso de objeto administrado.

Tipo de recurso

Tipo Java del recurso de objeto administrado (por ejemplo, javax.jms.Topic).

Nombre de clase

Nombre de clase de implantación asociado al tipo de recurso.

Descripción

Descripción del recurso de objeto administrado.

Estado

Si esta opción está seleccionada, el recurso de objeto administrado está activado. Esta opción se encuentra activada por defecto.

Propiedades Adicionales

Propiedades adicionales para el recurso de objeto administrado. Las propiedades disponibles dependen del adaptador de recursos para el pool de conexiones de conector. Los pares nombre-valor especificados en esta tabla se pueden usar para sustituir los valores por defecto de las propiedades definidas por el proveedor del adaptador de recursos.

Destinos

(Esta opción sólo se muestra si en el dominio se ha creado algún cluster o instancia independiente).

Clusters e instancias independientes para el recurso. Mueva los destinos seleccionados a la columna Destinos Seleccionados mediante los botones Agregar y Agregar Todos. Mueva los destinos que no necesite a la columna Destinos Disponibles mediante los botones Eliminar y Eliminar Todos.
