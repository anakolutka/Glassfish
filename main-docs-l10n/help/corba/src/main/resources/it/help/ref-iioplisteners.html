Listener IIOP

Utilizzare la pagina Listener IIOP per configurare i listener IIOP.

Per un broker ORB possono essere configurati più listener IIOP, ognuno dei quali accetta le connessioni su combinazioni di porta e indirizzo di rete diverse. I listener possono essere configurati in modo diverso per garantire comunicazioni e autenticazioni client sicure.

Per impostazione predefinita, sono configurati tre listener IIOP: orb-listener-1, SSL e SSL_MUTUALAUTH.

Per ciascun listener IIOP la pagina Listener IIOP contiene le informazioni riportate di seguito.

Nome

Il nome del listener IIOP.

Abilitato

L'impostazione è True se il listener IIOP è abilitato oppure è False se il listener IIOP è disabilitato.

Indirizzo di rete

L'indirizzo IP (Internet Protocol) o il nome host risolvibile da DNS per il listener IIOP.

Porta listener

Il numero di porta del listener IIOP.
