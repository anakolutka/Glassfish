Para Criar um Recurso do JDBC

Um recurso de Java DataBase Connectivity (fonte de dados) fornece aplicações com todos os meios para conectar-se a um banco de dados.

Antes de Começar

Para criar um recurso do JDBC, crie primeiro um pool de conexões JDBC. Consulte Para Criar um Pool de Conexões do JDBC.

  1. Na árvore de navegação, expanda o nó Recursos.

  2. No nó Recursos, expanda o nó JDBC.

  3. No nó JDBC, selecione o nó Recursos JDBC.

    A página Recursos JDBC será aberta.

  4. Na página Recursos JDBC, clique em Novo.

    A página Novo Recurso do JDBC será aberta.

  5. No campo Nome JNDI, digite um nome de recurso exclusivo.

    O nome JNDI organiza e localiza os componentes em um ambiente de computação distribuído, de maneira semelhante ao catálogo de cartão que organiza e representa os locais de livros em uma biblioteca. Consequentemente, o nome JNDI se torna um método importante de acesso ao recurso JDBC. Por convenção, o nome começa com a string jdbc/, por exemplo, jdbc/payrolldb. Não se esqueça de usar a barra para a direita.

    O nome deve conter somente caracteres alfanuméricos, sublinhado, traço ou ponto.

  6. Na lista drop-down Nome do Pool, selecione um pool de conexões a ser associado ao novo recurso JDBC.

  7. No campo Descrição, digite uma descrição breve do novo recurso do JDBC.

  8. Marque a caixa de seleção Status Ativado para tornar o recurso disponível durante o runtime.

    Por default, a opção está ativada.

  9. Na seção Propriedades Adicionais, especifique as propriedades adicionais.

    O GlassFish Server não define nenhuma propriedade adicional para recursos JDBC.

    Para adicionar uma propriedade, clique no botão Adicionar Propriedade. Na linha em branco que for exibida, digite o nome da propriedade no campo Nome e digite o valor da propriedade no campo Valor.

  10. Selecione um Alvo Disponível para adicioná-lo ou removê-lo da lista Alvos Selecionados.

  11. Clique em OK.
