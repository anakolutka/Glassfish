Um recurso JNDI personalizado especifica uma fábrica de objetos de recursos personalizada para todos os servidores.

  1. Na árvore de navegação, expanda o nó Recursos.

  2. No nó Recursos, expanda o nó JNDI.

  3. No nó JNDI, selecione Recursos Personalizados.

    A página Recursos Personalizados é aberta.

  4. Na página Recursos Personalizados, clique em Novo.

    Abre-se a página Novo Recurso Personalizado.

  5. No campo Nome da JNDI, digite um nome exclusivo.

    Esse nome será registrado no serviço atribuição de atribuição de nomes de JNDI. O nome JNDI organiza e localiza os componentes em um ambiente de computação distribuído, de maneira semelhante ao catálogo de cartão que organiza e representa os locais de livros em uma biblioteca. Por conseguinte, o nome JNDI se torna um método importante de acesso ao recurso personalizado.

  6. No campo Tipo de Recurso, efetue um dos procedimentos a seguir.

    • Na lista drop-down, selecione uma definição de tipo.

    • Selecione o segundo botão de opção e digite uma definição de tipo totalmente qualificada que siga o formato xxx.xxx.

  7. No campo Classe de Fábrica, digite um nome de classe de fábrica para o recurso personalizado a ser criado.

    A Classe de Fábrica é o nome especificado de usuário para a classe de fábrica. Essa classe implementa a interface javax.naming.spi.ObjectFactory.

  8. No campo Descrição, digite uma descrição curta do recurso.

    Esta descrição é um valor de string e pode incluir no máximo 250 caracteres.

  9. Marque a caixa de seleção Status ativo para ativar o recurso.

    Por default, o recurso está ativado, significando que está disponível assim que é criado.

  10. Na seção Propriedades Adicionais, especifique propriedades adicionais do recurso personalizado.

    O GlassFish Server não define propriedades adicionais para recursos personalizados.

    Para adicionar uma propriedade, clique no botão Adicionar Propriedade. Na linha em branco que for exibida, digite o nome da propriedade no campo Nome e digite o valor da propriedade no campo Valor.

  11. Clique em OK.
