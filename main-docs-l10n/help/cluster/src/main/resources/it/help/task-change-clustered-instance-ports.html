Per modificare i numeri di porta di un'istanza di GlassFish Server in cluster

I numeri di porta di un'istanza di GlassFish Server in cluster sono impostati inizialmente nella configurazione che l'istanza eredita dal cluster padre. È necessario modificare questi numeri di porta se, ad esempio, due istanze in cluster risiedono nello stesso host. In questa situazione è necessario modificare i numeri di porta per una delle istanze, in quanto le due istanze condividono la stessa configurazione.

I numeri di porta di un'istanza vengono memorizzati come proprietà di sistema Java. Queste proprietà vengono passate al launcher dell'applicazione Java tramite la relativa opzione -D all'avvio di GlassFish Server.

  1. Espandere il nodo Cluster nell'albero di navigazione.

  2. Selezionare il nodo per il cluster sotto il nodo Cluster espanso.

    Viene visualizzata la pagina Informazioni generali per il cluster.

  3. Nella pagina Informazioni generali, fare clic sulla scheda Istanze.

    Viene visualizzata la pagina Istanze server in cluster per il cluster.

  4. Selezionare l'istanza per la quale modificare i numeri di porta.

    Viene visualizzata la pagina Informazioni generali per l'istanza.

  5. Nella pagina Informazioni generali, fare clic sulla scheda Proprietà.

    Viene visualizzata la pagina Proprietà di sistema dell'istanza.

    Nella pagina vengono visualizzati i valori correnti delle proprietà di sistema per l'istanza riportate di seguito.

    • I numeri di porta predefiniti nella configurazione dell'istanza

    • Le eventuali proprietà personalizzate definite per l'istanza

  6. Per ogni proprietà esistente da modificare, digitare il nuovo valore nel campo Valore di sostituzione per la proprietà.

  7. Impostare i valori per eventuali proprietà aggiuntive necessarie.

    1. Nella tabella Proprietà aggiuntive fare clic su Aggiungi proprietà.

      Viene aggiunta una riga alla tabella Proprietà aggiuntive.

    2. Nel campo Nome variabile istanza digitare il nome della proprietà.

    3. Nel campo Valore di sostituzione digitare il valore.

  8. Fare clic su Salva.
