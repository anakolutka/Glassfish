GlassFish Project - Web Services CLI command usage

In this page, the usage of Web Services Management CLI commands is illustrated.  There are four web services management CLI  commands namely. configure-webservice-management, create-transformation-rule, delete-transformation-rule and list-tranformation-rules. These commands usage is explained in the following using 10 user secanarios.
 

  • To turn on a monitoring for a web service endpoint


    • The following command turns on monitoring level to LOW for the web service endpoint HelloIF in the application jaxrpc-simple. This web service endpoint is pacakaged in jaxrpc-simple.war module in that application. Thats why the fully qualified web service endpoint name is specified as jaxrpc-simple#jaxrpc-simple.war#HelloIF.

    configure-webservice-management --monitoring=LOW jaxrpc-simple#jaxrpc-simple.war#HelloIF
     

  • To turn on message tracing facility for a web service endpoint


    • To turn on message tracing facility, monitoring should be turned on to HIGH and message trace history size must be greater than 0.

     configure-webservice-management --monitoring=HIGH --maxhistorysize=250 jaxrpc-simple#jaxrpc-simple.war#HelloIF

     (default for --maxhistorysize=25)
     

  •  To turn off message trace from step #2


    • The following command turns off message trace facility and sets the monitoring level to LOW.  In the LOW setting the maxhistorysize value has no significance.

    configure-webservice-management --monitoring=LOW --maxhistorysize=25
     jaxrpc-simple#jaxrpc-simple.war#HelloIF
     

  •  To completely turn off management capabaility


    • The following turns off monitoring and message trace for the web service endpoint.

     configure-webservice-management --monitoring=OFF jaxrpc-simple#jaxrpc-simple.war#HelloIF
     

  •  To create a tranformation for that apply to a request operation


    •  create-transformation-rule --webservicename
     jaxrpc-simple#jaxrpc-simple.war#HelloIF
     --enabled=true  --applyto=request --rulefilelocation
     $<INSTALL_ROOT>/generated/xml/req.xslt  ChangeRequest_Rule
     

  • To create a tranformation for that apply to a response operation


    • create-transformation-rule --webservicename jaxrpc-simple#jaxrpc-simple.war#HelloIF  --enabled=true  --applyto=response --rulefilelocation  $<INSTALL_ROOT>/generated/xml/res.xslt  ChangeResponse_Rule
     

  • To create a tranformation for that apply to both request and response operations


    • create-transformation-rule --webservicename
     jaxrpc-simple#jaxrpc-simple.war#HelloIF
     --enabled=true  --applyto=both --rulefilelocation
     $<INSTALL_ROOT>/generated/xml/both.xslt  NormalizeContent_Rule
     

  • To list all the transformation rules for all web service end points


    • list-transformation-rules
     

  •  To list to which rules are being applied ( enabled rules) for a web service endpoint and it shows the order they get applied.


    • list-transformation-rules --webservicename webservice_name jaxrpc-simple#jaxrpc-simple.war#HelloIF
     

  • To remove any of the above rules


    • delete-transformation-rule --webservicename
     jaxrpc-simple#jaxrpc-simple.war#HelloIF <transformation-rule-name>
