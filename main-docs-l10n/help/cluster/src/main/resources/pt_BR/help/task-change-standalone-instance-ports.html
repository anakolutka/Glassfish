Para Modificar os Números de Porta de uma Instância do GlassFish Server Standalone

Os números de porta de uma instância standalone do GlassFish Server são definidos inicialmente na configuração que a instância faz referência. É necessário modificar estes números de porta se, por exemplo, duas instâncias que fazem referência à mesma configuração residirem no mesmo host. Nesse caso, os números de porta de uma instância devem ser modificados porque ambas as instâncias fazem referência à mesma configuração.

Os números de porta de uma instância são armazenados como propriedades de sistema Java. Essas propriedades são passadas para o acionador da aplicação Java por meio da opção -D do acionador da aplicação Java quando o GlassFish Server é iniciado.

  1. Na árvore de navegação, selecione o nó Instâncias Standalone.

    A página Instância de Servidor Standalone será aberta.

  2. Selecione a instância cujos números de porta deseja modificar.

    A página Informações Gerais da instância se abre.

  3. Na página Informações Gerais, clique na guia Propriedades.

    A página Propriedades de Sistema da Instância será aberta.

    Esta página exibe os valores atuais das seguintes propriedades do sistema para a instância:

    • Números de porta predefinidos na configuração da instância

    • Propriedades personalizadas definidas para a instância, se houver

  4. Digite o novo valor de cada propriedade que estiver sendo modificada no campo Valor Substituto.

  5. Defina os valores de quaisquer propriedades adicionais que forem necessárias.

    1. Na tabela Propriedades Adicionais, clique em Adicionar Propriedade.

      Uma linha é adicionada à tabela Propriedades Adicionais.

    2. No campo Nome da Variável da Instância, digite o nome da propriedade.

    3. No campo Valor Substituto, digite o valor.

  6. Clique em Salvar.
