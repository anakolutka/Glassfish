Managing Resources on a Stand-alone Server Instance

The Resources page enables you to perform the following operations:

  • View all the resources for a standalone instance or resources belonging to a specific resource type. You can do this by selecting the required option from the Filter drop-down list.

  • Create a new resource. To do this, from the New drop-down list, select the resource type you want to create. The New Resource page opens depending on the resource type you selected.

  • Enable a resource. The value in the Enabled column will be 'false' if the resource is not enabled.

  • Disable a resource. The value in the Enabled column will be 'true' if the resource is enabled.
