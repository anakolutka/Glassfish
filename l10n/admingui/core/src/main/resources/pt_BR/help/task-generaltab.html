Para Exibir Informações Gerais do Enterprise Server

  1. Na árvore de navegação, selecione o nó Servidor de Aplicativos.

    A página Informações Gerais é exibida.

  2. Na página Informações Gerais, realize qualquer uma das etapas a seguir.
    • Clique no botão Parar para interromper o Enterprise Server.
    • Clique no botão Reiniciar para interromper e reiniciar o Enterprise Server.
    • Clique no botão Exibir Arquivos de Log para exibir o log do servidor.

      Consulte Para exibir o log do servidor para obter detalhes.

    • Clique no Relatório JVM para abrir uma janela separada que lhe permite exibir relatórios no Java Virtual Machine, incluindo um relatório de resumo, um relatório de gerenciamento de memória, um relatório de carregamento de classes e um despejo de segmentos atual.
