Modification du mot de passe administrateur

  1. Dans l'arborescence de navigation, sélectionnez le noeud Domaine.

    La page Attributs de domaine apparaît.

  2. Sélectionnez l'onglet Mot de passe de l'administrateur.

    La page Nouveau mot de passe de l'administrateur apparaît.

    La page affiche l'ID utilisateur et le nom du groupe du compte d'administrateur en cours. Par défaut, l'ID utilisateur est admin et le nom du groupe est asadmin.

  3. Dans le champ Nouveau mot de passe, saisissez le nouveau mot de passe du compte administrateur en cours.

  4. Dans le champ Confirmer le nouveau mot de passe, saisissez encore une fois le nouveau mot de passe.

  5. Cliquez sur Enregistrer.
