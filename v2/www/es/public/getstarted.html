GlassFish es un servidor de aplicaciones de código abierto que implementa las últimas funciones de la plataforma Java EE 5. La plataforma Java EE 5 incluye las últimas versiones de tecnologías como JavaServer Pages (JSP) 2.1, JavaServer Faces (JSF) 1.2, Servlet 2.4, Enterprise JavaBeans 3.0, Java API for Web Services (JAX-WS) 2.0, Java Architecture for XML Binding (JAXB) 2.0, Web Services Metadata para Java Platform 1.0, entre otras muchas.

¿Es nuevo en GlassFish? Aprenda a descargar e instalar GlassFish, pruebe la guía de procedimientos iniciales, vea algunas películas o ensaye nuestros ejemplos en el tutorial. Vaya a la página de procedimientos iniciales.

Descargar y ejecutar

Puede descargar versiones de acceso rápido del servidor de aplicaciones GlassFish. La descarga incluye una versión binaria del servidor de aplicaciones GlassFish Java EE 5, de manera que sólo tiene que descomprimir la descarga y ejecutarla. Hay descargas para ejecutar en los sistemas operativos más conocidos.

Consiga ya las descargas.
