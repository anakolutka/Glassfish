Editar propiedades de pool de conexiones

Utilice la página Editar propiedades de pool de conexiones para modificar las propiedades adicionales de un pool de conexiones JDBC existente.

Propiedades Adicionales

Propiedades adicionales del pool de conexiones JDBC.

Para obtener una descripción de algunas de las propiedades disponibles, consulte Propiedades específicas de JDBC pools de conexiones.

Si no especifica un valor para una propiedad, ésta se elimina de la tabla al finalizar la edición del pool de conexiones. Si no desea definir un valor para una propiedad ahora, pero desea que la propiedad permanezca en la tabla para que el valor se pueda definir más adelante, introduzca un juego de paréntesis vacío en el campo Valor:

()
