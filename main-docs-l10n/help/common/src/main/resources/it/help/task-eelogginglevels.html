Per configurare i livelli di log

  1. Espandere il nodo Configurazioni nell'albero di navigazione.

  2. Nel nodo Configurazione fare clic sulla configurazione dell'istanza server o del cluster per cui si desidera configurare le impostazioni del logger.

    Viene visualizzata la pagina Configurazione.

  3. Nella pagina Configurazione fare clic su Impostazioni logger.

    Viene visualizzata la pagina Impostazioni logger per la destinazione della configurazione selezionata.

  4. Nella pagina Impostazioni logger fare clic sulla scheda Livelli di log.

    Viene visualizzata la pagina Livelli di log dei moduli.

  5. Nella pagina Livelli di log dei moduli modificare i livelli di log di uno o più moduli attenendosi alla seguente procedura.

    • Per modificare il livello di log di un singolo modulo, selezionare un nuovo valore dall'elenco a discesa corrispondente al modulo.

    • Per modificare il livello di log di un gruppo di moduli, effettuare le operazioni riportate di seguito.

      1. Selezionare le caselle di controllo in corrispondenza dei nomi dei moduli per i quali si desidera modificare i livelli di log.

      2. Selezionare un valore dall'elenco a discesa nella parte superiore della tabella.

      3. Fare clic sul pulsante Cambia livello.

    Il log visualizzerà i messaggi corrispondenti al livello selezionato o ai livelli superiori. Scegliere uno dei valori seguenti (elencati in ordine decrescente):

    • SEVERE

    • WARNING

    • INFO

    • CONFIG

    • FINE

    • FINER

    • FINEST

    • OFF

    Per informazioni dettagliate sul significato dei livelli di log, vedere Livelli di log dei moduli. Il livello predefinito è INFO, pertanto il log conterrà per impostazione predefinita i messaggi di livello INFO, WARNING o SEVERE.

  6. Fare clic su Salva.

    Le modifiche delle impostazioni Livello di log hanno effetto immediato. Vengono inoltre salvate in modo da poter essere utilizzate al riavvio del server.
