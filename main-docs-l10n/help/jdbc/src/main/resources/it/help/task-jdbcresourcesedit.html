Per modificare una risorsa JDBC

  1. Espandere il nodo Risorse nell'albero di navigazione.

  2. Nel nodo Risorse espandere il nodo JDBC.

  3. Nel nodo JDBC selezionare il nodo Risorse JDBC.

    Viene visualizzata la pagina Risorse JDBC.

  4. Nella pagina Risorse JDBC fare clic sul nome della risorsa da modificare.

    Viene visualizzata la pagina Modifica risorsa JDBC.

  5. Nell'elenco a discesa Nome pool selezionare il connection pool da associare a questa risorsa JDBC.

  6. Nel campo Ordine di distribuzione digitare un numero per specificare l'ordine di distribuzione della risorsa JDBC.

    Le risorse con numero inferiore vengono caricate per prime all'avvio del server. Il valore predefinito è 100.

  7. Nel campo Descrizione digitare una breve descrizione della risorsa JDBC.

  8. Nella sezione Proprietà aggiuntive, specificare le proprietà aggiuntive.

    GlassFish Server non definisce alcuna proprietà aggiuntiva per le risorse JDBC.

    • Per aggiungere una proprietà, fare clic sul pulsante Aggiungi proprietà. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Nome e il valore della proprietà nel campo Valore.

    • Per modificare una proprietà, è necessario modificare il valore del campo Valore della proprietà.

    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Nome corrispondente alla proprietà da eliminare, quindi fare clic sul pulsante Elimina proprietà.

  9. Fare clic sulla scheda Destinazione per gestire le destinazioni della risorsa.

    1. Abilitare o disabilitare le destinazioni definite per la risorsa in base alle esigenze.

    2. Fare clic sul pulsante Gestisci destinazioni per aggiungere o rimuovere destinazioni per la risorsa.

    Per ulteriori informazioni, vedere Per modificare le destinazioni di una risorsa JDBC.

  10. Fare clic sulla scheda Generale per tornare alla pagina Modifica risorsa JDBC.

  11. Fare clic su Salva.
