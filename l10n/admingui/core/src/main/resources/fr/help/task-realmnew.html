Créer un domaine

Le Enterprise Server est préconfiguré avec trois domaines : fichier, certificat et administrateur. Il est également possible de créer d'autres domaines ldap, JDBC, solaris et des domaines personnalisés. Un serveur comprend généralement un domaine de chaque type, tandis que le Enterprise Server comprend deux domaines de fichiers : fichier et administrateur. Il s'agit de deux domaines du même type, mais ayant deux utilisations différentes. Il est également possible d'avoir une base de données de certificats différente pour chaque serveur virtuel du système.

  1. Dans l'arborescence de navigation, développez le nœud Configuration.
  2. Si la configuration du domaine prend en charge des clusters, sélectionnez la configuration à modifier.

    Exemples : domaines créés avec le profil Cluster ou entreprise .

  3. Développez le nœud Sécurité.
  4. Sélectionnez le nœud Domaines.
  5. Sur la page Domaines, cliquez sur Nouveau.

    La page Nouveau domaine s'affiche.

  6. Entrez le nom du domaine dans le champ Nom.
  7. Indiquez le nom de classe du domaine à créer.

    Les choix possibles sont affichés dans le tableau suivant.

    Nom de domaine

    Nom de la classe

    fichier

    com.sun.enterprise.security.auth.realm.file.FileRealm

    certificat

    com.sun.enterprise.security.auth.realm.certificate.CertificateRealm

    ldap

    com.sun.enterprise.security.auth.realm.ldap.LDAPRealm

    JDBC

    com.sun.enterprise.security.auth.realm.jdbc.JDBCRealm

    solaris

    com.sun.enterprise.security.auth.realm.solaris.SolarisRealm

    personnalisé

    Nom de classe du domaine de connexion
  8. Ajoutez toute autre propriété de votre choix au domaine.
    1. Cliquez sur Ajouter une propriété.
    2. Dans le champ Nom, saisissez le nom de la propriété.

      • Pour une description des propriétés de domaine de fichier, consultez Propriétés spécifiques de la classe FileRealm.

      • Pour une description des propriétés de domaine de certificat, consultez Propriétés spécifiques de la classe CertificateRealm.

    3. Dans le champ Valeur, saisissez la valeur de la propriété.
  9. Cliquez sur OK.
