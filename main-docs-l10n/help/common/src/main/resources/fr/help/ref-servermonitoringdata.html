Surveillance de serveur

La page Surveillance de serveur permet de visualiser les données de surveillance pour l'instance de serveur.

Elle contient les informations ci-après.

Nom d'instance

Nom de l'instance de serveur.

Afficher

Serveur virtuel, processus d'écoute HTTP, opération Garbage Collector ou système de threads pour lequel les statistiques de surveillance sont affichées.

Pour obtenir une description des propriétés surveillées, reportez-vous au chapitre 8, Administering the Monitoring Service, du manuel Oracle GlassFish Server Administration Guide.
