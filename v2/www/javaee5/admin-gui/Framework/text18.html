Architecture

  • Example:

  • public class CCButtonDescriptor extends DisplayFieldDescriptor {

  • public CCButtonDescriptor(String name) {

    • super(name);

  • }

  • public View getInstance(RequestContext ctx, ContainerView container, String name) {

    • String buttonText = (String)getParameter("name");
    • return new CCButton(container, name, buttonText);

  • }

  • }


Notes:

CCButonDescriptor extends from DisplayFieldDescriptor, which extends from ViewDescriptor. In this case, CCButtonDescriptor is not using any functionality of DisplayFieldDescriptor, however, in the future it may get its model name and default value automatically from its super class.
