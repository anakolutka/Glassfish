新建環境服務

使用「新建環境服務」頁面即可建立環境服務資源。

「新建環境服務」頁面包含下列選項。

JNDI 名稱

可識別資源的唯一名稱。

環境資訊

資源的環境：類別載入器、JNDI、安全性或工作區。

描述

此資源的描述。

狀態

資源狀態。您可啟用或停用資源。預設值為「啟用」。

其他特性

資源的其他特性。GlassFish Server 未定義環境服務資源的任何其他特性。

相關的 asadmin 命令

  • create-XXX(1)

  • list-XXX(1)

  • delete-XXX(1)
