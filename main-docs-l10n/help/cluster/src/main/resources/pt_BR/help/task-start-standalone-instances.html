Para Iniciar Instâncias do GlassFish Server Standalone

Antes de Começar

Certifique-se de que os seguintes pré-requisitos foram atendidos:

  • Todos os nós nos quais se encontram as instâncias estão ativados para comunicação remota.

  • O usuário do servidor de administração de domínios (DAS) pode usar o protocolo remoto DCOM (Distributed Component Object Model) ou secure shell (SSH) para efetuar log-in no host para qualquer nó em que residam as instâncias no cluster.

Se algum desses pré-requisitos não for atendido, inicie cada instância individualmente pela linha de comandos, usando o subcomando start-local-instance(1).

  1. Na árvore de navegação, selecione o nó Instâncias Standalone.

    A página Instância de Servidor Standalone será aberta.

  2. Para cada instância que estiver sendo iniciada, marque a caixa de seleção ao lado da instância na tabela Instâncias do servidor.

  3. Clique em Iniciar.

    Será solicitado que você confirme se deseja iniciar as instâncias.

  4. Confirme se deseja iniciar as instâncias.

    O status de cada instância é atualizado na tabela Instâncias do servidor quando a instância for iniciada.
