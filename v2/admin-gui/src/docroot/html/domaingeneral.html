Domain Attributes in the Cluster and Enterprise Profile

The General Information page displays the following information:

  • Application Root: the directory from which applications are deployed.

  • Log Root: the location of the server instance's log files.

  • Locale: defines the locale setting and overrides the locale setting of the operating system.

In addition, you can select the following tabs to perform these additional tasks:

  • Diagnostic Tab: configure the diagnostic service settings.

  • Administrator Password Tab: change the password of the administrator currently logged in.

  • Advanced Tab: set general properties for deploying applications.
