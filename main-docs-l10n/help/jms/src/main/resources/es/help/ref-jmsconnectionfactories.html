Fábricas de conexiones de JMS

Utilice la página Fábricas de Conexiones JMS para configurar recursos de la fábrica de conexiones JMS.

Para cada recurso, aparece la siguiente información.

Nombre JNDI

Nombre JNDI que identifica la fábrica de conexiones.

Nombre de JNDI lógico

Nombre de JNDI lógico para la fábrica de conexiones. Consulte Nombres de JNDI lógicos para obtener más información.

Activado

(Esta columna sólo se muestra si la instancia del servidor por defecto, server, existe en el dominio).

Una marca de control si el recurso está activado, o una X si está desactivado.

Estado

(Esta columna sólo se muestra si en el dominio se ha creado algún cluster o instancia independiente).

Número de clusters e instancias independientes a los que se dirige el recurso y el número de estos destinos en los que el recurso está activado. Por ejemplo, "Activado en 2 de 4 de destinos" significa que el recurso se dirige a cuatro clusters e instancias independientes y que está activado en dos de estos cuatro destinos.

Tipo de recurso

Tipo de recurso de la fábrica de conexiones.

Descripción

Descripción de la fábrica de conexiones.

La tabla Fábricas de Conexiones también contiene las siguientes opciones.

Nuevo

Botón para crear una nueva fábrica de conexiones.

Suprimir

Botón para suprimir una o varias fábricas de conexiones seleccionadas.

Activar

Botón para activar una o varias fábricas de conexiones seleccionadas.

Desactivar

Botón para desactivar una o varias fábricas de conexiones seleccionadas.
