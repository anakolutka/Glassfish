编辑已部署的应用程序

  1. 在导航树中，选择“应用程序”节点。

    此时将打开“应用程序”页面。

  2. 从“已部署的应用程序”表中，单击要编辑的应用程序的名称。

    此时将打开“编辑应用程序”页面。

  3. 对于 Web 应用程序，请在“上下文根目录”字段中，键入应用程序的路径。

    在 Web 应用程序的 URL 中，上下文根目录要紧跟着端口号 (http://host:port /context-root/...)。确保上下文根路径以正斜杠开头，例如 /hello。

  4. 在“说明”字段中，键入此应用程序的说明。
  5. 取消选择“启用状态”复选框以禁用应用程序状态，使其变为不可用。

    默认情况下，只要一部署应用程序，其状态便为“启用”。

  6. 选中“启用 Java Web Start”复选框，以使用 Java Web Start 软件来部署应用程序。
  7. 单击“保存”。
