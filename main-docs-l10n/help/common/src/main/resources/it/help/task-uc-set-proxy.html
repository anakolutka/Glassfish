Per configurare le impostazioni proxy di rete per l'accesso ai repository di rete

Le impostazioni proxy di rete determinano la modalità di connessione di GlassFish Server a Internet. Per abilitare Strumento di aggiornamento per l'accesso ai repository di rete mediante Internet, configurare le impostazioni proxy di rete.

  1. Selezionare il nodo Strumento di aggiornamento nell'albero di navigazione.

    Viene visualizzata la pagina Componenti aggiuntivi disponibili.

  2. Nella pagina Componenti aggiuntivi disponibili fare clic sulla scheda Configurazione.

    Viene visualizzata la pagina Configurazione Strumento di aggiornamento.

  3. Nella pagina Configurazione Strumento di aggiornamento specificare la modalità di connessione a Internet del computer in cui è eseguito GlassFish Server.

    • Se il computer si connette a Internet direttamente, selezionare l'opzione Connessione diretta.

    • Se il computer si connette a Internet mediante un proxy, selezionare l'opzione Tramite un proxy.

  4. Se il computer in cui viene eseguito GlassFish Server si connette a Internet tramite un proxy, specificare il nome del proxy in Host proxy e il numero della porta in Porta proxy.

    Se il computer nel quale viene eseguito GlassFish Server si connette a Internet direttamente, saltare questo passo.

    È possibile che un browser Web sia in esecuzione nel computer in cui viene eseguito GlassFish Server. In questo caso è possibile ottenere il nome e il numero di porta dell'host del proxy dal browser Web. In caso contrario, chiedere queste informazioni all'amministratore di sistema.

  5. Fare clic su Salva.
