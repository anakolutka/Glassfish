Affichage des fichiers journaux pour une instance GlassFish Server incluse dans un cluster

Avant de commencer

L'instance pour laquelle vous souhaitez afficher les fichiers journaux doit être en cours d'exécution.

  1. Dans l'arborescence de navigation, développez le noeud Clusters.

  2. Sous le noeud Clusters développé, sélectionnez le noeud correspondant au cluster.

    La page Informations générales du cluster apparaît.

  3. Sur la page Informations générales, cliquez sur l'onglet Instances.

    La page Instances de serveur clusterisées du cluster apparaît.

  4. Sélectionnez l'instance pour laquelle vous souhaitez afficher les fichiers journaux.

    La page Informations générales de l'instance apparaît.

  5. Sur la page Informations générales, cliquez sur Afficher les fichiers journaux.

    La fenêtre du visualiseur de journaux apparaît.

    Remarque:

    L'instance doit déjà être en cours d'exécution. Si elle ne l'est pas, le bouton Afficher les fichiers journaux est désactivé.
