사용자를 추가하는 방법

  1. 새로 만들기를 눌러 파일 영역에 새 사용자를 추가합니다.

    새 파일 영역 사용자 페이지가 열립니다.

  2. 사용자 ID 필드에 사용자 이름을 입력합니다. 필수 필드입니다.
  3. 비밀번호 필드에 고유한 비밀번호를 입력합니다. 필수 필드입니다.
  4. 검증을 위해 사용자 비밀번호를 다시 확인합니다. 필수 필드입니다.
  5. 그룹 목록 필드에 사용자가 속한 그룹의 쉼표로 구분된 목록을 입력합니다. 이 그룹은 다른 곳에서 정의할 필요가 없습니다.
  6. 파일 영역의 사용자 목록에 이 사용자를 추가하려면 확인을 선택하거나 저장 없이 종료하려면 취소를 선택합니다.
