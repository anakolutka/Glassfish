Nouveau pool de connexions JDBC (étape 1 sur 2)

La page Nouveau pool de connexions JDBC (étape 1 sur 2) permet de créer un pool de connexions JDBC.

Elle contient les options ci-après.

Nom du pool

Nom du pool de connexions.

Type de ressource

Les types de ressource disponibles incluent : javax.sql.XADataSource (transactions globales), java.sql.ConnectionPoolDataSource (transactions locales, améliorations possibles des performances), javax.sql.DataSource (transactions locales uniquement) et java.sql.Driver.

Fournisseur du pilote de base de données

Nom du fournisseur de base de données.

Introspecter

Si cette option est activée, le pilote de base de données est introspecté en fonction du type de ressource et du fournisseur de pilote de base de données, et tous les noms de source de données ou de classe de pilote repérés sont affichés sur la page Nouveau pool de connexions JDBC (étape 2 sur 2) lorsque vous cliquez sur le bouton Suivant.

Pour que l'introspection fonctionne, le pilote JDBC doit être disponible pour GlassFish Server. Reportez-vous à Intégration d'un pilote JDBC.
