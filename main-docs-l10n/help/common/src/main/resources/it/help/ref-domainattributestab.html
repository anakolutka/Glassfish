Attributi dominio

Utilizzare la pagina Attributi dominio per impostare attributi di dominio avanzati per GlassFish Server.

La pagina Attributi dominio contiene le opzioni riportate di seguito.

Carica console dopo avvio DAS

Determina quando caricare Console di amministrazione in base alle opzioni riportate di seguito.

Basato sull'uso

Carica Console di amministrazione dopo il server di amministrazione del dominio (DAS) se la console è stata utilizzata nelle ultime 24 ore o se esistono altre istanze server oltre al server DAS. Questa è l'impostazione predefinita.

Sempre

Carica sempre Console di amministrazione dopo il server DAS. La console viene caricata al termine della sequenza di avvio principale di GlassFish Server.

Mai

Non carica mai Console di amministrazione dopo il server DAS. La console viene caricata quando viene richiesto di accedervi.

Radice applicazioni

Il percorso completo della directory in cui risiedono le applicazioni distribuite. Il valore predefinito è domain-dir/applications.

Radice log

Indica la posizione di memorizzazione dei file di log del server. Il valore predefinito è domain-dir/logs.

Impostazioni nazionali

Se questo campo viene lasciato vuoto, vengono utilizzate le impostazioni nazionali predefinite dell'host.

Per impostazioni nazionali si intende un identificativo che specifica una determinata combinazione di lingua e regione. Ad esempio, le impostazioni nazionali per l'Inglese americano sono en_US, mentre per l'Italiano sono it_IT. Per utilizzare impostazioni nazionali non in lingua Inglese, è necessario che sia stata eseguita la localizzazione di GlassFish Server, un processo che prevede la traduzione dall'Inglese a un'altra lingua.
