Per modificare una risorsa per un'istanza standalone di GlassFish Server

  1. Selezionare il nodo Istanze standalone nell'albero di navigazione.

    Viene visualizzata la pagina Istanze server standalone.

  2. Nella pagina Istanze server standalone selezionare l'istanza.

    Viene visualizzata la pagina Informazioni generali per l'istanza.

  3. Nella pagina Informazioni generali fare clic sulla scheda Risorse.

    Viene visualizzata la pagina Risorsa per l'istanza.

  4. Fare clic sul nome della risorsa.

    Viene visualizzata la pagina per la modifica della risorsa.

  5. Nella pagina per la modifica della risorsa, impostare e salvare le modifiche necessarie.

    Per i dettagli, consultare la Guida in linea relativa alla pagina per la modifica della risorsa.
