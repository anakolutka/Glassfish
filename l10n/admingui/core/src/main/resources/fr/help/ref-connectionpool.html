Pool de connexion

La page Pool de connexion contient les options suivantes :

Nb max. de connexions en attente

Indique le nombre maximum de connexions en attente autorisées pour un listener HTTP. La valeur par défaut est 4096.

Taille de la file d'attente

Indique la taille maximale, en octets, de la file d'attente de connexion. Cette valeur indique également le nombre maximum de connexions en attente acceptées par le serveur. La valeur par défaut est 4096.

Taille du tampon de réception

Indique la taille, en octets, du tampon de réception d'un listener HTTP. La valeur par défaut est 4096.

Taille du tampon d'envoi

Indique la taille, en octets, du tampon d'envoi d'un listener HTTP. La valeur par défaut est 8192.
