To Manage JBI Targets

  1. In the tree view, click on the Integrations node and then the JBI Components node.

    A list of JBI binding components and service engines appears in the JBI Components page.

  2. Under the JBI Components node, click the JBI component whose targets you want to manage.

  3. In the Component Properties page, click Manage Targets. The Manage Targets page enables you to choose the targets for the component you have selected in the previous step.

  4. Use the Add or Remove buttons to select the targets for the component.

  5. Click OK. The Component Properties page will list the targets for the component.
