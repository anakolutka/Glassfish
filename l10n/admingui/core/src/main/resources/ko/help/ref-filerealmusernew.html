새 파일 영역 사용자

새 파일 영역 사용자 페이지에서는 현재 영역에 대한 액세스 권한을 새 사용자에게 부여할 수 있습니다.

새 파일 영역 사용자 페이지에는 다음 옵션이 포함되어 있습니다.

영역 이름

현재 선택된 영역의 이름입니다. 읽기 전용 필드입니다.

사용자 ID

새 사용자 계정의 사용자 ID입니다.

그룹 목록

새 사용자가 속할 그룹입니다. 여러 그룹 이름은 쉼표로 구분합니다.

admin-realm 사용자의 경우 이는 읽기 전용 필드이며 그룹은 asadmin입니다.

새 비밀번호

새 사용자의 비밀번호입니다.

새 비밀번호 확인

새 사용자의 비밀번호입니다.

관련 작업

  • 사용자를 파일 영역에 추가하는 방법

  • 파일 영역 사용자 정보를 편집하는 방법

  • 파일 영역에서 사용자를 삭제하는 방법

관련 asadmin 명령

  • list-file-users(1)

  • create-file-user(1)

  • update-file-user(1)

  • delete-file-user(1)
