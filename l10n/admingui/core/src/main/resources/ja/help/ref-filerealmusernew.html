新しいファイルレルムユーザー

「新しいファイルレルムユーザー」ページを使用して、新しいユーザーに現在のレルムに対するアクセス権を付与します。

レルム名

現在選択されているレルムの名前を表示する読み取り専用フィールド。

ユーザー ID

新しいユーザーアカウントのユーザー ID。

グループ リスト

新しいユーザが属するグループ。複数のグループ名をコンマで区切ります。

新しいパスワード

新しいユーザーのパスワード。

新しいパスワードを確認

新しいユーザーのパスワード。
