设置高级域属性

  1. 在导航树中，选择 "Application Server" 节点。
  2. 在“常规信息”页面上，单击“高级”选项卡。
  3. 在“应用程序配置”页面上，单击“域属性”选项卡。

    此时将打开“域属性”页面。

  4. 在“应用程序根目录”字段中，键入将部署应用程序的目录的完整路径。
  5. 在“日志根目录”字段中，键入将保存服务器日志文件的目录的完整路径。
  6. 将“语言环境”字段保留为空，以便使用主机的默认语言环境。

    语言环境是一种标识符，用于指定特定的语言和区域组合。例如，美国英语的语言环境是 en_US，日语的语言环境是 ja_JP。要使用非英语语言环境，Enterprise Server 必须经过本地化，即必须将英语翻译成其他语言。

  7. 单击“保存”。
  8. 重新启动服务器。
