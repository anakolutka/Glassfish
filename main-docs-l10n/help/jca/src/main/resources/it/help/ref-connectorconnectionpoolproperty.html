Modifica proprietà connection pool connettore

Utilizzare la pagina Modifica proprietà connection pool connettore per modificare le impostazioni per un connection pool del connettore.

La pagina Modifica proprietà connection pool connettore contiene le opzioni riportate di seguito.

Proprietà aggiuntive

Proprietà aggiuntive per il connection pool del connettore. Le proprietà disponibili dipendono dal Resource Adapter per il connection pool del connettore. Le coppie nome-valore specificate in questa tabella possono essere utilizzate per sostituire i valori predefiniti per le proprietà specificate dal fornitore del Resource Adapter.

Se non si specifica un valore per una proprietà, la proprietà interessata verrà rimossa dalla tabella al termine della modifica del connection pool. Se non si desidera impostare il valore di una proprietà in questa fase, conservando tuttavia la proprietà nella tabella in modo da impostarne il valore in un secondo momento, digitare un set di parentesi vuote nel campo Valore:

()
