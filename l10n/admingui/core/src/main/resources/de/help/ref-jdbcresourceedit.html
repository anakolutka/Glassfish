JDBC-Ressource bearbeiten

J.N.D.I.-Name

Der J.N.D.I.-Name ist ein schreibgeschütztes Feld. Sie können den J.N.D.I.-Namen nur angeben, wenn Sie eine neue JDBC-Ressource erstellen.

Poolname

Der mit der JDBC-Ressource verbundene Verbindungspool.

Beschreibung

Eine kurze Beschreibung der JDBC-Ressource.

Status

Der Status der JDBC-Ressource. Die Ressource kann aktiviert oder deaktiviert werden.
