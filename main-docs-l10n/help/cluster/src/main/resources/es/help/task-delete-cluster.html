Para suprimir un cluster

Al suprimir un cluster se suprime el cluster y todas las instancias de GlassFish Server que contenga. No es necesario suprimir las instancias que contiene el cluster antes de suprimirlo.

Si la configuración que utiliza el cluster no es compartida, la configuración también se suprime.

Antes de Empezar

Asegúrese de que el cluster que vaya a suprimir está parado. Consulte Para parar un cluster para obtener instrucciones.

  1. En el árbol de navegación, seleccione el nodo Clusters.

    Se abrirá la página Clusters.

  2. En la página Clusters, active la casilla de control que aparece junto al nombre del cluster.

  3. Haga clic en Suprimir.

    Se le pedirá que confirme que desea suprimir el cluster y todas las instancias que contenga.

  4. Confirme que desea suprimir el cluster y todas las instancias que contenga.

    Se suprimirá el cluster y se eliminará de la tabla de clusters de la página Clusters.
