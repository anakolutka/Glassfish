파일 영역에서 사용자를 삭제하는 방법

  1. 탐색 트리에서 구성 노드를 선택합니다.
  2. 보안 노드를 확장합니다.
  3. 영역 노드를 확장합니다.
  4. 파일 노드를 선택합니다.

    영역 편집 페이지가 열립니다.

  5. 영역 편집 페이지에서 사용자 관리 버튼을 누릅니다.

    파일 사용자 페이지가 열립니다.

  6. 파일 사용자 페이지에서 삭제할 사용자 이름의 왼쪽에 있는 확인란을 선택합니다.
  7. 삭제를 누릅니다.

    사용자를 삭제해야 하는지 확인하는 대화 상자가 표시됩니다.

  8. 대화 상자에서 확인을 누릅니다.
참조

  • 사용자를 파일 영역에 추가하는 방법

  • 파일 영역 사용자 정보를 편집하는 방법
