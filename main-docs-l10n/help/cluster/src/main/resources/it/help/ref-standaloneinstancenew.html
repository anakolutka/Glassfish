Nuova istanza server standalone

Utilizzare la pagina Nuova istanza server standalone per creare un'istanza standalone di GlassFish Server.

La pagina Nuova istanza server standalone contiene le opzioni riportate di seguito.

Nome istanza

Il nome dell'istanza che si sta creando.

Il nome deve soddisfare i requisiti riportati di seguito.

  • Il nome può contenere solo caratteri ASCII.

  • Il nome deve iniziare con una lettera, un numero o un carattere di sottolineatura.

  • Il nome può contenere solo i caratteri seguenti:

    • Lettere minuscole

    • Lettere maiuscole

    • Numeri

    • Trattino

    • Punto

    • Carattere di sottolineatura

  • Il nome deve essere univoco nel dominio e non deve essere il nome di un'altra istanza di GlassFish Server, di un cluster, di una configurazione denominata o di un nodo.

  • Il nome non deve essere domain, server né un'altra parola chiave riservata di GlassFish Server.

Nodo

Elenco a discesa dei nodi esistenti che definiscono gli host in cui può risiedere l'istanza. È necessario selezionare un nodo dalla lista. Se l'istanza deve essere creata nell'host in cui è in esecuzione il server di amministrazione del dominio (DAS), selezionare il nodo predefinito localhost-domain.

Configurazione

Elenco a discesa delle configurazioni denominate esistenti che possono essere utilizzate dall'istanza. È necessario selezionare una configurazione dalla lista. L'istanza utilizzerà la configurazione selezionata al momento della creazione dell'istanza.

Crea una copia della configurazione selezionata

Se questa opzione è selezionata, la configurazione selezionata viene copiata al momento della creazione dell'istanza. Alla copia della configurazione viene assegnato il nome nome-istanza-config, dove nome-istanza è il nome dell'istanza digitato nel campo Nome istanza.

Se nell'elenco a discesa Configurazione è selezionata la voce default-config, questa opzione deve essere selezionata. La configurazione default-config può essere solo copiata e non utilizzata come riferimento.

Crea riferimento alla configurazione selezionata

Se questa opzione è selezionata, l'istanza utilizzerà la configurazione denominata esistente specificata.

Se nell'elenco a discesa Configurazione è selezionata la voce default-config, questa opzione non deve essere selezionata. La configurazione default-config non può essere utilizzata come riferimento, ma solo copiata.
