Welcome to the Integration Technologies home page. This page is dedicated to discussing the collection of Integration related technologies in GlassFish. Technologies included on this page are the J2EE Connector Architecture, Java Transaction Service (JTS), Java Transaction API (JTA) and JDBC/JMS/Connector connection pooling infrastructure.
