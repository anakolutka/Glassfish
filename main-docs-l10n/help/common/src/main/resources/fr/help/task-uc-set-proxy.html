Configuration des paramètres de proxy réseau permettant d'accéder à des référentiels réseau

Les paramètres de proxy réseau déterminent le mode de connexion de GlassFish Server à Internet. Pour permettre à l'Outil de mise à jour d'accéder aux référentiels réseau via Internet, configurez les paramètres de proxy réseau.

  1. Dans l'arborescence de navigation, sélectionnez le noeud Outil de mise à jour.

    La page Extensions disponibles apparaît.

  2. Sur la page Extensions disponibles, cliquez sur l'onglet Configuration.

    La page Configuration de l'Outil de mise à jour apparaît.

  3. Sur la page Configuration de l'Outil de mise à jour, indiquez le mode de connexion à Internet de l'ordinateur sur lequel GlassFish Server est exécuté.

    • Si l'ordinateur se connecte directement à Internet, sélectionnez l'option Connexion directe.

    • Si l'ordinateur se connecte à Internet via un proxy, sélectionnez l'option Via un proxy.

  4. Si l'ordinateur sur lequel GlassFish Server est exécuté se connecte à Internet via un proxy, indiquez le nom de l'hôte proxy et son numéro de port.

    Si l'ordinateur sur lequel GlassFish Server est exécuté se connecte directement à Internet, ignorez cette étape.

    Il est possible d'utiliser un navigateur Web sur l'ordinateur sur lequel GlassFish Server est exécuté. Dans ce cas, vous pouvez obtenir le nom d'hôte et le numéro de port du proxy à partir du navigateur Web. Sinon, demandez ces informations à l'administrateur système.

  5. Cliquez sur Enregistrer.
