Modifier le service de gestion de groupes

La page Service de gestion de groupes permet de modifier les paramètres GMS.

Le service de gestion de groupes (GMS) est un service inclus dans un processus offrant des services de surveillance de clusters, d'appartenance au cluster et de communication de groupe. Le service GMS est construit sur une pile configurable de protocoles de surveillance de l'intégrité et de détection d'appartenance à des groupes. Il exécute les fonctions suivantes :

  • Il signale les modules inscrits dans une instance GlassFish Server lorsque des instances membres sont ajoutées à un cluster et enlevées d'un cluster, ou lorsque leur échec est suspecté ou a été confirmé.

  • Il permet l'envoi et la réception de messages entre les membres du cluster.

La page Service de gestion de groupes contient les options suivantes :

Charger les valeurs par défaut

Bouton permettant de restaurer les valeurs par défaut des paramètres qui en ont. Les paramètres qui n'ont pas de valeurs par défaut ne sont pas modifiés.

Nom de configuration

Nom de la configuration à laquelle s'appliquent les paramètres de cette page. Ce champ est en lecture seule.

Nb maximal de signaux d'activité manqués

Nombre de signaux d'activité manqués d'un membre après lequel un échec d'instance de serveur est suspecté. La valeur par défaut est 3.

Fréquence de signal

Délai entre les signaux d'activité manqués d'un membre. La valeur par défaut est 2 000 millisecondes.

Délai de détection de groupe expiré

Durée d'attente du service GMS pour la détection d'autres membres du cluster. La valeur par défaut est 5 000 millisecondes.

Temps d'attente de vérification d'échec

Temps d'attente avant la vérification de l'indication d'échec d'un membre suspecté. La valeur par défaut est 1 500 millisecondes.

Autres propriétés

Table des propriétés supplémentaires pour le service de gestion de groupes.

GlassFish Server ne définit pas d'autre propriété GMS.

Pour chaque propriété, les informations suivantes sont affichées :

Nom

Nom de la propriété.

Valeur

Valeur de la propriété définie pour l'instance sélectionnée.

Description

Texte de description qui fournit de plus amples informations sur la propriété.

La table Autres propriétés contient également les options ci-après.

Ajouter une propriété

Bouton permettant d'ajouter une propriété. Un clic sur ce bouton permet d'ajouter une ligne à la table Autres propriétés.

Supprimer des propriétés

Bouton permettant de supprimer les propriétés sélectionnées. Toute propriété supprimée revient à sa valeur par défaut ou, si aucune valeur par défaut n'est configurée, n'est pas définie.
