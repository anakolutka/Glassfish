Configurations de fournisseurs

Utilisez la page Configurations de fournisseurs pour configurer les configurations de fournisseurs de sécurité des messages pour le niveau d'authentification SOAP ou HttpServlet.

Par défaut, quatre fournisseurs sont configurés pour le niveau d'authentification SOAP.

Pour chaque configuration de fournisseur, la page Configurations de fournisseurs contient les informations suivantes.

ID fournisseur

Nom de la configuration du fournisseur.

Type de fournisseur

Type du fournisseur, qui peut être l'un des éléments suivants :

client

Spécifie que le fournisseur est le fournisseur d'authentification du client.

serveur

Spécifie que le fournisseur est le fournisseur d'authentification du serveur.

client-serveur

Spécifie que le fournisseur est à la fois le fournisseur d'authentification du client et du serveur.

Fournisseur par défaut

S'il est défini à true, le fournisseur est le fournisseur par défaut.

Nom de classe

Nom de la classe d'implémentation Java du fournisseur.
