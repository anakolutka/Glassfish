Per modifica le proprietà del connection pool connettore

  1. Espandere il nodo Risorse nell'albero di navigazione.

  2. Espandere il nodo Connettori.

  3. Nel nodo Connettori selezionare il nodo Connection pool connettore.

    Viene visualizzata la pagina Connection pool connettore.

  4. Nella pagina Connection pool connettore fare clic sul nome del connection pool del connettore da modificare.

    Viene visualizzata la pagina Modifica connection pool connettore.

  5. Nella pagina Modifica connection pool connettore fare clic sulla scheda Proprietà aggiuntive.

    Viene visualizzata la pagina Modifica proprietà connection pool connettore.

  6. Nella tabella Proprietà aggiuntive specificare le proprietà aggiuntive per il connection pool del connettore.

    Le proprietà specificate dipendono dal Resource Adapter utilizzato dal pool. Le coppie nome-valore specificate dall'utente mediante questa tabella possono essere utilizzate per sostituire i valori predefiniti per le proprietà specificate dal fornitore del Resource Adapter.

    Se non si specifica un valore per una proprietà, la proprietà interessata verrà rimossa dalla tabella al termine della modifica del connection pool. Se non si desidera impostare il valore di una proprietà in questa fase, conservando tuttavia la proprietà nella tabella in modo da impostarne il valore in un secondo momento, digitare un set di parentesi vuote nel campo Valore:

    ()

    • Per aggiungere una proprietà, fare clic sul pulsante Aggiungi proprietà. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Nome e il valore della proprietà nel campo Valore.

    • Per modificare una proprietà, è necessario modificare il valore del campo Valore della proprietà.

    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Nome corrispondente alla proprietà da eliminare, quindi fare clic sul pulsante Elimina proprietà.

  7. Fare clic su Salva.
