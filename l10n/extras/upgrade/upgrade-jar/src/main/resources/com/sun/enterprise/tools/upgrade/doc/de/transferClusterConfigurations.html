Das Fenster "Transfer Cluster Configurations" ("Clusterkonfigurationen übertragen")

Für ein Clusterkonfigurations-Upgrade von Sun Java System Application Server 8.x Enterprise Edition (EE) auf Sun Java System Application Server 9.1 ist eine gültige clinstance.conf- Datei erforderlich. Das Upgrade-Tool geht davon aus, dass die clinstance.conf- Datei das 8.x EE-Cluster präzise widerspiegelt. Falls diese Datei die tatsächliche Clusterkonfiguration nicht präzise widerspiegelt, muss die Clusterdefinition vor Durchführung des Upgrades manuell bearbeitet werden.

Geben Sie im Fenster "Transfer Cluster Configurations" ("Clusterkonfigurationen übertragen") die Clusterkonfigurationsdatei(en) an, die in die neue Installation migriert werden soll(en).

  1. Klicken Sie auf "Hinzufügen". Das Dialogfenster "clinstance-Konfigurationsdatei auswählen" wird geöffnet.
  2. Wählen Sie die clinstance.conf- Clusterdatei, die die zu migrierenden Cluster-Konfigurationsinformationen enthält.
  3. Klicken Sie auf die Schaltfläche "Öffnen".
  4. Wiederholen Sie diesen Vorgang so lange, bis alle zu migrierenden Clusterkonfigurationsdateien hinzugefügt wurden.
  5. Klicken Sie anschließend auf "Weiter". Das Fenster "Upgrade-Ergebnisse" wird geöffnet und zeigt den Status des Upgrades an.
