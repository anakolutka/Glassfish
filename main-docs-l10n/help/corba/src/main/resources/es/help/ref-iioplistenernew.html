Nuevo listener IIOP

Utilice la página Nuevo listener IIOP para crear un listener IIOP nuevo.

La página Nuevo listener IIOP contiene las siguientes opciones.

Nombre de Configuración

Nombre de la configuración a la que se aplican los valores de esta página. Este campo es de sólo lectura.

Nombre

Nombre único que identifica el listener IIOP.

Dirección de red

Dirección de red del listener IIOP. La dirección de red puede ser una dirección IP o un nombre de host con resolución DNS.

Puerto del Listener

Número de puerto en el que el listener IIOP recibirá las conexiones.

Listener

Estado del listener. Si la casilla de control Activado está activada, estarán activadas las conexiones entrantes ORB IIOP para GlassFish Server. Esta opción se encuentra activada por defecto.

Seguridad

Si la casilla de control Activado está activada, SSL está activado para el listener IIOP. Esta opción se activada por defecto.

Propiedades Adicionales

Propiedades adicionales del listener IIOP. GlassFish Server no define ninguna propiedad adicional para los listener IIOP.
