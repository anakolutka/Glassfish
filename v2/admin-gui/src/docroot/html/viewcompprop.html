To View Properties

  1. In the tree view, click on the JBI node, then the Components node

  2. Click the Components and Service Engines whose properties you want to view.

    The properties page displays the details for the selected Components and Service Engines including its name, type, and metadata.

    In the Clusters profile, the Targets table shows all the targets available for the selected Components and Service Engines and identifies the lifecycle state indicating if the Components and Service Engines is enabled/started, stopped or disabled/shut down. You can use the State drop-down menu to filter the view and see only the Enabled or Disabled targets.

  3. To start a target for a component, click the checkbox next to the target and click Start.

  4. To stop a target, click the checkbox near the target and click Stop.

  5. To shutdown a target, click the check near the target and click Shut Down.
