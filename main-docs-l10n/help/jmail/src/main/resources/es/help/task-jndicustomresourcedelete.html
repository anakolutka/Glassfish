Para suprimir un recurso JNDI personalizado

  1. En el árbol de navegación, amplíe el nodo Recursos.

  2. En el nodo Recursos, amplíe el nodo JNDI.

  3. En el nodo JNDI, seleccione Recursos Personalizados.

    Se abrirá la página Recursos Personalizados.

  4. En la página Recursos Personalizados, active la casilla de control situada junto al nombre del recurso que vaya a suprimir.

  5. Haga clic en Suprimir.
