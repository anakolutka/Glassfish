JVM 監視データを表示する

  1. ナビゲーションツリーで、「Application Server」ノードを選択します。
  2. 「一般情報」ページで、「監視」タブをクリックします。
  3. 「Web コンテナの監視」ページで、「JVM」タブをクリックします。

    「JVM 監視」ページが開きます。

  4. (省略可能) 表示されている監視データを更新するには、「更新」ボタンをクリックします。
