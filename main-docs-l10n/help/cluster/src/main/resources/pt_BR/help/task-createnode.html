Para Criar um Nó

Um nó representa um host no qual o software GlassFish Server está instalado. Um nó deve estar presente em cada host no qual as instâncias GlassFish Server residem

  1. Na árvore de navegação, selecione o nó Nós.

    A página Nós será aberta.

  2. Na página Nós, clique em Novo.

    A página Novo Nó será aberta.

  3. No campo Nome, digite o nome desejado para o nó.

  4. Na lista drop-down Tipo, selecione o tipo do nó a ser criado.

    • Para ativar o nó de modo a suportar a comunicação por meio do protocolo remoto DCOM (Distributed Component Object Model), selecione DCOM.

    • Para criar um nó que ofereça suporte à comunicação através de shell seguro (SSH), selecione SSH.

    • Para criar um nó que não ofereça suporte à comunicação remota, selecione CONFIG.

  5. No campo Host do Nó, digite o nome do host que o nó representa.

    Se este campo é obrigatório ou não, depende do tipo de nó:

    • Se o tipo de nó for CONFIG, o nome do host pode ser omitido. O nome do host pode ser determinado quando as instâncias que residem no nó forem criadas.

    • Se o tipo de nó for DCOM ou SSH, o nome do host deverá ser especificado. Caso contrário, ocorre um erro.

  6. No campo Diretório do Nó, digite o caminho para o diretório que contém as instâncias GlassFish Server que são criadas no nó.

    Se o tipo do nó for CONFIG, deixe este campo vazio. O diretório do nó pode ser determinado quando as instâncias que residem no nó forem criadas.

    Se você digitar um caminho relativo, o caminho é relativo ao diretório as-install, em que as-install corresponde ao diretório de instalação de base do software GlassFish Server no host.

  7. No campo Diretório de Instalação, digite o caminho completo para o pai do diretório de instalação base do software GlassFish Server no host, por exemplo: /export/glassfish3.

    Se o tipo do nó for CONFIG, deixe este campo vazio. O diretório de instalação pode ser determinado quando as instâncias que residem no nó forem criadas.

  8. Finalize a criação do nó, conforme requerido pelo tipo do nó.

    1. Se o tipo de nó for CONFIG, clique em Salvar.

    2. Se o tipo de nó for DCOM, siga as instruções contidas em Para Finalizar a Criação de um Nó do DCOM.

    3. Se o tipo de nó for SSH, siga as instruções contidas em Para Finalizar a Criação de um Nó SSH.
