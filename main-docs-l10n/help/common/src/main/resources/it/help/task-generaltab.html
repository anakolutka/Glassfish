Per visualizzare le informazioni generali per il server DAS

  1. Selezionare il nodo server (server di amministrazione) nell'albero di navigazione.

    Viene visualizzata la pagina Informazioni generali.

  2. Nella pagina Informazioni generali effettuare una delle operazioni seguenti.

    • Consultare la visualizzazione in sola lettura Stato per verificare se il server è in esecuzione.

    • Consultare la visualizzazione in sola lettura Tempo di attività per visualizzare il numero di ore e minuti di attività continua del server.

    • Fare clic sul pulsante Arresta per arrestare GlassFish Server.

    • Fare clic sul pulsante Riavvia per arrestare e riavviare GlassFish Server.

    • Fare clic sul pulsante Visualizza file di log per visualizzare i file di log per i cluster o le istanze di GlassFish Server.

      Per i dettagli, vedere Per visualizzare i file di log del server DAS.

    • Fare clic su Report JVM per aprire una finestra distinta in cui è possibile visualizzare i report relativi alla Java Virtual Machine, tra cui un report di riepilogo, un report sulla gestione della memoria, un report sul caricamento delle classi e un dump del thread corrente.
