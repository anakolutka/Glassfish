Editar servicio de gestión de grupos

Utilice la página Servicios de gestión de grupos para cambiar la configuración de GMS.

El servicio de gestión de grupos (GMS) es un servicio en curso que proporciona supervisión del cluster, pertenencia al cluster y servicios de comunicación de grupos. GMS se basa en una pila configurable de protocolos de detección de pertenencia a grupos y de supervisión de estado. Los GMS realizan las funciones siguientes:

  • Notifican los módulos registrados en una instancia de GlassFish Server cuando se agregan o eliminan una o varias instancias en un cluster, o cuando se sospecha o se confirma que han fallado.

  • Ofrece la posibilidad de enviar y recibir mensajes entre los miembros del cluster.

La página Servicio de gestión de grupos contiene las opciones siguientes.

Cargar Valores por Defecto

Botón para restaurar configuración que tiene valores por defecto a sus valores por defecto. Las configuraciones que no tienen valores por defecto no se modifican.

Nombre de Configuración

Nombre de la configuración a la que se aplican los valores de esta página. Este campo es de sólo lectura.

Número Máximo de Latidos que Faltan

Número de latidos que faltan en un miembro tras el que se creará un fallo en la instancia del servidor. El valor por defecto es 3.

Frecuencia de Latidos

Período de tiempo entre latidos que faltan en un miembro concreto. El valor por defecto es 2.000 milisegundos.

Timeout de Detección de Grupos

Tiempo que espera GMS para detectar otros miembros del cluster. El valor por defecto es 5.000 milisegundos.

Tiempo de Espera para la Verificación de Fallos

Tiempo de espera antes de verificar que un miembro que se cree que ha fallado se debe indicar como tal. El valor por defecto es 1.500 milisegundos.

Propiedades Adicionales

Tabla de propiedades adicionales para el servicio de gestión de grupos.

GlassFish Server no define propiedades adicionales de GMS.

Para cada propiedad se muestra la información siguiente:

Nombre

Nombre de la propiedad.

Valor

Valor de la propiedad que se ha definido para la instancia seleccionada.

Descripción

Descripción textual que ofrece más información sobre la propiedad.

La tabla Propiedades adicionales también contiene las opciones siguientes.

Agregar propiedad

Botón para agregar una propiedad. Al hacer clic en este botón se agrega una fila a la tabla Propiedades adicionales.

Suprimir propiedades

Botón para suprimir una o varias propiedades seleccionadas. Cualquier propiedad que se suprima vuelve a su valor por defecto o, si no se ha definido ningún valor por defecto, se queda sin definir.
