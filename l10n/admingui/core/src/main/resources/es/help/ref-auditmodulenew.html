Módulo de auditoría nuevo

Utilice la página Nuevo módulo de auditoría para definir un nuevo módulo de auditoría.

La página Nuevo módulo de auditoría contiene las siguientes opciones.

Nombre

Nombre del módulo de auditoría nuevo.

Nombre de clase

El nombre completo de la clase que implementa este módulo. El nombre de clase para el módulo de auditoría predeterminado es com.sun.enterprise.security.Audit.

Propiedades adicionales

Propiedades adicionales del módulo de auditoría. Para obtener una descripción de las propiedades adicionales disponibles para los módulos de auditoría, consulte Propiedades específicas de los módulos de auditoría.
