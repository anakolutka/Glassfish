Configuraciones de seguridad de mensajes

La página Configuraciones de seguridad de mensajes muestra una lista de las configuraciones disponibles para los proveedores de seguridad de mensajes.

Para configurar la seguridad de mensajes, necesita un conocimiento básico de seguridad y de conceptos de servicios web. Para obtener más información consulte el Capítulo 13, Administering Message Security de Sun GlassFish Enterprise Server v3 Administration Guide y el Capítulo 6, Administering Web Applications de Sun GlassFish Enterprise Server v3 Administration Guide.

Servidor empresarial implementa la seguridad de los mensajes mediante proveedores de seguridad de mensajes integrados en la capa de procesamiento SOAP. Los proveedores de seguridad de mensajes dependen de otras facilidades de seguridad de Servidor empresarial.

La página Configuraciones de seguridad de mensajes contiene la siguiente información de cada configuración.

Capa de autenticación

Capa del mensaje en la que se lleva a cabo la autenticación. De manera predeterminada, se proporciona una configuración SOAP. También es posible proporcionar una configuración HttpServlet.

Proveedor predeterminado

Proveedor de servidor predeterminado para esta configuración. No hay ningún proveedor de servidor configurado de manera predeterminada.

Proveedor de cliente predeterminado

Proveedor de cliente predeterminado para esta configuración. No hay ningún proveedor de cliente configurado de manera predeterminada.
