So zeigen Sie die Webcontainerüberwachungsdaten an

  1. Wählen Sie in der Navigationsstruktur den Anwendungsserverknoten aus.
  2. Klicken Sie auf der Seite "Allgemeine Informationen" auf die Registerkarte "Überwachen".

    Die Seite "Webcontainerüberwachung" wird angezeigt.

  3. (Optional) Wählen Sie aus der Dropdown-Liste eine Anwendung aus, um die Überwachungsdaten für diese Anwendung anzuzeigen.
  4. (Optional) Wählen Sie aus der Dropdown-Liste "Virtueller Server" einen virtuellen Server aus, um die Überwachungsdaten für diesen Server auszusuchen.
  5. (Optional) Klicken Sie auf die Schaltfläche "Aktualisieren", um die angezeigten Überwachungsdaten zu aktualisieren.
