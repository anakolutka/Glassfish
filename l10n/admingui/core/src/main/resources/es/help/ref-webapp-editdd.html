Descriptores del módulo

Utilice la ficha Descriptor para ver los descriptores del módulo de una aplicación.

La página Descriptores del módulo contiene las siguientes opciones.

Descriptores de implementación

Una tabla que muestra el nombre de aplicación, el del componente (si procede) y el del descriptor de archivo.
