Modifica risorsa connettore

Utilizzare la pagina Modifica risorsa connettore per modificare le impostazioni di una risorsa connettore.

La pagina Modifica risorsa connettore contiene le opzioni riportate di seguito.

Carica impostazioni predefinite

Pulsante per ripristinare i valori predefiniti delle impostazioni che dispongono di valori predefiniti. Le impostazioni che non dispongono di valori predefiniti non vengono modificate.

Nome JNDI

Il nome univoco che identifica la risorsa connettore. Il campo Nome JNDI è di sola lettura. È possibile specificare il nome JNDI solo quando si crea una nuova risorsa connettore.

Nome pool

Il connection pool del connettore associato a questa risorsa.

Ordine di distribuzione

L'ordine di distribuzione della risorsa connettore.

Descrizione

La descrizione della risorsa connettore.

Stato

Indica se la risorsa è disponibile in fase di esecuzione.

Se nel dominio esiste solo l'istanza server predefinita server, è possibile abilitare o disabilitare la disponibilità in fase di esecuzione della risorsa. La disponibilità in fase di esecuzione è abilitata per impostazione predefinita.

Se nel dominio sono stati creati cluster o istanze standalone, utilizzare la scheda Destinazione per controllare la disponibilità in fase di esecuzione della risorsa nelle relative destinazioni.

Proprietà aggiuntive

Proprietà aggiuntive per la risorsa connettore. Le proprietà disponibili dipendono dal Resource Adapter per il connection pool del connettore. Le coppie nome-valore specificate in questa tabella possono essere utilizzate per sostituire i valori predefiniti per le proprietà specificate dal fornitore del Resource Adapter.
