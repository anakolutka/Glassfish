Exercise 0: Configuring your environment

If you are using a lab machine provided at JavaOne the following steps in Exercise 0 have been completed for you.  Please skip this section and go to Exercise 1 if your environment is already configured.

  1. Launch NetBeans 6.0 IDE
    • On Windows, Start > Programs > NetBeans IDE > NetBeans IDE 6.0 > NetBeans IDE or click the NetBeans 6.0 desktop icon
    • On Solaris, UNIX, or Linux, open a terminal window, then type <netbeans60_HOME>/bin/netbeans
  2. Install the Maven 2 plugin for NetBeans 6.0 if you do not have that installed.  (TBD, this image shows the plugin installed.  should redo this)
    • Choose Plugins from the Tools menu.  Select the Available Plugins Tab.
    • Select the Maven 2 Plugin and press the Install Button

  3. Installing Subversion is optional.   Subversion is an open source version control system.  For the purposes of this lab you are not required to checkout the Glassfish V3 source code since it has already been installed for you.  If you would like to view the latest source code this plugin is necessary.  In step 4 you will learn how to checkout Glassfish V3 and build it.
    • Choose Plugins from the Tools menu.  Select the Available Plugins Tab.
    • Select the Subversion Plugin and press the Install Button
           

  4. This step is also optional and is not required for this lab.  A snap shot of the  GlassFish V3 source code is included in this lab's zip file, and GlassFish V3 has been installed for you under <lab4520_guiPlugin>/v3/glassfish.   However, if you want to checkout and build the latest version of the source code, then do the following:

    Checkout Source code

        svn checkout https://svn.dev.java.net/svn/glassfish-svn/trunk/v3

        Build the GlassFish server
        cd v3;   mvn install

        Installing the server by unzipping the image.   The instructions in the following exercise is based on the assumption that the installation is at v3/glassfish.
        cd v3;  unzip v3/distribution/glassfish/target/glassfish.zip

        For more information regarding GlassFish V3 development environment and the workspace, please refer to the full build instructions at http://wiki.glassfish.java.net/Wiki.jsp?page=V3FullBuildInstruction"
