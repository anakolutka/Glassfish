JMS 대상 리소스를 편집하는 방법

  1. 탐색 트리에서 [리소스] 노드를 확장합니다.

  2. [리소스] 노드에서 [JMS 리소스] 노드를 확장합니다.

  3. [JMS 리소스] 노드에서 [JMS 대상 리소스] 노드를 선택합니다.

    [JMS 대상 리소스] 페이지가 열립니다.

  4. [JMS 대상 리소스] 페이지에서 편집할 대상 리소스의 이름을 누릅니다.

    [JMS 대상 리소스 편집] 페이지가 열립니다.

  5. [물리적 대상 이름] 필드에 이 리소스와 연관될 물리적 대상의 이름을 입력합니다.

    GlassFish Server에서는 물리적 대상이 필요할 때 자동으로 물리적 대상을 생성하고 사용자가 대상 리소스를 삭제할 때 물리적 대상을 삭제합니다.

  6. [리소스 유형] 드롭다운 목록에서 javax.jms.Topic 또는 javax.jms.Queue를 선택합니다.

  7. [설명] 필드에 대상 리소스에 대한 설명을 입력합니다.

  8. 런타임에 리소스를 사용으로 설정하려면 [상태 사용] 체크 박스를 선택합니다.

    이 옵션은 기본적으로 사용으로 설정되어 있습니다.

  9. [추가 속성] 영역에서 대상 리소스에 대한 추가 속성을 지정합니다.

    GlassFish Server에서는 JMS 대상 리소스에 대한 추가 속성을 정의하지 않습니다.

    • 속성을 추가하려면 [속성 추가] 단추를 누릅니다. 빈 행이 표시되면 [이름] 필드에 속성 이름을 입력하고 [값] 필드에 속성값을 입력합니다.

    • 속성을 수정하려면 속성의 [값] 필드를 편집합니다.

    • 속성을 삭제하려면 삭제할 속성의 [이름] 필드 왼쪽에 있는 체크 박스를 선택한 다음 [속성 삭제] 단추를 누릅니다.

  10. [저장]을 누릅니다.

    팁:

    기본 설정으로 돌아가려면 [기본값 로드] 단추를 누른 다음 [저장]을 누릅니다.

  11. 선택적으로 [대상] 탭을 눌러 [JMS 대상 리소스 대상] 페이지로 이동합니다. 이 페이지에서 다음을 수행합니다.

    1. [대상 관리]를 눌러 [리소스 대상 관리] 페이지를 엽니다.

      이 페이지에서 [사용 가능] 열과 [선택됨] 열 간에 대상을 이동합니다. 리소스를 사용하는 응용 프로그램이 배치될 대상을 [선택됨] 열에 배치해야 합니다. 사용 가능한 대상에는 기본 서버 인스턴스인 서버뿐 아니라 사용 가능한 클러스터 및 서버 인스턴스가 포함됩니다. [확인]을 눌러 변경 사항을 저장합니다.

    2. 대상에 대한 체크 박스를 선택한 다음 [사용] 또는 [사용 안함]을 눌러 해당 대상에 대한 리소스를 사용 또는 사용 안함으로 설정합니다.
