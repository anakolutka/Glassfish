稽核模組

稽核是指 Enterprise Server 如何記錄重要事件，例如錯誤或安全性漏洞。Enterprise Server 在記錄檔中記錄所有認證事件。完整的存取記錄提供了 Enterprise Server 存取事件的循序追蹤。

[稽核模組] 頁面會顯示可用稽核模組的清單。對於每個稽核模組，均提供以下資訊。

名稱

稽核模組的名稱。

類別名稱

實作此模組的類別之完全合格的名稱。預設稽核模組的類別名稱為 com.sun.enterprise.security.Audit。
