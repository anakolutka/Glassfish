To Edit a Notification Rule

  1. Do one of the following:
    • For the developer profile, in the tree component, select the Configuration node.
    • For the cluster or enterprise profile, in the tree component select the Configurations node, then select the configuration to edit.
  2. Select the Management Rules node.
  3. Select the management rule to edit.

    The Edit Notification Event page appears.

  4. Check Enabled to activate this rule.
  5. Change the rule description, if needed.
  6. Select the Record Event checkbox to log the occurrence of the specified event.
  7. Change the log level for this management rule, if needed.
  8. Select the deployed custom MBean from the drop-down list or specify the Object Name of the custom MBean.

    The Object Name is of the form = domain:type=implementation-class-name,name=implementation-class-name, where domain is the JMX Domain where the MBean is registered.

  9. Specify the notification type.
  10. Enter a description for the event associated with this management rule.
  11. Specify the action you want performed when the monitored data meets the condition specified in the event type.
  12. Click Save.
