Per visualizzare le informazioni dettagliate su un componente

In Strumento di aggiornamento sono disponibili informazioni dettagliate sui componenti installati, sugli aggiornamenti disponibili per tali componenti e sui componenti aggiuntivi.

  1. Selezionare il nodo Strumento di aggiornamento nell'albero di navigazione.

    Viene visualizzata la pagina Componenti aggiuntivi disponibili.

  2. Fare clic sulla scheda corrispondente alla pagina in cui è elencato il componente.

    Il componente è elencato in una delle pagine seguenti:

    • Componenti aggiuntivi disponibili

    • Aggiornamenti disponibili

    • Componenti installati

  3. Selezionare il componente nella tabella dei componenti.

    Viene visualizzata la pagina Dettagli componenti per il componente selezionato.
