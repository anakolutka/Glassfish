Nuevo Recurso Personalizado

Utilice la página Nuevo recurso personalizado para crear un nuevo recurso JNDI personalizado.

La página Nuevo recurso personalizado contiene las siguientes opciones.

Nombre JNDI

Nombre único que identifica el recurso.

Tipo de recurso

Definición de tipo totalmente cualificada para el recurso.

Clase de fábrica

Nombre especificado por el usuario para la clase de fábrica. Esta clase implanta la interfaz javax.naming.spi.ObjectFactory.

Descripción

Descripción de este recurso personalizado.

Estado

Estado del recurso personalizado. El recurso personalizado se puede activar o desactivar. El valor por defecto es Activado.

Propiedades Adicionales

Propiedades adicionales para el recurso personalizado. GlassFish Server no define ninguna propiedad adicional de los recursos personalizados.

Destinos

(Esta opción sólo se muestra si en el dominio se ha creado algún cluster o instancia independiente).

Clusters e instancias independientes para el recurso. Mueva los destinos seleccionados a la columna Destinos Seleccionados mediante los botones Agregar y Agregar Todos. Mueva los destinos que no necesite a la columna Destinos Disponibles mediante los botones Eliminar y Eliminar Todos.
