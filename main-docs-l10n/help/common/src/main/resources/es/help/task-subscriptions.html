Ver la página Suscripciones para GlassFish Server

  1. En el árbol de navegación, seleccione el nodo Tareas comunes.

    Se abrirá la página Tareas comunes.

  2. En Noticias de GlassFish, haga clic en el enlace Suscripciones.

    Se abrirá la página web Suscripciones de GlassFish en un separador o una ventana del explorador independiente.
