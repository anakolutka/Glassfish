Edición de un módulo de ciclo de vida

  1. En el árbol de navegación, seleccione el nodo Módulo de ciclo de vida.

    Se abrirá la página Módulos de ciclo de vida.

  2. En la página Módulos de ciclo de vida, haga clic en el nombre del módulo de ciclo de vida que está editando.

    Se abrirá la página Editar módulo de ciclo de vida.

  3. En el campo Nombre de clase, escriba el nombre completo del archivo de clase del módulo de ciclo de vida.

    La clase debe implementar la interfaz com.sun.appserv.server.LifecycleListener.

  4. En el campo ruta de clase, introduzca la ruta de clase para el módulo de ciclo de vida.

    La ruta de clase especifica dónde está ubicado el módulo de ciclo de vida. El valor predeterminado es domain-dir/applications. Si el módulo ya se encuentra en este directorio (es decir, en la ruta de clase del servidor), este campo puede dejarse en blanco.

  5. En el campo Orden de carga, introduzca un valor que especifique el orden en que este módulo de ciclo de vida se cargará en el arranque.

    Los módulos con valores de orden de carga inferiores se cargarán antes. El valor puede estar entre 101 y el MAXINT del sistema operativo. Los valores de 0 a 100 están reservados.

  6. Seleccione la casilla de verificación Estado habilitado para habilitar el módulo de ciclo de vida.

    Esta opción está habilitada por defecto.

  7. Seleccione la casilla de verificación Arranque de instancia para impedir fallo en la carga para especificar que el servidor debe apagarse si se produce un error en la carga del módulo de ciclo de vida.

    Esta opción se encuentra deshabilitada de forma predeterminada.

  8. En la lista desplegable Servidores virtuales, seleccione el servidor o servidores virtuales que se asociará con el módulo de ciclo de vida.

    De manera predeterminada no hay ningún servidor virtual seleccionado.

  9. En el campo Descripción, introduzca una descripción del módulo de ciclo de vida.
  10. (Opcional) En la sección Propiedades adicionales, especifique las propiedades adicionales.

    Servidor empresarial no define ninguna propiedad adicional para los módulos de ciclo de vida.

    • Para añadir una propiedad, haga clic en el botón Agregar propiedad. En la fila vacía escriba el nombre de la propiedad en el campo Nombre y el valor de la propiedad en el campo Valor.
    • Para modificar una propiedad, edite el campo Valor de la propiedad.
    • Para eliminar una propiedad, marque la casilla de verificación situada a la izquierda del campo Nombre de la propiedad que quiera eliminar y, a continuación, haga clic en el botón Eliminar propiedades.
  11. Haga clic en Guardar.

    Consejo - Para volver a los valores de configuración predeterminados, haga clic en el botón Cargar predeterminados y, a continuación, en Guardar.
