구성 요소를 제거하는 방법

구성 요소 제거에는 다음 항목의 제거가 포함됩니다.

  • 구성 요소의 원래 설치에 의해 적용된 모든 파일 및 디렉토리

  • 원래 설치 후 구성 요소에 대한 업데이트에 의해 적용된 모든 파일 및 디렉토리

  1. 탐색 트리에서 업데이트 도구 노드를 선택합니다.

    사용 가능한 애드온 페이지가 열립니다.

  2. 사용 가능한 애드온 페이지에서 설치된 구성 요소 탭을 누릅니다.

    설치된 구성 요소 페이지가 열립니다.

  3. 설치된 구성 요소의 테이블에서 제거할 구성 요소를 선택합니다.
    • 개별 구성 요소를 선택하려면 구성 요소 이름 옆에 있는 확인란을 선택합니다.
    • 모든 구성 요소를 선택하려면 테이블 헤더에서 모두 선택 아이콘을 누릅니다.
    • 모든 구성 요소를 선택 해제하려면 테이블 헤더에서 모두 선택 해제 아이콘을 누릅니다.
  4. 제거를 누릅니다.

    선택한 구성 요소를 제거해야 하는지 확인하라는 대화 상자가 표시됩니다.

  5. 대화 상자에서 확인을 누릅니다.

    Enterprise Server는 제거가 완료되었음을 확인합니다. 구성 요소가 설치된 구성 요소 테이블에서 제거되고 사용 가능한 애드온 페이지의 사용 가능한 애드온 테이블에 추가됩니다.

참조

  • 네트워크 저장소 액세스를 위해 네트워크 프록시 설정을 구성하는 방법

  • 구성 요소에 대한 업데이트를 설치하는 방법

  • 애드온 구성 요소 설치 방법

  • 구성 요소에 대한 자세한 내용을 보는 방법
