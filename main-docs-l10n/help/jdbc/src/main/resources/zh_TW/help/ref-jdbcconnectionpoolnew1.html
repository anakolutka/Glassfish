新建 JDBC 連線集區 (步驟 1，共 2 個步驟)

使用「新建 JDBC 連線集區 (步驟 1，共 2 步)」頁面可建立新的 JDBC 連線集區。

「新建 JDBC 連線集區 (步驟 1，共 2 步)」頁面包含下列選項。

集區名稱

連線集區的名稱。

資源類型

可用的資源類型包括 javax.sql.XADataSource (全域作業事件)、java.sql.ConnectionPoolDataSource (本機作業事件，可能可提昇效能)、javax.sql.DataSource (僅限於本機作業事件) 以及 java.sql.Driver。

資料庫驅動程式供應商

資料庫產品的供應商名稱。

自我檢查

如果啟用此選項，當您按一下「下一步」按鈕時，資料庫驅動程式會依據「資源類型」和「資料庫驅動程式供應商」進行自我檢查，並在新建 JDBC 連線集區 (步驟 2，共 2 個步驟)頁面中顯示任何探索到的資料來源或驅動程式類別名稱。

為使自我檢查能順利運作，GlassFish Server 必須能夠使用 JDBC 驅動程式。請參閱整合 JDBC 驅動程式。
