To Monitor Log Statistics for Cluster Instance

The Log Statistics page displays the available log records. By default, the log statistics are retained for five hours.

  1. In the Admin Console tree, select the Clusters node.
  2. In the Clusters page, click the desired server instance.
  3. Click the Monitor tab.

    The Log Statistics Monitoring page shows a list of log files, sorted by timestamp, along with the number of severe-level log messages and warning-level log messages in each log file. To sort the list by each column's value, click on that column.

  4. Click the radio button next to the log file you wish to view, then click Show Loggers.

    The page will show the loggers that have generated the log messages, along with the number of severe and warning messages. Click on the number of messages of each type to display the log records. Click Close when you are done.

  5. Click Refresh to refresh the list of log records.
