Configuration System Properties

In this page, you can do the following:

  • add system properties for a configuration. To add a property, click Add Property, type in the property name and value and click Save.

  • modify values of existing properties.

  • enable dynamic reconfiguration. If enabled, changes to the configuration are applied to the server instances without a server restart.

  • edit property values for all instances using this configuration. To do this, click Instance Values.

The following table describes the properties predefined for a configuration.

Property

Description

JMS_PROVIDER_PORT

Port number on which the JMS provider listens.

JMX_SYSTEM_CONNECTOR_PORT

Port number on which the JMX connector listens.

IIOP_SSL_LISTENER_PORT

ORB listener port for IIOP connections on which IIOP listener SSL listens.

HTTP_SSL_LISTENER_PORT

Port number for http-listener-2.

HTTP_LISTENER_PORT

Port number for http-listener-1.

IIOP_SSL_MUTUALAUTH_PORT

ORB listener port for IIOP connections on which the IIOP listener SSL_MUTUALAUTH listens.

IIOP_LISTENER_PORT

ORB listener port for IIOP connections on which orb-listener-1 listens.
