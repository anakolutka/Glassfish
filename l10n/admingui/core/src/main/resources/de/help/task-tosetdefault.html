So legen Sie den Standardbereich fest

Der Bereich default realm ist der Bereich, den Enterprise Server zur Authentifizierung und Autorisierung verwendet, wenn in der Bereitstellungsbeschreibung einer Anwendung kein Bereich angegeben ist.

  1. Erweitern Sie in der Navigationsstruktur den Konfigurationsknoten.
  2. Wenn die Domäne für die Unterstützung von Clustern konfiguriert ist, wählen Sie die zu ändernde Konfiguration aus.

    Beispiele sind Domänen, die mit dem Profil Cluster oder 'Unternehmen' erstellt werden.

  3. Wählen Sie den Knoten "Sicherheit".

    Die Seite "Sicherheit" wird geöffnet.

  4. Wählen Sie im Feld "Standardbereich" den gewünschten Bereich aus der Dropdown-Liste aus.
  5. Klicken Sie auf "Speichern", um die Änderungen zu speichern.
  6. Starten Sie den Server neu, wenn in der Konsole "Neustart erforderlich" angezeigt wird.
