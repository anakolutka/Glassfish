编辑上下文服务

  1. 在导航树中，展开“资源”节点。

  2. 在“资源”节点下，展开“并发资源”节点。

  3. 在“并发资源”节点下，选择“上下文服务”。

    此时将打开“上下文服务”页。

  4. 在“上下文服务”页上，单击要编辑的资源的名称。

    此时将打开“编辑上下文服务”页。

    注:

    JNDI 名称是只读字段。只能在创建新资源时指定 JNDI 名称。逻辑 JNDI 名称也是只读字段。

  5. 选择“启用上下文信息”复选框以将上下文传播到线程，然后选择要传播的上下文。

    默认情况下此选项已启用，并且选中所有上下文。通过按住 Shift 键单击或按住 Ctrl 键单击选择多个上下文。如果禁用了此选项，则不传播上下文，即使已选中了它们也是如此。

  6. 在“部署顺序”字段中，键入一个数字以指定资源的部署顺序。

    在服务器启动时先加载编号较低的资源。默认值为 100。

  7. 在“说明”字段中，键入资源的简短说明。

    此说明是字符串值，最多可以包含 250 个字符。

  8. 选择“启用状态”复选框以启用资源。

  9. 在“其他属性”部分中，不要添加任何其他属性。

    GlassFish Server 没有为此资源类型定义任何其他属性。此外，此资源类型当前不使用任何其他属性。

  10. 单击“保存”。
