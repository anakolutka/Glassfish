監査モジュール

監査は、Enterprise Server がエラーやセキュリティー侵入などの重大なイベントを記録する方法です。Enterprise Server は、すべての認証イベントをログファイルに記録します。完全なアクセスログには、Enterprise Server で行われるすべてのアクセスイベントが連続して記録されます。

「監査モジュール」ページには、使用可能な監査モジュールのリストが表示されます。監査モジュールごとに、次の情報が表示されます。

名前

監査モジュールの名前。

クラス名

このモジュールを実装するクラスの完全修飾名。デフォルトの監査モジュールのクラス名は、com.sun.enterprise.security.Audit です。
