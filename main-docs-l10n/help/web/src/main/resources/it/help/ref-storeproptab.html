Proprietà area di memorizzazione

Utilizzare la pagina Proprietà area di memorizzazione è possibile aggiungere o modificare le proprietà di persistenza (memorizzazione) delle sessioni per il contenitore Web.

La pagina Proprietà area di memorizzazione contiene le opzioni riportate di seguito.

Nome configurazione

Il nome della configurazione a cui si applicano le impostazioni presenti in questa pagina. Questo campo è di sola lettura.

Directory

Il percorso assoluto o relativo della directory in cui vengono scritti i singoli file di sessione. Il valore predefinito è domain-dir/generated/jsp/j2ee-apps/nome_app/nome_app_war. Il percorso relativo è relativo rispetto alla directory di lavoro temporanea di questa applicazione Web.

Proprietà aggiuntive

Proprietà aggiuntive per l'area di memorizzazione. GlassFish Server non definisce alcuna proprietà aggiuntiva per l'area di memorizzazione.
