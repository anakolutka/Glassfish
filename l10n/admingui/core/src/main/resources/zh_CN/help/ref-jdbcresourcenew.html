新建 JDBC 资源

J.N.D.I. 名称

J.N.D.I. 名称可以在一个分布式计算环境中组织和定位组件，这与使用卡片目录组织和表示图书馆中书籍位置的方法类似。因此，J.N.D.I. 名称是访问 JDBC 资源的重要方法之一。按照约定，名称以 jdbc/ 字符串开头，例如 jdbc/payrolldb。请不要忘记正斜杠。

池名称

与 JDBC 资源关联的连接池。

JDBC 资源的简短说明。

状态

JDBC 资源的状态。可以启用或禁用资源。
