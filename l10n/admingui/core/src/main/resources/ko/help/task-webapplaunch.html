웹 응용 프로그램을 시작하는 방법

시작하기 전에

시작할 웹 응용 프로그램이 배포되었는지 확인합니다.

  1. 탐색 트리에서 응용 프로그램 노드를 선택합니다.
  2. 웹 응용 프로그램 노드를 선택합니다.

    웹 응용 프로그램 페이지가 열립니다.

  3. 배포된 웹 응용 프로그램 테이블에서 시작할 응용 프로그램 행에서 시작을 누릅니다.

    응용 프로그램이 새 브라우저 창에서 열립니다.

참조

  • 웹 응용 프로그램을 배포하는 방법

  • 배포된 웹 응용 프로그램을 편집하는 방법

  • 웹 응용 프로그램의 설명자를 보는 방법

  • 배포된 웹 응용 프로그램을 재배포하는 방법
