Using the Clusters Common Tasks Page

The Clusters Profile Common Tasks page provides handy shortcuts for some of the most common Administration Console tasks. You can create clusters, deploy applications, create standalone instances, and access documentation.

The tasks available in the Common Tasks page varies depending on the profile you are running.

Use the Common Tasks page for the following operations:

  • To Create a Cluster

  • To Create a New Standalone Server Instance

  • To deploy an application

  • To deploy a web application

  • To create a JDBC connection pool

  • To view monitoring data

The Common Tasks page also provides links to the Application Server documentation.
