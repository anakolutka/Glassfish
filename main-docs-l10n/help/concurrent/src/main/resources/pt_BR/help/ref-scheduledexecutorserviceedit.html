Editor Serviço do Executor Programado Gerenciado

Use a página Editor de Serviço do Executor Programado Gerenciado para criar um recurso de serviço do executor gerenciado programado.

A página Editar Serviço de Contexto contém as opções a seguir.

Carregar Defaults

Botão para restaurar definições para seus valores default. Definições que não tiveram seus valores default alterados.

Nome JNDI

Um nome exclusivo que identifica o recurso. O nome JNDI é um campo somente para leitura. Você pode especificar o nome JNDI ao criar um novo recurso.

Informações de Contexto

O contexto do recurso: Classloader, JNDI, Segurança ou Área de Trabalho.

Status

O status do recurso. O recurso pode ser ativado ou desativado. O valor default é Ativado.

Prioridade do Thread

A prioridade de threads a ser criada para a instância de serviço do executor programado gerenciado. O default é 5 .

Tarefas de Longa Execução

XXX.

Suspensa Após

O tempo em segundos que uma tarefa pode ser executada antes de ser considerada suspensa.

Tamanho do Principal

O número de threads a serem mantidos em um pool de thread. O default é 0.

Tamanho Máximo do Pool

O número máximo de threads que um pool pode conter. O valor default é 2147483647.

Conexão Ativa

O tempo em segundos que o thread pode permanecer inativo quando o número de threads for maior que o tamanho do principal. O default é 60.

Vida Útil do Thread

O tempo em segundos que cada thread permanece em um pool de threads antes de ser expurgado, independentemente se o número de threads é maior que o tamanho do principal ou se o thread está inativo.

Capacidade da Fila da Tarefa

O tamanho da fila que mantém as tarefas submetidas que estão aguardando para serem executadas. O default é 2147483647 .

Descrição

Uma descrição deste recurso.

Propriedades Adicionais

Propriedades adicionais para o recurso personalizado. O GlassFish Server não define quaisquer propriedades adicionais para recursos de serviço do executor gerenciado programado.
