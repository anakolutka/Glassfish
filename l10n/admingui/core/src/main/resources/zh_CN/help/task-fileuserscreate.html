向 File 领域添加用户

  1. 在导航树中，展开“配置”节点。
  2. 展开“安全性”节点。
  3. 展开“领域”节点。
  4. 选择 "File" 节点。

    将打开“编辑领域”页面。

  5. 在“编辑领域”页面上，单击“管理用户”按钮。

    将打开“文件用户”页面。

  6. 在“文件用户”页面上，单击“新建”。

    将打开“新建 File 领域用户”页面。

  7. 在“用户 ID”字段中，键入用户的名称。
  8. 在“组列表”字段中，键入用户所属的组列表。

    使用逗号分隔组名。无需在其他位置定义这些组。

  9. 在“密码”字段中，键入一个唯一的密码。
  10. 在“确认新密码”字段中，再次键入用户密码。
  11. 单击“确定”。
