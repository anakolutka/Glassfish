使用【升級精靈】進行升級

使用【升級精靈螢幕】來指定來源和目標伺服器的安裝目錄、來源應用程式伺服器的管理使用者、管理密碼以及主密碼。

  1. 在【來源安裝目錄】欄位中，輸入要從其中匯入配置的現有安裝之位置。您必須輸入網域目錄。例如，<install-root>/domains/domain1。

  2. 在【目標安裝目錄】欄位中，輸入要將配置傳輸到 Application Server 安裝的哪一個位置。您必須提供網域根目錄做為此欄位的輸入資料。
  3. 提供來源應用程式伺服器的管理使用者名稱、管理密碼及主密碼。
  4. 完成後，請按【下一步】按鈕。
  5. 將顯示【升級結果】面板，指示升級作業的狀態。
  6. 當升級程序完成時，按一下【完成】按鈕以關閉升級工具。
