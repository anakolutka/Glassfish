Nachrichtensicherheitskonfiguration

Auf der Seite "Nachrichtensicherheitskonfigurationen" können Sie die verfügbaren Konfigurationen für Nachrichtensicherheitsprovider anzeigen und ändern.

Um Nachrichtensicherheit zu konfigurieren, benötigen Sie ein grundlegendes Verständnis von Sicherheits- und Webservicekonzepten. Weitere Informationen finden Sie in Chapter 3, Administering Message Security, im Oracle GlassFish Server Security Guide und Chapter 6, Administering Web Applications, im Oracle GlassFish Server Administration Guide.

GlassFish Server implementiert die Nachrichtensicherheit mithilfe von Nachrichtensicherheitsprovidern, die auf SOAP-Verarbeitungsebene integriert sind. Die Nachrichtensicherheitsprovider hängen von den Sicherheitsfunktionen von GlassFish Server ab.

Für jede Konfiguration werden die folgenden Informationen zur Verfügung gestellt.

Authentifizierungslayer

Die Nachrichtenlayer, in der Authentifizierung durchgeführt wird. Standardmäßig wird eine SOAP-Konfiguration bereitgestellt. Es ist darüber hinaus möglich, eine HttpServlet-Konfiguration bereitzustellen.

Standardprovider

Der Standardserverprovider für diese Konfiguration. Standardmäßig ist kein Standardserverprovider konfiguriert.

Standardclientprovider

Der Standardclientprovider für diese Konfiguration. Standardmäßig ist kein Standardclientprovider konfiguriert.

Die Tabelle "Nachrichtensicherheitskonfiguration" enthält auch die folgenden Optionen.

Neu

Schaltfläche zum Erstellen einer neuen Konfiguration.

Löschen

Schaltfläche zum Löschen von mindestens einer ausgewählten Konfiguration.

Verwandte Aufgaben

  • So werden GlassFish Server-Funktionalitäten zur Verwendung durch Nachrichtensicherheitsprovider konfiguriert

  • So wird eine Nachrichtensicherheitskonfiguration erstellt

  • So wird eine Nachrichtensicherheitskonfiguration bearbeitet

  • So wird eine Nachrichtensicherheitskonfiguration gelöscht

  • So wird eine Nachrichtensicherheitsproviderkonfiguration erstellt

  • So wird eine Nachrichtensicherheitsproviderkonfiguration bearbeitet

  • So wird eine Nachrichtensicherheitsproviderkonfiguration gelöscht

  • So wird die Nachrichtensicherheit für Anwendungsclients aktiviert
