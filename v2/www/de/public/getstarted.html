GlassFish ist ein kostenloser Open Source-Anwendungsserver, der die neuesten Features der Java EE 5-Plattform implementiert. Die Java EE 5-Plattform umfasst neueste Versionen von Technologien wie JavaServer Pages (JSP) 2.1, JavaServer Faces (JSF) 1.2, Servlet 2.4, Enterprise JavaBeans 3.0, Java API for Web Services (JAX-WS) 2.0, Java Architecture for XML Binding (JAXB) 2.0, Web Services Metadata für Java 1.0 sowie zahlreiche weitere Technologien.

Neu bei GlassFish? Erfahren Sie, wie Sie GlassFish herunterladen und installieren, machen Sie erste Schritte mit GlassFish, zeigen Sie einige Screencasts an, und testen Sie die Beispiele im Lernprogramm. Weitere Informationen finden Sie auf der Seite "Erste Schritte".

Download und Ausführung

Sie können Early-Access-Versionen des GlassFish-Anwendungsservers herunterladen. Der Download umfasst eine Binärversion des GlassFish Java EE 5-Anwendungsservers, Sie müssen diesen lediglich extrahieren und ausführen. Es stehen Downloads für die Ausführung auf allen gängigen Betriebssystemen bereit.
