GlassFish는 Java EE 5 플랫폼의 최신 기능을 구현하는 무료 오픈 소스 응용 프로그램 서버입니다. Java EE 5 플랫폼에는 JavaServer Pages(JSP) 2.1, JavaServer Faces(JSF) 1.2, Servlet 2.4, Enterprise JavaBeans 3.0, Java API for Web Services(JAX-WS) 2.0, Java Architecture for XML Binding(JAXB) 2.0, Web Services Metadata for the Java Platform 1.0 등과 같은 다양한 최신 버전 기술이 포함되어 있습니다. 

GlassFish의 신규 사용자이십니까? GlassFish를 다운로드하여 설치하는 방법, 시작 설명서 사용 방법, 스크린캐스트 보는 방법, 자습서에서 샘플을 사용하는 방법 등을 참조하십시오. 시작하기 페이지를 참조하십시오.
