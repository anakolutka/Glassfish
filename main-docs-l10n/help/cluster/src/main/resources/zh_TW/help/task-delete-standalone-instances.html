刪除獨立 GlassFish Server 執行處理

  1. 在瀏覽樹狀結構中，選取「獨立執行處理」節點。

    此時將開啟「獨立伺服器執行處理」頁面。

  2. 針對您要刪除的每個執行處理，請在伺服器執行處理表中選取該執行處理旁邊的核取方塊。

  3. 按一下「刪除」。

    系統會要求您確認是否要刪除這些執行處理。

  4. 請確認您要刪除這些執行處理。

    這些執行處理會刪除，並從頁面上的執行處理表中移除。
