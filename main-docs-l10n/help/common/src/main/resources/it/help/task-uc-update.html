Per visualizzare gli aggiornamenti disponibili per i componenti

La pagina Aggiornamenti disponibili fornisce informazioni sulle versioni più recenti dei componenti dell'immagine di installazione installati.

  1. Selezionare il nodo Strumento di aggiornamento nell'albero di navigazione.

    Viene visualizzata la pagina Componenti aggiuntivi disponibili.

  2. Nella pagina Componenti aggiuntivi disponibili fare clic sulla scheda Aggiornamenti disponibili.

    Viene visualizzata la pagina Aggiornamenti disponibili. Se non è disponibile alcun aggiornamento, la tabella è vuota.

Passi successivi

Per installare gli aggiornamenti disponibili, è necessario in primo luogo arrestare il dominio di GlassFish Server. Utilizzare quindi uno dei comandi riportati di seguito.

  • as-install-parent/bin/updatetool, che consente di avviare la utility grafica Strumento di aggiornamento.

  • as-install-parent/bin/pkg, la versione da riga di comando dello strumento di aggiornamento.

Per informazioni sull'uso del comando pkg, vedere il Capitolo 11, Extending and Updating GlassFish Server, in Oracle GlassFish Server Administration Guide. La utility Strumento di aggiornamento dispone di una Guida in linea specifica.

Una volta completata l'installazione degli aggiornamenti, riavviare il server.
