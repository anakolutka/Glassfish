Usuários do Arquivo

Gerencie os usuários do domínio arquivo com o Console de Administração. Usuários e grupos do domínio arquivo são listados no arquivo principal, cuja localização está especificada na propriedade arquivo.

Observação - Também é possível usar essas etapas para adicionar usuários a qualquer domínio arquivo, incluindo o domínio admin. Basta substituir o nome do domínio de destino no lugar do domínio arquivo mencionado nesta seção.

Um usuário do domínio arquivo pode pertencer a um grupo Java, uma categoria de usuários classificada por características comuns. Por exemplo, os clientes de um aplicativo de e-commerce podem pertencer ao grupo CLIENTE, mas os grandes consumidores podem pertencer ao grupo PREFERENCIAL. A categorização de usuários em grupos facilita ainda mais o controle do acesso de um grande número de usuários.

Inicialmente, após a instalação do Enterprise Server, o único usuário é o administrador especificado durante a instalação. Por padrão, esse usuário pertence ao grupo asadmin, no domínio admin, o que lhe direitos para modificar o Enterprise Server. Todos os usuários atribuídos a esse grupo terão privilégios de administrador, ou seja, eles terão acesso à ferramenta asadmin e ao Console de Administração.

A página Usuários do Arquivo exibe uma lista de usuários do domínio selecionado. Para cada usuário, as seguintes informações são fornecidas:

ID do Usuário

O nome de usuário.

Lista de Grupos

Os grupos aos quais o usuário pertence.
