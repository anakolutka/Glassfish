HTTP 수신기를 삭제하는 방법

시작하기 전에

기본 HTTP 수신기 http-listener-1, http-listener-2 또는 admin-listener를 삭제하려면 필요한 경우 설정을 복구할 수 있도록 먼저 Enterprise Server’s domain.xml 파일의 http-listener 요소에 대한 백업 복사본을 만듭니다.

  1. 탐색 트리에서 구성 노드를 선택합니다.
  2. 구성 노드에서 HTTP 서비스 노드를 확장합니다.
  3. HTTP 서비스 노드에서 HTTP 수신기 노드를 선택합니다.
  4. HTTP 서비스 페이지에서 삭제할 HTTP 수신기 이름 옆의 확인란을 선택합니다.
  5. 삭제를 누릅니다.
참조

  • HTTP 수신기를 만드는 방법

  • SSL을 사용하도록 HTTP 수신기를 구성하는 방법

  • HTTP 수신기를 편집하는 방법
