Sessioni JavaMail

GlassFish Server include l'interfaccia API JavaMail. JavaMail consente di accedere dalle applicazioni Java ai server di posta che supportano i protocolli IMAP (Internet Message Access Protocol) e SMTP (Simple Mail Transfer Protocol) nella rete locale o su Internet.

Utilizzare la pagina Sessioni JavaMail per configurare le sessioni JavaMail.

Per ogni sessione JavaMail vengono fornite le informazioni riportate di seguito.

Nome JNDI

Il nome univoco che identifica la sessione JavaMail.

Abilitato

L'impostazione è True se la sessione è abilitata oppure è False se la sessione è disabilitata.

Descrizione

La descrizione della sessione JavaMail.

La tabella Sessioni contiene inoltre le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare una nuova sessione JavaMail.

Elimina

Pulsante che consente di eliminare una o più sessioni JavaMail selezionate.

Abilita

Pulsante che consente di abilitare una o più sessioni JavaMail selezionate.

Disabilita

Pulsante che consente di disabilitare una o più sessioni JavaMail selezionate.
