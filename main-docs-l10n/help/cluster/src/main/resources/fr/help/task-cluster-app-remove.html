Suppression d'applications d'un cluster

La suppression d'une application enlève l'application de la cible en cours uniquement. Lorsqu'une application est enlevée d'une cible, elle reste répertoriée sur la page Applications, qui affiche toutes les applications déployées dans GlassFish Server. L'application reste également déployée sur les autres cibles sur lesquelles elle a été déployée.

  1. Dans l'arborescence de navigation, développez le noeud Clusters.

  2. Sous le noeud Clusters développé, sélectionnez le noeud correspondant au cluster.

    La page Informations générales du cluster apparaît.

  3. Sur la page Informations générales, cliquez sur l'onglet Applications.

    La page Applications du cluster apparaît.

  4. Pour chaque application à enlever, cochez la case en regard de l'application dans la table Applications déployées.

  5. Cliquez sur Enlever.

    Vous êtes invité à confirmer que vous souhaitez enlever les applications.

  6. Confirmez la suppression des applications.

    Les applications sont enlevées de la table des applications déployées sur la page.
