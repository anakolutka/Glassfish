ライフサイクルモジュール

「ライフサイクルモジュール」ページには、使用可能なライフサイクルモジュールのリストが表示されます。

ライフサイクルモジュールは、サーバーのライフサイクルの 1 つまたは複数のイベントによってトリガーされたときに、タスクを実行します。トリガーとなるサーバーイベントには、初期化、起動、要求を処理する準備の完了、およびシャットダウンがあります。ライフサイクルモジュールは Java EE 仕様の一部ではなく、Enterprise Server の拡張です。

ライフサイクルモジュールごとに、次の情報が表示されます。

名前

ライフサイクルモジュールの名前。

使用可能

ライフサイクルモジュールが有効な場合は true、無効な場合は false です。

読み込み順序

ライフサイクルモジュールを読み込む相対的な順序を指定する値。

クラス名

ライフサイクルモジュールのクラスファイル。

関連操作

  • 「ライフサイクルモジュールを作成する」

  • 「ライフサイクルモジュールを編集する」

  • 「ライフサイクルモジュールを削除する」

関連する asadmin コマンド

  • create-lifecycle-module(1)

  • list-lifecycle-modules(1)

  • delete-lifecycle-module(1)
