Cómo editar un dominio

  1. En el árbol de navegación, expanda el nodo Configuración.
  2. Si el dominio está configurado para admitir clústeres, seleccione la configuración que esté cambiando.

    Algunos ejemplos son los dominios que se crean con el perfil clúster o con el perfil grandes empresas .

  3. Expanda el nodo Seguridad.
  4. Expanda el nodo Dominios.
  5. Seleccione el nombre de un dominio existente.

    Aparecerá la página Editar dominio.

  6. Edite las propiedades específicas de esta clase según corresponda.
  7. Agregue las propiedades adicionales correspondientes del dominio.
    1. Haga clic en Agregar propiedad.
    2. En el campo Nombre introduzca el nombre de la propiedad.

      • Para obtener una descripción de las propiedades del dominio de archivo, consulte Propiedades específicas de la clase FileRealm.

      • Para obtener una descripción de las propiedades del dominio de certificado, consulte Propiedades específicas de la clase CertificateRealm .

    3. En el campo Valor introduzca el valor de la propiedad.
  8. Haga clic en Guardar para guardar los cambios.
