Admin-Objektressource bearbeiten

Auf der Seite "Admin-Objektressource bearbeiten" können Sie die Einstellungen für eine verwaltete Objektressource ändern.

Die Seite "Admin-Objektressource bearbeiten" enthält die folgenden Optionen.

Standardeinstellungen laden

Schaltfläche zur Wiederherstellung der Standardwerte von Einstellungen, die Standardwerte haben. Einstellungen, die keine Standardwerte haben, werden nicht geändert.

JNDI-Name

Ein eindeutiger Name, der die verwaltete Objektressource identifiziert. Der JNDI-Name ist ein schreibgeschütztes Feld. Sie können den JNDI-Namen nur festlegen, wenn Sie eine neue verwaltete Objektressource erstellen.

Ressourcenadapter

Der Ressourcenadapter für die verwaltete Objektressource.

Ressourcentyp

Der Java-Typ für die verwaltete Objektressource (z.B. ↓javax.jms.Topic).

Klassenname

Der mit dem Ressourcentyp verknüpfte Implementierungsklassenname.

Deployment-Reihenfolge

Die Deployment-Reihenfolge der verwalteten Objektressource.

Beschreibung

Eine Beschreibung der verwalteten Objektressource.

Status

Gibt an, ob die Ressource zur Laufzeit verfügbar ist.

Wenn in der Domain nur die Standardserverinstanz server vorhanden ist, kann die Laufzeitverfügbarkeit der Ressource aktiviert oder deaktiviert werden. Laufzeitverfügbarkeit ist standardmäßig aktiviert.

Wurden in der Domain Cluster oder Standalone-Instanzen erstellt, können Sie die Laufzeitverfügbarkeit der Ressource auf ihren Zielen über die Registerkarte "Ziel" steuern.

Weitere Eigenschaften

Weitere Eigenschaften für die verwaltete Objektressource. Die verfügbaren Eigenschaften richten sich nach dem Ressourcenadapter, der von dem Connector Connection-Pool verwendet wird. Mit den in dieser Tabelle angegebenen Name/Werte-Paaren können die vom Ressourcenadapterprovider definierten Standardwerte für die Eigenschaften überschrieben werden.

Verwandte Aufgaben

  • So wird eine verwaltete Objektressource erstellt

  • So wird eine verwaltete Objektressource bearbeitet

  • So wird eine verwaltete Objektressource gelöscht
