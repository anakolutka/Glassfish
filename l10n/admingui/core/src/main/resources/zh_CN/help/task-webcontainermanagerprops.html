配置管理器属性

会话管理器提供了用于配置会话的创建和销毁方式、存储会话状态的位置以及可用会话的最大数量的方法。

  1. 在导航树中，展开“配置”节点。
  2. 在“配置”节点下，选择“Web 容器”节点。

    此时将打开“Web 容器”页面。

  3. 在“Web 容器”页面上，单击“管理器属性”选项卡。
  4. 在“清除时间间隔”字段中，键入从库中删除非活动会话数据之前经过的秒数。
  5. 在“最大会话数”字段中，键入允许的最大会话数。
  6. 在“会话文件名”字段中，键入包含会话数据的文件的名称。
  7. 在“会话 ID 生成器类名称”字段中，指定用于生成唯一会话 ID 的自定义类。

    此类必须位于 Enterprise Server 类路径中。每个服务器实例只允许有一个会话 ID 生成器类。一个群集中的所有实例必须使用同一会话 ID 生成器，以防止会话密钥冲突。自定义会话 ID 生成器类必须实现 com.sun.enterprise.util.uuid.UuidGenerator 接口：

    package com.sun.enterprise.util.uuid;

public interface UuidGenerator {

    public String generateUuid();
    public String generateUuid(Object obj);  //obj is the session object
}
  8. （可选的）在“其他属性”部分中，指定其他属性。
  9. 单击“保存”。
