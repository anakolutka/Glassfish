Instance Values

In this page, you can edit the value of a configuration property applicable to all the instances using this configuration.

  • Instance Name. Displays the name of the instance for which this property value is applicable.

  • Cluster Name. Displays the name of the cluster to which the instance belongs.

  • Parameter Value. In this field, you can provide a new value for the property. This value will be applicable to all the instances using this configuration.

Click Save for the changes to take effect.
