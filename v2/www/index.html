The GlassFish community is building free, open source, production-quality, enterprise software. The main deliverables are an Application Server, the Java EE 5 Reference Implementation, and the Java Persistence API Reference Implementation, TopLink Essentials. The community also delivers Maven Repository, tools and much more.

Project SailFin, Open Source SIP Servlets Technology for GlassFish
SIP (Session Initiation Protocol) is behind many popular services we enjoy today, like Voice-over-IP (VoIP) phone service, instant messaging, presence and buddy list management and web conferencing. SailFin project adds SIP Servlet technology to the GlassFish Application Server. Visit https://sailfin.dev.java.net for more information.
