Configurações Gerais de JVM

A JavaTM Virtual Machine (JVMTM) permite que códigos de bytes Java sejam executados em um computador físico. O ajuste das configurações de JVM melhora o desempenho e evita erros de alocação de memória.

O Java Virtual Machine está incluído no Java Platform, Standard Edition (plataforma Java SE), que é exigido pelo Enterprise Server. Configurações de JVM incorretas impedirão que o servidor seja executado, portanto, tenha cuidado ao alterar essas configurações.

Observação - Os termos “Java Virtual Machine” e “JVM” significam uma Máquina Virtual para a plataforma Java.

A página Configurações Gerais de JVM contém as seguintes opções.

Java Home

O nome do diretório de instalação do software Java SE, necessário à execução do Enterprise Server. Para verificar se a versão do software Java SE especificada é compatível com esta versão, consulte Sun GlassFish Enterprise Server v3 Release Notes. Se você especificar um nome de diretório inexistente ou se a versão do diretório não for compatível com o software Java SE, o Enterprise Server não será iniciado.

Opções Javac

As opções de linha de comando do compilador da linguagem de programação Java. O Enterprise Server executa o compilador quando os componentes Enterprise JavaBeans TM (EJBTM) são implantados.

Depuração

Ativa a depuração com o Java Platform Debugger Architecture (JPDA).

O JPDA é utilizado pelos desenvolvedores de aplicativos. Para obter mais informações, consulte Sun GlassFish Enterprise Server v3 Preview Application Development Guide .

Opções de Depuração

Especifica as opções do JPDA passadas à JVM quando a depuração está ativada.

Opções de Compilação RMI

As opções de linha de comando do compilador rmic. O Enterprise Server executa o compilador rmic quando os componentes EJB são implantados.

Pré-processador de Código de Bytes

Nomes de classes para pré-processamento de códigos de bytes. Use vírgulas para separar os nomes de classes. Cada classe deve implementar a interface com.sun.appserv.BytecodePreprocessor . As classes são chamadas na ordem especificada.

Propriedades Adicionais

Propriedades adicionais do JVM. O Enterprise Server não define propriedades adicionais para o JVM.
