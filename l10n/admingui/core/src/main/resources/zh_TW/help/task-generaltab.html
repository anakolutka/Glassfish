檢視 Enterprise Server 的一般資訊

  1. 在瀏覽樹狀結構中，選取 [Enterprise Server] 節點。

    此時將開啟 [一般資訊] 頁面。

  2. 在 [一般資訊] 頁面上，執行下列任何步驟。
    • 按一下 [停止] 按鈕以停止 Enterprise Server。
    • 按一下 [重新啟動] 按鈕以停止和重新啟動 Enterprise Server。
    • 按一下 [檢視記錄檔] 按鈕以檢視伺服器記錄。

      請參閱檢視伺服器記錄以取得詳細資訊。

    • 按一下 [JVM 報告] 以開啟另外一個視窗，其中您可檢視 Java 虛擬機器的相關報告，包括摘要報告、記憶體管理報告、類別載入報告以及目前執行緒傾印。
