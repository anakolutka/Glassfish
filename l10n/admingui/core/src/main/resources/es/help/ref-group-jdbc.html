JDBC

La tecnología JavaTM DataBase Connectivity (JDBCTM) proporciona conectividad a bases de datos relacionales. En Servidor empresarial, se configura la conectividad a bases de datos agregando recursos JDBC (fuentes de datos) al dominio. Una fuente de datos es un método estándar de la plataforma Java, Enterprise Edition (plataforma Java EE), para configurar la conectividad a una base de datos. Cada fuente de datos contiene un conjunto de conexiones de base de datos. Las aplicaciones buscan la fuente de datos en el árbol de API de Java Naming and Directory InterfaceTM (J.N.D.I.) o en el contexto de la aplicación local y, a continuación, recuperan la conexión a una base de datos con el método getConnection. Las fuentes de datos y los conjuntos de conexiones correspondientes proporcionan procesos de administración de conexiones que contribuyen a la ejecución del sistema.
