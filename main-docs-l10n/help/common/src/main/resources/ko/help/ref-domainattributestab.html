도메인 속성

[도메인 속성] 페이지에서는 GlassFish Server에 대한 고급 도메인 속성을 설정할 수 있습니다.

[도메인 속성] 페이지에는 다음 옵션이 포함되어 있습니다.

DAS 시작 후 콘솔 로드

다음 중에서 관리 콘솔 로드 시기를 결정합니다.

사용량 기반

콘솔이 지난 24시간 내에 사용되었거나 다른 서버 인스턴스가 DAS(도메인 관리 서버)와 별개로 존재하는 경우 DAS 시작 후 관리 콘솔이 로드됩니다. 이 값은 기본값입니다.

항상

DAS 시작 후 관리 콘솔이 항상 로드됩니다. 콘솔은 기본 GlassFish Server 시작 시퀀스가 완료된 후에 로드됩니다.

사용 안함

DAS 시작 후 관리 콘솔이 로드되지 않습니다. 콘솔은 액세스가 요청될 때 로드됩니다.

응용 프로그램 루트

배치된 응용 프로그램이 있는 디렉토리의 전체 경로입니다. 기본값은 domain-dir/applications입니다.

로그 루트

서버 로그 파일을 저장하는 위치를 식별합니다. 기본값은 domain-dir/logs입니다.

로케일

이 필드를 비워 두면 호스트의 기본 로케일이 사용됩니다.

로케일은 언어 및 지역의 특정 조합을 지정하는 식별자입니다. 예를 들어 미국 영어의 로케일은 en_US이고 일본어의 로케일은 ja_JP입니다. 영어가 아닌 다른 로케일을 사용하려면 GlassFish Server를 현지화(영어를 다른 언어로 번역하는 작업이 포함된 프로세스)해야 합니다.
