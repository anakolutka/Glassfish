安装组件更新

更新安装的组件涉及以下操作：

  • 仅将修改的文件从网络系统信息库传输到您的系统

  • 仅在您的系统上更新 Enterprise Server 安装映像内修改的文件。

  • 在更新的组件之间应用新的依赖关系（如果有）

  1. 在导航树中，选择“更新工具”节点。

    此时将打开“可用的附加软件”页面。

  2. 在“可用的附加软件”页面上，单击“可用更新”选项卡。

    此时将打开“可用更新”页面。

  3. 在可用更新的表中，选择您要更新的组件。

    如果在版本或大小方面，组件的“已安装版本”和“新版本”之间没有区别，则无需更新组件。如果没有可用的更新，则此表为空。

    • 要选择单个组件，请选择该组件名称旁边的复选框。
    • 要选择所有组件，单击表头中的“全选”图标。
    • 要取消选择所有组件，请单击表头中的“取消全选”图标。
  4. 单击“安装”。
  5. 接受许可证协议。

    Enterprise Server 确认安装已完成。

  6. 重新启动服务器。

    有关详细说明，请参见以下文档：

    • 《Sun GlassFish Enterprise Server v3 Administration Guide》中的"To Stop a Domain"

    • 《Sun GlassFish Enterprise Server v3 Administration Guide》中的"To Start a Domain"
