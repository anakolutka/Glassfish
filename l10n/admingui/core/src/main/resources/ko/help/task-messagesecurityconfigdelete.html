메시지 보안 구성을 삭제하는 방법

  1. 탐색 트리에서 구성 노드를 선택합니다.
  2. 보안 노드를 확장합니다.
  3. 메시지 보안 노드를 선택합니다.

    메시지 보안 구성 페이지가 열립니다.

  4. 메시지 보안 구성 페이지에서 삭제 중인 메시지 보안 구성의 왼쪽에 있는 확인란을 선택합니다.
  5. 삭제를 누릅니다.
