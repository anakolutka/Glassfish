Servizio transazioni

Utilizzare la pagina Servizio transazioni per configurare le funzioni di recupero, timeout e registrazione delle transazioni.

La pagina Servizio transazioni contiene le seguenti opzioni.

Carica impostazioni predefinite

Pulsante per ripristinare i valori predefiniti delle impostazioni che dispongono di valori predefiniti. Le impostazioni che non dispongono di valori predefiniti non vengono modificate.

Nome configurazione

Il nome della configurazione a cui si applicano le impostazioni presenti in questa pagina. Questo campo è di sola lettura.

Al riavvio

Se la casella di controllo Abilitato è selezionata, viene eseguito un tentativo di recupero delle transazioni incomplete al riavvio del server. Questa opzione è disabilitata per impostazione predefinita.

Timeout transazione

Il numero di secondi di attesa del server prima di eseguire il rollback di una transazione non completata. Il valore predefinito, 0, indica che il server rimane in attesa del completamento di una transazione a tempo indeterminato.

Timeout tentativi

Il numero di secondi durante i quali GlassFish Server tenta di connettersi a un server irraggiungibile. Il valore predefinito è 600 (10 minuti).

Posizione log delle transazioni

La directory in cui vengono memorizzati i log del server. I log delle transazioni vengono memorizzati nella sottodirectory tx della directory specificata in questo campo. Il valore predefinito è la directory specificata nel campo Radice log della pagina Attributi dominio, situata nella scheda Avanzate del nodo GlassFish Server.

Decisione euristica

Indica se è necessario eseguire il commit o il rollback delle transazioni che coinvolgono server irraggiungibili. Il valore predefinito è Rollback. L'esecuzione del commit di transazioni non determinate può ripercussioni sull'integrità dei dati dell'applicazione.

Intervallo punti chiave

Il numero di transazioni tra un'operazione punto chiave e l'altra, che determinano il livello di compressione del file di log delle transazioni. Il valore predefinito è 65.536.
