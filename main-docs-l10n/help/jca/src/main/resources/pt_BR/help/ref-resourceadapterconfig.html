Configurações do Adaptador de Recursos

Use a página Configurações do Adaptador de Recursos para gerenciar as configurações do conector (adaptador de recurso).

A configuração do adaptador de recursos fornece informações de configuração para um adaptador de recursos. Com a Console de Administração, você cria um config de adaptador de recursos após a implantação do adaptador de recurso. Nesse caso, o adaptador de recurso é reiniciado com a nova configuração. Para criar uma configuração do adaptador de recursos antes de implantar o adaptador de recursos, use o comando asadmin create-resource-adapter-config.

Antes de poder criar uma configuração do adaptador de recursos, é preciso criar um pool de threads para que o adaptador de recursos a utilize.

Para cada config do adaptador de recursos, as seguintes informações são fornecidas.

Nome

O nome do adaptador de recursos.

A tabela Configurações também contém as seguintes opções.

Novo

Botão para criar uma nova configuração do adaptador de recursos.

Deletar

Botão para deletar uma ou mais configurações do adaptador de recursos selecionados.
