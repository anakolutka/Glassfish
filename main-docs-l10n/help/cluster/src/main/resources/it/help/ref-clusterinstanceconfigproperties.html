Proprietà di sistema dell'istanza

Nella pagina Proprietà di sistema dell'istanza viene visualizzata una lista delle proprietà di sistema Java impostate per l'istanza di GlassFish Server corrente. Le proprietà di sistema Java vengono passate al launcher dell'applicazione Java tramite la relativa opzione -D all'avvio di GlassFish Server.

Queste proprietà sostituiscono le definizioni delle proprietà relative alle impostazioni delle porte contenute nella configurazione dell'istanza. Le impostazioni di porta predefinite devono essere sostituite se, ad esempio, due istanze in cluster risiedono nello stesso host. In questa situazione è necessario che le impostazioni delle porte per una delle istanze vengano sostituite, perché le due istanze condividono la stessa configurazione.

Per informazioni sulle impostazioni di porta predefinite in una configurazione, vedere Proprietà di sistema di configurazione predefinite.

Il campo Nome istanza è un campo di sola lettura in cui viene visualizzato il nome dell'istanza corrente.

Per ogni proprietà vengono visualizzate le informazioni riportate di seguito.

Nome variabile istanza

Il nome della proprietà di sistema.

Valore corrente

Il valore impostato al momento per la proprietà. Questo campo è di sola lettura.

Valore di sostituzione

Il valore su cui verrà impostata la proprietà dopo il salvataggio delle modifiche.

La tabella Proprietà aggiuntive contiene inoltre le opzioni riportate di seguito.

Aggiungi proprietà

Pulsante che consente di aggiungere una proprietà. Se si fa clic su questo pulsante, viene aggiunta una riga alla tabella Proprietà aggiuntive.

Elimina proprietà

Pulsante che consente di eliminare una o più proprietà selezionate. L'eliminazione comporta il ripristino del valore predefinito della proprietà. Se il valore predefinito non è stato impostato, la proprietà rimane non definita.
