So wird ein JDBC-Connection-Pool gelöscht

Wenn Sie einen JDBC-Connection-Pool löschen, werden alle von diesem Connection-Pool abhängigen JDBC-Ressourcen ebenfalls gelöscht.

  1. Blenden Sie in der Navigationsbaumstruktur den Knoten "Ressourcen" ein.

  2. Blenden Sie unter dem Ressourcenknoten den JDBC-Knoten ein.

  3. Wählen Sie unter dem JDBC-Knoten den Knoten "Connection-Pools".

    Die Seite "Connection-Pools" wird geöffnet.

  4. Aktivieren Sie auf der Seite "Connection-Pool" das Kontrollkästchen links neben dem Pool, den Sie löschen möchten.

  5. Klicken Sie auf "Löschen".
