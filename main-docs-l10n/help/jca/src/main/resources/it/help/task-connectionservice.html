Per configurare il servizio connettore

  1. Espandere il nodo Configurazione nell'albero di navigazione.

  2. Nel nodo Configurazione selezionare il nodo Servizio connettore.

    Viene visualizzata la pagina Servizio connettore.

  3. Nel campo Timeout chiusura digitare il numero massimo di secondi consentito durante la chiusura di GlassFish Server per il completamento del metodo ResourceAdapter.stop di un'istanza di modulo connettore.

    I Resource Adapter che richiedono tempi di chiusura più lunghi vengono ignorati e la procedura di chiusura di GlassFish Server continua. Il valore predefinito è 30 secondi.

  4. Nell'elenco a discesa Criteri di caricamento classi del connettore selezionare il criterio da utilizzare.

    • Selezionare derived per indicare che i Resource Adapter vengono forniti in base ai riferimenti di un'applicazione a qualsiasi risorsa del Resource Adapter.

      Questo è il criterio predefinito.

    • Selezionare global per specificare che tutti i Resource Adapter saranno visibili per tutte le applicazioni.

    Questo criterio non si applica per il Resource Adapter di sistema JMS preinstallato, jmsra, che è sempre visibile per tutte le applicazioni.

  5. Fare clic su Salva.

    Suggerimento:

    Per ripristinare le impostazioni predefinite, fare clic sul pulsante Carica impostazioni predefinite, quindi su Salva.
