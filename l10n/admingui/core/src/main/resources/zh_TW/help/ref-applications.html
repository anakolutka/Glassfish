應用程式

[應用程式] 頁面會顯示 Enterprise Server 上部署的應用程式清單。您可以檢視和管理已部署的應用程式，還可以部署更多應用程式。

對於每個應用程式，均提供以下資訊：

名稱

應用程式名稱。

已啟用

若應用程式已啟用，則會出現核取記號；若應用程式已停用，則會顯示 X。

複合

若應用程式具有多種元件，則為 True；若只有一種元件，則為 False。

引擎

應用程式所使用的容器類型。容器類型可以是下列任何一項：

  • web

  • webservices

  • ejb

  • connector

  • appclient

  • jruby

動作

在部署後的元件上，您可以執行的動作之連結；[重新部署] 和 [重新啟動] 適用於所有元件；而 [啟動] 則適用於 Web 應用程式。

部署

部署企業應用程式的按鈕。

未部署

解除部署企業應用程式的按鈕。

啟用

啟用所選取企業應用程式的按鈕。

停用

停用所選取企業應用程式的按鈕。

篩選

依引擎篩選應用程式的下拉式清單。
