Configurer les paramètres proxy de réseau pour accéder aux référentiels de réseau

Les paramètres proxy de réseau permettent de déterminer le mode de connexion de Enterprise Server à Internet. Pour permettre au Outil de mise à jour d'accéder aux référentiels de réseau via l'Internet, configurez vos paramètres proxy de réseau.

  1. Dans l'arborescence de navigation, sélectionnez le nœud Outil de mise à jour.

    La page Add-ons disponibles s'affiche.

  2. Sur la page Add-ons disponibles, cliquez sur l'onglet Configuration.

    La page Configuration Outil de mise à jour s'affiche.

  3. Sur la page Configuration de Outil de mise à jour, indiquez le mode de connexion à l'Internet de la machine sur laquelle Enterprise Server fonctionne.
    • Si la machine se connecte directement à l'Internet, sélectionnez l'option Connexion directe.
    • Si la machine se connecte à l'Internet via un proxy, sélectionnez l'option Via un proxy.
  4. Si la machine sur laquelle s'exécute le Enterprise Server se connecte à Internet par l'intermédiaire d'un proxy, donnez le nom de l'hôte proxy et son numéro de port.

    Si la machine sur laquelle Enterprise Server s'exécute se connecte directement à Internet, ignorez cette étape.

    Il est possible qu'un navigateur Web s'exécute sur la même machine que Enterprise Server. Dans ce cas, vous pouvez obtenir le nom d'hôte et le numéro de port du proxy dans ce navigateur. Sinon, contactez votre administrateur système.

  5. Cliquez sur Enregistrer.
