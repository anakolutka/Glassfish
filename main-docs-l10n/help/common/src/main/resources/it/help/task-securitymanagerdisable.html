Per disabilitare Security Manager

La disabilitazione del Security Manager di GlassFish Server può comportare un miglioramento delle prestazioni per determinati tipi di applicazioni. Le funzioni di autorizzazione e autenticazione della piattaforma Java continuano a funzionare anche dopo la disabilitazione di Security Manager. È possibile disabilitare Security Manager in un ambiente di sviluppo, mentre si sconsiglia di disabilitarlo in un ambiente di produzione.

  1. Selezionare il nodo Configurazione nell'albero di navigazione.

  2. Selezionare la configurazione da modificare.

  3. Nel nodo Configurazione selezionare il nodo Impostazioni JVM.

    Viene visualizzata la pagina Impostazioni generali JVM.

  4. Nella pagina Impostazioni generali JVM fare clic sulla scheda Opzioni JVM.

    Viene visualizzata la pagina Opzioni JVM.

  5. Selezionare la casella di controllo a sinistra del campo Valore contenente l'opzione -Djava.security.policy.

  6. Fare clic su Elimina.

  7. Fare clic su Salva.

  8. Riavviare il server.
