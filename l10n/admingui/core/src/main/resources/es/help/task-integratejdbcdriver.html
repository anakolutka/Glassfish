Para integrar un controlador JDBC

Un controlador compatible con la API JDBC (controlador JDBC) convierte las llamadas JDBC de una aplicación en el protocolo del servidor de base de datos.

  • Para integrar el controlador JDBC en un dominio administrativo, haga una de las dos cosas siguientes:
    • Haga accesible el controlador al cargador de clases comunes.
      1. Copie los archivos JAR y ZIP del controlador en el directorio domain-dir /lib o copie los archivos de clase en el directorio domain-dir/lib/ext .
      2. Reinicie el dominio.
    • Haga accesible el controlador al cargador de clases del sistema.
      1. En el árbol de navegación, seleccione el nodo Application Server.
      2. En la página Información general, seleccione la ficha Configuración de JVM.
      3. En la página Configuración general de JVM, seleccione la ficha Configuración de ruta.
      4. En el campo Sufijo de ruta de clase, introduzca un nombre de ruta completo para el archivo JAR del controlador.
      5. Haga clic en "Guardar".
      6. Reinicie el servidor.
