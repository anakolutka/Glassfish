To Install a JBI Shared Library

  1. In the Shared Libraries page, click Install.
  2. In the Install JBI Shared Library page, specify the location of the shared library archive.

    You can either upload a shared library from your local file system or copy a shared library from the Application Server host's file system. Use the Browse Files button to navigate to the directory where the archive is located and select the archive.

  3. Click Next and in the General page, verify if the name and description are correct.
  4. Click Finish.

    The Shared Libraries page displays the new shared library.
