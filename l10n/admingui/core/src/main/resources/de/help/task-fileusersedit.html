So bearbeiten Sie Benutzerinformationen

  1. Wählen Sie in der Seite "Dateibenutzer" in der Spalte "Benutzer-ID" der Tabelle den Namen des zu ändernden Benutzers aus.

    Die Seite "Dateibereichsbenutzer bearbeiten" wird geöffnet.

  2. Zum Ändern des Benutzerpassworts geben Sie in den Feldern "Passwort" und "Passwort bestätigen" ein neues Passwort ein.
  3. Zum Ändern der Gruppen, zu denen der Benutzer gehört, fügen Sie im Feld "Gruppenliste" Gruppen hinzu bzw. löschen Gruppen aus diesem Feld.

    Trennen Sie Gruppen durch Kommata. Gruppen müssen zuvor nicht definiert werden.

  4. Wählen Sie "Speichern", um Änderungen an diesem Benutzer zu speichern, oder "Zurück", um zu beenden ohne zu speichern.
