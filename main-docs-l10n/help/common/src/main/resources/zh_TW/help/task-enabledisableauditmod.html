啟用或停用稽核記錄日誌

  1. 在瀏覽樹狀結構中，展開「組態」節點。

  2. 選取要組態的執行處理：

    • 若要組態特定的執行處理，請展開該執行處理的組態節點。例如，在預設執行處理 server 展開 server-config 節點。

    • 若要組態所有執行處理的預設設定，請展開 default-config 節點。

  3. 在「組態」節點下，選取「安全性」節點。

    「安全性」頁面會隨即開啟。

  4. 在「安全性」頁面上，選取「稽核日誌」的「已啟用」核取方塊來啟用稽核日誌。

    若要啟用稽核日誌，您也必須將使用中稽核模組的 auditOn 特性設為 true。

  5. 按一下「儲存」。
