JDBC 資源

Java 資料庫連結 (JDBC) 資源 (資料來源) 提供應用程式連線至資料庫的方法。管理員一般會為每個資料庫 (由部署於網域的應用程式所存取) 建立一個 JDBC 資源，但是您可為一個資料庫建立多個 JDBC 資源。

應用程式會在 Java Naming and Directory Interface (J.N.D.I.) API 樹狀結構中查找資料來源並請求連線，以便從連線池取得資料庫連線。與資料來源相關聯的連線池可提供與應用程式的連線。

「JDBC 資源」頁面會顯示在應用程式伺服器上建立的資源清單。對於每個資源，均提供以下資訊：

  • J.N.D.I. 名稱

  • 狀態

  • 連線池

  • 說明
