Se abrirá la página Editar Dominio.

En la página Editar Dominio, haga clic en el botón Gestionar Usuarios.

Se abrirá la página Usuarios administradores.

En la página Usuarios administradores, haga clic en Nuevo.

Se abrirá la página Nuevo Usuario de Dominio de Archivo.

En el campo ID de Usuario, introduzca el nombre del usuario.

En el campo Lista de Grupos, introduzca la lista de los grupos a los que pertenece el usuario.

Utilice comas para separar los nombres de grupo. Estos grupos no hace falta definirlos en otro lugar.

En el campo Contraseña, introduzca una contraseña única.

En el campo Confirmar Contraseña Nueva, escriba la contraseña de usuario de nuevo.

Haga clic en Aceptar.
