Fabriques de connexions JMS

La page Fabriques de connexions JMS permet de configurer les ressources de fabrique de connexions JMS.

Pour chaque ressource, les informations suivantes sont fournies.

Nom JNDI

Nom JNDI identifiant la fabrique de connexions.

Nom JNDI logique

Nom JNDI logique pour la fabrique de connexions. Pour plus d'informations, reportez-vous à Noms JNDI logiques.

Activé

(Cette colonne est affichée si seule l'instance de serveur par défaut server existe dans le domaine.)

Coche si la ressource est activée ou X si la ressource est désactivée.

Statut

(Cette colonne apparaît si des clusters ou des instances autonomes ont été créés dans le domaine.)

Nombre de clusters et d'instances autonomes sur lesquels la ressource est ciblée et nombre de cibles sur lesquelles la ressource est activée. Par exemple, Activé sur 2 cibles sur 4 signifie que la ressource est ciblée sur quatre clusters et instances autonomes et activée sur deux de ces quatre cibles.

Type de ressource

Type de ressource de la fabrique de connexions.

Description

Description de la fabrique de connexions.

La table Fabriques de connexions contient également les options ci-après.

Nouveau

Bouton permettant de créer une fabrique de connexions.

Supprimer

Bouton permettant de supprimer les fabriques de connexions sélectionnées.

Activer

Bouton permettant d'activer les fabriques de connexions sélectionnées.

Désactiver

Bouton permettant de désactiver les fabriques de connexions sélectionnées.
