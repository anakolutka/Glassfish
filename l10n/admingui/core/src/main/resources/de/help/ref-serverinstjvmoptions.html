JVM-Optionen

Auf der Seite JVM Options (JVM-Optionen) können Sie die Optionen für das Java-Anwendungsstarttool (java) festlegen, das Enterprise Server ausführt. Über die -D-Optionen können Sie Enterprise Server-spezifische Eigenschaften festlegen.

  • Um eine Option zu ändern, bearbeiten Sie das entsprechende Feld Value (Wert).

  • Um eine Option hinzuzufügen, klicken Sie auf die Schaltfläche Add JVM Option (JVM-Option hinzufügen). Geben Sie in der leeren Zeile im Feld Value (Wert) die Optionsinformationen ein.

  • Um eine Option zu löschen, aktivieren Sie das Kontrollkästchen links vom Feld Value (Wert) der zu löschenden Option, und klicken Sie anschließend auf Delete (Löschen).
