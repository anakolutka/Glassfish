Configurazione di Strumento di aggiornamento

La pagina Configurazione fornisce le informazioni sulla configurazione di Strumento di aggiornamento e consente di modificare la modalità di connessione di Strumento di aggiornamento a Internet. Per modificare altri aspetti della configurazione di Strumento di aggiornamento, utilizzare l'applicazione updatetool standalone disponibile nel percorso as-install-parent/bin.

La pagina Configurazione di Strumento di aggiornamento contiene le opzioni riportate di seguito.

Percorso di installazione

Il percorso del file system in cui si trova la directory radice o di livello superiore dell'immagine di installazione.

Repository preferito

Il nome del repository di rete preferito associato a questa immagine di installazione, ad esempio glassfish.org.

Connessione Internet

Una delle modalità di connessione di GlassFish Server a Internet riportate di seguito.

Connessione diretta

Questa opzione specifica che GlassFish Server si connette a Internet in modo diretto.

Tramite un proxy

Questa opzione specifica che GlassFish Server si connette a Internet mediante un server proxy. Se questa opzione è selezionata, i campi Host proxy e Porta proxy vengono aggiunti alla pagina.

Host proxy

Il nome host del server proxy per il computer su cui è in esecuzione GlassFish Server.

Questo campo è valido solo se il computer su cui è in esecuzione GlassFish Server si connette a Internet tramite un server proxy.

Porta proxy

Il numero di porta del server proxy per il computer su cui è in esecuzione GlassFish Server.

Questo campo è valido solo se il computer su cui è in esecuzione GlassFish Server si connette a Internet tramite un server proxy.

Repository

Questa tabella fornisce le informazioni riportate di seguito per ogni repository di rete associato a questa immagine di installazione.

Repository

Il nome del repository, ad esempio dev.glassfish.org.
