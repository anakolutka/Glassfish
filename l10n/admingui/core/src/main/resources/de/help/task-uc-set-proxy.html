So konfigurieren Sie Netzwerk-Proxy-Einstellungen für den Zugriff auf Netzwerk-Repositorys

Netzwerk-Proxy-Einstellungen legen fest, wie Enterprise Server eine Verbindung mit dem Internet herstellt. Damit Update-Tool über das Internet auf Netzwerk-Repositorys zugreifen kann, müssen Sie Ihre Netzwerk-Proxy-Einstellungen konfigurieren.

  1. Wählen Sie in der Navigationsstruktur den Knoten Update-Tool aus.

    Die Seite "Verfügbare Add-Ons" wird geöffnet.

  2. Klicken Sie auf der Seite "Verfügbare Add-Ons" auf die Registerkarte "Konfiguration".

    Die Seite Update-Tool-Konfiguration wird angezeigt.

  3. Geben Sie auf der Seite "Update-Tool-Konfiguration" an, wie der Rechner, auf dem Enterprise Server ausgeführt wird, eine Verbindung zum Internet herstellt.
    • Stellt der Rechner eine Direktverbindung mit dem Internet her, so wählen Sie die Option "Direktverbindung".
    • Stellt der Rechner eine Internetverbindung über einen Proxy her, wählen Sie die Option "Über einen Proxy".
  4. Wenn der Computer, auf dem Enterprise Server ausgeführt wird, über einen Proxy eine Verbindung zum Internet herstellt, geben Sie den Hostnamen und die Port-Nummer des Proxy an.

    Wenn der Computer, auf dem Enterprise Server ausgeführt wird, eine direkte Verbindung zum Internet herstellt, lassen Sie diesen Schritt aus.

    Möglicherweise wird ein Webbrowser auf dem Computer ausgeführt, auf dem Enterprise Server ausgeführt wird. In diesem Fall können Sie den Hostnamen und die Port-Nummer des Proxy über den Webbrowser beziehen. Andernfalls erhalten Sie die benötigten Informationen von Ihrem Systemadministrator.

  5. Klicken Sie auf "Speichern".
