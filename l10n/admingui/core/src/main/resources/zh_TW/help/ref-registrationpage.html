產品註冊

若要於 Sun Connection 註冊您的 Sun GlassFishTMEnterprise Server 安裝，您必須有 Sun Online 帳號。如果您沒有 Sun Online 帳號，Enterprise Server 可讓您在註冊期間建立一個帳號。

[產品註冊] 頁面包含下列選項。

我已經具有 Sun Online 帳號或 Sun Developer Network (SDN) 帳號

若已有帳號，請選取此選項。如果選取此選項，將會顯示以下欄位。

使用者名稱

註冊 Sun Online 帳號時所指定的使用者名稱。

密碼

註冊 Sun Online 帳號時所指定的密碼。

代理主機

Enterprise Server 執行所在機器的代理伺服器主機名稱。

此欄位僅在 Enterprise Server 執行所在機器透過代理伺服器連線至網際網路時才適用。

代理連接埠

Enterprise Server 執行所在機器的代理伺服器連接埠號碼。

此欄位僅在 Enterprise Server 執行所在機器透過代理伺服器連線至網際網路時才適用。

我沒有 Sun Online 帳號。立即註冊。

若您目前沒有帳號，且希望在 Sun Connection 中註冊 Enterprise Server 安裝，請選取此選項。如果選取此選項，將會顯示以下欄位。

電子郵件地址

您的電子郵件地址，即您的 Sun Online 帳號的使用者名稱。

密碼

Sun Online 帳號所使用的密碼。密碼要區分大小寫，並且長度必須為 6 至 10 個字元。

重新鍵入密碼

您必須重新鍵入在 [密碼] 欄位中所指定的密碼。

名字

您的名字。

姓氏

您的姓氏。

公司

您服務的公司 (如果適用)。

國家/地區

您所居住的國家或地區。

代理主機

Enterprise Server 執行所在機器的代理伺服器主機名稱。

此欄位僅在 Enterprise Server 執行所在機器透過代理伺服器連線至網際網路時才適用。

代理連接埠

Enterprise Server 執行所在機器的代理伺服器連接埠號碼。

此欄位僅在 Enterprise Server 執行所在機器透過代理伺服器連線至網際網路時才適用。
