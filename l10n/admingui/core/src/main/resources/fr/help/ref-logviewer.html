Afficheur du journal

Utilisez la page Afficheur du journal pour afficher le journal du serveur et personnaliser l'affichage du journal.

La page Afficheur du journal se compose de trois zones :

  • Critères de recherche

  • Recherche avancée

  • Résultats de la recherche

La zone Critères de recherche contient les options suivantes.

Recherche avancée

Un clic sur ce lien ouvre une zone qui vous permet d'affiner davantage l'afficheur du journal.

Horodatage

Le bouton-radio Le plus récent affiche les messages les plus récents. Le bouton-radio Plage spécifique ouvre les champ De et A qui vous permettent d'afficher les messages d'une certaine période uniquement.

Pour la valeur Heure, la syntaxe doit prendre la forme suivante (SSS est mis pour millisecondes) :

hh:mm:ss.SSS

Par exemple :

17:10:00.000

Si la valeur de début est postérieure à la valeur de fin, un message d'erreur apparaît.

Niveau de journalisation

Niveau de journalisation sélectionné pour le filtrage. Par défaut, tous les messages figurant dans le journal du serveur à partir du niveau de journalisation choisi sont affichés. Sélectionnez la case « Ne pas inclure des messages plus graves » pour afficher uniquement les messages du niveau choisi.

Pour vous assurer que les messages que vous voulez afficher apparaissent dans le journal du serveur, commencez par définir les niveaux de journalisation appropriés sur la page Niveaux de journalisation de module. Voir Configurer les niveaux de journalisation.

Si vous choisissez de filtrer les messages du journal en fonction du niveau de journalisation, seuls les messages correspondant aux critères de filtre spécifiés s'affichent. Toutefois, ce filtrage n'affecte pas les messages consignés dans le journal du serveur.

La zone Recherche avancée contient les options suivantes.

Journal

Espaces de noms du journal pour filtrer par module. Utilisez maj + clic ou Ctrl + clic pour sélectionner plusieurs espaces de noms.

Si vous sélectionnez un espace de noms à un niveau supérieur, tous les espaces de noms situés en dessous seront également sélectionnés. Par exemple, si vous sélectionnez javax.enterprise.system, seront également sélectionnés les journaux de tous les modules situés sous cet espace de noms : javax.enterprise.system.core, javax.enterprise.system.tools.admin , etc.

Journal personnalisé

Journals propres à une application particulière, un par ligne.

Si l'application comprend plusieurs modules, vous pouvez les afficher tous ou seulement une partie. Prenons l'exemple d'une application comprenant les journaux suivants :

com.mycompany.myapp.module1
com.mycompany.myapp.module2
com.mycompany.myapp.module3

Pour afficher des messages de tous les modules dans l'application, tapez com.mycompany.myapp . Pour afficher les messages du module2 uniquement, entrez com.mycompany.myapp.module2.

Si vous spécifiez un ou plusieurs enregistreurs personnalisés, les messages des modules du Enterprise Server n'apparaissent que si vous les avez explicitement spécifiés dans la zone Enregistreur.

Paires de valeurs de nom

Nom clé et valeur d'un thread pour afficher le résultat d'un thread particulier. Le nom clé est _ThreadID. Par exemple :

_ThreadID=13

Supposons que com.mycompany.myapp.module2 fonctionne sur plusieurs threads. Pour affiner le visionneur de journaux et afficher uniquement le résultat d'un seul thread, spécifiez le journal de ce module dans le champ Enregistreur personnalisé, puis spécifiez l'ID thread.

Affichage

Nombre de messages à afficher à la fois. Le nombre de messages affichés par défaut est de 40. Les autres choix sont 100, 250 et 1000.

Pour afficher les suivis de pile, désélectionnez la case « Tronquer les messages trop longs ». Par défaut, les suivis de pile n'apparaissent pas dans le visionneur. Pour les afficher, cliquez sur le lien (détails) pour un message.

Le tableau des Résultats de la recherche affiche les messages du journal du serveur.

Pour chaque message, le tableau des Résultats de la recherche contient les informations suivantes.

Numéro d'enregistrement

Numéro du message dans le journal du serveur.

Niveau de journalisation

Niveau de journalisation du message.

Message

Texte bref du message du journal. Cliquez sur le lien (détails) pour afficher la version formatée d'un message.

Journal

Le module de journalisation qui a généré le message du journal.

Horodatage

Date et heure d'émission du message du journal.

Nom/Paires de valeurs

Nom du thread et ID du message du journal.
