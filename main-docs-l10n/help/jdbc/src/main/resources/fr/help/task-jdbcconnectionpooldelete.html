Suppression d'un pool de connexions JDBC

Lorsque vous supprimez un pool de connexions JDBC, toutes les ressources JDBC qui dépendent de ce pool sont également supprimées.

  1. Dans l'arborescence de navigation, développez le noeud Ressources.

  2. Sous le noeud Ressources, développez le noeud JDBC.

  3. Sous le noeud JDBC, sélectionnez le noeud Pools de connexions.

    La page Pools de connexions apparaît.

  4. Dans la page Pools de connexions, cochez la case située à gauche du pool de connexions à supprimer.

  5. Cliquez sur Supprimer.
