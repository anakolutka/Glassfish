JVM 類別路徑設定

類別路徑是 Java 執行階段環境搜尋類別與其他資源的 JAR 檔案清單。

[JVM 類別路徑設定] 頁面包含下列選項。

環境類別路徑

若選取 [忽略] 核取方塊，則會忽略 CLASSPATH 環境變數。CLASSPATH 環境變數對於程式設計基本指導非常方便，但是不建議用於企業環境。

伺服器類別路徑

Enterprise Server 類別路徑。此為唯讀欄位。

系統類別路徑

系統上的類別的路徑。

類別路徑前綴

前置於伺服器的類別路徑之 JAR 檔案完整路徑，一行一個。

類別路徑後綴

附加於伺服器的類別路徑之 JAR 檔案完整路徑，一行一個。

本機程式庫路徑前綴

前置於本機程式庫路徑之 JAR 檔案完整路徑，一行一個。

本機程式庫路徑後綴

附加於本機程式庫路徑之 JAR 檔案完整路徑，一行一個。

本機程式庫路徑串連了伺服器的原生共用程式庫相對路徑、標準執行階段環境本機程式庫路徑、Shell 環境設定 (在 UNIX® 系統上為 LD_LIBRARY_PATH) ，以及在JVM 效能評測器設定頁面上指定的任何路徑。
