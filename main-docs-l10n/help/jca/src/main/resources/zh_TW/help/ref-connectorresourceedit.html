編輯連線器資源

使用「編輯連線器資源」頁面可修改連線器資源的設定。

「編輯連線器資源」頁面包含下列選項。

載入預設值

用於將具有預設值的設定復原為設定值的按鈕。沒有預設值的設定則保持不變。

JNDI 名稱

可識別連線器資源的唯一名稱。「JNDI 名稱」是唯讀欄位。您僅能在建立新的連線器資源時指定 JNDI 名稱。

集區名稱

與此資源關聯的連線器連線集區。

建置順序

連線器資源的建置順序。

描述

連線器資源的描述。

狀態

是否可在執行階段使用資源。

如果網域中只有預設伺服器執行處理 server 存在，可以選擇啟用或停用資源的執行階段可用性。預設會啟用執行階段可用性。

如果已在網域中建立任何叢集或獨立執行處理，可使用「目標」頁籤控制其目標之資源的執行階段可用性。

其他特性

連線器資源的其他特性。可用的特性會視連線器連線集區的資源轉接器而定。此表中指定的名/值配對，可用於置換資源轉接器供應商定義的特性預設值。
