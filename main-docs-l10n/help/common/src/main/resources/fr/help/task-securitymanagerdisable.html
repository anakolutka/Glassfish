Désactivation du gestionnaire de sécurité

La désactivation du gestionnaire de sécurité de GlassFish Server peut améliorer les performances de certains types d'application. Les fonctionnalités d'autorisation et d'authentification de la plate-forme Java pourront toujours être utilisées, même en cas de désactivation du gestionnaire de sécurité. Vous pouvez désactiver le gestionnaire de sécurité dans un environnement de développement. Toutefois, ne le désactivez pas dans un environnement de production.

  1. Dans l'arborescence de navigation, sélectionnez le noeud Configuration.

  2. Sélectionnez la configuration à modifier.

  3. Sous le noeud Configuration, sélectionnez le noeud Paramètres JVM.

    La page Paramètres généraux JVM apparaît.

  4. Dans la page Paramètres généraux JVM, cliquez sur l'onglet Options JVM.

    La page Options JVM apparaît.

  5. Cochez la case située à gauche du champ Valeur contenant l'option -Djava.security.policy.

  6. Cliquez sur Supprimer.

  7. Cliquez sur Enregistrer.

  8. Redémarrez le serveur.
