Building a simple blog server on Java EE 5 using NetBeans 5.5
This screen cast will demonstrate the relative ease of creating a basic blog server built upon the generated CRUD (CReate - Update - Delete) application using new features found in NetBeans 5.5.  Once the basic CRUD application is in place, we'll quickly customize the application using various JSF components to transform the generated application into a usable blog server (complete with comments, ratings, and AJAX).  Finally, we'll demonstrate how quickly we can expose the blog server as a Web Service.  This demonstration is a testament of developer productivity using good tools and Java EE 5.
