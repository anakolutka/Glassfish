Verfügbare Updates

Die Seite Available Updates (Verfügbare Updates) liefert Informationen zu neueren Versionen von Komponenten des Installation-Image, die installiert sind.

Die Seite Available Updates (Verfügbare Updates) umfasst die folgenden Optionen.

Installationspfad

Der Dateisystempfad des Root-Verzeichnisses oder des Verzeichnisses auf der obersten Ebene des Installations-Image.

Install (Installieren)

Diese Schaltfläche aktualisiert die ausgewählten Komponenten.

Available Updates (Verfügbare Updates)

Diese Tabelle liefert die folgenden Informationen zu jeder Komponente, für die eine neuere Version verfügbar ist:

Komponente

Der Name der Komponente.

Kategorie

Die Softwarekategorie, mit der die Komponente verbunden ist, zum Beispiel Anwendungsserver, Systemtools und Datenbanken. Die Kategorie wird der Komponente vom Autor der Komponente bei deren Erstellung zugewiesen.

Version

Die Versionsnummer des Pakets, in dem die Komponente geliefert wird.

Installationsgröße

Der Festplattenspeicherplatz, den die Installation der Komponente beansprucht.

Quelle

Für installierte Komponenten ist dies der Name des Netzwerk-Repository, von dem die Komponente abgerufen wurde.

Für eine Add-On-Komponente oder ein Update ist dies der Name des Netzwerk-Repositorys, von dem die Komponente abzurufen ist.
