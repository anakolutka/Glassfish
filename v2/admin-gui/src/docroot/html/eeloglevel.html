The Logger Namespace Hierarchy

The Application Server provides a logger for each of its modules. The following table lists the names of the modules and the namespace for each logger in alphabetical order, as they appear on the Log Levels page. (see To configure log levels). The last three modules in the table do not appear on the Log Levels page.
