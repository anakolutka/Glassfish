Para suprimir un pool de conexiones del conector

Cuando se suprime un pool de conexiones de conector, los recursos de conectores que dependen de ese pool también se suprimirán.

  1. En el árbol de navegación, amplíe el nodo Recursos.

  2. Amplíe el nodo Conectores.

  3. En el nodo Configuración, seleccione el nodo Pools de conexiones de conector.

    Se abrirá la página Pools de conexiones de conectores.

  4. En la página Pools de conexiones de conectores, active la casilla de control situada junto al nombre del pool de conexiones que desee suprimir.

  5. Haga clic en Suprimir.
