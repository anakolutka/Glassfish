Provides for loading the custom mbeans that are created in the application server.

Package Specification

Classes and interfaces required to register the MBeans in the MBeanServer. Once the MBean definitions are created in the domain's configuration, the code to actually register the classes as MBeans in the server resides here. It is expected that the code that wants to register MBeans calls the various methods defined in CustomMBeanRegistration interface that is defined elsewhere.
