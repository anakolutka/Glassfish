Recursos JDBC

Un recurso de Java DataBase Connectivity (JDBC) (fuente de datos) proporciona a las aplicaciones un método para conectarse a una base de datos. Normalmente, el administrador crea un recurso JDBC para cada base de datos a la que se accede a través de las aplicaciones implementadas en un dominio; sin embargo, se puede crear más de un recurso JDBC para una base de datos.

Las aplicaciones obtienen una conexión a la base de datos de un conjunto de conexiones al buscar una fuente de datos en el árbol de API Java Naming and Directory Interface (J.N.D.I.) y, a continuación, solicitan una conexión. El conjunto de conexiones asociado a la fuente de datos proporciona la conexión a la aplicación.

La página Recursos JDBC muestra una lista de los recursos que se han creado en el servidor de aplicaciones. En cada recurso, aparece la siguiente información:

  • Nombre J.N.D.I.

  • Estado

  • Conjunto de conexiones

  • Descripción
