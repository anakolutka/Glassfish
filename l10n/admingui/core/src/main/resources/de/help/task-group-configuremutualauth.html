Aktivieren der gegenseitigen Authentifizierung

Bei der gegenseitigen Authentifizierung ist die Authentifizierung sowohl auf Server- als auch auf Clientseite aktiviert. Zum Testen der gegenseitigen Authentifizierung muss ein Client mit gültigem Zertifikat vorhanden sein. Informationen zur gegenseitigen Authentifizierung finden Sie im Abschnitt zur Sicherheit im Java EE 5-Lernprogramm.

Die Aktivierung der gegenseitigen Authentifizierung umfasst folgende Schritte:

  • So aktivieren Sie die gegenseitige Authentifizierung für alle Anwendungen

  • So aktivieren Sie die gegenseitige Authentifizierung für eine bestimmte Anwendung
