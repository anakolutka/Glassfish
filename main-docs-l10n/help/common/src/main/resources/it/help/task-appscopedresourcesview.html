Per visualizzare le risorse con ambito applicazione

  1. Selezionare il nodo Applicazioni nell'albero di navigazione.

    Viene visualizzata la pagina Applicazioni.

  2. Selezionare l'applicazione per la quale si desidera visualizzare le risorse con ambito applicazione.

    Il nome applicazione può includere un identificativo di versione opzionale, che segue il nome ed è separato da questo mediante due punti (:). Per ulteriori informazioni sulle versioni di moduli e applicazioni, vedere Module and Application Versions in Oracle GlassFish Server Application Deployment Guide.

    Viene visualizzata la pagina per l'applicazione.

  3. Selezionare la scheda Risorse.

    Viene visualizzata la pagina Risorse con ambito applicazione e le risorse attualmente specificate per l'applicazione vengono visualizzate in una tabella con le colonne riportate di seguito.

    Nome applicazione

    Il nome dell'applicazione.

    Nome modulo

    Il nome del modulo all'interno dell'applicazione.

    Nome risorsa

    Il nome della risorsa con ambito applicazione per l'applicazione o il modulo.

    Tipo di risorsa

    Il tipo di risorsa, ad esempio Connection pool JDBC o Risorse connettore.
