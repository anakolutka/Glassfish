Nuova password amministratore

Utilizzare la pagina Nuova password amministratore per modificare la password per l'account amministratore corrente.

Nota:

Se l'amministrazione sicura è abilitata, non è possibile modificare la password in un valore vuoto.

La pagina Nuova password amministratore contiene le opzioni riportate di seguito.

ID utente

L'ID utente dell'amministratore corrente. Questo campo è di sola lettura.

Lista gruppi

Il gruppo al quale appartiene l'amministratore. Questo campo è di sola lettura. Il valore è asadmin.

Nuova password

La nuova password dell'amministratore.

Conferma nuova password

La nuova password dell'amministratore.

Task correlato

Per modificare la password dell'amministratore

Comando asadmin correlato

change-admin-password(1)
