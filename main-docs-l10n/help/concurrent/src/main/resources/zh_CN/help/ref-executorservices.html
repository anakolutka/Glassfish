托管执行程序服务

使用“托管执行程序服务”页可配置托管执行程序服务资源。

对于每个上下文服务资源，将提供以下信息。

JNDI 名称

标识资源的唯一名称。

已启用

如果资源已启用，则为“真”；如果资源已禁用，则为“假”。

上下文信息

资源的上下文：类加载器、JNDI、安全性或工作区。

线程优先级

要为托管执行程序服务实例创建的线程优先级。默认值为 5。

说明

资源的说明。

“托管执行程序服务”表还包含以下选项。

新建

用于创建新资源的按钮。

删除

用于删除一个或多个所选资源的按钮。

启用

用于启用一个或多个所选资源的按钮。

禁用

用于禁用一个或多个所选资源的按钮。

相关的 asadmin 命令

  • create-XXX(1)

  • list-XXX(1)

  • delete-XXX(1)
