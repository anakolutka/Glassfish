リソース

「リソース」ページを使用して、選択したGlassFish Serverインスタンスまたはクラスタの次のリソースを構成します。

  • JDBC

  • コネクタ

  • リソース・アダプタ構成

  • JMSリソース

  • JavaMailセッション

  • JNDI

  • 並行処理リソース

「リソース」ページには、次のオプションも表示されます。

リソースの追加

glassfish-resources.xmlファイルから1つまたは複数のリソースを作成するボタン。
