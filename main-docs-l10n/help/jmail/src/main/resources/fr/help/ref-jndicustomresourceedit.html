Modifier une ressource personnalisée

La page Modifier une ressource personnalisée permet de modifier une ressource JNDI personnalisée.

Elle contient les options ci-après.

Nom JNDI

Nom unique identifiant la ressource. Le champ Nom JNDI est en lecture seule. Vous ne pouvez spécifier le nom JNDI qu'à la création d'une ressource personnalisée.

Type de ressource

Définition qualifiée complète du type de la ressource.

Classe de fabrique

Nom de classe de fabrique spécifié par l'utilisateur. Cette classe implémente l'interface javax.naming.spi.ObjectFactory.

Ordre de déploiement

Ordre de déploiement de la ressource personnalisée.

Description

Description de cette ressource personnalisée.

Statut

Indique si la ressource est disponible lors de l'exécution.

Si seule l'instance de serveur par défaut server, existe dans le domaine, la disponibilité d'exécution de la ressource peut être activée ou désactivée. La disponibilité d'exécution est activée par défaut.

Si des clusters ou des instances autonomes ont été créés dans le domaine, utilisez l'onglet Cible pour contrôler la disponibilité d'exécution de la ressource sur ses cibles.

Autres propriétés

Propriétés supplémentaires de la ressource personnalisée. GlassFish Server ne définit aucune autre propriété pour les ressources personnalisées.
