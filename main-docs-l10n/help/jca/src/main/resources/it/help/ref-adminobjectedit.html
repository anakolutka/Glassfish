Modifica risorsa oggetto amministrato

Utilizzare la pagina Modifica risorsa oggetto amministrato per modificare le impostazioni di una risorsa oggetto amministrato.

La pagina Modifica risorsa oggetto amministrato contiene le opzioni riportate di seguito.

Carica impostazioni predefinite

Pulsante per ripristinare i valori predefiniti delle impostazioni che dispongono di valori predefiniti. Le impostazioni che non dispongono di valori predefiniti non vengono modificate.

Nome JNDI

Il nome univoco che identifica la risorsa oggetto amministrato. Il campo Nome JNDI è di sola lettura. È possibile specificare il nome JNDI solo quando si crea una nuova risorsa oggetto amministrato.

Resource Adapter

Il Resource Adapter per la risorsa oggetto amministrato.

Tipo di risorsa

Il tipo Java per la risorsa oggetto amministrato, ad esempio javax.jms.Topic.

Nome classe

Il nome della classe di implementazione associato al tipo di risorsa.

Descrizione

La descrizione della risorsa oggetto amministrato.

Stato

Se questa opzione è selezionata, la risorsa oggetto amministrato è abilitata. Questa opzione è abilitata per impostazione predefinita.

Proprietà aggiuntive

Proprietà aggiuntive per la risorsa oggetto amministrato. Le proprietà disponibili dipendono dal Resource Adapter per il connection pool del connettore. Le coppie nome-valore specificate in questa tabella possono essere utilizzate per sostituire i valori predefiniti per le proprietà specificate dal fornitore del Resource Adapter.
