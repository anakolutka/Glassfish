新建 File 範圍使用者

使用 [新建 File 範圍使用者] 頁面可授予新使用者對目前範圍的存取權。

範圍名稱

顯示目前選取範圍之名稱的唯讀欄位。

使用者 ID

新使用者帳號的使用者 ID。

群組清單

新使用者所屬的群組。以逗號分隔多個群組名稱。

新密碼

新使用者的密碼。

確認新密碼

新使用者的密碼。
