更改管理员密码

对管理员密码进行加密可以增强应用程序的安全性。有关安全性步骤的更多详细信息，请参见 《Sun GlassFish Enterprise Server v3 Prelude Administration Guide》中的第 6  章 “Administering System Security”。

  1. 在导航树中，展开“配置”节点。
  2. 如果域配置为支持群集，请选择您要更改的配置。

    示例是使用群集配置文件或企业配置文件创建的域。

  3. 展开“安全性”节点。
  4. 展开“领域”节点。
  5. 选择 "admin-realm" 节点。
  6. 在“编辑领域”页面中，单击“管理用户”按钮。
  7. 选择名为 admin 的用户。
  8. 输入新密码并确认密码。
  9. 单击“保存”。
