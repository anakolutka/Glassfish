Anwendungen

Die Seite Anwendungen zeigt eine Liste von Anwendungen an, die im &ProductName eingesetzt werden;. Sie können die bereits bereitgestellten Anwendungen anzeigen und verwalten, und Sie können weitere Anwendungen bereitstellen.

Für jede Anwendung werden die folgenden Informationen zur Verfügung gestellt.

Name (Name)

Der Name der Anwendung.

Enabled (Aktiviert)

Ein Häkchen, wenn die Anwendung aktiviert ist. Ein X, wenn die Anwendung deaktiviert ist.

Composite (Zusammengesetzt)

Wahr, wenn die Anwendung mehr als eine Komponente besitzt. Falsch, wenn sie nur über einen Komponententyp verfügt.

Engines

Die von der Anwendung verwendeten Container-Typen. Es stehen folgende Container-Typen zur Verfügung:

  • web

  • webservices

  • ejb

  • connector

  • appclient

  • jruby

Aktion

Links zu den Aktionen, die Sie nach der Bereitstellung mit der Komponente ausführen können: erneut bereitstellen und für alle Komponenten neu starten oder für Webanwendungen starten.

Deploy (Bereitstellen)

Schaltfläche zum Bereitstellen einer Unternehmensanwendung.

Undeploy (Bereitstellung aufheben)

Schaltfläche zum Aufheben der Bereitstellung einer Unternehmensanwendung.

Enable (Aktivieren)

Schaltfläche zum Aktivieren der ausgewählten Unternehmensanwendung.

Disable (Deaktivieren)

Schaltfläche zum Deaktivieren der ausgewählten Unternehmensanwendung.

Filter (Filter)

Dropdown-Liste, die Anwendungen nach Engine filtert.

Verwandte Aufgaben

  • So zeigen Sie eine Liste eingesetzter Anwendungen an

  • So stellen Sie eine Webanwendung bereit

  • So stellen Sie eine Unternehmensanwendung bereit

  • So stellen Sie einen Anwendungs-Client bereit

  • So stellen Sie ein Verbindungsmodul bereit

  • So stellen Sie ein EJB Jar bereit

  • So stellen Sie eine Ruby-Anwendung bereit

  • So bearbeiten Sie eine bereitgestellte Anwendung

  • So zeigen Sie die Beschreibung einer Anwendung an

  • So starten Sie eine Anwendung

  • Eine bereitgestellte Anwendung erneut bereitstellen

  • Eine bereitgestellte Anwendung erneut starten
