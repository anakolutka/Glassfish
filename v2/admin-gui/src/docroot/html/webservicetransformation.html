Using XSL Transformation Rules

You can apply XSL transformation rules for a web service endpoint. This enables fine grained control of web service request and response. Multiple XSL transformation rules can be applied to a web service endpoint. The order in which the transformation rules are applied can be configured in the Administration Console. All the generated XSLT files are stored in the generated/xml/application or module name/. Transformation rules help manage web services operation.
