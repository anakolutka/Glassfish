Configurer le traitement des requêtes de service HTTP

  1. Dans l'arborescence de navigation, développez le nœud Configuration.
  2. Sélectionnez le nœud Service HTTP.
  3. Dans la page Service HTTP, sélectionnez l'onglet Traitement des requêtes.

    La page Traitement des requêtes s'ouvre.

  4. Dans le champ Nombre de threads, entrez le nombre maximal de threads de traitement de requêtes.

    La valeur par défaut est 20.

  5. Dans le champ Nombre de threads initial, entrez le nombre de threads de traitement de requêtes disponibles au démarrage du serveur.

    La valeur par défaut est 2.

  6. Dans le champ Incrémentation des threads, entrez le nombre de threads de traitement de requêtes à ajouter lorsque le nombre de requêtes dépasse le nombre de threads initial.

    La valeur par défaut est 1.

  7. Dans le champ Délai d'expiration de la requête, entrez le nombre de secondes devant s'écouler avant expiration de la requête.

    La valeur par défaut est 30.

  8. Dans le champ Taille du tampon, entrez la taille, en octets, du tampon utilisé par les threads de traitement de requêtes pour lire les données demandées.

    La valeur par défaut est 8192.

  9. Cliquez sur Enregistrer.
