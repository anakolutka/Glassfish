Modifica mappa di sicurezza lavoro

Utilizzare la pagina Modifica mappa di sicurezza lavoro per modificare le impostazioni di una mappa di sicurezza lavoro.

La pagina Modifica mappa di sicurezza lavoro contiene le opzioni riportate di seguito.

Nome mappa di sicurezza

Il nome della mappa di sicurezza lavoro. Il campo del nome è di sola lettura. È possibile specificare un nome per la mappa di sicurezza solo quando si crea una nuova mappa di sicurezza lavoro.

Gruppo

Se il pulsante di scelta Gruppo è selezionato, nel campo di testo deve essere specificata una mappa di ciascun gruppo di utenti EIS backend al gruppo di utenti corrispondente definito nel dominio di sicurezza di GlassFish Server. Per specificare più mapping, utilizzare una lista separata da virgole. Ad esempio:

eis-group-1=server-group-1,eis-group-2=server-group-2
Principal

Se il pulsante di scelta Principal è selezionato, nel campo di testo deve essere specificata la mappa di ciascun principal EIS backend al principal corrispondente definito nel dominio di sicurezza di GlassFish Server. Per specificare più mapping, utilizzare una lista separata da virgole. Ad esempio:

eis-principal-1=server-principal-1,eis-principal-2=server-principal-2
Resource Adapter

Il Resource Adapter per il sistema EIS.

Ordine di distribuzione

L'ordine di distribuzione della mappa di sicurezza lavoro.

Descrizione

La descrizione della mappa di sicurezza lavoro.

Stato

Se questa opzione è selezionata, la mappa di sicurezza lavoro è abilitata. Questa opzione è abilitata per impostazione predefinita.
