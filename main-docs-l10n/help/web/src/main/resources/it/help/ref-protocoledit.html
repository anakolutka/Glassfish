Modifica protocollo

Utilizzare la pagina Modifica protocollo per modificare le impostazioni di un protocollo di rete esistente.

La pagina Modifica protocollo contiene le opzioni riportate di seguito.

Carica impostazioni predefinite

Pulsante per ripristinare i valori predefiniti delle impostazioni che dispongono di valori predefiniti. Le impostazioni che non dispongono di valori predefiniti non vengono modificate.

Nome configurazione

Il nome della configurazione a cui si applicano le impostazioni presenti in questa pagina. Questo campo è di sola lettura.

Nome

Il nome del protocollo. Il campo Nome è di sola lettura. È possibile specificare il nome solo quando si crea un nuovo protocollo.

Sicurezza

Se questa opzione è selezionata, la sicurezza viene abilitata per il protocollo. Se la sicurezza è abilitata, è possibile utilizzare la scheda SSL per modificare le impostazioni di sicurezza.

Se si sta configurando admin-listener per il server di amministrazione del dominio (DAS), che utilizza la configurazione server-config, questa opzione è di sola lettura. Per abilitare la sicurezza per questo protocollo, è necessario abilitare l'amministrazione sicura. Vedere Amministrazione sicura.
