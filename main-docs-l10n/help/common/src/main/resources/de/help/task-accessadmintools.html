So wird Zugriff auf Administrationstools erteilt

Nur Benutzer der Gruppe asadmin können auf die Administrationskonsole und das Befehlszeilenutility asadmin zugreifen. Um Benutzern den Zugriff auf diese Administrationstools zu erteilen, fügen Sie sie der Gruppe asadmin in admin-realm hinzu.

  1. Blenden Sie unter dem Knoten "Konfigurationen" die Serverinstanz oder die Clusterkonfiguration ein, für die Sie Administrationszugriffsberechtigungen konfigurieren möchten.

  2. Blenden Sie den Knoten "Sicherheit" ein.

  3. Blenden Sie den Knoten "Realms" ein.

  4. Wählen Sie den Knoten admin-realm.

    Die Seite "Realm bearbeiten" wird geöffnet.

  5. Klicken Sie auf der Seite "Realm bearbeiten" auf die Schaltfläche "Benutzer verwalten".

    Die Seite "Admin-Benutzer" wird geöffnet.

    Nach der Installation werden die während der Installation eingegebenen Werte für Benutzername und Kennwort des Administrators in einer Datei mit dem Namen admin-keyfile aufgelistet. Standardmäßig gehört dieser Benutzer zur Gruppe asadmin, die über Berechtigungen zum Ändern von GlassFish Server verfügt. Fügen Sie zu dieser Gruppe nur Benutzer hinzu, denen Sie Administratorberechtigungen für die ausgewählte GlassFish Server-Instanz oder das Cluster gewähren möchten.

    Wenn Sie einen Benutzer der Realm admin-realm hinzufügen, jedoch nicht der Gruppe asadmin zuweisen, werden die Benutzerinformationen zwar in die Datei admin-keyfile geschrieben, der Benutzer ist jedoch nicht für den Zugriff auf die Administrationstools oder Anwendungen in der Realm file berechtigt.

  6. Klicken Sie auf der Seite "Admin-Benutzer" auf "Neu", um der Realm admin-realm einen neuen Benutzer hinzuzufügen.

    Die Seite "Neuer File Realm-Benutzer" wird geöffnet.

  7. Geben Sie in das Feld "Benutzer-ID" den neuen Benutzernamen ein.

  8. Geben Sie in das Feld "Gruppenliste" die Namen der Gruppen ein, zu denen der Benutzer gehört.

    Um einem Benutzer Berechtigungen zum Durchführen von Änderungen an GlassFish Server zuzuweisen, fügen Sie die Gruppe asadmin im Feld "Gruppenliste" hinzu.

  9. Geben Sie in die Felder "Neues Kennwort" und "Neues Kennwort bestätigen" das Kennwort für den neuen Benutzer ein.

  10. Klicken Sie auf "OK", um diesen Benutzer der Realm admin-realm hinzuzufügen, oder klicken Sie auf "Abbrechen", um den Vorgang ohne Speichern zu beenden.
