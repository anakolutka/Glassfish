So konfigurieren Sie die JVM-Profilereinstellungen

Ein Profiler erzeugt Daten, die zur Analyse der Leistung sowie zur Ermittlung von potenziellen Engpässen verwendet werden können.

  1. Wählen Sie in der Navigationsstruktur den Konfigurationsknoten.
  2. Wählen Sie unter dem Konfigurationsknoten den JVM-Einstellungsknoten aus.

    Die Seite "Allgemeine JVM-Einstellungen" wird geöffnet.

  3. Klicken Sie auf der Seite "Allgemeine JVM-Einstellungen" auf die Registerkarte "Profiler".

    Die Seite "JVM-Profilereinstellungen" wird geöffnet.

  4. Geben Sie die Profilereinstellungen an.

    Die eingegebenen Informationen richten sich danach, welchen Profiler Sie verwenden. Beispiele und Anweisungen finden Sie in Kapitel 4, Debugging Applications in Sun GlassFish Enterprise Server v3 Preview Application Development Guide.

  5. Klicken Sie auf "Speichern".
  6. Starten Sie den Server neu.
