Modifica risorsa personalizzata

Utilizzare la pagina Modifica risorsa personalizzata per modificare una risorsa JNDI personalizzata.

La pagina Modifica risorsa personalizzata contiene le opzioni riportate di seguito.

Nome JNDI

Il nome univoco che identifica la risorsa. Il campo Nome JNDI è di sola lettura. È possibile specificare il nome JNDI solo quando si crea una nuova risorsa personalizzata.

Tipo di risorsa

La definizione completamente qualificata del tipo di risorsa.

Classe factory

Il nome specificato dall'utente per la classe factory. Questa classe implementa l'interfaccia javax.naming.spi.ObjectFactory.

Descrizione

La descrizione di questa risorsa personalizzata.

Stato

Lo stato della risorsa personalizzata. La risorsa personalizzata può essere abilitata o disabilitata. Lo stato predefinito è Abilitato.

Proprietà aggiuntive

Proprietà aggiuntive per la risorsa personalizzata. GlassFish Server non definisce alcuna proprietà aggiuntiva per le risorse personalizzate.
