HTTP 서비스 모니터링 데이터를 보는 방법

  1. 탐색 트리에서 Application Server 노드를 선택합니다.
  2. 일반 정보 페이지에서 모니터 탭을 누릅니다.
  3. 웹 컨테이너 모니터링 페이지에서 HTTP 서비스 탭을 누릅니다.

    HTTP 서비스 모니터링 페이지가 열립니다.

  4. (옵션) 가상 서버 드롭다운 목록에서 해당 서버에 대한 요청 통계를 볼 서버를 선택합니다.
  5. (옵션) HTTP 수신기 드롭다운 목록에서 해당 수신기에 대한 요청 통계를 볼 수신기를 선택합니다.
  6. (옵션) 갱신 버튼을 눌러 표시된 모니터링 데이터를 업데이트합니다.
참조

  • 모니터링 서비스를 구성하는 방법

  • 웹 컨테이너 모니터링 데이터를 보는 방법

  • JVM 모니터링 데이터를 보는 방법

  • 스레드 풀 모니터링 데이터를 보는 방법
