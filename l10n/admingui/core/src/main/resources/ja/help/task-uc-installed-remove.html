コンポーネントをアンインストールする

コンポーネントのアンインストールには次の項目の削除が含まれます。

  • コンポーネントの最初のインストールによって導入されたすべてのファイルとディレクトリ

  • 最初のインストール後に、コンポーネントの更新によって導入されたすべてのファイルとディレクトリ

  1. ナビゲーションツリーで、更新ツール ノードを選択します。

    「利用可能なアドオン」ページが開きます。

  2. 「利用可能なアドオン」ページで、「インストール済みコンポーネント」タブをクリックします。

    「インストール済みコンポーネント」ページが開きます。

  3. インストール済みコンポーネントの表で、アンインストールするコンポーネントを選択します。
    • 個々のコンポーネントを選択するには、そのコンポーネントの名前の隣にあるチェックボックスを選択します。
    • すべてのコンポーネントを選択するには、表ヘッダーの「すべてを選択」アイコンをクリックします。
    • すべてのコンポーネントを選択解除するには、表ヘッダーの「すべてを選択解除」アイコンをクリックします。
  4. 「削除」をクリックします。

    選択したコンポーネントのアンインストールが必要であることを確認するダイアログボックスが表示されます。

  5. ダイアログボックスの「了解 (OK)」をクリックします。

    Enterprise Server はアンインストールが完了したことを確認します。コンポーネントがインストール済みコンポーネントの表から削除され、Available Add-Onsページの利用可能なアドオンの表に追加されます。
