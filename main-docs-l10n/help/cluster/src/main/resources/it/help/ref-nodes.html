Nodi

Nella pagina Nodi viene visualizzata la lista di nodi del dominio. Per ogni nodo vengono visualizzate le informazioni riportate di seguito.

Nome

Il nome assegnato al nodo al momento della creazione. Se si fa clic sul nome, viene visualizzata la pagina Modifica nodo per il nodo.

Host nodo

Il nome dell'host rappresentato dal nodo.

Tipo

Il tipo del nodo, che può essere uno dei tipi riportati di seguito.

CONFIG

Il nodo non supporta la comunicazione remota.

DCOM

Il nodo supporta la comunicazione sul protocollo remoto DCOM (Distributed Component Object Model). Il protocollo DCOM è disponibile solo nei sistemi Windows.

SSH

Il nodo supporta la comunicazione su SSH (Secure Shell).

Istanze

Lista delle istanze che risiedono nel nodo con indicazione relativa all'eventuale stato di esecuzione di ciascuna istanza:

  • se l'istanza è in esecuzione, viene visualizzato un segno di spunta (✓);

  • se l'istanza è stata arrestata, viene visualizzato un punto esclamativo (!).

Se si fa clic sul nome di un'istanza viene visualizzata la pagina Informazioni generali per l'istanza.

Azione

Solo per i nodi di tipo DCOM e SSH: collegamento all'azione Ping per il nodo. Se si fa clic sul collegamento Ping viene eseguito un test per verificare se il nodo è raggiungibile sul protocollo di comunicazione supportato dal nodo.

Per i nodi CONFIG non sono possibili azioni.

La tabella Nodi contiene inoltre le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare un nodo.

Elimina

Pulsante che consente di eliminare uno o più nodi selezionati.

Elimina e disinstalla

Pulsante che consente di eliminare uno o più nodi selezionati e di disinstallare il software di GlassFish Server dall'host rappresentato da ciascun nodo eliminato.

Nota:

Il software di GlassFish Server non viene disinstallato da alcun host rappresentato da un nodo di tipo CONFIG. Per i nodi di tipo CONFIG, l'azione di questo pulsante è identica a quella del pulsante Elimina.
