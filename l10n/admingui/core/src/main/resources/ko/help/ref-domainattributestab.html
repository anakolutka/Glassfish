도메인 속성

이 페이지에서는 Enterprise Server에 대한 고급 도메인 속성을 설정할 수 있습니다.

응용 프로그램 루트

응용 프로그램이 배포되는 디렉토리의 전체 경로입니다.

로그 루트

서버 로그 파일이 저장되는 위치를 식별합니다.

로켈

이 필드를 비워 두면 호스트의 기본 로켈이 사용됩니다.

로켈은 언어 및 지역의 특정 조합을 지정하는 식별자입니다. 예를 들어 미국 영어의 로켈은 en_US이고 일본어의 로켈은 ja_JP입니다. 영어가 아닌 로켈을 사용하려면 Application Server가 현지화(영어를 다른 언어로 번역하는 작업이 포함된 프로세스)되어 있어야 합니다.
