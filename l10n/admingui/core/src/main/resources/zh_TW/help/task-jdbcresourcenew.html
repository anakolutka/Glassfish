建立 JDBC 資源

Java 資料庫連結 (JDBC) 資源 (資料來源) 提供應用程式連線至資料庫的方法。

開始之前

您必須先建立 JDBC 連線池，才可建立 JDBC 資源。請參閱建立 JDBC 連線池。

  1. 在瀏覽樹狀結構中，展開 [資源] 節點。
  2. 選取 [JDBC 資源] 節點。
  3. 在 [JDBC 資源] 頁面上，按一下 [新增]。

    [新 JDBC 資源] 頁面會隨即開啟。

  4. 在 [J.N.D.I. 名稱] 欄位中，鍵入唯一的資源名稱。

    J.N.D.I. 名稱可用以為分散式運算環境中的元件加以組織與定位，就如同圖書館裡的卡片目錄，可用以將書籍分門別類並表示擺放位置。因此，J.N.D.I. 名稱便成為存取 JDBC 資源的重要方法。依慣例，會是以 jdbc/ 字串開始的名稱，例如 jdbc/payrolldb。請勿遺漏正斜線。

    名稱僅能包含字母數字、底線、破折號或點字元。

  5. 從 [池名稱] 下拉式清單中，選取要與新 JDBC 資源產生關聯的連線池。
  6. 在 [說明] 欄位中，鍵入新 JDBC 資源的簡短說明。
  7. 若要停用資源使其無法使用，請取消選取 [已啟用] 核取方塊。

    依預設，資源在建立之後即可使用 (已啟用)。

  8. 按一下 [確定]。
