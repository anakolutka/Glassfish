제품 등록

설치된 Sun GlassFishTMEnterprise Server를 Sun Connection에 등록하려면 Sun 온라인 계정이 있어야 합니다. Sun 온라인 계정이 없는 경우 Enterprise Server를 등록하는 동안 Enterprise Server를 사용하여 계정을 만들 수 있습니다.

제품 등록 페이지에는 다음 옵션이 포함되어 있습니다.

이미 Sun 온라인 계정 또는 Sun 개발자 네트워크(SDN) 계정이 있습니다.

계정이 이미 있는 경우 선택합니다. 이 옵션을 선택하면 다음 필드가 표시됩니다.

사용자 이름

Sun 온라인 계정을 등록할 때 지정한 사용자 이름입니다.

비밀번호

Sun 온라인 계정을 등록할 때 지정한 비밀번호입니다.

프록시 호스트

Enterprise Server가 실행 중인 시스템에 대한 프록시 서버의 호스트 이름입니다.

이 필드는 Enterprise Server가 실행 중인 시스템이 프록시 서버를 통해 인터넷에 연결되는 경우에만 적용됩니다.

프록시 포트

Enterprise Server가 실행 중인 시스템에 대한 프록시 서버의 포트 번호입니다.

이 필드는 Enterprise Server가 실행 중인 시스템이 프록시 서버를 통해 인터넷에 연결되는 경우에만 적용됩니다.

Sun 온라인 계정이 없습니다. 등록해 주십시오.

계정이 없고 Sun Connection을 사용하여 Enterprise Server 설치를 등록하려는 경우 선택합니다. 이 옵션을 선택하면 다음 필드가 표시됩니다.

전자 메일 주소

Sun 온라인 계정의 사용자 이름으로 사용할 전자 메일 주소입니다.

비밀번호

Sun 온라인 계정에 사용할 비밀번호입니다. 비밀번호는 대소문자를 구분하며 6–10자 이내로 입력해야 합니다.

비밀번호 재입력

비밀번호 필드에 지정한 비밀번호를 다시 입력해야 합니다.

이름

이름입니다.

성

성입니다.

회사

적용 가능한 경우 직장을 입력합니다.

국가/지역

거주 국가 또는 지역입니다.

프록시 호스트

Enterprise Server가 실행 중인 시스템에 대한 프록시 서버의 호스트 이름입니다.

이 필드는 Enterprise Server가 실행 중인 시스템이 프록시 서버를 통해 인터넷에 연결되는 경우에만 적용됩니다.

프록시 포트

Enterprise Server가 실행 중인 시스템에 대한 프록시 서버의 포트 번호입니다.

이 필드는 Enterprise Server가 실행 중인 시스템이 프록시 서버를 통해 인터넷에 연결되는 경우에만 적용됩니다.
