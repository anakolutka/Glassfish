Afficher les informations détaillées d'un composant

Le Outil de mise à jour contient des informations détaillées sur les composants installés, les mises à jour disponibles pour ces composants et les composants add-ons.

  1. Dans l'arborescence de navigation, sélectionnez le nœud Outil de mise à jour.

    La page Composants installés s'affiche.

  2. Cliquez sur l'onglet de la page où le composant est répertorié.

    Le composant est répertorié sur l'une des pages suivantes :

    • Composants installés

    • Mises à jour disponibles

    • Add-ons disponibles

  3. Dans le tableau des composants, sélectionnez le composant.

    La page Détails des composants du composant sélectionné s'affiche.
