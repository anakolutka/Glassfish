Affichage des données de surveillance de ressource pour une instance GlassFish Server incluse dans un cluster

Avant de commencer

Avant d'afficher les données de surveillance, vous devez configurer le service de surveillance.

  1. Dans l'arborescence de navigation, développez le noeud Clusters.

  2. Sous le noeud Clusters développé, sélectionnez le noeud correspondant au cluster.

    La page Informations générales du cluster apparaît.

  3. Sur la page Informations générales, cliquez sur l'onglet Instances.

    La page Instances de serveur clusterisées du cluster apparaît.

  4. Sélectionnez l'instance pour laquelle vous souhaitez visualiser les données de surveillance de ressource.

    La page Informations générales de l'instance apparaît.

  5. Sur la page Informations générales, cliquez sur l'onglet Surveiller.

    La page Surveillance d'application apparaît.

  6. Cliquez sur l'onglet Ressources.

    La page Surveillance de ressource apparaît.

  7. Dans la liste déroulante Afficher, sélectionnez la ressource pour laquelle vous souhaitez afficher les données de surveillance.

  8. Pour mettre à jour les données de surveillance affichées, cliquez sur Actualiser.
