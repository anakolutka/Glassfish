Recursos de conectores

Utilice la página Recursos de conectores para configurar los recursos de conectores.

Un recurso de conector es un objeto de programa que proporciona una conexión a un EIS a una aplicación.

Para cada recurso, aparece la siguiente información.

Nombre JNDI

Un nombre único que identifica el recurso de conector.

Activado

Verdadero si el recurso está activado, o falso si no lo está.

Pool de conexiones

Pool de conexiones de conector asociado al recurso de conector.

Descripción

Descripción del recurso de conector.

La tabla Recursos también contiene las siguientes opciones.

Nuevo

Botón para crear un nuevo recurso de conector.

Suprimir

Botón para suprimir uno o más recursos de conector seleccionados.

Activar

Botón para activar uno o más recursos de conector seleccionados.

Desactivar

Botón para desactivar uno o más recursos de conector seleccionados.
