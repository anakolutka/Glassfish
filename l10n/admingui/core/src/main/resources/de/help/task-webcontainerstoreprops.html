So konfigurieren Sie die Speichereigenschaften

  1. Erweitern Sie in der Navigationsstruktur den Konfigurationsknoten.
  2. Wählen Sie unter dem Konfigurationsknoten den Webcontainerknoten aus.

    Die Seite "Webcontainer" wird geöffnet.

  3. Klicken Sie auf der Seite "Webcontainer" auf die Registerkarte "Eigenschaften speichern".
  4. Geben Sie im Feld "Reap-Intervall" die Zeitdauer in Sekunden an, bis die Daten der inaktiven Sitzungen aus dem Speicher gelöscht werden.
  5. Geben Sie im Feld "Verzeichnis" den Pfad zu der Datei an, in der die Sitzungsdaten gespeichert werden.

    Diese Einstellung ist nur erforderlich, wenn Sie als Persistenztyp "Datei" gewählt haben.

  6. (Optional) Geben Sie im Abschnitt "Weitere Eigenschaften" weitere Eigenschaften an.
  7. Klicken Sie auf "Speichern".
