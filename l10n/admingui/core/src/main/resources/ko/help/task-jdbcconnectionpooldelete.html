JDBC 연결 풀을 삭제하는 방법

  1. 탐색 트리에서 자원 노드를 확장합니다.
  2. 자원 노드에서 JDBC 노드를 확장합니다.
  3. JDBC 노드에서 연결 풀 노드를 선택합니다.

    연결 풀 페이지가 열립니다.

  4. 연결 풀 페이지에서 삭제할 풀의 왼쪽에 있는 확인란을 선택합니다.
  5. 삭제를 누릅니다.

  • 데이터베이스 액세스를 설정하는 방법

  • JDBC 드라이버를 통합하는 방법

  • JDBC 자원을 만드는 방법

  • JDBC 자원을 편집하는 방법

  • JDBC 자원을 삭제하는 방법

  • JDBC 연결 풀을 만드는 방법

  • JDBC 연결 풀 일반 설정을 편집하는 방법

  • JDBC 연결 풀 고급 속성을 편집하는 방법

  • 연결 풀 설정을 확인하는 방법

  • JDBC 연결 풀 등록 정보를 편집하는 방법
