Deploying Applications and Modules

The Applications page lists the types of applications you can deploy. You can view and manage the applications that are already deployed and you can deploy more applications.

Use the Applications page to perform the following tasks:

  • To view, deploy, and manage enterprise applications

  • To view, deploy, and manage deployed web applications

  • To view, deploy, and manage EJB modules

  • To view, deploy, and manage connector modules

  • To view, deploy, and manage lifecycle modules

  • To deploy an application client module
