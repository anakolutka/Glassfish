更新ツールの設定

「構成」のページには 更新ツール に関する設定情報が含まれ、更新ツール の設定を変更できます。

「更新ツール の設定」ページには、次のトピックが含まれます。

インストールパス

インストールイメージのルートディレクトリまたは最上位ディレクトリのファイルシステムパス。

優先リポジトリ

インストールイメージに関連付けられている優先ネットワークリポジトリの名前。たとえば、glassfish.org などです。

インターネット接続

Enterprise Server のインターネットへの接続方法は次のとおりです。

直接接続

このオプションは、Enterprise Server がインターネットに直接接続するように指定します。

プロキシ経由

このオプションは、Enterprise Server がプロキシサーバー経由でインターネットに接続するように指定します。このオプションを選択する場合、「プロキシホスト」フィールドと「プロキシポート」フィールドがこのページに追加されます。

プロキシホスト

Enterprise Server を実行しているマシンのプロキシサーバーのホスト名。

このフィールドは、Enterprise Server を実行しているマシンがプロキシサーバーを介してインターネットに接続する場合のみ適用可能です。

プロキシポート

Enterprise Server を実行しているマシンのプロキシサーバーのポート番号。

このフィールドは、Enterprise Server を実行しているマシンがプロキシサーバーを介してインターネットに接続する場合のみ適用可能です。

リポジトリ

この表には、このインストールイメージに関連付けられている各ネットワークリポジトリに関する次の情報が含まれます。

リポジトリ

リポジトリの名前。たとえば、glassfish.org などです。
