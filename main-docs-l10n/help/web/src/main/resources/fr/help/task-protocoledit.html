Modification d'un protocole

  1. Dans l'arborescence de navigation, développez le noeud Configuration.

  2. Sous le noeud Configuration, développez le noeud Configuration réseau.

  3. Sous le noeud Configuration réseau, développez le noeud Protocoles.

    La page Protocoles apparaît.

  4. Sur la page Protocoles, cliquez sur le nom du protocole que vous modifiez.

    La page Modifier un protocole apparaît.

  5. Cochez la case Sécurité activée pour activer la sécurité du protocole.

    Si la sécurité est activée, vous pouvez utiliser l'onglet SSL pour modifier les paramètres de sécurité.

  6. Cliquez sur Enregistrer.

    Conseil:

    Pour revenir aux paramètres par défaut, cliquez sur le bouton Charger les valeurs par défaut, puis sur Enregistrer.

  7. Si la case Activé de l'option Sécurité est cochée, suivez les étapes décrites dans Modification des paramètres SSL pour un protocole.
