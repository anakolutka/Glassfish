编辑 JMS 目标资源

  1. 在导航树中，展开“资源”节点。

  2. 在“资源”节点下，展开“JMS 资源”节点。

  3. 在“JMS 资源”节点下，选择“JMS 目标资源”节点。

    此时将打开“JMS 目标资源”页。

  4. 在“JMS 目标资源”页上，单击您要编辑的目标资源的名称。

    此时将打开“编辑 JMS 目标资源”页。

  5. 在“物理目标名称”字段中，键入要与此资源关联的物理目标的名称。

    GlassFish Server 在需要时自动创建物理目标，并在您删除目标资源时删除物理目标。

  6. 从“资源类型”下拉列表中，选择 javax.jms.Topic 或 javax.jms.Queue。

  7. 在“说明”字段中，键入目标资源的说明。

  8. 选择“启用状态”复选框以在运行时启用该资源。

    默认情况下启用此选项。

  9. 在“其他属性”区域中，指定目标资源的其他属性。

    GlassFish Server 不定义 JMS 目标资源的任何其他属性。

    • 要添加属性，请单击“添加属性”按钮。在显示的空白行中，在“名称”字段中键入属性名称，在“值”字段中键入属性值。

    • 要修改属性，请编辑该属性的“值”字段。

    • 要删除属性，请选中位于您要删除属性的“名称”字段左边的复选框，然后单击“删除属性”按钮。

  10. 单击“保存”。

    提示:

    要返回到默认设置，请单击“加载默认值”按钮，然后单击“保存”。

  11. 可以选择性地单击“目标”选项卡来转到“JMS 目标资源目标”页。在此页上，执行以下操作：

    1. 单击“管理目标”以打开“管理资源目标”页。

      在此页上的“可用”列和“所选”列之间移动目标。对于要将使用资源的应用程序部署到的目标，确保将这些目标放置在“所选”列中。可用目标包括可用集群和服务器实例，还有默认的服务器实例 server。单击“确定”保存更改。

    2. 选中目标对应的复选框，然后单击“启用”或“禁用”，启用或禁用该目标的资源。
