Informações de Ponto Final de Web Service

Usa a página Informações de Ponto Final de Web Service para exibir informações sobre um ponto final do Web service.

A página Informações de Ponto Final de Web Service contém as iinformações a seguir.

Nome da Aplicação

O nome da aplicação implantada. Clique neste link ou no botão Voltar para retornar à página Editar Aplicação.

O nome pode incluir um identificador de versão opcional, que segue o nome e é separado do nome por dois-pontos (:). Para obter mais informações sobre versões modulares ou da aplicação, consulte Module and Application Versions em Oracle GlassFish Server Application Deployment Guide.

Testador

O link do Testador ficará visível se o Tipo de Implantação não for RI (por exemplo, se for 109). Clique neste link para exibir uma página que permite fornecer parâmetros para testar os métodos do Web Service.

WSDL

A WSDL (Web Services Definition Language) do ponto final. Clique neste link para exibir o link WSDL em uma guia ou janela separada do browser.

Observe que alguns Web browsers tentam interpretar o código-fonte WSDL quando você clica no link WSDL e exibem uma página em branco. Nesses browsers, utilize o comando Exibir código-fonte na página em branco para exibir o código-fonte WSDL.

Nome do Ponto Final

O nome do ponto final.

Nome do Serviço

O nome do Web Service.

Nome da Porta

O nome da porta do Web Service.

Tipo de Implantação

O tipo de implantação para o Web Service (por exemplo, RI ou 109).

Tipo de Implementação

O tipo de implementação para o Web Service (por exemplo, SERVLET ou EJB).

Nome da Classe de Implementação

O nome da classe de implementação do Web Service.

URI do Endereço do Ponto Final

O URI do endereço do ponto final.

Namespace

O namespace do alvo associado com o Web Service.

Descrição

Descrição do ponto final, se houver um.
