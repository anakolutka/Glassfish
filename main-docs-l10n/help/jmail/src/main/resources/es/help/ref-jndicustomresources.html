Recursos Personalizados

Un recurso personalizado especifica una fábrica de objetos de recursos personalizada para todo el servidor.

Utilice la página Recursos personalizados para configurar los recursos JNDI personalizados.

Para cada recurso personalizado, aparece la siguiente información.

Nombre JNDI

Nombre único que identifica el recurso personalizado.

Activado

"True" si el recurso personalizado está activado y "False" si está desactivado.

Tipo de recurso

Tipo de recurso personalizado.

Descripción

Descripción del recurso personalizado.

La tabla Recursos también contiene las siguientes opciones.

Nuevo

Botón para crear un nuevo recurso personalizado.

Suprimir

Botón para suprimir uno o varios recursos personalizados seleccionados.

Activar

Botón para activar uno o varios recursos personalizados seleccionados.

Desactivar

Botón para desactivar uno o varios recursos personalizados seleccionados.
