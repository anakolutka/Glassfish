Überwachung von Enterprise Server-Ereignissen

Mit diesen Modulen wird überwacht, wie Enterprise Server wichtige Ereignisse, wie z. B. Fehler oder Sicherheitsverletzungen, aufzeichnet. Enterprise Server zeichnet sämtliche Authentifizierungsereignisse in Protokolldateien auf. Ein vollständiges Zugriffsprotokoll umfasst eine Reihe von Enterprise Server-Zugriffsereignissen.

Die Überwachung von Enterprise Server-Ereignissen umfasst die folgenden Aufgaben:

  • So erstellen Sie ein Überwachungsmodul

  • So bearbeiten Sie ein Überwachungsmodul

  • So löschen Sie ein Überwachungsmodul

  • So aktivieren oder deaktivieren Sie die Überwachungsprotokollierung

  • So legen Sie das aktive Überwachungsmodul fest

  • So verwenden Sie das standardmäßige Überwachungsmodul
