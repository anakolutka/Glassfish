To Set Instance-Only Properties

  1. In the tree component, expand the Clusters node.
  2. Expand the node for the cluster that contains the server instance.
  3. Select the server instance node.
  4. In the General Information page, click the Properties tab.
  5. If necessary, click the Instance Properties tab.
  6. Modify any existing properties for which you require a new value.
  7. Set values for any additional properties that you require.
    1. In the Additional Properties table, click Add Property.
    2. In the Name field, type the property name.
    3. In the Value field, type the value.
  8. Click Save.
