업그레이드 도구: 개요

업그레이드 도구를 사용하여 Application Server 8.x 또는 Application Server 9 설치에서 Application Server 9.1 설치로 구성 정보를 전송합니다. 전송되는 정보에는 배포된 응용 프로그램, 파일 영역, 보안 인증서, 기타 리소스 및 서버 구성 설정 등에 대한 데이터가 포함됩니다.

다음 표에는 지원되는 Application Server 업그레이드가 표시됩니다. 여기에서 PE는 Platform Edition, EE는 Enterprise Edition을 의미합니다.

업그레이드 용어

다음은 업그레이드 프로세스와 관련된 중요 용어입니다.

  • 소스 서버: 새 버전으로 업그레이드할 설치입니다.

  • 대상 서버: 업그레이드 대상 설치입니다.

  • 도메인 루트: 도메인이 생성되는 디렉토리입니다. 기본적으로 이 디렉토리는 asenv.conf에서 AS_DEF_DOMAINS_PATH로 지정되는 위치입니다.

  • 도메인 디렉토리 또는 domain-dir: 특정 도메인에 해당하는 도메인 루트 내 디렉토리입니다. 도메인과 관련된 모든 구성 및 기타 데이터가 이 디렉토리에 있습니다.

  • 설치 루트: Application Server가 설치될 디렉토리입니다.

Application Server 설치 과정에서 업그레이드 확인란을 선택하는 경우 설치가 완료된 후 업그레이드 마법사 화면이 자동으로 표시됩니다. 업그레이드 프로세스가 시작됩니다.
