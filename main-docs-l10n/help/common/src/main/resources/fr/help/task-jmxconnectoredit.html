Modification du connecteur JMX

Utilisez le service d'administration pour configurer un connecteur JMX distant conforme à la JSR-160.

Le connecteur JMX distant conforme à la JSR-160 gère la communication entre les clients JMX (comme par exemple, JConsole) et le service JMX, à des fins de gestion de domaine.

  1. Dans l'arborescence de navigation, développez le noeud Configuration.

  2. Sous le noeud Configuration, sélectionnez le noeud Service d'administration.

    La page Modifier un connecteur JMX apparaît.

  3. Cochez la case Sécurité activée pour que la communication JMX soit cryptée.

    Cette option est désactivée par défaut.

  4. Dans le champ Adresse, saisissez l'adresse IP du service de dénomination auprès duquel le stub de serveur de connecteur JMX est inscrit.

    Il ne s'agit pas du port du socket de serveur qui effectue la véritable communication JMX. Il s'agit de l'adresse de l'interface réseau à partir de laquelle le registre RMI est démarré. Si votre système est doté de plusieurs interfaces réseau, modifiez cette valeur afin de sélectionner une seule interface. La valeur par défaut est 0.0.0.0.

  5. Dans le champ Port, saisissez le numéro de port sur lequel le service de dénomination (registre RMI) écoute les connexions client RMI.

    La seule fonction de ce service de dénomination consiste à télécharger les stubs RMI. Si le port par défaut est occupé, un port libre est utilisé à la place. Les valeurs valides sont comprises entre 1 et 65535. Sous les systèmes UNIX, la création de sockets qui écoutent sur les ports 1 à 1024 exige des privilèges de superutilisateur. La valeur par défaut est 8686.

  6. Dans le champ Nom de domaine, saisissez le nom qui représente le domaine d'administration spécial.

    Toutes les authentifications sont gérées par ce domaine. La valeur par défaut est admin-realm.

  7. Dans la section Autres propriétés, spécifiez les propriétés supplémentaires.

    GlassFish Server spécifie plusieurs propriétés par défaut.

    • Pour ajouter une propriété, cliquez sur le bouton Ajouter une propriété. Sur la ligne vide qui apparaît, saisissez le nom de la propriété dans le champ Nom et la valeur de la propriété dans le champ Valeur.

    • Pour modifier une propriété, modifiez le champ Valeur de cette propriété.

    • Pour supprimer une propriété, sélectionnez la case située à gauche du champ Nom de la propriété à supprimer, puis cliquez sur le bouton Supprimer des propriétés.

  8. Cliquez sur Enregistrer.

  9. Si vous avez coché la case Sécurité activée, suivez la procédure décrite dans Modifier des paramètres SSL pour le connecteur JMX.
