메시지 보안 구성

메시지 보안 구성 페이지에는 메시지 보안 공급자에 대해 사용 가능한 구성이 나열됩니다.

메시지 보안을 구성하려면 보안 및 웹 서비스 개념을 기본적으로 이해해야 합니다. 자세한 내용은 Sun GlassFish Enterprise Server v3 Administration Guide의 13 장, Administering Message Security 및 Sun GlassFish Enterprise Server v3 Administration Guide의 6 장, Administering Web Applications를 참조하십시오.

Enterprise Server는 SOAP 처리 계층에 통합된 메시지 보안 공급자를 사용하여 메시지 보안을 구현합니다. 메시지 보안 공급자는 Enterprise Server의 다른 보안 기능에 따라 달라집니다.

각 구성에 대해 메시지 보안 구성 페이지에는 다음 정보가 포함됩니다.

인증 계층

인증이 수행되는 메시지 계층입니다. 기본적으로 SOAP 구성이 제공됩니다. HttpServlet 구성을 제공할 수도 있습니다.

기본 공급자

이 구성에 대한 기본 서버 공급자입니다. 기본적으로 기본 서버 공급자가 구성되어 있지 않습니다.

기본 클라이언트 공급자

이 구성에 대한 기본 클라이언트 공급자입니다. 기본적으로 기본 클라이언트 구성자가 구성되어 있지 않습니다.

관련 작업

  • 메시지 보안 공급자 사용을 위해 Enterprise Server 기능을 구성하는 방법

  • 메시지 보안 구성을 만드는 방법

  • 메시지 보안 구성을 편집하는 방법

  • 메시지 보안 구성을 삭제하는 방법

  • 메시지 보안 공급자 구성을 만드는 방법

  • 메시지 보안 공급자 구성을 편집하는 방법

  • 메시지 보안 공급자 구성을 삭제하는 방법

  • 응용 프로그램 클라이언트에 대한 메시지 보안을 활성화하는 방법
