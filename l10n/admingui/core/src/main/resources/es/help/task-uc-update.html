Cómo actualizar componentes instalados

La actualización de un componente instalado incluye las siguientes operaciones:

  • Transferir sólo los archivos modificados de un depósito de red a su sistema

  • Actualizar sólo los archivos modificados dentro de la imagen de instalación de Servidor empresarial en su sistema

  • Aplicar nuevas dependencias (si fuera necesario) entre los componentes actualizados

  1. En el árbol de navegación, seleccione el nodo Herramienta de actualización.

    Se abrirá la página Componentes instalados.

  2. En la página Componentes instalados, haga clic en la ficha Actualizaciones disponibles.

    Aparecerá la página Actualizaciones disponibles.

  3. En la tabla de actualizaciones disponibles, seleccione los componentes que desee actualizar.

    Si no hay actualizaciones disponibles, la tabla aparecerá vacía.

    • Para seleccionar un componente individual, marque la casilla de verificación situada al lado del nombre del componente.
    • Para seleccionar todos los componentes, haga clic en el icono Seleccionar todo situado en el encabezado de la tabla.
    • Para anular la selección de todos los componentes, haga clic en el icono Anular toda la selección situado en el encabezado de la tabla.
  4. Haga clic en "Instalar".
  5. Acepte el acuerdo de licencia.

    Servidor empresarial confirma que la instalación ha finalizado. Los componentes desaparecerán de la tabla de actualizaciones disponibles.

  6. Reinicie el servidor de administración de dominios.

    Para obtener instrucciones detalladas, consulte la documentación siguiente:

    • To Stop a Domain (or Server) de Sun GlassFish Enterprise Server v3 Prelude Administration Guide

    • To Start a Domain (or Server) de Sun GlassFish Enterprise Server v3 Prelude Administration Guide
