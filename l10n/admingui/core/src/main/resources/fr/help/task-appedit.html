Modification d'une application déployée

  1. Dans l'arborescence de navigation, sélectionnez le nœud Applications.

    La page Applications s'affiche.

  2. Dans le tableau Applications déployées, cliquez sur le nom de l'application à modifier.

    La page Modification de l'application s'affiche.

  3. Pour une application Web, dans le champ Racine contexte, saisissez le chemin de l'application.

    Dans l'URL de l'application Web, la racine contexte suit immédiatement le numéro de port (http://host: port/context-root/...). Assurez-vous que la racine contexte commence par une barre oblique, comme par exemple /hello.

  4. Dans le champ Description, entrez une brève description de cette application.
  5. Décochez la case État activé pour désactiver l'état de l'application afin de la rendre indisponible.

    Par défaut, l'état d'une application est Activé dès son déploiement.

  6. Cochez la case Java Web Start Enabled (Java Web Start activé) pour utiliser le logiciel Java Web Start afin de déployer l'application.
  7. Cliquez sur Enregistrer.
