安裝附加的元件

安裝附加的元件包括下列作業：

  • 從網路儲存庫傳送元件的內容及元件的相依性

  • 在系統上安裝 Enterprise Server 安裝影像下的元件

如果元件需要任何其他尚未安裝的元件，也會一併安裝所需的元件。

  1. 在瀏覽樹狀結構中，選取 [更新工具] 節點。

    已安裝的元件頁面會隨即顯示。

  2. 在 [已安裝的元件] 頁面上，按一下 [可用的附加元件] 標籤。

    可用的附加元件頁面會隨即顯示。

  3. 在可用的附加元件表格中，選取您要安裝的元件。

    如果沒有可用的附加元件，表格會是空的。

    • 若要選取個別元件，請選取元件名稱旁的核取方塊。
    • 若要選取所有元件，請按一下表格標頭中的 [全選] 圖示。
    • 若要取消選取所有元件，請按一下表格標頭中的 [全不選] 圖示。
  4. 按一下 [安裝]。

    會出現對話方塊請您確認所選應安裝的元件及其所需的元件。

  5. 在對話方塊中，按一下 [確定]。

    系統會詢問您是否接受正在安裝之元件的授權合約。

  6. 接受正在安裝之元件的授權合約。

    Enterprise Server 會確認安裝已完成。再從附加元件表格中移除元件，並在 [已安裝的元件] 頁面上，將這些元件增加至安裝的元件表格中。

  7. 重新啟動網域管理伺服器。

    如需詳細說明，請參閱下列文件：

    • 「Sun GlassFish Enterprise Server v3 Prelude Administration Guide」中的「To Stop a Domain (or Server)」

    • 「Sun GlassFish Enterprise Server v3 Prelude Administration Guide」中的「To Start a Domain (or Server)」
