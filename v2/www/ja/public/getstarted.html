GlassFish は、Java EE 5 プラットフォームの最新機能を実装した、フリーのオープンソースで開発されたアプリケーションサーバーです。 Java EE 5 プラットフォームでは、JavaServer Pages(JSP) 2.1、JavaServer Faces(JSF) 1.2、 Servlet 2.4、 Enterprise JavaBeans 3.0、 Java API for Web Services(JAX-WS) 2.0、 Java Architecture for XML Binding(JAXB) 2.0、 Web Services Metadata for the Java Platform 1.0 などの最新のテクノロジーを提供しています。  

GlassFish についてあまりご存じでない場合は、実際に GlassFish をダウンロードしてインストールし、スタートガイドに沿っていろいろと試したり、スクリーンキャストを見てみたり、チュートリアルにあるいくつかのサンプルを実際に使ったりしながら学んでみてください。 下記にリンクされているスタートガイドのページも見てみてください。

ダウンロードし、動かしてみる

GlassFish アプリケーションサーバーのアーリーアクセス版をダウンロードできます。 ダウンロードしたファイルには、GlassFish Java EE 5 application server の バイナリが含まれていますので、 アーカイブファイルを unzip することで、バイナリを使うことができます。 ダウンロードサイトには、主なオペレーティングシステム用のダウンロードファイルが置かれています。

ダウンロードサイトはこちら です。
