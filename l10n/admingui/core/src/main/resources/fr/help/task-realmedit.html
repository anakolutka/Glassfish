Modifier un domaine

  1. Dans l'arborescence de navigation, développez le nœud Configuration.
  2. Si la configuration du domaine prend en charge des clusters, sélectionnez la configuration à modifier.

    Exemples : domaines créés avec le profil Cluster ou entreprise .

  3. Développez le nœud Sécurité.
  4. Développez le nœud Domaines.
  5. Sélectionnez le nom d'un domaine existant.

    La page Modification du domaine s'ouvre.

  6. Modifiez les propriétés spécifiques de cette classe, si nécessaire.
  7. Ajoutez toute autre propriété de votre choix au domaine.
    1. Cliquez sur Ajouter une propriété.
    2. Dans le champ Nom, saisissez le nom de la propriété.

      • Pour une description des propriétés de domaine de fichier, consultez Propriétés spécifiques de la classe FileRealm.

      • Pour une description des propriétés de domaine de certificat, consultez Propriétés spécifiques de la classe CertificateRealm.

    3. Dans le champ Valeur, saisissez la valeur de la propriété.
  8. Cliquez sur Enregistrer pour sauvegarder les modifications.
