Paramètres de l'analyseur de perfomarnces JVM

Un analyseur de performances génère des données servant à analyser les performances et à identifier les éventuels goulots d'étranglement.

Définissez les paramètres de cet outil. Les informations saisies dépendent de l'analyseur de performances utilisé. Vous trouverez des exemples et des instructions au chapitre 3, Debugging Applications, du manuel Oracle GlassFish Server Application Development Guide.
