Módulos de auditoría

La auditoría es la manera en que Servidor empresarial registra sucesos significativos, como errores o violaciones de seguridad. Servidor empresarial registra todos los sucesos de autenticación en archivos de registro. Un registro de acceso completo ofrece un rastro secuencial de los sucesos de acceso de Servidor empresarial.

La página Módulos de auditoría muestra una lista de los módulos de auditoría disponibles. En cada módulo de auditoría se proporciona la siguiente información.

Nombre

El nombre del módulo de auditoría.

Nombre de clase

El nombre completo de la clase que implementa este módulo. El nombre de clase para el módulo de auditoría predeterminado es com.sun.enterprise.security.Audit.
