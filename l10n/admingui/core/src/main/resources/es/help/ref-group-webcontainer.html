Contenedor web

Contenedor web que aloja aplicaciones web. Proporciona el entorno en el que se ejecutan los servlets y los archivos de JavaServer PagesTM (JSPTM). No hay ningún valor de configuración a nivel del contenedor para el contenedor web en la Consola de administración.

Para configurar el contenedor web, utilice las siguientes páginas:

  • Propiedades generales

  • Propiedades de sesión

  • Propiedades de administrador

  • Propiedades de almacén
