Propriétés spécifiques de la classe FileRealm

Les propriétés suivantes sont requises pour un domaine fichier :

Contexte JAAS

Type de module de connexion à utiliser pour ce domaine. La seule valeur valide est fileRealm.

Fichier clé

Chemin d'accès complet et nom du fichier où le serveur va stocker toutes les informations relatives à l'utilisateur, au groupe et au mot de passe du domaine. La valeur par défaut est rép-domaine/config/keyfile pour le domaine fichier et rép-domaine/config/admin-keyfile pour le domaine administrateur .

Au départ, le fichier clé du domaine fichier est vide, donc les utilisateurs doivent être ajoutés sur la page Utilisateurs de fichiers... avant d'utiliser le domaine fichier.

Au départ, le fichier clé du domaine administrateur contient le nom utilisateur de l'administrateur, le mot de passe associé dans un format chiffré et le groupe auquel appartient l'utilisateur (par défaut, asadmin).

Remarque – Les utilisateurs du groupe asadmin dans le domaine administrateur sont autorisés à utiliser la Console d’administration et les outils asadmin. N'ajoutez à ce groupe que des utilisateurs ayant des privilèges d'administrateur.
