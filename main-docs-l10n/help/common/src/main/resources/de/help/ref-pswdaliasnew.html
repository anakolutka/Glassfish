Neuer Kennwortalias

Verwenden Sie die Seite "Neuer Kennwortalias", um einen neuen Kennwortalias zu erstellen, der in einer Kennwortdatei oder in der Domainkonfigurationsdatei verwendet wird.

Die Seite "Neuer Kennwortalias" enthält die folgenden Optionen:

Aliasname

Der Name des Alias. Dieser Name muss für alle Kennwortaliasnamen in der Domain eindeutig sein.

Kennwort

Das Kennwort, das den Kennwortalias in verschlüsselter Form speichert.

Kennwort bestätigen

Das Kennwort, das den Kennwortalias in verschlüsselter Form speichert. Die Werte "Kennwort" und "Kennwort bestätigen" werden verglichen, um sicherzustellen, dass das Kennwort fehlerfrei eingegeben wurde.
