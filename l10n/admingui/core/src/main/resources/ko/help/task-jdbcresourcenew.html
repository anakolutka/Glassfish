JDBC 자원을 만드는 방법

JDBC(Java DataBase Connectivity) 자원(데이터 소스)은 응용 프로그램을 데이터베이스에 연결하는 방법을 제공합니다

시작하기 전에

JDBC 자원을 만들려면 먼저 JDBC 연결 풀을 만들어야 합니다. JDBC 연결 풀을 만드는 방법을 참조하십시오.

  1. 탐색 트리에서 자원 노드를 확장합니다.
  2. JDBC 자원 노드를 선택합니다.
  3. JDBC 자원 페이지에서 새로 만들기를 누릅니다.

    새 JDBC 자원 페이지가 열립니다.

  4. J.N.D.I 이름 필드에 고유한 자원 이름을 입력합니다.

    J.N.D.I 이름은 카드 카탈로그가 도서관 내 책의 위치를 조직화하여 나타내는 것과 유사한 방식으로 분산 컴퓨팅 환경 내에서 구성 요소를 조직화하고 구성 요소의 위치를 찾아냅니다. 이에 따라 J.N.D.I 이름은 JDBC 자원에 액세스하는 중요한 방법이 됩니다. 일반적으로 이름은 jdbc/ 문자열로 시작합니다. 예를 들어 jdbc/payrolldb입니다. 슬래시를 반드시 입력합니다.

    이름에는 영숫자, 밑줄, 대시 또는 점 문자만 사용해야 합니다.

  5. 풀 이름 드롭다운 목록에서 새 JDBC 자원과 연결할 연결 풀을 선택합니다.
  6. 설명 필드에 새 JDBC 자원에 대한 간단한 설명을 입력합니다.
  7. 자원을 사용할 수 없도록 비활성화하려면 사용 가능 확인란의 선택을 취소합니다.

    기본적으로 자원이 만들어졌을 때 사용(활성화)할 수 있습니다.

  8. 확인을 누릅니다.
