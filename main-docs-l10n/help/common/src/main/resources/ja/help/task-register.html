GlassFish Serverの「登録」ページを表示するには

GlassFish Serverの「登録」ページを使用すると、GlassFish Serverのインストールを登録できます。

GlassFish Serverの登録ユーザーは次のような利点を得ることができます。

  • 新しいバージョン、パッチおよび更新に関する通知

  • 開発者向けの製品、サービスおよびトレーニングに関する特別なオファー

  • ホワイト・ペーパー、指導教材、およびGlassFish製品チームによるライブWebセミナーへのアクセス

  1. ナビゲーション・ツリーで、「共通タスク」ノードを選択します。

    「共通タスク」ページが開きます。

  2. 「GlassFishニュース」の下で、「登録」のリンクをクリックします。

    「GlassFish登録」Webページが、新しいブラウザ・タブまたはウィンドウで開きます。
