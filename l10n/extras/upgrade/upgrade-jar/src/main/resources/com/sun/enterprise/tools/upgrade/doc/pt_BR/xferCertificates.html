Tela Transfer Security Certificates

Use a tela Transfer Security Certificates para especificar os domínios que contêm um conjunto (ou vários conjuntos) de certificados de segurança que precisam ser migrados para o Sun Java Systems Application Server 9.1:

  1. Pressione o botão Add para exibir a caixa de diálogo Add Domain na qual o nome do domínio e as senhas dos arquivos de certificado são introduzidos.

    O painel Domains with Certificates to be Transferred exibe os domínios que foram adicionados.

  2. Repita esse processo até que todos os domínios que contêm os certificados que precisam ser transferidos tenham sido adicionados.
  3. Para excluir um domínio adicionado, selecione o domínio a ser excluído e pressione o botão Delete.
  4. Ao terminar, pressione o botão Next.

    Será exibida a tela Upgrade Results mostrando o status da atualização.
