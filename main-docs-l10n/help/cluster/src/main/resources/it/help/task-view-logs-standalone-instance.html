Per visualizzare i file di log per un'istanza standalone di GlassFish Server

Prima di iniziare

È necessario che l'istanza per la quale si visualizzano i file di log sia in esecuzione.

  1. Selezionare il nodo Istanze standalone nell'albero di navigazione.

    Viene visualizzata la pagina Istanze server standalone.

  2. Selezionare l'istanza per la quale visualizzare i file di log.

    Viene visualizzata la pagina Informazioni generali per l'istanza.

  3. Nella pagina Informazioni generali, fare clic su Visualizza file di log.

    Viene visualizzata la finestra Visualizzatore log.

    Nota:

    L'istanza deve essere già in esecuzione. Se l'istanza non è in esecuzione, il pulsante Visualizza file di log è disattivato.
