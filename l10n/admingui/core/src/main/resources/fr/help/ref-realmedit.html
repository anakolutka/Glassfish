Modification du domaine

Pour modifier le domaine, utilisez la page Modification du domaine. La page Modification du domaine contient les options suivantes :

Nom

Champ en lecture seule indiquant le nom du domaine que vous éditez

Nom de classe

Nom de classe du domaine que vous éditez.

Autres propriétés

Autres propriétés du domaine.

  • Pour ajouter une propriété, cliquez sur le bouton Ajouter propriété. Sur la ligne vide qui apparaît, saisissez le nom de la propriété dans le champ Nom et la valeur de la propriété dans le champ Valeur. Cliquez sur Enregistrer.

  • Pour modifier une propriété, modifiez le champ Valeur de cette propriété. Cliquez sur Enregistrer.

  • Pour supprimer une propriété, cochez la case située à gauche du champ Nom de la propriété que vous supprimez, puis cliquez sur le bouton Supprimer propriétés. Cliquez sur Enregistrer.

  • Pour une description des propriétés de domaine de fichier, consultez Propriétés spécifiques de la classe FileRealm.

  • Pour une description des propriétés de domaine de certificat, consultez Propriétés spécifiques de la classe CertificateRealm.
