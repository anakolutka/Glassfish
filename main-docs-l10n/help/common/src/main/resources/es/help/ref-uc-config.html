Configuración de Herramienta de Actualización

La página Configuración ofrece información de configuración sobre Herramienta de Actualización y permite cambiar el modo de conexión de Herramienta de Actualización a Internet. Para cambiar otros aspectos de la configuración de Herramienta de Actualización, utilice la aplicación independiente updatetool, que se encuentra en as-install-parent/bin.

La página Configuración de Herramienta de Actualización contiene las siguientes opciones.

Ruta de instalación

Ruta de acceso del sistema de archivos del directorio raíz o nivel superior de la imagen de instalación.

Repositorio preferido

Nombre del repositorio de red preferido asociado a esta imagen de instalación, por ejemplo, glassfish.org.

Conexión a Internet

Cómo conectar GlassFish Server a Internet:

Conexión directa

Esta opción especifica que GlassFish Server se conecta directamente a Internet.

Mediante un servidor proxy

Esta opción especifica que GlassFish Server se conecta a Internet mediante un servidor proxy. Cuando esta opción está seleccionada, se incluirán en esta página el campo Host del proxy y el campo Puerto del proxy.

Host del proxy

Nombre de host del servidor proxy del equipo en el que se ejecuta GlassFish Server.

Este campo sólo es aplicable si el equipo en el que se está ejecutando GlassFish Server se conecta a Internet a través de un servidor proxy.

Puerto del proxy

Número de puerto del servidor proxy del equipo en el que se ejecuta GlassFish Server.

Este campo sólo es aplicable si el equipo en el que se está ejecutando GlassFish Server se conecta a Internet a través de un servidor proxy.
