编辑应用程序

使用“编辑应用程序”页面可以修改现有的应用程序。

“编辑应用程序”页面包含以下选项：

名称

显示所编辑的应用程序名称的只读字段。

虚拟服务器

与此应用程序相关联的虚拟服务器。

上下文根目录

对于 Web 应用程序，指定应用程序路径。在 Web 应用程序的 URL 中，上下文根目录要紧跟着端口号 (http://host:port /context-root/...)。上下文根目录必须以正斜杠开头，例如：/hello。

说明

应用程序的简要说明。

状态

如果选中“已启用”复选框，则将启用应用程序。默认情况下，此选项已选中。

Java Web Start

如果选中“已启用”复选框，则应用程序将使用 Java Web Start 软件。Java Web Start 提供了一种与浏览器无关的部署方式，可将 Java 应用程序部署为在专用 Java 虚拟机中运行。

部署目录

如果选中此复选框，则应用程序将从目录进行部署。

位置

显示已部署应用程序位置的只读字段。

库

显示应用程序所需的 JAR 库文件的只读字段。

子模块

显示应用程序子模块及其引擎名称的表。
