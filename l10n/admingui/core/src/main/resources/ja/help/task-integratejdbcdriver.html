JDBC ドライバを統合する

JDBC API をサポートするドライバ (JDBC ドライバ) は、アプリケーションの JDBC 呼び出しを、データベースサーバーのプロトコルに変換します。

  • JDBC ドライバを管理ドメインに統合するには、次のどれかを実行します。
    • 共通クラスローダーにアクセスできるドライバを作成する
      1. ドライバの JAR および ZIP ファイルを domain-dir/lib ディレクトリにコピーするか、またはクラスファイルを domain-dir/lib/ext ディレクトリにコピーします。
      2. ドメインを再起動します。
    • システムクラスローダーにアクセスできるドライバを作成する
      1. ナビゲーションツリーで、「Application Server」ノードを選択します。
      2. 「一般情報」ページで、「JVM 設定」タブを選択します。
      3. 「JVM 一般設定」ページで、「パス設定」タブを選択します。
      4. 「クラスパスのサフィックス」フィールドで、ドライバの JAR ファイルの完全修飾パス名を入力します。
      5. 「保存」をクリックします。
      6. サーバーを再起動します。
