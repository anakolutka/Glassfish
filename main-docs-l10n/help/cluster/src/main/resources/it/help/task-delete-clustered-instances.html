Per eliminare le istanze di GlassFish Server in cluster

Attenzione:

Se si utilizza un cluster JMS (Java Message Service) con un broker principale, non eliminare l'istanza associata al broker principale. Se è necessario eliminare l'istanza, utilizzare il comando secondario change-master-broker(1) per assegnare il broker principale a un'altra istanza.

  1. Espandere il nodo Cluster nell'albero di navigazione.

  2. Selezionare il nodo per il cluster sotto il nodo Cluster espanso.

    Viene visualizzata la pagina Informazioni generali per il cluster.

  3. Nella pagina Informazioni generali, fare clic sulla scheda Istanze.

    Viene visualizzata la pagina Istanze server in cluster per il cluster.

  4. Per ogni istanza da eliminare, selezionare la casella di controllo in corrispondenza dell'istanza nella tabella Istanze server.

  5. Fare clic su Elimina.

    Viene richiesto di confermare l'eliminazione delle istanze.

  6. Confermare che si desidera eliminare le istanze.

    Le istanze vengono eliminate e rimosse dalla tabella di istanze della pagina.
