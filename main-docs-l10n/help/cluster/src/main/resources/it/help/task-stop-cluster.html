Per arrestare un cluster

Quando si arresta un cluster, vengono arrestate tutte le istanze di GlassFish Server in esecuzione nel cluster.

  1. Selezionare il nodo Cluster nell'albero di navigazione.

    Viene visualizzata la pagina Cluster.

  2. Nella pagina Cluster selezionare la casella di controllo accanto al nome del cluster.

  3. Fare clic su Arresta cluster.

    Viene richiesto di confermare l'arresto del cluster.

  4. Confermare che si desidera arrestare il cluster.

    Il cluster viene arrestato. Lo stato di ogni istanza del cluster viene aggiornato nella pagina Cluster quando l'istanza viene arrestata.
