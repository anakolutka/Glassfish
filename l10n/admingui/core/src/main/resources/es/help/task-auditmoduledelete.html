Cómo eliminar un módulo de auditoría

  1. Expanda el nodo Configuración.
  2. Seleccione la instancia que va a configurar:
    • Para configurar una instancia específica, expanda el nodo de configuración de dicha instancia. Por ejemplo, en la instancia predeterminada server expanda el nodo server-config.
    • Para configurar los valores de configuración predeterminados de todas las instancias, expanda el nodo default-config.
  3. Expanda el nodo Seguridad.
  4. Seleccione el nodo Módulos de auditoría.
  5. Haga clic en la casilla a la izquierda del módulo de auditoría que quiere eliminar.
  6. Haga clic en Delete.
