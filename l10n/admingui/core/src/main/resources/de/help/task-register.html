So registrieren Sie Sun GlassFish Enterprise Server für Sun Connection

Um Ihre Installation von Sun GlassFish Enterprise Server für Sun Connection zu registrieren, benötigen Sie ein Sun-Onlinekonto. Wenn Sie nicht über ein Sun-Onlinekonto verfügen, ermöglicht Enterprise Server die Erstellung eines Kontos bei der Registrierung.

Hinweis – Wenn Sie ein Sun-Onlinekonto bei der Registrierung von Enterprise Server erstellen, werden sie lediglich zur Angabe der erforderlichen Informationen aufgefordert. Nach dem Registrieren von Enterprise Server können Sie zusätzliche, optionale Informationen angeben, indem Sie Ihr Sun-Onlinekonto aktualisieren.

Bevor Sie beginnen

Wenn der Computer, auf dem Enterprise Server ausgeführt wird, über einen Proxy eine Verbindung zum Internet herstellt, benötigen Sie den Hostnamen und die Port-Nummer des Proxy. Wenn ein Webbrowser auf dem Computer ausgeführt wird, auf dem Enterprise Server ausgeführt wird, können Sie diese Informationen vom Webbrowser beziehen. Die benötigten Informationen erhalten Sie ansonsten von Ihrem Systemadministrator.

  1. Wählen Sie in der Navigationsstruktur den Registrierungsknoten.

    Die Seite "Produktregistrierung" wird geöffnet.

  2. Geben Sie auf der Seite zur Produktregistrierung die erforderlichen Informationen an.
    • Wenn Sie bereits über ein Sun-Onlinekonto verfügen, geben Sie die erforderlichen Informationen wie folgt an:
      1. Wählen Sie die Option "Ich habe bereits ein Sun-Onlinekonto".
      2. Geben Sie im Feld "Benutzername" den Benutzernamen ein, den Sie bei der Registrierung für das Sun-Onlinekonto angegeben haben.
      3. Geben Sie im Feld "Passwort" das Passwort ein, das Sie bei der Registrierung für das Sun-Onlinekonto angegeben haben.
    • Wenn Sie nicht über ein Sun-Onlinekonto verfügen, geben Sie die erforderlichen Informationen wie folgt an:
      1. Wählen Sie die Option "Ich habe kein Sun-Onlinekonto".
      2. Geben Sie im Feld für die E-Mail-Adresse Ihre E-Mail-Adresse ein.

        Ihre E-Mail-Adresse wird der Benutzername Ihres Sun-Onlinekontos sein.

      3. Geben Sie im Feld "Passwort" das Passwort für Ihr Sun-Onlinekonto ein.

        Für das Passwort wird die Groß- und Kleinschreibung beachtet, und es muss 6–10 Zeichen umfassen.

      4. Geben Sie im Feld zur Passwortbestätigung erneut das Passwort ein, das Sie im vorherigen Schritt eingegeben haben.
      5. Geben Sie im Feld "Vorname" Ihren Vornamen ein.
      6. Geben Sie im Feld "Nachname" Ihren Nachnamen ein.
      7. Wählen Sie in der Dropdown-Liste das Land aus, in dem Sie leben.
  3. Wenn der Computer, auf dem Enterprise Server ausgeführt wird, über einen Proxy eine Verbindung zum Internet herstellt, geben Sie den Hostnamen und die Port-Nummer des Proxy an.

    Wenn der Computer, auf dem Enterprise Server ausgeführt wird, eine direkte Verbindung zum Internet herstellt, lassen Sie diesen Schritt aus.

    Ein Webbrowser wird möglicherweise auf dem Computer ausgeführt, auf dem Enterprise Server ausgeführt wird. In diesem Fall können Sie den Hostnamen und die Port-Nummer des Proxy über den Webbrowser beziehen. Die benötigten Informationen erhalten Sie ansonsten von Ihrem Systemadministrator.

  4. Klicken Sie auf "Registrieren".

    Zur Bestätigung der erfolgreichen Registrierung wird die Supportseite geöffnet.
