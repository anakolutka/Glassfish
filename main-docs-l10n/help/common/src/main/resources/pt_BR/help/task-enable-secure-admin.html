Para Ativar a Administração Segura

  1. Na árvore de navegação, selecione o nó do servidor (Servidor Admin).

    A página Informações Gerais será aberta.

  2. Na página Informações Gerais, clique no botão Administração Segura.

    A página Administração Segura será aberta.

  3. Se a instalação do GlassFish Server usar valores personalizados de alias da administração ou da instância, atualize os valores default mostrados nos campos Alias de Administração e Alias da Instância, conforme necessário.

  4. Clique em Ativar Admin. Segura.

    Uma mensagem será exibida, indicando que a ativação da administração segura reiniciará o DAS imediatamente.

  5. Clique em OK.

    A administração segura é ativada e a página de reinicialização do servidor será aberta.
