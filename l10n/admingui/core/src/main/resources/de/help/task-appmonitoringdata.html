So zeigen Sie Überwachungsdaten der Anwendung an

Bevor Sie beginnen

Bevor Sie Überwachungsdaten anzeigen können, müssen Sie die Überwachung konfigurieren. Einzelheiten finden Sie unter So konfigurieren Sie den Überwachungsdienst.

  1. Wählen Sie im Navigationsbaum den Knoten Enterprise Server aus.

    Die Seite "Allgemeine Informationen" wird geöffnet.

  2. Klicken Sie auf der Seite "Allgemeine Informationen" auf die Registerkarte "Überwachen".

    Die Seite "Anwendungsüberwachung" wird geöffnet.

  3. Wählen Sie aus der Dropdown-Liste eine Anwendung aus, um die Überwachungsdaten für diese Anwendung anzuzeigen.
  4. Klicken Sie auf die Schaltfläche "Statistiken abrufen", um die Überwachungsstatistiken abzurufen.

    Überwachungsstatistiken werden in der Tabelle angezeigt.

  5. (Optional) Klicken Sie auf die Schaltfläche "Aktualisieren", um die angezeigten Überwachungsdaten zu aktualisieren.
