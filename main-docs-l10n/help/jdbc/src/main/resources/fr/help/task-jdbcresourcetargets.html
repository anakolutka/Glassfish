Modification des cibles de ressource JDBC

  1. Dans l'arborescence de navigation, développez le noeud Ressources.

  2. Développez le noeud JDBC.

  3. Développez le noeud Ressources JDBC.

  4. Sélectionnez la ressource JDBC à modifier.

  5. Sur la page Modifier une ressource JDBC, sélectionnez l'onglet Cibles.

  6. Pour affecter une nouvelle cible à la ressource, cliquez sur le bouton Gérer les cibles.

  7. Sur la page Gérer les cibles, sélectionnez une cible disponible pour l'ajouter à la liste Cibles sélectionnées ou l'en enlever.

  8. Cliquez sur OK pour appliquer vos modifications.
