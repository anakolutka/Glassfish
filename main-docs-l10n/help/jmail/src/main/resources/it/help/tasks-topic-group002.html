Per visualizzare le esecuzioni di processi batch

  1. Nella struttura di navigazione, selezionare il cluster o l'istanza server di GlassFish Server per cui si desidera visualizzare le esecuzioni di processi batch.

    Viene visualizzata la pagina Informazioni generali.

  2. Selezionare la scheda Batch.

    Viene visualizzata la pagina Esecuzioni processi batch.

  3. Nella pagina Esecuzioni processi batch fare clic su un ID esecuzione per visualizzare i dettagli di quella esecuzione specifica.

    Viene visualizzata la pagina Dettagli esecuzione processo batch.

  4. Per visualizzare i dettagli sui passi inclusi nell'esecuzione del processo batch, fare clic sulla scheda Passi di esecuzione.

    Viene visualizzata la pagina Passi esecuzione processo batch.

  5. Fare clic sul pulsante Indietro per tornare alla pagina Esecuzioni processi batch.
