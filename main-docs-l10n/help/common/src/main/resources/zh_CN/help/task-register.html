查看 GlassFish Server 的“注册”页

使用 GlassFish Server 的“注册”页可以注册您的 GlassFish Server 安装。

作为 GlassFish Server 的注册用户，您将获得以下好处：

  • 收到新版本、补丁程序和更新的通知

  • 开发者产品、服务和培训的特别优惠

  • 访问白皮书、指导文章以及与 GlassFish 产品团队召开实时网络研讨会

  1. 在导航树中，选择“常见任务”节点。

    此时将打开“常见任务”页。

  2. 在“GlassFish 新闻”下，单击“注册”链接。

    此时将在单独的浏览器选项卡或窗口中打开“GlassFish 注册”网页。
