This technology preview allows you to experiment with a very early version of the GlassFish V3 app server.

Note: This software is feature-incomplete, has undergone no formal testing, and is not suitable for production deployments.

Even so, you might find it interesting to experiment with this code to experience the highly modular and lightweight approach we are taking in V3.

Click on the “technology preview bundle” link below to download.  Some browsers may require you to right-click/CTRL-click to save these files to your computer.

The majority of the GlassFish code is available under the Common Development and Distribution License (CDDL) v1.0  The following page contains details about the components in GlassFish and the licenses under which they are covered.

Instructions to unbundle the GlassFish V3 technology preview

The GlasssFish V3 technology preview installation requires JDK 5 or JDK 6. Make sure that the Java bin directory is in your path. 

  1. Download the technology preview bundle.

  2. cd to the directory into which you want to install the V3 technology preview.

  3. Run: 

    java -jar filename.jar

    where filename.jar is your just-downloaded bundle. Read the license agreement and accept it or decline it. This installs the V3 technology preview into a new glassfish directory under your current default directory.


To get started with the GlassFish V3 technology preview, please see the quick-start guide.
