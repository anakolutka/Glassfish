Nuevo usuario de dominio de archivo

Utilice la página Nuevo usuario de dominio de archivo para otorgar acceso al dominio actual a un usuario nuevo.

Nota:

Si la administración segura está activada, no se puede crear un usuario admin-realm con una contraseña en blanco.

La página Nuevo usuario de dominio de archivo contiene las siguientes opciones.

Nombre de Configuración

Nombre de la configuración a la que se aplican los valores de esta página. Este campo es de sólo lectura.

Nombre de dominio

Nombre del dominio actualmente seleccionado. Éste es un campo de sólo lectura.

ID de Usuario

ID de usuario para una cuenta de usuario nueva.

Lista de Grupos

Grupos a los que pertenecerá el usuario nuevo. Si introduce varios grupos, sepárelos con comas.

Para usuarios de admin-realm, éste es un campo de sólo lectura y el grupo es asadmin.

Contraseña Nueva

Contraseña del usuario nuevo.

Confirmar Contraseña Nueva

Contraseña del usuario nuevo.
