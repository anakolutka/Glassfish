Überwachungsmodule

Mit diesen Modulen wird überwacht, wie Enterprise Server wichtige Ereignisse, z. B. Fehler oder Sicherheitsverletzungen, aufzeichnet. Enterprise Server zeichnet sämtliche Authentifizierungsereignisse in Protokolldateien auf. Ein vollständiges Zugriffsprotokoll umfasst eine Reihe von Enterprise Server-Zugriffsereignissen.

Auf der Seite "Überwachungsmodule" wird eine Liste mit verfügbaren Überwachungsmodulen angezeigt. Folgende Informationen werden für jedes Überwachungsmodul zur Verfügung gestellt.

Name

Der Name des Überwachungsmoduls.

Klassenname

Der vollqualifizierte Name der Klasse, welche dieses Modul implementiert. Der Klassenname für das standardmäßige Überwachungsmodul lautet com.sun.enterprise.security.Audit.

Verwandte asadmin-Befehle

  • create-audit-module(1)

  • list-audit-modules(1)

  • delete-audit-module(1)
