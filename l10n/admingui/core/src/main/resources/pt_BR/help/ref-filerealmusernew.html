Novo Usuário do Domínio Arquivo

A página Novo Usuário do Domínio Arquivo concede acesso ao domínio atual para um novo usuário.

A página Novo Usuário do Domínio Arquivo contém estas opções.

Nome de Domínio

O nome do domínio selecionado atualmente. Este é um campo somente leitura.

ID do Usuário

A ID de usuário da nova conta de usuário.

Lista de Grupos

Os grupos aos quais o novo usuário pertence. Os nomes dos vários grupos são separados por vírgulas.

Para usuários do admin-domínio, esse é um campo somente leitura, e o grupo é asadmin.

Nova Senha

Senha do novo usuário.

Confirmar Nova Senha

Senha do novo usuário.
