網域屬性

使用「網域屬性」頁面可設定 GlassFish Server 的進階網域屬性。

「網域屬性」頁面包含下列選項：

在 DAS 啟動後載入主控台

決定何時載入管理主控台：

以使用方法為基礎

在網域管理伺服器 (DAS) 啟動之後載入 管理主控台 (如果在過去 24 小時內使用了主控台，或如果除了 DAS 以外還有其他伺服器執行處理的話)。此為預設值。

一律載入

在 DAS 啟動後一律載入管理主控台。主控台會在主要的 GlassFish Server 啟動序列完成後載入。

一律不載入

在 DAS 啟動後一律不載入管理主控台。請求存取主控台時才會載入主控台。

應用程式根

已建置應用程式所在的目錄完整路徑。預設值為 domain-dir/applications。

日誌檔根目錄

指定儲存伺服器日誌檔的位置。預設值為 domain-dir/logs。

語言環境

若此欄位保留為空白，則會使用主機的預設語言環境。

語言環境是指定特定的語言和區域組合的識別碼。例如，美式英文的語言環境為 en_US，日文的語言環境為 ja_JP。若要使用非英文的語言環境，必須對 GlassFish Server 進行本土化 ，此處理作業包括將英文翻譯為其他語言。
