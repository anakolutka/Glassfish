Afficher les données de contrôle JVM

  1. Dans l'arborescence de navigation, sélectionnez le nœud Application Server.
  2. Dans la page Informations générales, cliquez sur l'onglet Contrôle.
  3. Dans la page Contrôle du conteneur Web, cliquez sur l'onglet JVM.

    La page Contrôle JVM s'ouvre.

  4. (Facultatif) Cliquez sur le bouton Actualiser pour mettre à jour les données de contrôle affichées.
