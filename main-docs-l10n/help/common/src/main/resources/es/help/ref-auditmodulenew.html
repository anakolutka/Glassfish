Nuevo módulo de auditoría

Utilice la página Nuevo módulo de auditoría para definir un nuevo módulo de auditoría.

La página Nuevo módulo de auditoría contiene las siguientes opciones.

Nombre de Configuración

Nombre de la configuración a la que se aplican los valores de esta página. Este campo es de sólo lectura.

Nombre

Nombre del módulo de auditoría nuevo.

Nombre de clase

Nombre completo de la clase que implanta este módulo. El nombre de clase para el módulo de auditoría por defecto es com.sun.enterprise.security.Audit.

Propiedades Adicionales

Propiedades adicionales del módulo de auditoría. Para ver una descripción de las propiedades adicionales disponibles para los módulos de auditoría, consulte Propiedades específicas de módulos de auditoría.
