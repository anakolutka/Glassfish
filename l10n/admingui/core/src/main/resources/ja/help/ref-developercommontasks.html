共通操作

「共通操作」ページには、管理コンソールの一般的な操作のためのショートカットが用意されています。

「共通操作」ページを使用して次の操作を実行できます。

  • 「Sun Connection に Sun GlassFish Enterprise Server を登録する」

  • 「Sun GlassFish Enterprise Server の「GlassFish ニュース」ページを表示する」

  • 「Sun GlassFish Enterprise Server の「サブスクリプション」ページを表示する」

  • 「配備されているアプリケーションのリストを表示する」

  • 「Web アプリケーションを配備する」

  • 「エンタープライズアプリケーションを配備する」

  • 「アプリケーションクライアントを配備する」

  • 「コネクタモジュールを配備する」

  • 「EJB JAR を配備する」

  • 「コンポーネントをアンインストールする」

  • 「アドオンコンポーネントをインストールする」

  • 「コンポーネントの更新をインストールする」

  • To Create a JDBC Connection Pool

「共通操作」ページには、一部の Enterprise Server マニュアルへのリンクも表示されます。
