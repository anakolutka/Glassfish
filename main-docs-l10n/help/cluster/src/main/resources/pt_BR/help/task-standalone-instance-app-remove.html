Para Remover Aplicações de uma Instância do GlassFish Server Standalone

A remoção de uma aplicação remove a aplicação somente do alvo atual. Quando uma aplicação é removida de um alvo, a aplicação continua a ser listada na página Aplicações que exibe todas as aplicações implantadas no GlassFish Server. A aplicação também permanece implantada nos outros alvos nos quais já havia sido implantada.

  1. Na árvore de navegação, selecione o nó Instâncias Standalone.

    A página Instância de Servidor Standalone será aberta.

  2. Na página Instância de Servidor Standalone, selecione a instância.

    A página Informações Gerais da instância se abre.

  3. Na página Informações Gerais da instância, clique na guia Aplicações.

    A página Aplicações da instância se abre.

  4. Para cada aplicação que estiver removendo, marque a caixa de seleção ao lado da aplicação na tabela Aplicações Implantadas.

  5. Clique em Remover.

    Será solicitado que você remova as aplicações.

  6. Confirme se deseja remover as aplicações.

    As aplicações serão removidas da tabela de aplicações implantadas na página.
