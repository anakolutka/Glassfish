设置默认领域

默认领域是当应用程序的部署描述符未指定领域时 Enterprise Server 用于验证和授权的领域。

  1. 在导航树中，展开“配置”节点。
  2. 如果域配置为支持群集，请选择您要更改的配置。

    示例是使用群集配置文件或企业配置文件创建的域。

  3. 选择“安全性”节点。

    将打开“安全性”页面。

  4. 在“默认领域”字段中，从下拉式列表中选取所需的领域。
  5. 单击“保存”以保存更改。
  6. 如果控制台中显示“必须重新启动”，请重新启动服务器。
