JDBC

JDBCTM(JavaTM DataBase Connectivity) 기술은 관계형 데이터베이스에 대한 연결을 제공합니다. Enterprise Server에서 도메인에 JDBC 자원(데이터 소스)을 추가하여 데이터베이스 연결을 구성합니다. 데이터 소스는 데이터베이스에 대한 연결을 구성하는 Java 플랫폼, Enterprise Edition(Java EE 플랫폼)의 표준 메소드입니다. 각 데이터 소스에는 데이터베이스 연결의 풀이 포함되어 있습니다. 응용 프로그램은 J.N.D.I.(Java Naming and Directory InterfaceTM) API 트리나 로컬 응용 프로그램 컨텍스트에서 데이터 소스를 찾은 다음 getConnection 메소드를 사용하여 데이터베이스 연결을 검색합니다. 데이터 소스 및 연결 풀은 시스템을 실행 상태로 유지하는 데 도움을주는 연결 관리 프로세스를 제공합니다.

JDBC 자원을 구성하려면 다음 페이지를 사용합니다.

  • 새 JDBC 자원

  • JDBC 자원 편집

  • 연결 풀

  • 새 JDBC 연결 풀(단계 1/2)

  • 새 JDBC 연결 풀(단계 2/2)
