Anwendung bearbeiten

Auf der Seite Edit Application (Anwendung bearbeiten) können Sie eine vorhandene Anwendung ändern.

Die Seite Edit Application (Anwendung bearbeiten) umfasst die folgenden Optionen.

Name (Name)

Schreibgeschütztes Feld mit dem Namen der Anwendung, die Sie bearbeiten.

Virtual Servers (Virtuelle Server)

Die mit dieser Anwendung verbundenen virtuellen Server.

Context Root (Kontext-Root)

Gibt für eine Webanwendung den Pfad zu der Anwendung an. Im URL der Webanwendung folgt die Kontext-Root gleich hinter der Portnummer (http://host: port/context-root/...). Die Kontext-Root sollte mit einem Schrägstrich beginnen, wie z. B. /hallo.

Description (Beschreibung)

Eine kurze Beschreibung der Anwendung.

Status (Status)

Die Anwendung ist aktiviert, wenn das Kontrollkästchen Enabled (Aktiviert) aktiviert ist. Diese Option ist standardmäßig aktiviert.

Java Web Start

Wenn das Kontrollkästchen Enabled (Aktiviert) aktiviert ist, verwendet die Anwendung Java Web Start-Software. Java Web Start bietet eine browser-unabhängige Möglichkeit zum Bereitstellen von Java-Anwendungen für die Ausführung auf einer dedizierten Java Virtual Machine.

Directory Deployed (Aus Verzeichnis bereitgestellt)

Wenn dieses Kontrollkästchen aktiviert ist, wird die Anwendung aus einem Verzeichnis bereitgestellt.

Location (Ort)

Schreibgeschütztes Feld mit dem Pfad der bereitgestellten Anwendung.

Libraries (Bibliotheken)

Schreibgeschütztes Feld, das die JAR-Bibliotheksdateien anzeigt, die für die Anwendung erforderlich sind.

Sub Modules (Untergeordnete Module)

Tabelle mit den Namen der untergeordneten Module der Anwendung und ihren Engines.

Verwandte Aufgaben

  • So bearbeiten Sie eine bereitgestellte Anwendung

  • So stellen Sie eine Webanwendung bereit

  • So stellen Sie eine Unternehmensanwendung bereit

  • So stellen Sie einen Anwendungs-Client bereit

  • So stellen Sie ein Verbindungsmodul bereit

  • So stellen Sie ein EJB Jar bereit

  • So stellen Sie eine Ruby-Anwendung bereit

  • So zeigen Sie die Beschreibung einer Anwendung an

  • So starten Sie eine Anwendung

  • Eine bereitgestellte Anwendung erneut bereitstellen

  • Eine bereitgestellte Anwendung erneut starten
