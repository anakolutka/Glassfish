Suppression d'un service de contexte

  1. Dans l'arborescence de navigation, développez le noeud Ressources.

  2. Sous le noeud Ressources, développez le noeud Ressources simultanées.

  3. Sous le noeud Ressources simultanées, sélectionnez Services de contexte.

    La page Services de contexte est ouverte.

  4. Sur la page Services de contexte, cochez la case située en regard du nom de la ressource à supprimer.

  5. Cliquez sur Supprimer.
