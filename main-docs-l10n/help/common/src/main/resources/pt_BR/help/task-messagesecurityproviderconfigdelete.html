Para Deletar uma Configuração do Provedor de Segurança de Mensagem

Antes de Começar

É possível criar ou deletar os provedores de segurança de mensagem da configuração server-config default. Os únicos provedores de segurança de mensagem que são suportados no server-config default são SOAP e HttpServlet. Como estes dois provedores de mensagem já são criados por default quando o GlassFish Server é instalado, o botão Deletar não é exibido na página Configurações de Segurança de Mensagem no server-config default.

  1. Na árvore de navegação, expanda o nó Configuração.

  2. Expanda o nó Segurança.

  3. Selecione o nó Segurança de Mensagem.

    A página Configurações de Segurança de Mensagem será aberta.

  4. Na página Configurações de Segurança de Mensagem, selecione SOAP.

    A página Editar Configuração de Segurança de Mensagem será aberta.

  5. Na página Editar Configuração de Segurança de Mensagem, clique na guia Provedores.

    A página Editar Configuração do Provedor será aberta.

  6. Na página Editar Configuração do Provedor, marque a caixa de seleção à esquerda da configuração do provedor que você está deletando.

  7. Clique em Deletar.
