Managing JavaMail Session Targets

  1. Expand Resources node, and select the JavaMail Sessions.
  2. In the JavaMail Sessions screen, click the JavaMail Session resource whose targets you want to manage.
  3. In the Edit JavaMail Session screen, click the Target tab.
  4. To enable a target, click the check box next to the desired server instance or cluster and click Enable.
  5. To disable a target, click the check box next to the desired server instance or cluster and click Disable.
