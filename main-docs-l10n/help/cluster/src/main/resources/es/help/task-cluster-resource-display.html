Para mostrar los recursos para un cluster

  1. En el árbol de navegación, amplíe el nodo Clusters.

  2. En el nodo Clusters ampliado, seleccione el nodo para el cluster.

    Se abre la página Información General para el cluster.

  3. En la página Información General, haga clic en el separador Recursos.

    Se abrirá la página Recursos para el cluster.

  4. Para mostrar sólo los recursos de un determinado tipo, en la lista desplegable Filtro, seleccione el tipo de recurso que se va a mostrar.
