Sesiones de JavaMail

GlassFish Server incluye la API de JavaMail. JavaMail proporciona acceso desde aplicaciones Java a servidores de correo compatibles con Internet Message Access Protocol (IMAP) y Simple Mail Transfer Protocol (SMTP) situados en su red o en Internet.

Utilice la página Sesiones JavaMail para configurar sesiones JavaMail.

Para cada sesión JavaMail, se proporciona la siguiente información.

Nombre JNDI

Nombre único que identifica las sesiones JavaMail.

Activado

(Esta columna sólo se muestra si la instancia del servidor por defecto, server, existe en el dominio).

Una marca de control si el recurso está activado, o una X si está desactivado.

Estado

(Esta columna sólo se muestra si en el dominio se ha creado algún cluster o instancia independiente).

Número de clusters e instancias independientes a los que se dirige el recurso y el número de estos destinos en los que el recurso está activado. Por ejemplo, "Activado en 2 de 4 de destinos" significa que el recurso se dirige a cuatro clusters e instancias independientes y que está activado en dos de estos cuatro destinos.

Descripción

Descripción de las sesiones JavaMail.

La tabla Sesiones también contiene las siguientes opciones.

Nuevo

Botón para crear una nueva sesión JavaMail.

Suprimir

Botón para suprimir una o más sesiones JavaMail seleccionadas.

Activar

Botón para activar una o más sesiones JavaMail seleccionadas.

Desactivar

Botón para desactivar una o más sesiones JavaMail seleccionadas.
