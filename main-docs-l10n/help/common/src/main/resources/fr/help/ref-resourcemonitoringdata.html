Surveillance de ressource

La page Surveillance de ressource permet d'afficher les données de surveillance d'une ressource sélectionnée.

Elle contient les informations ci-après.

Nom d'instance

Nom de l'instance de serveur.

Pool de connexions

Pool de connexions pour lequel les statistiques de surveillance sont affichées.

Pour obtenir une description des propriétés surveillées, reportez-vous au chapitre 8, Administering the Monitoring Service, du manuel Oracle GlassFish Server Administration Guide.
