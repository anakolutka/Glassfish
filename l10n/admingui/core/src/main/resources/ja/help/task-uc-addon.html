アドオンコンポーネントをインストールする

アドオンコンポーネントのインストールには次の作業が含まれます。

  • ネットワークリポジトリからコンポーネントとコンポーネントの依存関係の内容を転送する

  • システム上の Enterprise Server のインストールイメージの下のコンポーネントをインストールする

コンポーネントにまだインストールされていないほかのコンポーネントが必要な場合は、そのコンポーネントもインストールされます。

  1. ナビゲーションツリーで、更新ツール ノードを選択します。

    「インストール済みコンポーネント」ページが表示されます。

  2. 「インストール済みコンポーネント」ページで、「利用可能なアドオン」タブをクリックします。

    「利用可能なアドオン」ページが表示されます。

  3. 利用可能なアドオンコンポーネントの表で、インストールするコンポーネントを選択します。

    利用可能なアドオンコンポーネントがない場合、表は空になります。

    • 個々のコンポーネントを選択するには、そのコンポーネントの名前の隣にあるチェックボックスを選択します。
    • すべてのコンポーネントを選択するには、表ヘッダーの「すべてを選択」アイコンをクリックします。
    • すべてのコンポーネントを選択解除するには、表ヘッダーの「すべてを選択解除」アイコンをクリックします。
  4. 「インストール」をクリックします。

    選択したコンポーネントとそれが必要とするコンポーネントのインストールが必要であることを確認するダイアログボックスが表示されます。

  5. ダイアログボックスの「了解 (OK)」をクリックします。

    インストールするコンポーネントのライセンス契約書への同意が求められます。

  6. インストールするコンポーネントのライセンス契約書に同意します。

    Enterprise Server はインストールが完了したことを確認します。アドオンコンポーネントの表からコンポーネントが削除され、「インストール済みコンポーネント」ページのインストール済みコンポーネントの表に追加されます。

  7. ドメイン管理サーバーを再起動します。

    詳細な手順については、次のマニュアルを参照してください。

    • 『Sun GlassFish Enterprise Server v3 Prelude Administration Guide』の「To Stop a Domain (or Server)」

    • 『Sun GlassFish Enterprise Server v3 Prelude Administration Guide』の「To Start a Domain (or Server)」
