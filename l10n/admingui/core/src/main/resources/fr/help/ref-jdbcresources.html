Ressources JDBC

Une ressource JDBC (Java DataBase Connectivity) (source de données) fournit des applications via une connexion à une base de données. En général, l'administrateur crée une ressource JDBC pour chaque base de données accédée par les applications déployées dans un domaine. Plusieurs ressources JDBC peuvent cependant être créées pour une même base de données.

Les applications obtiennent une connexion à la base de données à partir d'un pool de connexions en recherchant une source de données dans l'arborescence J.D.N.I. (Java Naming and Directory Interface ), puis demandent une connexion. Le pool de connexions associé à la source de données fournit la connexion à l'application.

La page Ressources JDBC présente la liste des ressources créées sur le serveur d'application. Pour chaque ressource, les informations suivantes sont fournies :

  • Nom JNDI

  • Statut

  • Pool de connexion

  • Description
