删除 HTTP 侦听器

开始之前

如果您计划删除默认的 HTTP 侦听器 http-listener-1、 http-listener-2 或 admin-listener，请首先创建 Enterprise Server 的 domain.xml 文件中 http-listener 元素的备份副本，以便在需要时可以还原这些设置。

  1. 在导航树中，展开“配置”节点。
  2. 在“配置”节点下，展开“HTTP 服务”节点。
  3. 在“HTTP 服务”节点下，选择“HTTP 侦听器”节点。
  4. 在“HTTP 服务”页面上，选中要删除的 HTTP 侦听器的名称旁边的复选框。
  5. 单击“删除”。
