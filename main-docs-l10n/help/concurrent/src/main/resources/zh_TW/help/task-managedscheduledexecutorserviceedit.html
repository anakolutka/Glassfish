編輯受管理排定的執行程式服務

  1. 在瀏覽樹狀結構中，展開「資源」節點。

  2. 在「資源」節點下，展開「並行資源」節點。

  3. 在「並行資源」節點下，選取「受管理排定的執行程式服務」。

    就會開啟「受管理排定的執行程式服務」頁面。

  4. 在「受管理排定的執行程式服務」頁面中，按一下要編輯的資源名稱。

    就會開啟「編輯受管理排定的執行程式服務」頁面。

    備註:

    「JNDI 名稱」是唯讀欄位。您僅能在建立新資源時指定 JNDI 名稱。

  5. XXX

  6. 在「描述」欄位中，鍵入資源的簡短描述。

    該描述是一個字串值，最多可以包含 250 個字元。

  7. 選取「啟用狀態」核取方塊，以啟用資源。

  8. 在「其他特性」區段中，指定資源的其他特性。

    GlassFish Server 未定義受管理排定的執行程式服務資源的任何其他特性。

    • 若要增加特性，請按一下「增加特性」按鈕。在出現的空白列中，於「名稱」欄位中鍵入特性名稱，然後在「值」欄位中鍵入特性值。

    • 若要修改特性，請編輯特性的「值」欄位。

    • 若要刪除特性，請選取您要刪除之特性「名稱」欄位左側的核取方塊，然後按一下「刪除特性」。

  9. 按一下「儲存」。
