新規管理対象スレッド・ファクトリ

管理対象スレッド・ファクトリ・リソースを作成するには、「新規管理対象スレッド・ファクトリ」ページを使用します。

「新規管理対象スレッド・ファクトリ」ページには、次のオプションが含まれます。

JNDI名

リソースを識別する一意の名前。

コンテキスト情報

リソースのコンテキスト: 「クラス・ローダー」、「JNDI」、「セキュリティ」、またはワークエリア

ステータス

リソースのステータス。リソースは、有効または無効の状態になります。デフォルト値は「有効」です。

スレッド優先度

管理対象スレッド・ファクトリ・インスタンスに作成するスレッドの優先度。デフォルトは5です。

説明

リソースの説明。

追加プロパティ

リソースの追加プロパティ。GlassFish Serverは管理対象スレッド・ファクトリ・リソースの追加プロパティを定義しません。
