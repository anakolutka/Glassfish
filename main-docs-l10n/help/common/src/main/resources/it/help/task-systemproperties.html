Per gestire le proprietà di sistema

  1. Selezionare il nodo GlassFish Server nell'albero di navigazione.

    Viene visualizzata la pagina Informazioni generali.

  2. Nella pagina Informazioni generali fare clic sulla scheda Proprietà di sistema.

    Viene visualizzata la pagina Proprietà di sistema.

  3. Nella sezione Proprietà aggiuntive specificare le proprietà di sistema per la configurazione.

    Alcune proprietà di sistema sono predefinite, ma non vengono visualizzate in questa pagina. Per i dettagli, vedere Proprietà di sistema predefinite.

    • Per aggiungere una proprietà, fare clic sul pulsante Aggiungi proprietà. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Nome e il valore della proprietà nel campo Valore.

    • Per modificare una proprietà, è necessario modificare il valore del campo Valore della proprietà.

    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Nome corrispondente alla proprietà da eliminare, quindi fare clic sul pulsante Elimina proprietà.

  4. Fare clic su Salva.
