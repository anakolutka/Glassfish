Mot de passe administrateur

Utilisez la page Mot de passe administrateur pour modifier le mot de passe du compte administrateur actuel.

La page Mot de passe administrateur contient les options suivantes.

ID utilisateur

ID utilisateur de l'administrateur actuel. Ce champ est en lecture seule.

Liste des groupes

Groupe auquel l'administrateur appartient. Ce champ est en lecture seule. La valeur est asadmin.

Nouveau mot de passe

Nouveau mot de passe pour l'administrateur.

Confirmer le mot de passe

Nouveau mot de passe pour l'administrateur.
