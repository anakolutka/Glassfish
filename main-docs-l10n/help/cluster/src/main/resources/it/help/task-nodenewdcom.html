Per completare la creazione di un nodo DCOM

Questo task è la continuazione di Per creare un nodo per un nodo DCOM.

  1. Per copiare il software di GlassFish Server dall'host del server DAS all'host del nodo, abilitare l'opzione Installa GlassFish Server.

  2. Per forzare la creazione del nodo nella configurazione del server di amministrazione del dominio (DAS) anche se la convalida dei parametri del nodo non riesce, abilitare l'opzione Imponi.

  3. Se si desidera che GlassFish Server esegua il test della connessione su DCOM all'host del nodo prima di creare il nodo, attenersi alla procedura riportata di seguito.

    1. Abilitare l'opzione Convalida DCOM.

    2. Nel campo Directory di test remota, digitare il nome della cartella dell'host remoto in cui GlassFish Server deve creare un file per eseguire il test della connessione.

  4. Nel campo Dominio Windows digitare il nome del dominio Windows che contiene l'utente specificato dall'opzione Nome utente Windows.

  5. Nel campo Nome utente Windows digitare il nome dell'utente nell'host del nodo che deve eseguire il processo per la connessione all'host tramite DCOM.

  6. Specificare la modalità di autenticazione dell'utente Windows per il login all'host del nodo.

    Nota:

    GlassFish Server supporta anche l'autenticazione della password tramite la voce AS_ADMIN_WINDOWSPASSWORD del password file. Il password file è specificato nell'opzione --passwordfile della utility asadmin(1).

    1. Per specificare l'autenticazione tramite una password memorizzata nella configurazione del server DAS, attenersi alla procedura riportata di seguito.

      Attenzione:

      La password viene memorizzata come testo in chiaro nella configurazione del server DAS. Per maggiore sicurezza, si consiglia di utilizzare un alias password.

      1. Nell'elenco a discesa Autenticazione utente Windows selezionare Password.

      2. Nel campo Password utente Windows digitare la password che verrà utilizzata dall'utente Windows per collegarsi all'host del nodo.

    2. Per specificare l'autenticazione tramite un alias password, attenersi alla procedura riportata di seguito.

      1. Nell'elenco a discesa Autenticazione utente Windows selezionare Alias password.

      2. Nell'elenco a discesa Alias password selezionare un alias esistente.

  7. Fare clic su Salva.
