JVM 구성

JVMTM(JavaTM 가상 머신)을 통해 Java 바이트 코드를 물리적 시스템에서 실행할 수 있습니다. JVM 설정을 조정하여 성능을 향상시키고 메모리 할당 오류를 방지할 수 있습니다.

JVM 구성에는 다음과 같은 작업이 포함됩니다.

  • JVM 일반 설정을 구성하는 방법

  • JVM 경로 설정을 구성하는 방법

  • JVM 옵션을 구성하는 방법

  • 보안 관리자를 비활성화하는 방법

  • JVM 프로필러 설정을 구성하는 방법

  • 고급 응용 프로그램 구성을 설정하는 방법

  • 고급 도메인 속성을 설정하는 방법

주 – "Java 가상 머신"과 "JVM"은 Java 플랫폼용 가상 머신을 의미합니다.
