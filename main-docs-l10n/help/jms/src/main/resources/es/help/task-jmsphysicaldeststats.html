Para ver las estadísticas de un destino físico JMS

  1. En el árbol de navegación, amplíe el nodo Configuración.

  2. En el nodo Configuración, amplíe el nodo Java Message Service.

  3. En el nodo Java Message Service, seleccione el nodo Destinos Físicos.

    Se abrirá la página Destinos Físicos.

  4. En la página Destinos Físicos, haga clic en el enlace Ver para el destino cuyas estadísticas desee ver.

    Se abrirá la página Ver estadísticas de destino físico JMS.

  5. Cuando haya terminado de ver las estadísticas, haga clic en el botón Atrás.
