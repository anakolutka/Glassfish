Gerenciar Alvos

Utilize a página Gerenciar Alvos do Recurso para exibir os clusters e as instâncias de servidores standalone do alvo nos quais um módulo do ciclo de vida pode ser ativado. O módulo do ciclo de vida pode ser ativado somente nos alvos da coluna Alvos Selecionados.

A página Gerenciar Alvos do Recurso contém as seguintes informações.

Alvos Disponíveis

Os clusters e as instâncias nos quais o módulo do ciclo de vida não pode ser ativado.

Alvos Selecionados

Os clusters e as instâncias nos quais o módulo do ciclo de vida pode ser ativado.

Adicionar

Botão para mover o alvo selecionado de Alvos Disponíveis para Alvos Selecionados.

Adicionar Tudo

Botão para mover todos os Alvos Disponíveis para os Alvos Selecionados.

Remover

Botão para mover o alvo selecionado de Alvos Selecionados para Alvos Disponíveis.

Remover Tudo

Botão para mover todos os Alvos Selecionados para Alvos Disponíveis.
