Activer ou désactiver la journalisation d'audit

  1. Dans l'arborescence de navigation, développez le nœud Configurations.
  2. Sélectionnez l'instance à configurer :
    • Pour configurer une instance particulière, développez le nœud de configuration de l'instance. Par exemple, pour l'instance par défaut, server, développez le nœud server-config.
    • Pour configurer les paramètres par défaut de toutes les instances, développez le noeud default-config.
  3. Sous le nœud Configuration, sélectionnez le nœud Sécurité.

    La page Sécurité s'ouvre.

  4. Cochez la case Journalisation d'audit activée.

    Pour désactiver la journalisation d'audit, désélectionnez la case.

    L'activation de cette option entraîne le chargement des modules d'audit et permet de s'assurer qu'ils sont appelés par la bibliothèque d'audit de Enterprise Server aux points d'audit.

  5. Si vous activez la journalisation d'audit, spécifiez un module d'audit par défaut tel qu'indiqué dans la rubrique Définir le module d'audit actif.
  6. Cliquez sur Enregistrer.
  7. Redémarrez le Enterprise Server si « Redémarrage requis » s'affiche dans la console.
