관리자 등록 정보

세션 관리자는 세션을 만들고 삭제하는 방법, 세션 상태 저장 위치 및 사용 가능한 최대 세션 수를 구성할 수 있는 방법을 제공합니다.

관리자 등록 정보 페이지를 사용하여 세션 관리자 등록 정보를 추가하거나 수정합니다. 관리자 등록 정보 페이지에는 다음 옵션이 포함되어 있습니다.

리프 간격

비활성 세션 데이터가 저장소에서 삭제되기까지의 시간(초)입니다.

최대 세션

허용되는 최대 세션 수입니다.

세션 파일 이름

세션 데이터가 포함된 파일 이름입니다.

세션 아이디 생성기 클래스 이름

Enterprise Server 클래스 경로에서 식별되는 아이디 생성기 클래스 이름입니다.

추가 등록 정보

세션 관리자의 추가 등록 정보입니다.

  • 등록 정보를 추가하려면 등록 정보 추가 버튼을 누릅니다. 빈 행이 표시되면 이름 필드에 등록 정보 이름을 입력하고 값 필드에 등록 정보 값을 입력합니다. 저장을 누릅니다.

  • 등록 정보를 수정하려면 등록 정보의 값 필드를 편집합니다. 저장을 누릅니다.

  • 등록 정보를 삭제하려면 삭제할 등록 정보의 이름 필드 왼쪽에 있는 확인란을 선택한 다음 등록 정보 삭제 버튼을 누릅니다. 저장을 누릅니다.

관련 작업

  • 세션 시간 초과 값을 설정하는 방법

  • 관리자 등록 정보를 구성하는 방법

  • 저장소 등록 정보를 구성하는 방법
