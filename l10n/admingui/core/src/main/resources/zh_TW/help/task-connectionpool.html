配置 HTTP 服務連線池

  1. 在瀏覽樹狀結構中，展開 [配置] 節點。
  2. 在 [配置] 節點下，選取 [HTTP 服務] 節點。
  3. 在 [HTTP 服務] 頁面上，按一下 [連線池] 標籤。

    [連線池] 頁面會隨即開啟。

  4. 在 [最大擱置計數] 欄位中，鍵入 HTTP 偵聽程式允許的最大擱置連線數。

    預設值為 4096。

  5. 在 [佇列大小] 欄位中，鍵入連線佇列的最大大小 (以位元組為單位)。

    該值還指定伺服器可以擁有的最大未執行連線數目。預設值為 4096。

  6. 在 [接收緩衝區大小] 欄位中，鍵入 HTTP 偵聽程式接收緩衝區的大小 (以位元組為單位)。

    預設值為 4096。

  7. 在 [傳送緩衝區大小] 欄位中，鍵入 HTTP 偵聽程式傳送緩衝區的大小 (以位元組為單位)。

    預設值為 8192。

  8. 按一下 [儲存]。

    提示 – 若要回到預設設定，請按一下 [載入預設值] 按鈕，然後按一下 [儲存]。
