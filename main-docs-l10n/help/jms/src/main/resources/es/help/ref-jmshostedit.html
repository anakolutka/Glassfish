Editar host JMS

Utilice la página Editar Host JMS para modificar la configuración de un host JMS.

La página Editar Host JMS contiene las siguientes opciones.

Cargar Valores por Defecto

Botón para restaurar configuración que tiene valores por defecto a sus valores por defecto. Las configuraciones que no tienen valores por defecto no se modifican.

Nombre de Configuración

Nombre de la configuración a la que se aplican los valores de esta página. Este campo es de sólo lectura.

Nombre

Nombre del host. El campo Nombre es un campo de sólo lectura. Sólo puede especificar el campo Nombre cuando crea un nuevo host JMS.

Host

Nombre o dirección IP (Internet Protocol) del sistema donde se ejecutará el host JMS (localhost o el nombre del sistema remoto o local).

Puerto

Número de puerto del servicio JMS. Cambie este campo sólo si el servicio JMS que va a utilizar se ejecuta en un puerto que no es el que se usa por defecto. El puerto por defecto es 7676.

Nombre de Usuario Administrador y Contraseña de Administrador

El nombre de usuario y la contraseña del broker de cola de mensajes. Éstos son diferentes del nombre de usuario y la contraseña de GlassFish Server. Edite estos campos solamente si los valores del broker de cola de mensajes se han modificado mediante el comando as-install-parent/mq/bin/imqusermgr. Los valores por defecto son admin y admin.
