新建環境服務

使用「新建環境服務」頁面即可建立環境服務資源。

「新建環境服務」頁面包含下列選項。

JNDI 名稱

可識別資源的唯一 JNDI 名稱。依慣例會是以資源類型指示符與斜線開頭的名稱，例如 concurrent/、jdbc/ 或 jms/。

環境資訊

要傳輸至繫線的環境：Classloader、JNDI、Security 或 WorkArea。您可以啟用或停用環境傳輸。如果啟用，就會傳輸選取的環境。如果停用，就不會傳輸任何環境 (包括選取的環境)。預設值為「啟用」(包括全部的四個環境)。

描述

此資源的描述。

狀態

資源狀態。您可啟用或停用資源。預設值為「啟用」。

其他特性

資源的其他特性。GlassFish Server 並未定義此資源類型的任何其他特性。此外，此資源類型目前也並未使用任何其他特性。

目標

(已在網域中建立叢集或獨立執行處理時才會顯示此選項。)

資源的叢集和獨立執行處理。可使用「加入」與「全部加入」按鈕，將所需的目標移至「選取的目標」欄。可使用「移除」與「全部移除」按鈕，將任何不需要的目標移至「可用的目標」欄。
