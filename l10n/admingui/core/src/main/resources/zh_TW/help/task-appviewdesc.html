檢視應用程式的描述元

  1. 在瀏覽樹狀結構中，選取 [應用程式] 節點。

    [應用程式] 頁面會隨即開啟。

  2. 在 [應用程式] 頁面上，按一下您要檢視其描述元的應用程式名稱。

    [編輯應用程式] 頁面會隨即開啟。

  3. 在 [編輯應用程式] 頁面上，按一下 [描述元] 標籤。

    [模組描述元] 頁面會隨即開啟。

  4. 在 [部署描述元] 表格中，按一下您要檢視的 XML 檔案名稱。

    該 XML 檔案會在新的視窗中開啟。
