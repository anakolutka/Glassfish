Manage JBI Targets

This page helps you manage the targets (clusters and standalone server instances) for JBI Service Assemblies, components and Shared Libraries.

  • For Service Assemblies, this page allows you to select targets for Service Assembly deployment and remove a Service Assembly from a deployed target.

  • For JBI components, this page allows you to select targets for JBI component installation and remove a JBI component from an installed target.

  • For Shared Libraries, this page allows you to select targets for Shared Library installation and remove a Shared Library from an installed target.
