JACC プロバイダ

「JACC プロバイダ」ページには、使用可能な Java Authorization Contract for Containers (JACC) プロバイダのリストが表示されます。

JACC は Java EE 仕様の一部で、プラグイン可能な承認プロバイダ用のインタフェースを定義しています。これによって、管理者は認証を行うためにサードパーティー製の「プラグイン」モジュールを設定できます。デフォルトで、Enterprise Server は JACC に準拠する単純なファイルベースの承認エンジンを提供します。

JACC プロバイダの実装クラスの記述方法については、http://jcp.org/en/jsr/detail?id=115 の JACC 仕様を参照してください。
