Nouvelle mappe de sécurité pour le pool de connexions du connecteur

La page Nouvelle mappe de sécurité pour le pool de connexions du connecteur permet de créer une mappe de sécurité pour un pool de connexions de connecteur.

Elle contient les options ci-après.

Nom du pool de connexions

Nom du pool de connexions de connecteur auquel cette mappe de sécurité appartient. Le nom du pool est un champ en lecture seule.

Nom de la mappe de sécurité

Nom de la mappe de sécurité.

Groupes d'utilisateurs

Si ce bouton radio est sélectionné, le champ de texte doit contenir les noms des groupes d'utilisateurs, séparés par des virgules.

Principaux

Si ce bouton radio est sélectionné, le champ de texte doit contenir les noms des principaux, séparés par des virgules.

Nom utilisateur

Nom utilisateur du principal back-end requis par le système EIS.

Mot de passe

Mot de passe du principal back-end requis par le système EIS.
