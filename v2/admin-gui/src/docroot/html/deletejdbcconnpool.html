To delete a JDBC connection pool

  1. Expand the Resources and then JDBC node.
  2. Select the Connection Pools node.
  3. Select the checkbox for the pool to be deleted.
  4. Click Delete.
More Information
Equivalent asadmin command

delete-jdbc-connection-pool
