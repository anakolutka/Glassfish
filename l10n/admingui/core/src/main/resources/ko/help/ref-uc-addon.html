사용 가능한 애드온

사용 가능한 애드온 페이지에서는 아직 설치되지 않은 설치 이미지의 구성 요소에 대한 정보를 제공합니다.

사용 가능한 애드온 페이지에는 다음 옵션이 포함되어 있습니다.

설치 경로

설치 이미지의 루트 또는 최상위 수준 디렉토리에 대한 파일 시스템 경로입니다.

설치

이 버튼을 누르면 선택한 애드온 구성 요소가 설치됩니다.

사용 가능한 애드온

이 테이블에서는 아직 설치되지 않은 설치 이미지의 각 구성 요소에 대해 다음 정보를 제공합니다.

구성 요소

구성 요소의 이름입니다.

Category

예를 들어 Application Server, 시스템 도구 및 데이터베이스와 같이 구성 요소가 연결된 소프트웨어의 범주입니다. 구성 요소가 만들어지면 범주가 구성 요소의 작성자에 의해 해당 구성 요소에 할당됩니다.

버전

구성 요소를 제공하는 패키지의 버전 번호입니다.

설치된 크기

구성 요소의 설치 시 사용하는 디스크 공간의 크기입니다.

Source

설치된 구성 요소의 경우 구성 요소를 가져온 네트워크 저장소의 이름입니다.

애드온 구성 요소 또는 업데이트의 경우 구성 요소를 가져온 네트워크 저장소의 이름입니다.
