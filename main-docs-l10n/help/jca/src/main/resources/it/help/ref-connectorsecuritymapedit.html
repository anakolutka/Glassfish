Modifica mappa di sicurezza connection pool connettore

Utilizzare la pagina Modifica mappa di sicurezza connection pool connettore per modificare una mappa di sicurezza per un connection pool del connettore.

La pagina Modifica mappa di sicurezza connection pool connettore contiene le opzioni riportate di seguito.

Nome connection pool

Il nome del connection pool del connettore a cui appartiene la mappa di sicurezza. Il campo del nome del pool è di sola lettura.

Nome mappa di sicurezza

Il nome della mappa di sicurezza. Il campo del nome è di sola lettura. È possibile specificare il nome di una mappa di sicurezza solo quando si crea una nuova mappa di sicurezza.

Nome mappa di sicurezza

Il nome della mappa di sicurezza.

Gruppi di utenti

Se questo pulsante di scelta è selezionato, il campo di testo deve contenere i nomi dei gruppi di utenti separati da virgole.

Principal

Se questo pulsante di scelta è selezionato, il campo di testo deve contenere i nomi dei principal separati da virgole.

Nome utente

Il nome utente del principal backend richiesto dal sistema EIS.

Password

La password del principal backend richiesto dal sistema EIS.
