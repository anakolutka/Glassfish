Configurations d'adaptateurs de ressources

La page Configurations d'adaptateurs de ressources permet de gérer les configurations de connecteur (adaptateur de ressources).

La configuration d'adaptateur de ressources fournit les informations de configuration d'un adaptateur de ressources. Avec la Console d'administration, vous créez une configuration d'adaptateur de ressources une fois que ce dernier est déployé. Dans ce cas, l'adaptateur de ressources est redémarré avec la nouvelle configuration. Pour créer une configuration d'adaptateur de ressources avant de déployer ce dernier, utilisez la commande asadmin create-resource-adapter-config.

Avant de pouvoir créer une configuration d'adaptateur de ressources, vous devez créer un pool de threads pour l'adaptateur de ressources à utiliser.

Pour chaque configuration d'adaptateur de ressources, les informations suivantes sont fournies.

Nom

Nom de l'adaptateur de ressources.

La table Configurations contient également les options ci-après.

Nouveau

Bouton permettant de créer une configuration d'adaptateur de ressources.

Supprimer

Bouton permettant de supprimer les configurations d'adaptateurs de ressources sélectionnées.
