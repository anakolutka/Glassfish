バッチ・ジョブ実行を表示するには

  1. ナビゲーション・ツリーで、バッチ・ジョブ実行を表示するGlassFish Serverのクラスタまたはサーバー・インスタンスを選択します。

    「一般情報」ページが開きます。

  2. バッチ・タブを選択します。

    バッチ・ジョブ実行ページが表示されます。

  3. バッチ・ジョブ実行ページで、実行IDをクリックし、特定の実行の詳細を表示します。

    バッチ・ジョブ実行の詳細ページが表示されます。

  4. バッチ・ジョブ実行のステップの詳細を表示するには、実行ステップ・タブをクリックします。

    バッチ・ジョブ実行のステップ・ページが表示されます。

  5. バッチ・ジョブ実行ページに戻るには、「戻る」ボタンをクリックします。
