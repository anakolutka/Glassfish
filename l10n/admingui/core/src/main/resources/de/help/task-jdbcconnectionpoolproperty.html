So bearbeiten Sie die JDBC-Verbindungspooleigenschaften

  1. Erweitern Sie in der Navigationsstruktur den Ressourcenknoten.
  2. Erweitern Sie unter dem Ressourcenknoten den JDBC-Knoten.
  3. Wählen Sie unter dem JDBC-Knoten den Verbindungspoolknoten aus.

    Die Seite "Verbindungspool" wird geöffnet.

  4. Klicken Sie auf den Namen des zu bearbeitenden Verbindungspools.

    Die Seite "Verbindungspool bearbeiten" wird geöffnet.

  5. Klicken Sie auf der Seite "Verbindungspool bearbeiten" auf die Registerkarte "Weitere Eigenschaften".
  6. Um eine Eigenschaft zu ändern, bearbeiten Sie das Feld "Wert" dieser Eigenschaft.
  7. So fügen Sie eine Eigenschaft hinzu:
    1. Klicken Sie auf die Schaltfläche "Eigenschaft hinzufügen".

      Eine leere Zeile wird angezeigt.

    2. Geben Sie im Feld "Name" den Eigenschaftsnamen ein.
    3. Geben Sie im Feld "Wert" den Wert ein.
  8. So löschen Sie eine Eigenschaft:
    1. Aktivieren Sie das Kontrollkästchen links neben dem Feld "Name" der zu löschenden Eigenschaft.
    2. Klicken Sie auf "Eigenschaften löschen".
  9. Klicken Sie auf "Speichern".
