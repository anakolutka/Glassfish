So legen Sie das aktive Überwachungsmodul fest

Bevor Sie beginnen

Zum Festlegen des vom Server verwendeten Überwachungsmoduls aktivieren Sie zunächst die Überwachungsprotokollierung, wie unter So aktivieren oder deaktivieren Sie die Überwachungsprotokollierung beschrieben.

  1. Wählen Sie im Feld "Überwachungsmodule" den Namen des Überwachungsmoduls aus, das vom Server verwendet werden soll.

    Der Standardwert ist default. Stellen Sie sicher, dass auditOn für dieses Überwachungsmodul auf true gesetzt ist, wie unter So verwenden Sie das standardmäßige Überwachungsmodul beschrieben.

  2. Klicken Sie auf "Speichern".
  3. Starten Sie Enterprise Server neu, wenn in der Konsole "Neustart erforderlich" angezeigt wird.
