Instale um driver que suporte a API da JDBC (driver JDBC) do banco de dados.

Verifique se o driver a ser utilizado para criar as conexões com bancos de dados está instalado em todos os servidores em que a fonte de dados será implantada.

Certifique-se de o arquivo JAR do driver esteja acessível à instância do servidor do domínio.
