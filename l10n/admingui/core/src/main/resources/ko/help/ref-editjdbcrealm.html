JDBCRealm 클래스와 관련된 등록 정보

웹 페이지 또는 웹 응용 프로그램을 보호하려면 등록된 사용자만 액세스할 수 있도록 보안을 설정할 수 있습니다. 이 기능을 인증 기능이라고 합니다. 이 영역 유형은 데이터베이스에 사용자 자격 증명을 저장하는 것과 관련됩니다. Enterprise Server는 구성 파일에서 데이터베이스 정보 및 활성화된 JDBC 영역 옵션을 사용합니다. JDBC 영역에 대해 다음 등록 정보가 필요합니다.

JAAS 컨텍스트

이 영역에 사용할 로그인 모듈의 식별자입니다. 유일한 유효 값은 jdbcRealm입니다.

J.N.D.I.

이 영역의 J.N.D.I. 이름입니다. 기본값은 jdbc/security 입니다.

사용자 테이블

이 영역의 인증된 사용자 목록을 포함하는 테이블입니다. 기본값은 usertable입니다.

사용자 이름 열

사용자 테이블에서 사용자 목록을 포함하는 열 이름입니다. 기본값은 userid입니다.

비밀번호 열

사용자 테이블에서 각 사용자의 비밀번호를 포함하는 열 이름입니다. 기본값은 password입니다.
