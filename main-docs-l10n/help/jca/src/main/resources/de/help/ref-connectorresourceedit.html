Connector-Ressource bearbeiten

Auf der Seite "Connector-Ressource bearbeiten" können Sie die Einstellungen für eine Connector-Ressource ändern.

Die Seite "Connector-Ressource bearbeiten" enthält die folgenden Optionen.

Standardeinstellungen laden

Schaltfläche zur Wiederherstellung der Standardwerte von Einstellungen, die Standardwerte haben. Einstellungen, die keine Standardwerte haben, werden nicht geändert.

JNDI-Name

Ein eindeutiger Name, der die Connector-Ressource identifiziert. Der JNDI-Name ist ein schreibgeschütztes Feld. Sie können den JNDI-Namen nur festlegen, wenn Sie eine neue Connector-Ressource erstellen.

Poolname

Der mit dieser Ressource verknüpfte Connector Connection-Pool.

Beschreibung

Eine Beschreibung der Connector-Ressource.

Status

Wenn diese Option ausgewählt wird, wird die Connector-Ressource aktiviert. Diese Option ist standardmäßig aktiviert.

Weitere Eigenschaften

Weitere Eigenschaften für die Connector-Ressource. Die verfügbaren Eigenschaften richten sich nach dem Ressourcenadapter, der von dem Connector Connection-Pool verwendet wird. Mit den in dieser Tabelle angegebenen Name/Werte-Paaren können die vom Ressourcenadapterprovider definierten Standardwerte für die Eigenschaften überschrieben werden.

Verwandte Aufgaben

  • So wird eine Connector-Ressource erstellt

  • So wird eine Connector-Ressource bearbeitet

  • So wird eine Connector-Ressource gelöscht
