JDBC 接続プールを削除する

  1. ナビゲーションツリーで、「リソース」ノードを展開します。
  2. 「リソース」ノードで、「JDBC」ノードを展開します。
  3. 「JDBC」ノードで、「接続プール」ノードを選択します。

    「接続プール」ページが開きます。

  4. 「接続プール」ページで、削除する接続プールの左側にあるチェックボックスを選択します。
  5. 「削除」をクリックします。
