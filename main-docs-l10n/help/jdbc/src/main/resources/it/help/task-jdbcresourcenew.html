Per creare una risorsa JDBC

Una risorsa (origine dati) JDBC (Java DataBase Connectivity) fornisce alle applicazioni i mezzi per connettersi a un database.

Prima di iniziare

Prima di creare una risorsa JDBC, è necessario creare un connection pool JDBC. Vedere Per creare un connection pool JDBC.

  1. Espandere il nodo Risorse nell'albero di navigazione.

  2. Nel nodo Risorse espandere il nodo JDBC.

  3. Nel nodo JDBC selezionare il nodo Risorse JDBC.

    Viene visualizzata la pagina Risorse JDBC.

  4. Nella pagina Risorse JDBC fare clic su Nuovo.

    Viene visualizzata la pagina Nuova risorsa JDBC.

  5. Nel campo Nome JNDI digitare un nome di risorsa univoco.

    Il nome JNDI consente di organizzare e individuare i componenti all'interno di un ambiente di elaborazione distribuito, in modo analogo agli schedari in una biblioteca, con cui viene organizzata e rappresentata la posizione dei libri. Il nome JNDI diventa quindi un metodo importante per accedere alla risorsa JDBC. Per convenzione, il nome inizia con la stringa jdbc/, ad esempio jdbc/payrolldb. Ricordare di includere la barra.

    Il nome deve contenere solo caratteri alfanumerici, caratteri di sottolineatura, lineette e punti.

  6. Nell'elenco a discesa Nome pool selezionare il connection pool da associare alla nuova risorsa JDBC.

  7. Nel campo Descrizione digitare una breve descrizione della nuova risorsa JDBC.

  8. Selezionare la casella di controllo Abilitato dell'opzione Stato per rendere disponibile la risorsa in fase di esecuzione.

    Questa opzione è abilitata per impostazione predefinita.

  9. Nella sezione Proprietà aggiuntive, specificare le proprietà aggiuntive.

    GlassFish Server non definisce alcuna proprietà aggiuntiva per le risorse JDBC.

    Per aggiungere una proprietà, fare clic sul pulsante Aggiungi proprietà. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Nome e il valore della proprietà nel campo Valore.

  10. Selezionare una destinazione disponibile per aggiungerla o rimuoverla dalla lista Destinazioni selezionate.

  11. Fare clic su OK.
