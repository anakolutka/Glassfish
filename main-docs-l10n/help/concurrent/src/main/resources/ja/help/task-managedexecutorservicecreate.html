管理対象エグゼキュータ・サービスを作成するには

管理対象エグゼキュータ・サービスでXXXを指定します。

  1. ナビゲーション・ツリーで、「リソース」ノードを展開します。

  2. 「リソース」ノードで、「並行処理リソース」ノードを展開します。

  3. 「並行処理リソース」ノードで、「管理対象エグゼキュータ・サービス」を選択します。

    「管理対象エグゼキュータ・サービス」ページが開きます。

  4. 「管理対象エグゼキュータ・サービス」ページで、「新規」をクリックします。

    「新規管理対象エグゼキュータ・サービス」ページが開きます。

  5. 「JNDI名」フィールドに、一意の名前を入力します。

    この名前はJNDIネーミング・サービスに登録されます。カード式の目録によって図書館の蔵書の分類と保管場所の管理を行う場合と同様の方法で、分散コンピューティング環境に配置されたコンポーネントの編成と場所の管理がJNDI名によって行われます。そのため、JNDI名はリソースへの重要なアクセス方法となります。

  6. XXX

  7. 「説明」フィールドに、リソースの簡単な説明を入力します。

    この説明は文字列値で、最大250文字を入力することができます。

  8. 「ステータス」の「有効」チェック・ボックスを選択し、リソースを有効にします。

    デフォルトでは、リソースは有効で、作成後すぐに使用できます。

  9. 「追加プロパティ」セクションで、リソースの追加プロパティを指定します。

    GlassFish Serverは管理対象エグゼキュータ・サービス・リソースの追加プロパティを定義しません。

    プロパティを追加するには、「プロパティを追加」ボタンをクリックします。表示される空白行で、「名前」フィールドにプロパティ名を入力し、「値」フィールドにプロパティ値を入力します。

  10. 「OK」をクリックします。
