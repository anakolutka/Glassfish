Page Service de contrôle

La page Service de contrôle permet de configurer les options de contrôle pour les composants ou services des serveurs individuels.

JVM

Niveau de contrôle JVM. Les niveaux disponibles sont BAS, HAUT et DÉSACTIVÉ. Dans cette version, il n'existe aucune différence entre les options BAS et HAUT.

Service HTTP

Niveau de contrôle du service HTTP. Les niveaux disponibles sont BAS, HAUT et DÉSACTIVÉ. Dans cette version, il n'existe aucune différence entre les options BAS et HAUT.

Conteneur Web

Niveau de contrôle du conteneur Web. Les niveaux disponibles sont BAS, HAUT et DÉSACTIVÉ. Dans cette version, il n'existe aucune différence entre les options BAS et HAUT.

Pool de threads

Niveau de contrôle des pools de threads. Les niveaux disponibles sont BAS, HAUT et DÉSACTIVÉ. Dans cette version, il n'existe aucune différence entre les options BAS et HAUT.
