Proprietà manager

Il Session Manager consente di configurare le modalità di creazione ed eliminazione delle sessioni, la posizione di memorizzazione della condizione della sessione e il numero massimo di sessioni disponibili.

Utilizzare la pagina Proprietà manager per aggiungere o modificare le proprietà del Session Manager per il contenitore Web.

La pagina Proprietà manager contiene le opzioni riportate di seguito.

Carica impostazioni predefinite

Pulsante per ripristinare i valori predefiniti delle impostazioni che dispongono di valori predefiniti. Le impostazioni che non dispongono di valori predefiniti non vengono modificate.

Nome configurazione

Il nome della configurazione a cui si applicano le impostazioni presenti in questa pagina. Questo campo è di sola lettura.

Intervallo di recupero

Il numero di secondi prima dell'eliminazione dei dati di una sessione inattiva dall'area di memorizzazione. Il valore predefinito è 60.

Impostare questa opzione su un valore minore del valore della frequenza con cui i dati della sessione vengono modificati. Ad esempio, nel caso di un sito Web utilizzato di frequente, si consiglia di specificare il valore più basso possibile (1 secondo) per un servlet del contatore di accessi, altrimenti potrebbero andare perduti gli ultimi accessi ogni volta che si riavvia il server.

Numero massimo sessioni

Il numero massimo di sessioni che possono essere inserite nella cache. Il valore -1 indica l'assenza di limiti per il numero delle sessioni. Il valore predefinito è -1.

Una volta raggiunto il limite, il tentativo di creare una nuova sessione comporta la restituzione di un'eccezione IllegalStateException.

Nome file sessione

Il percorso assoluto o relativo della directory in cui viene conservata la condizione della sessione a ogni riavvio dell'applicazione, se la conservazione della condizione è possibile. Se il percorso non è stato impostato, la condizione della sessione non viene salvata. Per impostazione predefinita, tale percorso non è impostato.

Il percorso relativo è relativo rispetto alla directory temporanea di questa applicazione Web.

Proprietà aggiuntive

Proprietà aggiuntive per il Session Manager. GlassFish Server non definisce alcuna proprietà aggiuntiva per il Session Manager.
