Modification d'une ressource de destination JMS

  1. Dans l'arborescence de navigation, développez le noeud Ressources.

  2. Sous le noeud Ressources, développez le noeud Ressources JMS.

  3. Sous le noeud Ressources JMS, sélectionnez le noeud Ressources de destination JMS.

    La page Ressources de destination JMS apparaît.

  4. Sur la page Ressources de destination JMS, cliquez sur le nom de la ressource de destination à modifier.

    La page Modifier une ressource de destination JMS apparaît.

  5. Dans le champ Nom de la destination physique, saisissez le nom de la destination physique à associer à cette ressource.

    GlassFish Server crée la destination physique automatiquement lorsque cela est nécessaire et la supprime lorsque vous supprimez la ressource de destination.

  6. Dans la liste déroulante Type de ressource, sélectionnez javax.jms.Topic ou javax.jms.Queue.

  7. Dans le champ Description, saisissez la description de la ressource de destination.

  8. Sélectionnez la case Activé du champ Statut pour activer la ressource lors de l'exécution.

    Cette option est activée par défaut.

  9. Dans la zone Autres propriétés, spécifiez les propriétés supplémentaires de la ressource de destination.

    GlassFish Server ne définit aucune autre propriété pour les ressources de destination JMS.

    • Pour ajouter une propriété, cliquez sur le bouton Ajouter une propriété. Sur la ligne vide qui apparaît, saisissez le nom de la propriété dans le champ Nom et la valeur de la propriété dans le champ Valeur.

    • Pour modifier une propriété, modifiez le champ Valeur de cette propriété.

    • Pour supprimer une propriété, sélectionnez la case située à gauche du champ Nom de la propriété à supprimer, puis cliquez sur le bouton Supprimer des propriétés.

  10. Cliquez sur Enregistrer.

    Conseil:

    Pour revenir aux paramètres par défaut, cliquez sur le bouton Charger les valeurs par défaut, puis sur Enregistrer.

  11. Vous pouvez cliquer sur l'onglet Cibles pour accéder à la page Cibles de la ressource de destination JMS. Sur cette page, procédez comme suit :

    1. Cliquez sur Gérer les cibles pour ouvrir la page Gérer les cibles des ressources.

      Sur cette page, déplacez les cibles entre les colonnes Disponible et Sélectionné. Dans la colonne Sélectionné, veillez à placer les cibles sur lesquelles les applications qui utilisent la ressource seront déployées. Les cibles disponibles comprennent les instances de serveur et les clusters disponibles, ainsi que l'instance de serveur par défaut server. Cliquez sur OK pour enregistrer les modifications.

    2. Cochez la case correspondant à une cible, puis cliquez sur Activer ou sur Désactiver pour activer ou désactiver la ressource pour cette cible.
