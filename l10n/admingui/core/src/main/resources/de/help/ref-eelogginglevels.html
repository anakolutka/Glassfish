Modul-Protokollebenen

Verwenden Sie die Seite Modul-Protokollebenen, um die Protokollebenen für einzelne Module zu konfigurieren.

Für jedes Modul enthält die Seite Modul-Protokollebenen die folgenden Informationen.

Name der Protokollfunktion

Der Name der Protokollfunktion für das Modul (z. B. javax.enterprise.system.tools.admin .

Protokollfunktionsebene

Die aktuelle Protokollierungsebene für das Modul.

Die folgenden Protokollebenen sind verfügbar. Sie sind in der Reihenfolge vom höchsten zum niedrigsten aufgelistet.

SCHWERWIEGEND

Ereignisse, die die normale Programmausführung stören.

WARNUNG

Warnungen, einschließlich Ausnahmen.

INFO

Nachrichten, die mit Serverkonfiguration oder Serverstatus in Zusammenhang stehen, ausgenommen Fehler.

KONFIGURATION

Nachrichten, die mit der Serverkonfiguration in Zusammenhang stehen.

FINE

Minimale Überlänge

FINER

Moderate Überlänge.

FINEST

Maximale Überlänge.

DEAKTIVIERT

Keine Protokollnachrichten.

Verwandte Aufgaben

  • So konfigurieren Sie allgemeine Protokollierungseinstellungen

  • So konfigurieren Sie Protokollebenen

  • So zeigen Sie das Serverprotokoll an

Verwandte asadmin-Befehle

  • set-log-level

  • rotate-log(1)
