新建 File 领域用户

使用“新建 File 领域用户”页面可以授予新用户对当前领域的访问权限。

“新建 File 领域用户”页面包含以下选项：

领域名

当前选中领域的名称。这是只读字段。

用户 ID

新用户帐户的用户 ID。

组列表

新用户所属的组。使用逗号分隔多个组名称。

对于 admin-realm 用户，这是只读字段，并且组为 asadmin。

新密码

新用户的密码。

确认新密码

新用户的密码。

  • 编辑 File 领域用户信息

  • 从 File 领域中删除用户
