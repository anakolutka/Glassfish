Editar Serviço de Gerenciamento de Grupos

Utilize a página Serviço de Gerenciamento de Grupos para alterar as definições do GMS.

O Serviço de Gerenciamento de Grupos (GMS) é um serviço em andamento que oferece monitoramento de cluster, associação de cluster e serviços de comunicação de grupos. O GMS está construído sobre uma pilha configurável de protocolos de monitoramento da integridade e de descoberta da associação do grupo. O GMS realiza as seguintes funções:

  • Notifica os módulos registrados em uma instância do GlassFish Server quando uma ou mais instâncias do membro são adicionadas ou removidas de um cluster, ou quando tais instâncias são suspeitas de ter falhas ou realmente tenham falhas.

  • Fornece a capacidade de enviar e receber mensagens entre membros do cluster.

A página Serviço de Gerenciamento de Grupos contém as seguintes opções.

Carregar Defaults

Botão para restaurar definições para seus valores default. Definições que não tiveram seus valores default alterados.

Nome da Configuração

O nome da configuração à qual as definições nesta página se aplicam. Este campo é somente para leitura.

Máximo de Pulsações Perdidas

O número de pulsações perdidas de um membro após o qual há suspeita de uma falha da instância do servidor. O default é 3.

Frequência das Pulsações

O período entre pulsações perdidas de um membro. O default é 2000 milissegundos.

Time-out para Descoberta de Grupo

Quantidade de tempo que o GMS aguarda pela a descoberta de outros membros do grupo. O default é 5000 milissegundos.

Tempo de Espera para Confirmação de Falha)

O tempo a ser aguardado antes de confirmar que um membro com suspeita de falha deve ser relatado como apresentado falha. O default é 1500 milissegundos.

Propriedades Adicionais

Tabela de propriedades adicionais do Serviço de Gerenciamento de Grupos.

O GlassFish Server não define propriedades adicionais do GMS.

São exibidas as seguintes informações para cada propriedade:

Nome

O nome da propriedade.

Valor

Valor da propriedade definida para a instância selecionada.

Descrição

Uma descrição textual que fornece mais informações sobre a propriedade.

A tabela Propriedades Adicionais também contém as seguintes opções.

Adicionar Propriedade

Botão para adicionar uma propriedade. Clicar neste botão adiciona uma linha à tabela Propriedades Adicionais.

Deletar Propriedades

Botão para deletar uma ou mais propriedades selecionadas. Qualquer propriedade deletada retorna ao seu valor default ou, se nenhum valor tiver sido definido, fica com valor indefinido.
