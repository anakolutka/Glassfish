일반 정보

일반 정보 페이지에는 다음 정보가 표시됩니다.

이름

서버의 이름입니다.

상태

서버의 현재 상태입니다. 서버는 중지됨, 시작됨 또는 실행 중일 수 있습니다.

HTTP 로드 밸런서

사용 가능한 경우 로드 밸런서의 이름입니다.

HTTP 포트

HTTP 요청 및 응답에 대해 현재 구성된 포트입니다.

IIOP 포트

IIOP 요청 및 응답에 대해 현재 구성된 포트입니다.

구성 디렉토리

domain.xml과 같은 구성 파일이 포함된 호스트 시스템 상의 디렉토리입니다.

설치된 버전

제품 버전 식별자입니다.

디버그

디버깅을 사용하도록 설정되어 있는지 여부를 식별합니다. 디버깅을 사용하도록 설정된 경우 포트 번호도 표시됩니다.
