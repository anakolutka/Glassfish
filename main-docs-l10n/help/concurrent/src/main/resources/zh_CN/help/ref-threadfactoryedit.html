编辑托管线程工厂

使用“编辑托管线程工厂”页可编辑托管线程工厂资源。

“编辑托管线程工厂”页包含以下选项。

加载默认值

用于将具有默认值的设置还原为其默认值的按钮。没有默认值的设置不发生更改。

JNDI 名称

标识资源的唯一名称。JNDI 名称是只读字段。只能在创建新资源时指定 JNDI 名称。

逻辑 JNDI 名称

资源的逻辑 JNDI 名称。这是一个 Java EE 标准名称，使用该名称可以访问资源。逻辑 JNDI 名称是只读字段。有关详细信息，请参阅逻辑 JNDI 名称。

上下文信息

传播到线程的上下文：类加载器、JNDI、安全性或工作区。可以启用或禁用上下文传播。如果启用，则将传播所选上下文。如果禁用，则不传播任何上下文，即使已选中也是如此。默认值为“已启用”并选中所有四个上下文。

状态

资源在运行时是否可用。

如果域中只存在默认服务器实例 server，则可以启用或禁用资源的运行时可用性。默认情况下启用运行时可用性。

如果域中已创建了任何集群或独立实例，则使用“目标”选项卡来控制其目标上资源的运行时可用性。

线程优先级

分配给已创建线程的优先级。默认值为 5。

部署顺序

此资源的部署顺序。在服务器启动时先加载编号较低的资源。默认值为 100。

说明

此资源的说明。

其他属性

资源的其他属性。GlassFish Server 不定义此资源类型的任何其他属性。此外，此资源类型当前不使用任何其他属性。
