Configuración de ruta de acceso de JVM

Utilice la página Configuración de Ruta de Acceso de JVM para configurar la ruta de acceso de biblioteca nativa. La ruta de acceso de biblioteca nativa consiste en una concatenación de las rutas de acceso siguientes:

  • El prefijo de la ruta de acceso de biblioteca nativa

  • La ruta de acceso del servidor para sus bibliotecas nativas (es decir, as-install/lib)

  • Los directorios de biblioteca nativa de 64 bits, si procede

  • El valor de la variable java.library.path para la JVM utilizada para iniciar GlassFish Server (este valor es distinto en diferentes sistemas operativos)

  • Las rutas de acceso que se especifican en la página Configuración de analizador de perfiles de JVM, si el analizador está activado

  • El sufijo de la ruta de acceso de biblioteca nativa

Otros parámetros de configuración se muestran como valores de sólo lectura para los dominios que se han actualizado desde una versión anterior de GlassFish Server. No están soportados en la versión v3.

La página Configuración de Ruta de Acceso de JVM contiene las siguientes opciones.

Nombre de Configuración

Nombre de la configuración a la que se aplican los valores de esta página. Este campo es de sólo lectura.

Classpath de Entorno

Si se activa la casilla de control Ignorar, se omitirá la variable de entorno CLASSPATH. La variable de entorno CLASSPATH resulta cómoda para tutoriales básicos de programación, pero no se recomienda en entornos de empresa. Éste es un campo de sólo lectura.

Classpath del Sistema

Ruta de acceso a las clases del sistema. Éste es un campo de sólo lectura.

Prefijo de Classpath

Rutas de acceso completas a los archivos JAR que se van a anteponer a la classpath del servidor, una por línea. Éste es un campo de sólo lectura.

Sufijo de Classpath

Rutas de acceso completas a los archivos JAR que se van a agregar a la classpath del servidor, una por línea. Éste es un campo de sólo lectura.

Prefijo de Ruta de Acceso de Biblioteca Nativa

Rutas de acceso completas a los archivos JAR que se van a anteponer a la ruta de acceso de biblioteca nativa, una por línea.

Sufijo de Ruta de Acceso de Biblioteca Nativa

Rutas de acceso completas a los archivos JAR que se van a agregar a la ruta de acceso de biblioteca nativa, una por línea.
