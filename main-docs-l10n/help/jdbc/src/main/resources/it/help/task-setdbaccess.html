Per la memorizzazione, l'organizzazione e il recupero di dati, la maggior parte delle applicazioni utilizza i database relazionali.

GlassFish Server include una copia del database Java DB ed è preconfigurato con la funzione di accesso a tale database.

  1. Installare un prodotto database supportato.

    Per la lista dei database supportati, vedere Oracle GlassFish Server Release Notes.

  2. Installare un driver che supporti l'interfaccia API JDBC (driver JDBC) per il database.

    Assicurarsi che il driver da utilizzare per creare le connessioni al database sia installato in tutti i server nei quali si desidera distribuire l'origine dati.

  3. Assicurarsi che il file JAR del driver sia accessibile per l'istanza server del dominio.

    Vedere Per integrare un driver JDBC.

  4. Creare il database.

    In genere il provider di applicazioni fornisce script per la creazione e l'inserimento dei dati nel database.

  5. Creare un connection pool per il database.

    Vedere Per creare un connection pool JDBC.

  6. Creare una risorsa JDBC che punti al connection pool.

    Vedere Per creare una risorsa JDBC.
