Cómo crear un dominio

Servidor empresarial viene preconfigurado con tres dominios: file, certificate y admin-realm. También es posible crear dominios ldap, JDBC, solaris y personalizados. Normalmente se tiene un dominio de cada tipo en un servidor, pero en Servidor empresarial existen dos dominios de archivo: file y admin-realm. Se trata de dos dominios del mismo tipo utilizados para dos fines diferentes. También es posible tener una base de datos de certificados diferente para cada servidor virtual de su sistema.

  1. En el árbol de navegación, expanda el nodo Configuración.
  2. Expanda el nodo Seguridad.
  3. Seleccione el nodo Dominios.

    Se abrirá la página Dominios.

  4. En la página Dominios haga clic en Nuevo.

    Se abrirá la página Nuevo dominio.

  5. En el campo Nombre, introduzca un nombre para el dominio.
  6. Para especificar el nombre de clase del dominio, realice una de las siguientes acciones:
    • En la lista desplegable Nombre de clase, seleccione el nombre de clase para el dominio creado.

      En la tabla siguiente se muestran las opciones válidas.

      Nombre de dominio

      Nombre de clase

      file

      com.sun.enterprise.security.auth.realm.file.FileRealm

      certificate

      com.sun.enterprise.security.auth.realm.certificate.CertificateRealm

      ldap

      com.sun.enterprise.security.auth.realm.ldap.LDAPRealm

      JDBC

      com.sun.enterprise.security.auth.realm.jdbc.JDBCRealm

      solaris

      com.sun.enterprise.security.auth.realm.solaris.SolarisRealm
    • Seleccione el botón de opción inferior y escriba el nombre de clase del dominio personalizado creado.
  7. (Opcional) En la sección Propiedades adicionales, especifique las propiedades adicionales.

    Para añadir una propiedad, haga clic en el botón Agregar propiedad. En la fila vacía escriba el nombre de la propiedad en el campo Nombre y el valor de la propiedad en el campo Valor.

    Las propiedades válidas dependen del tipo de dominio.

    • Para obtener una descripción de las propiedades del dominio de archivo, consulte Propiedades específicas de la clase FileRealm.

    • Para obtener una descripción de las propiedades del dominio de certificado, consulte Propiedades específicas de la clase CertificateRealm .

    • Para obtener una descripción de las propiedades del dominio JDBC, consulte Propiedades específicas de la clase JDBCRealm .

    • Para obtener una descripción de las propiedades del dominio LDAP, consulte Propiedades específicas de la clase LDAPRealm .

    • Para obtener una descripción de las propiedades del dominio Solaris, consulte Propiedades específicas de la clase SolarisRealm .

  8. Haga clic en Aceptar.
