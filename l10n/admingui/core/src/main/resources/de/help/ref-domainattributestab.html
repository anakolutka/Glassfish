Domänen-Attribute

Verwenden Sie diese Seite, um die erweiterten Domänenattribute für Enterprise Server festzulegen.

Anwendungs-Root

Der vollständige Pfad zu dem Verzeichnis, in dem Anwendungen bereitgestellt werden.

Protokoll-Root

Gibt an, wo die Serverprotokolldateien gespeichert sind.

Ländereinstellung

Wenn Sie dieses Feld leer lassen, wird die standardmäßige Ländereinstellung des Hosts verwendet.

Eine Ländereinstellung ist ein Bezeichner, der eine bestimmte Kombination aus Sprache und Region angibt. Beispielsweise lautet die Ländereinstellung für Englisch (USA) en_US , die Ländereinstellung für Japanisch lautet ja_JP. Um eine nicht englische Ländereinstellung zu verwenden, muss Anwendungsserver lokalisiert, d. h. in eine andere Sprache übersetzt werden.
