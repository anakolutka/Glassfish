Modifica modulo di audit

Utilizzare la pagina Modifica modulo di audit per modificare un modulo di audit.

La pagina Modifica modulo di audit contiene le opzioni riportate di seguito.

Nome configurazione

Il nome della configurazione a cui si applicano le impostazioni presenti in questa pagina. Questo campo è di sola lettura.

Nome

Il nome del modulo di audit. Questo campo è di sola lettura. È possibile specificare il nome solo quando si crea un nuovo modulo di audit.

Nome classe

Il nome completamente qualificato della classe che implementa questo modulo. Il nome della classe per il modulo di audit predefinito è com.sun.enterprise.security.Audit.

Proprietà aggiuntive

Proprietà aggiuntive per il modulo di audit. Per la descrizione della proprietà aggiuntiva disponibile per i moduli di audit, vedere Proprietà specifiche dei moduli di audit.
