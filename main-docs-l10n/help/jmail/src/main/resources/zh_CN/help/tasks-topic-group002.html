查看批处理作业执行

  1. 在导航树中，选择要查看批处理作业执行的 GlassFish Server 集群或服务器实例。

    将打开“一般信息”页。

  2. 选择“批处理”选项卡。

    此时将打开“批处理作业执行”页。

  3. 在“批处理作业执行”页上，单击一个执行 ID 以查看有关特定执行的详细信息。

    此时将打开“批处理作业执行详细信息”页。

  4. 要查看有关批处理作业执行中的步骤的详细信息，请单击“执行步骤”选项卡。

    此时将打开“批处理作业执行步骤”页。

  5. 单击“后退”按钮可返回“批处理作业执行”页。
