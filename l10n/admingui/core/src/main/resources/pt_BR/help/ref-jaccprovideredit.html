Editar provedor de JACC

Use a página Editar Provedor de JACC para modificar um provedor de JACC.

A página Editar Provedor de JACC contém as seguintes opções.

Nome

O nome do provedor JACC. Este é um campo somente leitura. Você só pode especificar o nome quando cria um novo provedor de JACC.

Configuração da Diretiva

O nome da classe que implementa a fábrica de configuração de diretivas.

Provedor de Diretivas

O nome da classe que implementa a fábrica de diretivas.

Propriedades Adicionais

Propriedades adicionais do provedor de JACC. Para obter uma descrição das propriedades adicionais disponíveis para provedores de JACC, consulte Propriedades específicas de provedores de JACC.
