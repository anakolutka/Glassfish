Affichage des données de surveillance d'application pour une instance GlassFish Server autonome

Avant de commencer

Avant d'afficher les données de surveillance, vous devez configurer le service de surveillance.

  1. Dans l'arborescence de navigation, sélectionnez le noeud Instances autonomes.

    La page Instances de serveur autonomes apparaît.

  2. Sélectionnez l'instance pour laquelle vous souhaitez afficher les données de surveillance d'application.

    La page Informations générales de l'instance apparaît.

  3. Sur la page Informations générales, cliquez sur l'onglet Surveiller.

    La page Surveillance d'application apparaît.

  4. Dans la liste déroulante Afficher, sélectionnez l'application pour laquelle vous souhaitez afficher les données de surveillance.

  5. Pour mettre à jour les données de surveillance affichées, cliquez sur Actualiser.
