Connector-Service

Auf der Seite "Connector-Service" können Sie allgemeine Einstellungen für den Connector-Service ändern, mit dem Ressourcenadapter gesteuert werden.

Die Seite "Connector-Service" enthält die folgenden Optionen.

Standardeinstellungen laden

Schaltfläche zur Wiederherstellung der Standardwerte von Einstellungen, die Standardwerte haben. Einstellungen, die keine Standardwerte haben, werden nicht geändert.

Name der Konfiguration

Der Name der Konfiguration, für die die Einstellungen auf dieser Seite angewendet werden. Dieses Feld ist schreibgeschützt.

Timeout beim Herunterfahren

Die maximale Anzahl Sekunden, die während des Herunterfahrens von GlassFish Server auf die Ausführung der ResourceAdapter.stop-Methode einer Connector-Modulinstanz gewartet wird. Ressourcenadapter, deren Herunterfahren länger dauert, werden ignoriert, und das Herunterfahren von GlassFish Server wird fortgesetzt. Der Standardwert ist 30 Sekunden.

Policy zum Laden von Connector-Klassen

Die beim Laden von Klassen anzuwendende Policy. Zur Auswahl stehen:

derived

Gibt an, dass die Ressourcenadapter entsprechend den Verweisen einer Anwendung auf beliebige Ressourcen des Ressourcenadapters bereitgestellt werden. Diese Policy ist die Standardeinstellung.

global

Gibt an, dass alle Ressourcenadapter für alle Anwendungen sichtbar sind.

Diese Policy gilt nicht für den vorinstallierten JMS-Systemressourcenadapter, jmsra, der immer für alle Anwendungen sichtbar ist.

Verwandte Aufgaben

  • So stellen Sie ein Connector-Modul bereit
