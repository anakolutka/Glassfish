JDBC

JavaTM 資料庫連結 (JDBCTM) 技術提供關聯式資料庫的連結。在 Enterprise Server 中，請將 JDBC 資源 (資料來源) 增加至網域以配置資料庫連結。資料來源是 Java Platform, Enterprise Edition (Java EE 平台) 配置資料庫連結的標準方法。每個資料來源都包含一個資料庫連線池。應用程式會在 Java Naming and Directory InterfaceTM (J.N.D.I.) API 樹狀結構或本機應用程式環境中查找資料來源，然後使用 getConnection 方法擷取資料庫連線。資料來源及其連線池提供了有助於保持系統執行的連線管理程序。

若要配置 JDBC 資源，請使用下列頁面：

  • JDBC 資源

  • 新建 JDBC 資源

  • 編輯 JDBC 資源

  • 連線池

  • 新建 JDBC 連線池 (步驟 1，共 2 步)

  • 新建 JDBC 連線池 (步驟 2，共 2 步)

  • 編輯連線池
