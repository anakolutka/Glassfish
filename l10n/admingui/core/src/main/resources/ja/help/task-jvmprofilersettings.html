JVM のプロファイラ設定を行う

プロファイラツールは、パフォーマンスを分析し、潜在的な障害を特定するための情報を生成します。

  1. ナビゲーションツリーで、「設定」ノードを選択します。
  2. 「設定」ノードの下で、「JVM 設定」ノードを選択します。

    「JVM 一般設定」ページが開きます。

  3. 「JVM 一般設定」ページで、「プロファイラ」タブをクリックします。

    「JVM プロファイラ設定」ページが開きます。

  4. プロファイラ設定を指定します。

    指定する情報は、使用するプロファイラ製品によって異なります。具体例と手順については、『Sun GlassFish Enterprise Server v3 Preview Application Development Guide』の第 4 章「Debugging Applications」を参照してください。

  5. 「保存」をクリックします。
  6. サーバーを再起動します。
