提供者配置

使用 [提供者配置] 頁面可配置 SOAP 或 HttpServlet 認證層的訊息安全性提供者配置。

根據預設，SOAP 認證層會設定四種提供者。

[提供者配置] 頁面可顯示每則訊息的下列資訊。

提供者 ID

提供者配置的名稱。

提供者類型

提供者的類型，可以是下列其中任何一項：

client

指定提供者是用戶端認證提供者。

server

指定提供者是伺服器認證提供者。

client-server

指定提供者既是用戶端認證提供者，也是伺服器認證提供者。

預設提供者

若設為 true，提供者即是預設提供者。

類別名稱

提供者的 Java 實作類別名稱。
