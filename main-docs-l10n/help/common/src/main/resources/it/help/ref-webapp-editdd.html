Descrittori modulo

Utilizzare la pagina Descrittori modulo per visualizzare i descrittori di modulo per un'applicazione.

La pagina Descrittori modulo contiene le opzioni riportate di seguito.

Descrittori di distribuzione

Tabella in cui vengono visualizzati il nome dell'applicazione, il nome del componente secondario (se applicabile) e il nome del file del descrittore.
