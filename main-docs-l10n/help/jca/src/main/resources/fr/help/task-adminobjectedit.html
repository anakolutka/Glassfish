Modification d'une ressource d'objet administré

  1. Dans l'arborescence de navigation, développez le noeud Ressources.

  2. Développez le noeud Connecteurs.

  3. Sous le noeud Connecteurs, sélectionnez le noeud Ressources d'objet d'administration.

    La page Ressources d'objet d'administration apparaît.

  4. Sur la page Ressources d'objet d'administration, cliquez sur le nom de la ressource d'objet administré à modifier.

    La page Modifier une ressource d'objet d'administration apparaît.

  5. Dans la liste déroulante Adaptateur de ressources, sélectionnez l'adaptateur de ressources pour la ressource d'objet administré.

  6. Dans la liste déroulante Type de ressource, sélectionnez le type Java de la ressource d'objet administré.

  7. Dans la liste déroulante Nom de classe, sélectionnez le nom de la classe d'implémentation associée au type de ressource.

  8. Dans le champ Description, saisissez la description de la ressource d'objet administré.

  9. Dans le champ Ordre de déploiement, saisissez un nombre pour indiquer l'ordre de déploiement de la ressource d'objet administré.

    Les ressources avec les plus petits numéros sont chargés en premier au démarrage du serveur. La valeur par défaut est 100.

  10. Sélectionnez la case Activé du champ Statut pour activer la ressource d'objet administré.

    Cette option est activée par défaut.

  11. Dans la table Autres propriétés, spécifiez les propriétés supplémentaires de la ressource d'objet administré.

    Les propriétés disponibles dépendent de l'adaptateur de ressources du pool de connexions de connecteur.

    • Pour ajouter une propriété, cliquez sur le bouton Ajouter une propriété. Sur la ligne vide qui apparaît, saisissez le nom de la propriété dans le champ Nom et la valeur de la propriété dans le champ Valeur.

    • Pour modifier une propriété, modifiez le champ Valeur de cette propriété.

    • Pour supprimer une propriété, sélectionnez la case située à gauche du champ Nom de la propriété à supprimer, puis cliquez sur le bouton Supprimer des propriétés.

  12. Dans l'onglet Cible, cliquez sur Gérer les cibles pour modifier les cibles sur lesquelles l'objet administré est déployé.

    Revenez à la page Modifier une ressource d'objet d'administration en cliquant sur l'onglet Général.

  13. Cliquez sur Enregistrer.

    Conseil:

    Pour revenir aux paramètres par défaut, cliquez sur le bouton Charger les valeurs par défaut, puis sur Enregistrer.
