응용 프로그램

응용 프로그램 페이지에는 &ProductName에 배포된 응용 프로그램의 목록이 표시됩니다. 여기에서 이미 배포된 응용 프로그램을 보고 관리할 뿐만 아니라 더 많은 응용 프로그램을 배포할 수 있습니다.

각 응용 프로그램에 대해 제공되는 정보는 다음과 같습니다.

이름

응용프로그램 이름입니다.

사용 가능

응용 프로그램이 사용 가능한 경우 확인 표시가, 사용 가능하지 않은 경우 X가 표시됩니다.

복합

응용 프로그램에 구성 요소가 두 개 이상이면 True, 구성 요소가 하나만 있으면 False입니다.

엔진

응용 프로그램에서 사용되는 컨테이너의 유형입니다. 컨테이너 유형에는 다음과 같은 것이 있습니다.

  • web

  • webservices

  • ejb

  • connector

  • appclient

  • jruby

작업

배포한 구성 요소에서 수행할 수 있는 작업에 대한 링크입니다. 모든 구성 요소에 대해 재배포와 다시 시작이 표시되고 웹 응용 프로그램의 경우 시작이 표시됩니다.

배포

엔터프라이즈 응용 프로그램을 배포하는 버튼입니다.

배포 해제

엔터프라이즈 응용 프로그램을 배포 해제하는 버튼입니다.

활성화

선택한 엔터프라이즈 응용 프로그램을 활성화하는 버튼입니다.

비활성화

선택한 엔터프라이즈 응용 프로그램을 비활성화하는 버튼입니다.

필터

엔진별로 응용 프로그램을 필터링하는 드롭다운 목록입니다.

관련 작업

  • 배포된 응용 프로그램의 목록을 보는 방법

  • 웹 응용 프로그램을 배포하는 방법

  • 엔터프라이즈 응용 프로그램을 배포하는 방법

  • 응용 프로그램 클라이언트를 배포하는 방법

  • 커넥터 모듈을 배포하는 방법

  • EJB Jar를 배포하는 방법

  • Ruby 응용 프로그램을 배포하는 방법

  • 배포된 응용 프로그램을 편집하는 방법

  • 응용 프로그램의 설명자를 보는 방법

  • 응용 프로그램을 시작하는 방법

  • 배포된 응용 프로그램을 재배포하는 방법

  • 배포된 응용 프로그램을 다시 시작하는 방법
