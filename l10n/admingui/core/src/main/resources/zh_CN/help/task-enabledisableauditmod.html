启用或禁用审计日志记录

  1. 在导航树中，展开“配置”节点。
  2. 选择要配置的实例：
    • 要配置特定的实例，请展开该实例的配置节点。例如，对于默认实例 server，请展开 "server-config" 节点。
    • 要配置所有实例的默认设置，请展开 "default-config" 节点。
  3. 在“配置”节点下，选择“安全性”节点。

    将打开“安全性”页面。

  4. 选中“启用审计日志记录”复选框可以启用审计日志记录。

    要禁用审计日志记录，请取消选中“启用审计日志记录”复选框。

    启用此选项可以装入审计模块并确保这些模块在审计时被 Enterprise Server 的审计库调用。

  5. 如果要启用审计日志记录，请指定一个默认的审计模块，如设置活动的审计模块中所述。
  6. 单击“保存”。
  7. 如果控制台中显示“必须重新启动”，请重新启动 Enterprise Server。
