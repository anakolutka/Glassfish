라이프사이클 모듈

라이프사이클 모듈 페이지에는 사용 가능한 라이프사이클 모듈 목록이 표시됩니다.

라이프사이클 모듈은 서버의 라이프사이클에서 하나 이상의 이벤트에 의해 트리거될 경우에 작업을 수행합니다. 발생할 수 있는 트리거 서버 이벤트에는 초기화, 시작, 요청 서비스 준비 및 종료 등이 있습니다. 라이프사이클 모듈은 Java EE 사양의 일부가 아니지만 Enterprise Server의 개선 사항입니다.

각 라이프사이클 모듈에 대해 제공되는 정보는 다음과 같습니다.

이름

라이프사이클 모듈의 이름입니다.

사용 가능

라이프사이클 모듈이 사용 가능한 경우 true, 사용 가능하지 않은 경우 false입니다.

로드 순서

라이프사이클 모듈을 로드해야 할 상대적 순서를 지정하는 값입니다.

클래스 이름

라이프사이클 모듈의 클래스 파일입니다.

관련 작업

  • 라이프사이클 모듈을 만드는 방법

  • 라이프사이클 모듈을 편집하는 방법

  • 라이프사이클 모듈을 삭제하는 방법

관련 asadmin 명령

  • create-lifecycle-module(1)

  • list-lifecycle-modules(1)

  • delete-lifecycle-module(1)
