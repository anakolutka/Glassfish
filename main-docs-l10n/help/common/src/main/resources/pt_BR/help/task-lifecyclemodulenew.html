Para Criar um Módulo do Ciclo de Vida

  1. Na árvore de navegação, selecione o nó Módulos de Ciclo de Vida.

    A página Módulos de Ciclo de Vida será aberta.

  2. Na página Módulos de Ciclo de Vida, clique em Novo.

    A página Novo Módulo de Ciclo de Vida será aberta.

  3. No campo Nome, digite um nome para o módulo de ciclo de vida.

  4. No campo Nome da Classe, digite o nome totalmente qualificado do arquivo de classe do módulo de ciclo de vida.

    A classe deve implementar a interface com.sun.appserv.server.LifecycleListener.

  5. No campo Classpath, digite o classpath do módulo do ciclo de vida.

    O classpath especifica onde o módulo do ciclo de vida está localizado. O valor default é domain-dir/applications. Se o módulo já estiver neste diretório (isto é, no classpath do servidor), esse campo poderá ser deixado em branco.

  6. No campo Ordem de Carga, digite um valor especificando a ordem na qual esse módulo do ciclo de vida deve ser carregado na inicialização.

    Os módulos com valores inteiros menores de ordem de carga são carregados antes. O valor pode variar de 101 até o MAXINT do sistema operacional. Valores de 1 a 100 são reservados.

  7. No campo Descrição, digite uma descrição do módulo de ciclo de vida.

  8. Marque a caixa de seleção Status Ativado para ativar o módulo de ciclo de vida.

    Por default, a opção está ativada.

  9. Marque a caixa de seleção Inicialização de Instância de Prevenção no Caso de Falha ao Carregar para especificar que o servidor seja submetido a shutdown em caso de falha ao carregar o módulo do ciclo de vida.

    Por default, essa opção está desativada.

  10. Para adicionar instâncias de servidor e clusters do alvo nos quais serão implantados o módulo do ciclo de vida, selecione-os na lista Alvos disponíveis e utilize o botão Adicionar para movê-los para a lista Alvos selecionados.

    A opção Alvo aparece se clusters ou instâncias de servidores standalone, além da instância default do servidor (server), existirem. Por default, o módulo do ciclo de vida está ativado na instância default do servidor, com o nome server. Utilize a botão Adicionar Tudo para implantar em todos os alvos. Utilize os botões Remover e Remover Tudo para remover os alvos da lista Alvos Selecionados.

  11. Clique em OK.
