Modification d'un utilisateur de domaine de fichiers

Pour modifier les privilèges d'accès au domaine actuel d'un utilisateur, utilisez la page Modification d'un utilisateur de domaine de fichiers.

Nom de domaine

Champ en lecture seule indiquant le nom du domaine sélectionné.

ID utilisateur

Champ en lecture seule indiquant le nom du compte que vous éditez.

Liste des groupes

Groupes auxquels cet utilisateur va appartenir. Séparez les noms de groupes par des virgules.

Nouveau mot de passe

Nouveau mot de passe de cet utilisateur.

Confirmer le mot de passe

Nouveau mot de passe de cet utilisateur.
