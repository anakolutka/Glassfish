Modificar los Recursos de Ámbito de Aplicación

  1. En el árbol de navegación, seleccione el nodo Aplicaciones.

    Se abrirá la página Aplicaciones.

  2. Seleccione una aplicación para la que desee ver los recursos de ámbito de aplicación.

    El nombre de la aplicación puede incluir un identificador de versión opcional, que va a continuación del nombre y está separado de él por dos puntos (:). Para obtener más información sobre las versiones de módulo y de aplicación, consulte el apartado sobre las versiones de módulo y de aplicación en Oracle GlassFish Server Application Deployment Guide.

    Se abrirá la página de la aplicación.

  3. Seleccione el separador Recursos.

    Se abrirá la página Recursos de ámbito de aplicación y los recursos actualmente especificados para la aplicación aparecen en una tabla con las columnas siguientes:

    Nombre de Aplicación

    Nombre de la aplicación.

    Nombre de Módulo

    Nombre del módulo dentro de la aplicación.

    Nombre de recurso

    Nombre del recurso de ámbito de aplicación para la aplicación o el módulo.

    Tipo de recurso

    Tipo de recurso, por ejemplo Pools de Conexiones JDBC o Recursos de Conectores.

  4. Haga clic en el nombre del recurso que desee modificar.

    La página que se abra dependerá del tipo de recurso. Por ejemplo, si el recurso es un pool de conexiones JDBC, se abrirá la página Editar Pool de Conexiones JDBC.
