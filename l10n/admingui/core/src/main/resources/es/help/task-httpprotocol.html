Para configurar el protocolo HTTP

  1. En el árbol de navegación, expanda el nodo Configuración.
  2. En el nodo Configuración, seleccione el nodo Servicio HTTP.
  3. En la página Servicio HTTP, haga clic en la ficha Protocolo HTTP.

    Se abrirá la página Protocolo HTTP.

  4. En el campo Versión, introduzca la versión del protocolo HTTP que se va a utilizar.

    Los valores aceptados son HTTP/1.0 y HTTP/1.1. El valor predeterminado es HTTP/1.1.

  5. Marque la casilla de verificación Búsqueda de DNS habilitada para habilitar la búsqueda de la entrada DNS del cliente.

    Esta opción se encuentra deshabilitada de forma predeterminada.

  6. Anule la selección de la casilla de verificación SSL habilitado para deshabilitar la seguridad del servidor globalmente.

    Seleccione esta casilla de verificación para utilizar SSL para cualquier escucha que tenga la seguridad habilitada. Esta opción está habilitada por defecto.

  7. En el campo Tipo obligado, introduzca el tipo de respuesta que se debe utilizar si no hay ninguna asignación MIME disponible que coincida con la extensión.

    El valor predeterminado es text/html; charset=iso-8859-1.

  8. En el campo Tipo predeterminado, introduzca el tipo de respuesta predeterminado.

    El valor predeterminado es text/html; charset=iso-8859-1. El valor es una cadena separada por punto y coma que incluye el tipo de contenido, la codificación, el idioma y el conjunto de caracteres.

  9. Haga clic en "Guardar".

    Consejo – Para volver a los valores de configuración predeterminados, haga clic en el botón Cargar predeterminados y, a continuación, en Guardar.
