Providing Key Value Pair Example

  • java.util.HashMap keyValues = new java.util.HashMap( );

  • keyValues.put('RequestId', '1111' );

  • keyValues.put('TransactionId', '2222' );

  • jtsLogger.log(Level.FINE, <Message Key>, keyValues);

  • or

  • Object[] params = {param1, keyValues };

  • jtsLogger.log(Level.FINE, <Message Key>, params);

  • will generate the following log

  • [#|2003-07-28T12:54:43.453-0700|FINE|javax.enterprise.resource.jts|RequestId=1111;TransactionId=2222;_ThreadId=77|<A Log Message>|#]
