감사 로깅을 활성화하거나 비활성화하는 방법

  1. 탐색 트리에서 구성 노드를 확장합니다.
  2. 구성할 인스턴스를 선택합니다.
    • 특정 인스턴스를 구성하려면 해당 인스턴스 구성 노드를 확장합니다. 예를 들어, 기본 인스턴스 server의 경우 server-config 노드를 확장합니다.
    • 모든 인스턴스의 기본 설정을 구성하려면 default-config 노드를 확장합니다.
  3. 구성 노드에서 보안 노드를 선택합니다.

    보안 페이지가 열립니다.

  4. 감사 로깅을 활성화하려면 감사 로깅 활성화 확인란을 선택합니다.

    감사 로깅을 비활성화하려면 감사 로깅 활성화 확인란을 선택 취소합니다.

    이 옵션을 선택하면 감사 모듈이 로드되고 Enterprise Server의 감사 라이브러리가 감사 지점에서 이 감사 모듈을 호출합니다.

  5. 감사 로깅을 활성화한 경우에는 활성 감사 모듈을 설정하는 방법에서 설명한 대로 기본 감사 모듈을 지정합니다.
  6. 저장을 누릅니다.
  7. 콘솔에 다시 시작해야 함이 표시되면 Enterprise Server를 다시 시작합니다.
참조

  • 감사 모듈을 만드는 방법

  • 감사 모듈을 편집하는 방법

  • 감사 모듈을 삭제하는 방법

  • 활성 감사 모듈을 설정하는 방법

  • 기본 감사 모듈을 사용하는 방법
