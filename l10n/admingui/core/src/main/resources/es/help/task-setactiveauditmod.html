Cómo establecer el módulo de auditoría activo

Antes de empezar

Para especificar el módulo de auditoría que utilizará el servidor, primero active el registro de auditoría tal y como se describe en Cómo habilitar o deshabilitar el registro de auditoría.

  1. En el campo Módulos de auditoría, seleccione el nombre del módulo de auditoría que utilizará el servidor.

    El valor predeterminado es default. Asegúrese de que este módulo de auditoría tiene el valor de la propiedad auditOn establecido en true tal y como se describe en Cómo utilizar el módulo de auditoría predeterminado.

  2. Haga clic en "Guardar".
  3. Reinicie Servidor empresarial si aparece el mensaje Reinicio necesario en la consola.
