更改管理领域中用户的密码

要更改当前管理员帐户的密码，请参见更改管理员密码。如果 admin-realm 有多个用户，您还可以更改其他用户的密码。

对管理员密码进行加密可以增强应用程序的安全性。有关安全性步骤的更多详细信息，请参见《Sun GlassFish Enterprise Server v3 Administration Guide》中的第 11  章 "Administering System Security"。

  1. 在导航树中，展开“配置”节点。
  2. 展开“安全性”节点。
  3. 展开“领域”节点。
  4. 选择 "admin-realm" 节点。

    将打开“编辑领域”页面。

  5. 在“编辑领域”页面上，单击“管理用户”按钮。

    将打开“文件用户”页面。

  6. 在“文件用户”页面上，单击要为其添加或更改密码的用户。

    将打开“编辑 File 领域用户”页面。

  7. 在“新密码”和“确认新密码”字段中，键入新的密码。
  8. 单击“保存”。
