To Manage the Lifecycle States of a Service Assembly

  1. Select the Service Assemblies whose lifecycle state you want to manage.
  2. From the Operations drop-down list, select the desired lifecycle state. Typically, you manage lifecycle states in this order:

    • Start: If a Service Assembly has been shutdown or stopped, select Start to start or restart it. Check the State column to know the current lifecycle state of the Service Assembly.

    • Stop: Select Stop only if the Service Assembly has been started. See the component's documentation for why and when to use the stop command.

    • Shutdown: Select Shutdown before you undeploy a Service Assembly.
