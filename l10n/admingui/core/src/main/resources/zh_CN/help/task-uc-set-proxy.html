为访问网络系统信息库配置网络代理设置

网络代理设置确定 Enterprise Server 连接到 Internet 的方式。要使更新工具可以通过 Internet 访问网络系统信息库，请配置您的网络代理设置。

  1. 在导航树中，选择“更新工具”节点。

    将显示安装的组件页面。

  2. 在“安装的组件”页面上，单击“配置”选项卡。

    将显示更新工具配置页面。

  3. 在“更新工具配置”页面，指定运行 Enterprise Server 的计算机连接到 Internet 的方式。
    • 如果此计算机直接连接到 Internet，请选择“直接连接”选项。
    • 如果此计算机通过代理连接到 Internet，请选择“通过代理服务器”选项。
  4. 如果运行 Enterprise Server 的计算机通过代理连接到 Internet，请提供代理的主机名和端口号。

    如果运行 Enterprise Server 的计算机直接连接到 Internet，则忽略此步骤。

    运行 Enterprise Server 的计算机上可能正在运行 Web 浏览器。在这种情况下，可以通过 Web 浏览器获取代理的主机名和端口号。否则，请您的系统管理员提供这些信息。

  5. 单击“保存”。
