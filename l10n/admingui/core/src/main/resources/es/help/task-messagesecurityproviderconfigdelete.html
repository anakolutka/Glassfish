Eliminación de la configuración del proveedor de seguridad de mensajes

  1. En el árbol de navegación, expanda el nodo Configuración.
  2. Expanda el nodo Seguridad.
  3. Seleccione el nodo Seguridad de mensajes

    Se abrirá la página Configuraciones de seguridad de mensajes.

  4. En la página Configuraciones de seguridad de mensajes, seleccione SOAP.

    Se abrirá la página Editar configuración de seguridad de mensajes.

  5. En la página Configuración de seguridad de mensajes, haga clic en la ficha Proveedores.

    Se abrirá la página Editar configuración del proveedor.

  6. En la página Editar configuración del proveedor, seleccione la casilla de verificación que aparece a la izquierda de la configuración del proveedor que está eliminando.
  7. Haga clic en Delete.
