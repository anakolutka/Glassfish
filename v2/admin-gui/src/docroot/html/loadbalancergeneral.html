To Change Load Balancer Device Settings

  1. In the tree component, select the HTTP Load Balancers node.
  2. In the right pane, click the load balancer that you want to edit.

    The Load Balancer Device Settings page appears.

  3. To communicate changes in the load balancer automatically to the web server or other load balancing device, click the Enable checkbox.

    The device settings must be specified and the connection working for changes to be automatically applied.

  4. To set up and test the connection to the load balancer device (for example, the Sun Java System Web Server:
    1. Enter the hostname of the web server or other load balancer device.
    2. Enter the port used to communicate with the load balancing device. It must be SSL-enabled.
    3. Enter the proxy host used for outbound HTTP.
    4. Enter the proxy port used for outbound HTTP.
    5. Click Test Connection to test the connection to the load balancer device.
  5. To add, change or delete properties:
    1. To add a property, click Add Property and fill in the name and value.
    2. To delete a property, click the checkbox next to the property name and click Delete Properties.
  6. Click Save.
