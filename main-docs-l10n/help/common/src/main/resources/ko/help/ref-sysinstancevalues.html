인스턴스 값

[인스턴스 값] 페이지에서는 선택한 명명된 구성을 참조하는 GlassFish Server 인스턴스에 대한 구성 시스템 속성의 값을 편집할 수 있습니다.

[인스턴스 값] 페이지에는 속성이 설정된 선택한 구성 요소를 참조하는 인스턴스의 목록이 표시됩니다.

인스턴스 이름

인스턴스가 생성될 때 인스턴스에 지정된 이름입니다. 이 이름을 누르면 인스턴스에 대한 [일반 정보] 페이지가 열립니다.

클러스터 이름

인스턴스가 멤버인 클러스터의 이름(있는 경우)입니다. 이 필드는 읽기 전용입니다.

기본값

인스턴스가 참조하는 명명된 구성의 속성에 대해 설정된 값입니다. 이 필드는 읽기 전용입니다.

대체 값

선택된 인스턴스에 대해 설정된 속성의 값입니다. 이 값은 기본값을 대체합니다.
