Valores de instancia

Utilice la página Valores de instancia para editar los valores de una propiedad de configuración del sistema para instancias de GlassFish Server que hagan referencia a la configuración con nombre seleccionada.

La página Valores de instancia muestra una lista de instancias que hacen referencia a la configuración seleccionada, en la que se define la propiedad.

Nombre de instancia

Nombre que se asignó a la instancia en el momento de su creación. Al hacer clic en el nombre se abre la página Información general para la instancia.

Nombre de cluster

Nombre del cluster del que la instancia es miembro, si lo hubiera. Este campo es de sólo lectura.

Valor por defecto

Valor que se define para la propiedad en la configuración especificada a la que hace referencia la instancia. Este campo es de sólo lectura.

Valor de sustitución

Valor de la propiedad que se ha definido para la instancia seleccionada. Este valor sustituye el valor por defecto.
