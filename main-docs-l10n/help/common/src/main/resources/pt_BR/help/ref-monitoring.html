Monitoramento

Use a página Monitoramento para configurar o monitoramento e exibir dados de monitoramento para instâncias do servidor clusterizadas e não clusterizadas.

Para cada instância do servidor, as seguintes informações são fornecidas.

Nome da Instância

O nome da instância do servidor. Clique no nome da instância para ir para a página Informações Gerais para a instância.

Nome do Cluster

O nome do cluster ao qual a instância do serviço pertence, se for uma instância clusterizada. Clique no nome do cluster para ir para a página Informações Gerais para o cluster.

Ação

Links que permitem executar as seguintes operações nas instâncias do servidor.

Configurar Monitoramento

Exibe a página de Serviço de Monitoramento para a configuração do cluster ou da instância standalone.

Exibir Dados de Monitoramento

Links que permitem exibir os seguintes tipos de dados de monitoramento para instâncias do servidor.

Aplicação

Exibe a página Monitoramento da Aplicação para a instância do servidor.

Servidor

Exibe a página de Monitoramento de Servidor para a instância do servidor.

Recursos

Exibe a página de Monitoramento de Recursos para a instância do servidor.
