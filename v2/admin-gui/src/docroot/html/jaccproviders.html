JACC Providers

JACC (Java Authorization Contract for Containers) is part of the Java 1.4 specification that defines an interface for pluggable authorization providers. This enables the administrator to set up third-party plug in modules to perform authorization. By default, the Application Server provides a simple, JACC-compliant file-based authorization engine.

Use Admin Console:

  • To create a JACC provider

  • To edit a JACC provider

  • To delete a JACC provider

  • To set the active JACC provider
