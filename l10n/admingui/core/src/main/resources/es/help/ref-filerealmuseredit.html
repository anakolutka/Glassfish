Editar usuario de dominio de archivo

Utilice la página Editar usuario de dominio de archivo para modificar cualquier privilegio de acceso del usuario en el dominio actual.

Nombre de dominio

Campo de sólo lectura que muestra el nombre del dominio seleccionado actualmente.

Id. de usuario

Campo de sólo lectura que muestra el nombre de la cuenta que está editando.

Lista de grupos

Grupos al que pertenecerá este usuario. Si introduce varios grupos, sepárelos con comas.

Contraseña nueva

Contraseña nueva para este usuario.

Contraseña nueva (confirmar)

Contraseña nueva para este usuario.
