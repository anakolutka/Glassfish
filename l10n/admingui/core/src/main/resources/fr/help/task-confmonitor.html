Configurer le contrôle du service

  1. Dans l'arborescence de navigation, développez le nœud Configuration.
  2. Sous le nœud Configuration, sélectionnez le nœud Contrôle.

    La page Service de contrôle s'affiche.

  3. Dans le tableau, effectuez l'une des actions suivantes.
    • Dans la colonne Niveau de contrôle, sélectionnez HAUT ou BAS dans la liste déroulante pour un ou plusieurs composants de la colonne Composant pour activer le contrôle de ces composants.
    • Cochez les cases en regard d'un ou de plusieurs composants, puis sélectionnez HAUT ou BAS dans la liste déroulante Niveau au sommet du tableau et cliquez sur le bouton Modifier le niveau pour définir le niveau de contrôle à cette valeur pour tous ces composants.

    Dans cette version, il n'existe aucune différence entre les options BAS et HAUT. Pour désactiver le contrôle d'un ou de plusieurs composants, sélectionnez DÉSACTIVÉ. La valeur par défaut est OFF.

  4. Cliquez sur Enregistrer.
