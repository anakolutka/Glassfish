Hosts JMS

Utilice la página Hosts JMS para configurar los hosts del servicio JMS.

Para cada host, se muestra la siguiente información.

Nombre

Nombre del host JMS.

Host

Nombre o dirección IP (Internet Protocol) del sistema donde el host JMS se está ejecutando.

Puerto

Número de puerto del servicio JMS en el host.

La tabla de hosts también contiene las siguientes opciones.

Nuevo

Botón para crear un nuevo host JMS.

Suprimir

Botón para suprimir uno o más hosts JMS seleccionados.
