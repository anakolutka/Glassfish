Modifier le fournisseur JACC

Utilisez la page Modifier le fournisseur JACC pour modifier un fournisseur JACC.

La page Modifier le fournisseur JACC contient les options suivantes.

Nom

Nom du fournisseur JACC. Ce champ est en lecture seule. Vous ne pouvez spécifier le nom qu'à la création d'un nouveau fournisseur JACC.

Configuration de stratégies

Nom de la classe qui implémente la fabrique de configuration de stratégies.

Fournisseur de stratégies

Nom de la classe qui implémente la fabrique de stratégies.

Autres propriétés

Autres propriétés du fournisseur JACC. Pour obtenir une description des autres propriétés disponibles pour les fournisseurs JACC, consultez la rubrique Propriétés propres aux fournisseurs JACC.
