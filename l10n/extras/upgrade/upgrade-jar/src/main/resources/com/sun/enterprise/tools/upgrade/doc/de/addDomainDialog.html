Dialogfenster "Domäne hinzufügen"

Wählen Sie mithilfe des Dialogfensters "Domäne hinzufügen" den Namen der Domäne und die Passwörter des Quell- und Zielzertifikats.

  1. Klicken Sie auf das Einblendmenü der Quelldomäne und wählen Sie die Domäne, die die zu migrierenden Sicherheitszertifikate enthält.
  2. Geben Sie im Passwortfeld die erforderlichen Passwörter ein. Je nach Quell- und Zielinstallation müssen entweder zwei oder drei Passwörter eingegeben werden:
    • Geben Sie im Quell-NSS- und/oder Ziel-NSS-Feld die Passwörter für die NSS-Zertifikatdateien ein, die für die NSS-Vertrauensdatenbank verwendet werden.
    • Geben Sie im Feld "Benutzerschlüsseldatenbank" das Passwort für die Quell-Benutzerschlüsseldatenbank ein, die für die JSSE-Datei keystore.jks verwendet wird, die Benutzerschlüsselpaarzertifikate enthält.
    • Geben Sie im Feld "Zertifikatausstellerdatenbank" das Passwort für die Zertifikats-Berechtigungsdatenbank ein, das für die JSSE-Datei cacerts.jks verwendet wird, die ZA-Zertifikate enthält.
  3. Klicken Sie auf "OK", um den Vorgang abzuschließen. Das Fenster "Sicherheitszertifikate übertragen" wird geöffnet und zeigt die Domäne an, die im Fenster "Domains with Certificates to be Transferred" ("Domänen mit zu übertragenden Zertifikaten") hinzugefügt wurde.
