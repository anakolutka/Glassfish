SSL을 사용하도록 HTTP 수신기를 구성하는 방법

SSL을 사용하도록 HTTP 수신기를 구성하면 Enterprise Server에서 특정 포트를 서버 인증서에 바인딩할 수 있습니다.

  1. 탐색 트리에서 구성 노드를 선택합니다.
  2. 구성 노드에서 HTTP 서비스 노드를 확장합니다.
  3. HTTP 서비스 노드에서 HTTP 수신기 노드를 선택합니다.
  4. HTTP 수신기 페이지에서 구성할 수신기의 이름을 누릅니다.
  5. HTTP 수신기 편집 페이지에서 SSL 탭을 누릅니다.
  6. 클라이언트가 이 수신기를 사용할 때 서버에 자신을 인증하게 하려면 클라이언트 인증 확인란을 선택합니다.
  7. 인증서 별명 필드에서 기존 서버 키 쌍 및 인증서 이름을 입력합니다.
  8. 수신기에서 SSL3 보안 프로토콜을 활성화하려면 SSL3 사용 가능 확인란을 선택합니다.
  9. 수신기에서 SSL2 보안 프로토콜을 활성화하려면 SSL2 사용 가능 확인란을 선택합니다.
  10. 수신기에서 TLS 보안 프로토콜을 활성화하려면 TLS 사용 가능 확인란을 선택합니다.
  11. 암호 제품군 섹션에서 프로토콜에서 사용되는 사용 가능한 암호 제품군을 선택하고 추가 버튼을 누릅니다.

    모든 암호화 제품군을 활성화하려면 모두 추가 버튼을 누릅니다.

  12. 저장을 누릅니다.
  13. 서버를 다시 시작합니다.
