Modifica servizio esecutore pianificato gestito

Utilizzare la pagina Modifica servizio esecutore pianificato gestito per modificare una risorsa di servizio esecutore pianificato gestito.

La pagina Modifica servizio contesto contiene le opzioni riportate di seguito.

Carica impostazioni predefinite

Pulsante per ripristinare i valori predefiniti delle impostazioni che dispongono di valori predefiniti. Le impostazioni che non dispongono di valori predefiniti non vengono modificate.

Nome JNDI

Il nome univoco che identifica la risorsa. Il campo Nome JNDI è di sola lettura. È possibile specificare il nome JNDI solo quando si crea una nuova risorsa.

Informazioni contesto

Il contesto della risorsa: Classloader, JNDI, Sicurezza o Area di lavoro.

Stato

Lo stato della risorsa. La risorsa può essere abilitata o disabilitata. Lo stato predefinito è Abilitato.

Priorità thread

La priorità dei thread da creare per l'istanza di servizio esecutore pianificato gestito. L'impostazione predefinita è 5.

Task con tempi di esecuzione lunghi

XXX.

Sospendi dopo

Il periodo di tempo in secondi durante il quale un task può essere eseguito prima di essere considerato sospeso.

Dimensione di base

Il numero di thread da gestire in un pool di thread. L'impostazione predefinita è 0.

Dimensione massima pool

Il numero massimo di thread che un pool può contenere. Il valore predefinito è 2147483647.

Controllo attività

Il periodo di tempo in secondi durante il quale i thread possono rimanere inattivi quando il numero di thread è maggiore della dimensione di base. L'impostazione predefinita è 60.

Durata thread

Il periodo di tempo in secondi durante il quale ogni thread rimane nel pool di thread prima di essere rimosso, indipendentemente dal fatto che il numero di thread sia maggiore della dimensione di base o che il thread sia inattivo.

Capacità coda di task

La dimensione della coda che contiene i task sottomessi in attesa di esecuzione. L'impostazione predefinita è 2147483647.

Descrizione

La descrizione di questa risorsa.

Proprietà aggiuntive

Proprietà aggiuntive per la risorsa. GlassFish Server non definisce alcuna proprietà aggiuntiva per le risorse dei servizi esecutore pianificati gestiti.
