SSL-Zielgerät

Das SSL-Zielgerät verbindet ein Serverzertifikat mit dem HTTP-Zielgerät-Port. Die SSL-Einstellungen werden beim Erstellen eines neuen HTTP-Zielgeräts über die Seite Neues HTTP-Zielgerät (Schritt 2 von 2) definiert. SSL-Einstellungen werden darüber hinaus über die Seite "SSL-Zielgerät" bearbeitet.

Client-Authentifizierung

Aktivieren Sie diese Option, um bei Verwendung dieses Zielgeräts die Client-Authentifizierung gegenüber dem Server zu erzwingen.

Zertifikatspseudonym

Der Name eines vorhandenen Serverschlüsselpaars und Zertifikats.

SSL3

Wenn Sie diese Option aktivieren, wird das SSL3-Sicherheitsprotokoll auf dem Zielgerät verwendet.

SSL2

Wenn Sie diese Option aktivieren, wird das SSL2-Sicherheitsprotokoll auf dem Zielgerät verwendet.

TLS

Wenn Sie diese Option aktivieren, wird das TLS-Sicherheitsprotokoll auf dem Zielgerät verwendet.

Allgemeine Ciphersuites

Die vom/von den Protokoll(en) verwendeten Ciphersuites.

Flüchtige Dittie-Hellman-Ciphersuites

Die vom/ von den Protokoll(en) verwendeten flüchtigen Ciphersuites.
