Bereiche

Enterprise Server verfügt über drei vorkonfigurierte Bereiche: file, certificate und admin-realm. Darüber hinaus können die Bereiche ldap, JDBC, solaris sowie benutzerdefinierte Bereiche erstellt werden. Im Allgemeinen ist auf jedem Server ein Bereichstyp vorhanden, Enterprise Server verfügt jedoch über zwei Dateibereiche: file und admin-realm. Dies sind zwei Bereiche desselben Typs, die für unterschiedliche Zwecke verwendet werden. Darüber hinaus kann für jeden virtuellen Server auf dem System eine separate Zertifikatdatenbank bereitgestellt werden.

Auf der Seite "Bereiche" wird eine Liste mit verfügbaren Bereichen angezeigt. Für jeden Bereich werden die folgenden Informationen zur Verfügung gestellt:

  • Name

  • Klassenname

Verwandte Aufgaben

  • So erstellen Sie einen Bereich

  • So bearbeiten Sie einen Bereich

  • So löschen Sie einen Bereich

  • So legen Sie den Standardbereich fest

  • So erstellen Sie einen benutzerdefinierten Bereich

Verwandte asadmin-Befehle

  • list-auth-realms(1)
