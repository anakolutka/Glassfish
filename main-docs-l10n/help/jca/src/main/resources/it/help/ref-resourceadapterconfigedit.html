Modifica configurazione Resource Adapter

Utilizzare la pagina Modifica configurazione Resource Adapter per modificare le impostazioni di una configurazione di Resource Adapter.

La pagina Modifica configurazione Resource Adapter contiene le opzioni riportate di seguito.

Nome Resource Adapter

Il nome del Resource Adapter che si sta configurando. Il campo del nome è di sola lettura. È possibile specificare un nome per il Resource Adapter solo quando si crea una nuova configurazione di Resource Adapter.

ID pool di thread

Il pool di thread da usare per la configurazione di Resource Adapter.

Ordine di distribuzione

L'ordine di distribuzione di Resource Adapter.

Proprietà aggiuntive

Proprietà aggiuntive per il Resource Adapter. Le proprietà disponibili sono i nomi dei metodi setter della classe a cui fa riferimento l'elemento resourceadapter-class nel file ra.xml, che definisce il nome della classe del componente JavaBeans del Resource Adapter. Qualsiasi modifica apportata alle proprietà qui definite sostituisce i valori predefiniti presenti nel file ra.xml.
