卸载组件

卸载组件涉及到删除以下项：

  • 在组件的初始安装时引入的所有文件和目录

  • 在初始安装后更新组件时引入的所有文件和目录

  1. 在导航树中，选择“更新工具”节点。

    将显示安装的组件页面。

  2. 在安装的组件表中，选择您要卸载的组件。
    • 要选择单个组件，请选择该组件名称旁边的复选框。
    • 要选择所有组件，单击表头中的“全选”图标。
    • 要取消选择所有组件，请单击表头中的“取消全选”图标。
  3. 单击“删除”。

    会出现一个对话框，要求您确认是否应该卸载所选组件。

  4. 在此对话框中，单击“确定”。

    Enterprise Server 确认卸载已完成。各组件将从安装的组件表中删除，并添加到可用的附加软件页面中的可用附加组件表中。
