Protocolli

Utilizzare la pagina Protocolli per configurare i protocolli di rete.

Per ogni protocollo vengono fornite le informazioni riportate di seguito.

Nome

Il nome del protocollo.

Sicurezza abilitata

L'impostazione è True se la sicurezza è abilitata per il protocollo oppure è False se la sicurezza non è disabilitata per il protocollo.

La tabella Protocolli contiene inoltre le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare un nuovo protocollo.

Elimina

Pulsante che consente di eliminare una o più protocolli selezionati.
