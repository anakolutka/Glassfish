Per modificare una risorsa connettore

  1. Espandere il nodo Risorse nell'albero di navigazione.

  2. Espandere il nodo Connettori.

  3. Nel nodo Connettori selezionare il nodo Risorse connettore.

    Viene visualizzata la pagina Risorse connettore.

  4. Nella pagina Risorse connettore fare clic sul nome della risorsa da modificare.

    Viene visualizzata la pagina Modifica risorsa connettore.

  5. Nell'elenco a discesa Nome pool selezionare il connection pool del connettore a cui appartiene la nuova risorsa connettore.

  6. Nel campo Ordine di distribuzione digitare un numero per specificare l'ordine di distribuzione della risorsa connettore.

    Le risorse con numero inferiore vengono caricate per prime all'avvio del server. Il valore predefinito è 100.

  7. Nel campo Descrizione digitare la descrizione della risorsa connettore.

  8. Nella scheda Destinazione selezionare le destinazioni necessarie e fare clic su Abilita o Disabilita.

    Per aggiungere o rimuovere le destinazioni dalla lista, fare clic su Gestisci destinazioni.

    Tornare alla pagina Modifica risorsa connettore facendo clic sulla scheda Generale.

  9. Selezionare la casella di controllo Abilitato dell'opzione Stato per abilitare la risorsa connettore in fase di esecuzione.

    Questa opzione è abilitata per impostazione predefinita.

  10. Nella sezione Proprietà aggiuntive specificare le proprietà aggiuntive per la risorsa connettore.

    Le proprietà disponibili dipendono dal Resource Adapter per il connection pool del connettore.

    • Per aggiungere una proprietà, fare clic sul pulsante Aggiungi proprietà. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Nome e il valore della proprietà nel campo Valore.

    • Per modificare una proprietà, è necessario modificare il valore del campo Valore della proprietà.

    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Nome corrispondente alla proprietà da eliminare, quindi fare clic sul pulsante Elimina proprietà.

  11. Fare clic su Salva.

    Suggerimento:

    Per ripristinare le impostazioni predefinite, fare clic sul pulsante Carica impostazioni predefinite, quindi su Salva.
