Écran Transférer les certificats de sécurité

Utilisez l'écran Transférer les certificats de sécurité pour indiquer les domaines qui contiennent un (ou plusieurs) jeu de certificats de sécurité qui doivent être migrés vers Sun Java Systems Application Server 9.1 :

  1. Appuyez sur le bouton Ajouter pour afficher la boîte de dialogue Ajouter un domaine, où le nom du domaine et les mots de passe pour les fichiers de certificat sont entrés.

    Le panneau Domaines contenant des certificats à transférer affiche les domaines ajoutés.

  2. Répétez la procédure jusqu'à ce que tous les domaines qui contiennent les certificats devant être transférés soient ajoutés.
  3. Pour supprimer un domaine déjà ajouté, sélectionnez le domaine à supprimer et appuyez sur le bouton Supprimer le domaine.
  4. Une fois la procédure terminée, appuyez sur le bouton Suivant.

    L'écran Résultats de la mise à niveau, qui indique le statut de la mise à niveau, s'affiche.
