配置监视服务

  1. 在导航树中，展开“配置”节点。
  2. 在“配置”节点下，选择“监视”节点。

    此时将打开“监视服务”页面。

  3. 从“HTTP 服务”下拉式列表中，选择“低”或“高”来启用 HTTP 服务监视。

    在此发行版中，“低”和“高”选项所提供的监视级别之间没有差别。要禁用 HTTP 服务监视，请选择“关闭”。默认值为“关闭”。

  4. 从“Web 容器”下拉式列表中，选择“低”或“高”来启用 Web 容器监视。

    在此发行版中，“低”和“高”选项所提供的监视级别之间没有差别。要禁用 Web 容器监视，请选择“关闭”。默认值为“关闭”。

  5. 从 "JVM" 下拉式列表中，选择“低”或“高”启用 JVM 监视。

    在此发行版中，“低”和“高”选项所提供的监视级别之间没有差别。要禁用 JVM 监视，请选择“关闭”。默认值为“关闭”。

  6. 从“线程池”下拉式列表中，选择“低”或“高”启用线程池监视。

    在此发行版中，“低”和“高”选项所提供的监视级别之间没有差别。要禁用线程池监视，请选择“关闭”。默认值为“关闭”。

  7. 单击“保存”。
