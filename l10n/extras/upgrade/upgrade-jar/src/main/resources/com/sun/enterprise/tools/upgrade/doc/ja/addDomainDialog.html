「ドメインを追加」ダイアログ

「ドメインを追加」ダイアログを使用して、ドメイン名とソースおよびターゲットの証明書パスワードを選択します。

  1. 「ソースドメイン」プルダウンをクリックし、移行するセキュリティー証明書を含むドメインを選択します。
  2. 「パスワード」パネルで、要求されるパスワードを入力します。ソースおよびターゲットのインストール環境のタイプに応じて、2 つまたは 3 つのパスワードの入力が必要となります。
    • 「ソース NSS」フィールドおよび「ターゲット NSS」フィールドに、NSS の信頼できるデータベースで使用される、NSS 証明書ファイルのパスワードを入力します。
    • 「ユーザーキーデータベース」フィールドに、ソースユーザーキーデータベースのパスワードを入力します。このパスワードは、ユーザーキーペア証明書を含む JSSE の keystore.jks ファイルで使用されます。
    • 「認証局データベース」フィールドに、認証局データベースのパスワードを入力します。このパスワードは、認証局の証明書を含む JSEE の cacerts.jks ファイルで使用されます。
  3. 入力が完了したら、「了解」ボタンを押します。「セキュリティー証明書を転送」画面が表示され、追加したドメインが「転送する証明書付きのドメイン」パネルに示されます。
