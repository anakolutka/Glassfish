Servicio de supervisión

Utilice la página Servicio de supervisión para configurar opciones de supervisión de servicios o componentes del servidor individual.

Debe desplegar una aplicación, componente o servicio antes de configurar su supervisión.

La página Servicio de supervisión contiene las siguientes opciones.

Nombre de Configuración

Nombre de la configuración a la que se aplican los valores de esta página. Este campo es de sólo lectura.

Servicio de supervisión

Si la casilla de control de activación de servicio de supervisión está activada, se activará la supervisión para GlassFish Server. Esta opción se encuentra activada por defecto.

MBeans de supervisión

Si la casilla de control de activación de MBeans de supervisión está activada, todos los MBeans necesarios para la supervisión se desplegarán en GlassFish Server. Esta opción se encuentra activada por defecto.

Si la casilla de control no está activada, no podrá ver los datos de supervisión en la Consola de administración incluso si la casilla de control Servicio de supervisión sí está activada. En su lugar, debe utilizar la línea de comandos para ver los datos de supervisión. Si desactiva la casilla de control, la huella de memoria de GlassFish Server se reduce.

DTrace

Si la casilla de control DTrace activado está activada y dispone de supervisión de Oracle Enterprise Manager DTrace, puede usar la utilidad Solaris DTrace. Esta opción se activada por defecto.

Para cada componente disponible, se proporciona la siguiente información.

Componente

Nombre del componente de GlassFish Server.

Nivel de supervisión

Nivel de supervisión determinado por el componente. Para cada componente están disponibles los niveles BAJO, ALTO Y DESACTIVADO. El nivel por defecto es DESACTIVADO. No existe diferencia alguna entre los niveles de supervisión que ofrecen las opciones BAJO y ALTO de esta versión.

La tabla Configuración de nivel de componentes también contiene las siguientes opciones.

Cambiar Nivel

Botón para cambiar el nivel de supervisión de uno o más componentes seleccionados.

Nivel

Lista desplegable en la que puede seleccionar un nivel de supervisión para que se aplique a todos los componentes seleccionados.
