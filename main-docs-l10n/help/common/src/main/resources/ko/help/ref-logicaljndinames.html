논리적 JNDI 이름

리소스를 생성할 때 리소스에 대해 고유한 JNDI 이름을 지정합니다. 응용 프로그램은 이 이름을 사용하여 리소스에 액세스합니다.

Java EE 표준에서는 특정 기본 리소스가 응용 프로그램에 제공되도록 지정하고, 이러한 기본 리소스에 대해 특정 JNDI 이름을 정의합니다. GlassFish Server는 논리적 JNDI 이름을 통해 이러한 이름을 제공합니다. 논리적 JNDI 이름은 다음과 같이 Java EE 표준 JNDI 이름을 특정 GlassFish Server 리소스에 매핑합니다.

java:comp/DefaultContextService

이 Java EE 표준 이름은 concurrent/__defaultContextService 리소스에 매핑됩니다.

java:comp/DefaultManagedThreadFactory

이 Java EE 표준 이름은 concurrent/__defaultManagedThreadFactory 리소스에 매핑됩니다.

java:comp/DefaultManagedExecutorService

이 Java EE 표준 이름은 concurrent/__defaultManagedExecutorService 리소스에 매핑됩니다.

java:comp/DefaultManagedScheduledExecutorService

이 Java EE 표준 이름은 concurrent/__defaultManagedScheduledExecutorService 리소스에 매핑됩니다.

java:comp/DefaultDataSource

이 Java EE 표준 이름은 jdbc/__default 리소스에 매핑됩니다.

java:comp/DefaultJMSConnectionFactory

이 Java EE 표준 이름은 jms/__defaultConnectionFactory 리소스에 매핑됩니다.
