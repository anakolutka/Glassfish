Para Configurar as Definições de Caminho de JVM

  1. Na árvore de navegação, selecione o nó Configuração.

  2. Selecione a configuração que está sendo alterada.

  3. No nó Configuração, selecione o nó Definições de JVM.

    A página Definições Gerais de JVM será aberta.

  4. Na página Definições Gerais de JVM, clique na guia Definições de Caminho.

    A página Definições do Caminho JVM será aberta.

  5. No campo Prefixo do Caminho da Biblioteca Nativa, digite os caminhos completos dos arquivos JAR a serem colocados no início do caminho da biblioteca nativa.

    Para acrescentar vários arquivos, digite um caminho por linha.

  6. No campo Sufixo do Caminho da Biblioteca Nativa, digite os caminhos completos dos arquivos JAR a serem acrescentados ao caminho da biblioteca nativa.

    Para acrescentar vários arquivos, digite um caminho por linha.

  7. Clique em Salvar.

  8. Reinicie o servidor.
