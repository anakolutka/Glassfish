Para suprimir la configuración del adaptador de recursos

  1. En el árbol de navegación, amplíe el nodo Recursos.

  2. En el nodo Recursos, seleccione el nodo Configuración de adaptadores de recursos.

    Se abrirá la página Configuración del adaptador de recursos.

  3. En la página Configuraciones del adaptador de recursos, active la casilla de control situada junto al nombre de la configuración del adaptador de recursos que está suprimiendo

  4. Haga clic en Suprimir.
