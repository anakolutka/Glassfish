Visualisation des exécutions de traitement batch

  1. Dans l'arborescence de navigation, sélectionnez le cluster ou l'instance de serveur GlassFish Server dont vous voulez visualiser les exécutions de traitement batch.

    La page Informations générales apparaît.

  2. Sélectionnez l'onglet Batch.

    La page Exécutions de traitement batch apparaît.

  3. Sur la page Exécutions de traitement batch, cliquez sur un ID d'exécution pour visualiser les détails relatifs à cette exécution.

    La page Détails d'exécution de traitement batch apparaît.

  4. Pour visualiser les détails sur les étapes de l'exécution de traitement batch, cliquez sur l'onglet Etapes d'exécution.

    La page Etapes d'exécution de traitement batch apparaît.

  5. Cliquez sur le bouton Précédent pour revenir à la page Exécutions de traitement batch.
