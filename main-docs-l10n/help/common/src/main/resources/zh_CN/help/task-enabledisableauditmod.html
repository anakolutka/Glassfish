启用或禁用审计日志记录

  1. 在导航树中，展开“配置”节点。

  2. 选择要配置的实例：

    • 要配置特定的实例，请展开该实例的配置节点。例如，对于默认实例 server，请展开 server-config 节点。

    • 要配置所有实例的默认设置，请展开 default-config 节点。

  3. 在“配置”节点下，选择“安全性”节点。

    将打开“安全性”页。

  4. 在“安全性”页上，选中“启用审计日志记录”复选框以启用审计日志记录。

    要激活审计日志记录，还必须将活动审计模块的 auditOn 属性设置为真。

  5. 单击“保存”。
