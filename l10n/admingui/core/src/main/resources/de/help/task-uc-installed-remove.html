So deinstallieren Sie Komponenten

Zum Deinstallieren einer Komponente müssen folgende Elemente entfernt werden:

  • Alle Dateien und Verzeichnisse aus der ursprünglichen Installation der Komponente

  • Alle Dateien und Verzeichnisse aus Updates der Komponente nach der ursprünglichen Installation

  1. Wählen Sie in der Navigationsstruktur den Knoten "Update-Tool" aus.

    Die Seite "Verfügbare Add-Ons" wird geöffnet.

  2. Klicken Sie auf der Seite "Verfügbare Add-Ons" auf die Registerkarte "Installierte Komponenten".

    Die Seite "Installierte Komponenten" wird angezeigt.

  3. Wählen Sie in der Tabelle der installierten Komponenten die Komponenten aus, die Sie deinstallieren.
    • Um eine einzelne Komponente auszuwählen, aktivieren Sie das Kontrollkästchen neben dem Namen der Komponente.
    • Um alle Komponenten auszuwählen, klicken Sie in der Kopfzeile der Tabelle auf das Symbol "Alle auswählen".
    • Um die Auswahl aller Komponenten aufzuheben, klicken Sie in der Kopfzeile der Tabelle auf das Symbol "Auswahl für alle aufheben".
  4. Klicken Sie auf "Löschen".

    Sie werden in einem Dialogfeld aufgefordert zu bestätigen, dass die ausgewählten Komponenten deinstalliert werden sollen.

  5. Klicken Sie in diesem Dialogfeld auf "OK".

    Enterprise Server bestätigt den Abschluss der Deinstallation. Die Komponenten werden aus der Tabelle der installierten Komponenten gelöscht und zur Tabelle der verfügbaren Add-Ons auf der Seite "Verfügbare Add-Ons" hinzugefügt.
