Atributos de dominio

Utilice esta página para establecer atributos de dominio avanzados de Servidor empresarial.

Raíz de aplicaciones

La ruta completa del directorio en el que se implementan las aplicaciones.

Raíz de registro

Identifica el lugar en el que se guardan los archivos de registro de la del servidor.

Configuración regional

Si se deja este campo en blanco, se utilizará la configuración regional predeterminada del host.

La configuración regional es un identificador que especifica una combinación concreta de idioma y región. Por ejemplo, la configuración regional para inglés de Estados Unidos es en_US y para japonés es ja_JP. Para utilizar una configuración local que no sea en idioma inglés, Application Server tiene que estar localizado, un proceso que incluye traducir el inglés a otro idioma.
