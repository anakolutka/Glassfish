配置 HTTP 服务保持活动

  1. 在导航树中，展开“配置”节点。
  2. 在“配置”节点下，选择“HTTP 服务”节点。
  3. 从“HTTP 服务”页面中，选择“保持活动”选项卡。

    此时将打开“保持活动”页面。

  4. 在“线程计数”字段中，键入要使用的保持活动线程的数量。

    默认值为 1。

  5. 在“最大连接数”字段中，键入要保持的持久连接的最大数量。

    默认值为 256。

  6. 在“超时”字段中，键入保持活动连接应保持打开状态的最大秒数。

    默认值为 30。

  7. 单击“保存”。

    提示 – 要返回到默认设置，请单击“装入默认值”按钮，然后单击“保存”。
