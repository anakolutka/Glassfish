So zeigen Sie allgemeine Informationen für Enterprise Server an

  1. Wählen Sie im Navigationsbaum den Knoten Enterprise Server aus.

    Die Seite "Allgemeine Informationen" wird geöffnet.

  2. Führen Sie auf der Seite "Allgemeine Informationen" einen der folgenden Schritte aus.
    • Klicken Sie auf die Schaltfläche Stop, um Enterprise Server anzuhalten.
    • Klicken Sie auf die Schaltfläche "Neustart", um Enterprise Server anzuhalten und neu zu starten.
    • Klicken Sie auf die Schaltfläche "Protokolldateien anzeigen", um das Serverprotokoll anzuzeigen.

      Einzelheiten finden Sie unter So zeigen Sie das Serverprotokoll an .

    • Klicken Sie auf JVM-Bericht, um ein separates Fenster zu öffnen, das es Ihnen ermöglicht, Berichte auf der Java Virtual Machine anzuzeigen, u. a. einen zusammenfassenden Bericht, Speichermanagement-Bericht, Klassenladebericht und einen aktuellen Thread-Pool.
