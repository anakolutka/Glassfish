[새 관리 대상 예약된 실행기 서비스] 페이지에는 다음 옵션이 포함되어 있습니다.

JNDI 이름

리소스를 식별하는 고유한 이름입니다.

컨텍스트 정보

리소스의 컨텍스트입니다(Classloader, JNDI, Security 또는 WorkArea).

상태

리소스의 상태입니다. 리소스는 사용 또는 사용 안함으로 설정할 수 있습니다. 기본값은 사용입니다.

스레드 우선순위

관리 대상 예약된 실행기 서비스 인스턴스에 대해 생성할 스레드의 우선순위입니다. 기본값은 5입니다.

오랫동안 실행 중인 작업

XXX.

작업 정지 간주 시간

정지 상태로 간주되기 전에 작업을 실행할 수 있는 시간(초)입니다.

코어 크기

최대 풀 크기

연결 유지

스레드 수명

스레드 수가 코어 크기보다 큰지 여부 또는 스레드가 유휴 상태인지 여부에 관계없이 비우기 전에 각 스레드가 스레드 풀에 남아 있을 수 있는 시간(초)입니다.

실행 대기 중인 대기열에 보류된 제출된 작업의 크기입니다. 기본값은 2147483647입니다.
