Pour afficher la page Forums GlassFish du Sun GlassFish Enterprise Server

La page Forums GlassFish fournit un ensemble de liens permettant d'accéder à des informations supplémentaires sur Sun GlassFish Enterprise Server, telles que des bulletins d'information, des articles, des conseils techniques, des forums de discussion et des blogs sur le sujet.

  1. Dans l'arborescence de navigation, sélectionnez le noeud Forums GlassFish.

    La page Forums GlassFish s'ouvre.

  2. Cliquez sur un lien pour ouvrir une page dans un onglet ou une autre fenêtre distincte du navigateur.
