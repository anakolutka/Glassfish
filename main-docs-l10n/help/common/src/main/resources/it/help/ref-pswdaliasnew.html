Nuovo alias password

Utilizzare la pagina Nuovo alias password per creare un nuovo alias password da utilizzare in un password file o nel file di configurazione del dominio.

La pagina Nuovo alias password contiene le opzioni riportate di seguito.

Nome alias

Il nome dell'alias. Questo nome deve essere univoco nel gruppo di alias password presenti nel dominio.

Password

La password che l'alias memorizza in forma cifrata.

Conferma password

La password che l'alias memorizza in forma cifrata. I valori Password e Conferma password vengono confrontati per assicurarsi che la password sia stata immessa senza errori di digitazione.
