Per modificare un'applicazione distribuita a un cluster

  1. Espandere il nodo Cluster nell'albero di navigazione.

  2. Selezionare il nodo per il cluster sotto il nodo Cluster espanso.

    Viene visualizzata la pagina Informazioni generali per il cluster.

  3. Nella pagina Informazioni generali fare clic sulla scheda Applicazioni.

    Viene visualizzata la pagina Applicazioni per il cluster.

  4. Nella tabella Applicazioni distribuite della pagina Applicazioni, fare clic sul nome dell'applicazione.

    Viene visualizzata la pagina Modifica applicazione.

  5. Nella pagina Modifica applicazione fornire le informazioni relative all'applicazione necessarie e salvare le modifiche.

    Per i dettagli, vedere l'argomento Modifica applicazione della Guida in linea.
