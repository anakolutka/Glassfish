Nueva asignación de seguridad de trabajo

Utilice la página Nueva asignación de seguridad de trabajo para crear una nueva asignación de seguridad de trabajo.

La página Nueva asignación de seguridad de trabajo contiene las siguientes opciones.

Nombre de asignación de seguridad

Nombre de la asignación de seguridad de trabajo.

Grupo

Si el botón de radio Asignación de Grupo está activado, el campo de texto debe especificar una asignación de cada grupo de usuarios EIS de backend al grupo de usuarios correspondiente definido en el dominio de seguridad de GlassFish Server. Utilice una lista separada por comas para especificar más de una asignación. Por ejemplo:

eis-group-1=server-group-1,eis-group-2=server-group-2
Principal

Si el botón de radio Asignación de Principal está activado, el campo de texto debe especificar una asignación de cada principal EIS de backend al principal correspondiente definido en el dominio de seguridad de GlassFish Server. Utilice una lista separada por comas para especificar más de una asignación. Por ejemplo:

eis-principal-1=server-principal-1,eis-principal-2=server-principal-2
Adaptador de recursos

Adaptador de recursos para el EIS.

Descripción

Descripción de la asignación de seguridad de trabajo.

Estado

Si esta opción está seleccionada, la asignación de seguridad de trabajo está activada. Esta opción se encuentra activada por defecto.
