Modifica host JMS

Utilizzare la pagina Modifica host JMS per modificare le impostazioni per un host JMS.

La pagina Modifica host JMS contiene le opzioni riportate di seguito.

Carica impostazioni predefinite

Pulsante per ripristinare i valori predefiniti delle impostazioni che dispongono di valori predefiniti. Le impostazioni che non dispongono di valori predefiniti non vengono modificate.

Nome configurazione

Il nome della configurazione a cui si applicano le impostazioni presenti in questa pagina. Questo campo è di sola lettura.

Nome

Il nome dell'host. Il campo Nome è di sola lettura. È possibile specificare il nome solo quando si crea un nuovo host JMS.

Host

Il nome o l'indirizzo IP (Internet Protocol) del sistema su cui verrà eseguito l'host JMS (localhost o il nome del sistema locale o remoto).

Porta

Il numero di porta del servizio JMS. Modificare questo campo solo se il servizio JMS da utilizzare è in esecuzione su una porta non predefinita. La porta predefinita è la porta numero 7676.

Nome utente amministratore e Password amministratore

Il nome utente e la password del broker Message Queue. Sono diversi dal nome utente e dalla password di GlassFish Server. Modificare questi campi solo se i valori del broker di Message Queue sono stati modificati tramite il comando as-install-parent/mq/bin/imqusermgr. I valori predefiniti sono admin e admin.
