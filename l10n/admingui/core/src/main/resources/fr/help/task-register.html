Enregistrer Sun GlassFish Enterprise Server avec Sun Connection

Pour enregistrer l'installation de Sun GlassFish Enterprise Server avec Sun Connection , vous devez disposer d'un compte en ligne Sun. Si vous n'avez pas encore de compte en ligne Sun, Enterprise Server vous permet d'en créer un lors de l'enregistrement.

Remarque – Si vous optez pour cette solution, vous êtes invitée à saisir uniquement les informations obligatoires. Après l'enregistrement de Enterprise Server, vous pouvez fournir des informations supplémentaires en mettant à jour votre compte Sun Online.

Avant de commencer

Si la machine sur laquelle Enterprise Server s'exécute se connecte à Internet par l'intermédiaire d'un proxy, procurez-vous le nom de l'hôte et son numéro de port. Si un navigateur Web s'exécute sur la même machine que Enterprise Server, vous pouvez obtenir ces informations dans ce navigateur. Sinon, contactez votre administrateur système.

  1. Dans l'arborescence de navigation, sélectionnez le nœud Enregistrement.

    La page Enregistrement du produit s'ouvre.

  2. Dans la page d'enregistrement du produit, saisissez les informations requises.
    • Si vous disposez déjà d'un compte en ligne Sun, procédez comme suit :
      1. Sélectionnez l'option Je possède déjà un compte en ligne Sun.
      2. Dans le champ Nom d'utilisateur, entrez le nom spécifié à la création de votre compte en ligne.
      3. Faites de même pour le champ Mot de passe.
    • Si vous ne disposez pas d'un compte en ligne Sun, procédez comme suit :
      1. Sélectionnez l'option Je ne possède pas de compte en ligne Sun.
      2. Dans le champ Adresse électronique, entrez votre adresse e-mail.

        Cette dernière sera le nom d'utilisateur de votre compte en ligne Sun.

      3. Dans le champ Mot de passe, entrez le mot de passe choisi pour accéder à votre compte.

        Celui-ci est sensible à la casse et doit comporter entre 6 et 10 caractères.

      4. Dans le champ Confirmation du mot de passe, entrez à nouveau le mot de passe choisi précédemment.
      5. Dans le champ Prénom, entrez votre prénom.
      6. Dans le champ Nom, entrez votre nom.
      7. À partir de la liste déroulante, sélectionnez votre pays ou territoire de résidence.
  3. Si la machine sur laquelle Enterprise Server s'exécute se connecte à Internet par l'intermédiaire d'un proxy, entrez le nom de l'hôte et son numéro de port.

    Si la machine sur laquelle Enterprise Server s'exécute se connecte directement à Internet, ignorez cette étape.

    Il est possible qu'un navigateur Web s'exécute sur la même machine que Enterprise Server. Dans ce cas, vous pouvez obtenir le nom d'hôte et le numéro de port du proxy dans ce navigateur. Sinon, contactez votre administrateur système.

  4. Cliquez sur S'enregistrer.

    Pour confirmer la fin de l'enregistrement, la page Support s'ouvre.
