Définir la valeur du délai d'attente de la session

  1. Dans l'arborescence de navigation, développez le nœud Configuration.
  2. Sous le nœud Configuration, sélectionnez le nœud Conteneur Web.

    La page Conteneur Web s'affiche.

  3. Dans la page Conteneur Web, cliquez sur l'onglet Propriétés des sessions.
  4. Dans le champ Délai d'attente de la session, saisissez le nombre de secondes durant lequel une session est valide.
  5. (Facultatif) Dans la section Autres propriétés, spécifiez d'autres propriétés.
  6. Cliquez sur Enregistrer.
