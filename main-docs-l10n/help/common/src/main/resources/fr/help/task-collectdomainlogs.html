Collecte et téléchargement de journaux de domaine

Vous pouvez collecter tous les fichiers journaux disponibles pour une instance ou un cluster GlassFish Server spécifié et les compiler en une seule archive ZIP.

Remarque:

Si vous spécifiez un cluster, l'archive ZIP inclut également le fichier journal pour le serveur d'administration de domaine (DAS).

  1. Dans l'arborescence de navigation, sélectionnez le noeud Domaine.

    La page Attributs de domaine apparaît.

  2. Sélectionnez l'onglet Journaux de domaine.

    La page Journaux de domaine apparaît.

    S'il existe des clusters, les listes déroulantes des clusters et des instances sont affichées. S'il n'existe pas de clusters, seule une liste déroulante des instances est affichée.

  3. Sélectionnez Cluster ou Instance.

  4. Sélectionnez le cluster ou l'instance dont vous collectez et téléchargez les fichiers journaux.

  5. Cliquez sur Collecter les journaux.

    Le mécanisme de téléchargement du navigateur vous invite à télécharger le fichier log-files-name.zip, où name est le nom du cluster ou de l'instance.
