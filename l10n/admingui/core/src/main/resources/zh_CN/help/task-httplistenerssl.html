配置 HTTP 侦听器以使用 SSL

将 HTTP 侦听器配置为使用 SSL 使 Enterprise Server 可以将特定的端口绑定到服务器证书。

  1. 在导航树中，展开“配置”节点。
  2. 在“配置”节点下，展开“HTTP 服务”节点。
  3. 在“HTTP 服务”节点下，选择“HTTP 侦听器”节点。
  4. 在“HTTP 侦听器”页面上，单击要配置的侦听器的名称。
  5. 在“编辑 HTTP 侦听器”页面上，单击 "SSL" 选项卡。
  6. 选中“客户机验证”复选框，以强制客户机在使用此侦听器时向服务器验证自身。
  7. 在“证书昵称”字段中，键入现有服务器密钥对和证书的名称。
  8. 选中“启用 SSL3”复选框可以在侦听器上启用 SSL3 安全协议。
  9. 选中“启用 SSL2”复选框可以在侦听器上启用 SSL2 安全协议。
  10. 选中“启用 TLS”复选框可以在侦听器上启用 TLS 安全协议。
  11. 在“密码组”部分，选择协议所使用的可用密码组，然后单击“添加”按钮。

    要启用所有密码组，请单击“全部添加”按钮。

  12. 单击“保存”。
  13. 重新启动服务器。
