Managing HTTP Load Balancers

The HTTP Load Balancers page provides you the following information about existing load balancers.

  • Name. Displays the name of the load balancer. Click the load balancer name to edit its settings.

  • Targets. Displays the targets (stand-alone instances and clusters) managed by the load balancer.

  • Status. Indicates if the changes to the load balancer configuration have been pushed to the physical load balancer.

From this page, you can also do the following:

  • Create a new load balancer. To create a new load balancer, click New. For more steps, see To Create an HTTP Load Balancer.

  • Delete an existing load balancer. To delete a load balancer, click the box next to the load balancer you want to delete and then click Delete.
