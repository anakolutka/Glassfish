Para iniciar una aplicación web

Antes de empezar

Asegúrese de que se ha implementado la aplicación web que está iniciando.

  1. En el árbol de navegación, expanda el nodo Aplicaciones.
  2. Seleccione el nodo Aplicaciones web.

    Se abrirá la página Aplicaciones web.

  3. En la tabla Aplicaciones web implementadas, haga clic en Iniciar en la fila de la aplicación que esté iniciando.

    La aplicación se abrirá en una nueva ventana del navegador.
