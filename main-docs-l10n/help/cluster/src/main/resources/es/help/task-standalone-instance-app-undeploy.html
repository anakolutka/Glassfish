Anular despliegue de aplicaciones de una instancia de GlassFish Server independiente

Al anular el despliegue de una aplicación, se elimina la aplicación de todos los objetivos y del dominio. Cuando se anula el despliegue de una aplicación, la aplicación ya no aparece en la página Aplicaciones que muestra todas las aplicaciones desplegadas en GlassFish Server.

  1. En el árbol de navegación, seleccione el nodo Instancias Independientes.

    Se abrirá la página Instancias de Servidor Independientes.

  2. En la página Instancias de Servidor Independientes, seleccione la instancia.

    Se abrirá la página Información General para la instancia.

  3. En la página Información General para la instancia, haga clic en el separador Aplicaciones.

    Se abrirá la página Aplicaciones para la instancia.

  4. Para cada aplicación cuyo despliegue vaya a anular, active la casilla de control que aparece junto a ella en la tabla Aplicaciones Desplegadas.

  5. Haga clic en Anular Despliegue.

    Se le pedirá que confirme que desea anular el despliegue de las aplicaciones.

  6. Confirme que desea anular el despliegue de las aplicaciones.

    Se anula el despliegue de las aplicaciones y éstas se eliminan de la tabla de aplicaciones desplegadas de la página.
