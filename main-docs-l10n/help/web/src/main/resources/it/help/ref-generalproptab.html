Proprietà generali

Utilizzare la pagina Proprietà generali per aggiungere o modificare le proprietà del contenitore Web.

Il contenitore Web funge da host per le applicazioni Web. Fornisce l'ambiente di esecuzione per i servlet e i file JSP (JavaServer Pages). Per la descrizione delle proprietà disponibili, vedere Proprietà specifiche del contenitore Web.

La pagina Proprietà generali contiene le informazioni riportate di seguito per ogni proprietà.

Nome

Il nome della proprietà.

Valore

Il valore della proprietà.

Descrizione

La descrizione della proprietà.
