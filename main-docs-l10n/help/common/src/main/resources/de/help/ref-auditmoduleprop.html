Für Auditmodule spezifische Eigenschaften

Die folgende Eigenschaft ist für ein Auditmodul verfügbar.

auditOn

Ein Wert True verursacht das Laden des Auditmoduls und sorgt dafür, dass es von der GlassFish Server-Audit-Library bei den Audit-Points aufgerufen wird. Ein Wert False deaktiviert die Audit-Logging für das Modul.
