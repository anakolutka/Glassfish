管理檔案使用者

管理檔案使用者會透過安全性範圍節點在 Enterprise Server 中完成。

  1. 在瀏覽樹狀結構中，展開 [配置] 節點。
  2. 若將網域配置為支援叢集，請選取您要變更的配置。

    例如，透過 cluster 設定檔或 enterprise 設定檔建立的網域。

  3. 展開 [安全性] 節點。
  4. 展開 [範圍] 節點。
  5. 選取 [檔案] 節點。
  6. 在 [編輯範圍] 頁面中按一下 [管理使用者] 按鈕。

    [檔案使用者] 頁面會隨即開啟。在此頁面中執行以下作業：

    • 增加使用者

    • 編輯使用者資訊

    • 刪除使用者
