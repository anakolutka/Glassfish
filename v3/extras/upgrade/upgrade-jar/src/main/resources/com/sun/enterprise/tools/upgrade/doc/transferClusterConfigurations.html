Transfer Cluster Configurations Screen

A Sun Java System Application Server 8.x Enterprise Edition (EE) cluster configuration upgrade to Sun Java System Application Server 9.1 requires a valid clinstance.conf file. The Upgrade Tool assumes that the clinstance.conf file is an accurate reflection of the 8.x EE cluster. If, for some reason, this file does not accurately represent the actual cluster configuration, the cluster definition must be manually edited before the upgrade can be performed.

Use the Transfer Cluster Configurations screen to specify the cluster configuration file(s) that need to be migrated to the new installation.

  1. Press the Add button and the Select clinstance.conf file dialog displays.
  2. Select the clinstance.conf cluster file, which contains the cluster configuration information to be migrated.
  3. Press the Open button.
  4. Repeat this process until all the cluster configuration files that must be migrated have been added.
  5. Press the Next button when done. The Upgrade Results screen will display, showing the status of the upgrade.
