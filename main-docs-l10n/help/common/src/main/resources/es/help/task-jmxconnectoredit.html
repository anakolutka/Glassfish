En el campo Puerto, introduzca el número de puerto en el que el servicio de asignación de nombres (registro RMI) recibe las conexiones del cliente RMI.

La única función de este servicio de nomenclatura es descargar los stubs de RMI. Si el puerto por defecto está ocupado, se utilizará otro puerto que esté libre. Los valores válidos se encuentran entre 1 y 65535. En sistemas UNIX, para la creación de sockets de recepción para los puertos 1-1024 se necesitan privilegios de superusuario. El valor por defecto es 8686.

En el campo Nombre de Dominio, escriba el nombre que representa el dominio administrativo especial.

Este dominio maneja toda la autenticación. El valor por defecto es admin-realm.

En la sección Propiedades Adicionales, especifique las propiedades adicionales.

GlassFish Server especifica varias propiedades por defecto.

  • Para agregar una propiedad, haga clic en el botón Agregar Propiedad. En la fila en blanco que aparece, introduzca el nombre de propiedad en el campo Nombre y escriba el valor de propiedad en el campo Valor.

  • Para modificar una propiedad, edite el campo Valor.

  • Para suprimir una propiedad, active la casilla de control a la izquierda del campo Nombre de la propiedad que está suprimiendo y, a continuación, haga clic en el botón Suprimir Propiedades.

Haga clic en Guardar.

Si activa la casilla de control Seguridad Activada, siga los pasos descritos en Editar configuración de SSL para el conector JMX.
