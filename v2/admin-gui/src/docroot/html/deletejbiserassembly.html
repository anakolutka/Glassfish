To Delete Service Assemblies

Deleting Service Assemblies is available only for the Clusters profile.

  1. In the tree view, select the JBI node and then the Service Assemblies node.

  2. Click the service assembly you want to delete.

  3. Click Delete.

    A status message displays asking for confirmation that you want to remove the selected application from this target.

  4. Click OK.

    The Service Assembly is removed from the JBI Service Assemblies table.
