“传输群集配置”屏幕

要将 Sun Java System Application Server 8.x Enterprise Edition (EE) 群集配置升级到 Sun Java System Application Server 9.1，需要有效的 clinstance.conf 文件。升级工具假定 clinstance.conf 文件是 8.x EE 群集的精确反映。如果出于某种原因，此文件没有精确地表示实际的群集配置，则必须手动编辑群集定义，然后才能执行升级。

使用“传输群集配置”屏幕可以指定需要迁移到新安装的群集配置文件。

  1. 按“添加”按钮，此时将显示“选择 clinstance.conf 文件”对话框。
  2. 选择包含要迁移的群集配置信息的 clinstance.conf 群集文件。
  3. 按“打开”按钮。
  4. 重复此过程，直至必须迁移的所有群集配置文件已全部添加为止。
  5. 完成后，请按“下一步”按钮。此时将显示“升级结果”屏幕，表明升级的状态。
