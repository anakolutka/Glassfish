Para reimplantar um aplicativo implantado

A reimplantação de um aplicativo implanta uma nova versão do aplicativo, embora preserve as configurações e outras opções do aplicativo implantado originalmente.

Antes de começar

Verifique se o aplicativo a ser reimplantado já está implantado.

  1. Na árvore de navegação, selecione o nó Aplicativos.

    A página Aplicativos é exibida.

  2. Na tabela Aplicativos Implantados, clique em Reimplantar na linha do aplicativo a ser reimplantado.

    A página Reimplantar Aplicativos ou Módulos é exibida.

  3. Se precisar que sessões ativas do aplicativo sejam preservadas e então restauradas após a conclusão da reimplantação, marque a caixa de seleção Manter Sessões.

    Se alguma sessão ativa do aplicativo não for preservada ou restaurada, nenhuma das sessões será disponibilizada quando a reimplantação estiver concluída. Entretanto, a reimplantação continuará e um aviso será registrado em log.

  4. Especifique o local do arquivo compacto que contém o aplicativo a ser reimplantado.
    • Se o arquivo compacto residir em ou for acessado do computador cliente, selecione a opção Arquivo Empacotador a Ser Carregado no Servidor.

      O computador cliente é o host em que você visualiza o Console de Administração por meio de um navegador.

      Clique em Procurar para localizar o arquivo ou digite o caminho completo para o arquivo.

    • Se o arquivo compacto residir no computador servidor ou se for um aplicativo não empacotado localizado em um diretório destacado, selecione a opção Arquivo Empacotado Local ou Diretório Acessado no Servidor de Aplicativos.

      O computador servidor é o host que executa o servidor de administração de domínio (DAS).

      Clique em Procurar Arquivos para localizar um arquivo ou clique em Procurar Pastas para localizar um diretório. Como alternativa, digite o caminho completo para o arquivo ou diretório. A implantação a partir de um diretório avançado destina-se a desenvolvedores avançados e não é recomendada para ambientes de produção.

  5. Clique em OK.
