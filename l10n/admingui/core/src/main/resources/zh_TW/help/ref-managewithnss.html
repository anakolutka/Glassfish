用於使用者管理的 Network Security Services (NSS)

您可以使用檔案使用者頁面中討論的管理主控台或使用 NSS 工具管理使用者。NSS 是一組程式庫，功能是支援在跨平台的環境下，開發具有安全性的用戶端與伺服器應用程式。使用 NSS 建立的應用程式可以支援 SSL v2 和 v3、TLS、PKCS #5、PKCS #7、PKCS #11、PKCS #12、S/MIME、X.509 v3 等憑證以及其他安全標準。如需詳細資訊，請連結到以下 URL：

  • 網路安全服務 (NSS)，位於 http://www.mozilla.org/projects/security/pki/nss/

  • NSS 安全工具，位於 http://www.mozilla.org/projects/security/pki/nss/tools/

  • NSS 簡介，位於 http://www.mozilla.org/projects/security/pki/nss/overview.html
