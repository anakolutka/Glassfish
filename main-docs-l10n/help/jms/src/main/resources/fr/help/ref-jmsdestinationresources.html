Ressources de destination JMS

La page Ressources de destination JMS permet de configurer les ressources de destination JMS.

Pour chaque ressource de destination, les informations suivantes sont fournies.

Nom JNDI

Nom unique identifiant la ressource de destination.

Activé

(Cette colonne est affichée si seule l'instance de serveur par défaut server existe dans le domaine.)

Coche si la ressource est activée ou X si la ressource est désactivée.

Statut

(Cette colonne apparaît si des clusters ou des instances autonomes ont été créés dans le domaine.)

Nombre de clusters et d'instances autonomes sur lesquels la ressource est ciblée et nombre de cibles sur lesquelles la ressource est activée. Par exemple, Activé sur 2 cibles sur 4 signifie que la ressource est ciblée sur quatre clusters et instances autonomes et activée sur deux de ces quatre cibles.

Type de ressource

Type de la ressource de destination.

Description

Description de la ressource de destination.

La table Ressources de destination contient également les options ci-après.

Nouveau

Bouton permettant de créer une ressource de destination.

Supprimer

Bouton permettant de supprimer les ressources de destination sélectionnées.

Activer

Bouton permettant d'activer les ressources de destination sélectionnées.

Désactiver

Bouton permettant de désactiver les ressources de destination sélectionnées.
