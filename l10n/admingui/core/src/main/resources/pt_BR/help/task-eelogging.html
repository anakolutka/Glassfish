Para Definir as configurações gerais de log

  1. Na árvore de navegação, expanda o nó Configuração.
  2. Sob o nó Configuração, selecione o nó Configurações do Agente de Log.

    A página Configurações do Agente de Log é exibida.

  3. Marque a caixa de seleção Gravar no Log do Sistema Ativado para enviar a saída de log para o recurso syslog, além do log do servidor.

    Esse recurso está disponível apenas nos sistemas Solaris e Linux. Por padrão, a opção está desativada.

  4. Marque a caixa de seleção Registrar em Log no Console Ativado para NÃO FAÇO IDÉIA.

    Por padrão, a opção está desativada.

  5. No campo Arquivo de Log, digite um nome alternativo ou local para o arquivo de log do servidor.

    O local padrão é glassfish/domains/domain1/logs/server.log.

  6. No campo Manipulador de Logs, digite o nome da classe absoluta de um manipulador de logs personalizado.

    Um manipulador de logs personalizado permite que você envie logs para um destino diferente de server.log ou syslog O manipulador personalizado deve estender a classe java.util.logging.Handler (uma API compatível com JSR 047). Coloque a classe do manipulador no caminho da classe do Enterprise Server, para que o manipulador seja instalado durante a inicialização do servidor.

  7. No campo Limite de Rotação de Arquivo, digite o tamanho em bytes que o arquivo de log do servidor deve alcançar antes de ser rotacionado.

    Quando o log do servidor alcança o tamanho especificado, o Enterprise Server cria um arquivo novo, vazio, chamado server.log e renomeia o arquivo antigo para server.log_date, onde date é a data e a hora em que o arquivo foi rotacionado. O valor padrão é 2 megabytes. O valor mínimo para o limite é de 500 quilobytes; se você especificar um valor menor, o arquivo será rotacionado quando alcançar 500 KB. Para desativar a rotação de arquivos de log, defina o valor para 0.

  8. No campo Limite de Tempo de Rotação de Arquivo, digite o número de minutos após o qual o log do servidor deve ser rotacionado.

    O valor padrão é 0, o que significa que o arquivo é rotacionado quando ele alcança o tamanho especificado no campo Limite de Rotação de Arquivo.

    Observação - Se você especificar um ou mais minutos, o limite de tempo tem precedência sobre o limite de tamanho especificado pelo campo Limite de Rotação de Arquivo.

  9. Clique em Salvar.
  10. Pare e reinicie o Enterprise Server.
