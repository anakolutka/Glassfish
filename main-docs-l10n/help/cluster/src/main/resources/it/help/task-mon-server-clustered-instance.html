Per visualizzare i dati di monitoraggio dei server virtuali per un'istanza di GlassFish Server in cluster

Prima di iniziare

Per visualizzare i dati di monitoraggio, è necessario configurare il servizio di monitoraggio.

  1. Espandere il nodo Cluster nell'albero di navigazione.

  2. Selezionare il nodo per il cluster sotto il nodo Cluster espanso.

    Viene visualizzata la pagina Informazioni generali per il cluster.

  3. Nella pagina Informazioni generali, fare clic sulla scheda Istanze.

    Viene visualizzata la pagina Istanze server in cluster per il cluster.

  4. Selezionare l'istanza per la quale visualizzare i dati di monitoraggio per i server virtuali.

    Viene visualizzata la pagina Informazioni generali per l'istanza.

  5. Nella pagina Informazioni generali, fare clic sulla scheda Monitoraggio.

    Viene visualizzata la pagina Monitoraggio applicazioni.

  6. Fare clic sulla scheda Server.

    Viene visualizzata la pagina Monitoraggio server.

  7. Nell'elenco a discesa Visualizza, selezionare il componente per il quale si desidera visualizzare i dati di monitoraggio.

  8. Per aggiornare i dati di monitoraggio visualizzati, fare clic su Aggiorna.
