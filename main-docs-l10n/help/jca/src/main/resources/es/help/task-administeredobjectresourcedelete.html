Para suprimir un recurso de objeto administrado

  1. En el árbol de navegación, amplíe el nodo Recursos.

  2. Amplíe el nodo Conectores.

  3. En el nodo Conectores, seleccione el nodo Recursos de objeto administrado.

    Se abrirá la página Recursos de objeto administrado

  4. En la página Recursos de objeto administrado, active la casilla de control situada junto al nombre del recurso de objeto administrado que está suprimiendo.

  5. Haga clic en Suprimir.
