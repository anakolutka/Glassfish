To configure a JMX connector using the Admin Service

The Admin Service determines whether the server instance is a regular instance, a domain administration server (DAS), or a combination. Platform Edition has only one server instance, and it is a combination. Use the Admin Service to configure a JSR-160 compliant remote JMX connector, which handles communication between the domain administration server and the node agents, which manage server instances on a host machine, for remote server instances.

  1. Expand the Configuration node, then the Admin Service node.

  2. Select the admin service to be modified.

  3. Set the security properties by following the instructions in To configure security for the Admin Service's JMX connector.

  4. Click Save to save the changes, or click Load Defaults to restore the properties to their default values.
