웹 응용 프로그램

웹 응용 프로그램을 사용하여 기존 웹 응용 프로그램을 수정합니다. 페이지에는 다음 옵션이 포함되어 있습니다.

이름

편집 중인 웹 응용 프로그램 이름을 표시하는 읽기 전용 필드입니다.

컨텍스트 루트

웹 응용 프로그램에 대한 경로를 지정합니다. 웹 응용 프로그램의 URL에서 컨텍스트 루트 바로 앞에는 포트 번호( http://host:port/context-root/...)가 있습니다. 컨텍스트 루트는 슬래시로 시작해야 합니다(예: /hello).

가상 서버

이 웹 응용 프로그램과 연결된 가상 서버입니다.

설명

웹 응용 프로그램에 대한 설명입니다.

상태 사용 가능

이 옵션을 선택하면 웹 응용 프로그램이 활성화됩니다. 이 옵션은 기본적으로 선택됩니다.

위치

응용 프로그램이 배포된 아카이브 파일의 위치를 표시하는 읽기 전용 필드입니다.
