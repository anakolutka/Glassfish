Your server is up and running!

To replace this page, overwrite the file index.html. The location of this file depends on your product and platform:

  • Sun Java Enterprise System on Solaris: /var/opt/SUNWappserver/domains/<domain_name>/docroot
  • Sun Java Enterprise System on Linux: /var/opt/sun/appserver/domains/<domain_name>/docroot
    where <domain_name> is the domain name (for example, domain1).
  • Stand-alone Application Server (file-based installations): <install_dir>/domains/<domain_name>/docroot, where <install_dir> is the Application Server installation directory and <domain_name> is the domain name.
