파일 사용자

관리 콘솔을 사용하여 file 영역 사용자를 관리합니다. file 영역의 사용자 및 그룹은 키 파일에 나열되며, 키 파일의 위치는 file 등록 정보에 지정되어 있습니다.

주 - 이 단계를 사용하여 admin-realm을 포함한 file 영역에 사용자를 추가할 수도 있습니다. 이 절에 언급된 file 영역을 대상 영역의 이름으로 바꾸기만 하면 됩니다.

file 영역의 사용자는 공통된 특성으로 분류된 사용자 범주인 Java 그룹에 속할 수 있습니다. 예를 들어, 전자 상거래 응용 프로그램 사용자는 CUSTOMER 그룹에 속할 수 있지만, 대량 소비자는 PREFERRED 그룹에 속할 수 있습니다. 사용자를 그룹으로 범주화하면 많은 수의 사용자 액세스를 더 쉽게 제어할 수 있습니다.

Enterprise Server 설치 후 처음에는 설치 시 입력한 관리자만이 유일한 사용자입니다. 기본적으로 이 사용자는 Enterprise Server를 수정할 수 있는 권한을 부여하는 admin-realm 영역의 asadmin 그룹에 속합니다. 이 그룹에 할당된 사용자는 관리자 권한을 갖습니다. 즉, asadmin 도구 및 관리 콘솔에 액세스할 수 있는 권한을 부여 받습니다.

파일 사용자 페이지는 선택한 영역의 사용자 목록을 표시합니다. 각 사용자에 대해 제공되는 정보는 다음과 같습니다.

사용자 ID

사용자 이름입니다.

그룹 목록

사용자가 속한 그룹입니다.
