Para Deletar um Módulo de Auditoria

  1. Na árvore de navegação, expanda o nó Configuração.

  2. Selecione a instância que será configurada:

    • Para configurar uma instância em particular, expanda o nó de configuração da instância. Por exemplo, na instância default, server, expanda o nó server-config.

    • Para definir as definições default de todas as instâncias, expanda o nó default-config.

  3. Sob o nó Configuração, expanda o nó Segurança.

  4. Sob o nó Segurança, selecione o nó Módulos de Auditoria.

    A página Módulos de Auditoria será aberta.

  5. Na página Módulos de Auditoria, marque a caixa de seleção à esquerda do módulo de auditoria a ser deletado.

  6. Clique em Deletar.
