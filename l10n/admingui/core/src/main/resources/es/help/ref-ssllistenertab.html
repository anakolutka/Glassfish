Escucha SSL

La escucha SSL enlaza un certificado del servidor con el puerto de escucha HTTP. La configuración de SSL se define al crear una escucha HTTP nueva a través de la página Nueva escucha HTTP (paso 2 de 2). La configuración de SSL se edita adicionalmente mediante la página Escucha SSL.

Autenticación de cliente

Permitir obligar a los clientes a autenticarse en el servidor cada vez que utilicen esta escucha.

Sobrenombre del certificado

Nombre del par de claves y del certificado del servidor existente.

SSL3

Si se activa esta opción, se utilizará en la escucha el protocolo de seguridad SSL3.

SSL2

Si se activa esta opción, se utilizará en la escucha el protocolo de seguridad SSL2.

TLS

Si se activa esta opción, se utilizará en la escucha el protocolo de seguridad TLS.

Conjuntos de cifrados comunes

Conjuntos de cifrados disponibles utilizados por los protocolos.

Conjuntos de cifrados efímeros Dittie-Hellman

Conjuntos de cifrados efímeros utilizados por los protocolos.
