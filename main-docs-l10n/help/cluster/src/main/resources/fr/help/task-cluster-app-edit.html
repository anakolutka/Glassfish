Modification d'une application déployée vers un cluster

  1. Dans l'arborescence de navigation, développez le noeud Clusters.

  2. Sous le noeud Clusters développé, sélectionnez le noeud correspondant au cluster.

    La page Informations générales du cluster apparaît.

  3. Sur la page Informations générales, cliquez sur l'onglet Applications.

    La page Applications du cluster apparaît.

  4. Dans la table Applications déployées de la page Applications, cliquez sur le nom de l'application.

    La page Modifier une application apparaît.

  5. Sur la page Modifier une application, vous devez fournir les informations requises concernant l'application et enregistrer vos modifications.

    Pour plus d'informations, reportez-vous à Modifier une application dans l'aide en ligne.
