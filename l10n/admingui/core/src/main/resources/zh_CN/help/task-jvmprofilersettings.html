配置 JVM 事件探查器设置

事件探查器工具可生成用于分析性能和识别潜在瓶颈的数据。

  1. 在导航树中，选择 "Application Server" 节点。
  2. 在“常规信息”页面上，单击“JVM 设置”选项卡。
  3. 在“JVM 常规设置”页面上，单击“事件探查器”选项卡。

    此时将打开“JVM 事件探查器设置”页面。

  4. 指定事件探查器设置。

    指定的信息取决于所使用的事件探查器产品。有关示例和说明，请参见《Sun GlassFish Enterprise Server v3 Prelude Developer’s Guide》中的第 3  章 “Debugging Applications”。

  5. 单击“保存”。
  6. 重新启动服务器。
