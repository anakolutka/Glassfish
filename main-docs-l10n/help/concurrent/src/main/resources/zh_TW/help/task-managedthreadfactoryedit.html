編輯受管理繫線工廠

  1. 在瀏覽樹狀結構中，展開「資源」節點。

  2. 在「資源」節點下，展開「並行資源」節點。

  3. 在「並行資源」節點下，選取「受管理繫線工廠」。

    就會開啟「受管理繫線工廠」頁面。

  4. 在「受管理繫線工廠」頁面中，按一下要編輯的資源名稱。

    就會開啟「編輯受管理繫線工廠」頁面。

    備註:

    「JNDI 名稱」是唯讀欄位。您僅能在建立新資源時指定 JNDI 名稱。邏輯 JNDI 名稱也是唯讀欄位。

  5. 選取「啟用環境資訊」核取方塊即可將環境傳輸至繫線，並選取想要傳輸的環境。

    預設會啟用此選項並選取所有環境。如果要選擇多個環境，請在按住 Shift 鍵或 Ctrl 鍵的同時以滑鼠點選環境。如果停用此選項，就不會傳輸任何環境 (包括選取的環境)。

  6. 選取「啟用狀態」核取方塊，以啟用資源。

    此選項預設為啟用。

  7. 在「繫線優先順序」欄位中，鍵入要指定給已建立繫線的優先順序。

    預設值為 5。

  8. 在「建置順序」欄位中，鍵入數字來指定資源的建置順序。

    系統會在伺服器啟動時先載入數字較小的資源。預設值為 100。

  9. 在「描述」欄位中，鍵入資源的簡短描述。

    該描述是一個字串值，最多可以包含 250 個字元。

  10. 請勿在「其他特性」區段中新增任何其他特性。

    GlassFish Server 並未定義任何此資源類型的其他特性。此外，此資源類型目前也並未使用任何其他特性。

  11. 按一下「儲存」。
