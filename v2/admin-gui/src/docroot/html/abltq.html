To delete an HTTP listener

  1. In the Developer profile, expand the Configuration, HTTP Service node, then HTTP Listener node.
  2. In the Clusters and Enterprise profile, expand the Configuration node, select the configuration you want to configure, then select HTTP service, then the HTTP Listeners node.
  3. Check the box next to the name of the HTTP listener to be deleted.
  4. Click Delete.

    It is possible to delete the http-listener-1, http-listener-2, and admin-listener HTTP listeners, but this is not recommended. If you plan to do so, first copy the http-listener elements of the Application Server’s domain.xml file to a safe place so that the settings can be restored if needed.

More Information

delete-http-listener
