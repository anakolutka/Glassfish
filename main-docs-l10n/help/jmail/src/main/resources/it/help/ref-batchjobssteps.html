Passi esecuzione processo batch

Nella pagina Passi esecuzione processo batch vengono visualizzati i dettagli relativi ai passi di una esecuzione di processo batch specifica. Vengono visualizzate le informazioni riportate di seguito.

Nome istanza
Nome cluster

Il nome del cluster o dell'istanza server di GlassFish cui si applica la configurazione.

Nome processo

Il nome del processo batch.

ID esecuzione

L'ID assegnato all'esecuzione del processo batch. Una nuova esecuzione viene creata al primo avvio di un processo e a ogni riavvio dell'esecuzione esistente.

Passi processo

I passi in una esecuzione di processo batch specifica. Per ogni passo vengono visualizzate le informazioni riportate di seguito.

Nome passo

Il nome del passo.

Stato batch

Lo stato del passo impostato dal runtime batch.

Stato di uscita

Lo stato del passo impostato dall'XML di processo per il processo o dall'applicazione batch. Per impostazione predefinita, lo stato di uscita e lo stato del batch sono uguali a meno che lo stato di uscita non venga sostituito in modo esplicito.

Ora di inizio

L'ora in cui è stato iniziato il passo.

Ora di fine

L'ora in cui è stato terminato il passo.

Metriche passo

Metriche per il passo, con l'indicazione del numero di elementi letti, scritti, con commit eseguito e così via.
