Ajouter un utilisateur

  1. Cliquez sur Nouveau pour ajouter un nouvel utilisateur au domaine fichier.

    La page Nouvel utilisateur du domaine fichier s'ouvre.

  2. Dans le champ ID utilisateur, saisissez le nom de l'utilisateur. Ce champ est obligatoire.
  3. Dans le champ Mot de passe, saisissez un mot de passe unique. Ce champ est obligatoire.
  4. Confirmez de nouveau le mot de passe de l'utilisateur, par sécurité. Ce champ est obligatoire.
  5. Dans le champ Liste des groupes, saisissez, en les séparant par des virgules, les groupes auxquels l'utilisateur appartient. Ces groupes n'ont pas besoin d'être définis ailleurs.
  6. Sélectionnez OK pour ajouter cet utilisateur à la liste d'utilisateurs du domaine fichier ou sur Annuler pour quitter la page sans enregistrer.
