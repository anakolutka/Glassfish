Création d'une ressource d'objet administré

  1. Dans l'arborescence de navigation, développez le noeud Ressources.

  2. Développez le noeud Connecteurs.

  3. Sous le noeud Connecteurs, sélectionnez le noeud Ressources d'objet d'administration.

    La page Ressources d'objet d'administration apparaît.

  4. Sur la page Ressources d'objet d'administration, cliquez sur le bouton Nouveau.

    La page Nouvelle ressource d'objet d'administration apparaît.

  5. Dans le champ Nom JNDI, saisissez le nom unique qui identifie la ressource d'objet administré.

  6. Dans la liste déroulante Adaptateur de ressources, sélectionnez l'adaptateur de ressources pour la ressource d'objet administré.

  7. Dans la liste déroulante Type de ressource, sélectionnez le type Java de la ressource d'objet administré.

  8. Dans la liste déroulante Nom de classe, sélectionnez le nom de la classe d'implémentation associée au type de ressource.

  9. Dans le champ Description, saisissez la description de la ressource d'objet administré.

  10. Sélectionnez la case Activé du champ Statut pour activer la ressource d'objet administré.

    Cette option est activée par défaut.

  11. Dans la table Autres propriétés, spécifiez les propriétés supplémentaires de la ressource d'objet administré.

    Les propriétés disponibles dépendent de l'adaptateur de ressources du pool de connexions de connecteur.

    Pour ajouter une propriété, cliquez sur le bouton Ajouter une propriété. Sur la ligne vide qui apparaît, saisissez le nom de la propriété dans le champ Nom et la valeur de la propriété dans le champ Valeur.

  12. Dans la section Cibles disponibles, sélectionnez le domaine, le cluster ou les instances de serveur où doit résider l'objet administré, puis cliquez sur Ajouter.

    Pour enlever l'un des domaines, clusters ou l'une des instances de serveur répertoriés dans le champ Sélectionné, sélectionnez l'élément dans le champ, puis cliquez sur Enlever.

  13. Cliquez sur OK.
