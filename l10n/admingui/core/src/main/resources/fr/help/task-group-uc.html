Mise à jour et développement de Enterprise Server

La mise à jour et le développement de Enterprise Server comprend les tâches suivantes :

  • Configurer les paramètres proxy de réseau pour accéder aux référentiels de réseau

  • Mettre à jour les composants installés

  • Installer des composants add-ons

  • Désinstaller des composants

  • Afficher les informations détaillées d'un composant
