Komponentendetails

Die Seite "Komponentendetails" enthält detaillierte Informationen zu einer Komponente.

Die Seite "Komponentendetails" umfasst die folgenden Optionen.

Zurück

Diese Schaltfläche wechselt zu der Seite zurück, auf der Sie die Komponente ausgewählt haben, deren Details Sie gerade anzeigen.

Packagename

Der Name des Packages, in dem die Komponente geliefert wird.

Eindeutige ID

Die vollständig angegebene ID dieser Komponente wie im Repository vorhanden.

Kategorie

Die Softwarekategorie, mit der die Komponente verbunden ist, z.B. Application Server, Systemtools und Datenbanken. Die Kategorie wird der Komponente vom Autor der Komponente bei deren Erstellung zugewiesen.

Version

Die Versionsnummer des Packages, in dem die Komponente geliefert wird.

Veröffentlichungsdatum

Datum und Uhrzeit der Veröffentlichung dieser Komponentenversion im Repository. Dieses Datum ist nicht das Datum und die Uhrzeit der Installation der Komponente innerhalb des Installationsimages.

Installationsgröße

Der Speicherplatz, den die Installation der Komponente beansprucht.

Quelle

Für installierte Komponenten ist dies der Name des Netzwerk-Repository, aus dem die Komponente abgerufen wurde.

Für eine Add-on-Komponente oder ein Update ist dies der Name des Netzwerk-Repositorys, von dem die Komponente abzurufen ist.

Beschreibung

Eine Übersicht der Komponente.

Verwandte Aufgaben

So werden detaillierte Informationen über eine Komponente angezeigt

Zugehörige asadmin-Befehle
