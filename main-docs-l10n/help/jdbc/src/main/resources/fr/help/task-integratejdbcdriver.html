Intégration d'un pilote JDBC

Un pilote prenant en charge l'API JDBC (pilote JDBC) convertit les appels JDBC d'une application en protocole du serveur de base de données.

Pour intégrer le pilote JDBC à un domaine GlassFish Server, effectuez l'une des opérations suivantes :

  • Configurez le pilote afin qu'il soit accessible au chargeur de classe commun.

    1. Copiez les fichiers JAR et ZIP du pilote dans le répertoire domain-dir/lib.

      Remarque:

      Il existe une exception à cette règle. Si vous utilisez le pilote Oracle JDBC et que vous voulez utiliser des extensions EclipseLink disponibles avec les bases de données Oracle, vous devez placer les fichiers JAR du pilote Oracle JDBC dans le répertoire domain-dir/lib/ext.

    2. Redémarrez le domaine.

  • Configurez le pilote afin qu'il soit accessible au chargeur de classe système.

    1. Dans l'arborescence de navigation, développez le noeud Configuration.

    2. Sous le noeud Configuration, sélectionnez le noeud Paramètres JVM.

      La page Paramètres généraux JVM apparaît.

    3. Sur la page Paramètres généraux JVM, cliquez sur l'onglet Paramètres du chemin.

    4. Dans le champ Suffixe de classpath, saisissez le nom qualifié complet du chemin du fichier JAR du pilote.

    5. Cliquez sur Enregistrer.

    6. Redémarrez le serveur.
