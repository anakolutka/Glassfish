JACC 提供者

[JACC 提供者] 頁面會顯示可用的 Java 容器授權合約 (JACC) 提供者清單。

JACC 屬於 Java EE 規格，其定義可插接式授權提供者的介面。這可以讓管理員設定協力廠商外掛程式模組以執行授權。依預設，Enterprise Server 提供一個簡單的與 JACC 相容並基於檔案的授權引擎。

如需了解如何撰寫 JACC 提供者的實作類別，請參閱 JACC 規格，網址為 http://jcp.org/en/jsr/detail?id=115。

對於每個 JACC 提供者，均提供以下資訊：

名稱

JACC 提供者的名稱。

策略提供者

實作策略工廠的類別名稱。
