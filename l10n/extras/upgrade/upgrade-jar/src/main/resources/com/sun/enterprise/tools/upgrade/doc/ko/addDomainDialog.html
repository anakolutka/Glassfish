도메인 추가 대화 상자

도메인 추가 대화 상자를 사용하여 도메인 이름과 소스 및 대상 인증서 비밀번호를 선택합니다.

  1. 소스 도메인 풀다운 메뉴를 클릭하고 마이그레이션할 보안 인증서가 있는 도메인을 선택합니다.
  2. 비밀번호 패널에서 입력하라는 메시지가 표시되면 비밀번호를 입력합니다. 소스 및 대상 설치 유형에 따라 2개 또는 3개의 비밀번호를 입력해야 할 수 있습니다.
    • 소스 NSS 및/또는 대상 NSS 필드에서 NSS 트러스트된 데이터베이스에 사용하는 NSS 인증서 파일용 비밀번호를 입력합니다.
    • 사용자 키 데이터베이스 필드에서 사용자 키 쌍 인증서가 있는 JSSE.keystore.jks 파일에 사용하는 소스 사용자 키 데이터베이스용 비밀번호를 입력합니다.
    • 인증 기관 데이터베이스 필드에서 인증 기관 인증서가 있는 JSSE.cacerts.jks 파일에 사용되는 인증 기관 데이터베이스용 비밀번호를 입력합니다.
  3. 완료되면 확인 버튼을 누릅니다. 보안 인증서 전송 화면이 나타나, 전송할 인증서가 있는 도메인 패널에 추가된 도메인이 표시됩니다.
