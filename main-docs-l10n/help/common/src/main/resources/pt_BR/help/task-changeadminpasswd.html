Para Alterar a Senha de um Usuário no Realm Admin

Para alterar a senha da conta atual do administrador, consulte Para Alterar a Senha do Administrador. Se o realm-admin tiver mais de um usuário, você também poderá alterar a senha de outros usuários.

Criptografar a senha do administrador aumenta a segurança da aplicação. Consulte o Capítulo 1, Administering System Security, no Oracle GlassFish Server Security Guide para obter informações mais detalhadas sobre procedimentos de segurança.

  1. No nó Configurações, expanda a configuração de cluster ou instância de servidor na qual deseja modificar a senha de administração.

  2. Expanda o nó Segurança.

  3. Expanda o nó Realms.

  4. Selecione o nó admin-realm.

    A página Editar Realm será aberta.

  5. Na página Editar Realm, clique no botão Gerenciar Usuários.

    A página Usuários Admin será aberta.

  6. Na página Usuários Admin, clique no usuário cuja senha você quer adicionar ou alterar.

    A página Editar Usuário de Realm de Arquivos será aberta.

  7. Nos campos Nova Senha e Confirmar Nova Senha, digite a nova senha.

  8. Clique em Salvar.
