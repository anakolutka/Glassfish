Para Coletar e Fazer Download de Logs do Domínio

Você pode coletar todos os arquivos de log disponíveis para uma instância ou cluster específico do GlassFish Server e compilar os logs em um arquivo compactado ZIP simples.

Observação:

Se você especificar um cluster, o arquivo compactado ZIP também incluirá o arquivo de log para o servidor de administração de domínios (DAS).

  1. Na árvore de navegação, selecione o nó Domínio.

    A página Atributos do Domínio será aberta.

  2. Selecione a guia Logs de Domínio.

    A página Logs do Domínio será aberta.

    Se existirem clusters, serão exibidas listas drop-down para clusters e instâncias. Se não existirem clusters, somente uma lista drop-down para instâncias será exibida.

  3. Selecione Cluster ou Instância.

  4. Selecione o cluster ou instância cujos arquivos de log você está coletando e fazendo download.

  5. Clique em Coletar Logs.

    O mecanismo de download do seu browser’ solicita que você faça download do arquivo log-files-name.zip, em que name é o nome do cluster ou instância.
