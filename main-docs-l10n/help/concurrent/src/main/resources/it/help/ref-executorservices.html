Servizi esecutore gestiti

Utilizzare la pagina Servizi esecutore gestiti per configurare le risorse dei servizi esecutore gestiti.

Per ogni risorsa servizio contesto vengono fornite le informazioni riportate di seguito.

Nome JNDI

Il nome univoco che identifica la risorsa.

Abilitato

L'impostazione è True se la risorsa è abilitata oppure è False se la risorsa è disabilitata.

Informazioni contesto

Il contesto della risorsa: Classloader, JNDI, Sicurezza o Area di lavoro.

Priorità thread

La priorità dei thread da creare per l'istanza di servizio esecutore gestito. L'impostazione predefinita è 5.

Descrizione

La descrizione della risorsa.

La tabella Servizi esecutore gestiti contiene inoltre le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare una nuova risorsa.

Elimina

Pulsante che consente di eliminare una o più risorse selezionate.

Abilita

Pulsante che consente di abilitare una o più risorse selezionate.

Disabilita

Pulsante che consente di disabilitare una o più risorse selezionate.
