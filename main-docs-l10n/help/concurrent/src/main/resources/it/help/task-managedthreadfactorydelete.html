Per eliminare un thread factory gestito

  1. Espandere il nodo Risorse nell'albero di navigazione.

  2. Nel nodo Risorse espandere il nodo Risorse concorrenti.

  3. Nel nodo Risorse concorrenti selezionare Thread factory gestiti.

    Viene visualizzata la pagina Thread factory gestiti.

  4. Nella pagina Thread factory gestiti selezionare la casella di controllo in corrispondenza del nome della risorsa da eliminare.

  5. Fare clic su Elimina.
