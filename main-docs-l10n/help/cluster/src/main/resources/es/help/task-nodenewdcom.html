Terminar la creación de un nodo DCOM

Esta tarea es la continuación para un nodo DCOM de Crear un nodo.

  1. Para copiar el software GlassFish Server desde el host del DAS al host del nodo, active la opción Instalar GlassFish Server.

  2. Para forzar la creación del nodo en la configuración del servidor de administración de dominio (DAS) aunque falle la validación de los parámetros del nodo, active la opción Forzar.

  3. Si necesita que GlassFish Server pruebe la conexión mediante DCOM con el host del nodo antes de crear el nodo, lleve a cabo estos pasos:

    1. Active la opción Validar DCOM.

    2. En el campo Directorio de Prueba Remoto, escriba el nombre de la carpeta del host remoto en la que GlassFish Server deba crear un archivo para probar la conexión.

  4. En el campo Dominio de Windows, escriba el nombre del domino de Windows que contenga el usuario que la opción Nombre de Usuario de Windows especifica.

  5. En el campo Nombre de Usuario de Windows, escriba el nombre del usuario del host de este nodo que ejecutará el proceso para conectar al host mediante DCOM.

  6. Especifique cómo se autenticará el usuario de Windows al iniciar sesión en el host de este nodo.

    Nota:

    GlassFish Server también soporta la autenticación con contraseña a través de la entrada AS_ADMIN_WINDOWSPASSWORD en el archivo de contraseñas. El archivo de contraseñas se especifica en la opción -paswordfile de la utilidad asadmin(1).

    1. Para especificar la autenticación mediante una contraseña guardada en la configuración de DAS, efectúe los pasos siguientes:

      Atención:

      Esta contraseña se almacena en forma de texto no cifrado en la configuración de DAS. Para mejorar la seguridad, utilice un alias de contraseña

      1. Seleccione Contraseña en la lista desplegable Autenticación de Usuario de Windows.

      2. En el campo Contraseña de Usuario de Windows, escriba la contraseña que el usuario de Windows utilizará al conectarse al host de este nodo.

    2. Para especificar la autenticación mediante un alias de contraseña, efectúe los pasos siguientes:

      1. Seleccione Alias de Contraseña en la lista desplegable Autenticación de Usuario de Windows.

      2. En la lista desplegable Alias de Contraseña, seleccione un alias existente.

  7. Haga clic en Guardar.
