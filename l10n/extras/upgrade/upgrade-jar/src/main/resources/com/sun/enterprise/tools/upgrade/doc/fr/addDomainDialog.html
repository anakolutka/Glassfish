Boîte de dialogue Ajouter un domaine

Utilisez la boîte de dialogue Ajouter un domaine pour sélectionner le nom du domaine et les mots de passe des certificats source et cible.

  1. Cliquez sur le menu déroulant Domaine source pour sélectionner le domaine qui contient les certificats de sécurité à migrer.
  2. Dans le panneau Mots de passe, saisissez les mots de passe demandés par le système. Selon le type d'installation source et cible, deux ou trois mots de passe peuvent être nécessaires :
    • Saisissez les mots de passe pour les fichiers du certificat NSS, utilisés pour les bases de données de confiance NSS, dans les champs NSS source et/ou NSS cible.
    • Saisissez le mot de passe de la base de données de clés utilisateur, utilisée pour le fichier JSSE keystore.jks qui contient les certificats de paire de clés utilisateur, dans le champ Bases de données de clés utilisateur.
    • Saisissez le mot de passe pour la base de données de l'autorité de certification, utilisée pour le fichier JSSE cacerts.jks qui contient les certificats de l'autorité de certification, dans le champ Base de données d'autorité de certification.
  3. Appuyez ensuite sur OK. L'écran Transférer les certificats de sécurité s'affiche, indiquant le domaine ajouté au panneau Domaines contenant des certificats à transférer.
