Nuevo usuario del dominio de archivo

Utilice la página Nuevo usuario del dominio de archivo para garantizar el acceso al dominio actual a un usuario nuevo.

Nombre de dominio

Campo de sólo lectura que muestra el nombre del dominio seleccionado actualmente.

Id. de usuario

Id. de usuario para una cuenta de usuario nueva.

Lista de grupos

Grupos al que pertenecerá el usuario nuevo. Si introduce varios nombres de grupo, sepárelos con comas.

Contraseña nueva

Contraseña del usuario nuevo.

Contraseña nueva (confirmar)

Contraseña del nuevo usuario.
