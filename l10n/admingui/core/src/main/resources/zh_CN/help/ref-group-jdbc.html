JDBC

JavaTM DataBase Connectivity (JDBCTM) 技术提供到关系数据库的连接。在 Enterprise Server 中，可通过向域中添加 JDBC 资源（数据源）来配置数据库连接。数据源是 Java Platform, Enterprise Edition（Java EE 平台）配置到数据库的连接的标准方法。每个数据源均包含一个数据库连接池。应用程序在 Java Naming and Directory InterfaceTM (J.N.D.I.) API 树或本地应用程序上下文中查找数据源，然后使用 getConnection 方法检索数据库连接。数据源及其连接池提供的连接管理进程有助于保持系统的运行。

要配置 JDBC 资源，请使用以下页面：

  • JDBC 资源

  • 新建 JDBC 资源

  • 编辑 JDBC 资源

  • 连接池

  • 新建 JDBC 连接池（步骤 1，共 2 步）

  • 新建 JDBC 连接池（步骤 2，共 2 步）

  • 编辑连接池

  • 编辑连接池属性
