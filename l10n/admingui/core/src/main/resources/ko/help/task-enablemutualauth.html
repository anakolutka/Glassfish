모든 응용 프로그램에 대해 상호 인증을 활성화하는 방법

Enterprise Server는 HTTPS 인증 시 certificate 영역을 사용합니다.

  1. 탐색 트리에서 구성 노드를 선택합니다.
  2. 구성 노드에서 보안 노드를 확장합니다.
  3. 보안 노드에서 영역 노드를 확장합니다.
  4. 영역 노드에서 certificate 영역을 선택합니다.

    영역 편집 페이지가 열립니다.

  5. 영역 편집 페이지의 추가 등록 정보 테이블에서 등록 정보 추가 버튼을 누릅니다.
    1. 이름 필드에 clientAuth를 입력합니다.
    2. 값 필드에 true를 입력합니다.
  6. 저장을 누릅니다.
  7. 콘솔에 다시 시작해야 함이 표시되면 Enterprise Server를 다시 시작합니다.

    서버를 다시 시작한 후에는 certificate 영역을 사용하는 모든 응용 프로그램에 클라이언트 인증이 필요합니다.
