연결 풀

연결 풀 페이지에는 다음 옵션이 포함되어 있습니다.

보류 중인 최대 연결 수

HTTP 수신기에 대해 허용할 보류 중인 최대 연결 수를 지정합니다. 기본값은 4096입니다.

대기열 크기

연결 대기열의 최대 크기(바이트)를 지정합니다. 이 값은 서버가 보유할 수 있는 해결되지 않은 최대 연결 수도 지정합니다. 기본값은 4096입니다.

수신 버퍼 크기

HTTP 수신기의 수신 버퍼 크기(바이트)를 지정합니다. 기본값은 4096입니다.

전송 버퍼 크기

HTTP 수신기의 전송 버퍼 크기(바이트)를 지정합니다. 기본값은 8192입니다.
