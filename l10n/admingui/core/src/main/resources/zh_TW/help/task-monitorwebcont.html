檢視 Web 容器監視資料

  1. 在瀏覽樹狀結構中，選取 [Application Server] 節點。
  2. 在 [一般資訊] 頁面上，按一下 [監視] 標籤。

    [Web 容器監視] 頁面會隨即開啟。

  3. (可選擇) 從 [應用程式] 下拉式清單中，選取一個應用程式以檢視該應用程式的監視資料。
  4. (可選擇) 從 [虛擬伺服器] 下拉式清單中，選取一個虛擬伺服器以檢視該伺服器的監視資料。
  5. (可選擇) 按一下 [重新整理] 按鈕可更新顯示的監視資料。
