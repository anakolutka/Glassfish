Configurer l'accès à la base de données

Pour stocker, organiser et récupérer les données, la plupart des applications utilisent des bases de données relationnelles.

  1. Installez un produit de base de données pris en charge.

    Pour consulter la liste des bases de données prises en charge, reportez-vous aux Sun GlassFish Enterprise Server v3 Prelude Release Notes .

  2. Installez un pilote prenant en charge l'API JDBC (pilote JDBC) pour la base de données.

    Assurez-vous que le pilote que vous souhaitez utiliser pour créer des connexions à la base de données est installé sur tous les serveurs sur lesquels vous souhaitez déployer la source de données.

  3. Assurez-vous que le fichier JAR du pilote soit accessible à l'instance de serveur du domaine.

    Voir Intégrer un pilote JDBC.

  4. Créez la base de données.

    Généralement, le fournisseur d'applications offre des scripts pour créer et renseigner la base de données.

  5. Créez un pool de connexions pour la base de données.

    Voir Créer un pool de connexions JDBC.

  6. Créez une ressource JDBC pointant vers le pool de connexions.

    Voir Créer une ressource JDBC.
