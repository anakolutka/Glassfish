Déploiement d'une application vers un cluster

  1. Dans l'arborescence de navigation, développez le noeud Clusters.

  2. Sous le noeud Clusters développé, sélectionnez le noeud correspondant au cluster.

    La page Informations générales du cluster apparaît.

  3. Sur la page Informations générales, cliquez sur l'onglet Applications.

    La page Applications du cluster apparaît.

  4. Sur la page Applications, cliquez sur Déployer.

    La page Déployer des applications ou des modules apparaît.

  5. Sur la page Déployer des applications ou des modules, indiquez les informations requises sur l'application, puis cliquez sur OK.

    Pour plus d'informations, reportez-vous à Déployer des applications ou des modules dans l'aide en ligne.
