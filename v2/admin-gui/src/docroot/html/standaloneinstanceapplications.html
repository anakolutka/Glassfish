To Manage Applications on a Server Instance

  1. In the Admin Console tree, expand the Stand-alone Instances node and select the desired server instance.

    The General Information page opens.

  2. Click the Applications tab.

    The Applications page appears. In this page, you can enable, disable or remove applications associated with the server instance.

  3. From the Filter drop down list, select the type of applications you want to be displayed in the Application table.
  4. To deploy an application on the server instance, click Deploy.
  5. To enable an application on the server instance, select the application by clicking the check box next to the application name and select Enable from the More Actions drop down list.
  6. To disable an application on the server instance, select the application by clicking the check box next to the application name and select Disable from the More Actions drop down list.
  7. To enable an application for load balancing, select the application by clicking the check box next to the application name and select Load Balancer Enable from the More Actions drop down list.
  8. If you do not want an application on the server instance to be load balanced, select the application by clicking the check box next to the application name and select Enable from the More Actions drop down list.
  9. To remove an application from the server instance, select the application by clicking the check box next to the application name and click Remove.
