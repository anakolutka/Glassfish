Proprietà di sessione

Utilizzare la pagina Proprietà di sessione per aggiungere o modificare le proprietà per la sessione del contenitore Web.

La pagina Proprietà di sessione contiene le opzioni riportate di seguito.

Carica impostazioni predefinite

Pulsante per ripristinare i valori predefiniti delle impostazioni che dispongono di valori predefiniti. Le impostazioni che non dispongono di valori predefiniti non vengono modificate.

Nome configurazione

Il nome della configurazione a cui si applicano le impostazioni presenti in questa pagina. Questo campo è di sola lettura.

Timeout sessione

Specifica il numero massimo di secondi durante i quali una sessione inattiva rimane valida. Se si imposta 0 o un valore inferiore, le sessioni non scadono mai. Il valore predefinito è 1800.

Proprietà aggiuntive

Proprietà aggiuntive per le sessioni del contenitore Web. Per la descrizione delle proprietà disponibili, vedere Proprietà specifiche delle sessioni del contenitore Web.
