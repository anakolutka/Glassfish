GlassFish 是一款免費的開放原始碼應用程式伺服器，可實作 Java EE 5 平台中的最新功能。Java EE 5 平台包含最新版本的技術，如 JavaServer Pages (JSP) 2.1、JavaServer Faces (JSF) 1.2、Servlet 2.5、Enterprise JavaBeans 3.0、Java API for Web Services (JAX-WS) 2.0、Java Architecture for XML Binding (JAXB) 2.0、Web Services Metadata for the Java Platform 1.0 以及其他許多新技術。


您是 GlassFish 的新手嗎？瞭解如何下載並安裝 GlassFish、嘗試閱覽「入門指南」、查看某些螢幕擷取影片或嘗試指導中的某些範例。請查看 GlassFish 簡介頁面。

下載並執行

瞭解並使用最新 Java EE 5 技術撰寫應用程式

GlassFish 應用程式伺服器實作所有最新的 Java EE 5 技術。例如，可使用最新 Web 層技術 (如 JSF1.2) 撰寫應用程式。除了嘗試使用最新技術外，您還可以在 GlassFish 上嘗試現有的 J2EE 1.4 (或較舊版本) 應用程式，在遷移方面領先一步。
  • 請查看 GlassFish 的入門指南。該指南提供了 GlassFish 伺服器的安裝和執行說明，以及用於部署的小型應用程式範例。
  • 請查看 Java BluePrints 首頁，瞭解適用於 Java EE 5 的內容。該首頁包含用於在 Java EE 5 平台上建立 AJAX 應用程式的藍圖 (BluePrint)。
  • 您可從兩個指導中 (簡單的 Hello World 應用程式和使用 Java EE 5 Persistence 的進階應用程式) 選擇一個，以便使用 NetBeans 開發 Java EE 應用程式。
  • java 文件 是線上提供的，會隨各個升級版本進行更新。
  • 在論壇上提問
  • 查看不斷擴充的技術提示和部落格清單，其中包含在 GlassFish 中使用 Java EE 5 技術的一些範例。
  • 查看 The Aquarium，瞭解有關 GlassFish 相關技術的更多新聞。
