관리 대상 예약된 실행기 서비스를 삭제하는 방법

  1. 탐색 트리에서 [리소스] 노드를 확장합니다.

  2. [리소스] 노드에서 [동시 리소스] 노드를 확장합니다.

  3. [동시 리소스] 노드에서 [관리 대상 예약된 실행기 서비스]를 선택합니다.

    [관리 대상 예약된 실행기 서비스] 페이지가 열립니다.

  4. [관리 대상 예약된 실행기 서비스] 페이지에서 삭제하려는 리소스 이름 옆의 체크 박스를 선택합니다.

  5. [삭제]를 누릅니다.
