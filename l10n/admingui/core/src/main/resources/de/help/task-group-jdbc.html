Verbinden von Anwendungen mit Datenbanken über die JDBC-Technologie

Die JavaTM DataBase Connectivity (JDBCTM)-Technologie bietet Konnektivität mit relationalen Datenbanken. In Enterprise Server konfigurieren Sie Datenbankkonnektivität, indem Sie JDBC-Ressourcen (Datenquellen) zur Domäne hinzufügen. Unter einer Datenquelle versteht man eine auf der Java-Plattform, Enterprise Edition (Java EE-Plattform), basierende standardmäßige Methode zum Konfigurieren der Konnektivität mit einer Datenbank. Jede Datenquelle enthält einen Pool aus Datenbankverbindungen. Anwendungen durchsuchen die Datenquelle im Java Naming and Directory InterfaceTM (J.N.D.I.) API-Baum oder im lokalen Anwendungskontext und rufen dann über die getConnection-Methode eine Datenbankverbindung ab. Datenquellen und die zugehörigen Verbindungspools stellen Prozesse zur Verbindungsverwaltung bereit, die den Systembetrieb aufrechterhalten.
