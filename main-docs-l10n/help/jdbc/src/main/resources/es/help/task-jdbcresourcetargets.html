Para editar destinos de recurso JDBC

  1. En el árbol de navegación, amplíe el nodo Recursos.

  2. Amplíe el nodo JDBC.

  3. Amplíe el nodo Recursos JDBC.

  4. Seleccione el recurso JDBC que desee editar.

  5. En la página Editar recurso JDBC, seleccione el separador Destinos.

  6. Para asignar un nuevo destino al recurso, seleccione el botón Gestionar destinos.

  7. En la página Gestionar destinos, seleccione un destino disponible para agregarlo o eliminarlo de la lista Destinos seleccionados.

  8. Haga clic en Aceptar para aplicar las ediciones.
