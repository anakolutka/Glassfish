Protocolo HTTP

La página Protocolo HTTP contiene las opciones siguientes:

Versión

Versión del protocolo HTTP que se debe utilizar (HTTP/1.0 o HTTP/1.1). El valor predeterminado es HTTP/1.1.

Búsqueda DNS activada

Si se selecciona esta opción, se activará la búsqueda de entrada de DNS para el cliente. Esta opción está desactivada de forma predeterminada.

SSL activado

Si se selecciona esta opción, SSL se utilizará en cualquier escucha que tenga la seguridad activada. Si no se selecciona esta opción, la seguridad estará desactivada completamente en el servidor. Esta opción está seleccionada de forma predeterminada.

Tipo obligado

Tipo de respuesta que se debe utilizar si no hay ninguna asignación MIME disponible que coincida con la extensión. El valor predeterminado es text/html; charset=iso-8859-1.

Tipo predeterminado

El valor predeterminado es text/html; charset=iso-8859-1. El valor es una cadena separada por punto y coma que incluye el tipo de contenido , la codificación, el idioma y el conjunto de caracteres.
