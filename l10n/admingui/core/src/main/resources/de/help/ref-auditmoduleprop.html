Besondere Eigenschaften von Überwachungsmodulen

Die folgende Eigenschaft ist für ein Überwachungsmodul verfügbar.

auditOn

Ein Wert wahr verursacht das Laden des Überwachungsmoduls und sorgt dafür, dass es von der Überwachungsbibliothek in den Überwachungspunkten von Enterprise Server aufgerufen wird. Ein Wert falsch deaktiviert die Überwachungs-Protokollierung für das Modul.
