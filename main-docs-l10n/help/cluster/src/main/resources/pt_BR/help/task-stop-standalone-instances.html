Para Interromper Instâncias do GlassFish Server Standalone

  1. Na árvore de navegação, selecione o nó Instâncias Standalone.

    A página Instância de Servidor Standalone será aberta.

  2. Para cada instância que estiver sendo interrompida, marque a caixa de seleção ao lado da instância na tabela Instâncias do Servidor.

  3. Clique em Interromper.

    Será solicitado que você confirme se deseja interromper as instâncias.

  4. Confirme se deseja interromper as instâncias.

    O status de cada instância é atualizado na tabela Instâncias do Servidor quando a instância for interrompida.
