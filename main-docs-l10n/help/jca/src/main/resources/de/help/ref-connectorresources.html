Connector-Ressourcen

Auf der Seite "Connector-Ressourcen" können Sie Connector-Ressourcen konfigurieren.

Eine Connector-Ressource ist ein Programmobjekt, das eine Anwendung mit einer Verbindung zu einem EIS versorgt.

Für jede Ressource werden die folgenden Informationen zur Verfügung gestellt.

JNDI-Name

Ein eindeutiger Name, der die Connector-Ressource identifiziert.

Aktiviert

"True", wenn die Connector-Ressource aktiviert ist, bzw. "False", wenn sie deaktiviert ist.

Connection-Pool

Der mit der Connector-Ressource verknüpfte Connector Connection-Pool.

Beschreibung

Eine Beschreibung der Connector-Ressource.

Die Tabelle "Ressourcen" enthält darüber hinaus die folgenden Optionen.

Neu

Schaltfläche zum Erstellen einer neuen Connector-Ressource.

Löschen

Schaltfläche zum Löschen von mindestens einer ausgewählten Connector-Ressource.

Aktivieren

Schaltfläche zum Aktivieren von mindestens einer ausgewählten Connector-Ressource.

Deaktivieren

Schaltfläche zum Deaktivieren von mindestens einer ausgewählten Connector-Ressource.

Verwandte Aufgaben

  • So wird eine Connector-Ressource erstellt

  • So wird eine Connector-Ressource bearbeitet

  • So wird eine Connector-Ressource gelöscht

Zugehörige asadmin-Befehle

  • create-connector-resource(1)

  • list-connector-resources(1)

  • delete-connector-resource(1)
