刪除稽核模組

  1. 在瀏覽樹狀結構中，展開「組態」節點。

  2. 選取要組態的執行處理：

    • 若要組態特定的執行處理，請展開該執行處理的組態節點。例如，在預設執行處理 server 展開 server-config 節點。

    • 若要組態所有執行處理的預設設定，請展開 default-config 節點。

  3. 在「組態」節點下，展開「安全性」節點。

  4. 在「安全性」節點下，選取「稽核模組」節點。

    「稽核模組」頁面會隨即開啟。

  5. 在「稽核模組」頁面上，選取您要刪除的稽核模組左側的核取方塊。

  6. 按一下「刪除」。
