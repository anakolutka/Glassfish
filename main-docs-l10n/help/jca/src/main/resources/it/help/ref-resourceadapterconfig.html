Configurazioni Resource Adapter

Utilizzare la pagina Configurazioni Resource Adapter per gestire le configurazioni del connettore (Resource Adapter).

Una configurazione di Resource Adapter contiene le informazioni di configurazione per un Resource Adapter. Console di amministrazione consente di creare una configurazione di Resource Adapter dopo la distribuzione del Resource Adapter. In questo caso il Resource Adapter viene riavviato con la nuova configurazione. Per creare una configurazione di Resource Adapter prima di distribuire il Resource Adapter, usare il comando asadmin create-resource-adapter-config.

Per poter creare un configurazione di Resource Adapter, è necessario creare un pool di thread che verrà usato dal Resource Adapter.

Per ogni configurazione di Resource Adapter vengono fornite le informazioni riportate di seguito.

Nome

Il nome del Resource Adapter.

La tabella Configurazioni contiene inoltre le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare una nuova configurazione di Resource Adapter.

Elimina

Pulsante che consente di eliminare una o più configurazioni di Resource Adapter selezionate.
