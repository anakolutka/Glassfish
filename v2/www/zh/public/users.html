GlassFish 是一款免费的开放源代码应用服务器，可实现 Java EE 5 平台中的最新功能。Java EE 5 平台包含最新版本的技术，如 JavaServer Pages(JSP) 2.1、JavaServer Faces(JSF) 1.2、Servlet 2.5、Enterprise JavaBeans 3.0、Java API for Web Services(JAX-WS) 2.0、Java Architecture for XML Binding(JAXB) 2.0、Web Services Metadata for the Java Platform 1.0，以及许多其他新技术。


对 GlassFish 不熟悉吗？请了解如何下载并安装 GlassFish，尝试阅读入门指南，查看某些截屏视频 (screencast)，或尝试教程中的某些样例。请查看 GlassFish 概述页面。

下载并运行

可以下载 Stable 版本或最新的 Early Access 版本的 GlassFish 应用服务器。下载内容包括二进制版本的 GlassFish Java EE 5 应用服务器，所以您只需遵循简单的说明并运行它即可。提供了适于在所有常用操作系统上运行的下载内容。

了解并使用最新的 Java EE 5 技术编写应用程序

GlassFish 应用服务器实现所有最新的 Java EE 5 技术。例如，可使用最新的 Web 层技术（如 JSF1.2）编写应用程序。除了尝试使用最新技术外，您还可以在 GlassFish 上尝试现有的 J2EE 1.4（或早期版本）应用程序，在迁移方面领先一步。
  • 查看 GlassFish 的入门指南。该指南提供了有关安装并运行 GlassFish 服务器的说明，以及一个用于部署的小型样例应用程序。
  • 查看 Java BluePrints 主页，了解适用于 Java EE 5 的内容。该主页包括用于在 Java EE 5 平台上构建 AJAX 应用程序的计划 (BluePrints)。
  • 使用 NetBeans  开发 Java EE 应用程序。使用两个教程之一开发一个简单的 hello world 应用程序和一个更为高级的、使用 Java EE 5 Persistence 的应用程序。
  • javadocs 是联机提供的，它们会随各个提升版本 (promoted build) 进行更新。
  • 在论坛上提出问题
  • 查看技术提示和博客的增长列表，其中会包含一些在 GlassFish 中使用 Java EE 5 技术的示例。
  • 查看 The Aquarium，了解 GlassFish 相关技术的更多新闻。
