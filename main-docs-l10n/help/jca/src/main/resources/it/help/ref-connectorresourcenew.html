Nuova risorsa connettore

Utilizzare la pagina Nuova risorsa connettore per creare una nuova risorse connettore.

La pagina Nuova risorsa connettore contiene le opzioni riportate di seguito.

Nome JNDI

Il nome univoco che identifica la risorsa connettore.

Nome pool

Il connection pool a cui appartiene la nuova risorsa connettore.

Descrizione

La descrizione della risorsa connettore.

Stato

Se questa opzione è selezionata, la risorsa connettore è abilitata. Questa opzione è abilitata per impostazione predefinita.

Proprietà aggiuntive

Proprietà aggiuntive per la risorsa connettore. Le proprietà disponibili dipendono dal Resource Adapter per il connection pool del connettore. Le coppie nome-valore specificate in questa tabella possono essere utilizzate per sostituire i valori predefiniti per le proprietà specificate dal fornitore del Resource Adapter.

Destinazioni

I cluster e le istanze standalone per la risorsa connettore. Per spostare le destinazioni desiderate nella colonna Destinazioni selezionate, utilizzare i pulsanti Aggiungi e Aggiungi tutto. Per spostare le destinazioni non necessarie nella colonna Destinazioni disponibili, utilizzare i pulsanti Rimuovi e Rimuovi tutto.

Questa opzione viene visualizzata solo se nel dominio sono stati creati cluster o istanze standalone.
