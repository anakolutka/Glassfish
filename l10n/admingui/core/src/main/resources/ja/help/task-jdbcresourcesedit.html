JDBC リソースを編集する

  1. ナビゲーションツリーで、「リソース」ノードを展開します。
  2. 「JDBC リソース」ノードを選択します。
  3. 「JDBC リソース」ページで、編集するリソースの名前をクリックします。

    「JDBC リソースを編集」ページが開きます。

    注 – 「JNDI 名」は読み取り専用フィールドです。J.N.D.I. 名を指定できるのは、新しい JDBC リソースの作成時のみです。

  4. 「プール名」ドロップダウンリストから、この JDBC リソースに関連付ける接続プールを選択します。
  5. 「説明」フィールドで、この JDBC リソースの簡単な説明を入力します。
  6. 「保存」をクリックします。
