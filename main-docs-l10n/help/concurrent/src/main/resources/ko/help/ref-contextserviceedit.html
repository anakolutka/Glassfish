컨텍스트 서비스 편집

[컨텍스트 서비스 편집] 페이지에서는 컨텍스트 서비스 리소스를 편집할 수 있습니다.

[컨텍스트 서비스 편집] 페이지에는 다음 옵션이 포함되어 있습니다.

기본값 로드

기본값을 사용하는 설정을 기본값으로 복원하는 단추입니다. 기본값을 사용하지 않는 설정은 변경되지 않습니다.

JNDI 이름

리소스를 식별하는 고유한 이름입니다. [JNDI 이름]은 읽기 전용 필드입니다. 새 리소스를 생성할 때만 JNDI 이름을 지정할 수 있습니다.

컨텍스트 정보

리소스의 컨텍스트입니다(Classloader, JNDI, Security 또는 WorkArea).

설명

해당 리소스에 대한 설명입니다.

상태

리소스의 상태입니다. 리소스는 사용 또는 사용 안함으로 설정할 수 있습니다. 기본값은 사용입니다.

추가 속성

리소스에 대한 추가 속성입니다. GlassFish Server에서는 컨텍스트 서비스 리소스에 대한 추가 속성을 정의하지 않습니다.
