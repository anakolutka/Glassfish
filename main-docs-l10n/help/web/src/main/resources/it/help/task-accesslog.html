Per configurare il servizio HTTP

  1. Espandere il nodo Configurazione nell'albero di navigazione.

  2. Selezionare la configurazione da modificare.

  3. Nel nodo Configurazione selezionare il nodo Servizio HTTP.

    Viene visualizzata la pagina Servizio HTTP.

  4. Selezionare la casella di controllo Abilitato dell'opzione SSO per abilitare il sistema SSO per impostazione predefinita per tutte le applicazioni Web in tutti i server virtuali di questo server configurate per lo stesso realm.

    È possibile ignorare questa impostazione per un singolo server virtuale. Questa opzione è disabilitata per impostazione predefinita.

  5. Selezionare la casella di controllo Abilitato dell'opzione Log degli accessi per abilitare il log degli accessi per tutti i sottoelementi del server virtuale in cui non è specificata tale proprietà.

    Se la casella di controllo Abilitato dell'opzione Log degli accessi è deselezionata, il log degli accessi viene disabilitato per tutti i sottoelementi del server virtuale in cui non è specificata tale proprietà. Questa opzione è disabilitata per impostazione predefinita.

  6. Selezionare la casella di controllo Abilitato dell'opzione Rotazione per abilitare la rotazione dei file di log.

    Questa opzione è abilitata per impostazione predefinita.

  7. Nell'elenco a discesa Criteri di rotazione selezionare un criterio di rotazione.

    L'unico criterio di rotazione disponibile è time.

  8. Nel campo Intervallo di rotazione digitare il numero di minuti che intercorrono tra le rotazioni del log degli accessi.

    Questo campo è valido solo se il criterio di rotazione è time. Il valore predefinito è 1.440 minuti (24 ore).

  9. Nel campo Suffisso rotazione digitare il valore di stringa da aggiungere alla fine del nome file di log in seguito alla rotazione.

    Il valore predefinito è yyyy-MM-dd.

    Assicurarsi che il suffisso di rotazione contenga valori sufficienti per la creazione di un suffisso univoco per ogni rotazione di log. Ad esempio, se il log di accesso viene ruotato più di una volta al giorno, includere l'ora del giorno nel suffisso di rotazione.

    Il suffisso univoco consente di assicurare che venga eseguita la rotazione del file di log di accesso a ogni scadenza dell'intervallo di rotazione. Se il suffisso di rotazione del nuovo file di log degli accessi è identico al suffisso di rotazione del file di log degli accessi esistente, la rotazione del file non viene eseguita.

  10. Nel campo Numero massimo di file digitare il numero massimo di file di log degli accessi di cui è stata eseguita la rotazione da conservare.

    Un valore negativo indica che il numero dei file è illimitato. Il valore predefinito è -1.

  11. Nel campo Dimensione buffer digitare le dimensioni in byte del buffer del log degli accessi.

    Un valore minore o uguale a 0 consente di disabilitare l'inserimento in buffer del log degli accessi. Il valore predefinito è 32768.

  12. Nel campo Intervallo di scrittura digitare il numero di secondi al termine dei quali il log viene scritto sul disco.

    Il log degli accessi viene scritto quando il buffer è pieno o allo scadere dell'intervallo di tempo previsto. Se il valore è minore o uguale a 0, il buffer viene sempre scritto, anche se non è pieno. Ciò significa che ogni volta che si accede al server, i messaggi del log vengono memorizzati direttamente nel file. Il valore predefinito è 300.

  13. Nel campo Formato digitare il valore di stringa per specificare le seguenti proprietà del log degli accessi:

    • le informazioni da acquisire nel log degli accessi;

    • l'ordine di acquisizione delle informazioni.

    Di seguito è riportato il valore predefinito.

    %client.name% %auth-user-name% %datetime% %request% %status% %response.length%

    Nella stringa del formato è possibile includere i valori riportati di seguito.

    %attribute.name%

    Registra il valore (mediante la chiamata del metodo toString rispettivo) dell'attributo della richiesta con il valore name specificato.

    %auth-user-name%

    Nome dell'utente autorizzato

    %client.name%

    Nome host client

    %client.dns%

    DNS client

    %cookie.name%

    Il valore del cookie di richiesta con il valore name specificato

    %cookie.value%

    Il valore del primo cookie trovato nella richiesta

    %datetime%

    Data di sistema

    %request%

    Riga di richiesta HTTP completa

    %status%

    Stato

    %response.length%

    Lunghezza contenuto risposta

    %header.referer%

    Intestazione Referer

    %header.user-agent%

    Intestazione User-Agent

    %http-method%

    Metodo HTTP

    %http-uri%

    URI HTTP

    %query-str%

    Stringa query HTTP

    %http-version%

    Versione protocollo HTTP

    %header.accept%

    Intestazione Accept

    %header.date%

    Intestazione Date

    %header.if-mod-since%

    Intestazione If-Modified-Since

    %header.auth%

    Intestazione Authorization

    %header.any%

    Qualsiasi valore di intestazione HTTP valido definito nella RFC 2616 (anche any è un valore di intestazione valido; qui è specificato come variabile)

    %session.name%

    Il valore dell'attributo di sessione con nome name o NULL-SESSION-ATTRIBUTE-name se l'attributo denominato non esiste nella sessione o NULL-SESSION se la sessione non esiste

    %time-taken%

    Il tempo necessario per eseguire ogni richiesta in millisecondi

    %vs.id%

    ID server virtuale

  14. Nella sezione Proprietà aggiuntive, specificare le proprietà aggiuntive.

    Per la descrizione delle proprietà disponibili, vedere Proprietà specifiche del servizio HTTP.

    • Per aggiungere una proprietà, fare clic sul pulsante Aggiungi proprietà. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Nome e il valore della proprietà nel campo Valore.

    • Per modificare una proprietà, è necessario modificare il valore del campo Valore della proprietà.

    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Nome corrispondente alla proprietà da eliminare, quindi fare clic sul pulsante Elimina proprietà.

  15. Fare clic su Salva.

    Suggerimento:

    Per ripristinare le impostazioni predefinite, fare clic sul pulsante Carica impostazioni predefinite, quindi su Salva.
