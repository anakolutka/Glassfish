Monitoraggio server

Utilizzare la pagina Monitoraggio server per visualizzare i dati di monitoraggio per l'istanza server.

La pagina Monitoraggio server contiene le informazioni riportate di seguito.

Nome istanza

Il nome dell'istanza server.

Visualizza

Il server virtuale, listener HTTP, Garbage Collector o sistema di thread per il quale vengono visualizzate le statistiche di monitoraggio.

Per una descrizione delle proprietà monitorate, vedere il Capitolo 8, Amministrazione del servizio di monitoraggio, in Oracle GlassFish Server Administration Guide.
