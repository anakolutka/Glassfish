Para exibir dados de monitoramento do servidor

Antes de começar

Antes de você poder exibir os dados de monitoramento, é preciso configurar o monitoramento. Consulte Para configurar o serviço de monitoramento para obter os detalhes.

  1. Na árvore de navegação, selecione o nó Enterprise Server.

    A página Informações Gerais é exibida.

  2. Na página Informações Gerais, clique na guia Monitor.

    A página Monitoramento do Aplicativo é exibida.

  3. Clique na guia Servidor.

    A página Monitoramento do Servidor é exibida.

  4. Na lista suspensa Servidor Virtual, selecione o servidor virtual para visualizar os dados de monitoramento desse servidor.
  5. Clique no botão Obter Estatísticas para recuperar as estatísticas de monitoramento.

    As estatísticas de monitoramento aparecem na tabela.

  6. (Opcional) Clique no botão Atualizar para atualizar os dados de monitoramento exibidos.
