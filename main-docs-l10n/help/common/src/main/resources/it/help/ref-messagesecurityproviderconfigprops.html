Proprietà specifiche per la configurazione di provider di sicurezza dei messaggi

Per una configurazione di provider di sicurezza dei messaggi sono disponibili le proprietà riportate di seguito.

server.config

La directory e il nome del file XML che contiene informazioni relative alla configurazione del server. Il valore predefinito è domain-dir/config/wss-server-config-1.0.xml.

debug

Se l'impostazione è true, consente di eseguire il dump dei messaggi di debug del provider server nel log del server. Il valore predefinito è false.

dynamic.username.password

Se impostato su true, comunica al runtime del provider di raccogliere il nome utente e la password da CallbackHandler per ogni richiesta. Se l'impostazione è false, il nome utente e la password di wsse:UsernameToken(s) vengono raccolti una volta sola durante l'inizializzazione del modulo. Questa proprietà è applicabile solo per un ClientAuthModule. Il valore predefinito è false.

encryption.key.alias

Specifica la chiave di cifratura utilizzata dal provider. La chiave viene identificata dall'alias keystore corrispondente. Il valore predefinito è s1as.

signature.key.alias

Specifica la chiave di firma utilizzata dal provider. La chiave viene identificata dall'alias keystore corrispondente. Il valore predefinito è s1as.
