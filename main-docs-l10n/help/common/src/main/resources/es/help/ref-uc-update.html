Actualizaciones disponibles

La página Actualizaciones disponibles ofrece información sobre las últimas versiones de los componentes de la imagen de instalación que están instalados.

Para instalar las actualizaciones disponibles, primero debe parar el dominio de GlassFish Server. A continuación, utilice uno de estos comandos:

  • glassfish3/bin/updatetool, que inicia la utilidad gráfica Herramienta de actualización

  • glassfish3/bin/pkg, una versión de línea de comandos de la herramienta de actualización

Para obtener más información sobre el uso del comando pkg, consulte el Capítulo 11, sobre ampliación y actualización de GlassFish Server, en Oracle GlassFish Server Administration Guide. La utilidad Herramienta de Actualización tiene su propia ayuda en pantalla.

Reinicie el servidor después de instalar las actualizaciones.

La página Actualizaciones disponibles contiene las siguientes opciones.

Ruta de instalación

Ruta de acceso del sistema de archivos del directorio raíz o nivel superior de la imagen de instalación.

Actualizaciones disponibles

Para cada componente para el que haya disponible una versión más reciente, en esta tabla se proporciona la siguiente información.

Componente

Nombre del componente.

Categoría

Categoría del software al que se asocia el componente, por ejemplo, servidores de aplicaciones, herramientas del sistema y bases de datos. El autor del componente le asigna la categoría al mismo cuando se crea el componente.

Versión Instalada

Número de versión del paquete en el que se proporciona el componente actual.

Nueva versión

Número de versión de la actualización disponible.

Cambio

Diferencia de tamaño entre la versión instalada y la actualización disponible.

Origen

Nombre del Repositorio de red del que se obtendrá el componente.
