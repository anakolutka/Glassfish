To Configure Monitoring for a Web Service

  1. In the tree view, expand the Web Services node.
  2. Select the web service for which you are configuring monitoring.
  3. Click the Monitor tab. Then click the Config tab.
  4. From the Monitoring Level drop down list, select the level. The following options are available:

    • LOW- Enables monitoring for the whole web service. No method level monitoring will be done.

    • HIGH- Message Trace is also enabled in addition to enabling number of requests per second, average response time, and throughput attributes.

    • OFF- Disables monitoring.

  5. Enter a value for the Message History. Default is 25.

    This is the total number of messages that will be displayed for a web service.

  6. Click the Reset button to clear all statistics and the running averages are restarted.
