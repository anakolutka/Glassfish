JACC-Anbieter

Die Seite JACC-Anbieter zeigt eine Liste verfügbarer Anbieter von Java Authorization Contract for Containers (JACC) an.

JACC ist der Teil der Java EE Spezifikation, der eine Schnittstelle für plugfähige Autorisierungsanbieter definiert. Dies ermöglicht es dem Administrator, Plugin-Module von Drittanbietern einzurichten, um die Autorisierung durchzuführen. Enterprise Server bietet standardmäßig ein einfaches JACC-fähiges dateibasiertes Autorisierungsmodul.

Informationen zum Schreiben der Implementierungsklassen für einen JACC-Anbieter finden Sie unter der JACC-Spezifikation unter http://jcp.org/en/jsr/detail?id=115.

Für jeden JACC-Anbieter werden die folgenden Informationen bereitgestellt.

Name

Der Name des JACC-Anbieters.

Richtlinienanbieter

Der Name der Klasse, die die Richtlinien-Factory implementiert.
