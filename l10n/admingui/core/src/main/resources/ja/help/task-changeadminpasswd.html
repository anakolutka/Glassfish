管理パスワードを変更する

管理パスワードを暗号化すると、アプリケーションのセキュリティーが向上します。セキュリティー設定手順の詳細については、『Sun GlassFish Enterprise Server v3 Prelude Administration Guide』の第 6 章「Administering System Security」を参照してください。

  1. ナビゲーションツリーで、「設定」ノードを展開します。
  2. ドメインがクラスタをサポートするように設定されている場合は、変更する設定を選択します。

    たとえば、クラスタ プロファイルや エンタープライズ プロファイルで作成されるドメインなどです。

  3. 「セキュリティー」ノードを開きます。
  4. 「レルム」ノードを開きます。
  5. 「admin-realm」ノードを選択します。
  6. 「レルムを編集」ページで、「ユーザーを管理」ボタンをクリックします。
  7. admin という名前のユーザーを選択します。
  8. 新しいパスワードを入力し、そのパスワードをもう一度入力します。
  9. 「保存」をクリックします。
