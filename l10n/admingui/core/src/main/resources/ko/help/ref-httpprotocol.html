HTTP 프로토콜

HTTP 프로토콜 페이지에는 다음 옵션이 포함되어 있습니다.

버전

사용된 HTTP 프로토콜 버전입니다(HTTP/1.0 또는 HTTP/1.1). 기본값은 HTTP/1.1입니다.

DNS 조회 사용 가능

이 옵션을 선택하면 클라이언트의 DNS 항목 조회가 활성화됩니다. 이 옵션은 기본적으로 선택 취소됩니다.

SSL 사용 가능

이 옵션을 선택하면 보안이 활성화된 모든 수신기에 대해 SSL이 사용됩니다. 이 옵션을 선택 취소하면 서버에서 전역적으로 보안이 비활성화됩니다. 이 옵션은 기본적으로 선택됩니다.

강제 유형

확장자에 일치하는 사용 가능한 MIME 매핑이 없을 때 사용되는 응답 유형입니다. 기본값은 text/html; charset=iso-8859-1입니다.

기본 유형

기본값은 text/html; charset=iso-8859-1입니다. 값은 content-type, encoding, language 및 charset이 세미콜론으로 구분된 문자열입니다.
