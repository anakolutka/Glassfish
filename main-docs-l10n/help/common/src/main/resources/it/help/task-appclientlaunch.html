Per avviare un client di applicazione

Prima di iniziare

Assicurarsi che il client di applicazione da avviare sia abilitato. Il client deve essere un client Web che può essere eseguito con Java Web Start.

  1. Selezionare il nodo Applicazioni nell'albero di navigazione.

    Viene visualizzata la pagina Applicazioni.

  2. Nella tabella Applicazioni distribuite fare clic sul nome del client di applicazione da avviare.

    Il nome può includere un identificativo di versione opzionale, che segue il nome ed è separato da questo mediante due punti (:). Per ulteriori informazioni sulle versioni di moduli e applicazioni, vedere Module and Application Versions in Oracle GlassFish Server Application Deployment Guide.

    Viene visualizzata la pagina Modifica applicazione.

  3. Verificare che la casella di controllo Java Web Start sia selezionata.

    Java Web Start deve essere abilitato per poter avviare un client di applicazione. Se la casella di controllo Java Web Start non è selezionata, selezionarla, quindi fare clic su Salva.

  4. Nella tabella Moduli e componenti fare clic su Avvia nella colonna Azione.

    Viene visualizzata la pagina di avvio del client dell'applicazione.

  5. Nella pagina di avvio del client dell'applicazione digitare gli argomenti da passare al client di applicazione.

  6. Fare clic su Avvia.
