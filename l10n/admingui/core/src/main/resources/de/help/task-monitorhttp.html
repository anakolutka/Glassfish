So zeigen Sie die HTTP-Dienstüberwachungsdaten an

  1. Wählen Sie in der Navigationsstruktur den Anwendungsserverknoten.
  2. Klicken Sie auf der Seite "Allgemeine Informationen" auf die Registerkarte "Überwachen".
  3. Klicken Sie auf der Seite "Webcontainerüberwachung" auf die Registerkarte "HTTP-Dienst".

    Die Seite "HTTP-Dienstüberwachung" wird geöffnet.

  4. (Optional) Wählen Sie aus der Dropdown-Liste "Virtueller Server" einen Server aus, um die Anforderungsstatistik für diesen Server anzuzeigen.
  5. (Optional) Wählen Sie aus der Dropdown-Liste "HTTP-Zielgeräte" ein Zielgerät aus, um die Anforderungsstatistik für dieses Zielgerät auszuwählen.
  6. (Optional) Klicken Sie auf die Schaltfläche "Aktualisieren", um die angezeigten Überwachungsdaten zu aktualisieren.
