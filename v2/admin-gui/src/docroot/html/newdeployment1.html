To Deploy a Service Assembly

Deploying a Service Assembly automatically starts all the components targeted with the Service Units in this Service Assembly, if the Auto Start property on the JBI Runtime Configuration tab is enabled.

  1. In the Service Assemblies page, click Deploy.

    The Deploy JBI Services Assembly page is displayed.

  2. Select the location of the Service Assembly archive you want to deploy and click Next.

    You can either upload a Service Assembly archive from your local file system or copy a Service Assembly archive from the Application Server host's file system. Use the Browse Files button to navigate to the directory where the archive is located and select the archive. This archive can be a .jar or a .zip file.

  3. Click the Enabled checkbox if you want to start the Service Assembly immediately after it is deployed.
  4. Click Finish.

    The Service Assemblies page lists the deployed Service Assembly.
