To Change Load Balancer General Information

  1. In the tree component, select the HTTP Load Balancers node.

  2. In the right pane, click the load balancer to edit.

    The General information page appears.

  3. To set up and test the connection to the load balancer device (for example the Sun Java System Web Server:

    1. Enter the hostname of the web server or other load balancer device.

    2. Enter the port used to communicate with the load balancing device. It must be SSL enabled.

    3. Enter the proxy host used for outbound HTTP.

    4. Enter the proxy port used for outbound HTTP.

    5. Click Test Connection to test the connection to the load balancer device.

  4. To change the settings for the load balancer:

    1. To communicate changes to the load balancer automatically to the web server or other load balancing device, click the Enable checkbox.

      The device settings must be specified and the connection working for changes to be automatically applied.

    2. Enter the reload poll interval, which sets how often the load balancer checks for an updated configuration.

      When the check detects changes, the configuration file is reloaded. A value of 0 disables reloading.

    3. Click the checkbox to enable HTTPS routing.

      If enabled, HTTPS requests to the load balancer result in HTTPS requests to the server instance. If disabled, HTTPS requests to the load balancer result in HTTP requests to the server instance.

    4. Enter the response timeout in seconds.

      A server instance must return a response within this time. If no response is received within the time period, the server is considered unhealthy. If set to a positive number, and the request is idempotent, the request is retried. If the request is not idempotent, an error page is returned. If set to 0 no timeout is used. The default is 60.

  5. To add, change or delete properties:

    1. To add a property, click Add Property and fill in the name and value.

    2. To delete a property, click the checkbox next to the property name and click Delete Properties.

  6. Click Save.
