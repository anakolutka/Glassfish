Ejecuciones de trabajo por lotes

GlassFish Server proporciona un tiempo de ejecución por lotes para la programación y ejecución de trabajos por lotes. Los trabajos por lotes suelen ser de larga ejecución, tareas orientadas a lotes que contienen varios pasos. Las aplicaciones por lotes envían trabajos al tiempo de ejecución por lotes y proporcionan instrucciones sobre cómo y cuándo ejecutar los pasos. El tiempo de ejecución por lotes procesa los pasos y almacena información sobre los trabajos en un repositorio de trabajo. En GlassFish Server, el repositorio de trabajo es una base de datos.

Utilice la página Ejecuciones de trabajo por lotes para ver los detalles sobre las ejecuciones de trabajo por lotes registradas.

La página Ejecuciones de trabajo por lotes contiene la siguiente información.

Nombre de la instancia
Nombre del cluster

Nombre de la instancia de servidor o cluster de GlassFish al que se aplica la configuración.

Trabajos por lotes

Lista de ejecuciones de trabajo por lotes registradas para la instancia o el cluster. Para cada ejecución, se muestra la información siguiente:

ID de ejecución

ID asignado para la ejecución del trabajo por lotes. Al hacer clic en el ID de ejecución se abre la página Detalles de ejecución de trabajo por lotes. La primera vez que se inicia un trabajo y cada vez que se reinicia la ejecución existente, se crea una nueva ejecución.

Nombre del trabajo

Nombre del trabajo.

Estado de lote

Estado de la ejecución tal y como lo ha definido el tiempo de ejecución por lotes.

Estado de salida

Estado de la ejecución tal y como lo ha definido el XML de trabajo o la aplicación por lotes. Por defecto, el estado de salida y el estado de lotes son el mismo a menos que el estado de salida se sustituya de forma explícita.

Identificador de instancia

Identificador asignado a la instancia del trabajo por lotes. La instancia de un trabajo por lotes puede tener varias ejecuciones.

Hora de inicio

Hora a la que se inicia la ejecución.

Hora de finalización

Hora a la que termina la ejecución.
