Fournisseurs JACC

La page Fournisseurs JACC affiche une liste des fournisseurs JACC (Java Authorization Contract for Containers).

JACC fait partie de la spécification Java EE qui définit une interface pour les fournisseurs d'autorisation enfichables. Cela permet à l'administrateur de configurer des modules de branchement tiers pour appliquer l'autorisation. Par défaut, le Enterprise Server fournit un moteur d'autorisation simple, basé sur le fichier et compatible JACC.

Pour en savoir plus sur l'écriture de classes d'implémentation pour un fournisseur JACC, consultez la spécification JACC sur http://jcp.org/en/jsr/detail?id=115.

Pour chaque fournisseur JACC, les informations suivantes sont fournies.

Nom

Nom du fournisseur JACC.

Fournisseur de stratégies

Nom de la classe qui implémente la fabrique de stratégies.
