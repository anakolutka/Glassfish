新管理員密碼

使用「新管理員密碼」頁面可變更目前管理員帳號的密碼。

備註:

如果啟用安全管理，則不能將密碼變更為空白值。

「新管理員密碼」頁面包含下列選項。

使用者 ID

目前管理員的使用者 ID。此為唯讀欄位。

群組清單

管理員所屬的群組。此為唯讀欄位。此值為 asadmin。

新密碼

管理員的新密碼。

確認新密碼

管理員的新密碼。

相關作業

變更管理員密碼

相關的 asadmin 命令

change-admin-password(1)
