Editar propiedades de pool de conexiones de conector

Utilice la página Editar propiedades de pool de conexiones de conector para modificar las propiedades de un pool de conexiones del conector.

La página Editar propiedades de pool de conexiones de conector presenta las opciones siguientes:

Propiedades Adicionales

Propiedades adicionales para el pool de conexiones de conector. Las propiedades disponibles dependen del adaptador de recursos para el pool de conexiones de conector. Los pares nombre-valor especificados en esta tabla se pueden usar para sustituir los valores por defecto de las propiedades definidas por el proveedor del adaptador de recursos.

Si no especifica un valor para una propiedad, ésta se elimina de la tabla al finalizar la edición del pool de conexiones. Si no desea definir un valor para una propiedad ahora, pero desea que la propiedad permanezca en la tabla para que el valor se pueda definir más adelante, introduzca un juego de paréntesis vacío en el campo Valor:

()
