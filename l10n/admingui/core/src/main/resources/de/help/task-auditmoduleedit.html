So bearbeiten Sie ein Überwachungsmodul

Überwachungsmodule sind standardmäßig nicht aktiviert. Weitere Informationen zum Aktivieren von Überwachungsmodulen finden Sie unter So aktivieren oder deaktivieren Sie die Überwachungsprotokollierung.

  1. Erweitern Sie den Knoten "Konfigruationen".
  2. Wählen Sie die zu konfigurierende Instanz:
    • Zum Konfigurieren einer bestimmten Instanz erweitern Sie den config-Knoten der Instanz. Für die Standardinstanz server erweitern Sie z. B. den Knoten server-config.
    • Zum Konfigurieren der Standardeinstellungen für alle Instanzen erweitern Sie den Knoten default-config.
  3. Erweitern Sie den Knoten "Sicherheit".
  4. Erweitern Sie den Knoten "Überwachungsmodule".
  5. Klicken Sie auf den Knoten des Überwachungsmoduls, das bearbeitet werden soll.
  6. Ändern Sie bei Bedarf auf der Seite "Überwachungsmodul bearbeiten" den Klassennamen.
  7. Fügen Sie weitere Eigenschaften für das Modul hinzu, indem Sie auf "Hinzufügen" klicken und den Namen und den Wert einer Eigenschaft eingeben.

    Eine der zulässigen Eigenschaften ist auditOn, welche angibt, ob diese Implementierungsklasse aktiviert ist oder nicht. Zulässige Werte sind true und false.

  8. Ändern Sie vorhandene Eigenschaften, indem Sie den zu ändernden Namen oder Wert auswählen und die Änderung direkt in das Textfeld eingeben.
  9. Löschen Sie eine Eigenschaft, indem sie das Kontrollkästchen links neben der Eigenschaft auswählen und auf "Eigenschaften löschen" klicken.
  10. Zum Speichern der Änderungen klicken Sie auf "Speichern", oder klicken Sie auf die Zurück-Schaltfläche des Browsers, um den Vorgang abzubrechen, ohne die Änderungen zu speichern.
