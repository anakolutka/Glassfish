Affichage, déploiement et gestion des applications Web

L'affichage, le déploiement et la gestion des applications Web comprennent les tâches suivantes :

  • Déployer une application Web

  • Modifier une application Web déployée

  • Afficher les descripteurs d'une application Web

  • Lancer une application Web

  • Redéployer une application Web déployée
