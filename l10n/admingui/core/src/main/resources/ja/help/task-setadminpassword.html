管理者パスワードを変更する

  1. ナビゲーションツリーで、Enterprise Server ノードを選択します。

    「一般情報」ページが開きます。

  2. 「管理者パスワード」タブを選択します。

    このページには、現在の管理者アカウントのユーザー ID とグループ名が表示されます。デフォルトでは、ユーザー ID は admin、グループ名は asadmin です。

  3. 「新しいパスワード」フィールドに、現在の管理者アカウントの新しいパスワードを入力します。
  4. 「新しいパスワードを確認」フィールドに、新しいパスワードを再度入力します。
  5. 「保存」をクリックします。
