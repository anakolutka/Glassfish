Archivar usuarios

Administre los usuarios del dominio file por medio de la Consola de administración. <Check Alignment of PHs>Los usuarios y grupos del dominio file aparecen en el archivo de claves, cuya ubicación está especificada por la propiedad file.

Nota - También es posible seguir estos pasos para agregar usuarios a cualquier dominio de archivo, incluido admin-realm. Sólo tiene que sustituir el nombre del dominio de destino en el dominio file indicado en esta sección.

Los usuarios del dominio file pueden pertenecer a un grupo Java, una categoría de usuarios clasificados por rasgos en común. <Check Alignment of PHs>Por ejemplo, los clientes de una aplicación de comercio electrónico pueden pertenecer al grupo CLIENTE, pero los que más gastan pertenecerían al grupo PREFERENTE. La clasificación de usuarios en grupos simplifica el control del acceso de grandes números de usuarios.

Inicialmente, tras la instalación de Servidor empresarial, el único usuario es el administrador introducido durante la instalación. Por defecto, este usuario pertenece al grupo asadmin, en el dominio admin-realm, lo que le da derecho a modificar Servidor empresarial. Los usuarios asignados a este grupo tendrán privilegios de administrador, es decir que podrán acceder a la herramienta asadmin y a la Consola de administración.

La página Usuarios del archivo muestra una lista de usuarios del dominio seleccionado. Para cada usuario aparece la siguiente información:

Id. de usuario

El nombre de usuario.

Lista de grupos

Los grupos a los que pertenece el usuario.
