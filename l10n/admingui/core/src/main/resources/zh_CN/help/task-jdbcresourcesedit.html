编辑 JDBC 资源

  1. 在导航树中，展开“资源”节点。
  2. 选择“JDBC 资源”节点。
  3. 从“JDBC 资源”页面中，单击要编辑的资源的名称。

    此时将打开“编辑 JDBC 资源”页面。

    注 – “J.N.D.I. 名称”是只读字段。只能在创建新的 JDBC 资源时指定 J.N.D.I. 名称。

  4. 从“池名称”下拉式列表中，选择要与此 JDBC 资源关联的连接池。
  5. 在“说明”字段中，键入此 JDBC 资源的简短说明。
  6. 单击“保存”。
