Servidores virtuales

Un servidor virtual, a veces llamado host virtual, es un objeto que permite al mismo servidor físico alojar varios nombres de dominio de Internet. Todos los servidores virtuales alojados en el mismo servidor físico comparten la dirección de Protocolo de Internet (IP) de dicho servidor físico. Un servidor virtual asocia el nombre de un dominio de un servidor (como www.sun.com) con el servidor particular en el que se ejecuta Servidor empresarial.
