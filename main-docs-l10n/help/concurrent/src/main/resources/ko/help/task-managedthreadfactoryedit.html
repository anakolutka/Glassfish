관리 대상 스레드 팩토리를 편집하는 방법

  1. 탐색 트리에서 [리소스] 노드를 확장합니다.

  2. [리소스] 노드에서 [동시 리소스] 노드를 확장합니다.

  3. [동시 리소스] 노드에서 [관리 대상 스레드 팩토리]를 선택합니다.

    [관리 대상 스레드 팩토리] 페이지가 열립니다.

  4. [관리 대상 스레드 팩토리] 페이지에서 편집할 리소스의 이름을 누릅니다.

    [관리 대상 스레드 팩토리 편집] 페이지가 열립니다.

    주:

    [JNDI 이름]은 읽기 전용 필드입니다. 새 리소스를 생성할 때만 JNDI 이름을 지정할 수 있습니다.

  5. XXX

  6. [설명] 필드에 리소스에 대한 간단한 설명을 입력합니다.

    이 설명은 최대 250자가 포함될 수 있는 문자열 값입니다.

  7. 리소스를 사용으로 설정하려면 [상태 사용] 체크 박스를 선택합니다.

  8. [추가 속성] 섹션에서 리소스에 대한 추가 속성을 지정합니다.

    GlassFish Server에서는 관리 대상 스레드 팩토리 리소스에 대한 추가 속성을 정의하지 않습니다.

    • 속성을 추가하려면 [속성 추가] 단추를 누릅니다. 빈 행이 표시되면 [이름] 필드에 속성 이름을 입력하고 [값] 필드에 속성값을 입력합니다.

    • 속성을 수정하려면 속성의 [값] 필드를 편집합니다.

    • 속성을 삭제하려면 삭제할 속성의 [이름] 필드 왼쪽에 있는 체크 박스를 선택한 다음 [속성 삭제]를 누릅니다.

  9. [저장]을 누릅니다.
