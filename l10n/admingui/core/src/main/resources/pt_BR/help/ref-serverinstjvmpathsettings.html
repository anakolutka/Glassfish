Configurações de Caminho de Classe de JVM

O caminho de classe é a lista de arquivos JAR que o ambiente de tempo de execução JAVA pesquisa para detectar classes e outros recursos.

A página Configurações de Caminho de Classe de JVM contém as seguintes opções.

Caminho de Classe de Ambiente

Se a caixa de seleção Ignorar estiver marcada, a variável de ambiente CAMINHO DE CLASSE será ignorada. A variável de ambiente CAMINHO DE CLASSE é conveniente para tutoriais básicos de programação, mas não é recomendada para ambientes empresariais.

Caminho de Classe de Servidor

O caminho de classe do Enterprise Server. Este é um campo somente leitura.

Caminho de Classe do Sistema

O caminho para as classes no seu sistema.

Prefixo de Caminho de Classe

Caminhos completos para os arquivos JAR a serem colocados no início do caminho de classe do servidor, um por linha.

Sufixo de Caminho de Classe

Caminhos completos para os arquivos JAR a serem anexados ao caminho de classe do servidor, um por linha.

Prefixo de Caminho da Biblioteca Nativa

Caminhos completos para os arquivos JAR a serem colocados no início do caminho da biblioteca nativa, um por linha.

Sufixo de Caminho da Biblioteca Nativa

Caminhos completos para os arquivos JAR a serem anexados ao caminho da biblioteca nativa, um por linha.

O caminho da biblioteca nativa é uma concatenação do caminho relativo do servidor para as respectivas bibliotecas compartilhadas, o caminho da biblioteca nativa do ambiente de tempo de execução padrão, a configuração do ambiente de shell (LD_LIBRARY_PATH nos sistemas UNIX®) e qualquer caminho especificado na página Configurações do Gerador de Perfil de JVM.
