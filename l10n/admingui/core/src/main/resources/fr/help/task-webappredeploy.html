Redéployer une application Web déployée

Le redéploiement d'une application Web déploie une nouvelle version de l'application tout en conservant les paramètres et les options avec lesquels l'application a été déployée initialement.

Avant de commencer

Assurez-vous que l'application Web à redéployer est déjà déployée.

  1. Dans l'arborescence de navigation, développez le nœud Applications.
  2. Sous le nœud Applications, sélectionnez le nœud Applications Web.

    La page Applications Web s'ouvre.

  3. Dans le tableau Applications Web déployées, cliquez sur Redéployer sur la ligne de l'application que vous voulez redéployer.

    La page Redéploiement des applications/modules d'entreprise s'affiche.

  4. Pour conserver puis restaurer les sessions actives de l'application à la fin du redéploiement, cochez la case Conserver les sessions.

    Si la conservation ou la restauration d'une session active de l'application échoue, aucune des sessions ne sera disponible une fois le redéploiement terminé. Toutefois, le redéploiement se poursuit et un avertissement est enregistré.

  5. Indiquez l'emplacement de l'archive qui contient l'application à redéployer.
    • Si l'archive est un fichier stocké sur la machine client ou accessible depuis cette dernière, sélectionnez l'option Fichier empaqueté à télécharger sur le serveur.

      L'ordinateur client est l'hôte sur lequel vous visualisez la Console d’administration via un navigateur.

      Cliquez sur Parcourir pour accéder au fichier ou tapez le chemin d'accès complet au fichier.

    • Si l'archive est un fichier stocké sur la machine serveur ou une application non empaquetée dans un répertoire éclaté, sélectionnez l'option Fichier local empaqueté ou Répertoire accessible depuis le serveur d'application.

      La machine serveur est l'hôte sur lequel fonctionne le serveur d'administration du domaine.

      Saisissez le nom du chemin d'accès complet au fichier ou au répertoire. Le déploiement à partir d'un répertoire éclaté est réservé aux développeurs expérimentés et est déconseillé pour les environnements de production.

  6. Cliquez sur OK.
