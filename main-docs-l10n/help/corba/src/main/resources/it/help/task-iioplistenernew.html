Per creare un listener IIOP

  1. Espandere il nodo Configurazione nel componente della struttura.

  2. Selezionare l'istanza da configurare:

    • Per configurare un'istanza particolare, selezionare il nodo di configurazione dell'istanza. Ad esempio, per l'istanza predefinita server, selezionare il nodo server-config.

    • Per configurare le impostazioni predefinite per le istanze future che utilizzano una copia di default-config, selezionare il nodo default-config.

  3. Nel nodo Configurazione espandere il nodo Sicurezza.

  4. Selezionare Listener IIOP.

    Viene visualizzata la pagina Listener IIOP.

  5. Fare clic su Nuovo.

  6. Nel campo Nome digitare un nome per identificare il listener.

  7. Nel campo Indirizzo di rete digitare l'indirizzo di rete del listener.

    È possibile digitare un indirizzo IP o un nome host risolvibile da DNS.

  8. Nel campo Porta listener digitare il numero della porta su cui il listener IIOP è in ascolto.

  9. Selezionare la casella di controllo Abilitato dell'opzione Listener per abilitare le connessioni in entrata IIOP ORB per GlassFish Server.

    Questa opzione è abilitata per impostazione predefinita.

  10. Selezionare la casella di controllo Abilitato dell'opzione Sicurezza per abilitare SSL per il listener IIOP.

    Questa opzione è disabilitata per impostazione predefinita.

  11. Nella sezione Proprietà aggiuntive, specificare le proprietà aggiuntive.

    GlassFish Server non definisce alcuna proprietà aggiuntiva per i listener IIOP.

    Per aggiungere una proprietà, fare clic sul pulsante Aggiungi proprietà. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Nome e il valore della proprietà nel campo Valore.

  12. Fare clic su OK.

  13. Se è stata selezionata la casella di controllo Abilitato dell'opzione Sicurezza, attenersi alla procedura descritta in Per modificare le impostazioni SSL per un listener IIOP sicuro.
