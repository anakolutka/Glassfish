Para editar una sesión JavaMail

  1. En el árbol de navegación, amplíe el nodo Recursos.

  2. En el nodo Recursos, seleccione Sesiones JavaMail.

    Se abrirá la página Sesiones JavaMail.

  3. En la página Sesiones JavaMail, haga clic en el nombre de la sesión que esté editando.

    Se abrirá la página Editar Sesión JavaMail.

  4. En el campo Host de Correo, especifique el nombre de host del servidor de correo por defecto.

    Los métodos de conexión de los objetos de almacenamiento y transporte utilizan este valor si no se suministra una propiedad de host específica del protocolo. El nombre debe poder resolverse en un nombre de host real.

  5. En el campo Usuario por Defecto, escriba otro nombre de usuario que se debe proporcionar al conectarse a un servidor de correo electrónico.

    Los métodos de conexión de los objetos de almacenamiento y transporte utilizan este valor si no se suministra una propiedad de nombre de usuario específica del protocolo. El nombre de usuario debe contener únicamente caracteres alfanuméricos, de subrayado, guiones o puntos.

  6. En el campo Dirección de Remitente por Defecto, escriba la dirección de correo electrónico del usuario por defecto.

    El formato es nombre_usuario@host.dominio.

  7. En el campo Orden de despliegue, escriba un número para especificar el orden de despliegue del recurso de sesión JavaMail.

    Los recursos con un número inferior se cargan en primer lugar al iniciar el servidor. El valor por defecto es 100.

  8. En el campo Descripción, cambie la descripción de la sesión de correo.

  9. Active la casilla de control Estado activado para activar la sesión de correo.

    El valor por defecto es Activado.

  10. En el campo Protocolo de Almacenamiento, defina el método de comunicación de objetos de almacenamiento que se va a utilizar.

    Por defecto, el protocolo de almacenamiento es imap.

  11. En el campo Clase de Protocolo de Almacenamiento, especifique la clase del método de comunicación de almacenamiento que implanta el protocolo de almacenamiento deseado.

    Por defecto, la clase de protocolo de almacenamiento es com.sun.mail.imap.IMAPStore.

  12. En el campo Protocolo de Transporte, especifique el método de comunicación de transporte.

    Por defecto, el protocolo de transporte es smtp.

  13. En el campo Clase de Protocolo de Transporte, especifique el método de comunicación para la clase de transporte.

    Por defecto, la clase de protocolo de transporte es com.sun.mail.smtp.SMTPTransport .

  14. Active la casilla de control Depurar para activar una salida de depuración adicional para esta sesión de correo, incluido un rastreo de protocolo.

    Si el nivel de log de JavaMail se ha definido como FINE o FINER, la salida de depuración se genera y se incluye en el archivo log del sistema. Consulte Para configurar niveles de log para obtener más información sobre la definición del nivel de log.

  15. En la tabla Propiedades Adicionales, especifique otras propiedades requeridas por las aplicaciones, como una propiedad de host o nombre de usuario específico del protocolo.

    Para obtener una lista de las propiedades disponibles, consulte la documentación de la API de JavaMail (http://java.sun.com/products/javamail/javadocs/index.html). Las propiedades adicionales deben comenzar por mail- y como separadores se deben utilizar guiones en lugar de puntos. Por ejemplo, mail-debug es correcta, pero mail.debug no lo es. El backend de GlassFish Server convierte los guiones en los puntos que espera la API de JavaMail.

    • Para agregar una propiedad, haga clic en el botón Agregar Propiedad. En la fila en blanco que aparece, introduzca el nombre de propiedad en el campo Nombre y escriba el valor de propiedad en el campo Valor.

    • Para modificar una propiedad, edite el campo Valor.

    • Para suprimir una propiedad, active la casilla de control a la izquierda del campo Nombre de la propiedad que está suprimiendo y, a continuación, haga clic en el botón Suprimir Propiedades.

  16. Haga clic en Guardar.

    Sugerencia:

    Para volver a la configuración por defecto, haga clic en el botón Cargar Valores por Defecto y, a continuación, en Guardar.
