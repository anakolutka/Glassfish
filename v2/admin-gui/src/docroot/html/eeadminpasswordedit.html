To set the Administrator password

  1. In the Developer profile, select the Application Server node.
  2. In the Clusters profile, select the Domain node.
  3. Select the Administrator Password tab.

    The page displays the user ID and group list of the current administrator account. By default, the asadmin user ID has administrator privileges for the Application Server.

  4. Enter the new password for the current administrator account. Confirm the password, by entering it again.
  5. Click Save.

    Note - To create, edit, or delete a user account for the currently selected security realm, expand the Configuration —> Security —> Realms nodes and select the desired realm. Click on the Manage Users button in the Edit Realms page to change the user accounts for that realm.
