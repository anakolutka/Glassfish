Cómo cambiar la contraseña del administrador

El cifrado de la contraseña del administrador mejora la seguridad de su aplicación. Consulte el Capítulo 6, Administering System Security de Sun GlassFish Enterprise Server v3 Prelude Administration Guide para obtener información detallada sobre los procedimientos de seguridasd.

  1. En el árbol de navegación, expanda el nodo Configuración.
  2. Si el dominio está configurado para admitir clústeres, seleccione la configuración que esté cambiando.

    Algunos ejemplos son los dominios que se crean con el perfil clúster o con el perfil grandes empresas .

  3. Expanda el nodo Seguridad.
  4. Expanda el nodo Dominios.
  5. Seleccione el nodo admin-realm.
  6. Haga clic en el botón Administrar usuarios de la página Editar dominio.
  7. Seleccione el usuario llamado admin.
  8. Introduzca la contraseña nueva y confírmela.
  9. Haga clic en "Guardar".
