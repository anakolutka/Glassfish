가상 서버를 삭제하는 방법

  1. 탐색 트리에서 구성 노드를 선택합니다.
  2. 구성 노드에서 HTTP 서비스 노드를 확장합니다.
  3. HTTP 서비스 노드에서 가상 서버 노드를 선택합니다.
  4. 가상 서버 페이지에서 삭제할 가상 서버 이름 왼쪽의 확인란을 선택합니다.
  5. 삭제를 누릅니다.

    __asadmin 가상 서버는 삭제할 수 있지만 권장하지 않습니다. 삭제할 계획일 때는 필요한 경우 설정을 복원할 수 있도록 먼저 Enterprise Server domain.xml 파일의 virtual-server 요소를 안전한 곳에 복사해 두십시오.
