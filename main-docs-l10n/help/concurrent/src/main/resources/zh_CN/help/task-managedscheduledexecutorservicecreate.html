创建托管调度执行程序服务

托管调度执行程序服务指定 XXX。

  1. 在导航树中，展开“资源”节点。

  2. 在“资源”节点下，展开“并发资源”节点。

  3. 在“并发资源”节点下，选择“托管调度执行程序服务”。

    此时将打开“托管调度执行程序服务”页。

  4. 在“托管调度执行程序服务”页上，单击“新建”。

    此时将打开“新建托管调度执行程序服务”页。

  5. 在“JNDI 名称”字段中，键入一个唯一的名称。

    此名称将被注册到 JNDI 命名服务中。JNDI 名称可以在一个分布式计算环境中组织和定位组件，这与使用卡片目录组织和表示图书馆中书籍位置的方法类似。因此，JNDI 名称是访问资源的重要方法之一。

  6. XXX

  7. 在“说明”字段中，键入资源的简短说明。

    此说明是字符串值，最多可以包含 250 个字符。

  8. 选择“启用状态”复选框以启用资源。

    默认情况下，该资源已启用，也就是说该资源一经创建即可使用。

  9. 在“其他属性”部分中，指定资源的其他属性。

    GlassFish Server 不定义托管调度执行程序服务资源的任何其他属性。

    要添加属性，请单击“添加属性”按钮。在显示的空白行中，在“名称”字段中键入属性名称，在“值”字段中键入属性值。

  10. 单击“确定”。
