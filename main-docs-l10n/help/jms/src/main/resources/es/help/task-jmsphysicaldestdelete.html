Para suprimir un destino físico JMS

No se recomienda suprimir el destino del sistema, mq.sys.dmq.

  1. En el árbol de navegación, amplíe el nodo Configuración.

  2. En el nodo Configuración, amplíe el nodo Java Message Service.

  3. En el nodo Java Message Service, seleccione el nodo Destinos Físicos.

    Se abrirá la página Destinos Físicos.

  4. En la página Destinos Físicos, active la casilla de control situada junto al nombre del destino que va a suprimir.

  5. Haga clic en Suprimir.
