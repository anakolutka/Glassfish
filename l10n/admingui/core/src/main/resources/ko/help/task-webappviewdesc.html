웹 응용 프로그램의 설명자를 보는 방법

  1. 탐색 트리에서 응용 프로그램 노드를 선택합니다.
  2. 응용 프로그램 노드에서 웹 응용 프로그램 노드를 선택합니다.

    웹 응용 프로그램 페이지가 열립니다.

  3. 웹 응용 프로그램 페이지에서 설명자를 보려는 웹 응용 프로그램의 이름을 누릅니다.

    웹 응용 프로그램 페이지가 열립니다.

  4. 웹 응용 프로그램 페이지에서 설명자 탭을 누릅니다.

    배포 설명자 사용자 정의 페이지가 열립니다.

  5. 배포 설명자 필드에서 볼 XML 파일의 이름을 누릅니다.

    XML 파일이 새 페이지에 열립니다.

  6. 파일을 검색하고 닫기를 눌러 배포 설명자 사용자 정의 페이지로 돌아갑니다.
참조

  • 웹 응용 프로그램을 배포하는 방법

  • 배포된 웹 응용 프로그램을 편집하는 방법

  • 웹 응용 프로그램을 시작하는 방법

  • 배포된 웹 응용 프로그램을 재배포하는 방법
