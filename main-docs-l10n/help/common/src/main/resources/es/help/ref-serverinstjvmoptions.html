Opciones de JVM

Utilice la página Opciones de JVM para especificar las opciones del iniciador de aplicaciones Java (java) que ejecuta GlassFish Server. Las opciones -D designan propiedades que son específicas de GlassFish Server.

  • Para modificar una opción, edite el campo Valor de la opción.

  • Para agregar una opción, haga clic en el botón Agregar Opción de JVM. En la fila en blanco que aparece, introduzca la información de la opción en el campo Valor.

    Si la información de la opción contiene uno o más espacios, coloque toda la cadena entre comillas dobles.

  • Para suprimir una opción, active la casilla de control situada en la parte izquierda del campo Valor de la opción que se va a suprimir y, a continuación, haga clic en Suprimir.
