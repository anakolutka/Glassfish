建立範圍

Enterprise Server 預先配置了三個範圍： file、certificate 以及 admin-realm。您也可以建立 ldap、JDBC、solaris 以及自訂範圍。伺服器上的每種類型通常只有一個範圍，但是在 Enterprise Server 上有以下兩個 file 範圍︰ file 和 admin-realm。同一類型有兩個範圍，用於兩種不同目的。 系統上的每個虛擬伺服器可以有不同的憑證資料庫。

  1. 在瀏覽樹狀結構中，展開 [配置] 節點。
  2. 展開 [安全性] 節點。
  3. 選取 [範圍] 節點。

    [範圍] 頁面會隨即開啟。

  4. 在 [範圍] 頁面上按一下 [新建]。

    [新建範圍] 頁面會隨即開啟。

  5. 在 [名稱] 欄位中，鍵入範圍的名稱。
  6. 若要指定範圍的類別名稱，請執行以下作業之一：
    • 從 [類別名稱] 下拉式清單中，選取要建立之範圍的類別名稱。

      下表顯示有效的選項。

      範圍名稱

      類別名稱

      file

      com.sun.enterprise.security.auth.realm.file.FileRealm

      certificate

      com.sun.enterprise.security.auth.realm.certificate.CertificateRealm

      ldap

      com.sun.enterprise.security.auth.realm.ldap.LDAPRealm

      JDBC

      com.sun.enterprise.security.auth.realm.jdbc.JDBCRealm

      solaris

      com.sun.enterprise.security.auth.realm.solaris.SolarisRealm
    • 選取下方的單選按鈕，然後鍵入所要建立之自訂範圍的類別名稱。
  7. (可選擇) 在 [其他特性] 區段中，指定其他特性。

    若要增加特性，請按一下 [增加特性] 按鈕。在出現的空白列中，於 [名稱] 欄位中鍵入特性名稱，然後在 [值] 欄位中鍵入特性值。

    有效的特性取決於範圍的類型。

    • 如需「File 範圍」特性的說明，請參閱專屬於 FileRealm 類別的特性。

    • 如需「Certificate 範圍」特性的說明，請參閱專屬於 CertificateRealm 類別的特性。

    • 如需「JDBC 範圍」特性的說明，請參閱專屬於 JDBCRealm 類別的特性。

    • 如需「LDAP 範圍」特性的說明，請參閱LDAPRealm 類別特有的特性。

    • 如需「Solaris 範圍」特性的說明，請參閱SolarisRealm 類別特有的特性。

  8. 按一下 [確定]。
