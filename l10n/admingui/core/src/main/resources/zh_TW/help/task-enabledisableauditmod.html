啟用或停用稽核記錄

  1. 在瀏覽樹狀結構中，展開 [配置] 節點。
  2. 選取要配置的實例：
    • 若要配置特定的實例，請展開該實例的配置節點。例如，對於預設實例 server，請展開 [server-config] 節點。
    • 若要配置所有實例的預設設定，請展開 default-config 節點。
  3. 在 [配置] 節點下，選取 [安全性] 節點。

    [安全性] 頁面會隨即開啟。

  4. 選取 [稽核記錄已啟用] 核取方塊，以啟用稽核記錄。

    若要停用稽核記錄，請取消選取 [稽核記錄已啟用] 核取方塊。

    啟用此選項會載入稽核模組，並確定 Enterprise Server 的稽核程式庫會在稽核點呼叫稽核模組。

  5. 如果要啟用稽核記錄功能，請依照設定使用中的稽核模組中所述，指定預設稽核模組。
  6. 按一下 [儲存]。
  7. 如果主控台中顯示 [需要重新啟動]，請重新啟動 Enterprise Server。
