编辑托管调度执行程序服务

使用“编辑托管调度执行程序服务”页可编辑调度的托管执行程序服务资源。

“编辑上下文服务”页包含以下选项。

加载默认值

用于将具有默认值的设置还原为其默认值的按钮。没有默认值的设置不发生更改。

JNDI 名称

标识资源的唯一名称。JNDI 名称是只读字段。只能在创建新资源时指定 JNDI 名称。

上下文信息

资源的上下文：类加载器、JNDI、安全性或工作区。

状态

资源的状态。可以启用或禁用资源。默认值为“已启用”。

线程优先级

要为托管调度执行程序服务实例创建的线程优先级。默认值为 5。

长时间运行的任务

XXX。

以下时间后挂起

认为某个任务挂起之前该任务可以执行的时间 (秒)。

核心大小

要保留在线程池中的线程数量。默认值为 0。

最大池大小

池可包含的最大线程数。默认值为 2147483647。

保持活动

线程数量大于核心大小时，线程可以保持空闲的时间 (秒)。默认值为 60。

线程寿命

每个线程在清除之前保留在线程池中的时间 (秒)，无论线程数量大于核心大小或者线程为空闲。

任务队列容量

拥有等待执行的已提交任务的队列大小。默认值为 2147483647。

说明

此资源的说明。

其他属性

资源的其他属性。GlassFish Server 不定义调度的托管执行程序服务资源的任何其他属性。
