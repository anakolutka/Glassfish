Utilisateurs de fichiers

La page Utilisateurs de fichiers permet de gérer les utilisateurs de domaine file. Les utilisateurs et les groupes du domaine file sont répertoriés dans le fichier de clés, dont l'emplacement est spécifié par la propriété file.

Un utilisateur du domaine file peut appartenir à un groupe Java, c'est-à-dire une catégorie d'utilisateurs classés selon des caractéristiques communes. Par exemple, les clients d'une application d'e-commerce appartiendront au groupe CUSTOMER, tandis que les plus gros clients appartiendront au groupe PREFERRED. La classification des utilisateurs dans des groupes facilite le contrôle de l'accès d'un grand nombre d'utilisateurs.

Juste après l'installation de GlassFish Server, le seul utilisateur est l'administrateur configuré lors de l'installation. Par défaut, cet utilisateur appartient au groupe asadmin, du domaine admin-realm, qui autorise la modification de GlassFish Server. Les utilisateurs affectés à ce groupe auront des privilèges d'administrateur, c'est-à-dire qu'ils auront accès à l'outil asadmin et à la Console d'administration.

La page Utilisateurs de fichiers affiche la liste des utilisateurs du domaine sélectionné. Pour chaque utilisateur, les informations suivantes sont fournies :

ID utilisateur

Nom utilisateur.

Liste des groupes

Groupes auxquels l'utilisateur appartient.

La table Utilisateurs de fichiers contient également les options ci-après.

Nouveau

Bouton permettant de créer un utilisateur.

Supprimer

Bouton permettant de supprimer les utilisateurs sélectionnés.
