域日志

使用“域日志”页可以收集和下载指定 GlassFish Server 实例或集群的日志文件，并将日志编译到单个 ZIP 档案中。

注:

如果指定集群，则 ZIP 档案还将包含域管理服务器 (DAS) 的日志文件。

“域日志”页包含以下选项。

集群

集群 (如果存在任何集群) 的下拉列表。

实例

服务器实例的下拉列表。

收集日志

用于将日志收集到 log-files-name.zip 文件中的按钮，其中 name 为集群或实例的名称。

log-files-name.zip 文件是使用浏览器的下载机制下载的。
