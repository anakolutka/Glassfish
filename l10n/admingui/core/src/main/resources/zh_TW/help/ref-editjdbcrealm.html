專屬於 JDBCRealm 類別的特性

若要保護網頁或 Web 應用程式，您可以設定安全性，僅讓註冊的使用者進行存取。此作業稱為認證功能。此類範圍涉及在資料庫中儲存使用者憑證。Enterprise Server 使用資料庫資訊和配置檔案中的已啟用 JDBC 範圍選項。以下是 JDBC 範圍所需的特性：

JAAS 環境

此範圍所使用的登入模組識別碼。唯一的有效值為 jdbcRealm。

J.N.D.I.

此範圍的 J.N.D.I. 名稱。預設值為 jdbc/security。

使用者表格

包含此範圍的授權使用者清單之表格。預設值為 usertable。

使用者名稱欄

使用者表格內包含使用者清單的欄名稱。預設值為 userid。

密碼欄

使用者表格內包含各個使用者密碼的欄名稱。預設值為 password。

群組表格

預設值為 grouptable。

群組名稱欄

預設值為 groupid。
