To Manage JBI Components in the Cluster Profile

The Components page displays the binding components and service engines currently deployed into the JBI environment. From this page, you can install or delete components as well as manage their lifecycle states.n

  1. In the tree view, click on the JBI node and then the Components node.

    The Components page displays a list of JBI Binding Components and Service Engines.

  2. Use the State drop-down list to view components that are in a particular state. For example, select Enabled if you want to view components that are currently in the Enabled state.

  3. Use the Type drop-down list, if you want to view only the Binding Components or only the Service Engines.

  4. To install a JBI component, click Install. For more steps, see the ***insert link to clus profile section

  5. To enable a component, click the check box next to the component that you want to enable and from the State drop-down list, select Enabled.

  6. To disable a component, click the check box next to the component that you want to disable and from the State drop-down list, select Disable.

  7. To delete a component, click the check box next to the component that you want to delete and from the State drop-down list, select Delete.
