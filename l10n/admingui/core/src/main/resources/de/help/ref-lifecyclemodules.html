Lebenszyklusmodule

Die Seite Lebenszyklusmodule zeigt eine Liste der verfügbaren Lebenszyklusmodule an.

Ein Lebenszyklusmodul führt Aufgaben aus, wenn es durch mindestens ein Ereignis im Serverlebenszyklus ausgelöst wird. Mögliche Server-Auslösungsereignisse sind: Initialisierung, Start, Service-bereite Anfragen und Herunterfahren. Lebenszyklusmodule sind nicht Bestandteil der Java EE-Spezifikation, stellen jedoch eine Erweiterung von Enterprise Server dar.

Folgende Informationen werden für jedes Lebenszyklusmodul zur Verfügung gestellt.

Name

Der Name des Lebenszyklusmoduls.

Enabled (Aktiviert)

Wahr, wenn das Lebenszyklusmodul aktiviert ist, falsch, wenn dieses nicht aktiviert ist.

Ladereihenfolge

Ein Wert, der die relative Reihenfolge festlegt, in der das Lebenszyklusmodul geladen werden soll.

Klassenname

Die Klassendatei des Lebenszyklusmoduls.

Verwandte Aufgaben

  • So erstellen Sie ein Lebenszyklusmodul

  • So bearbeiten Sie ein Lebenszyklusmodul

  • So löschen Sie ein Lebenszyklusmodul

Verwandte asadmin-Befehle

  • create-lifecycle-module(1)

  • list-lifecycle-modules(1)

  • delete-lifecycle-module(1)
