Para Deletar um Servidor Virtual

  1. Na árvore de navegação, expanda o nó Configuração.

  2. Selecione a configuração que está sendo alterada.

  3. No nó Configuração, selecione o nó Servidores Virtuais.

    A página Servidores Virtuais será aberta.

  4. NA página Servidores Virtuais, marque a caixa de seleção à esquerda do nome do servidor virtual a ser deletado.

  5. Clique em Deletar.

    Não é recomendado deletar qualquer um dos servidores virtuais default, __asadmin e server.
