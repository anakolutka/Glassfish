Risorse di destinazione JMS

Utilizzare la pagina Risorse di destinazione JMS per configurare le risorse di destinazione JMS.

Per ogni risorsa di destinazione vengono fornite le informazioni riportate di seguito.

Nome JNDI

Il nome univoco che identifica la risorsa di destinazione.

Abilitato

L'impostazione è True se la risorsa è abilitata oppure è False se la risorsa è disabilitata.

Tipo di risorsa

Il tipo della risorsa di destinazione.

Descrizione

La descrizione della risorsa di destinazione.

La tabella Risorse di destinazione contiene inoltre le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare una nuova risorsa di destinazione.

Elimina

Pulsante che consente di eliminare una o più risorse di destinazione selezionate.

Abilita

Pulsante che consente di abilitare una o più risorse di destinazione selezionate.

Disabilita

Pulsante che consente di disabilitare una o più risorse di destinazione selezionate.
