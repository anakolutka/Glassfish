Para editar propiedades de pool de conexiones de conector

  1. En el árbol de navegación, amplíe el nodo Recursos.

  2. Amplíe el nodo Conectores.

  3. En el nodo Configuración, seleccione el nodo Pools de conexiones de conector.

    Se abrirá la página Pools de conexiones de conectores.

  4. En la página Pool de conexiones de conector, haga clic en el nombre del pool de conexiones de conector que desee editar.

    Se abrirá la página Editar pool de conexiones del conector.

  5. En la página Editar pool de conexiones de conector, haga clic en el separador Propiedades adicionales.

    Se abrirá la página Editar propiedades de pool de conexiones de conector.

  6. En la tabla Propiedades adicionales, especifique las propiedades adicionales para el pool de conexiones de conector.

    Las propiedades especificadas dependen del adaptador de recursos utilizado por el pool. Los pares nombre-valor especificados por el usuario de esta tabla se pueden emplear para sustituir los valores por defecto de las propiedades definidas por el proveedor del adaptador de recursos.

    Si no especifica un valor para una propiedad, ésta se elimina de la tabla al finalizar la edición del pool de conexiones. Si no desea definir un valor para una propiedad ahora, pero desea que la propiedad permanezca en la tabla para que el valor se pueda definir más adelante, introduzca un juego de paréntesis vacío en el campo Valor:

    ()

    • Para agregar una propiedad, haga clic en el botón Agregar Propiedad. En la fila en blanco que aparece, introduzca el nombre de propiedad en el campo Nombre y escriba el valor de propiedad en el campo Valor.

    • Para modificar una propiedad, edite el campo Valor.

    • Para suprimir una propiedad, active la casilla de control a la izquierda del campo Nombre de la propiedad que está suprimiendo y, a continuación, haga clic en el botón Suprimir Propiedades.

  7. Haga clic en Guardar.
