詳細ドメイン属性を設定する

  1. ナビゲーションツリーで、「Application Server」ノードを選択します。
  2. 「一般情報」ページで、「詳細」タブをクリックします。
  3. 「アプリケーション設定」ページで、「ドメイン属性」タブをクリックします。

    「ドメイン属性」ページが開きます。

  4. 「アプリケーションルート」フィールドに、アプリケーションを配備するディレクトリの完全パスを入力します。
  5. 「ログルート」フィールドに、サーバーのログファイルを保持するディレクトリの完全パスを入力します。
  6. 「ロケール」フィールドを空白のままにして、ホストのデフォルトロケールを使用するようにします。

    ロケールは、言語と地域の特定の組み合わせを指定する識別子です。たとえば、米国英語のロケールは en_US で、日本語のロケールは ja_JP です。英語以外のロケールを使用するには、英語を目的の言語に翻訳して、Enterprise Server をローカライズする必要があります。

  7. 「保存」をクリックします。
  8. サーバーを再起動します。
