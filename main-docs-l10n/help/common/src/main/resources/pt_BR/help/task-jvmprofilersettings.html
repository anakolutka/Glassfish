Para Configurar as Definições do Gerador de Perfil JVM

Uma ferramenta geradora de perfis que gera dados, os quais são utilizados para analisar o desempenho e identificar o potencial de afunilamento.

  1. Na árvore de navegação, selecione o nó Configuração.

  2. Selecione a configuração que está sendo alterada.

  3. No nó Configuração, selecione o nó Definições de JVM.

    A página Definições Gerais de JVM será aberta.

  4. Na página Definições Gerais de JVM, clique na guia Gerador de Perfil.

    A página Definições do Gerador de Perfil JVM será aberta.

  5. Especifique as definições do gerador de perfil.

    As informações especificadas dependem do produto gerador de perfis em uso. Para obter exemplos e instruções, consulte Capítulo 3, Debugging Applications, em Oracle GlassFish Server Application Development Guide.

  6. Clique em Salvar.

  7. Reinicie o servidor.
