Para Adicionar uma Instância de GlassFish Server em um Cluster

As instâncias do GlassFish Server formam a base da implantação de uma aplicação. Uma instância clusterizada é um membro de um cluster. Cada instância pertence a um único domínio e tem sua própria estrutura de diretório, configuração e aplicações implantadas. Cada instância também inclui o contêiner Web Java Platform, Enterprise Edition (Java EE) e o contêiner da tecnologia Enterprise JavaBeans (tecnologia EJB). Cada instância nova deve conter uma referência a um nó que defina o host no qual a instância residirá.

Observação:

Para criar uma instância em um nó do tipo CONFIG que represente um host remoto, você deverá usar o subcomando create-local-instance(1). Não é possível utilizar o Console de Administração para este propósito.
Antes de Começar

Certifique-se de que os seguintes itens existam:

  • O cluster ao qual você está adicionando a instância. Consulte Para Criar um Cluster para obter instruções.

  • O nó no qual a instância deve residir. Consulte Para Criar um Nó para obter instruções.

  1. Na árvore de navegação, expanda o nó Clusters.

  2. No nó Clusters expandidos, selecione o nó do cluster.

    A página Informações Gerais do cluster será aberta.

  3. Na página Informações Gerais, clique na guia Instâncias.

    A página Instâncias de Servidor Clusterizado será aberta com informações sobre o cluster.

  4. Na página Instâncias de Servidor Clusterizado, clique em Novo.

    A página Nova Instância de Servidor Clusterizado será aberta.

  5. No campo Nome, digite um nome para a instância.

    O nome deve atender aos seguintes requisitos:

    • O nome só pode conter caracteres ASCII.

    • O nome deve começar com uma letra, um número ou um sublinhado.

    • O nome pode conter só os seguintes caracteres:

      • Letras minúsculas

      • Letras maiúsculas

      • Números

      • Hífen

      • Ponto

      • Sublinhado

    • O nome deve ser único no domínio e não deve ser o nome de outra instância, um cluster, uma configuração nomeada ou um nó do GlassFish Server.

    • O nome não deve ser domain, server, nem nenhuma outra palavra-chave que esteja reservada pelo GlassFish Server.

  6. Na lista drop-down Nó, selecione o nó no qual a instância residirá.

  7. Clique em OK.
