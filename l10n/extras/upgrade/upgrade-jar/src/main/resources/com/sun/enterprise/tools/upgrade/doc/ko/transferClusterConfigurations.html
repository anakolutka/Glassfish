클러스터 구성 전송 화면

Sun Java System Application Server 8.x Enterprise Edition(EE) 클러스터 구성을 Sun Java System Application Server 9.1로 업그레이드하려면 유효한 clinstance.conf 파일이 필요합니다. 업그레이드 도구는 clinstance.conf 파일이 8.x EE 클러스터를 정확하게 반영한다고 가정합니다. 어떤 이유로든 이 파일이 실제 클러스터 구성을 정확하게 나타내지 않는 경우 업그레이드를 수행하려면 먼저 클러스터 정의를 수동으로 편집해야 합니다.

클러스터 구성 전송 화면을 사용하여 새 설치로 마이그레이션해야 하는 클러스터 구성 파일을 지정합니다.

  1. 추가 버튼을 누르면 clinstance.conf 파일 선택 대화 상자가 나타납니다.
  2. 마이그레이션할 클러스터 구성 정보가 있는 clinstance.conf 클러스터 파일을 선택합니다.
  3. 열기 버튼을 누릅니다.
  4. 마이그레이션해야 하는 모든 클러스터 구성 파일이 추가될 때까지 이 과정을 반복합니다.
  5. 완료되면 다음 버튼을 누릅니다. 업그레이드 결과 화면이 나타나 업그레이드 상태가 표시됩니다.
