編輯應用程式

使用 [編輯應用程式] 頁面可修改現有的應用程式。

[編輯應用程式] 頁面包含下列選項。

名稱

顯示正在編輯中的應用程式名稱的唯讀欄位。

虛擬伺服器

與此應用程式相關的虛擬伺服器。

環境根目錄

若為 Web 應用程式，則指定應用程式的路徑。在 Web 應用程式的 URL 中，環境根目錄緊跟在連接埠號碼之後 (http://host: port/context-root/...)。環境根目錄的開頭必須是正斜線，例如 /hello。

說明

應用程式的簡短說明。

狀態

若選取 [已啟用] 核取方塊，則會啟用應用程式。此選項預設為選取。

Java Web Start

若選取 [已啟用] 核取方塊，則應用程式會使用 Java Web Start 軟體。Java Web Start 能透過各種瀏覽器部署執行於專用 Java 虛擬機器上的 Java 應用程式。

部署目錄

若選取此核取方塊，則會從目錄部署應用程式。

位置

顯示已部署應用程式位置的唯讀欄位。

程式庫

顯示應用程式所需程式庫 JAR 檔案的唯讀欄位。

子模組

顯示應用程式子模組名稱及其引擎的表格。
