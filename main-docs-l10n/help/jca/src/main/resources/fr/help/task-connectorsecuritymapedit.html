Modification d'une mappe de sécurité pour un pool de connexions de connecteur

  1. Dans l'arborescence de navigation, développez le noeud Ressources.

  2. Développez le noeud Connecteurs.

  3. Sous le noeud Connecteurs, sélectionnez le noeud Pools de connexions du connecteur.

    La page Pools de connexions du connecteur apparaît.

  4. Sur la page Pools de connexions du connecteur, cliquez sur le nom du pool de connexions à modifier.

    La page Modifier le pool de connexions du connecteur apparaît.

  5. Sur la page Modifier le pool de connexions du connecteur, cliquez sur l'onglet Mappes de sécurité.

    La page Mappes de sécurité des pools de connexions du connecteur apparaît.

  6. Sur la page Mappes de sécurité des pools de connexions du connecteur, cliquez sur le nom de la mappe de sécurité à modifier.

    La page Modifier une mappe de sécurité du pool de connexions du connecteur apparaît.

  7. Effectuez l'une des opérations suivantes uniquement :

    • Sélectionnez le bouton radio Groupes d'utilisateurs et entrez les noms des groupes d'utilisateurs, séparés par des virgules.

    • Sélectionnez le bouton radio Principaux et entrez les noms des principaux, séparés par des virgules.

  8. Dans le champ Nom utilisateur, spécifiez le nom utilisateur du principal back-end requis par le système EIS.

  9. Dans le champ Mot de passe, spécifiez le mot de passe du principal back-end requis par le système EIS.

  10. Cliquez sur Enregistrer.
