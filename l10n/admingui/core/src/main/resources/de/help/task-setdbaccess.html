So richten Sie den Datenbankzugriff ein

Zum Speichern, Organisieren und Abrufen von Daten verwenden die meisten Anwendungen relationale Datenbanken.

  1. Installieren Sie ein unterstütztes Datenbankprodukt.

    Eine Liste mit unterstützten Datenbankprodukten finden Sie in den Sun GlassFish Enterprise Server v3 Prelude Release Notes .

  2. Installieren Sie einen Treiber, der die JDBC-API (JDBC-Treiber) für die Datenbank unterstützt.

    Stellen Sie sicher, dass die Treiber, die Sie zum Erstellen der Datenbankverbindungen verwenden möchten, auf allen Servern installiert sind, auf denen Sie die Datenquelle bereitstellen möchten.

  3. Stellen Sie die JAR-Datei des Treibers für die Serverinstanz der Domäne zur Verfügung.

    Siehe auch So integrieren Sie einen JDBC-Treiber.

  4. Erstellen Sie die Datenbank.

    Üblicherweise stellt der Anwendungsanbieter Skripts zum Erstellen und Füllen der Datenbank bereit.

  5. Erstellen Sie einen Verbindungspool für die Datenbank.

    Siehe So erstellen Sie einen JDBC-Verbindungspool.

  6. Erstellen Sie eine JDBC-Ressource, die auf den Verbindungspool verweist.

    Siehe So erstellen Sie eine JDBC-Ressource.
