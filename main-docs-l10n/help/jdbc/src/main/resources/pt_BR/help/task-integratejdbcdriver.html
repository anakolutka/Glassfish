Para Integrar um Driver JDBC

Um driver que suporte a API da JDBC (driver JDBC) converte as chamadas JDBC de uma aplicação no protocolo do servidor de bancos de dados.

Para integrar o driver JDBC a um domínio do GlassFish Server, execute qualquer um desses procedimentos:

  • Torne o driver acessível ao carregador de classe comum.

    1. Copie os arquivos JAR e ZIP do driver no diretório domain-dir/lib.

      Observação:

      Há uma exceção a essa regra. Se você estiver usando o driver JDBC da Oracle e deseja aproveitar as extensões EclipseLink que estão disponíveis para uso com bancos de dados da Oracle, é necessário colocar os arquivos JAR para o driver JDBC da Oracle no diretório domain-dir/lib/ext.

    2. Reinicie o domínio.

  • Torne o driver acessível ao carregador de classe do sistema.

    1. Na árvore de navegação, expanda o nó Configuração.

    2. No nó Configuração, selecione o nó Definições de JVM.

      A página Definições Gerais de JVM será aberta.

    3. Na página Definições Gerais de JVM, clique na guia Definições de Caminho.

    4. No campo Sufixo do Classpath, digite o nome de caminho totalmente qualificado para o arquivo JAR do driver.

    5. Clique em Salvar.

    6. Reinicie o servidor.
