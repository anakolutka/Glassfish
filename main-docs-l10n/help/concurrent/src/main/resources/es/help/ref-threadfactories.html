Fábricas de threads gestionadas

Utilice la página Fábricas de threads gestionadas para configurar recursos de la fábrica de threads gestionada.

En cada recurso de la fábrica de threads gestionada, aparece la siguiente información.

Nombre JNDI

Nombre único que identifica el recurso.

Activado

True si el recurso está activado o false, si no lo está.

Información de contexto

Contexto del recurso: classloader, JNDI, seguridad o área de trabajo.

Prioridad de threads

Prioridad de threads que se van a crear para la instancia de la fábrica de threads gestionada. El valor por defecto es 5.

Descripción

Descripción del recurso.

La tabla Fábricas de threads gestionadas también contiene las siguientes opciones.

Nuevo

Botón para crear un nuevo recurso.

Suprimir

Botón para suprimir uno o varios recursos seleccionados.

Activar

Botón para activar uno o varios recursos seleccionados.

Desactivar

Botón para desactivar uno o varios recursos seleccionados.
