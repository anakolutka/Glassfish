Viewing Messages

To view messages received for monitoring a web service:

  1. In the tree view component, expand the Web Services node.
  2. Select the web service for which you want to view the messages.
  3. Click the Monitor tab. Then click the Messages tab.
  4. Select the instance for which you want to view the monitoring messages.

    Displays a list of messages received for the web service. The number of messages displayed depends on the monitoring configuration. The Actions column links to call flow data, if call flow is turned on when the web service is executed. If call flow data is not available for this message, the Actions column will be empty.

    You can also select a filter to view only the success messages or the failure messages.
