Using the Cluster Profile Common Tasks Page

The Common Tasks page provides handy shortcuts for some of the most common Admin Console tasks. You can create clusters, deploy applications and JBI Service Assemblies, create standalone instances, and access documentation.

Use the Common Tasks page for the following operations:

  • To Create a Cluster

  • To Create a New Standalone Server Instance

  • To deploy an application

  • To deploy a web application

  • To Deploy a Service Assembly

  • To create a JDBC connection pool

  • To view monitoring data

The Common Tasks page also provides links to the Application Server documentation.
