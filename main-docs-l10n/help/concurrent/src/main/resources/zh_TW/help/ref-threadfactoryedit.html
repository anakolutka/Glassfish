編輯受管理繫線工廠

您可以使用「編輯受管理繫線工廠」頁面來編輯受管理繫線工廠資源。

「編輯受管理繫線工廠」頁面包含下列選項。

載入預設值

用於將具有預設值的設定復原為設定值的按鈕。沒有預設值的設定則保持不變。

JNDI 名稱

可識別資源的唯一名稱。「JNDI 名稱」是唯讀欄位。您僅能在建立新資源時指定 JNDI 名稱。

環境資訊

資源的環境：類別載入器、JNDI、安全性或工作區。

狀態

資源狀態。您可啟用或停用資源。預設值為「啟用」。

繫線優先順序

要為受管理繫線工廠執行處理建立之繫線的優先順序。預設值為 5。

描述

此資源的描述。

其他特性

資源的其他特性。GlassFish Server 未定義受管理繫線工廠資源的任何其他特性。
