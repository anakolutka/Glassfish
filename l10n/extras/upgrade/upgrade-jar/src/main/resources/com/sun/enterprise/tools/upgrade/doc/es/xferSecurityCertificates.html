Pantalla Transferir certificados de seguridad

Utilice la pantalla Transferir certificados de seguridad para especificar dominios que contengan uno o varios conjuntos de certificados de seguridad que deben migrarse a Sun Java Systems Application Server 9:

  1. Pulse el botón Agregar para abrir el cuadro de diálogo Agregar dominio en el que se especifican el nombre del dominio y las contraseñas de los certificados.

    En el panel Dominios con certificados que transferir se muestran los dominios agregados.

  2. Repita este proceso hasta haber agregado todos los dominios que contengan certificados que deben transferirse.
  3. Para eliminar un dominio agregado, selecciónelo y pulse el botón Eliminar dominio.
  4. Cuando haya finalizado, pulse el botón Siguiente.

    Se abrirá la pantalla Resultados de la actualización, en la que se muestra el estado de la actualización.
