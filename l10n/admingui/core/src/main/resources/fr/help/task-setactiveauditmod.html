Définir le module d'audit actif

Avant de commencer

Pour spécifier le module d'audit utilisé par le serveur, vous devez d'abord activer la journalisation d'audit tel qu'indiqué dans la rubrique Activer ou désactiver la journalisation d'audit.

  1. Dans le champ Modules d'audit, sélectionnez le nom du module d'audit à utiliser par le serveur.

    La valeur par défaut est default. Assurez-vous que la propriété auditOn de ce module d'audit est définie sur true (vrai) tel qu'indiqué dans la rubrique Utiliser le module d'audit par défaut.

  2. Cliquez sur Enregistrer.
  3. Redémarrez le Enterprise Server si « Redémarrage requis » s'affiche dans la console.
