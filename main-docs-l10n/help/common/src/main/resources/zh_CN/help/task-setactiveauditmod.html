设置活动审计模块

  1. 在导航树中，展开“配置”节点。

  2. 选择要配置的实例：

    • 要配置特定的实例，请展开该实例的配置节点。例如，对于默认实例 server，请展开 server-config 节点。

    • 要配置所有实例的默认设置，请展开 default-config 节点。

  3. 在“配置”节点下，选择“安全性”节点。

    将打开“安全性”页。

  4. 从“审计模块”下拉列表中，选择服务器要使用的审计模块的名称。

    默认情况下，只有一个名为默认的可用审计模块。

  5. 单击“保存”。
