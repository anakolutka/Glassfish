Activation ou désactivation d'une ressource de manière globale

  1. Dans l'arborescence de navigation, développez le noeud Clusters.

  2. Sous le noeud Clusters développé, sélectionnez le noeud correspondant au cluster.

    La page Informations générales du cluster apparaît.

  3. Sur la page Informations générales, cliquez sur l'onglet Ressources.

    La page Ressources du cluster apparaît.

  4. Cochez la case située en regard de la ressource.

  5. Cliquez sur le bouton approprié.

    • Pour activer la ressource, choisissez Activer.

    • Pour désactiver la ressource, choisissez Désactiver.

      La désactivation d'une ressource n'entraîne pas sa suppression.
