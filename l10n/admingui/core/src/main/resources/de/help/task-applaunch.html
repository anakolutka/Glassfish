So starten Sie eine Anwendung

Bevor Sie beginnen

Stellen Sie sicher, dass die zu startende Anwendung bereitgestellt ist.

  1. Wählen Sie in der Navigationsstruktur den Anwendungsknoten.

    Die Seite "Anwendungen" wird geöffnet.

  2. Klicken Sie in der Tabelle "Bereitgestellte Anwendungen" in der Zeile für die zu startende Anwendung auf "Starten".

    Die Anwendung wird in einem neuen Browserfenster geöffnet.
