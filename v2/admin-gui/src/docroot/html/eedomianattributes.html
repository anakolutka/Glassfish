To set the Advanced Domain Attributes

  1. In the Developer profile, select the Application Server node then select the Advanced tab, then the Domain Attributes tab.
  2. In the Clusters profile, select Domain node.
  3. Define the Domain Attributes as follows:

    • Application Root: Identify the full directory path where the applications will be deployed.

    • Log Root: Specify where the server instance log files are kept.

    • Locale: Typically, you will leave the Locale field blank to use the default locale of the host.

      A locale is an identifier that specifies a particular combination of language and region. For example, the locale for American English is en_US, and for Japanese it is ja_JP. In order to use a non-English locale, the Application Server must be localized, a process that includes translating English into another language.

  4. Click Save.
  5. Restart the server.
