Para alterar a senha de um usuário no domínio do admin

Para alterar a senha da conta do administrador atual, consulte Para alterar a Senha do Administrador. Se o domínio-admin tiver mais de um usuário, você também poderá alterar a senha de outros usuários.

Criptografar a senha do administrador aumenta a segurança do aplicativo. Para obter informações mais detalhadas sobre procedimentos de segurança, consulte o Capítulo 11, Administering System Security, no Sun GlassFish Enterprise Server v3 Administration Guide.

  1. Na árvore de navegação, expanda o nó Configuração.
  2. Expanda o nó Segurança.
  3. Expanda o nó Domínios.
  4. Selecione o nó admin-realm.

    A página Editar Domínio é exibida.

  5. Na página Editar Domínio, clique no botão Gerenciar Usuários.

    A página Usuários do Arquivo é exibida.

  6. Na página Usuários do Arquivo, clique no usuário cuja senha você quer adicionar ou alterar.

    A página Editar Usuário do Domínio Arquivo é exibida.

  7. Nos campos Nova Senha e Confirmar Nova Senha, digite a nova senha.
  8. Clique em Salvar.
