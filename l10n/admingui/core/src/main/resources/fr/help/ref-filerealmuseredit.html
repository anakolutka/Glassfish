Modification d'un utilisateur de domaine de fichiers

Utilisez la page Modification d'un utilisateur de domaine de fichiers pour modifier le mot de passe d'accès au domaine actuel.

La page Modification d'un utilisateur de domaine de fichiers contient les options suivantes :

Nom de domaine

Nom du domaine actuellement sélectionné. Ce champ est en lecture seule.

ID utilisateur

Nom du compte que vous modifiez. Ce champ est en lecture seule.

Liste des groupes

Groupes auxquels cet utilisateur appartient. Séparez les noms de groupe par une virgule.

Pour les utilisateurs du domaine d'administration, ce champ est en lecture seule, et le groupe est asadmin.

Nouveau mot de passe

Nouveau mot de passe de cet utilisateur.

Confirmer le mot de passe

Nouveau mot de passe de cet utilisateur.
