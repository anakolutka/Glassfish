Promoted builds are weekly builds of GlassFish which have passed a minimum level of testing which includes the quicklook tests, a sub-set of CTS EE5 smoketest (representative collection of 347 tests) and a sub-set of SQE tests (22 in number). A build is not promoted unless it passes all quicklook, CTS smoke and SQE smoke tests.

Instructions to unbundle and configure GlassFish

GlasssFish installation requires JDK 5.

  1. Download one of the bundles
  2. Run: 
    java -Xmx256m -jar filename.jar
  3. cd glassfish
  4. Run: 
    ant -f setup.xml

Ant is included in the GlassFish bundle that was downloaded.  We recommend that you use the version from \glassfish\lib\ant directory.

To get started with GlassFish, see the quick start guide.
