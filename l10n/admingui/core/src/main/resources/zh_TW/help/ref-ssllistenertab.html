SSL 偵聽程式

SSL 偵聽程式會將伺服器憑證連結至 HTTP 偵聽程式連接埠。使用新建 HTTP 偵聽程式 (步驟 2，共 2 步)頁面建立新的 HTTP 偵聽程式時，就會定義 SSL 設定。您可在此 [SSL 偵聽程式] 頁面上進一步編輯 SSL 設定。

用戶端認證

啟用此欄位以強制用戶端在使用此偵聽程式時對伺服器進行自我認證。

憑證暱稱

現有伺服器金鑰組與憑證的名稱。

SSL3

若啟用此選項，則會在偵聽程式上使用 SSL3 安全性協定。

SSL2

若啟用此選項，則會在偵聽程式上使用 SSL2 安全性協定。

TLS

若啟用此選項，則會在偵聽程式上使用 TLS 安全性協定。

常用密碼組

協定所使用的可用密碼組。

暫用 Dittie-Hellman 密碼組

協定所使用的暫用密碼組。
