So löschen Sie eine JDBC-Ressource

  1. Erweitern Sie in der Navigationsstruktur den Ressourcenknoten.
  2. Wählen Sie den Knoten "JDBC-Ressourcen" aus.
  3. Aktivieren Sie auf der Seite "JDBC-Ressourcen" das Kontrollkästchen links neben der zu löschenden Ressource.
  4. Klicken Sie auf „Löschen“.
