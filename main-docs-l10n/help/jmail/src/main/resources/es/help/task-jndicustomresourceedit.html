Para editar un recurso JNDI personalizado

  1. En el árbol de navegación, amplíe el nodo Recursos.

  2. En el nodo Recursos, amplíe el nodo JNDI.

  3. En el nodo JNDI, seleccione Recursos Personalizados.

    Se abrirá la página Recursos Personalizados.

  4. En la página Recursos Personalizados, haga clic en el nombre del recurso que desee editar.

    Se abrirá la página Editar Recurso Personalizado.

    Nota:

    El nombre JNDI es un campo de sólo lectura. Únicamente puede especificar el nombre JNDI al crear un nuevo recurso personalizado.

  5. Para modificar el tipo de recurso, realice una de las acciones siguientes.

    • En la lista desplegable, seleccione una definición de tipo.

    • Seleccione el segundo botón de radio e introduzca una definición de tipo completamente cualificada que siga el formato xxx.xxx.

  6. En el campo Clase de Fábrica, escriba un nombre de clase de fábrica para el recurso personalizado.

    La Clase de Fábrica es el nombre especificado por el usuario para la clase de fábrica. Esta clase implanta la interfaz javax.naming.spi.ObjectFactory.

  7. En el campo Orden de despliegue, escriba un número para especificar el orden de despliegue del recurso personalizado.

    Los recursos con un número inferior se cargan en primer lugar al iniciar el servidor. El valor por defecto es 100.

  8. En el campo Descripción, introduzca una breve descripción del recurso.

    Esta descripción es un valor de cadena y puede incluir un máximo de 250 caracteres.

  9. Active la casilla de control Estado activado para activar el recurso.

    Por defecto, el recurso está activado, lo que significa que está disponible en cuanto se crea.

  10. En la sección Propiedades Adicionales, especifique propiedades adicionales del recurso personalizado.

    GlassFish Server no define ninguna propiedad adicional para los recursos personalizados.

    • Para agregar una propiedad, haga clic en el botón Agregar Propiedad. En la fila en blanco que aparece, introduzca el nombre de propiedad en el campo Nombre y escriba el valor de propiedad en el campo Valor.

    • Para modificar una propiedad, edite el campo Valor.

    • Para suprimir una propiedad, active la casilla de control a la izquierda del campo Nombre de la propiedad que está suprimiendo y, a continuación, haga clic en el botón Suprimir Propiedades.

  11. Haga clic en Guardar.
