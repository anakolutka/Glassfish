Para editar una aplicación web implementada

  1. En el árbol de navegación, expanda el nodo Aplicaciones.
  2. Seleccione el nodo Aplicaciones web.
  3. En la tabla Aplicaciones web implementadas, haga clic en el nombre de la aplicación que se va a editar.

    Se abrirá la página Aplicación web.

  4. En el campo Raíz de contexto, introduzca la ruta para esta aplicación web.

    En la URL de la aplicación web, la raíz de contexto aparece justo después del número de puerto (http://host:port /context-root/...). Asegúrese de que la raíz de contexto empieza con una barra, por ejemplo, /hello.

  5. En el campo Descripción, introduzca una descripción de esta aplicación web.
  6. Haga clic en "Guardar".
