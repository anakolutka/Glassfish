Configurer les utilitaires du Enterprise Server pour permettre l'utilisation par des fournisseurs de sécurité des messages

  1. Si vous utilisez un jeton de nom d'utilisateur, configurez une base de données utilisateur, si nécessaire.

    Si vous utilisez un jeton nom d'utilisateur/mot de passe, configurez d'abord un domaine approprié et une base de données utilisateur pour le domaine.

    Pour savoir comment configurer une base de données utilisateur, reportez-vous à la rubrique Modifier un domaine.

  2. Gérez les certificats et les clés privées, si nécessaire.
Étapes suivantes

Une fois les utilitaires du Enterprise Server configurés pour être utilisés par les fournisseurs de sécurité des messages, les fournisseurs installés sur le Enterprise Server peuvent être activés conformément à la rubrique Pour modifier une configuration de sécurité des messages.
