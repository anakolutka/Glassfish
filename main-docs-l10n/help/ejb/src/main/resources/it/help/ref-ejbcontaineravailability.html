Disponibilità contenitore EJB

Utilizzare la pagina Disponibilità contenitore EJB per configurare le impostazioni di disponibilità per il contenitore EJB.

La pagina Disponibilità contenitore EJB contiene le opzioni riportate di seguito.

Carica impostazioni predefinite

Pulsante per ripristinare i valori predefiniti delle impostazioni che dispongono di valori predefiniti. Le impostazioni che non dispongono di valori predefiniti non vengono modificate.

Nome configurazione

Il nome della configurazione a cui si applicano le impostazioni presenti in questa pagina. Questo campo è di sola lettura.

Servizio disponibilità

Se la casella di controllo Abilitato è selezionata, la disponibilità è abilitata per il contenitore EJB.

Tipo di persistenza HA

Il meccanismo di persistenza utilizzato quando la casella di controllo Abilitato dell'opzione Servizio disponibilità è selezionata. L'impostazione predefinita è replicated, che fornisce la persistenza in base al failover in un cluster. Non modificare questo valore.

Tipo di persistenza SFSB

Il meccanismo di persistenza utilizzato quando la casella di controllo Abilitato dell'opzione Servizio disponibilità non è selezionata. L'impostazione predefinita è file, che fornisce la persistenza in base al file system. Non modificare questo valore.

Il campo Posizione di memorizzazione sessione determina la posizione in cui gli stati dei checkpoint SFSB vengono resi persistenti. Per impostare il campo Posizione di memorizzazione sessione, vedere Per configurare le impostazioni EJB.

Proprietà aggiuntive

Proprietà aggiuntive per la disponibilità del contenitore EJB.

GlassFish Server non definisce alcuna proprietà aggiuntiva per la disponibilità del contenitore EJB.
