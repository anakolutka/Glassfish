Configurer la connexion Keep-Alive du service HTTP

  1. Dans l'arborescence de navigation, développez le nœud Configuration.
  2. Sous le nœud Configuration, sélectionnez le nœud Service HTTP.
  3. Dans la page Service HTTP, sélectionnez l'onglet Keep-Alive.

    La page Keep-Alive s'ouvre.

  4. Dans le champ Nombre de threads, entrez le nombre de threads keep-alive à utiliser.

    La valeur par défaut est 1.

  5. Dans le champ Nombre max. de connexions, entrez le nombre maximal de connexions permanentes à conserver.

    La valeur par défaut est 256.

  6. Dans le champ Délai, saisissez la durée maximale (en secondes) pendant laquelle la connexion Keep-Alive doit être maintenue.

    La valeur par défaut est 30.

  7. Cliquez sur Enregistrer.

    Astuce – Pour revenir aux paramètres par défaut, cliquez sur le bouton Charger valeurs par défaut, puis sur Enregistrer.
