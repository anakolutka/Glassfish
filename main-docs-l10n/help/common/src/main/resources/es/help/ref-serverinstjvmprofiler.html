Configuración de analizador de perfiles de JVM

Una herramienta de analizador de perfiles genera datos que se utilizan para analizar el rendimiento e identificar cuellos de botella potenciales.

Especifique la configuración del analizador de perfiles. La información que debe especificar depende del producto de analizador de perfiles que esté utilizando. Para obtener ejemplos e instrucciones, consulte el Capítulo 3, sobre depuración de aplicaciones, en Oracle GlassFish Server Application Development Guide.
