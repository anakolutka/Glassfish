Gestisci destinazioni

Utilizzare la pagina Gestisci destinazioni risorse per modificare i cluster e le istanze server standalone di destinazione su cui è possibile abilitare un modulo del ciclo di vita. Il modulo del ciclo di vita può essere abilitato solo nelle destinazioni riportate nella colonna Destinazioni selezionate.

La pagina Gestisci destinazioni risorse contiene le informazioni riportate di seguito.

Destinazioni disponibili

I cluster e le istanze in cui il modulo del ciclo di vita non può essere abilitato.

Destinazioni selezionate

I cluster e le istanze in cui il modulo del ciclo di vita può essere abilitato.

Aggiungi

Pulsante che consente di spostare la destinazione selezionata dalla colonna Destinazioni disponibili alla colonna Destinazioni selezionate.

Aggiungi tutto

Pulsante che consente di spostare tutte le destinazioni dalla colonna Destinazioni disponibili alla colonna Destinazioni selezionate.

Rimuovi

Pulsante che consente di spostare la destinazione selezionata dalla colonna Destinazioni selezionate alla colonna Destinazioni disponibili.

Rimuovi tutto

Pulsante che consente di spostare tutte le destinazioni dalla colonna Destinazioni selezionate alla colonna Destinazioni disponibili.
