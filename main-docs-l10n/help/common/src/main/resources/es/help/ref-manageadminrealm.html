Usuarios administradores

Utilice la página Usuarios administradores para gestionar los usuarios de admin-realm. admin-realm es un dominio de archivo. Los usuarios y grupos del dominio file aparecen en el archivo de claves, cuya ubicación está especificada por la propiedad file.

Inicialmente, tras la instalación de GlassFish Server, el único usuario es el administrador introducido durante la instalación. Por defecto, este usuario pertenece al grupo asadmin, en el dominio admin-realm, lo que le da derecho a modificar GlassFish Server. Los usuarios asignados a este grupo tendrán privilegios de administrador, es decir que podrán acceder a la herramienta asadmin y a la Consola de administración.

La página Usuarios administradores muestra una lista de los usuarios para el dominio seleccionado. Para cada recurso, se proporciona la siguiente información:

ID de Usuario

Nombre de usuario.

Lista de Grupos

Grupos a los que pertenece el usuario.

La tabla Usuarios administradores también contiene las siguientes opciones.

Nuevo

Botón para crear un nuevo usuario.

Suprimir

Botón para suprimir uno o varios usuarios seleccionados.
