Servizio di monitoraggio

Utilizzare la pagina Servizio di monitoraggio per configurare le opzioni di monitoraggio per singoli componenti o servizi server.

Prima di configurare il monitoraggio di un'applicazione, un componente o un servizio, è necessario eseguirne la distribuzione.

La pagina Servizio di monitoraggio contiene le opzioni riportate di seguito.

Nome configurazione

Il nome della configurazione a cui si applicano le impostazioni presenti in questa pagina. Questo campo è di sola lettura.

Servizio di monitoraggio

Se la casella di controllo Abilitato dell'opzione Servizio di monitoraggio è selezionata, il monitoraggio è abilitato per GlassFish Server. Questa opzione è abilitata per impostazione predefinita.

MBean di monitoraggio

Se la casella di controllo Abilitato dell'opzione MBean di monitoraggio è selezionata, tutti gli MBean necessari per il monitoraggio vengono distribuiti in GlassFish Server. Questa opzione è abilitata per impostazione predefinita.

Se la casella di controllo non è selezionata, non è possibile visualizzare i dati di monitoraggio nella Console di amministrazione, anche se è selezionata la casella di controllo Servizio di monitoraggio. In tal caso, per visualizzare i dati di monitoraggio è necessario utilizzare la riga di comando. Se si deseleziona la casella di controllo, l'utilizzo di memoria di GlassFish Server risulterà ridotto.

DTrace

Se la casella di controllo Abilitato dell'opzione DTrace è selezionata e si dispone di Oracle Enterprise Manager DTrace Monitoring, è possibile utilizzare la funzione DTrace Solaris. Questa opzione è disabilitata per impostazione predefinita.

Per ogni componente disponibile vengono fornite le informazioni riportate di seguito.

Componente

Il nome del componente di GlassFish Server.

Livello di monitoraggio

Il livello di monitoraggio impostato per il componente. Per ogni componente i livelli disponibili sono BASSO, ALTO e DISATTIVATO. Il livello predefinito è DISATTIVATO. Nella release corrente i livelli di monitoraggio garantiti con le opzioni BASO e ALTO non presentano differenze.

La tabella Impostazioni a livello di componente contiene inoltre le opzioni riportate di seguito.

Cambia livello

Pulsante che consente di modificare livello di monitoraggio per uno o più componenti selezionati.

Livello

Elenco a discesa da cui è possibile selezionare il livello di monitoraggio da applicare a tutti i componenti selezionati.
