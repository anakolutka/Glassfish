檢視 Web 應用程式的描述元

  1. 在瀏覽樹狀結構中，展開 [應用程式] 節點。
  2. 在 [應用程式] 節點下，選取 [Web 應用程式] 節點。

    [Web 應用程式] 頁面會隨即開啟。

  3. 在 [Web 應用程式] 頁面上，按一下您要檢視其描述元的 Web 應用程式名稱。

    [Web 應用程式] 頁面會隨即開啟。

  4. 在 [Web 應用程式] 頁面上，按一下 [描述元] 標籤。

    [自訂部署描述元] 頁面會隨即開啟。

  5. 在 [部署描述元] 欄位中，按一下您要檢視的 XML 檔案名稱。

    將在新的頁面中開啟 XML 檔案。

  6. 瀏覽檔案，然後按一下 [關閉] 以回到 [自訂部署描述元] 頁面。
