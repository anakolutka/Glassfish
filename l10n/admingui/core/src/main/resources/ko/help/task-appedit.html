배포된 응용 프로그램을 편집하는 방법

  1. 탐색 트리에서 응용 프로그램 노드를 선택합니다.

    응용 프로그램 페이지가 열립니다.

  2. 배포된 응용 프로그램 테이블에서 편집할 응용 프로그램의 이름을 누릅니다.

    응용 프로그램 편집 페이지가 열립니다.

  3. 웹 응용 프로그램의 경우 컨텍스트 루트 필드에 응용 프로그램의 경로를 입력합니다.

    웹 응용 프로그램의 URL에서 컨텍스트 루트 바로 앞에는 포트 번호(http://host:port /context-root/...)가 있습니다. 컨텍스트 루트는 슬래시로 시작해야 합니다(예: /hello).

  4. 설명 필드에 이 응용 프로그램의 설명을 입력합니다.
  5. 응용 프로그램 상태를 비활성화하여 사용할 수 없게 만들려면 상태 사용 가능 확인란을 선택 해제합니다.

    기본적으로 응용 프로그램이 배포되는 즉시 상태는 사용 가능이 됩니다.

  6. Java Web Start 소프트웨어를 사용하여 응용 프로그램을 배포하려면 Java Web Start 사용 가능 확인란을 선택합니다.
  7. 저장을 누릅니다.
참조

  • 배포된 응용 프로그램의 목록을 보는 방법

  • 웹 응용 프로그램을 배포하는 방법

  • 엔터프라이즈 응용 프로그램을 배포하는 방법

  • 응용 프로그램 클라이언트를 배포하는 방법

  • 커넥터 모듈을 배포하는 방법

  • EJB Jar를 배포하는 방법

  • Ruby 응용 프로그램을 배포하는 방법

  • 응용 프로그램의 설명자를 보는 방법

  • 응용 프로그램을 시작하는 방법

  • 배포된 응용 프로그램을 재배포하는 방법

  • 배포된 응용 프로그램을 다시 시작하는 방법
