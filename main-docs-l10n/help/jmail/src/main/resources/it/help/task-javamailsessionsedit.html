Per modificare una sessione JavaMail

  1. Espandere il nodo Risorse nell'albero di navigazione.

  2. Nel nodo Risorse selezionare Sessioni JavaMail.

    Viene visualizzata la pagina Sessioni JavaMail.

  3. Nella pagina Sessioni JavaMail fare clic sul nome della sessione da modificare.

    Viene visualizzata la pagina Modifica sessione JavaMail.

  4. Nel campo Host posta specificare il nome host del server di posta predefinito.

    Se non viene fornita una proprietà host specifica per un protocollo, i metodi di connessione degli oggetti Store e Transport utilizzano questo valore. Il nome deve essere risolvibile in un nome host effettivo.

  5. Nel campo Utente predefinito digitare un altro nome utente da fornire per la connessione a un server di posta.

    Se non viene fornita una proprietà nome utente specifica per un protocollo, i metodi di connessione degli oggetti Store e Transport utilizzano questo valore. Il nome utente deve contenere solo caratteri alfanumerici, caratteri di sottolineatura, lineette e punti.

  6. Nel campo Indirizzo mittente predefinito digitare l'indirizzo di posta elettronica dell'utente predefinito.

    Il formato da usare è nome utente@host.dominio.

  7. Nel campo Ordine di distribuzione digitare un numero per specificare l'ordine di distribuzione della risorsa di sessione JavaMail.

    Le risorse con numero inferiore vengono caricate per prime all'avvio del server. Il valore predefinito è 100.

  8. Nel campo Descrizione modificare la descrizione per la sessione di posta.

  9. Selezionare la casella di controllo Abilitato dell'opzione Stato per abilitare la sessione di posta.

    Lo stato predefinito è Abilitato.

  10. Nel campo Protocollo di memorizzazione definire il metodo di comunicazione oggetto Store da utilizzare.

    Per impostazione predefinita il protocollo di memorizzazione è imap.

  11. Nel campo Classe protocollo di memorizzazione specificare la classe del metodo di comunicazione Store che implementa il protocollo Store desiderato.

    Per impostazione predefinita la classe protocollo di memorizzazione è com.sun.mail.imap.IMAPStore.

  12. Nel campo Protocollo di trasporto specificare il metodo di comunicazione di trasporto.

    Per impostazione predefinita, il protocollo di trasporto è smtp.

  13. Nel campo Classe protocollo di trasporto specificare il metodo di comunicazione per la classe Transport.

    Per impostazione predefinita la classe protocollo di trasporto è com.sun.mail.smtp.SMTPTransport.

  14. Selezionare la casella di controllo Debug per abilitare l'output di debug extra, compreso un trace del protocollo, per la sessione di posta.

    Se il livello di log JavaMail è impostato su FINE o FINER, l'output di debug viene generato e incluso nel file di log di sistema. Per informazioni sull'impostazione dei livelli del log, vedere Per configurare i livelli di log.

  15. Nella tabella Proprietà aggiuntive specificare le proprietà aggiuntive necessarie per le applicazioni, ad esempio una proprietà host o nome utente specifica del protocollo.

    Per la lista delle proprietà disponibili, consultare la documentazione dell'interfaccia API JavaMail (http://java.sun.com/products/javamail/javadocs/index.html). Le proprietà aggiuntive devono iniziare con mail- e utilizzare i trattini, e non i punti, come separatori. Ad esempio, mail-debug è una proprietà aggiuntiva corretta, mentre mail.debug non lo è. Il processo backend di GlassFish Server converte i trattini nei punti previsti dall'interfaccia API JavaMail.

    • Per aggiungere una proprietà, fare clic sul pulsante Aggiungi proprietà. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Nome e il valore della proprietà nel campo Valore.

    • Per modificare una proprietà, è necessario modificare il valore del campo Valore della proprietà.

    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Nome corrispondente alla proprietà da eliminare, quindi fare clic sul pulsante Elimina proprietà.

  16. Fare clic su Salva.

    Suggerimento:

    Per ripristinare le impostazioni predefinite, fare clic sul pulsante Carica impostazioni predefinite, quindi su Salva.
