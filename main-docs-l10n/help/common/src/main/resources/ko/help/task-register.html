GlassFish Server 등록 페이지를 보는 방법

GlassFish Server 등록 페이지에서는 GlassFish Server 설치를 등록할 수 있습니다.

GlassFish Server의 등록된 사용자는 다음과 같은 혜택을 받을 수 있습니다.

  • 새 버전, 패치 및 업데이트에 대한 통지

  • 개발자 제품, 서비스 및 교육에 대한 특별 제공

  • 백서, 교육 기사 및 GlassFish 제품 팀과의 라이브 웹 세미나 이용

  1. 탐색 트리에서 [일반 작업] 노드를 선택합니다.

    [일반 작업] 페이지가 열립니다.

  2. GlassFish 뉴스 아래에서 [등록] 링크를 누릅니다.

    GlassFish 등록 웹 페이지가 별도의 브라우저 탭이나 창에서 열립니다.
