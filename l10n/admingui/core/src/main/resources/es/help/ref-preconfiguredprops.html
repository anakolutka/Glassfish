Propiedades del sistema predefinidas

En Servidor empresarial existen las siguientes propiedades del sistema predefinidas.

com.sun.aas.installRoot

Directorio donde Servidor empresarial está instalado.

com.sun.aas.instanceRoot

Directorio de nivel superior para un dominio Servidor empresarial.

com.sun.aas.hostName

Nombre del sistema anfitrión (equipo).

com.sun.aas.javaRoot

Directorio de instalación para el tiempo de ejecución de Java.

com.sun.aas.imqLib

Directorio de biblioteca para el software GlassFish de Sun Información de resolución de problemas de.

com.sun.aas.domainName

Nombre del dominio. La propiedad no se utiliza en la configuración predeterminada, pero se puede utilizar para personalizar la configuración. El valor predeterminado es domain1.
