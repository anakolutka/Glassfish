Configurer les propriétés du magasin

  1. Dans l'arborescence de navigation, développez le nœud Configuration.
  2. Sous le nœud Configuration, sélectionnez le nœud Conteneur Web.

    La page Conteneur Web s'affiche.

  3. Dans la page Conteneur Web, cliquez sur l'onglet Propriétés du magasin.
  4. Dans le champ Intervalle d'obtention, entrez le nombre de secondes devant s'écouler avant que les données d'une session inactive ne soient supprimées du magasin.
  5. Dans le champ Répertoire, entrez le chemin du fichier dans lequel les données de session seront stockées.

    Ce dernier n'est nécessaire que si vous avez sélectionné le type de persistance Fichier.

  6. (Facultatif) Dans la section Autres propriétés, spécifiez d'autres propriétés.
  7. Cliquez sur Enregistrer.
