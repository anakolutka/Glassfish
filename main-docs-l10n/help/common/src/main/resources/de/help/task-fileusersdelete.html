So wird ein Benutzer aus der File Realm gelöscht

  1. Blenden Sie unter dem Knoten "Konfigurationen" die Serverinstanz oder die Clusterkonfiguration ein, für die Sie Admin-Realm-Benutzer verwalten möchten.

  2. Blenden Sie den Knoten "Sicherheit" ein.

  3. Blenden Sie den Knoten "Realms" ein.

  4. Wählen Sie den Knoten file.

    Die Seite "Realm bearbeiten" wird geöffnet.

  5. Klicken Sie auf der Seite "Realm bearbeiten" auf die Schaltfläche "Benutzer verwalten".

    Die Seite "Dateibenutzer" wird geöffnet.

  6. Aktivieren Sie auf der Seite "Dateibenutzer" das Kontrollkästchen links neben dem Namen des zu löschenden Benutzers.

  7. Klicken Sie auf "Löschen".

    Sie müssen in einem Dialogfeld bestätigen, dass der Benutzer gelöscht werden soll.

  8. Klicken Sie in diesem Dialogfeld auf "OK".
