Usuarios de archivos

Utilice la página Usuarios de archivos para gestionar los usuarios del dominio file. Los usuarios y grupos del dominio file aparecen en el archivo de claves, cuya ubicación está especificada por la propiedad file.

Los usuarios del dominio file pueden pertenecer a un grupo Java, categoría de usuarios clasificados por rasgos en común. Por ejemplo, los clientes de una aplicación de comercio electrónico pueden pertenecer al grupo CLIENTE, pero los que más gastan pertenecerían al grupo PREFERENTE. La clasificación de usuarios en grupos simplifica el control del acceso de gran cantidad de usuarios.

Inicialmente, tras la instalación de GlassFish Server, el único usuario es el administrador introducido durante la instalación. Por defecto, este usuario pertenece al grupo asadmin, en el dominio admin-realm, lo que le da derecho a modificar GlassFish Server. Los usuarios asignados a este grupo tendrán privilegios de administrador, es decir que podrán acceder a la herramienta asadmin y a la Consola de administración.

La página Usuarios de archivos muestra una lista de usuarios del dominio seleccionado. Para cada recurso, se proporciona la siguiente información:

ID de Usuario

Nombre de usuario.

Lista de Grupos

Grupos a los que pertenece el usuario.

La tabla Usuarios de archivos también contiene las siguientes opciones.

Nuevo

Botón para crear un nuevo usuario.

Suprimir

Botón para suprimir uno o varios usuarios seleccionados.
