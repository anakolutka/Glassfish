Per creare una risorsa di destinazione JMS

Quando si crea una risorsa di destinazione JMS, GlassFish Server crea una risorsa oggetto amministrato per la risorsa di destinazione. È possibile visualizzare questa risorsa nel nodo Connettori.

  1. Espandere il nodo Risorse nell'albero di navigazione.

  2. Nel nodo Risorse espandere il nodo Risorse JMS.

  3. Nel nodo Risorse JMS selezionare il nodo Risorse di destinazione JMS.

    Viene visualizzata la pagina Risorse di destinazione JMS.

  4. Nella pagina Risorse di destinazione JMS fare clic su Nuovo.

    Viene visualizzata la pagina Nuova risorsa di destinazione JMS.

  5. Nel campo Nome JNDI digitare il nome JNDI della risorsa.

    Si consiglia di utilizzare il prefisso del contesto secondario di denominazione jms/ per le risorse JMS. Ad esempio: jms/Queue.

  6. Nel campo Nome destinazione fisica digitare il nome della destinazione fisica da associare alla risorsa.

    GlassFish Server crea in modo automatico la destinazione fisica quando è necessaria e la elimina quando si elimina la risorsa di destinazione.

  7. Nell'elenco a discesa Tipo di risorsa selezionare javax.jms.Topic o javax.jms.Queue.

  8. Nel campo Descrizione digitare una descrizione della risorsa di destinazione.

  9. Selezionare la casella di controllo Abilitato dell'opzione Stato per abilitare la risorsa in fase di esecuzione.

    Questa opzione è abilitata per impostazione predefinita.

  10. Nell'area Proprietà aggiuntive specificare le proprietà aggiuntive per la risorsa di destinazione.

    GlassFish Server non definisce alcuna proprietà aggiuntiva per le risorse di destinazione JMS.

    Per aggiungere una proprietà, fare clic sul pulsante Aggiungi proprietà. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Nome e il valore della proprietà nel campo Valore.

  11. Nell'area Destinazioni, effettuare le operazioni riportate di seguito.

    1. Selezionare nella colonna Disponibili la destinazione o le destinazioni in cui verranno distribuite le applicazioni che utilizzano la risorsa.

      Le destinazioni disponibili includono tutti i cluster e le istanze server disponibili, oltre all'istanza server predefinita server.

    2. Fare clic su Aggiungi per spostare la destinazione nella colonna Selezionate.

  12. Fare clic su OK.
