So wird der GlassFish Server-Blog angezeigt

Im GlassFish Server-Blog finden Sie Neuigkeiten über die GlassFish Server-Community und Links zu zusätzlichen Informationen über GlassFish Server.

  1. Wählen Sie in der Navigationsbaumstruktur den Knoten "Allgemeine Aufgaben".

    Die Seite "Allgemeine Aufgaben" wird geöffnet.

  2. Klicken Sie unter "GlassFish News" auf den Link "Blog".

    Der GlassFish Server-Blog wird in einer separaten Registerkarte oder in einem Fenster Ihres Browsers geöffnet.
