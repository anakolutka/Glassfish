创建 JDBC 资源

Java DataBase Connectivity (JDBC) 资源（数据源）为应用程序提供连接到数据库的方法。

开始之前

为了创建 JDBC 资源，您必须先创建一个 JDBC 连接池。请参见创建 JDBC 连接池。

  1. 在导航树中，展开“资源”节点。
  2. 选择“JDBC 资源”节点。
  3. 在“JDBC 资源”页面上，单击“新建”。

    此时将打开“新建 JDBC 资源”页面。

  4. 在“J.N.D.I. 名称”字段中，键入一个唯一的资源名称。

    J.N.D.I. 名称可以在一个分布式计算环境中组织和定位组件，这与使用卡片目录组织和表示图书馆中书籍位置的方法类似。因此，J.N.D.I. 名称是访问 JDBC 资源的重要方法之一。按照约定，名称以 jdbc/ 字符串开头，例如 jdbc/payrolldb。请不要忘记正斜杠。

    名称只能包含字母数字、下划线、破折号或点字符。

  5. 从“池名称”下拉式列表中，选择要与新的 JDBC 资源关联的连接池。
  6. 在“说明”字段中，键入新 JDBC 资源的简短说明。
  7. 要禁用资源以便其不可用，请取消选中“启用”复选框。

    默认情况下，创建资源之后立即可以使用资源（已启用）。

  8. 单击“确定”。
