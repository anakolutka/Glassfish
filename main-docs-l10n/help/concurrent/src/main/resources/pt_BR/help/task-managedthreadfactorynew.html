Para Criar uma Fábrica de Threads Gerenciados

Uma fábrica de threads gerenciados especifica um XXX.

  1. Na árvore de navegação, expanda o nó Recursos.

  2. No nó Recursos, expanda o nó Recursos Concorrentes.

  3. No nó Recursos Concorrentes, selecione Fábricas de Threads Gerenciados.

    A página Fábricas de Threads Gerenciados será aberta.

  4. Na página Fábricas de Threads Gerenciados, clique em Novo.

    A página Nova Fábrica de Threads Gerenciados será aberta.

  5. No campo Nome da JNDI, digite um nome exclusivo.

    Esse nome será registrado no serviço atribuição de atribuição de nomes de JNDI. O nome JNDI organiza e localiza os componentes em um ambiente de computação distribuído, de maneira semelhante ao catálogo de cartão que organiza e representa os locais de livros em uma biblioteca. Consequentemente, o nome da JNDI torna-se um método importante de acessar o recurso.

  6. XXX

  7. No campo Descrição, digite uma descrição curta do recurso.

    Esta descrição é um valor de string e pode incluir no máximo 250 caracteres.

  8. Marque a caixa de seleção Status ativo para ativar o recurso.

    Por default, o recurso está ativado, significando que está disponível assim que é criado.

  9. Na seção Propriedades Adicionais, especifique propriedades adicionais para o recurso externo.

    O Productname; não define nenhuma propriedade adicional para os recursos da fábrica de threads gerenciados.

    Para adicionar uma propriedade, clique no botão Adicionar Propriedade. Na linha em branco que for exibida, digite o nome da propriedade no campo Nome e digite o valor da propriedade no campo Valor.

  10. Clique em OK.
