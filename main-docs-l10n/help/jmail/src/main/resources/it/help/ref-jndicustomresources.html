Risorse personalizzate

Una risorsa personalizzata specifica un object factory di risorse lato server personalizzato.

Utilizzare la pagina Risorse personalizzate per configurare le risorse JNDI personalizzate.

Per ogni risorsa personalizzata vengono fornite le informazioni riportate di seguito.

Nome JNDI

Il nome univoco che identifica la risorsa personalizzata.

Abilitato

L'impostazione è True se la risorsa personalizzata è abilitata oppure è False se la risorsa personalizzata è disabilitata.

Tipo di risorsa

Il tipo di risorsa personalizzata.

Descrizione

La descrizione della risorsa personalizzata.

La tabella Risorse contiene inoltre le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare una nuova risorsa personalizzata.

Elimina

Pulsante che consente di eliminare una o più risorse personalizzate selezionate.

Abilita

Pulsante che consente di abilitare una o più risorse personalizzate selezionate.

Disabilita

Pulsante che consente di disabilitare una o più risorse personalizzate selezionate.
