So integrieren Sie einen JDBC-Treiber

Ein Treiber, der die JDBC-API (JDBC-Treiber) unterstützt, übersetzt die JDBC-Aufrufe einer Anwendung in das Protokoll des Datenbankservers.

  • Um den JDBC-Treiber in eine administrative Domäne zu integrieren, führen Sie einen der folgenden Schritte aus:
    • Machen Sie den Treiber dem Ladeprogramm für gemeinsame Klassen zugänglich.
      1. Kopieren Sie die JAR- und ZIP-Dateien des Treibers in das Verzeichnis domain-dir/lib, oder kopieren Sie die zugehörigen Klassendateien in das Verzeichnis domain-dir/lib/ext.
      2. Starten Sie die Domäne neu.
    • Machen Sie den Treiber dem Ladeprogramm für Systemklassen zugänglich.
      1. Wählen Sie in der Navigationsstruktur den Anwendungsserverknoten.
      2. Wählen Sie auf der Seite "Allgemeine Informationen" die Registerkarte "JVM-Einstellungen" aus.
      3. Wählen Sie auf der Seite "Allgemeine JVM-Einstellungen" die Registerkarte "Pfadeinstellungen" aus .
      4. Geben Sie im Feld "Klassenpfadsuffix" den voll qualifizierten Pfadnamen für die JAR-Datei des Treibers ein.
      5. Klicken Sie auf "Speichern".
      6. Starten Sie den Server neu.
