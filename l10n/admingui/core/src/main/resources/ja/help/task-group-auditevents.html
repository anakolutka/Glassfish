Enterprise Server のイベントを監査する

Enterprise Server が記録したエラーやセキュリティー侵入などの重大なイベントを監査します。Enterprise Server は、すべての認証イベントをログファイルに記録します。完全なアクセスログには、Enterprise Server で行われるすべてのアクセスイベントが連続して記録されます。

Enterprise Server のイベントの監査には次の作業が含まれます。

  • 「監査モジュールを作成する」

  • 「監査モジュールを編集する」

  • 「監査モジュールを削除する」

  • 「監査ログを有効または無効にする」

  • 「有効な監査モジュールを設定する」

  • 「デフォルト監査モジュールを使用する」
