Configuraciones de seguridad de mensajes

Utilice la página Configuraciones de seguridad de mensajes para ver y modificar las configuraciones disponibles para los proveedores de seguridad de mensajes.

Para configurar la seguridad de mensajes, necesita un conocimiento básico de seguridad y de conceptos de servicios web. Para obtener más información, consulte el Capítulo 3, sobre administración de la seguridad de los mensajes, en Oracle GlassFish Server Security Guide y el Capítulo 6, sobre administración de aplicaciones web, en Oracle GlassFish Server Administration Guide.

GlassFish Server implanta la seguridad de los mensajes mediante proveedores de seguridad de mensajes integrados en la capa de procesamiento SOAP. Los proveedores de seguridad de mensajes dependen de otras utilidades de seguridad de GlassFish Server.

Para cada configuración, se proporciona la siguiente información.

Capa de autenticación

Capa del mensaje en la que se lleva a cabo la autenticación. Por defecto, se proporciona una configuración SOAP. También es posible proporcionar una configuración HttpServlet.

Proveedor por defecto

Proveedor de servidor por defecto para esta configuración. Por defecto, no hay ningún proveedor de servidor configurado.

Proveedor de cliente por defecto

Proveedor de cliente por defecto para esta configuración. No hay ningún proveedor de cliente configurado por defecto.

La tabla Configuración de seguridad de mensajes también contiene las siguientes opciones.

Nuevo

Botón para crear una nueva configuración.

Suprimir

Botón para suprimir una o varias configuraciones seleccionadas.
