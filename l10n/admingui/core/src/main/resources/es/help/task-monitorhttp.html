Para visualizar los datos de supervisión del servicio HTTP

  1. En el árbol de navegación, seleccione el nodo Application Server.
  2. En la página Información general, haga clic en la ficha Supervisar.
  3. En la página Supervisión del contenedor web, haga clic en la ficha Servicio HTTP.

    Se abrirá la página Supervisión del servicio HTTP.

  4. (Opcional) En la lista desplegable Servidor virtual, seleccione un servidor para visualizar las estadísticas de solicitudes de dicho servidor.
  5. (Opcional) En la lista desplegable Escucha HTTP, seleccione una escucha para visualizar las estadísticas de solicitudes de dicho servidor.
  6. (Opcional) Haga clic en el botón Actualizar para actualizar los datos de supervisión mostrados.
