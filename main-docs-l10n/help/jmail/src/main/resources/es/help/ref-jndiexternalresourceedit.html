Editar Recurso Externo

Utilice la página Editar recursos externos para modificar un recurso JNDI externo.

La página Editar recursos externos contiene las siguientes opciones.

Nombre JNDI

Nombre único que identifica el recurso. El nombre JNDI es un campo de sólo lectura. Sólo puede especificar el nombre JNDI al crear un recurso externo nuevo.

Tipo de recurso

Definición de tipo totalmente cualificada para el recurso.

Clase de fábrica

Nombre especificado por el usuario para la clase de fábrica. Esta clase implanta la interfaz javax.naming.spi.InitialContextFactory.

Búsqueda JNDI

El valor JNDI para buscar en el repositorio externo.

Por ejemplo, cuando se crea un recurso externo para conectar a un repositorio externo, para probar una clase de bean, la búsqueda JNDI puede seguir el formato: cn=testmybean.

Orden de despliegue

Orden de despliegue del recurso externo.

Descripción

Descripción de este recurso externo.

Estado

Indica si el recurso está disponible en el tiempo de ejecución.

Si sólo existe la instancia de servidor por defecto, server en el dominio, la disponibilidad en tiempo de ejecución del recurso puede estar activada o desactivada. La disponibilidad en tiempo de ejecución está activa por defecto.

Si se ha creado algún cluster o alguna instancia independiente en el dominio, utilice el separador Destino para controlar la disponibilidad del recurso en tiempo de ejecución en sus destinos.

Propiedades Adicionales

Propiedades adicionales del recurso externo. GlassFish Server no define ninguna propiedad adicional para los recursos externos.
