有効な監査モジュールを設定する

  1. ナビゲーションツリーで、「設定」ノードを展開します。
  2. 「設定」ノードの下で、「セキュリティー」ノードを選択します。

    「セキュリティ」ページが開きます。

  3. 「監査モジュール」ドロップダウンリストから、サーバーが使用する監査モジュールの名前を選択します。

    デフォルトでは、default という名前の監査モジュールだけが使用可能です。

  4. 「保存」をクリックします。
