在「工廠類別」欄位中，鍵入要建立的外部資源的 JNDI 工廠類別外部儲存區域。

「工廠類別」是使用者為工廠類別指定的名稱，例如 com.sun.jndi.ldap。此類別會實行 javax.naming.spi.InitialContextFactory 介面。

在「JNDI 查找」欄位中，鍵入要在外部儲存區域中查找的 JNDI 值。

例如，建立連線至外部儲存區域的外部資源之後，為了測試 Bean 類別，可進行如下 JNDI 查詢︰cn=testmybean。

在「描述」欄位中，鍵入資源的簡短描述。

該描述是一個字串值，最多可以包含 250 個字元。

選取「啟用狀態」核取方塊，以啟用資源。

依預設會啟用資源，表示資源於建立後就可以使用。

在「其他特性」區段中，指定外部資源的其他特性。

GlassFish Server 未定義外部資源的任何其他特性。

若要增加特性，請按一下「增加特性」按鈕。在出現的空白列中，於「名稱」欄位中鍵入特性名稱，然後在「值」欄位中鍵入特性值。

按一下「確定」。
