Per modificare un host JMS

  1. Espandere il nodo Configurazione nell'albero di navigazione.

  2. Selezionare l'istanza da configurare:

    • Per configurare un'istanza particolare, selezionare il nodo di configurazione dell'istanza. Ad esempio, per l'istanza predefinita server, selezionare il nodo server-config.

    • Per configurare le impostazioni predefinite per le istanze future che utilizzano una copia di default-config, selezionare il nodo default-config.

  3. Nel nodo Configurazione espandere il nodo Java Message Service.

  4. Nel nodo Java Message Service selezionare il nodo Host JMS.

    Viene visualizzata la pagina Host JMS.

  5. Nella pagina Host JMS selezionare l'host da modificare.

    Viene visualizzata la pagina Modifica host JMS.

  6. Nel campo Host digitare il nome o l'indirizzo IP (Internet Protocol) del sistema su cui verrà eseguito l'host JMS (localhost o il nome del sistema locale o remoto).

  7. Nel campo Porta digitare il numero di porta del servizio JMS.

    Modificare questo campo solo se il servizio JMS da utilizzare è in esecuzione su una porta non predefinita. La porta predefinita è la porta numero 7676.

  8. Nei campi Nome utente amministratore e Password amministratore digitare il nome utente e la password del broker Message Queue.

    Sono diversi dal nome utente e dalla password di GlassFish Server. Modificare questi campi solo se i valori del broker di Message Queue sono stati modificati tramite il comando as-install-parent/mq/bin/imqusermgr. I valori predefiniti sono admin e admin.

  9. Fare clic su Salva.

    Suggerimento:

    Per ripristinare le impostazioni predefinite, fare clic sul pulsante Carica impostazioni predefinite, quindi su Salva.
