Para desplegar una aplicación en un cluster

  1. En el árbol de navegación, amplíe el nodo Clusters.

  2. En el nodo Clusters ampliado, seleccione el nodo para el cluster.

    Se abre la página Información General para el cluster.

  3. En la página Información General, haga clic en el separador Aplicaciones.

    Se abre la página Aplicaciones para el cluster.

  4. En la página Aplicaciones, haga clic en Desplegar.

    Se abre la página Desplegar Aplicaciones o Módulos.

  5. En la página Desplegar Aplicaciones o Módulos, especifique la información necesaria sobre la aplicación y haga clic en Aceptar.

    Para obtener más información, consulte la ayuda en pantalla de Desplegar Aplicaciones o Módulos.
