範圍

Enterprise Server 預先配置了三個範圍： file、certificate 以及 admin-realm。您也可以建立 ldap、JDBC、solaris 以及自訂範圍。伺服器上的每種類型通常只有一個範圍，但是在 Enterprise Server 上有以下兩個 file 範圍︰ file 和 admin-realm。同一類型有兩個範圍，用於兩種不同目的。 系統上的每個虛擬伺服器可以有不同的憑證資料庫。

[範圍] 頁面會顯示可用範圍清單。會為每個範圍提供以下資訊：

  • 名稱

  • 類別名稱
