「レルムを編集」ページが開きます。

「レルムを編集」ページで、「ユーザーを管理」ボタンをクリックします。

「管理ユーザー」ページが開きます。

「管理ユーザー」ページで、「新規」をクリックします。

「新規ファイル・レルム・ユーザー」ページが開きます。

「ユーザーID」フィールドに、ユーザーの名前を入力します。

「グループ・リスト」フィールドに、ユーザーが属するグループのリストを入力します。

複数のグループ名はカンマで区切ります。これらのグループは、他の場所で定義される必要はありません。

「パスワード」フィールドに、一意のパスワードを入力します。

「新規パスワードの確認」フィールドに、ユーザー・パスワードを再度入力します。

「OK」をクリックします。
