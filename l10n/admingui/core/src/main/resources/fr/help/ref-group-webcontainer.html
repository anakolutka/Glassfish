Conteneur Web

Le conteneur Web héberge des applications Web. Il fournit l'environnement dans lequel sont executés les servlets et les fichiers JavaServer PagesTM (JSPTM). Il n'existe aucun paramètre à l'échelle du conteneur pour le conteneur Web dans la Console d’administration.
