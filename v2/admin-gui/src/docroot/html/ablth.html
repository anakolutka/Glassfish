To configure the HTTP Protocol

  1. Expand the Configuration node to select the HTTP Service node.

  2. Click the HTTP Protocol tab.

  3. Define the HTTP Protocol settings as follows:

    • Version: Enter the version of the HTTP protocol to be used (HTTP/1.0 or HTTP/1.1).

    • DNS lookup: Select this checkbox to enable lookup of the DNS entry for the client. Default is disabled.

    • SSL: Select to enable/disable security in the server globally. Leave this value enabled to be able to use SSL for any listener that has security enabled. The default is Enabled.

    • Forced Response Type: Enter the response type to be used if there is no MIME mapping available matching the extension. The default value is text/html; charset=iso-8859-1.

    • Default Response Type: Enter the default response type. The default value is text/html; charset=iso-8859-1. The value is a semicolon-delimited string consisting of content-type, encoding, language, and charset.

  4. Click Save to save the changes, or Load Defaults to return to the default settings.
