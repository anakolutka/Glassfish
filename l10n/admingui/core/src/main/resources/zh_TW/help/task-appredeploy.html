重新部署已部署的應用程式

重新部署應用程式會部署應用程式的新版本，同時保留應用程式最初部署時的設定及其他選項。

開始之前

請確定您要重新部署的應用程式已部署完成。

  1. 在瀏覽樹狀結構中，選取 [應用程式] 節點。

    [應用程式] 頁面會隨即開啟。

  2. 在 [已部署的應用程式] 表格中，於您要重新部署之應用程式的列中按一下 [重新部署]。

    [重新部署應用程式或模組] 頁面會隨即開啟。

  3. 如果需要保留應用程式的使用中階段作業，再於完成重新部署時加以復原，請選取 [保留階段作業] 核取方塊。

    如果應用程式有任何使用中的階段作業無法保留或復原，完成重新部署之後將無法使用任何階段作業。但是，重新部署會繼續並記錄一則警告。

  4. 指定包含要重新部署之應用程式的歸檔位置。
    • 如果歸檔是常駐在用戶端機器上或可從用戶端機器存取的檔案，請選取 [要上傳至伺服器的封裝檔案] 選項。

      用戶端是您要在其上透過瀏覽器檢視管理主控台的主機。

      按一下 [瀏覽] 以瀏覽至該檔案，或者鍵入該檔案的完整路徑。

    • 如果歸檔是常駐在伺服器機器上的檔案，或是展開的目錄中未封裝的應用程式，請選取 [可從應用程式伺服器存取的本機封裝檔案或目錄] 選項。

      伺服器機器是執行網域管理伺服器 (DAS) 的主機。

      按一下 [瀏覽檔案] 以瀏覽至檔案，或按一下 [瀏覽資料夾] 以瀏覽至目錄。或者，鍵入檔案或目錄的完整路徑名稱。從展開的目錄部署適用於進階開發者，但不建議在生產環境下進行該作業。

  5. 按一下 [確定]。
