So werden JVM-Pfadeinstellungen konfiguriert

  1. Wählen Sie in der Navigationsbaumstruktur den Knoten "Konfiguration".

  2. Wählen Sie die zu ändernde Konfiguration.

  3. Wählen Sie unter dem Konfigurationsknoten den JVM-Einstellungsknoten.

    Die Seite "Allgemeine JVM-Einstellungen" wird geöffnet.

  4. Klicken Sie auf der Seite "Allgemeine JVM-Einstellungen" auf die Registerkarte "Pfadeinstellungen".

    Die Seite "JVM-Pfadeinstellungen" wird geöffnet.

  5. Geben Sie im Feld "Präfix für Pfad zur nativen Library" die vollständigen Pfade der JAR-Dateien ein, die dem Pfad zur nativen Library vorangestellt werden sollen.

    Geben Sie pro Zeile einen Pfad ein, um mehrere Dateien voranzustellen.

  6. Geben Sie im Feld "Suffix für Pfad zur nativen Library" die vollständigen Pfade der JAR-Dateien ein, die an den Pfad zur nativen Library angehängt werden sollen.

    Geben Sie pro Zeile einen Pfad ein, um mehrere Dateien anzuhängen.

  7. Klicken Sie auf "Speichern".

  8. Starten Sie den Server neu.
