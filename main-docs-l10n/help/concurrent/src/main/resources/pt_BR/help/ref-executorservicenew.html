Novo Serviço do Executor Gerenciado

Use a página Novo Serviço do Executor Gerenciado para criar um recurso de serviço do executor gerenciado.

A página Novo Serviço do Executor Gerenciado contém as opções a seguir.

Nome JNDI

Um nome JNDI exclusivo que identifica o recurso. Por convenção, o nome começa com um indicador de tipo de recurso e uma barra, tais como concurrent/, jdbc/ ou jms/.

Informações de Contexto

Os contextos a serem propagados para threads:, Classloader; JNDI, Segurança ou WorkArea. A propagação de contexto pode ser ativada ou desativada. Se a opção estiver ativada, os contextos selecionados serão propagados. Se esta opção estiver desativada, nenhum dos contextos será propagado, mesmo que eles estejam selecionados. O valor default é Ativado, com todos os quatro contextos selecionados.

Status

O status do recurso. O recurso pode ser ativado ou desativado. O valor default é Ativado.

Prioridade do Thread

A prioridade a ser designada ao threads criados. O valor default é 5.

Tarefas de Longa Execução

Se o recurso deve ser usado para tarefas de longa execução. Se esta opção estiver ativada, tarefas de longa execução não serão relatadas como paradas. Por default, essa opção está desativada.

Suspensa Após

O número de segundos durante os quais as tarefas podem ser executadas antes de se considerar que não estão respondendo. O valor default é 0.

Descrição

Uma descrição deste recurso.

Tamanho do Principal

O número de threads a serem mantidos em um pool de thread. O valor default é 0.

Tamanho Máximo do Pool

O número máximo de threads que um pool pode conter. O valor default é 2147483647.

Conexão Ativa

O tempo em segundos durante os quais os threads podem permanecer inativos quando o número de threads for maior que o tamanho do núcleo. O valor default é 60.

Vida Útil do Thread

O número de segundos durante os quais os threads podem permanecer em um pool de threads antes de serem expurgados, independentemente de se o número de threads é maior do que o tamanho do núcleo ou de se os threads estão inativos. O valor default é 0.

Capacidade da Fila da Tarefa

O número de tarefas submetidas que podem ser armazenadas na fila de tarefas que estão aguardando execução. O valor default é 2147483647.

Propriedades Adicionais

Propriedades adicionais do recurso. O GlassFish Server não define propriedades adicionais para este tipo de recurso. Além disso, este tipo de recurso não usa atualmente propriedades adicionais.

Alvos

(Esta opção só será exibida se os clusters ou instâncias standalone tiverem sido criados no domínio.)

Clusters e instâncias standalone do recurso. Mova os alvos desejados para a coluna Alvos Selecionados utilizando os botões Adicionar e Adicionar Tudo. Mova os alvos desnecessários para a coluna Alvos Disponíveis utilizando os botões Remover e Remover Tudo.
