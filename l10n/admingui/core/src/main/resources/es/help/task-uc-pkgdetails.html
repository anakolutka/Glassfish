Cómo ver información detallada de un componente

Herramienta de actualización proporciona información detallada sobre los componentes instalados, las actualizaciones disponibles para los componentes instalados y los componentes de complementos.

  1. En el árbol de navegación, seleccione el nodo Herramienta de actualización.

    Se abrirá la página Componentes instalados.

  2. Haga clic en la ficha de la página donde aparezca el componente.

    El componente aparecerá en una de las siguientes paginas:

    • Componentes instalados

    • Actualizaciones disponibles

    • Complementos disponibles

  3. En la tabla de componentes, seleccione el componente.

    Aparecerá la página Detalles de los componentes del componente seleccionado.
