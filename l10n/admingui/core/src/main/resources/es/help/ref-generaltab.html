Información general

La página Información general muestra la siguiente información:

Nombre

Nombre del servidor .

Estado

Estado actual de la del servidor. El servidor puede detenerse, iniciarse o ejecutarse.

Equilibrador de la carga de HTTP

Nombre del equilibrador de carga, si está activado.

Puerto(s) HTTP

Puertos configurados actualmente para solicitudes y respuestas HTTP.

Puerto(s) IIOP

Puertos configurados actualmente para solicitudes y respuestas IIOP.

Directorio de configuración

El directorio del equipo host que contiene archivos de configuración como, por ejemplo, domain.xml.

Versión instalada

identificador de versión del producto.

Depuración

Identifica si la depuración está habilitada. Si la depuración está habilitada, también aparecerá el número de puerto.
