Per modificare una risorsa oggetto amministrato

  1. Espandere il nodo Risorse nell'albero di navigazione.

  2. Espandere il nodo Connettori.

  3. Nel nodo Connettori selezionare il nodo Risorse oggetto amministrato.

    Viene visualizzata la pagina Risorse oggetto amministrato.

  4. Nella pagina Risorse oggetto amministrato fare clic sul nome della risorsa oggetto amministrato da modificare.

    Viene visualizzata la pagina Modifica risorsa oggetto amministrato.

  5. Nell'elenco a discesa Resource Adapter selezionare il Resource Adapter per la risorsa oggetto amministrato.

  6. Nell'elenco a discesa Tipo di risorsa selezionare il tipo di dati Java per la risorsa oggetto amministrato.

  7. Nell'elenco a discesa Nome classe selezionare il nome della classe di implementazione associato al tipo di risorsa.

  8. Nel campo Descrizione digitare la descrizione della risorsa oggetto amministrato.

  9. Nel campo Ordine di distribuzione digitare un numero per specificare l'ordine di distribuzione della risorsa oggetto amministrato.

    Le risorse con numero inferiore vengono caricate per prime all'avvio del server. Il valore predefinito è 100.

  10. Selezionare la casella di controllo Abilitato dell'opzione Stato per abilitare la risorsa oggetto amministrato.

    Questa opzione è abilitata per impostazione predefinita.

  11. Nella tabella Proprietà aggiuntive specificare le proprietà aggiuntive per la risorsa oggetto amministrato.

    Le proprietà disponibili dipendono dal Resource Adapter per il connection pool del connettore.

    • Per aggiungere una proprietà, fare clic sul pulsante Aggiungi proprietà. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Nome e il valore della proprietà nel campo Valore.

    • Per modificare una proprietà, è necessario modificare il valore del campo Valore della proprietà.

    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Nome corrispondente alla proprietà da eliminare, quindi fare clic sul pulsante Elimina proprietà.

  12. Nella scheda Destinazione fare clic su Gestisci destinazioni per modificare le destinazioni in cui è distribuito l'oggetto amministrato.

    Tornare alla pagina Modifica risorsa oggetto amministrato facendo clic sulla scheda Generale.

  13. Fare clic su Salva.

    Suggerimento:

    Per ripristinare le impostazioni predefinite, fare clic sul pulsante Carica impostazioni predefinite, quindi su Salva.
