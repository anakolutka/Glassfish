Risorse JDBC

Una risorsa (origine dati) JDBC (Java DataBase Connectivity) fornisce alle applicazioni i mezzi per connettersi a un database. In genere l'amministratore crea una risorsa JDBC per ogni database al quale accedono le applicazioni distribuite in un dominio. È tuttavia possibile creare più risorse JDBC per un singolo database.

Le applicazioni ottengono una connessione al database da un connection pool mediante la ricerca di un'origine dati nella struttura dell'interfaccia API JNDI (Java Naming and Directory Interface) e la successiva richiesta della connessione. Il connection pool associato all'origine dati fornisce la connessione all'applicazione.

Utilizzare la pagina Risorse JDBC per configurare le risorse JDBC.

Per ogni risorsa vengono fornite le informazioni riportate di seguito.

Nome JNDI

Il nome univoco che identifica la risorsa JDBC.

Abilitato

L'impostazione è True se la risorsa è abilitata oppure è False se la risorsa è disabilitata.

Connection pool

Il connection pool JDBC associato alla risorsa.

Descrizione

Una breve descrizione della risorsa.

La tabella Risorse contiene inoltre le opzioni riportate di seguito.

Nuovo

Pulsante che consente di creare una nuova risorsa.

Elimina

Pulsante che consente di eliminare una o più risorse selezionate.

Abilita

Pulsante che consente di abilitare una o più risorse selezionate.

Disabilita

Pulsante che consente di disabilitare una o più risorse selezionate.
