Per modificare un servizio esecutore pianificato gestito

  1. Espandere il nodo Risorse nell'albero di navigazione.

  2. Nel nodo Risorse espandere il nodo Risorse concorrenti.

  3. Nel nodo Risorse concorrenti selezionare Servizi esecutore pianificati gestiti.

    Viene visualizzata la pagina Servizi esecutore pianificati gestiti.

  4. Nella pagina Servizi esecutore pianificati gestiti fare clic sul nome della risorsa da modificare.

    Viene visualizzata la pagina Modifica servizio esecutore pianificato gestito.

    Nota:

    Il campo Nome JNDI è di sola lettura. È possibile specificare il nome JNDI solo quando si crea una nuova risorsa.

  5. XXX

  6. Nel campo Descrizione digitare una breve descrizione della risorsa.

    La descrizione è un valore stringa con lunghezza limitata a 250 caratteri.

  7. Selezionare la casella di controllo Abilitato dell'opzione Stato per abilitare la risorsa.

  8. Nella sezione Proprietà aggiuntive specificare le proprietà aggiuntive per la risorsa.

    GlassFish Server non definisce alcuna proprietà aggiuntiva per le risorse dei servizi esecutore pianificati gestiti.

    • Per aggiungere una proprietà, fare clic sul pulsante Aggiungi proprietà. Nella riga vuota visualizzata, digitare il nome della proprietà nel campo Nome e il valore della proprietà nel campo Valore.

    • Per modificare una proprietà, è necessario modificare il valore del campo Valore della proprietà.

    • Per eliminare una proprietà, selezionare la casella di controllo a sinistra del campo Nome corrispondente alla proprietà da eliminare, quindi fare clic sul pulsante Elimina proprietà.

  9. Fare clic su Salva.
