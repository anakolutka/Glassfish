공급자 구성

공급자 구성 페이지에서는 SOAP 또는 HttpServlet 인증 계층에 대한 메시지 보안 공급자 구성을 구성할 수 있습니다.

기본적으로 SOAP 인증 계층에 대해 네 개의 공급자가 구성됩니다.

각 공급자 구성에 대해 공급자 구성 페이지에는 다음 정보가 포함됩니다.

공급자 아이디

공급자 구성의 이름입니다.

공급자 유형

공급자 유형으로, 다음 중 하나가 될 수 있습니다.

client

공급자가 클라이언트 인증 공급자임을 지정합니다.

server

공급자가 서버 인증 공급자임을 지정합니다.

client-server

공급자가 클라이언트 및 서버 인증 공급자임을 지정합니다.

기본 공급자

true로 설정하면 공급자는 기본 공급자입니다.

클래스 이름

공급자의 Java 구현 클래스의 이름입니다.

관련 작업

  • 메시지 보안 공급자 사용을 위해 Enterprise Server 기능을 구성하는 방법

  • 메시지 보안 구성을 만드는 방법

  • 메시지 보안 구성을 편집하는 방법

  • 메시지 보안 구성을 삭제하는 방법

  • 메시지 보안 공급자 구성을 만드는 방법

  • 메시지 보안 공급자 구성을 편집하는 방법

  • 메시지 보안 공급자 구성을 삭제하는 방법

  • 응용 프로그램 클라이언트에 대한 메시지 보안을 활성화하는 방법
