Affichage de fichiers journaux pour le serveur DAS

  1. Dans l'arborescence de navigation, sélectionnez le noeud du serveur (serveur d'administration).

    La page Informations générales apparaît.

  2. Sur la page Informations générales, cliquez sur le bouton Afficher les fichiers journaux.

    La fenêtre du visualiseur de journaux apparaît.
