Déployer les applications/modules d'entreprise

Pour déployer une application, utilisez la page Déployer les applications/modules d'entreprise. La page propose les options suivantes :

Type

Le type d'application que vous déployez. L'unique type d'application disponible dans cette version est Application Web (.war).

Lieu

Le lieu de l'archive de l'application que vous déployez.

Les options suivantes indiquent les points d'accès à l'archive et précisent si cette dernière est un fichier ou un répertoire.

Fichier empaqueté à télécharger sur le serveur

L'archive est contenue dans un fichier stocké sur la machine client ou accessible depuis cette même machine.

L'ordinateur client est l'hôte sur lequel vous visualisez la Console d’administration via un navigateur.

Fichier empaqueté ou répertoire local accessible depuis le serveur d'application

L'archive est un fichier stocké sur la machine serveur ou une application non empaquetée contenue dans un répertoire éclaté.

Le serveur est l'hôte qui exécute le serveur d'administration du domaine Application Server.

Nom de l'application

Le nom de l'application.

Racine contexte

Le chemin jusqu'à l'application. Dans l'URL de l'application, la racine contexte suit immédiatement le numéro de port (http://host:port/context-root/... ). La racine contexte doit commencer par une barre oblique, par exemple /hello.

Serveurs virtuels

Les serveurs virtuels associés à cette application.

Statut activé

Si cette option est sélectionnée, l'application est activée. Cette option est sélectionnée par défaut.

Exécution du vérificateur activée

Si cette option est sélectionnée, un utilitaire de vérification est exécuté pour vérifier si l'application est compatible avec JavaTM Platform, Enterprise Edition (plate-forme Java EE) et Enterprise Server.

Précompilation des JSP activée

Si cette option est sélectionnée, les fichiers JavaServer Pages ( JSPTM) sont précompilés. Si cette option n'est pas sélectionnée, les fichiers JSP sont compilés au moment de l'exécution lorsqu'ils sont utilisés pour la première fois. Par défaut, cette option n'est pas sélectionnée.

Bibliothèques

Les fichiers JAR de la bibliothèque requis par l'application.

Description

Une courte description de l'application.
