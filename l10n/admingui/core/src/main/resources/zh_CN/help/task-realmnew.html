创建领域

Enterprise Server 预先配置了三个领域：file、certificate 和 admin-realm。还可以创建 ldap、JDBC、solaris 和自定义领域。通常，一台服务器上每种类型的领域都会有一个，但在 Enterprise Server 上有两个 file 领域：file 和 admin-realm。这两个领域的类型相同但用于两种不同的目的。还可以在系统中为每台虚拟服务器创建不同的证书数据库。

  1. 在导航树中，展开“配置”节点。
  2. 如果域配置为支持群集，请选择您要更改的配置。

    示例是使用群集配置文件或企业配置文件创建的域。

  3. 展开“安全性”节点。
  4. 选择“领域”节点。
  5. 在“领域”页面中，单击“新建”。

    将显示“新建领域”页面。

  6. 在“名称”字段中为领域输入名称。
  7. 指定要创建的领域的类名。

    下表显示了有效选项。

    领域名

    类名

    file

    com.sun.enterprise.security.auth.realm.file.FileRealm

    certificate

    com.sun.enterprise.security.auth.realm.certificate.CertificateRealm

    ldap

    com.sun.enterprise.security.auth.realm.ldap.LDAPRealm

    JDBC

    com.sun.enterprise.security.auth.realm.jdbc.JDBCRealm

    solaris

    com.sun.enterprise.security.auth.realm.solaris.SolarisRealm

    custom

    登录领域类的名称
  8. 为领域添加任何所需的其他属性。
    1. 单击“添加属性”。
    2. 在“名称”字段中，输入属性的名称。

      • 有关“File 领域”属性的说明，请参见特定于 FileRealm 类的属性。

      • 有关“证书领域”属性的说明，请参见特定于 CertificateRealm 类的属性。

    3. 在“值”字段中输入属性的值。
  9. 单击“确定”。
