So zeigen Sie Batchjobausführungen an

  1. Wählen Sie in der Navigationsbaumstruktur das GlassFish Server-Cluster oder die -Serverinstanz, für die bzw. für das Sie die Batchjobausführungen anzeigen möchten.

    Die Seite "Allgemeine Informationen" wird geöffnet.

  2. Wählen Sie die Registerkarte "Batch".

    Die Seite "Batchjobausführungen" wird geöffnet.

  3. Klicken Sie auf der Seite "Batchjobausführungen" auf eine Ausführungs-ID, um Details zu der jeweiligen Ausführung anzuzeigen.

    Die Seite "Details zur Batchjobausführung" wird geöffnet.

  4. Um Details zu den Schritten in der Batchjobausführung anzuzeigen, klicken Sie auf die Registerkarte "Ausführungsschritte".

    Die Seite "Schritte der Batchjobausführung" wird geöffnet.

  5. Klicken Sie auf die Schaltfläche "Zurück", um zur Seite "Batchjobausführungen" zurückzukehren.
