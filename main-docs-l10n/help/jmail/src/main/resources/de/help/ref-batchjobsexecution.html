Details zur Batchjobausführung

Auf der Seite "Details zur Batchjobausführung" werden Details zu einer bestimmten Ausführung eines Batchjobs angezeigt. Es werden folgende Informationen angezeigt.

Instanzname
Clustername

Der Name des GlassFish-Clusters oder der GlassFish-Serverinstanz, auf das bzw. auf die sich die Konfiguration bezieht.

Jobname

Der Name des Batchjobs.

Ausführungs-ID

Die ID, die der Ausführung des Batchjobs zugewiesen ist. Beim erstmaligen Starten eines Jobs und bei jedem Neustart der vorhandenen Ausführung wird eine neue Ausführung erstellt.

Schrittzahl

Die Anzahl Schritte in der Ausführung des Batchjobs.

Batchstatus

Der von der Batchlaufzeit festgelegte Status der Batchjobausführung.

Beendigungsstatus

Der Status der Batchjobausführung, der von der Job-XML für den Job oder von der Batchanwendung festgelegt wurde. Standardmäßig sind Beendigungsstatus und Batchstatus identisch, es sei denn, der Beendigungsstatus wird explizit überschrieben.

Startzeit

Der Zeitpunkt, an dem die Batchjobausführung gestartet wurde.

Endzeit

Der Zeitpunkt, an dem die Batchjobausführung abgeschlossen wurde.

Jobparameter

Die an die Batchlaufzeit für die Batchjobausführung übergebenen Eigenschaften, ausgedrückt als Name/Wert-Paare.

Verwandte Aufgaben

  • So zeigen Sie Batchjobausführungen an

  • So konfigurieren Sie die Batchlaufzeit
