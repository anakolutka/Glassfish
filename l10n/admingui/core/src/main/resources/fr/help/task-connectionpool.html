Configurer le pool de connexions du service HTTP

  1. Dans l'arborescence de navigation, développez le nœud Configuration.
  2. Sous le nœud Configuration, sélectionnez le nœud Service HTTP.
  3. Dans la page Service HTTP, cliquez sur l'onglet Pool de connexions.

    La page Pool de connexions s'ouvre.

  4. Dans le champ Nombre max. de connexions en attente, entrez le nombre maximal de connexions en attente à autoriser pour un module d'écoute HTTP.

    La valeur par défaut est 4096.

  5. Dans le champ Taille de la file d'attente, entrez la taille maximale, en octets, de la file d'attente des connexions.

    Cette valeur indique également le nombre maximum de connexions en attente acceptées par le serveur. La valeur par défaut est 4096.

  6. Dans le champ Taille du tampon de réception, entrez la taille, en octets, du tampon de réception pour un module d'écoute HTTP.

    La valeur par défaut est 4096.

  7. Dans le champ Taille du tampon d'envoi, entrez la taille, en octets, du tampon d'envoi pour un module d'écoute HTTP.

    La valeur par défaut est 8192.

  8. Cliquez sur Enregistrer.

    Astuce – Pour revenir aux paramètres par défaut, cliquez sur le bouton Charger valeurs par défaut, puis sur Enregistrer.
