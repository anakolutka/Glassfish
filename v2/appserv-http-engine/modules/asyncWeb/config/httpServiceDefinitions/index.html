Congratulation, AsyncWeb is running on top of Grizzly!

The asyncWeb.xml configuration file is located under <install_dir>/domains/<domain_name>/config/asynWeb.xml where <install_dir> is the Application Server installation directory, and <domain_name<> is the domain name (for example, domain1).

To deploy HttpService in GlassFish, drop your <service>.xml under <install_dir>/domains/<domain_name>/config/httpServiceDefinitions/ and your application jars under <install_dir>/lib
