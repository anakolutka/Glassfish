Para Deletar Instâncias do GlassFish Server Clusterizadas

Cuidado:

Se estiver utilizando um cluster Java Message Service (JMS) com um broker mestre, não delete a instância que está associada ao broker mestre. Se esta instância precisar ser deletada, use o subcomando change-master-broker(1) para designar o broker mestre a outra instância.

  1. Na árvore de navegação, expanda o nó Clusters.

  2. No nó Clusters expandidos, selecione o nó do cluster.

    A página Informações Gerais do cluster será aberta.

  3. Na página Informações Gerais, clique na guia Instâncias.

    A página Instâncias de Servidor Clusterizado será aberta com informações sobre o cluster.

  4. Para cada instância que você estiver deletando, marque a caixa de seleção próxima à instância na tabela Instâncias do Servidor.

  5. Clique em Deletar.

    Será solicitado que você confirme se deseja deletar as instâncias.

  6. Confirme se deseja deletar as instâncias.

    As instâncias são deletadas e removidas da tabela de instâncias da página.
