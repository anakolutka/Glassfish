Nuovo servizio contesto

Utilizzare la pagina Nuovo servizio contesto per creare una risorsa servizio contesto.

La pagina Nuovo servizio contesto contiene le opzioni riportate di seguito.

Nome JNDI

Il nome univoco che identifica la risorsa.

Informazioni contesto

Il contesto della risorsa: Classloader, JNDI, Sicurezza o Area di lavoro.

Descrizione

La descrizione di questa risorsa.

Stato

Lo stato della risorsa. La risorsa può essere abilitata o disabilitata. Lo stato predefinito è Abilitato.

Proprietà aggiuntive

Proprietà aggiuntive per la risorsa. GlassFish Server non definisce alcuna proprietà aggiuntiva per le risorse dei servizi contesto.
