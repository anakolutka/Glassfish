Para alterar a Senha do Administrador

  1. Na árvore de navegação, selecione o nó Servidor de Aplicativos.

    A página Informações Gerais é exibida.

  2. Selecione a guia Senha do Administrador.

    A página exibr a ID do usuário e o nome do grupo da conta do administrador atual. Por padrão, a ID do usuário é anônima e o nome do grupo é asadmin .

  3. No campo Nova Senha, digite a nova senha da conta do administrador atual.
  4. No campo Confirmar Nova Senha, torne a digitar a nova senha.
  5. Clique em Salvar.
