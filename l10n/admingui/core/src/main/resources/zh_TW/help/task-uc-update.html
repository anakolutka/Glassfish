更新已安裝的元件

更新已安裝的元件包括下列作業：

  • 僅將已修改的檔案從網路儲存庫傳送至系統

  • 僅在系統上更新 Enterprise Server 安裝影像內已修改的檔案

  • 套用更新元件之間新的相依性 (如果有)

  1. 在瀏覽樹狀結構中，選取 [更新工具] 節點。

    已安裝的元件頁面會隨即顯示。

  2. 在 [已安裝的元件] 頁面上，按一下 [可用的更新] 標籤。

    可用的更新頁面會隨即顯示。

  3. 在可用更新表格中，選取您要更新的元件。

    如果沒有可用的更新，表格會是空的。

    • 若要選取個別元件，請選取元件名稱旁的核取方塊。
    • 若要選取所有元件，請按一下表格標頭中的 [全選] 圖示。
    • 若要取消選取所有元件，請按一下表格標頭中的 [全不選] 圖示。
  4. 按一下 [安裝]。
  5. 接受授權合約。

    Enterprise Server 會確認安裝已完成。再從可用更新表格中移除元件。

  6. 重新啟動網域管理伺服器。

    如需詳細說明，請參閱下列文件：

    • 「Sun GlassFish Enterprise Server v3 Prelude Administration Guide」中的「To Stop a Domain (or Server)」

    • 「Sun GlassFish Enterprise Server v3 Prelude Administration Guide」中的「To Start a Domain (or Server)」
