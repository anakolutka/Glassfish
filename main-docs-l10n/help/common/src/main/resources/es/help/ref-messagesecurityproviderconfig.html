Configuraciones de proveedor

Utilice la página Configuraciones de proveedor para configurar el proveedor de seguridad de mensajes en una capa de autenticación SOAP o HttpServlet.

Por defecto, hay cuatro proveedores configurados en la capa de autenticación SOAP.

Para cada configuración de proveedor, se proporciona la siguiente información.

ID de Proveedor

Nombre de la configuración del proveedor.

Tipo de proveedor

Tipo de proveedor, que puede ser cualquiera de los siguientes:

client

Especifica que el proveedor es el proveedor de autenticación de clientes.

server

Especifica que el proveedor es el proveedor de autenticación del servidor.

client-server

Especifica que el proveedor es tanto el proveedor de autenticación del servidor como del cliente.

Proveedor por defecto

Si se define en true, el proveedor será el proveedor por defecto.

Nombre de clase

Nombre de la clase de implantación de Java del proveedor.

La tabla Configuraciones de proveedor también contiene las siguientes opciones.

Nuevo

Botón para crear una nueva configuración de proveedor.

Suprimir

Botón para suprimir una o más configuraciones de proveedor seleccionadas.
