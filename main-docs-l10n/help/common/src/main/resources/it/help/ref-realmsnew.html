Nuovo realm

Utilizzare la pagina Nuovo realm per creare un realm.

La pagina Nuovo realm contiene le opzioni riportate di seguito.

Nome configurazione

Il nome della configurazione a cui si applicano le impostazioni presenti in questa pagina. Questo campo è di sola lettura.

Nome

Il nome del nuovo realm.

Nome classe

Il nome di classe per il nuovo realm.

Le scelte valide vengono mostrate nella tabella seguente.

Nome realm Nome classe
file com.sun.enterprise.security.auth.realm.file.FileRealm
certificate com.sun.enterprise.security.auth.realm.certificate.CertificateRealm
ldap com.sun.enterprise.security.auth.realm.ldap.LDAPRealm
JDBC com.sun.enterprise.security.auth.realm.jdbc.JDBCRealm
solaris com.sun.enterprise.security.auth.realm.solaris.SolarisRealm
PAM com.sun.enterprise.security.auth.realm.solaris.PamRealm

Per specificare un realm personalizzato, selezionare il pulsante di scelta inferiore e digitare il nome classe per il realm personalizzato da creare.

Proprietà aggiuntive

Proprietà aggiuntive per il realm. Le proprietà valide dipendono dal tipo di realm.

  • Proprietà specifiche della classe FileRealm

  • Proprietà specifiche della classe CertificateRealm

  • Proprietà specifiche della classe JDBCRealm

  • Proprietà specifiche della classe LDAPRealm

  • Proprietà specifiche della classe SolarisRealm

  • Proprietà specifiche della classe PamRealm
