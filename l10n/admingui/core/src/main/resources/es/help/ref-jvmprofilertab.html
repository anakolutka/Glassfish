Página Configuración de analizador JVM

Una herramienta de tipo analizador genera datos que son utilizados para analizar el rendimiento e identificar atascos potenciales.

Especifique la configuración del analizador. La información que especifique dependerá del analizador que utilice. Para obtener ejemplos e instrucciones, consulte el Capítulo 3, Debugging Applications de Sun GlassFish Enterprise Server v3 Prelude Developer’s Guide.
