Page des tâches courantes

La page Tâches courantes fournit des raccourcis vers les tâches Console d’administration.

Utilisez la page Tâches courantes pour effectuer les tâches suivantes :

  • Enregistrer Sun GlassFish Enterprise Server avec Sun Connection

  • Afficher la page Support pour Sun GlassFish Enterprise Server

  • Déployer une application Web

  • Créer un pool de connexions JDBC

La page Tâches courantes offre également des liens vers la documentation Enterprise Server.
