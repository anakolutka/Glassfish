Clusters

La página Clusters muestra una lista de los clusters del dominio. Para cada cluster, se muestra la información siguiente:

Nombre

Nombre que se asignó al cluster en el momento de su creación. Al hacer clic en el nombre se abrirá la página Información general del cluster.

Configuración

Configuración que se seleccionó para el cluster en el momento de su creación. Al hacer clic en la configuración se abrirá la página Configuraciones del cluster.

Instancias

Nombre y estado de cada instancia del cluster. Al hacer clic en el nombre de la instancia se abrirá la página Información general de la instancia.

La tabla Clusters también contiene las opciones siguientes.

Nuevo

Botón para crear un cluster.

Suprimir

Botón para suprimir un cluster.

Iniciar cluster

Botón para iniciar un cluster.

Parar cluster

Botón para parar un cluster.
