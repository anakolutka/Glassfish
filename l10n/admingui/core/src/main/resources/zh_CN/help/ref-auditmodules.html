审计模块

审计指的是 Enterprise Server 对错误或安全性破坏等重要事件进行记录。Enterprise Server 在日志文件中记录所有验证事件。完整的访问日志提供了 Enterprise Server 访问事件的顺序线索。

“审计模块”页面显示了可用的审计模块列表。对于每个审计模块，将提供以下信息：

名称

审计模块名称。

类名

实现此模块的类的全限定名称。默认审计模块的类名为 com.sun.enterprise.security.Audit。

相关的 asadmin 命令

  • create-audit-module(1)

  • list-audit-modules(1)

  • delete-audit-module(1)
