Crear un listener IIOP

  1. En el componente de árbol, amplíe el nodo Configuración.

  2. Seleccione la instancia que va a configurar:

    • Para configurar una instancia determinada, seleccione el nodo de configuración de la instancia. Por ejemplo, para la instancia por defecto, server, seleccione el nodo server-config.

    • Para definir la configuración por defecto para instancias futuras que utilicen una copia de default-config, seleccione el nodo default-config.

  3. En el nodo Configuración, amplíe el nodo ORB.

  4. Seleccione Listeners IIOP.

    Se abrirá la página Listeners IIOP.

  5. Haga clic en Nuevo.

  6. En el campo Nombre, escriba un nombre para identificar el listener.

  7. En el campo Dirección de red, introduzca la dirección de red del listener.

    Puede introducir una dirección IP o un nombre de host que se pueda resolver mediante DNS.

  8. En el campo Puerto del listener, escriba el número del puerto en el que recibirá el listener IIOP.

  9. Active la casilla de control Listener activado para activar las conexiones entrantes ORB IIOP para GlassFish Server.

    Esta opción se encuentra activada por defecto.

  10. Active la casilla de control Seguridad activada para activar SSL para el listener IIOP.

    Esta opción se activada por defecto.

  11. En la sección Propiedades Adicionales, especifique las propiedades adicionales.

    GlassFish Server no define ninguna propiedad adicional para los listener IIOP.

    Para agregar una propiedad, haga clic en el botón Agregar Propiedad. En la fila en blanco que aparece, introduzca el nombre de propiedad en el campo Nombre y escriba el valor de propiedad en el campo Valor.

  12. Haga clic en Aceptar.

  13. Si activa la casilla de control Seguridad Activada, siga los pasos descritos en Editar configuración SSL para un listener IIOP seguro.
