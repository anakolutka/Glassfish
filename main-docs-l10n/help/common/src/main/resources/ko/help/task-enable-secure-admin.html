보안 관리를 사용으로 설정하는 방법

  1. 탐색 트리에서 서버(관리 서버) 노드를 선택합니다.

    [일반 정보] 페이지가 열립니다.

  2. [일반 정보] 페이지에서 [보안 관리] 단추를 누릅니다.

    [보안 관리] 페이지가 열립니다.

  3. GlassFish Server 설치가 사용자 정의 관리 또는 인스턴스 별칭 값을 사용하는 경우, 필요에 따라 [관리 별칭] 및 [인스턴스 별칭] 필드에 표시된 기본값을 업데이트합니다.

  4. [보안 관리 사용]을 누릅니다.

    보안 관리를 사용으로 설정하면 DAS가 즉시 재시작됨을 알리는 메시지가 나타납니다.

  5. [확인]을 누릅니다.

    보안 관리가 사용으로 설정되고 서버 재시작 페이지가 열립니다.
