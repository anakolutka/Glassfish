Contraseña de administrador

Consulte la página Contraseña de administrador para cambiar la contraseña de la cuenta de administrador actual.

La página Contraseña de administrador contiene las siguientes opciones.

Id. de usuario

Id. de usuario del administrador actual. Éste es un campo de sólo lectura.

Lista de grupos

Grupo al que pertenece el administrador. Éste es un campo de sólo lectura. El valor es asadmin.

Contraseña nueva

Contraseña nueva del administrador.

Contraseña nueva (confirmar)

Contraseña nueva del administrador.

Tarea relacionada

  • Cambio de contraseña del administrador

Comando asadmin relacionado

  • change-admin-password(1)
