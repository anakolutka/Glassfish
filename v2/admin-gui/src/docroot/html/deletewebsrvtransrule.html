To Remove a Transformation Rule

  1. In the tree view, expand the Web Services node.
  2. Select the web service for which you want to remove a transformation rule.
  3. Click the Transformation tab.

    A list of transformation rules available for this web service end point is displayed.

  4. Select the transformation rule by selecting the check box.
  5. Click the Remove button.

    This removes the transformation rule from the list. If this transformation rule is applied to a web service endpoint, it automatically gets disabled. However, the XSLT file remains in the file path location. This XSLT file can be used by any other web service endpoint.
