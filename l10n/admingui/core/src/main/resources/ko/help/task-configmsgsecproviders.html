메시지 보안 공급자 사용을 위해 Enterprise Server 기능을 구성하는 방법

  1. 사용자 이름 토큰을 사용할 경우 필요하면 사용자 데이터베이스를 구성합니다.

    사용자 이름/비밀번호 토큰을 사용할 경우에는 먼저 영역과 해당 영역의 사용자 데이터베이스를 적절하게 구성합니다.

    사용자 데이터베이스 구성은 영역을 편집하는 방법을 참조하십시오.

  2. 필요한 경우 인증서 및 개인 키를 관리합니다.
다음 순서

Enterprise Server의 기능을 메시지 보안 공급자가 사용할 수 있도록 구성하면 Enterprise Server와 함께 설치된 공급자를 메시지 보안 구성을 편집하는 방법의 설명대로 활성화할 수 있습니다.
