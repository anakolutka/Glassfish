Para eliminar aplicaciones de un cluster

Al eliminar una aplicación, sólo se elimina del destino actual. Cuando una aplicación se elimina de un destino, la aplicación sigue apareciendo en la página Aplicaciones que muestra todas las aplicaciones desplegadas en GlassFish Server. La aplicación también seguirá desplegada en otros destinos en los que se hubiera desplegado.

  1. En el árbol de navegación, amplíe el nodo Clusters.

  2. En el nodo Clusters ampliado, seleccione el nodo para el cluster.

    Se abre la página Información General para el cluster.

  3. En la página Información General, haga clic en el separador Aplicaciones.

    Se abre la página Aplicaciones para el cluster.

  4. Para cada aplicación que vaya a eliminar, active la casilla de control que aparece junto a ella en la tabla Aplicaciones Desplegadas.

  5. Haga clic en Eliminar.

    Se le pedirá que confirme que desea eliminar las aplicaciones.

  6. Confirme que desea eliminar las aplicaciones.

    Las aplicaciones se eliminan de la tabla de aplicaciones desplegadas de la página.
