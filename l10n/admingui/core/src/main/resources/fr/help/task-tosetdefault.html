Définir le domaine par défaut

Le domaine par défaut est le domaine utilisé par le Enterprise Server pour l'authentification et l'autorisation si aucun domaine n'est spécifié par le descripteur de déploiement de l'application.

  1. Dans l'arborescence de navigation, développez le nœud Configuration.
  2. Si la configuration du domaine prend en charge des clusters, sélectionnez la configuration à modifier.

    Exemples : domaines créés avec le profil Cluster ou entreprise .

  3. Sélectionnez le nœud Sécurité.

    La page Sécurité s'ouvre.

  4. Dans le champ Domaine par défaut, choisissez le domaine de votre choix dans la liste déroulante.
  5. Cliquez sur Enregistrer pour sauvegarder les modifications.
  6. Redémarrez le serveur si « Redémarrage requis » s'affiche dans la console.
