관리 비밀번호를 변경하는 방법

관리 비밀번호를 암호화하면 응용 프로그램의 보안이 향상됩니다. 보안 절차에 대한 자세한 내용은 Sun GlassFish Enterprise Server v3 Prelude Administration Guide의 6 장, Administering System Security.

  1. 탐색 트리에서 구성 노드를 선택합니다.
  2. 클러스터를 지원하도록 도메인이 구성된 경우 변경할 구성을 선택합니다.

    cluster 프로필 또는 enterprise 프로필을 통해 생성된 도메인을 예로 듭니다.

  3. 보안 노드를 확장합니다.
  4. 영역 노드를 확장합니다.
  5. admin-realm 노드를 선택합니다.
  6. 영역 편집 페이지에서 사용자 관리 버튼을 누릅니다.
  7. 이름이 admin인 사용자를 선택합니다.
  8. 새 비밀번호를 입력하고 비밀번호를 확인합니다.
  9. 저장을 누릅니다.
참조

  • 보안 설정을 구성하는 방법

  • 관리 도구에 액세스를 부여하는 방법
