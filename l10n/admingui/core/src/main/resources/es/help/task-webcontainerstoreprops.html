Para configurar las propiedades del almacén

  1. En el árbol de navegación, expanda el nodo Configuración.
  2. En el nodo Configuración, seleccione el nodo Contenedor web.

    Se abrirá la página Contenedor web.

  3. En la página Contenedor web, haga clic en la ficha Propiedades del almacén.
  4. En el Intervalo de cierre, introduzca el número de segundos que deben transcurrir antes de que los datos de sesión inactiva se eliminen del almacén.
  5. En el campo Directorio, introduzca la ruta al archivo en el que se almacenarán los datos de sesión.

    This is needed only if you have selected File as the persistence type.

  6. (Opcional) En la sección Propiedades adicionales, especifique las propiedades adicionales.
  7. Haga clic en "Guardar".
