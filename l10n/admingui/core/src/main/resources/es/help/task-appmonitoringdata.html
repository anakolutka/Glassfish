Visualización de datos de supervisión de la aplicación

Antes de empezar

Antes de poder ver datos de supervisión, es necesario que configure la supervisión. Consulte Cómo configurar el servicio de supervisión para obtener más información.

  1. En el árbol de navegación, seleccione el nodo Application Server.

    Aparecerá la página de Información general.

  2. En la página Información general, haga clic en la ficha Supervisar.

    Se abrirá la página Supervisión de aplicación.

  3. En la lista desplegable Aplicación, seleccione una aplicación para visualizar los datos de supervisión de dicha aplicación.
  4. Haga clic en el botón Estadísticas para recuperar estadísticas de supervisión.

    Las estadísticas de supervisión aparecen en la tabla.

  5. (Opcional) Haga clic en el botón Actualizar para actualizar los datos de supervisión mostrados.
