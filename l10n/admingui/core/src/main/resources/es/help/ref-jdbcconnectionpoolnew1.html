Nuevo conjunto de conexiones JDBC (paso 1 de 2)

Nombre

El nombre del conjunto de conexiones.

Tipo de recurso

Los tipos de recursos disponibles son javax.sql.XADataSource (transacciones globales), java.sql.ConnectionPoolDataSource (transacciones locales, posibles mejoras de rendimiento) y javax.sql.DataSource (sólo transacciones locales).

Proveedor de la base de datos

Nombre del proveedor del producto de la base de datos.
