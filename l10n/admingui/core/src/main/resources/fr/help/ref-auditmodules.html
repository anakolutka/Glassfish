Modules d'audit

Les modules d'audit contrôlent la manière dont le Enterprise Server consigne les événements importants, comme les erreurs et les brèches de sécurité. Le Enterprise Server consigne tous les événements d'authentification dans des fichiers journaux. Un journal d'accès fournit un suivi séquentiel des événements d'accès du Enterprise Server.

La page Modules d'audit affiche la liste des modules d'audit disponibles. Pour chaque module d'audit, les informations suivantes sont fournies.

Nom

Nom du module d'audit.

Nom de classe

Nom complet de la classe qui implémente ce module. Le nom de classe du module d'audit par défaut est com.sun.enterprise.security.Audit.
