So werden anwendungsspezifische Ressourcen angezeigt

  1. Wählen Sie in der Navigationsbaumstruktur den Anwendungsknoten.

    Die Seite "Anwendungen" wird geöffnet.

  2. Wählen Sie eine Anwendung, für die Sie die Anwendungsgeltungsbereichsressourcen anzeigen möchten.

    Der Anwendungsname kann eine optionale Versions-ID enthalten, die dem Namen folgt und von diesem durch einen Doppelpunkt (:) getrennt ist. Weitere Informationen zu Modul- und Anwendungsversionen finden Sie unter Module and Application Versions im Oracle GlassFish Server Application Deployment Guide.

    Die Seite für die Anwendung wird geöffnet.

  3. Wählen Sie die Registerkarte "Ressourcen".

    Die Seite "Anwendungsspezifische Ressourcen" wird geöffnet, und die aktuell für die Anwendung angegebenen Ressourcen werden in einer Tabelle mit den folgenden Spalten angezeigt:

    Anwendungsname

    Der Name der Anwendung.

    Modulname

    Der Name des Moduls innerhalb der Anwendung.

    Ressourcenname

    Der Name der anwendungsspezifischen Ressource für die Anwendung oder das Modul.

    Ressourcentyp

    Der Ressourcentyp, Beispiel: JDBC-Connection Pools oder Connector-Ressourcen.
