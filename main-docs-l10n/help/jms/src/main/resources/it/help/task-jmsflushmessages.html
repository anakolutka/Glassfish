Per rimuovere i messaggi da una destinazione fisica JMS

  1. Espandere il nodo Configurazione nell'albero di navigazione.

  2. Nel nodo Configurazione espandere il nodo Java Message Service.

  3. Nel nodo Java Message Service selezionare il nodo Destinazioni fisiche.

    Viene visualizzata la pagina Destinazioni fisiche.

  4. Nella pagina Destinazioni fisiche selezionare la casella di controllo in corrispondenza del nome della destinazione da cui rimuovere i messaggi.

  5. Fare clic sul pulsante Elimina tutti i messaggi.
