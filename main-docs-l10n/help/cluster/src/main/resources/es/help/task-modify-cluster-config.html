Para modificar la configuración de un cluster

  1. En el árbol de navegación, amplíe el nodo Clusters.

  2. Seleccione el nodo para el cluster.

  3. Haga clic en el nombre de la configuración.

    Se abrirá la página Configuración para el cluster.

  4. En la página Configuración, seleccione el componente de la configuración que se va a modificar.

    Se abrirá la página para editar el componente seleccionado de la configuración.

  5. En la página para editar el componente seleccionado, realice los cambios oportunos y guárdelos.

    Para obtener más información, consulte la ayuda en pantalla de la página para editar el componente seleccionado.
