特定于 JDBCRealm 类的属性

要保护您的 Web 页或 Web 应用程序，您可以设置安全性，以便只有注册用户才能访问它们。这也称为验证工具。这种类型的领域涉及在数据库中存储用户证书。Enterprise Server 使用数据库信息和配置文件内启用的 JDBC 领域选项。JDBC 领域需要以下属性：

JAAS 上下文

用于此领域的登录模块的标识符。唯一的有效值是 jdbcRealm。

J.N.D.I.

此领域的 J.N.D.I. 名称。默认值为 jdbc/security。

用户表

包含该领域的授权用户列表的表。默认值为 usertable。

用户名列

用户表内包含用户列表的列的名称。默认值为 userid。

密码列

用户表内包含各个用户密码的列的名称。默认值为 password。

组表

默认值为 grouptable。

组名列

默认值为 groupid。
