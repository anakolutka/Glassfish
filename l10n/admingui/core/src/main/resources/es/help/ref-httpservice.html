Servicio HTTP

La página Servicio HTTP contiene las siguientes opciones:

Registro de acceso

Si se selecciona esta opción, se habilitará el registro de acceso de todos los subelementos de servidores virtuales que no tengan especificada esta propiedad. Si no selecciona esta opción, se deshabilitará el registro de acceso de todos los subelementos de servidores virtuales que no tengan especificada esta propiedad.

Tamaño del búfer de registro de acceso

Tamaño, en bytes, del búfer de registro de acceso, o un valor inferior o igual a cero para los registros de acceso sin búfer. El valor predeterminado es 4096.

Intervalo de escritura de registro de acceso

El número de segundos antes de que el registro se escriba en el disco. La escritura del registro de acceso tiene lugar cuando el búfer está lleno o cuando caduca el intervalo. Si el valor es igual o menor que cero, el búfer siempre se escribirá aunque no esté lleno. Esto significa que, cada vez que se accede al servidor, el mensaje de registro se almacena directamente en el archivo.

Propiedades adicionales

Propiedades adicionales del Servicio HTTP.

Están disponibles las propiedades siguientes:

connectionTimeout

Especifica el número de milisegundos que deben esperar las escuchas HTTP, después de aceptar una conexión, para que se presente la línea URI de solicitud.

El valor predeterminado es 12.000 (12 segundos).

maxKeepAliveRequests

Especifica el número máximo de solicitudes HTTP que se pueden canalizar hasta que el servidor cierre la conexión. Establezca esta propiedad en 1 para deshabilitar el mantenimiento HTTP/1.0, así como la canalización y el mantenimiento HTTP/1.1.

El valor predeterminado es 1.000.

traceEnabled

Si se establece en "true", permite el método TRACE. Establezca esta propiedad en "false" para que Servidor empresarial sea menos susceptible a ataques de secuencia de comandos entre sitios.

Para obtener más información sobre el método TRACE, consulte RFC 2616: protocolo de transferencia de hipertexto -- HTTP/1.1.

El valor predeterminado es Verdadero.
