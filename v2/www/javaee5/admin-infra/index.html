Administrative Infrastructure comprises of the framework that goes into configuration and monitoring of the distributed server components, interaction of various administrative tools with administrative tools, a public API to assist independent administrative client development, life cycle management of the server side entities like domains and servers, interaction with operating platform to leverage some of the system services and provision to extend the built in management functionality to exploit self healing capabilities.There are various JSRs that the infrastructure is based on, most notably: JMX (JSR 003) and JSR77.
