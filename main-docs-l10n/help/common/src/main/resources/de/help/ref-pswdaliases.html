Kennwortaliasnamen

Verwenden Sie die Seite "Kennwortaliasnamen", um Kennwortaliasnamen zu erstellen und zu verwalten.

Auf der Seite "Kennwortaliasnamen" wird eine Liste der Kennwortaliasnamen in der Domain angezeigt. Für jeden Alias werden folgende Informationen angezeigt:

Name

Der Name, der dem Kennwortalias bei seiner Erstellung zugewiesen wurde. Wenn Sie auf den Namen klicken, wird die Seite Kennwortalias bearbeiten für den Alias geöffnet.

Die Tabelle "Kennwortaliasnamen" enthält darüber hinaus die folgenden Optionen.

Neu

Schaltfläche zum Erstellen eines Kennwortalias.

Löschen

Schaltfläche zum Löschen von mindestens einem Kennwortaliasnamen.
