Per configurare le impostazione del profiler JVM

Uno strumento profiler genera dati utilizzabili per analizzare le prestazioni e identificare potenziali punti critici.

  1. Selezionare il nodo Configurazione nell'albero di navigazione.

  2. Selezionare la configurazione da modificare.

  3. Nel nodo Configurazione selezionare il nodo Impostazioni JVM.

    Viene visualizzata la pagina Impostazioni generali JVM.

  4. Nella pagina Impostazioni generali JVM fare clic sulla scheda Profiler.

    Viene visualizzata la pagina Impostazioni profiler JVM.

  5. Specificare le impostazioni del profiler.

    Le informazioni specificate dipendono dal prodotto profiler in uso. Per esempi e istruzioni, vedere il Capitolo 3, Debugging Applications, in Oracle GlassFish Server Application Development Guide.

  6. Fare clic su Salva.

  7. Riavviare il server.
