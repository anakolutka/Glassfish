To Edit a Timer Rule

  1. Do one of the following:
    • For the developer profile, in the tree component, select the Configuration node.
    • For the cluster or enterprise profile, in the tree component select the Configurations node, then select the configuration to edit.
  2. Select the Management Rules node.
  3. Select the management rule to edit.

    The Edit Timer Event page appears.

  4. Check Enabled to activate this rule.
  5. Change the rule description, if needed.
  6. Select the Record Event checkbox to log the occurrence of the specified event.
  7. Change the log level for this management rule, if needed.
  8. Specify the date from when this rule is enabled.
  9. Specify the pattern to search for.

    The default pattern is mm/dd/yyyy hh:mm:ss. The pattern must match the format in which you have specified the date.

  10. Specify the time period in milliseconds at which notifications will be sent to the action MBean.
  11. Specify the number of times the pattern occurs in the notification.
  12. Specify the message that will be written to the log when the pattern occurs.
  13. Enter a description for the event associated with this management rule.
  14. Specify the action you want performed when the monitored data meets the condition specified in the event type.
  15. Click Save.
