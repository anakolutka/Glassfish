So konfigurieren Sie die JVM-Profilereinstellungen

Ein Profiler erzeugt Daten, die zur Analyse der Leistung sowie zur Ermittlung von potenziellen Engpässen verwendet werden können.

  1. Wählen Sie in der Navigationsstruktur den Anwendungsserverknoten.
  2. Klicken Sie auf der Seite "Allgemeine Informationen" auf die Registerkarte "JVM-Einstellungen".
  3. Klicken Sie auf der Seite "Allgemeine JVM-Einstellungen" auf die Registerkarte "Profiler".

    Die Seite "JVM-Profilereinstellungen" wird geöffnet.

  4. Geben Sie die Profilereinstellungen an.

    Die eingegebenen Informationen richten sich danach, welchen Profiler Sie verwenden. Beispiele und Anweisungen finden Sie in Kapitel 3, Debugging Applications in Sun GlassFish Enterprise Server v3 Prelude Developer’s Guide.

  5. Klicken Sie auf "Speichern".
  6. Starten Sie den Server neu.
