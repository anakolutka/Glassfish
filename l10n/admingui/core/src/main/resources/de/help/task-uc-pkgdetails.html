So zeigen Sie detaillierte Informationen zu einer Komponente an

Update-Tool liefert detaillierte Informationen zu installierten Komponenten , verfügbaren Updates für installierte Komponenten und Add-On-Komponenten.

  1. Wählen Sie in der Navigationsstruktur den Knoten "Update-Tool" aus.

    Die Seite "Verfügbare Add-Ons" wird angezeigt.

  2. Klicken Sie auf die Registerkarte für die Seite, auf der die Komponente aufgeführt ist.

    Die Komponente wird auf einer der folgenden Seiten aufgeführt:

    • Verfügbare Add-Ons

    • Verfügbare Updates

    • Installierte Komponenten

  3. Wählen Sie in der Tabelle der Komponenten die Komponente aus.

    Die Seite Komponentendetails für die ausgewählte Komponente wird angezeigt.
