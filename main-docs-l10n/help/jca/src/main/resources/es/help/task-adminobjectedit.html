Para editar un recurso de objeto administrado

  1. En el árbol de navegación, amplíe el nodo Recursos.

  2. Amplíe el nodo Conectores.

  3. En el nodo Conectores, seleccione el nodo Recursos de objeto administrado.

    Se abrirá la página Recursos de objeto administrado

  4. En la página Recursos de objeto administrado, haga clic en el nombre del recurso de objeto administrado que se está editando.

    Se abrirá la página Editar recurso de objeto administrado.

  5. En la lista desplegable Adaptador de recursos, seleccione el adaptador de recursos para el recurso de objeto administrado.

  6. En la lista desplegable Tipo de recurso, seleccione el tipo Java para el recurso de objeto administrado.

  7. En la lista desplegable Nombre de clase, seleccione el nombre de clase de implantación asociado al tipo de recurso.

  8. En el campo Descripción, introduzca una descripción del recurso de objeto administrado.

  9. Active la casilla de control Estado activado para activar el recurso de objeto administrado.

    Esta opción se encuentra activada por defecto.

  10. En la tabla Propiedades adicionales, especifique las propiedades adicionales para el recurso de objeto administrado.

    Las propiedades disponibles dependen del adaptador de recursos para el pool de conexiones de conector.

    • Para agregar una propiedad, haga clic en el botón Agregar Propiedad. En la fila en blanco que aparece, introduzca el nombre de propiedad en el campo Nombre y escriba el valor de propiedad en el campo Valor.

    • Para modificar una propiedad, edite el campo Valor.

    • Para suprimir una propiedad, active la casilla de control a la izquierda del campo Nombre de la propiedad que está suprimiendo y, a continuación, haga clic en el botón Suprimir Propiedades.

  11. En el separador Destino, haga clic en Gestionar destinos para editar los destinos en los que el objeto administrado está desplegado.

    Vuelva a la página Editar recurso de objeto administrado haciendo clic en el separador General.

  12. Haga clic en Guardar.

    Sugerencia:

    Para volver a la configuración por defecto, haga clic en el botón Cargar Valores por Defecto y, a continuación, en Guardar.
