常规信息

使用“常规信息”页面可以验证应用服务器设置以及查看 Java 虚拟机数据。

“常规信息”页面包含以下信息：

“停止”按钮

单击“停止”按钮以停止 Enterprise Server。

“重新启动”按钮

单击“重新启动”按钮以重新启动 Enterprise Server。

“查看日志文件”按钮

单击“查看日志文件”按钮以查看服务器日志。

名称

服务器名称。

JVM 报告

如果单击“JVM 报告”，则会打开一个单独的窗口，此窗口显示有关 Java 虚拟机的报告，包括摘要报告、内存管理和垃圾收集报告、类装入报告和当前线程转储。

配置目录

主机上包含配置文件（例如 domain.xml）的目录。

安装版本

产品版本标识符。

调试

标识是否已启用调试。如果已启用调试，则还会显示端口号。

HTTP 端口

当前配置的用于 HTTP 请求和响应的端口。

IIOP 端口

当前配置的用于 IIOP 请求和响应的端口。
