To edit JDBC resource targets

  1. Expand the Resources, JDBC, and then the JDBC Resources node.
  2. Select the JDBC resource that you want to edit.
  3. Open the Targets tab.

    In the tab, you can enable or disable the existing targets.

  4. If you want to assign a new target to the resource, click Manage Targets. In the Manage Targets screen, you can add or remove targets for the resource.
