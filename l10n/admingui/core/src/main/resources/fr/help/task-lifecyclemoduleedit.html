Pour modifier un module de cycle de vie

  1. Dans l'arborescence de navigation, sélectionnez le noeud Modules de cycle de vie.

    La page Modules de cycle de vie s'affiche.

  2. Sur la page Modules de cycle de vie, cliquez sur le nom du module de cycle de vie que vous modifiez.

    La page Modification du module de cycle de vie s'ouvre.

  3. Dans le champ Nom de la classe, saisissez le nom complet du fichier de classe du module de cycle de vie.

    La classe doit implémenter l'interface com.sun.appserv.server.LifecycleListener.

  4. Dans le champ Classpath, saisissez le classpath du module de cycle de vie.

    Le classpath indique là où se trouve le module de cycle de vie. La valeur par défaut est domain-dir/applications. Si le module est déjà dans ce répertoire (c'est-à-dire dans le classpath du serveur), ce champ peut être laissé vierge.

  5. Dans le champ Ordre de chargement, saisissez une valeur indiquant l'ordre dans lequel ce module de cycle de vie doit être chargé au démarrage.

    Les modules aux valeurs d'ordre de chargement à petit nombre sont chargés plus tôt. La valeur peut aller de 101 à la valeur MAXINT du système d'exploitation. Les valeurs de 1 à 100 sont réservées.

  6. Cochez la case État activé pour activer le module de cycle de vie.

    Cette option est activée par défaut.

  7. Cochez la case Empêcher le démarrage de l'instance en cas d'échec du chargement pour indiquer que le serveur doit être arrêté en cas d'échec de chargement du module de cycle de vie.

    Cette option est désactivée par défaut.

  8. Dans la liste déroulante Serveurs virtuels, sélectionnez le ou les serveur(s) virtuel(s) à associer au module de cycle de vie.

    Par défaut, aucun serveur virtuel n'est sélectionné.

  9. Dans le champ Description, saisissez une description du module de cycle de vie.
  10. (Facultatif) Dans la section Autres propriétés, spécifiez d'autres propriétés.

    Le Enterprise Server ne définit aucune autre propriété pour les modules de cycle de vie.

    • Pour ajouter une propriété, cliquez sur le bouton Ajouter propriété. Sur la ligne vide qui apparaît, saisissez le nom de la propriété dans le champ Nom et la valeur de la propriété dans le champ Valeur.
    • Pour modifier une propriété, modifiez le champ Valeur de cette propriété.
    • Pour supprimer une propriété, cochez la case située à gauche du champ Nom de la propriété que vous supprimez, puis cliquez sur le bouton Supprimer propriétés.
  11. Cliquez sur Enregistrer.

    Astuce - Pour revenir aux paramètres par défaut, cliquez sur le bouton Charger valeurs par défaut, puis sur Enregistrer.
