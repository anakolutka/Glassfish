Activación de la autenticación mutua

Con la autenticación mutua se activa la autenticación del servidor y del cliente. Para probar la autenticación mutua, debe existir un cliente con un certificado válido. Para obtener más información sobre la autenticación mutua, consulte la sección Seguridad del Tutorial de Java EE 5.

La activación de la autenticación mutua incluye las tareas siguientes:

  • Cómo activar la autenticación mutua en todas las aplicaciones

  • Cómo activar la autenticación mutua en una aplicación específica
