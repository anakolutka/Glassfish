Per abilitare o disabilitare il log di audit

  1. Espandere il nodo Configurazione nell'albero di navigazione.

  2. Selezionare l'istanza da configurare:

    • Per configurare un'istanza particolare, espandere il nodo di configurazione specifico dell'istanza. Ad esempio, per l'istanza predefinita server, espandere il nodo server-config.

    • Per configurare le impostazioni predefinite per tutte le istanze, espandere il nodo default-config.

  3. Nel nodo Configurazione selezionare il nodo Sicurezza.

    Viene visualizzata la pagina Sicurezza.

  4. Nella pagina Sicurezza selezionare la casella di controllo Abilitato dell'opzione Log di audit per abilitare la funzione di log dell'audit.

    Per attivare il log di audit è inoltre necessario impostare la proprietà auditOn per il modulo di audit attivo su true.

  5. Fare clic su Salva.
