  • 建置 Web 應用程式

  • 建置企業應用程式

  • 建置應用程式從屬端

  • To Deploy a Connector Module

  • 建置 EJB JAR

  • 組態監視服務

  • 檢視應用程式監視資料

  • 檢視伺服器監視資料

  • 檢視資源監視資料

  • 檢視已安裝的元件

  • 安裝附加元件

  • 檢視元件的可用更新

  • 建立 JDBC 連線集區

「常用工作」頁面還提供部分 GlassFish Server 文件的連結。
