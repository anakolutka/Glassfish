Nuovo listener IIOP

Utilizzare la pagina Nuovo listener IIOP per creare un nuovo listener IIOP.

La pagina Nuovo listener IIOP contiene le opzioni riportate di seguito.

Nome configurazione

Il nome della configurazione a cui si applicano le impostazioni presenti in questa pagina. Questo campo è di sola lettura.

Nome

Il nome univoco che identifica il listener IIOP.

Indirizzo di rete

L'indirizzo di rete del listener IIOP. L'indirizzo di rete può essere un indirizzo IP o un nome host risolvibile da DNS.

Porta listener

Il numero della porta di ascolto del listener IIOP.

Listener

Lo stato del listener. Se la casella di controllo Abilitato è selezionata, le connessioni IIOP ORB in entrata per GlassFish Server sono abilitate. Questa opzione è abilitata per impostazione predefinita.

Sicurezza

Se la casella di controllo Abilitato è selezionata, SSL è abilitato per il listener IIOP. Questa opzione è disabilitata per impostazione predefinita.

Proprietà aggiuntive

Proprietà aggiuntive per il listener IIOP. GlassFish Server non definisce alcuna proprietà aggiuntiva per i listener IIOP.
