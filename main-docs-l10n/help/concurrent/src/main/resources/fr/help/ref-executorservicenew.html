Nouveau service exécuteur géré

Utilisez la page Nouveau service exécuteur géré pour créer une ressource de service exécuteur géré.

Elle contient les options ci-après.

Nom JNDI

Nom JNDI unique identifiant la ressource. Par convention, le nom commence par un indicateur de type de ressource et une barre oblique, comme concurrent/, jdbc/ ou jms/.

Informations de contexte

Contextes à propager vers les threads : Chargeur de classe, JNDI, Sécurité ou Zone de travail. La propagation de contexte peut être activée ou désactivée. Si elle est activée, les contextes sélectionnés sont propagés. Si elle est désactivée, aucun des contextes n'est propagé, même s'ils sont sélectionnés. La valeur par défaut est Activé, avec les quatre contextes sélectionnés.

Statut

Statut de la ressource. La ressource peut être activée ou désactivée. La valeur par défaut est Activé.

Priorité des threads

Priorité à affecter aux threads créés. La valeur par défaut est 5.

Tâches à longue durée d'exécution

Indique si la ressource doit être utilisée pour les tâches à longue durée d'exécution. Si cette option est activée, les tâches à longue durée d'exécution ne sont pas signalés comme étant bloquées. Cette option est désactivée par défaut.

Bloqué au bout de

Nombre de secondes pendant lesquelles les tâches peuvent être exécutées avant d'être considérées comme ne répondant pas. La valeur par défaut est 0.

Description

Description de la ressource.

Taille de base

Nombre de threads à conserver dans un pool de threads. La valeur par défaut est 0.

Taille maximale du pool

Nombre maximal de threads que le pool de threads peut contenir. La valeur par défaut est 2147483647.

Maintien des connexions

Nombre de secondes pendant lesquelles les threads peuvent rester inactifs lorsque le nombre de threads est supérieur à la taille de base. La valeur par défaut est 60.

Durée de vie des threads

Nombre de secondes pendant lesquelles les threads peuvent rester dans un pool de threads avant d'être purgés, que le nombre de threads soit supérieur à la taille de base ou non ou que les threads soient inactifs ou non. La valeur par défaut est 0.

Capacité de la file d'attente de tâches

Nombre de tâches soumises qui peuvent être stockées dans la file d'attente des tâches en attente d'exécution. La valeur par défaut est 2147483647.

Autres propriétés

Propriétés supplémentaires de la ressource. GlassFish Server ne définit aucune autre propriété pour ce type de ressource. De plus, ce type de ressource n'utilise actuellement aucune propriété supplémentaire.

Cibles

(Cette option apparaît si des clusters ou des instances autonomes ont été créés dans le domaine.)

Clusters et instances autonomes de la ressource. Déplacez les cibles souhaitées dans la colonne Cibles sélectionnées à l'aide des boutons Ajouter et Tout ajouter. Déplacez toutes les cibles inutiles dans la colonne Cibles disponibles à l'aide des boutons Enlever et Tout enlever.
