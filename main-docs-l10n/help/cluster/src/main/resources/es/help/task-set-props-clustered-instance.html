Definir propiedades de sólo instancia para una instancia de GlassFish Server en cluster

Las propiedades de una instancia de GlassFish Server incluyen información de configuración opcional acerca de la instancia.

  1. En el árbol de navegación, amplíe el nodo Clusters.

  2. En el nodo Clusters ampliado, seleccione el nodo para el cluster.

    Se abre la página Información General para el cluster.

  3. En la página Información General, haga clic en el separador Instancias.

    Se abrirá la página Instancias del Servidor en Cluster para el cluster.

  4. Seleccione la instancia para la cual se modificarán las propiedades de sólo instancia.

    Se abrirá la página Información General para la instancia.

  5. En la página Información General, haga clic en el separador Propiedades.

  6. Si es necesario, haga clic en el separador Propiedades de la Instancia.

    Se abre la página Propiedades de la Instancia.

    Esta página muestra una lista de las propiedades que se definen para la instancia seleccionada de GlassFish Server.

  7. Para cada propiedad que desee modificar, escriba el nuevo valor en el campo Valor de la propiedad.

  8. Defina los valores que necesite en las demás propiedades.

    1. En la tabla Propiedades Adicionales, haga clic en Agregar Propiedad.

      Se agrega una fila a la tabla Propiedades Adicionales.

    2. En el campo Nombre, escriba un nombre para la propiedad.

    3. En el campo Valor, escriba el valor.

    4. En el campo Descripción, escriba una breve descripción de la propiedad.

  9. Haga clic en Guardar.
