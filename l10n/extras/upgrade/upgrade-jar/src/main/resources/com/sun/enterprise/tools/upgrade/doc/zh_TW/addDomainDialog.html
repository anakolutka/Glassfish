【增加網域】對話方塊

使用【增加網域】對話方塊來選取網域名稱和來源與目標憑證密碼。

  1. 按一下【來源網域】下拉式清單，然後選取包含要遷移之安全性憑證的網域。
  2. 在【密碼】面板中，輸入提示您輸入的密碼。根據來源和目標安裝的類型，可能需要輸入兩個或三個密碼：
    • 在【來源 NSS】及/或【目標 NSS】欄位中，輸入 NSS 憑證檔案的密碼，該憑證檔案用於 NSS 可信任的資料庫。
    • 在【使用者金鑰資料庫】欄位中，輸入來源使用者金鑰資料庫的密碼， 該資料庫用於包含使用者金鑰組憑證的 JSSE keystore.jks 檔案。
    • 在【憑證授權單位資料庫】欄位中，輸入憑證授權單位資料庫的密碼， 該資料庫用於包含憑證授權單位憑證的 JSSE cacerts.jks 檔案。
  3. 完成後，請按下【確定】按鈕。將顯示【傳送安全憑證】螢幕，指出已在【含要傳輸的憑證之網域】面板中增加的網域。
