감사 모듈을 만드는 방법

감사는 Enterprise Server에서 오류나 보안 위반과 같은 중요한 이벤트를 기록하는 방법입니다. Enterprise Server는 모든 인증 이벤트를 로그 파일에 기록합니다. 전체 액세스 로그는 Enterprise Server 액세스 이벤트의 순차적인 추적을 제공합니다.

Enterprise Server에서는 간단한 기본 감사 모듈을 제공합니다. 자세한 내용은 기본 감사 모듈을 사용하는 방법 을 참조하십시오.

  1. 탐색 트리에서 구성 노드를 선택합니다.
  2. 구성 노드에서 보안 노드를 확장합니다.
  3. 보안 노드 아래에서 감사 모듈 노드를 선택합니다.

    감사 모듈 페이지가 열립니다.

  4. 감사 모듈 페이지에서 새로 만들기를 누릅니다.

    새 감사 모듈 페이지가 열립니다.

  5. 새 감사 모듈 페이지의 이름 필드에 감사 모듈 이름을 입력합니다.
  6. 클래스 이름 필드에 이 모듈을 구현하는 클래스의 정규화된 이름을 입력합니다.

    기본 감사 모듈의 클래스 이름은 com.sun.enterprise.security.Audit입니다.

  7. (옵션) 추가 등록 정보 섹션에서 추가 등록 정보를 지정합니다.

    등록 정보를 추가하려면 등록 정보 추가 버튼을 누릅니다. 빈 행이 표시되면 이름 필드에 등록 정보 이름을 입력하고 값 필드에 등록 정보 값을 입력합니다.

    감사 모듈에 사용 가능한 추가 등록 정보의 설명은 감사 모듈과 관련된 등록 정보를 참조하십시오.

  8. 확인을 누릅니다.
참조

  • 감사 모듈을 편집하는 방법

  • 감사 모듈을 삭제하는 방법

  • 감사 로깅을 활성화하거나 비활성화하는 방법

  • 활성 감사 모듈을 설정하는 방법

  • 기본 감사 모듈을 사용하는 방법
