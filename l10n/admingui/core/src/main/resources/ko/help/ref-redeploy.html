엔터프라이즈 응용 프로그램/모듈 재배포

엔터프라이즈 응용 프로그램/모듈 재배포 페이지를 사용하여 이전에 배포한 응용 프로그램을 재배포합니다. 페이지에는 다음 옵션이 포함되어 있습니다.

이름

재배포 중인 응용 프로그램의 이름을 표시하는 읽기 전용 필드입니다.

유형

응용 프로그램 유형을 표시하는 읽기 전용 필드입니다.

세션 유지

세션 유지 확인란을 선택할 경우 재배포 중인 응용 프로그램의 활성 세션이 보존되고 재배포가 완료되면 복구됩니다.

응용 프로그램의 활성 세션을 보존하거나 복구하지 못할 경우에는 재배포 완료 시 모든 세션을 사용할 수 없게 됩니다. 그러나 재배포가 계속 진행되고 경고가 기록됩니다.

활성 세션을 보존하기 위해 Enterprise Server는 세션을 일련화하여 메모리에 저장합니다. 세션을 복원하려면 새롭게 재배포한 응용 프로그램의 클래스 로더에서는 이전에 저장된 세션을 일련화 해제합니다.

위치

재배포 중인 응용 프로그램의 새 아카이브 위치입니다.

다음 옵션에서는 아카이브에서 액세스할 수 있는 위치 및 아카이브가 파일 또는 디렉토리인지 여부를 지정합니다.

서버에 업로드할 패키지화된 파일

아카이브는 클라이언트 시스템에 상주하거나 클라이언트 시스템에서 액세스할 수 있는 파일에 있습니다.

클라이언트 시스템은 브라우저를 통해 관리 콘솔을 보는 호스트입니다.

Application Server에서 액세스할 수 있는 패키지화된 로컬 파일 또는 디렉토리

아카이브는 서버 시스템에 상주하는 파일이거나 확장된 디렉토리의 압축 해제된 응용 프로그램입니다.

서버 시스템은 Application Server 도메인 관리 서버를 실행하는 호스트입니다.

관련 작업

  • 웹 응용 프로그램을 배포하는 방법

  • 배포된 웹 응용 프로그램을 편집하는 방법

  • 웹 응용 프로그램의 설명자를 보는 방법

  • 웹 응용 프로그램을 시작하는 방법

  • 배포된 웹 응용 프로그램을 재배포하는 방법

관련 asadmin 명령

  • list-applications(1)

  • list-modules(1)
