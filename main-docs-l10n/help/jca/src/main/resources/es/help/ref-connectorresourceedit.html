Editar Recurso de Conector

Utilice la página Editar recurso de conector para modificar la configuración para un recurso de conector.

La página Editar recurso de conector contiene las siguientes opciones.

Cargar Valores por Defecto

Botón para restaurar configuración que tiene valores por defecto a sus valores por defecto. Las configuraciones que no tienen valores por defecto no se modifican.

Nombre JNDI

Un nombre único que identifica el recurso de conector. El nombre JNDI es un campo de sólo lectura. Sólo puede especificar el nombre JNDI al crear un nuevo recurso de conector.

Nombre de pool

El pool de conexiones de conector asociado a este recurso.

Orden de despliegue

Orden de despliegue del recurso de conector.

Descripción

Descripción del recurso de conector.

Estado

Indica si el recurso está disponible en el tiempo de ejecución.

Si sólo existe la instancia de servidor por defecto, server en el dominio, la disponibilidad en tiempo de ejecución del recurso puede estar activada o desactivada. La disponibilidad en tiempo de ejecución está activa por defecto.

Si se ha creado algún cluster o alguna instancia independiente en el dominio, utilice el separador Destino para controlar la disponibilidad del recurso en tiempo de ejecución en sus destinos.

Propiedades Adicionales

Propiedades adicionales para el recurso de conector. Las propiedades disponibles dependen del adaptador de recursos para el pool de conexiones de conector. Los pares nombre-valor especificados en esta tabla se pueden usar para sustituir los valores por defecto de las propiedades definidas por el proveedor del adaptador de recursos.
