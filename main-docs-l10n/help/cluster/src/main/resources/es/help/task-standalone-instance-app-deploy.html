Desplegar una aplicación en una instancia de GlassFish Server independiente

  1. En el árbol de navegación, seleccione el nodo Instancias Independientes.

    Se abrirá la página Instancias de Servidor Independientes.

  2. En la página Instancias de Servidor Independientes, seleccione la instancia.

    Se abrirá la página Información General para la instancia.

  3. En la página Información General para la instancia, haga clic en el separador Aplicaciones.

    Se abrirá la página Aplicaciones para la instancia.

  4. En la página Aplicaciones, haga clic en Desplegar.

    Se abre la página Desplegar Aplicaciones o Módulos.

  5. En la página Desplegar Aplicaciones o Módulos, especifique la información necesaria sobre la aplicación y haga clic en Aceptar.

    Para obtener más información, consulte la ayuda en pantalla de Desplegar Aplicaciones o Módulos.
