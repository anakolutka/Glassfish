JDBC リソース

Java DataBase Connectivity (JDBC) リソース (データソース) は、データベースへの接続手段をアプリケーションに提供します。通常、管理者はドメイン内に配備されたアプリケーションがアクセスするデータベースごとに 1 つの JDBC リソースを作成します。ただし、1 つのデータベースに対して複数の JDBC リソースを作成できます。

アプリケーションは、Java Naming and Directory Interface (J.N.D.I.) API ツリーのデータソースを検索して接続プールからデータベース接続を取得してから、接続を要求します。データソースに関連付けられている接続プールが、アプリケーションへの接続を提供します。

「JDBC リソース」ページには、アプリケーションサーバーで作成されたリソースの一覧が表示されます。リソースごとに、次の情報が表示されます。

  • J.N.D.I. 名

  • 状態

  • 接続プール

  • 説明
