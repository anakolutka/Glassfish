Gérer les cibles

La page Gérer les cibles permet de modifier les clusters et les instances de serveur autonomes cible sur lesquels une application peut être activée. L'application peut être activée uniquement sur les cibles figurant dans la colonne Cibles sélectionnées.

La page Gérer les cibles contient les informations ci-après.

Cibles disponibles

Clusters et instances sur lesquels l'application n'est pas déployée.

Cibles sélectionnées

Clusters et instances sur lesquels l'application est déployée.

Ajouter

Bouton permettant de déplacer la cible sélectionnée de Cibles disponibles vers Cibles sélectionnées.

Tout ajouter

Bouton permettant de déplacer toutes les cibles disponibles vers Cibles sélectionnées.

Enlever

Bouton permettant de déplacer la cible sélectionnée de Cibles sélectionnées vers Cibles disponibles.

Tout enlever

Bouton permettant de déplacer toutes les cibles sélectionnées vers Cibles disponibles.
