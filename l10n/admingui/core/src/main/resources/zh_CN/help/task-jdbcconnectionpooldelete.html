删除 JDBC 连接池

  1. 在导航树中，展开“资源”节点。
  2. 在“资源”节点下，展开 "JDBC" 节点。
  3. 在 "JDBC" 节点下，选择“连接池”节点。

    此时将打开“连接池”页面。

  4. 在“连接池”页面中，选中要删除的连接池左侧的复选框。
  5. 单击“删除”。
