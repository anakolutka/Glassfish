查看 GlassFish Server 博客

GlassFish Server 博客提供了来自 GlassFish Server 社区的新闻，以及指向有关 GlassFish Server 的其他信息的链接。

  1. 在导航树中，选择“常见任务”节点。

    此时将打开“常见任务”页。

  2. 在“GlassFish 新闻”下，单击“博客”链接。

    此时将在单独的浏览器选项卡或窗口中打开 GlassFish Server 博客。
