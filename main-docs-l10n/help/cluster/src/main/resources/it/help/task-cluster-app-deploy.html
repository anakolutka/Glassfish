Per distribuire un'applicazione a un cluster

  1. Espandere il nodo Cluster nell'albero di navigazione.

  2. Selezionare il nodo per il cluster sotto il nodo Cluster espanso.

    Viene visualizzata la pagina Informazioni generali per il cluster.

  3. Nella pagina Informazioni generali fare clic sulla scheda Applicazioni.

    Viene visualizzata la pagina Applicazioni per il cluster.

  4. Nella pagina Applicazioni, fare clic su Distribuisci.

    Viene visualizzata la pagina Distribuisci applicazioni o moduli.

  5. Nella pagina Distribuisci applicazioni o moduli, specificare le informazioni richieste sull'applicazione e fare clic su OK.

    Per i dettagli, vedere l'argomento Distribuisci applicazioni o moduli della Guida in linea.
