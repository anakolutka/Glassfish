일반 작업

일반 작업 페이지에는 일반적인 관리 콘솔 작업에 대한 바로 가기가 있습니다.

일반 작업 페이지를 사용하여 다음 작업을 수행합니다.

  • Sun Connection을 사용하여 Sun GlassFish Enterprise Server를 등록하는 방법

  • Sun GlassFish Enterprise Server의 지원 페이지를 보는 방법

  • 웹 응용 프로그램을 배포하는 방법

  • JDBC 연결 풀을 만드는 방법

  • 모니터링 데이터를 보는 방법

  • 모니터링 서비스를 구성하는 방법

  • 설치된 구성 요소를 보는 방법

  • 설치된 구성 요소를 업데이트하는 방법

  • 애드온 구성 요소 설치 방법

또한 일반 작업 페이지는 Enterprise Server 설명서에 대한 링크를 제공합니다.
