Aktualisieren mithilfe des Upgrade-Assistenten

Spezifizieren Sie mithilfe des Upgrade-Assistenten die Installationsverzeichnisse der Quell- und Zielserver, den Admin-Benutzer, das Admin-Passwort und das Master-Passwort des Quellanwendungsservers.

  1. Geben Sie im Feld für das Quellinstallationsverzeichnis den Speicherort der aktuellen Installation an, aus welcher die Konfiguration importiert werden soll.  Sie müssen das Domänenverzeichnis eingeben. Beispiel: <install-root>/domains/domain1.

  2. Geben Sie im Feld "Zielinstallationsverzeichnis" den Speicherort der Application Server-Installation ein, auf welche die Konfiguration übertragen werden soll. Sie müssen in dieses Feld die Domänen-Root eingeben.
  3. Geben Sie den Admin-Benutzernamen, das Admin-Passwort und das Master-Passwort des Quellanwendungsservers an.
  4. Klicken Sie anschließend auf "Weiter".
  5. Das Fenster "Upgrade-Ergebnisse" wird geöffnet und zeigt den Upgrade-Status an.
  6. Klicken Sie auf "Fertig stellen", um das Upgrade-Tool zu schließen, wenn der Upgrade-Prozess abgeschlossen ist.
