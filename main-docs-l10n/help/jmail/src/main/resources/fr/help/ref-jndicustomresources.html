Ressources personnalisées

Une ressource personnalisée spécifie une fabrique personnalisée d'objets de ressource au niveau du serveur.

La page Ressources personnalisées permet de configurer des ressources JNDI personnalisées.

Pour chaque ressource personnalisée, les informations suivantes sont fournies.

Nom JNDI

Nom unique identifiant la ressource personnalisée.

Activé

(Cette colonne est affichée si seule l'instance de serveur par défaut server existe dans le domaine.)

Coche si la ressource est activée ou X si la ressource est désactivée.

Statut

(Cette colonne apparaît si des clusters ou des instances autonomes ont été créés dans le domaine.)

Nombre de clusters et d'instances autonomes sur lesquels la ressource est ciblée et nombre de cibles sur lesquelles la ressource est activée. Par exemple, Activé sur 2 cibles sur 4 signifie que la ressource est ciblée sur quatre clusters et instances autonomes et activée sur deux de ces quatre cibles.

Type de ressource

Type de la ressource personnalisée.

Description

Description de la ressource personnalisée.

La table Ressources contient également les options suivantes :

Nouveau

Bouton permettant de créer une ressource personnalisée.

Supprimer

Bouton permettant de supprimer les ressources personnalisées sélectionnées.

Activer

Bouton permettant d'activer les ressources personnalisées sélectionnées.

Désactiver

Bouton permettant de désactiver les ressources personnalisées sélectionnées.
