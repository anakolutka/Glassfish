编辑 File 领域用户

使用“编辑 File 领域用户”页面可以修改用户对当前领域的访问权限。

领域名

显示当前所选领域名称的只读字段。

用户 ID

显示您要编辑的帐户名称的只读字段。

组列表

此用户将属于的组。使用逗号分隔多个组名称。

新密码

此用户的新密码。

确认新密码

此用户的新密码。
