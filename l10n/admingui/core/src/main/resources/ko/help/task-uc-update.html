설치된 구성 요소를 업데이트하는 방법

설치된 구성 요소 업데이트에는 다음 작업이 포함됩니다.

  • 수정된 파일만 네트워크 저장소에서 시스템으로 전송

  • 시스템의 Enterprise Server 설치 이미지 내에서 수정된 파일만 업데이트

  • 업데이트된 구성 요소 사이 새 종속성(해당하는 경우) 적용

  1. 탐색 트리에서 업데이트 도구 노드를 선택합니다.

    설치된 구성 요소 페이지가 표시됩니다.

  2. 설치된 구성 요소 페이지에서 사용 가능한 업데이트 탭을 누릅니다.

    사용 가능한 업데이트 페이지가 표시됩니다.

  3. 사용 가능한 업데이트 테이블에서 업데이트할 구성 요소를 선택합니다.

    업데이트를 사용할 수 없는 경우 테이블이 비어 있습니다.

    • 개별 구성 요소를 선택하려면 구성 요소 이름 옆에 있는 확인란을 선택합니다.
    • 모든 구성 요소를 선택하려면 테이블 헤더에서 모두 선택 아이콘을 누릅니다.
    • 모든 구성 요소를 선택 해제하려면 테이블 헤더에서 모두 선택 해제 아이콘을 누릅니다.
  4. 설치를 누릅니다.
  5. 사용권 계약에 동의합니다.

    Enterprise Server는 설치가 완료되었음을 확인합니다. 구성 요소가 사용 가능한 업데이트 테이블에서 제거됩니다.

  6. 도메인 관리 서버를 다시 시작합니다.

    자세한 지침은 다음 설명서를 참조하십시오.

    • Sun GlassFish Enterprise Server v3 Prelude Administration Guide의 To Stop a Domain (or Server)

    • Sun GlassFish Enterprise Server v3 Prelude Administration Guide의 To Start a Domain (or Server)
