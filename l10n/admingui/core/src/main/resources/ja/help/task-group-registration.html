Sun GlassFish Enterprise Server の登録

Sun GlassFishTM Enterprise Server のユーザー登録をすると、次の情報を得ることができます。

  • パッチ情報およびバグのアップデート

  • スクリーンキャストおよびチュートリアル

  • ニュースおよびイベント

  • サポートおよびトレーニングの提供

登録には次の操作が含まれます。

  • 「Sun Connection に Sun GlassFish Enterprise Server を登録する」

  • 「Sun GlassFish Enterprise Server の「Support」ページを表示する」
