收集和下載網域日誌

您可以收集所指定 GlassFish Server 執行處理或叢集的所有可用日誌檔，然後將其編譯成單一 ZIP 存檔。

備註:

如果指定叢集，則 ZIP 存檔也會包括網域管理伺服器 (DAS) 的日誌檔。

  1. 在瀏覽樹狀結構中，選取「網域」節點。

    「網域屬性」頁面會隨即開啟。

  2. 選取「網域日誌」標籤。

    「網域日誌」頁面會隨即開啟。

    如果叢集存在，則會顯示叢集和執行處理的下拉式清單。如果叢集不存在，則只會顯示執行處理的下拉式清單。

  3. 選取「叢集」或「執行處理」。

  4. 選取要收集和下載日誌檔的叢集或執行處理。

  5. 按一下「收集日誌」。

    您瀏覽器的下載機制會提示您下載 log-files-name.zip 檔案，其中 name 是叢集或執行處理的名稱。
