Anwendungsziele

Verwenden Sie die Seite "Anwendungsziele", um Zielcluster und eigenständige Serverinstanzen anzuzeigen, auf denen die Anwendung aktiviert werden kann.

Die Seite "Anwendungsziele" enthält die folgenden Informationen.

Zielname

Der Name des Clusters oder der Standalone-Serverinstanz.

Aktiviert

Zeigt True an, wenn die Anwendung auf dem Ziel aktiviert ist, und False, wenn die Anwendung deaktiviert ist.

LB aktiviert

Zeigt True an, wenn die Anwendung für das Load Balancing auf dem Ziel aktiviert ist, und False, wenn die Anwendung für das Load Balancing deaktiviert ist.

Virtuelle Server

Um die Anwendung mit bestimmten virtuellen Servern auf dem Ziel zu verknüpfen, wählen Sie für das Ziel die Option "Virtuelle Server verwalten".

Die Tabelle "Ziele" enthält darüber hinaus die folgenden Optionen.

Ziele verwalten

Schaltfläche zum Verwalten von Anwendungszielen.

Weitere Aktionen

Dropdown-Liste mit folgenden Aktionen.

Aktivieren

Aktion zum Aktivieren der Anwendung auf dem ausgewählten Ziel.

Deaktivieren

Aktion zum Deaktivieren der Anwendung auf dem ausgewählten Ziel.

Load Balancer aktivieren

Aktion zum Aktivieren der Anwendung für das Load Balancing auf dem ausgewählten Ziel.

Load Balancer deaktivieren

Aktion zum Deaktivieren der Anwendung für das Load Balancing auf dem ausgewählten Ziel.
