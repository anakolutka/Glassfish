Cómo configurar el servicio de supervisión

  1. En el árbol de navegación, expanda el nodo Configuración.
  2. En el nodo Configuración, seleccione el nodo Supervisión.

    Se abrirá la página Servicio de supervisión.

  3. En la lista desplegable Servicio HTTP, seleccione BAJO o ALTO para habilitar la supervisión del servicio HTTP.

    No existe diferencia alguna entre los niveles de supervisión que ofrecen las opciones BAJO y ALTO de esta versión. Para deshabilitar la supervisión del servicio HTTP, seleccione DESACTIVADO. El valor predeterminado es DESACTIVADO.

  4. En la lista desplegable del contenedor Web, seleccione BAJO o ALTO para habilitar la supervisión del contenedor web.

    No existe diferencia alguna entre los niveles de supervisión que ofrecen las opciones BAJO y ALTO de esta versión. Para deshabilitar la supervisión del contenedor web, seleccione DESACTIVADO. El valor predeterminado es DESACTIVADO.

  5. En la lista desplegable JVM, seleccione BAJO o ALTO para habilitar la supervisión de JVM.

    No existe diferencia alguna entre los niveles de supervisión que ofrecen las opciones BAJO y ALTO de esta versión. Para deshabilitar la supervisión de JVM, seleccione DESACTIVADO. El valor predeterminado es DESACTIVADO.

  6. En la lista desplegable Conjuntos de subprocesos, seleccione BAJO o ALTO para habilitar la supervisión de conjuntos de subprocesos.

    No existe diferencia alguna entre los niveles de supervisión que ofrecen las opciones BAJO y ALTO de esta versión. Para deshabilitar la supervisión de conjuntos de subprocesos, seleccione DESACTIVADO. El valor predeterminado es DESACTIVADO.

  7. Haga clic en "Guardar".
