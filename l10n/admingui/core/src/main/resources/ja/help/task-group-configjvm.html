JVM の設定

JavaTM 仮想マシン (JVMTM) は、物理マシン上で Java バイトコードを実行できるようにします。JVM 設定を調整することで、パフォーマンスの改善や記憶域割り当てエラーの回避を行うことができます。

JVM の設定には次の操作が含まれます。

  • 「JVM 一般設定を行う」

  • 「JVM パスの設定を行う」

  • 「JVM オプションを設定する」

  • 「セキュリティーマネージャーを無効にする」

  • 「JVM のプロファイラ設定を行う」

  • 「詳細アプリケーション設定を行う」

  • 「詳細ドメイン属性を設定する」

注 – 「Java 仮想マシン」および「JVM」という用語は、Java プラットフォームの仮想マシンを意味します。
